各位是写技术文章的吗？这次我就写一下**写技术文章本身**。

# [](#should-i-write-technical-articles)该不该写技术文章？

即使你不是工程师**“信息传播”很重要**。这可能不是一个自由职业者，但也导致从那里工作。**与其写**，不如写。

当然，如果是以接活为目的而写，就应该写，因为这是一种精细的销售活动。拨打最新的信息等等将是品牌。

对于那些没有这样做的人来说，尽可能多的记录日志是一个好主意，比如被错误困扰的内容。3 个月后，你会得到同样的错误，你会看到它。哈哈

然而，这是微妙的写，直到不可能。为了不减少写代码的时间，我们还是放宽一点吧。

# 如果它写了，我愿意被阅读

首先，我们对技术文章进行了分类。对于翻译文章也是如此，内容本身并不是下面的感受。

*   趋势文章
*   总结烦恼
*   总结你所学内容的文章
*   作为一个案例介绍你在做什么
*   写关于人们问什么的文章
*   诗

对于每一个问题，我都会写下需要阅读的要点。

# [](#trend-article)潮流篇

反正潮流篇**速度重要**！

让我们快速发布 3 天内邻居说“哦，新图书馆出来了”或它是什么的文章。

而且趋势文章简洁是相当重要的。因为人们最想知道的是最后的总结。

所以，如果你能写出“我懂了”这样的内容，你就可以说“哦~ React hooks，完全懂了”并分享出来。

它可能根本不是一个博客。有时候在 Twitter 或者 Youtube 上也不错。

# [](#summarize-the-troubles)总结烦恼

无论如何，错误日志很重要！

如果可能的话，让我们把出现**错误信息的最重要的措辞放在标题上**。

这是为了让那些犯了错误的人能够到达。换句话说，不要期望它在写完之后马上被阅读。

之后，既然要上 google，起码 SEO 重要！如果输给“stackoverflow 的零回答问题”就太可怜了。哈哈

就算不是错误，也让它成为一个看似 googling 的标题吧，比如“foo 不是工作”。

但是，如果你写一篇小众错误措辞的文章，感觉还行。

当你忘记的时候，你会感谢我的过去。

## [](#article-summarizing-what-you-studied)总结你所学内容的文章

这很重要**全面性**！

即使是写对比文章，重要的是要认真写参考文章，即使很长也要写得全面。

“用这个和这个，最后哪个最快……”？我想避免像离开这样的事情。

此外，这也是重要的搜索引擎优化，因为它不会被阅读，除非它显示在谷歌的第一页。

## [](#introduces-what-you-were-making-as-a-case)以案例的形式介绍你在做什么

紧紧**写广告**很重要！

如果一个人最终没有意识到它所创造的东西，那它就没有存在的意义。

理想是变成“产品发布会”→“你们这里怎么做这个服务？”→“哇~评论文章有一定的份额和分享”。

**我只想做广告的文章都是噪音**

## [](#article-to-write-about-what-people-asked)写什么人问什么文章

把别人问的内容放到文章里的时候。即使被同一个人问两次，喊出文章的网址也是赞赏。

重要的是听到的人是否满意。如果一个人询问内容并检查它会更好吗？

之后，**被问肯定是被要求**既然是文章，如果有这样的机会，最好认真写在文章里。

# [](#poem)诗

**情感内容重要**。让我们尽可能地打开情感。如果你有情感内容，那么你不需要任何东西。

会被读得更赤裸裸。那就是互联网。

# [](#the-title-is-important)标题重要

重要的是加入你所使用的技术内容或要减去的东西。我想让它不超过 30 个字符之类的(这不是这样的限制，谷歌搜索削减字母)

## 让我知道你写这篇文章的日期

个人博客可能会忘记首先显示日期。当你写旧文章的时候，你以后会有麻烦的。

## [](#write-down-the-operating-environment-etc-as-much-as-possible)记下操作环境等。尽可能

而且我会尽可能多的贴出版本信息。最好是带有 codepen 的复制环境。

# [](#should-i-write-even-if-it-is-short)即使很短也要写吗？

不管多短，写总比不写好。

```
Error Wording

In the third line of foo.js, bar

I fixed it after writing this line. 
```

哪怕只是三行也是值得的！

它可能会帮助有同样错误的人。

即使信息变得陈旧和微妙，谷歌的工作就是不离开那篇文章，所以**你不必担心。**

如果有什么我能写的起的，那就写吧。永远不会是负面的。
