# 哪个虚构人物最能描述你的工作角色？

> 原文:[https://dev . to/alanhylands/which-ficial-character-best-describe-your-job-role-3m5k](https://dev.to/alanhylands/which-fictional-character-best-describes-your-job-role-3m5k)

我们有一些关于“你的职位是什么？”的好帖子关于 Dev 的问题([这里](https://dev.to/ben/whats-your-job-title-and-why-is-that-your-title-57h1)和[这里](https://dev.to/wuz/what-is-your-job-title-5545))。但我对职称有意见。他们总是不告诉我们你每天坐在椅子上 8 个多小时到底在做什么。

我想我们可以从中得到一点乐趣，看看是否有人能展示出他们认为哪个虚拟角色最能代表他们在工作中真正做的事情。可能来自一本书、一部电视剧、一部电影、漫画、歌曲，无论什么——只要你能告诉我们哪些特征或行为与你的角色有关。

## [](#ill-go-first)我先走。

[![Which character is Alan like?](../Images/23953d0ba341a42cf2caf7e6b366a943.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--Ec8gblic--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/dgrtwobcej4w8m69q9dl.jpg)

我的职位是“零售商业卓越总监”。(泄露的很少，不是吗？)如今，我的角色最好被描述为《低俗小说》中的温斯顿·沃尔夫和《律政俏佳人》中的 T2·连姆·尼森的结合体。

我用我的“特殊技能”来解决难题。也许这两个人处理问题的方式不太一样，但是...数据会变得很乱。

加入一点《教父》中的汤姆·哈根(Tom Hagen )( T1 ),在企业的不同派系和领域之间进行调解，以传达行政命令。当奥比万·克诺比成为原力的一员后，我帮助指导和建议人们研究我们的系统和商业数据的神秘艺术。你已经成功了 90%。

(是的，这个问题我也思考过！)

## [](#its-not-just-a-bit-of-fun)不仅仅是一点乐趣。

你永远不知道你会在聚会或网上和谁聊天。回答不可避免的问题“你是做什么的？”用“我是一个<insert generic="" meaningless="" job="" title="">”真的不能完全公正地对待你。</insert>

如果有人告诉我他们是狼和连姆·尼森的混合体，尽管我可能会给他们一个老白眼，所以不一定要押韵。但是除了通常的“我写代码”/“我分析数据”式的回答之外，至少要了解你**实际上**在做什么的主要概念。你永远不知道接下来的对话会把你带到哪里。

我进行过分析师面试，在面试中，候选人完全不能偏离他们(保守地)写在纸上的东西。有时候读起来就像是直接从他们上次被雇佣的招聘广告上抄袭来的。我从来没有信心，他们可能真的有街头智慧来操作，而不需要一直手握着。没有人希望这样。
