# 未来属于我们所有人

> 原文:[https://dev . to/dk reidler/the-future-own-to-all-of-us-1 h17](https://dev.to/dkreidler/the-future-belongs-to-all-of-us-1h17)

我 10 岁的女儿正在学习编码。断断续续地，她使用可汗学院和其他工具，用代码画画，用代码让机器人化身四处走动并遵循指示...她认为她只是在玩。但是*我也在*学习编码，尽管我深入 React 和 Redux 教程，我看到了我正在积极学习并试图实现的东西和她在玩耍状态中自然吸收的东西之间的相似之处。我确保尽量让她对编码的亲和力正常化(玩！)，告诉她有一天能够看到一个问题并以自己的方式提出自己的解决方案是多么酷的事情；变得独立，成为变革的力量。不是以一种“女孩不喜欢科技，所以你喜欢很酷”的偶然性别歧视的方式，而是以一种“太棒了，我喜欢你对这个感兴趣”的整体方式。

我希望看到整个社区接受这种方法。太多时候，我觉得我们都还在为达到非男性程序员的最低数量而沾沾自喜...当仅仅是事实，它不接近 50%是一个巨大的问题。(对我来说)解决办法不是对那些已经上山的人小题大做(尽管它*很棒，不要误会我的意思),而是致力于*平整*这座山，这样下一代就不会觉得多年后他们在克服同样的问题。*

我指的不是不真诚的“这没什么大不了的，因为女性已经获得了平等而忽视她们的成就”，而是“积极地拆除障碍和阻碍”。女性已经(一次又一次地)证明了她们可以和男性同事一样胜任任何工作。年轻女性不应该被期望不断地重复证明同样的事情。让更多女性的声音出现在播客和专题讨论中。在视频和教程中多放一些小众的声音。有这么多，他们实际上只是背景的一部分，而不是仅仅因为在那里而被放在基座上的象征，同时仍然强化了首先到达那里是多么困难的形象。因为这种形象同样具有破坏性，同样令人生畏，就好像我们从一开始就没有取得任何进展一样。

我是这样想的:从前，女人是不允许在剧院里演戏的(抱歉，*剧院*)。只有男人才能表演，如果需要的话，可以男扮女装。从莎士比亚时代的英格兰快进到今天，注意到电视上性别平等的相对增加。是的，有成长的空间，但至少现在唯一的男扮女装的人真的，真的想在那里。相比之下，我们在代表少数民族方面相对失败。除了一些引人注目的例外，你仍然很少在针对美国中产阶级观众的电视节目中找到多个少数民族角色。令人高兴的是，例外证明了他们打破的潜规则的愚蠢，所以即使这样也在慢慢向好的方向发展。网飞上的“Brainchild”可能是应该可能的终极表达:一个由印度裔美国女演员主持的科学节目，她让好奇号变得很酷。甚至是科学权威？艾莉·沃德。作为一名白人男性，我发现我缺乏代表性...提神醒脑。

一般来说，编码和技术也是如此。如果我的女儿看到自己在周围世界的反映，她将更有可能坚持她的“有趣的游戏”。女性需要无处不在，而不是被供奉。少数民族需要无处不在，而不仅仅是装饰门面，提供多样性的外表。希望有很多读者告诉我我是个白痴，因为他们就是这么做的。我希望如此。我想被证明是错的。我想环顾四周，看到一个多元化、多种族、热情的社区，几乎任何人都可以自称是他们自己的。

唯一需要克服的障碍是自我怀疑和冒名顶替综合症...我们也不需要在列表中添加真正的外部障碍。

今后..属于我们所有人。
