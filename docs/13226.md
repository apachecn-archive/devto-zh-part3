# 最积极的改变来自积极的怀疑

> 原文:[https://dev . to/ilonacodes/the-most-positive-changes-through-positive-questions-1n 61](https://dev.to/ilonacodes/the-most-positive-changes-through-positive-doubts-1n61)

每个人都有感觉不完美和情绪疲惫的时候。无论发生什么，我们每天都在不断怀疑自己。而且，就算是成功人士，在生活中也经历过这种感觉。

不幸的是，我们很多人都倾向于关注消极的方面和事件。我做得太多了，有一次在我的一次“重生”后，我停止了这样做。

我也听到脑子里那些声音像“你不行，你不够好，没人喜欢你，你不酷，你好讨厌……”日复一日。

最后，不管你有多成功，不管有多少人喜欢你，甚至崇拜你。你仍然怀疑和质疑自己。

你可以在任何领域成为第一，但有时你仍然会认为自己不够好。

这只是人之常情。

大约两年前，当我开始写博客时，我的脑海里总是想着:“你真是个笑话，你认为你能写吗？你没那么资格，闲下来就不干了。”

老实说，我决定不放弃并继续写作的唯一原因是，我明白我最终不会失去任何东西，只能从中受益。

所以，我很高兴我在那一刻做出了正确的决定，现在我无法想象没有博客的生活。除此之外，想想如果我经常屈服于这些声音，我会错过多少机会，这让我感到害怕。

重点是，我们将永远怀疑自己，成为我们最残酷的批评家。即使我们已经证明了自己有能力、聪明、美丽和成功。一次又一次，这些都会在那里。

这种自我怀疑没什么，但是低价值的低自信加上对自己无能的坚定信念会严重影响我们的生活。

但是，我觉得有点自我怀疑是正常的，有时候甚至是健康的。因为它防止一个人超越过度自信和傲慢之间的界限。

怀疑我们自己和我们的能力有一个非常麻痹的效果，但另一方面，它也激励我们找到不同的方法来管理我们内心的恐惧。

我把怀疑分成两组——积极的和消极的。这里有一个我能同意的好的和坏的怀疑的定义。本质上:

善意的怀疑是一种建设性的怀疑，出于对你成功的渴望，怀疑风险和陷阱。

坏怀疑是一种破坏性的怀疑主义，它停留在问题上，希望你的努力会失败。

## [](#benefits-of-positive-doubts)正面怀疑的好处

观察周围的其他人和我自己，我可以说怀疑自己的人一般都比较敏感。此外，积极的怀疑会引发成千上万个“为什么”的问题，扩大好奇心，打开我们的思维去寻找更多的观点并质疑它们。

1.  积极的自我怀疑可以刺激我们设定个人目标或行动计划，并做好充分准备迎接任何挑战。

2.  积极的自我怀疑的另一个光明面是，有这种经历的人通常会更努力地工作:他们有这种卓越的职业道德，这使他们在自己工作的领域里变得专业。

3.  积极的自我怀疑也鼓励仔细检查一个人的清单，以确保一切都是有序的，并意识到完成一项任务/待办事项所需的所有事情或技能或努力都被学习和应用了。

4.  通常，积极的怀疑会阻止我们停止创造。这里我的意思是我们用它们来提高自己。觉得自己不够优秀？然后训练！练习！获取你能找到的任何有用的资源。挑战自己成长。把这种积极的怀疑作为你提升自己和技能的灵感。

你看，坦率地说，积极的自我怀疑是职业发展和成就的重要组成部分。

当然，在追求目标的过程中，我们会犯很多错误，怀疑很多事情。然而，如果没有自我怀疑，我们就会停止自我成长。

你有导致焦虑和抑郁的自我怀疑吗？如果是的话，你有没有尝试过将你的自我怀疑转化为对生活的积极输出？怎么会？
