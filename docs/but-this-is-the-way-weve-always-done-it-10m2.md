# 但这是我们一贯的做法

## 这里有些不对劲

你花了几天时间调查一些事情。每次你写报告的时候都会有这种持续不断的抱怨，而且有些事情就是让你不舒服。

所以，在你跳，两脚。勇敢的心，坚定的意志，有勇无谋，足以忽略那些认为你在徒劳无功地浪费时间的疑虑。

你想有所作为。纠正错误。为未来几代运行报告的 BI 分析师重新设定真正的路线。

带着你的真理之剑和美德之盾，你继续前进，在遗留的意大利面条式代码和堆积如山的平面文件输入中挖掘，向着你的目标前进。

## 宾果游戏

然后你就找到了。确凿的证据。你知道的大规模杀伤性武器数据就藏在那里，现在你的手就在上面。

几年前，一些提线木偶在他们应该离开时加入了内部加入，这几年来，它将整个报告吹进了随机数生成器的领域。

一个哈莱姆福音唱诗班在你的脑海中响起，向天堂歌唱哈利路亚。

詹姆斯·布朗穿得像个牧师，在蹦床上又跳又跳。

一束光透过窗户照进来，以来自数据之神的清晰信息照亮你。

你是数据 Neo，被选中的一个，给原力和任何你想放在这里的流行文化图标带来平衡。

## 昂首阔步走向老板的办公室

你迫不及待地想把它带给你的老板，向他展示你是一个多么聪明、聪明、有动力的分析界的坏蛋。你飘飘然。

像蝴蝶一样飞舞，准备像蜜蜂一样蛰人，但这是什么？他看起来不太受欢迎。

恰恰相反。他的上唇向上翘着。现在他的双唇紧闭，眼睛眯了起来。

额头皱了起来，他竖起手指，看着屏幕上你的对比报告，几乎一分钟都没有呼吸。

## 哦他妈的。

看起来不太好。你的手心出汗了。膝盖发软。你的手臂很重。你脑海中的音乐已经从《天使的主人》切换到了《哦，福尔图娜》,而且声音越来越大了。

不祥的预感像一个铅制保龄球一样盘踞在你的胃里，当你的老板转过身来直视你时，它开始缓慢而稳定地进入你的喉咙。

“哦，该死的，我做了什么？”重复播放。你认为它在你的脑袋里，但它可能在办公室的广播系统里。

哦见鬼。

审判时间到了。他停顿了一下，说了一些让你震惊的话。

“是啊，你说得对”。

## 现在说什么？

你的心脏又开始跳动了，但是还有一个问题。

“尽管如此，我们还是无法改变它。与其告诉任何人我们在过去的五年里犯了一个错误，而且从现在开始它将是正确的，不如坚持错误”。

打扰一下。请再说一遍。我没听错吧？

因此，最好是让一份报告始终不正确，而不是纠正它，因为这将破坏所有的历史趋势，并通过改进某些东西实际上使该部门看起来更糟？

或者更糟糕的是，只要总体趋势是正确的，就故意报告不正确的数据，因为这才是“真正重要的”吗？

## 事实往往比小说更离奇

我从世界各地听到的故事会让普通分析师汗毛直竖，屁眼噘起。

如果你没有经历过这种“但这是我们这里一直做的方式”的特殊变化，你只是没有足够努力去解决一些你应该不去管的事情。它通常伴随着总是运行报告的资深者的睁大眼睛的死亡的锐利凝视。

站在经理的立场上，我当然不会接受这种态度，但那样的话，我就是一个十足的钻石王老五。总是踢未来的门。不断推动改进，从看似干涸的源头挤出最后一点进步。
