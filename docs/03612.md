1.  挑一个问题
2.  使用开源解决这个问题
3.  成为批判者，在系统中寻找错误。
4.  不要坏嘴，只是提出问题
5.  主动提出修理它们
6.  给出解决方案，并与他们合作使之更好
7.  祈祷所有的神，希望你的修复被合并。
