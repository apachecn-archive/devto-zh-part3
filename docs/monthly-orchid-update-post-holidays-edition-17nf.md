# 即将到来

作为兰花永无止境的追求成为代码文档最好的工具的一部分，我很兴奋地说，对 Groovy 代码文档的研究正在进行中！这在很大程度上仍处于探索阶段，但我希望在接下来的一个月左右，我能够完成其中的一大部分。请在这里继续关注它的最新进展！

我还在为我的公司做一个秘密项目，一个有趣的小型无服务器 SlackBot:

*   在 Node.js 中运行
*   这是用 Kotlin/JS 编写的
*   并被记录为兰花
*   产品页面和帮助表格也由兰花管理
*   都是和格雷一起组装的
*   并且使用 Netlify 服务(虽然 Netlify 没有数据库，所以我使用 Firebase)

它还没有完全完成，但欢迎您随时预览 [CrederaBot](https://crederabot.netlify.com/) ！您甚至可以在自己的 Slack 团队中安装这个 bot，但是如果我需要更改 DB 模式，我不能保证我不会删除您的数据😇。
