### 机器人网络又名僵尸网络

被称为恶意软件，由于它的工作控制大量的计算资源，干扰系统主机，未经同意，从业主与 it 代理-僵尸网络的一部分，在你的电脑。僵尸网络采用客户端-服务器层，以脚本或编译代码的形式使用客户端或代理服务器提供 web 服务。

通常，一个坏的僵尸网络会同时提供诸如传播方法、DDOS、甚至证书收集工具包等恶意功能。不，你说让僵尸网络做好事？当然了。

让我们改变它上面的一些东西，我们改变它以方便我们管理或轻松使用捆绑的计算资源，就像如果僵尸网络被别人用来控制你的资源和伤害行为，为什么你不做你自己的僵尸网络来控制你的资源和有益的功能。这样一个远程你的电脑备份，跟踪你的手机，当你忘记它，或浇水植物使用 arduino 集成到你的僵尸网络服务器。在浏览器中，让伟大的任务变得简单。
