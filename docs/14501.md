# 如何理解管理者？

> 原文：<https://dev.to/rlxdprogrammer/how-to-understand-the-managers-1p7h>

所有的软件开发人员都需要与经理合作，这也有助于你和你的经理保持良好的关系，因为大多数经理可以决定你的晋升，工资，项目等。开发人员的个人目标和经理的目标经常是相互冲突的，这意味着经理和开发人员之间的冲突。为了避免这种情况，作为第一步，所有开发人员都需要了解经理的职责和目标。利用这些知识，你可以从与经理的讨论中获得更好的结果。

首先简单地看一下作为一名开发人员，你的主要目标是什么:你希望有足够的时间以舒适的方式完成任务，你希望使用最现代的技术并尝试你所有的想法，你希望参加技术培训，你需要一份好的薪水和良好的工作生活平衡。这些要点中的大部分都会让你的经理付出代价，但是如果你能让所有经理明白这对他们也是有益的，你就可以向他们保证你的需求。

正如许多公司一样，许多不同的经理职位都有不同的角色描述。我想重点介绍两种主要类型的经理:负责项目或子项目的项目经理和负责开发团队的团队经理。在某些情况下，这两项责任由同一个人承担。在其他情况下，这些角色有不同的名称。当然，还有其他经理角色，比如销售经理，但大多数情况下，他们与开发人员的联系较少。

# 团队经理

团队经理有一个开发团队。通常团队经理的职责如下:

*   寻找合适的团队成员(进行面试等。)
*   支持新团队成员融入团队
*   监控他们的表现和技能
*   根据项目需求，让他们在确定的方向上提高技能(例如，送他们去培训，为他们找到合适的导师等。)
*   为正确的开发人员提供正确的项目技能和经验
*   收集项目经理的反馈
*   与团队成员定期举行反馈会议
*   为团队设定目标
*   为团队组织业余活动
*   做出薪资变动的决定
*   决定休假申请
*   将不适合团队的成员从团队中移除
*   定期向高层管理人员反馈团队的成果

太多了，对吧。为了能够做到这一点，团队经理需要有一个技术概述，以确定所需的技能和良好的人际交往技能。

还有一件更重要的事情:大多数团队经理都有自己的预算。这意味着团队经理也要负责一些钱。这里的收入来自团队完成的项目。例如，如果团队完成了项目的一个里程碑，那么团队预算中会有一个预定义的固定价格。

成本如下:团队成员的工资、培训成本、设备成本(计算机等)。)、办公室费用(租赁费、咖啡等。)、脱产计划的费用等。

除非收入大于成本，否则一切都很顺利，否则团队经理会有一些问题。项目价格通常是这样设计的，如果团队能够在计划的时间框架内完成项目，团队就有利润，一切都很好。如果拥有合适技能的积极进取的人在项目中工作，这将会发生。如果你在这样的情况下(你在当前的项目中工作有效且成功)，你将有很好的机会获得更高的薪水。例如，如果你能证明参加培训能让你在项目中更有效率，那么你就有很好的机会参加培训。否则，你的经理不会支付培训费用，如果培训不能使你当前(或即将到来的)项目的日常工作更有效的话。

让我们考虑一下像新团队成员的加速时间这样的情况。这就是所谓的管理费用，它花费很多(工资、培训等)。)但是不直接挣钱。这种时期需要尽可能短地计划。这就是为什么频繁地改变项目不是一个好主意。

总结:你的团队经理的目标是保持正的预算平衡，所以无论你向他提出什么要求，都需要提到这样一个事实，即你在过去为团队赚了很多钱，或者投资会让你在未来赚更多钱。当然使用这些词不是最好的主意，但是你应该把这些想法记在心里。

# 项目经理

项目经理负责一个项目或子项目。项目经理的目标是让客户满意，让项目对公司有利。因此，项目经理的任务如下:

*   定期与客户沟通
*   为项目制定时间计划(包括主要里程碑等。)
*   创建成本计划
*   创建项目的资源规划(在项目的不同阶段需要多少开发人员以及他们的技能)
*   跟踪项目的所有进展
*   定期向高级管理层汇报
*   组织项目会议

这里重要的是:让你开心不是项目经理的目标。他的目标是按时到达里程碑。如果为此需要加班，项目经理不会介意。如果你做得不好，项目经理会向团队经理请求另一个开发人员。经理的目标只是保持计划。

项目经理还负责项目预算。从成本的角度来看，它意味着:收入是客户支付的价格。成本是为开发人员的工作支付给团队经理的钱、会议费用、软件许可证和差旅费。

因此，如果一个项目经理让你加班，他只是想实现他的个人目标。另一方面，与项目经理保持良好的关系是件好事，因为他会向团队经理提供关于你工作的反馈，这会极大地影响你的职业生涯。为了能够建立良好的关系，试着在计划阶段帮助他做出尽可能好的估计，然后努力实现它们。

总之，和你的经理保持良好的关系是件好事。不要忘记:没有他们你就没有工作，他们也是大机器的一部分，就像你一样。为了能够有一个良好的关系，总是试图理解他们的目标，并试图支持他们实现这些目标。那是你能做的最好的。
