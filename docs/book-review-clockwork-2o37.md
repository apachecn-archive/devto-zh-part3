# 书评——时钟

> 原文：<https://dev.to/documentednerd/book-review-clockwork-2o37>

大家好，正如你们很多人所知，我读了很多书，我也喜欢用“听得见的”。旅行或开车时消磨时间的好方法是听有声读物。现在，我一直在努力学习更多关于商业视角和生产力的知识。我以前读过和谈论过的一些好书是:

*   勇气
*   实在说
*   10x 规则
*   深耕

但是我想花一点时间谈谈我刚刚读完的这本书,[clock time:design your business to run it self](https://www.amazon.com/Clockwork-Design-Your-Business-Itself/dp/0525534016/ref=sr_1_1?ie=UTF8&qid=1549572935&sr=8-1&keywords=clockwork+business)，作者是 Mike Michalowicz。现在我不得不承认，当我几次听说这本书时，我就知道了这本书，它几次出现在我的“推荐阅读”书上。

所以我对这本书持怀疑态度，主要是因为这本书讲述了人们如何开始自己的事业，而我在一家大公司工作。那么这对我有什么帮助呢？我不得不承认，我错了。

我发现这本书真的发人深省，它让我重新审视我所做的许多活动和工作，以衡量对成功的影响和重要性。作者认为，在任何组织中，每个人都有责任做到以下几点:

1.  保护蜂王角色
2.  扮演蜂王的角色

基本上，这项业务的关键部分是扮演 QBR(蜂王角色)，这是你工作的关键部分，让你的所有行动都专注于这一点。基本上，我的论点是，我应该把工作日的每一秒钟都花在 QBR 上，当一项活动占用了我的时间时，我应该专注于尽快完成这项工作，或者如果可能的话，把它从我的盘子里移走。

这样做的目的是让我把注意力集中在更大的画面上，并创造一个你可以从工作中解脱出来并感到舒适的场景。对我来说，我倾向于很难摆脱工作。最近我一直在设定目标来帮助自己摆脱烦恼。我发现，当我开始将这一点付诸实践时，我能够以更少的压力拔掉插头，这有助于我的整体心理健康。对我来说，我开始打算做以下事情:

*   每天留出 1 小时吃午饭
*   我不会在桌子上吃东西

这迫使我离开办公桌去午休，老实说，这听起来很小，但却有巨大的回报，我发现当我回到工作岗位时，我会更加专注，在一天结束时，从精神角度来看，不会那么疲惫。我发现工作的压力水平下降了，我也发现我正在做的工作更令人满意。
