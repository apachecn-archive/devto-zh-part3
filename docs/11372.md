### [](#coding-the-program-and-sending-our-first-tweet)**编写程序并发送我们的第一条推文**

#### 在空闲时创建一个 auth.py 文件，将你的密钥粘贴到里面，然后点击保存

```
consumer_key = 'ENTER_YOUR_CONSUMER_KEY'
consumer_secret = 'ENTER_YOUR_CONSUMER_SECRET'
access_token = 'ENTER_YOUR_ACCESS_TOKEN'
access_token_secret = 'ENTER_YOUR_ACCESS_TOKEN'
```

#### [](#create-another-file-and-import-twython-from-twython-and-the-requests-module)创建另一个文件，从 twython 和 requests 模块导入 Twython

```
from twython import Twython
import requests
```

#### [](#also-import-the-variables-from-authpy)也从 auth.py 导入变量

```
from auth import (
    consumer_key,
    consumer_secret,
    access_token,
    access_token_secret
)
```

将此另存为 twitter.py

#### [](#create-a-connection-with-the-twitter-api-using-twython)使用 Twython 创建与 Twitter API 的连接

```
twitter = Twython(
    consumer_key,
    consumer_secret,
    access_token,
    access_token_secret
)
```

#### [](#well-use-the-updatestatus-function-of-twitters-api-to-send-a-tweet)我们将使用 Twitter API 的

```
update_status()
```

函数来发送一条推文

```
message = "Hello world!"
twitter.update_status(status=message)
print('Tweeted: ' % message)
```

#### [](#now-we-run-the-module-by-using-f5-or-from-tweetpy-gt-run-gt-run-module)现在我们通过使用 F5 或者从 tweet.py - >运行- >运行模块来运行模块。

### [](#calling-the-icanhazdadjoke-api)**调用 icanhazdaddjoke API**

幸运的是， **GET** 方法可以在没有认证的情况下调用*icanhazdaddjoke*。如果你想了解更多，请阅读他们的文档[这里](https://icanhazdadjoke.com/api)

```
url = 'https://icanhazdadjoke.com/'
headers = {'Accept': 'application/json'}
joke_msg = requests.get(url, headers=headers).json().get('joke')
print(joke_msg)
```

#### [](#designing-the-code-structure)设计代码结构

我们希望在我们的 twitter 账户上发布*icanhazdaddjoke*的输出，所以我们使用 **url** 变量来调用他们的 API。我从输出开始，回到调用 API。

**我们以这段代码结束:**

```
url = 'https://icanhazdadjoke.com/'
headers = {'Accept': 'application/json'}
joke_msg = requests.get(url, headers=headers).json().get('joke')
twitter.update_status(status=joke_msg)

```
