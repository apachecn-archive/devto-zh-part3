# 生成代码以调用来自邮递员的请求

> 原文:[https://dev . to/praneetnadkar/generating-code-to-call-a-request-from-postman-4h 75](https://dev.to/praneetnadkar/generating-code-to-call-a-request-from-postman-4h75)

Postman 是我用来调用、测试和调试 API 的最好的工具之一。今天早上，当我在处理一些事情的时候，我发现了《邮差》中两个我没有意识到的出色特性。Postman 可以为您生成一些请求调用代码，也可以生成文档。对于那些不知道的人来说，是的，这是真的！

我已经用 C#创建了一个示例 API，看起来是这样的:

```
// POST api/values
[HttpPost]
public ActionResult<string> Post([FromBody] string value)
{
   return ""+value+" is posted";
} 
```
