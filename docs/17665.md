# 在 Windows 上开发:Docker，Linux VM，WSL 还是 Cygwin？

> 原文：<https://dev.to/vinibrsl/developing-on-windows-docker-linux-vm-wsl-or-cygwin-4847>

(我目前是 Mac 用户，正在改用戴尔笔记本电脑)

MAC 很贵，Linux 有一些硬件兼容性问题。我只剩下窗户了。

我认识的大多数开发人员都讨厌在 Windows 上开发，除非他们正在开发。NET 应用程序，这不是我的情况。此外，我对我的 Unix shell 非常满意，我不想使用 PowerShell 或糟糕的 Windows CMD。

别误会，Windows 10 很棒。微软正在找回自己的最佳状态。并且要在 Windows 上作为主操作系统进行开发(没有双引导)，有一些选择:

1)Docker for Windows
2)Linux VM(VMWare、Virtual Box、Hyper-V)
3) Windows 子系统 for Linux(相当不错)
4) Cygwin

这些选项恢复了我的 Unix shell，并使开发变得更加容易。

有人使用过这些选项吗？
