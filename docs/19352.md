# 兼职自由职业

> 原文:[https://dev.to/raddevon/freelancing-on-the-side-40e3](https://dev.to/raddevon/freelancing-on-the-side-40e3)

大多数人在过渡到 web 开发的过程中，都无法离开自己的工作岗位。做着最差工作、最需要改变的人，却最没有实现改变的灵活性。

别担心。自由职业者来救援！当你学习 web 开发和过渡到一个新的职业时，你不必失去你所有的收入，你可以采取一个渐进的方法。下班后和周末加快速度。然后，**开始接一些自由职业者的活**。

当你开始兼职自由职业时，有一些事情需要注意。

## [](#dont-skip-the-admin-%F0%9F%97%83)不要跳过管理员🗃

你会受到诱惑，因为你只是在兼职做自由职业者的开发，以跳过所有无聊的管理任务。在与你合作之前，你不想注册你的公司，不想与会计师交谈，不想让你的客户签署合同。

这是一个错误。如果你现在不开始做这些重要的事情，你永远都不会去做。这会在某个时候给你带来问题。当你意识到客户不打算付款时，你将在一个项目中投入 5000 美元…并且你没有让他们签合同。

从一开始就做正确的事情，以避免这种令人讨厌的情况。**练习你的玩法**。

## [](#cya)CYA🍑

确保你没有把你的私人生活和你的兼职工作混为一谈。使用你的个人银行账户，经营一家没有任何保护你个人资产的独资企业，这将是很有诱惑力的。

就像前面的建议一样，你可能可以像这样跑很长时间而没有任何影响。然而，一旦你遇到问题，后果将会很严重。客户可以起诉你，拿走你的个人资产。如果你有家庭，这意味着你兼职中的问题会影响你的家庭生活，影响你爱的人。**节省一点时间和几美元是不值得的。**

## [](#understand-its-not-all-code)明白这不全是代码🤔

当我刚开始的时候，我想我会建立一个网站，告诉客户我提供了什么服务，然后等待客户的到来。实际上*和*发生的事情一点也不像。相反，为了找到工作，我不得不出去，总是站在人们的前面。

今天，我大约一半的时间是不收费的。它被用来做市场营销、寻找客户、记账和其他无数的事情。理解这一点很重要，尤其是如果你只是兼职自由职业者的话。花点时间，你会认为 T1 将会完全致力于写代码，而 T2 会把它减半。*那是*你能开多少时间的账单。

如果你准备好开始，我刚刚推出了一门电子邮件课程，将在一周内带你从零开始从事自由职业。[参加我的自由职业速成班🚀](https://raddevon.com/enroll-in-the-freelancing-crash-course/)！

## [](#save-for-taxes)为税收而储蓄💸

你的雇主为你的全职工作收入支付他们的那部分税款，而你的那部分通常会从你的支票中扣缴。你甚至不用考虑这个问题，直到年底，你会以退税的形式拿回你多付的钱。

当你是个体户时——即使只占你时间的*部分——你必须支付雇主部分和雇员部分的税款。**确保你在赚钱的时候为此存钱**，而不是在纳税的时候想方设法把钱都存起来。*

要想知道该存多少钱，你应该和会计谈谈。你可能需要支付季度估计税款，你的会计师也可以在这方面给你建议。这可能要花你几百美元，但是会计的建议是值得的。

去年，我是一名全职员工，同时还兼职做自由职业者。我把自由职业收入的 20%存起来交税。事实证明这还不够。我不得不拿出几千美元来支付差额。今年，我要存 35%的钱。

## [](#flip-the-switch)扳动开关💡

做自由职业者需要很大的投入。当你在边上尝试时，你会看到潜力，但你也会看到如果没有更多的时间，你还有多少潜力无法挖掘。一旦你明白了如何找到工作，如何保持工作，开始思考什么时候从你的工作过渡到全职自由职业是有意义的。

当你迈出这一步时，你会有其他的考虑，比如更换雇主提供的保险，维持办公室提供的一些社会关系。当你转换时，确保这些是微积分的一部分，但是不要拖太久。你永远不会对自由职业有 100%的了解，但是你会学到足够的东西来开始你的新生活。
