# FOUC 代表什么？

> 原文：<https://dev.to/jess/what-does-fouc-stand-for-2llg>

我早些时候在我的手机上看到了下面的屏幕，并与团队分享了它。它在刷新时消失了，我无法再现它，但想确认没有其他人看到它。

够直截了当了，但这是什么缩写！它是怎么发音的？！
