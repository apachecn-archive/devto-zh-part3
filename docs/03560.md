Go 是一门相对较新的语言；于 2009 年在谷歌诞生。它是[静态类型](https://codetips.co.uk/intermediate/translation-and-types/)并且是[编译语言](https://codetips.co.uk/intermediate/translation-and-types/)。

它是一种后端语言，这意味着它用于编写位于后台的应用程序，执行其他服务(如网站)要求它执行的任务。

已经有很多后端语言，但是由于 Go 的简单性和速度，很多开发人员选择学习和使用它。

我们选择教你围棋的另一个原因是学习曲线。由于其简单性，它比其他后端语言更容易学习。

下载/安装 Go 后，您可以通过运行以下命令来验证您的安装:

```
go version 
```

如果命令输出一个版本，你需要安装一个代码编辑器(我们推荐 [Visual Studio Code](https://code.visualstudio.com/docs/introvideos/basics) )，然后你就可以继续下一篇文章了。
