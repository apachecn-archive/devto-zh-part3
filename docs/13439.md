# 在工作面试中

> 原文:[https://dev.to/voins/on-job-interviews-3o65](https://dev.to/voins/on-job-interviews-3o65)

想象一下，如果他们想雇佣一个已经出版的小说家，他们会问什么:

*   你知道逗号是什么吗？
*   请告诉我一些关于单词的事情。
*   你如何用字母写一个单词？你了解信件，是吗？
*   你能写一个问题吗？任何问题都行。

在他礼貌地指出他显然知道所有这些东西甚至更多之后，他们转向“高级”问题:

*   你听说过“牛津逗号”吗？(用非常怀疑的声音)

...

这是我最近在一家大型国际投资银行的经历。几乎是字面意思。
