# 我每天使用的 5 个网络风暴快捷方式

> 原文:[https://dev . to/Thomas _ ph35/5-webstorm-shortcut s-I-use-daily-3799](https://dev.to/thomas_ph35/5-webstorm-shortcuts-i-use-everyday-3799)

我正在使用 WebStorm IDE，因为我有一个学生许可证，我真正喜欢的事情之一是快捷方式！

我们每天都在写很多东西，而且大部分时间我们都在一遍又一遍地做同样的事情。

这就是为什么我寻找一个更快的方法来做它们。🚅

# [](#1-multiline-editing)1-多行编辑

`alt` + `left-click`这让你可以编辑你代码中的任何地方。
T3![multi-line](../Images/9bcfa486d17a4744125270ec86baa6a2.png)T5】

## [](#12-multiline-with-text-search)1.2 多行带文本搜索

您可以使用`alt` + `j`选择下一个文本，然后一次编辑多行。
[![multi-line 2](../Images/27c157d895a5f0549e54fc692ac318e2.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--CWe9CSsV--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_66%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/3lqacvqe628zgzmz2aic.gif)

> ⚠不认为这是一个重命名工具。

# [](#2-duplicate-a-line)2-复制一行

这一个看起来没用，但是它确实节省时间。
[![ctrl-D](../Images/f864452302f43383accd2944b1b5b189.png)T3】](https://res.cloudinary.com/practicaldev/image/fetch/s--8-yVVsC8--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_66%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/vuepp5ugvou12kfhafxj.gif)

> 我有时用它来“事后多行编辑”。

# [](#3-moving-things-around)3-四处移动的东西

你也可以用`ctrl` + `shift` +(向上或向下)来上下移动你的行，这样你就不会被迫复制/剪切。
T3![moving lines](../Images/a38a26f0f75b8a7e1887e17f89c18303.png)T5】

> 这对于 JSON 来说也非常有用...

# [](#4-clipboard-history)4-剪贴板历史

你知不知道 Webstorm 把你复制/剪切的内容保存在历史中？
现在你知道了！你可以用`ctrl` + `shift` + `v`选择过去什么。
[![clipboard history](../Images/0d3cc6d3b88d3db756c13c489a818dcd.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--4VLQvIUC--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/xa7h1hf4wx8758v7ptdj.png)

# [](#5-growing-selection)5-生长选择

当选择某样东西时，你可以用`ctrl` + `w`将当前选择扩大到下一级，以降低选择命中率`ctrl` + `shift` + `w`。
[![selection](../Images/83873a432c46423feb5fe4754d17405d.png)T8】](https://res.cloudinary.com/practicaldev/image/fetch/s--MiXkb483--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_66%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/vn2e78u6t4shzslwxn0d.gif)

# [](#conclusion)结论

这也是我喜欢 WebStorm 的原因之一。如果你有其他有用的捷径，请告诉我。如果你学会了其中的一些！🎉
