# 2019 年 DjangoCon Europe 上我最喜欢的 5 场演讲

> 原文:[https://dev . to/django tricks/my-5-favorite-talks-at-django con-Europe-2019-531 b](https://dev.to/djangotricks/my-5-favorite-talks-at-djangocon-europe-2019-531b)

[![Django people at the DjangoCon Europe 2019](../Images/8c053a4238cd1873e58359f3bec50c19.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--BvXufktW--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://1.bp.blogspot.com/-BzP7Pb1KYnc/XMy8IpYcmyI/AAAAAAAAB-c/7kaW3l-YHmI1iDar2sLjRgM-G82qIseIQCLcBGAs/s1600/my-5-favorite-talks-at-djangocon-europe-2019.png)

今年的 DjangoCon Europe 在丹麦的哥本哈根举行，地点是一个非常有创意和特别的地方[AFUK——不羁创意学院](http://www.afuk.dk/)。在艺术灵魂的包围下，我们了解了更多的网络技术，在生态可重复使用的一次性咖啡杯中认识了彼此，享受了美味正宗的食物，并与 Django 品牌的饮料进行了交流。和往常一样，还有一个聚会，人们喝 IPA(不要和 API 混淆)。周末，Django 开发人员在编码冲刺阶段为 Django 和相关开源软件解决 bug。

在这里，我想向你展示我最喜欢的五大演讲。

## [](#django-and-web-security-headers)Django 和 Web 安全头

亚当·约翰逊( [@AdamChainz](https://twitter.com/AdamChainz) )谈到了特殊的响应头，告诉浏览器更安全地处理网站的数据，以及哪些 Django 设置负责这些头。

[![Video record](../Images/7cea5a72913503b6d50e101dc7a55863.png)T2】](https://youtu.be/gvQW1vVNohg)

[rixx 总结](https://rixx.de/blog/djangocon-europe-2019-django-and-web-security-headers/)

## [](#docs-or-it-didnt-happen)Docs 还是没发生！

Mikey Ariel([@ ThatDocsLady](https://twitter.com/ThatDocsLady))正在谈论文档的必要性以及在那里写些什么。

[![Video record](../Images/679c837e8cf07364e1ea3408e8375551.png)T2】](https://youtu.be/gwVxNHO9Lh0)

[rixx 总结](https://rixx.de/blog/djangocon-europe-2019-docs-or-it-didn-t-happen-with-q-a/)

## [](#pushing-the-orm-to-its-limits)将 ORM 推到极限

齐格鲁德·约德尔( [@sigurdlj](https://twitter.com/sigurdlj) )在谈论高级 Django ORM 用例。

[![Video record](../Images/4c6f2c84bb202dae121a10f16a3e1ded.png)T2】](https://youtu.be/MPpPu6c8wsM)

[rixx 总结](https://rixx.de/blog/djangocon-europe-2019-pushing-the-orm-to-its-limits/)

## [](#honorable-mentions)优秀奖

*   Carlton Gibson 的《喂养小马:回馈 Django &如何让它为你工作》。
*   汤姆·克里斯蒂的《重新设计姜戈的草图》。
*   [“减少、重用、回收——与共享工作人员保持 WebSocket 连接”](https://2019.djangocon.eu/talks/reduce-reuse-recycle-persisting-websocket-connecti/)Aaronn Bassett。
*   [“带有 GeoDjango、PostGIS 和传单的地图”](https://2019.djangocon.eu/talks/maps-with-geodjango-postgis-and-leaflet/)Paolo Melchiorre。
*   《机器人来了——Django 和机器学习》作者汤姆·戴森。
