# 第一届 Slack ML 应用程序黑客马拉松-太多伟大的想法！

> 原文:[https://dev . to/ishiid/1st-slack-app-hackathon-too-many-great-ideas-435 o](https://dev.to/ishiid/1st-slack-app-hackathon-too-many-great-ideas-435o)

我们在周四举办了第一届 Slack App hackathon，得到了很多很棒的想法！这一次我们没有使用像 Zapier 和 IFTTT 这样的编码应用构建器。我们将继续开发应用程序，并与人工智能模型集成。

你最喜欢的来自下面的想法是什么？还是你有 Slack app 的想法？
请评论！我的邮箱:[dai@jenio.co](mailto:dai@jenio.co)

Slack App 创意；

*   具有 LSTM 深度学习的 Peom 生成机器人
*   github 动作检测器
*   具有强化学习的笑话生成机器人
*   对收到的 gmail 数据进行情感分析(愤怒的客户检测)
*   Trello 票证输出分析
*   金融外汇买入/卖出机器人
*   从 outlook 到 Slack DM(if TTT 可能，但 Zapier 不可能)
*   自动回复客户
*   从 Gmail 到电子表格(以便我们可以作为结构化数据进行分析)
*   策划 RSS(任何内容推荐 API 都可以？)
*   来自 Google 日历新活动的推送通知
*   在 Google 日历的临近事件前推送提醒
*   在线昵称和真实身份检查器
*   尝试使用用户本地 bot API
*   想用谷歌对话流试试吗
*   想用 Wit.ai 和 API.ai 试试
*   翻译机器人
*   来自 Trello 的关于新任务和完成任务的通知
*   餐馆搜索机器人
*   使用自然语言处理的 Trello 新任务筛选
