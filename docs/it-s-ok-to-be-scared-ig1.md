# 害怕是正常的

> 原文：<https://dev.to/ashleemboyer/it-s-ok-to-be-scared-ig1>

*最初于 2019 年 2 月下旬为[我的个人博客网站](https://ashleemboyer.com/it-s-ok-to-be-scared)撰写。*

未知是绝对恐怖的，可能是因为我们无法控制我们不知道的。但是你有没有花时间去思考过，当你拥有了所有你需要的关于某件事情的信息，却仍然知道你不能控制它的时候，那是一种什么感觉？这是可怕的*和*令人愤怒。

下周对我来说是全新挑战的开始。我将在离工作 2 小时的地方上一堂课，我必须改变我的工作时间表来适应通勤。每周 4 天，我会从早上 6 点到下午 2 点工作，然后去听一个不到一个小时的讲座。

我对未来 3 个月将会怎样的恐惧大多表现为对过去的愤怒。我想了很多事情，这些事情对我来说可能和应该有所不同。几周前，当我去校园为新学期做准备时，我情不自禁地走在走廊上，为年轻时的自己感到悲伤。她应该得到比当时更好的待遇。

然后我想我现在在哪里。我和我最好的朋友住在一起，我们一起建立的生活是我做梦也想不到的。我们有一个可爱的家和一只漂亮的小狗，我们一起分享它。在过去的一年里，我们的关系有了很大的发展，我从来没有和其他人有过如此多的联系。

如果我在大学的时候有一件事发生了变化会怎么样？我还会在现在的位置吗？我可能就不是原来的我了。我现在所知道的是，我对我的生活和我与自己的关系完全满意。

过了一段时间，一切才感觉正常。但是它到了那里，我也成功了。我已经到了，正在为下一次旅行休息。将近一年的休息给了我空间去认识到那些时候害怕是绝对没问题的。

恐惧让我们不断前进，不管方向是否正确。重要的是，我们定期观察我们的周围，以确保我们在正确的轨道上。当你看到可怕的东西直视你的眼睛时，要坚信你会没事的。

一切都会如其所愿。
