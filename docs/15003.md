# Reddit 上的农场因果报应与机器人 2019 年更新

> 原文：<https://dev.to/mrpowerscripts/farm-karma-on-reddit-with-bots-2019-update-2l35>

回到 2017 年 7 月，我注意到 Reddit 变得更加政治化。许多有着各种倾向的非常具体的政治子帖子和很少的订阅者在 Reddit 上被推得很高。这似乎有点奇怪，但这与之前的选举周期并没有太大的不同。不过不同的是，在奥巴马赢得第二个任期的四年里，Reddit 的用户数量激增。子记录的数量[增加了一倍](https://www.statista.com/chart/11882/number-of-subreddits-on-reddit/)。有更多的社区、更多的人和更多的讨论。因此，许多新的政治分支也会出现也就不足为奇了。

这太棒了。越来越多的人参与并意识到政治是如何运作的，我们有互联网来分享这些信息，这很好。你能想象试图理解一百年前选举的所有细微差别吗？你投了票，然后事情发生了，有一天你醒来，报纸告诉你谁在管理这个国家。这到底是怎么回事？人们如何能够做出明智的决定？

他们没有。人们经常被欺骗。一百多年前，我们有一个事件叫做美西战争。你不会听到太多这样的事情，因为这是一个完全混乱的事件，记者们编造了大量的狗屎来鼓励美国政府和人民攻击一个国家。他们为什么要这么做？金钱和权力。让我们从[这个维基百科条目](https://en.wikipedia.org/wiki/Propaganda_of_the_Spanish%E2%80%93American_War)中截取一段。

> 美西战争(1898 年 4 月至 8 月)被认为是宣传史上的一个转折点，也是黄色新闻实践的开始。
> 
> 这是第一次媒体介入导致军事行动的冲突。这场战争源于美国对西班牙军队和其古巴殖民地公民之间的革命斗争的兴趣。美国报纸通过制造暴行来煽动人们对战争的兴趣，这些暴行为在世界范围内干预西班牙殖民地提供了正当理由。
> 
> 美国国内的几股势力正在推动与西班牙的战争。他们的策略是广泛的，他们的目标是以任何可能的方式吸引美国人民的意见。像《纽约日报》的所有者威廉·蓝道夫·赫斯特这样的人卷入了与《纽约世界报》的约瑟夫·普利策的发行量之战，并将这场冲突视为出售报纸的一种方式。许多报纸刊登耸人听闻的文章，并派遣记者到古巴报道这场战争。记者不得不躲避西班牙当局；通常，他们无法获得可靠的消息，严重依赖线人提供的消息。许多故事来源于第二手或第三手资料，或者是记者精心制作、歪曲或完全捏造的，以增强其戏剧性效果。西奥多·罗斯福(Theodore Roosevelt)当时是海军部长助理，他希望利用这场冲突来帮助愈合美国内战留下的伤口，并增强美国海军的实力，同时确立美国在世界舞台上的地位。罗斯福向美国国会施压，要求其援助古巴人民。他强调古巴的弱点和女性特质，为美国的军事干预辩护。

你知道普利策奖吧？以约瑟夫·普利策命名的奖项，授予杰出的新闻工作者。它是以一个卷入流通战争的家伙命名的，他发起了黄色新闻。你不能瞎编。接下来，你要告诉我[联邦通信委员会正在与它打算监管的公司](https://www.theregister.co.uk/2019/01/25/fcc_accused_of_colluding/)勾结。或者[食品和药物管理局参与促进明知对美国公众有害的政策](https://www.lifeextension.com/Magazine/2018/1/As-We-See-It/Page-01)。当我们需要的时候，我们的真理部在哪里？

我链接的那些文章你看了吗？我没有。这些标题支持了我的论点，对我来说已经足够了。不知道他们说什么。

我们应该如何做出有意义的决定？历史清楚地表明，当有两件事悬而未决时，每个人都会对每个人撒谎:金钱和权力。规则是给失败者的，如果你遵守规则，你就会成为失败者。

听起来很刺耳，不是吗？认为自己不能相信任何人或任何事是痛苦的。拥有一个系统和相信这个系统是两码事。你可以拥有一个无人相信的神奇系统，也可以拥有一个人人都相信的完全腐败的系统。许多美国人相信他们的制度。他们相信报纸是来通知他们的。我是说，他们为什么要撒谎？他们是一家美国报纸。他们的大楼外面有一面旗子。那应该够好了。报纸是用来提供信息的，绝不会有任何不可告人的目的。至少足够多的人是这么认为的。当然不是所有人，但你不需要对每个人都耍诡计。你所需要的就是赢得比赛。

我们真的能指望人们到处质疑在报纸、电视或互联网上提供给他们的每一个主张和断言的真实性吗？当然不是，我们都接受这一点。如此循环往复。

自从导致美西战争的事件发生以来，事情并没有改变多少。事实上，现在普通人更难跟上不断变化的全球政治格局。许多人都在努力为自己找一个栖身之所，更不用说去核实一天中不断向他们袭来的所有信息了。更好的是，为什么要注意呢？电视、电影和许多其他爱好可以填补空虚。

本质上，我们被耍了。每天都有新一代有用的白痴诞生。“我不是一个有用的白痴”是很多有用的白痴都会说的话。所有那些迫不及待要入侵古巴的美国人都确信他们不是有用的白痴。他们消息灵通，因为他们看报纸。消息灵通的人就是这么做的。

> “如果你不看报纸，你就不了解情况。如果你看报纸，你就被误导了。”―马克·吐温

这句话通常被认为是马克·吐温说的，但当然，它无法真正得到证实。在文章的这一点上，错误的引用应该是我们最不关心的。

黄色新闻并没有消亡，只是现在对于美国政治话语的现行标准来说，它的音节太多了。假新闻是个新名词。这是共和党人在唐纳德·特朗普 2018 年竞选总统期间推广的。虽然现在你会听到每个人都用这个词来形容他们认为是假新闻的事情。我不得不承认，这比黄色新闻简洁多了。我能理解为什么它是人们喜欢到处扔的东西。

就我个人而言，我觉得在这个时代，人们更加怀疑。但反常的是，这并没有让人们变得更加见多识广。它已经退化到人们寻找信息来源，确认他们感到安慰的偏见。不管偏见是什么。无论你相信什么，总有人愿意重申这一信念——如果他们能从中获利或受益的话。除非你自己能够接触到所有的事实和数据，否则根本没有办法判断这是为了你的利益还是为了他们的利益。这种情况多久发生一次？你多久会成为你需要说的信息的第一方，选举那些制定管理你的法律的人？他们到底是谁，他们到底参与了什么？几乎没有。然而，我们仍然相信这是一个可行的系统，因为这是我们所拥有的，也是我们所知道的。

现阶段的互联网已经让我觉得消息不那么灵通了。我曾经尽可能关注每一个新闻网站，阅读每一篇可用的文章。即使是国际网站。我想了解情况，保持清醒，这样我才能做出明智的决定。现在，我很难找到兴趣。没办法知道是不是另一个普利策在幕后操纵。这个系统已经崩溃了，或者至少看起来是这样。

我们生活在一个神奇的时代。信息时代。我们解决了一些令人震惊的问题才来到这里。但是现在要克服的最大障碍不是技术。克服导致我们为自己的偏见而不是真理而战的人类条件是未来最大的挑战。但是，在这么多人意见不一、相互争斗的时候，解决这些重大问题的希望就不大了。只是重复利用过去的破坏性事件。真理是战争的第一个牺牲品，谎言很乐意为我们给手枪上膛。

不管怎样，悲观够了。当我们周围的世界分崩离析时，我们仍然可以开心。

为了开始新的一年，我更新了 karma farming 机器人，使它比以前更加简单和智能。现在你可以像专业人士一样耕种业力了。它甚至包括使用代理服务器的能力，这有助于隐藏你的机器人运行的位置。它还使用了一个名为 docker 的工具，该工具在一个容器内运行机器人，这就像是在你的计算机上运行的一个迷你操作系统。集装箱里发生的一切都被控制住了。这样它就不会影响或搞乱你的主机操作系统。这也意味着你可以很容易地同时运行多个机器人。你可以在 GitHub 上找到这个机器人的最新代码，以及如何运行它的一些说明。出去耕种那份业力。没有人会知道这是一个机器人，他们也不会在乎它是否告诉了他们想听的东西。那就坐下来，看着上升的选票滚滚而来。
