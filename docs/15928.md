有时，为了方便起见，我们可以查看我们正在使用的程序消耗了多少系统资源，这样我们就可以找到一些应用程序，可以随时显示 CPU、RAM 和网络的使用情况，而无需打开系统中的任何活动监视器或任务管理器。

我试图找到一些应用程序，但找不到任何免费的东西(显然，如果你付费，你很容易得到东西)，所以我突然想到为什么不为它写代码。

所以我所需要的就是 NodeJs +一些包来获得系统统计数据和电子数据，让这些统计数据有个样子。

所以这是到目前为止下图中的**输出***【15 | 8gb | 0.1】*，这只是它的基本实现。

# 监视器应用程序

[![alt text](../Images/6046c8355fb45f9eb12a7accd70a82ca.png)T2】](https://github.com/amitgiri0001/monitor-app/blob/master/Screenshot.png)

有史以来最简单的监测应用程序，坚持在屏幕的角落，并随时可见的监测 CPU，内存和网络。

# 装置

`nvm use 11; npm i`

# 奔跑

`npm start`

# 建设

`npm run build`

构建将根据提供的选项创建包。
