# 选择合适的网站托管服务

> 原文：<https://dev.to/flippedcoding/choosing-the-right-service-for-hosting-your-site-479j>

有 Azure、GoDaddy、AWS、InMotion 和一堆其他方法可以让你的网站在线可用。你可能正在寻找托管网站的最佳方式，这是一个重要的决定，无论是从技术上还是经济上。你的虚拟主机服务将决定你的网站能处理多少流量，以及你能访问多少核心功能。

## 虚拟主机的类型

虚拟主机只是一种花哨的说法，意思是你的网站在你付费的服务器上。您有四个选项可以选择托管服务器:共享、专用、云托管和虚拟专用服务器(VPS)。每个选项都有自己的优势，选择哪一个取决于您当前对网站的需求以及您对网站未来需求的预期。

### 共享托管

共享主机服务器是大多数人的起点。共享主机将是绝对最便宜的选择，它需要最少的技术知识。顾名思义，你和其他网站共享一个服务器。它可能是少数几个站点，也可能是成千上万个其他站点。

这里你需要知道的主要事情是，通常所有的网站共享相同的服务器资源，如 CPU 和 RAM。如果另一个网站正在使用更多的资源，这可能会导致您的网站运行缓慢。如果你刚刚开始，你不希望有太多的流量，共享主机包可能会很适合你。

### 专用主机

专用托管服务器与共享托管服务器相反。不同于在同一个服务器上有一堆网站，这个只有你的。这是你的单人服务器，你可以随心所欲。这将使您完全控制服务器所做的一切。随着流量越来越大，你可以轻松地增加资源。

这种类型的服务器需要更多的技术知识，您需要了解一些如何配置您的设置。因为你有绝对的控制权，而且没有其他人和你一起在服务器上，所以你要为共享主机支付更多的费用。通常这是为了更大的应用程序和公司，但如果你知道你会得到大量的流量，你可能会考虑这种托管。

### 云托管

接下来是云托管。你可能已经知道这一点，但 Azure 和 AWS 是云托管领域最大的品牌。这种类型的托管工作与其他不同，因为它使用一个服务器网络来托管多个网站。与共享主机相比，这种方式的优势在于服务器网络可以为每个站点提供适量的资源。

因为一个服务器网络可以有一堆服务器，这意味着你有能力扩展你的网站来处理更多的流量，而不受任何其他网站的影响。您将不能完全访问服务器根目录，但是您可以更改您可用的资源数量。大多数云托管会根据你使用多少资源来收费。如果您想尝试随增长付费的策略，这可能是一个不错的选择。

### 虚拟私有服务器

最后一种托管包是虚拟专用服务器。这是一个有趣的托管版本。实际情况是，多个网站共享一台服务器，但该服务器被分成多个虚拟服务器。因此，每个网站都表现得像是在自己的专用服务器上，但实际上它们共享一个服务器。

虚拟专用服务器比共享主机或云主机给你更多的控制权，但比专用主机便宜。它不像云主机那样可以扩展，但是你也不必像共享主机那样担心资源。如果你想对你的服务器有更多的控制，而又不想花钱买专用的，这是一个很好的选择。

## 如何选择托管包

选择正确的软件包实际上取决于你对网站的期望。你也可以随时切换主机包，这样你就不用太担心选错了。在选择托管包之前，您应该考虑以下几点:

*   你预计会有多少流量？
*   您真正需要对服务器设置/软件进行多大程度的控制？
*   这要花多少钱？
*   有订阅期吗，多长时间？
*   你以后能升级到一个不同的主机包吗？
*   服务器的可靠性和响应性怎么样？
*   有好的技术支持吗？
*   可以添加新域或子域吗？
