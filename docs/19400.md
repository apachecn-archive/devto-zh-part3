# iPad Pro 12.9 (2018)三周后的观感

> 原文:[https://dev . to/joshpuetz/ipad-pro-129-2018-impressions-after-three-weeks-2dn 6](https://dev.to/joshpuetz/ipad-pro-129-2018-impressions-after-three-weeks-2dn6)

说我是 iPad 的粉丝是一种保守的说法。自从 2010 年第一代 iPad 问世以来，我几乎拥有了每一台 iPad。2015 年 iPad Pro 的推出(以及 Apple Pencil 和键盘盖的推出)巩固了 iPad 作为我主要非工作电脑的地位。它也越来越成为我的工作电脑，我会在以后写更多。

因此，我预订了最近推出的第三代 iPad Pro 也就不足为奇了。过去几周，我一直在各种环境下(在家、度假、飞机上和技术会议上)使用支持 LTE 的 12.9 版 iPad Pro:以下是我迄今为止对硬件和软件的粗略印象:

## [](#hardware-ipad)硬件:iPad

*   新的设计是华丽的，手感也很好。我很想看到这种复古的未来设计在明年跨越生产线进入 iPhones。
*   这不是真正的边对边:仍然有一个明确的边框(Face ID 传感器和前置摄像头隐藏在其中)。
*   疏油涂层没有改变:屏幕仍然是一个可怕的指纹磁铁。
*   总的来说，USB-C 是一个很好的改变:该规范允许更快的充电时间和重复使用我的 MacBook Pro 的加密狗。当然，如果你还没有一大堆 USB-C 加密狗，你必须建立一个工具包。
*   由于 iPad 厚度减小，摄像头凸起大得可笑。智能键盘 Folio 有助于隐藏凸起，但如果你在没有外壳的情况下使用 iPad，它在平坦的表面上有点不均匀。

## [](#hardware-smart-keyboard-folio)硬件:智能键盘对开

*   由于单个键盘/外壳没有像以前型号的键盘外壳那样的接缝，在膝盖上感觉更加稳定。
*   单个前盖片的缺点是失去了旧键盘盖的“媒体配置”。以前的型号可以折叠在 iPad 后面，使它能够以几乎垂直的角度支撑。
*   有两个键盘角度是很棒的:更垂直的设置非常接近我提到的“媒体”配置，但整个键盘现在在这个视角下突出。没有悬挂在架子上的键盘，将 iPad 立在一个浅架子上是不可能的。
*   当它折叠起来时，感觉封面上的按键就像在演奏手风琴(或者至少是我想象的那种感觉)。这不是一种愉快的感觉，但足以作为世界上最昂贵的烦躁玩具在紧要关头。
*   在 folio 中使用更多的磁铁意味着它仍然超级粘在金属咖啡桌上。
*   视角通道上的磁铁有点太强了:折叠外壳需要很大的努力。

## [](#hardware-apple-pencil)硬件:苹果铅笔

*   与旧设计相比，新设计的铅笔更容易抓握，因为它的边缘是平的，表面是哑光的
*   铅笔侧面的双击动作需要一些时间来适应。由于我握笔的方式(非常不正确),在我握笔的时候敲击铅笔的侧面有点棘手。经过练习，我认为这仍然比在我的笔记应用程序中手动点击橡皮擦图标要快
*   磁力充电很棒，但是铅笔放在包里就不太好使了。结果我在旅行前把它拿掉了。
*   电池寿命似乎缩短了:我确实看到过随着使用时间的延长，电池寿命下降到 50%以下。然而，通过将铅笔放在 iPad 上的存储位置来为铅笔充电的便利性降低了这种做法对现实世界的影响。

## [](#software)软件

*   Face ID 感觉比在 iPhone 上更方便，因为我几乎总是把 iPad 以直角放在面前。
*   Face ID 总体感觉更快，它检测我面部的角度似乎比 iPhone X/Xs/Xr 上的 Face ID 更宽更远。
*   WWDC 2019 上的 iOS 13 公告将真正决定 iPad Pro 的未来:现在硬件已经设置好了，设备感觉需要软件功能来利用它。这款设备的苹果专业级应用(像 Logic 和 Final Cut Pro)在哪里？
