# 越来越好！

> 原文：<https://dev.to/burdettelamar/getting-better-56jk>

当我开始任何重要的开发任务时，我会努力记住我想要三样东西:

*   工作产品本身，通常是新代码。
*   一些可重用的代码。
*   一个改进的开发者(就是我)。

关于第三点:我试着设计任务，这样我就有学习和成长的空间。这通常很容易。

但是我有一个备用方案:选择一两个新的快捷键。

确定候选人很简单。我只是记下我用鼠标反复做的一些事情(我称之为*对*的鼠标操作)。然后我会看看是否有捷径可走。如果是这样，我试着养成使用快捷键而不是鼠标的新习惯。
