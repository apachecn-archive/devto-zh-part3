# 免费层最好的 Heroku 应用是什么？

> 原文：<https://dev.to/kokimame/what-is-the-best-working-heroku-app-with-the-free-tier-3k69>

标题几乎说明了一切，但让我给你一些背景。虽然我没有 web 编程的经验，但我有一个 web 应用程序的想法，我想在我的语言教学项目过程中实现它。我刚刚用一个免费账户在 Heroku 上部署了我的第一个 Django 项目。到目前为止，它运行良好。虽然有时需要几秒钟才能到达网站，但这对我的抓取式网站来说并不重要。最重要的是，与我从事的桌面应用程序开发相比，更新 web 应用程序和部署到所有平台是多么容易，这让我感到非常惊讶。

问题是我没有完全理解 Heroku 平台的许多方面，例如，我仍然让`DEBUG = True`当教程说我不应该的时候。另外，我想看看在免费层的限制下，web 应用程序能做得多好。所以，让我知道是否有一些有趣的网站在免费的 Heroku cloud 上工作，希望是在 GitHub 上。
