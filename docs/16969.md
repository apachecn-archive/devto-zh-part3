# 为了成为一名开发者，我放弃了一切

> 原文:[https://dev . to/Marek/I-throw-it-all-away-to-be-a-developer-16 C4](https://dev.to/marek/i-threw-it-all-away-to-become-a-developer-16c4)

当我决定放弃生物化学并成为一名开发人员时，我已经完成了 3 年的本科学位。

回想起来，这是一个相当疯狂的决定。我真的很幸运，因为我有家人的支持，允许我这样做，但这仍然是一个冒险的举动。

生物化学的学习其实真的很有趣。这都是关于细胞的内部结构和 DNA 是如何工作的。我仍然喜欢阅读关于生命科学的新闻，尤其是像基因编辑这样的新发展。

但是当我想象自己在实验室工作并在学术界建立自己的事业时，我缺少了一些东西。我不只是想做研究，我想做东西。

这就是我最终选择软件的原因。在软件方面，我可以拿起电脑制作东西。这其中蕴含着强大的力量。我不需要任何人的许可。

与研究和学术界相比。可以说，在你职业生涯的每一步，你都必须得到许可。你要求被一位教授接受去他们的实验室工作。你要求期刊发表一篇论文。你要求资助你的研究。你请求允许在会议上发言。如果你的工作真的很好，那么最终它会转过来，人们开始向你要东西，但这是一条漫长的道路。

这就是软件与众不同以及吸引我的原因。有了软件你不需要任何机构或实验室来学习，你可以自己做。

你可以创作并出版你的作品，没有人会阻止你。

如果你的作品很好，人们会分享它，甚至为此付钱。

这就是为什么我最终成为了一名软件开发人员，而这正是我一直在寻找的职业。
