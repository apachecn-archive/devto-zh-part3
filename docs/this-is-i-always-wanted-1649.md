2018 年初，我一直在学习 web 开发，无法真正专注于完成课程，但总是跟踪各种 web 技术的最新发展，不断阅读博客和故事，这样我就不会完全脱离 web 开发的世界。

**时间一致**

到 2018 年 10 月的第一周，我决定我需要做出重大改变，并站在正确的一边，所以我专注于完成我在中间留下的课程，并提高我的技能，随后是下班后漫长的令人沮丧的夜晚，并将周末的大部分时间用于学习。

**到目前为止**

1.  完成了 HTML，CSS，JavaScript 的初级课程。
2.  学习基本的用户界面设计。
3.  开始着手我的作品集。
4.  **发表了我的第一篇博文:D.**

虽然有这么多东西要学，这么多东西要做，但知道每天我都在向我的目标靠近，还是有一些满足感。

我将尽可能多地记录/博客我的旅程，分享我的经历。

完全被喜悦和**快乐的编码淹没了。**

干杯！！
