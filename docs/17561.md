#### [](#hi-how-do-i-even-begin)嗨！我该如何开始？

这似乎是我们总是问的第一件事，回答这个问题并不容易。每一次新的冒险总是出乎意料。你永远不知道从哪里开始。

也许，稍微介绍一下。
嘿！我叫本·普拉克斯，我在家上学。

我是**没有机器学习专家** *(还没有)*。
我**不是毕业生**。
我不擅长数学 ***(可能几乎每个人都会有同感，对吗？).*
我**勉强理解了机器学习模型罩下的数学**。
我**不总是专注**。我让小事分散我的注意力，顺便说一下，这是一个可怕的习惯。**

 **这也是为什么我决定写这篇文章来真正推动自己学习。

稍后会有更多的介绍。

我不知道在我的领域里有多少专家可以给我提供机器学习方面的指导或培训。

不过，我有以下几点:

1.  笔记本电脑
2.  互联网
3.  大脑
4.  谷歌
5.  愿意学习
6.  决心
7.  好奇心

我也知道一些社区对机器学习感兴趣。社区会帮助你，但大多数时候，**你只能自己搜索问题**。

好奇心、决心和学习的意愿可能是以上 7 项中最重要的。如果没有这些，你可能甚至不会费心去读这篇文章，对吗？所以，让你的好奇心流动起来，用知识喂养它。下定决心并愿意学习，充分意识到你会在前进的道路上犯错误。接受它，继续新的学习。

这篇文章(文章，日记，博客，无论什么)的目的是鼓励所有想开始机器学习之旅的人。

希望你能了解我将面临的一两件事情，试图理解这个叫做机器学习(ML)的神秘黑箱。🤔？

也许我们还可以一起庆祝，当我们最终揭开这些谜团和谜语，一起解开我们的知识之谜。🎉🎉

我希望你喜欢跟随我学习机器学习的旅程。

如果你能提出一些你认为可能对我和其他读者有帮助的建议，请不要犹豫发表评论。

感谢阅读。我们开始吧！😄**
