> 一点信息(数据)总是有一定的类型

[![What are data-types?](../Images/f97793eebffea93ef77529c97f5210fa.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--VGjdXOPa--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://www.codetips.co.uk/content/images/2019/03/data-type_4-on-stone.jpg)

每种编程语言都有不同的数据类型或不同的名称，但大多数都有以下四种类型的变体:

### [](#booleans)布尔

`Booleans`是简单的二进制值(`true`或`false`)。

```
isTrue = true
isFalse = false 
```

在我们上一篇关于 [if 语句](https://codetips.co.uk/beginner/what-is-an-if-statement/)的文章中，我们展示了程序如何决定是否执行一段代码。用于此的条件最终只能有两种结果:**做** ( `true`)或者**不做** ( `false`)。

### [](#integers)整数

`Integers`为整数(无小数位)。

```
number = 5 
```

### [](#floats)浮动

`Floats`是数字，即**可以**有小数，但不一定要有。

```
number = 3.5 
```

### [](#strings)字符串

`Strings`是简单的一系列字符(如单词、名称或整个句子)

```
id = ckjasdiwe98213
name = John
lastname = Doe
job = John is working at a farm, growing potatoes 
```
