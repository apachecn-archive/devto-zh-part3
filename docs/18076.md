# 商业社交网络 CMS 框架-有什么推荐吗？

> 原文:[https://dev . to/themazuma/social-network-CMS-framework-for-business-any-recommendation-58n](https://dev.to/themazuma/social-network-cms-framework-for-business---any-recommendation-58n)

嗨，我想为有利基市场的公司创建一个社交网络平台。在简介部分应该有一个路线图，团队和顾问。

在中间，张贴来自公司进展或相关新闻的更新。

和一个只有人可以评论和阅读，但不能发布的部分(只有公司会发布，只是邀请)。

我做了一些调查，因为我从来没有建立过任何社交网络(只有几个带有 Wordpress 和 themeforests 框架的网站)。

有什么建议吗？短暂性缺血性发作

Elgg 3.0:

[http://learn . elgg . org/en/master/appendix/upgrade-notes/2 . x-to-3.0 . html](http://learn.elgg.org/en/master/appendix/upgrade-notes/2.x-to-3.0.html)

[https://github.com/Elgg/Elgg/tree/3.0](https://github.com/Elgg/Elgg/tree/3.0)

老社区，没有更多的新闻题材，也很难从平台上改变核心(看资源)。

头脑:

类似于 Elgg，他们改变了 Elgg 的内核，但是花了 2 年时间和很多开发人员才完成！我们没有足够的资源，我们也不需要区块链或密码来启动平台(至少在开始时，我们可以通过 API 从 0x 或其他已经存在的来源集成一些令牌或硬币)。

Wordpress:

[https://buddypress.org/](https://buddypress.org/)

支付大约 200 欧元，这是 wordpress 背后的一个漂亮的框架。是有些东西但是不能改变很多东西。

框架示例:[http://crypterio.stylemixthemes.com/landing/](http://crypterio.stylemixthemes.com/landing/)

Joomla:

还有 Jomsocial，一个构建社交网络的框架，集成在 Joomla 的框架中。

[http://www.jomsocial.com/demo](http://www.jomsocial.com/demo)

展柜:[https://www.elogos.ro/](https://www.elogos.ro/)

我喜欢这个社区、价格和平台，但就像是 elgg 的升级版。

轻松社交的展示:[https://stackideas.com/easysocial/showcase](https://stackideas.com/easysocial/showcase)

Drupal:

还有就是开放社交:[https://www.getopensocial.com/](https://www.getopensocial.com/)

它是部分开源的，那么你必须每月为更多的用户付费。非常好的内容，但价格昂贵，我们现在不需要。

展示:[https://www.getopensocial.com/showcases](https://www.getopensocial.com/showcases)T4[https://www.connectingbusiness.org/index.php/about](https://www.connectingbusiness.org/index.php/about)

开放的社交网络:

[https://www.opensource-socialnetwork.org/](https://www.opensource-socialnetwork.org/)

非常类似于 facebook，但是免费，有一个大的社区和开发者。有 210 美元的额外费用

新海豚框架: [https://una.io](https://una.io) (这个是我的最爱。)
