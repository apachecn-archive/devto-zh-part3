# ERC 是做什么的？看看 ERC 20，ERC 721，还有更多！

> 原文：<https://dev.to/dapp_com/what-do-ercs-do-looking-at-erc-20-erc-721-and-more-45lc>

没有关于一些伦理委员会的解释的小儿科是不完整的。我们将解释几乎每次我们在以太坊网络上使用 dapp 时都会用到的主要 ERCs 以及它们的一些鲜为人知的改进。用最简单的术语来说，当你阅读“ERC”时，你可能正在阅读以太网上与令牌相关的功能。对于许多人来说，ERC 似乎是一项可怕的任务，所以我们试图分解一些我们看到人们在推广他们的 dapps 时参考的最重要的 ERC。

ERC-20:只是一枚硬币？

在以太坊网络上，可以自由兑换加密货币以太。以太是一种加密货币或硬币，可被视为可用于进行各种交易的代币。在以太坊网络中，还有其他令牌可以根据它们与以太之间的交换价值来使用。要在以太坊区块链上实现另一个令牌，你必须在名为 ERC-20 的智能合约中设置代码或技术标准。首字母缩写“ERC”代表“以太坊评论请求”，而“20”是请求的编号。简而言之，ERC-20 只是让代币表现得像市场上的加密货币一样。

一个使用 ERC-20 的令牌的简单例子可以在一个流行的名为“分散的”的 dapp 中看到。在分散土地，用户可以访问创世纪城，一个虚拟世界，用户可以使用平台内的货币法力购买土地，支付经验，或交易 NFT(稍后将详细介绍)。ERC-20 基本上允许 dapps 的开发者应用他们的新令牌，同时给他们一个货币价值。然而，ERC-20 令牌需要尊重令牌在以太网上如何表现的明确准则。由于大多数代币遵守 ERC 20 规则，dapp 用户可以将这些代币存放在大多数钱包中。

然而，该系统并不完美，因为试图通过令牌的智能合约发送令牌会导致令牌丢失。当令牌被发送到另一个用户的钱包时，来自接收钱包的请求将被发送，要求令牌的合同，以便它可以更新分类帐。如果您通过令牌的智能合约发送令牌，回复将被视为空，而您实际发送的令牌将消失在虚拟的空气中。

ERC-223 来救援了！

以太坊社区并没有袖手旁观，让 ERC-20 故障一直存在，让代币意外消失。尽管如此，随着更多的 dapp 项目实现了 ERC-20，许多令牌丢失了。据估计，损失在 70 万美元到 100 万美元之间。

ERC-223 修复了 ERC-20 在直接与令牌的智能合约一起发送时丢失令牌的问题，它还在此过程中创造了另一个好处。本质上，它允许通过智能合约一次性将令牌移动到另一个钱包。通过该功能，钱包不需要令牌的智能合约来更新分类帐。这个新的副产品(你会说性感吗？)函数的一个优点是，由于只有一个函数而不是两个函数需要验证，因此汽油费将减半。天然气费是执行智能合同或简单交易所需的小额付款——它们构成了非常小的费用，本应奖励以太坊网络中的矿工。

另一个竞争者:ERC-777

针对 ERC-20 漏洞的另一个解决方案于 2017 年 11 月 20 日提出，并作为“ERC-777”提交给了社区这不仅是 ERC-20 的解决方案，而且像 ERC-223 一样，它也有自己的好处。ERC-777 采用 ERC-820 的功能，允许每个人看到智能合约的地址并检查其功能，并使用它来验证交易或请求的操作。通过将 ERC-20 简单的“传送和确认”功能的性质改为 ERC-777 的“发送”，它允许令牌和智能合同本身做更多的事情。

有了这些新功能，ERC-777 可以检查它试图执行的动作的性质，并得出是否要执行该动作的结论。因此，在令牌交易被错误发送的情况下，将令牌发送回令牌的原始所有者，交易将被取消。为了使 ERC-777 更有趣，它被设计成允许 ERC-20 容易地升级到 ERC-777。然而，不幸的是，ERC-777 并没有被所有使用 ERC-20 的项目立即采用，一般以太坊社区的共识是，它最终将被实现，尽管它可能需要更多的时间。

ERC-721:另一个令牌？

ERC-721 是一个令牌，但它是一个不可替代的令牌，或者更好地为区块链社区所知，是一个 NFT。ERC-721 允许开发者创建不能在多个所有者之间共享的令牌，并且可以包含每个 NFT 创建的个人的统计和/或特征。不足为奇的是，ERC-721 函数已经被许多 dapp 游戏广泛采用，他们创建了代表人物或物品的 NFT。

这方面的完美例子是 Etheremon，其中每个 Etheremon 都是实现 ERC-721 功能的 NFT。玩家可以互相交换他们的以太精灵。没有玩家可以分享以太精灵的一部分；如果它需要从一个所有者转移到另一个所有者，它必须作为整个 NFT 转移。与此同时，在游戏中，玩家可以训练和战斗他们的以太精灵，使他们的以太精灵的统计数据增加或减少。以游戏中的繁殖功能为例，每当创建一个新的以太妖怪，以太坊网络中就会创建一个新的 NFT，所有权会自动归请求该动作的个人所有。

ERC-721 本质上允许代币被制成收藏品，可以在 dapp 内部共享，并影响未来 NTFs 的创建。这些 NFT 的供应可以是无限的；然而，大多数 dapp 项目限制了总供应量，以免降低 NFTs 的价值。让事情变得更有趣的是，ERC-998 可以与 ERC-721 一起工作，允许玩家一次性转移整个角色及其所有的统计数据、物品，甚至令牌。

其他伦理委员会

随着时间的推移，我们可以期待更多的 ERC 出现，以改善以太坊网络，并使智能合同更加智能。还有很多我们没有讨论的 ERC，如果您想更仔细地了解它们，您可以访问以太坊改进提案网站，那里列出了它们。大多数提议在各种项目中的实现都非常缓慢，因为人们总是担心实现新的 ERC 后会发现有不可预见的错误。在 ERC-20 的情况下，错误是已知的，这使得大多数开发人员很容易避免这个问题，并在没有任何其他不可预见的问题的项目中实现它。随着开发人员越来越熟悉新的 ERC，我们也将看到功能更好的 dapps，提供更好的用户体验。
