# 你的有机排名在谷歌下降的 9 个原因以及如何解决

> 原文:[https://dev . to/promozseo/9-reasons-your-organic-rankings-dropped-in-Google-how-to-fix-3a pn](https://dev.to/promozseo/9-reasons-your-organic-rankings-dropped-in-google-and-how-to-fix-that-3apn)

也许你知道今天有一个网站是不够的。除非你的目标关键词在谷歌和其他搜索引擎上排名靠前，否则你的网站毫无用处。在当今竞争激烈的市场中，每个企业都在尽最大努力让自己的商业关键词在谷歌上排名靠前。但是大多数都一再失败。出现这种情况大多是因为缺乏 SEO 知识和预算。如果你觉得太难了，那么是时候深入研究了。

SEO 不是一日之功。SEO 在开始得到结果之前需要时间。SEO 也需要持续的努力和投资。如果你对投资搜索引擎优化犹豫不决，那么这很可能不适合你。雇佣廉价的搜索引擎优化公司只会浪费你的金钱和时间，而不是雇佣一个有着良好记录的知名专家搜索引擎优化顾问。

在这里，我们将列出 9 个最有可能导致你的网站在谷歌和其他主要搜索引擎中搜索引擎优化排名下降的原因。

## [](#you-picked-up-the-wrong-keywords)你拿错了关键词

关键词对排名很重要，也很关键。用户使用与其需求相关的各种搜索查询进行搜索。谷歌在蜂鸟、RankBrain 等算法的帮助下理解这些带有相关关键词的搜索查询。此后，谷歌向我们展示最相关和高质量的网页结果。

如果您的网站不包含正确的关键字集，那么很难对与您的业务、产品和服务相关的竞争性关键字进行排名。

如果你已经雇佣了一个搜索引擎优化代理，请他们发送详细的报告，包括页面使用的关键词，以及每月平均搜索量和 KEI(关键词效率指数)。此外，你需要一个新鲜的关键字分析你的网站的每个重要页面。这将让你更清楚地知道是否已经使用了正确的关键字。

今后，每当你在你的网站上创建一个新的销售页面或文章/博客页面，在做任何优化之前，你需要一个与页面主题相关的深度关键词报告和数据。

## [](#your-website-metadata-are-not-properly-optimized)您的网站元数据没有得到适当的优化

当你知道你为一个网页选择了正确的关键词，你的下一步工作就是确认元数据优化的质量。

对于一个网页，有三个重要的元数据需要优化；页面标题、元描述和图像替代文本(不完全是元数据)。

检查页面标题和描述中是否使用了最佳关键词。这两篇文章分别用 70 和 150 个字符完成。这两者都必须是有意义的，人类可读的，而不是塞满了关键字。

图像替代或图像替代文本不应为空白，并且必须包含一个与页面和图像相关的关键字。这也必须有语境意义。

此外，检查页面是否有 H1，内容标题是否有最佳关键词(每个网页一个 H1)。类似于 H1，一个页面应该有 H2，H3 等夫妇。每一个段落标题都应该包含一个有意义地适合段落上下文的关键词。

## 你的网页内容很差

首先，你可以在任何事情上妥协，但不能在网站内容上妥协。网站的每一个重要页面都必须有足够的与其主题相关的新鲜内容。不这样做不仅会使搜索引擎排名下降，还会让你的访客失望。

一般长内容比较好排名。因此，在保持人性化的同时，创建尽可能多细节的内容。

将内容分成小段，尽量不要一遍又一遍地重复同样的内容。从长远来看，这不会有什么帮助，而且会造成糟糕的用户体验。

在内部和上下文中链接页面，以增加页面浏览量和观众停留时间。这是改善用户体验和排名的最佳实践之一。

如果你曾经外包内容，确保检查他们的原创性。互联网上有许多在线工具，它们会告诉你内容是否已经提前发布。重复和虚假的内容对网站弊大于利。

此外，你应该定期更新你的网站内容，以保持它们的新鲜和相关性。谷歌的咖啡因算法倾向于在搜索结果中对最近更新的内容进行排名。

## 你的网站没有(好的)反向链接

像内容一样，反向链接对于有机排名同样重要和强大。谷歌和各大搜索引擎都把反向链接作为衡量一个网站权威性的重要参数。

如果你的网站没有排名，你的网站的反向链接可能是原因。如果你的网站有少量的反向链接，那么很可能是时候集中精力获取更多高质量的链接了。此外，这些链接必须来自与您的业务和行业相关的权威网站或网页。

如果你已经外包了你的网站的搜索引擎优化，要求该机构发送一份每日反向链接报告，其中应包含所有链接的完整细节，以及日期、链接网站的域权限、垃圾邮件分数和链接页面 URL。一个一个地交叉检查，并定期监控细节。

此外，您还应该监控您是否丢失了任何高权限链接。这可能是由多种原因造成的；也许链接页面被删除了，或者链接网站删除了你的反向链接。当失去许多这样强大的链接可能是你的网站有机排名低的一个原因，解决办法是继续获得自然和一贯的高权威反向链接。

记住，反向链接的总数很重要，但是这些链接的相关性和权威性更重要。因此，多投资于高质量的链接建设，而不是从差的或不相关的网站建立低反向链接。此外，你的主要目标应该是创造高质量的内容，而不是建立低质量的反向链接。因为，如果你一直沉迷于创造高质量的内容，这些将自动吸引高权威的反向链接。

## 你的网站在技术上并不完善

这是 SEO 中的一个关键问题，大多数新网站都面临着这个问题。作为一个网站站长，你在技术优化上有很多东西需要去检查和确认。

首先，检查站点的域版本是否设置好了。它应该要么打开 www，要么不打开。非设定版本必须将自己永久重定向到设定版本。

其次，检查每个网页是否有正确的规范标签。规范 URL 告诉搜索引擎要索引和排名的 URL 版本。

接下来，你应该在需要的时候处理好网站的链接。你可以使用谷歌搜索控制台工具或任何其他断链分析工具找出该网站的 404 错误页面。解决方案是将 404 URL 永久重定向到现有的相关内部页面 URL。

HTTPS 永远比 HTTP 好。谷歌推荐 HTTPS，并将其作为一个强有力的排名信号。因此，如果你的网站还没有托管在 HTTPS，考虑尽快得到它。

确保你网站的 robots.txt 文件写了正确的指令，并且放在正确的位置。确认是否有任何东西被阻止使用机器人文件。阻塞任何资源，如。css，。js 或图像文件是不好的 SEO 实践。

当网站地图文件不是必须的时候，把它放在网站上有助于搜索引擎容易快速地理解内部 URL。因此，在逻辑上保持包含重要内部 URL 的小站点地图文件。

定期监控谷歌搜索控制台工具和谷歌分析工具，以了解网站的许多技术和可用性错误。

## [](#your-website-is-taking-too-much-time-to-load)您的网站加载时间过长

网站速度是当今最强的排名信号之一。如果你的网站在浏览器和移动设备上花费太多的时间来加载，那么这不仅会在排名上产生问题，还会给用户留下很差的体验。

作为一个网站所有者，你必须检查一些事情，使你的网站在浏览器和所有设备上运行得更快。

检查是否启用了 GZip 压缩。这将压缩所有网络文件，使网站加载速度更快。

确认是否启用了浏览器缓存。你可以使用 WordPress 插件为你的 WP 网站启用浏览器缓存。否则，直接在站点的 htaccess 文件上使用浏览器缓存代码。

在上传到网站之前，优化图像尺寸和文件大小。图像占用最大空间，需要很长时间才能完全下载。因此，优化图像的宽度、高度和文件大小对网站的加载时间有很大的影响。

考虑通过合并多个。css 文件合并成一个大的. css 文件，多个。js 文件压缩到最小。js 文件。

此外，你也可以要求搜索引擎优化机构使用 minified。css 和. js 文件以及用于小图像和图标的 CSS-sprite。

如果你有一个像样的预算，你也可以尝试付费 CDN 技术，使您的网站更快。

## [](#your-website-isnt-mobile-friendly)您的网站不支持移动设备

在当今的移动世界，每个人都在他们的移动设备上搜索东西，移动响应网站是一件非常正常的事情。但是，仍然有许多商业网站在设计时没有考虑到他们的移动用户。如果你的网站是那些非移动响应网站之一，那么你每月都在承受巨大的损失。因此，这是关注移动搜索引擎优化和重新设计你的网站使其对移动设备友好的好时机。

此外，谷歌已经表示，他们把网站的移动响应作为一个严重的排名信号。随着时间的推移，那些还不适合移动的网站将会失去他们的有机排名。这意味着，如果你想让你的网站在谷歌和其他大型搜索引擎上排名靠前，你就不能再忽视这一点了。

咨询你的网站设计者和开发者，按照现代的响应式网站设计理念和技术来改造你的网站。这不仅会提升你的网站在搜索引擎中的排名，还会给你带来更多忠实的客户。

记住，在重新设计网站的时候，尽量不要改变网站的内部 URL 结构。

## [](#you-are-not-tracking-your-visitors-behavior-and-experience)你没有跟踪你的访客的行为和体验

未能衡量网站访问者的行为会付出很大的代价。用户体验是当今最强的排名信号之一。如果你网站的大部分访问者离开你的网站，而没有停留在网站上或访问其他页面，这对你的网站来说是致命的。

最糟糕的情况是，当你网站的有机访问者离开你的网站，回到搜索引擎，然后访问你的竞争对手的网站。这被称为弹簧贴。谷歌对此非常重视，最终这会导致搜索引擎排名下降。

你可以做几件事来让你的访问者关注你的网站及其内容。

首先，分析网站的设计和加载速度。你的网站设计必须是用户友好的。确保菜单链接在各种设备和屏幕尺寸上都易于访问。

开发人类可读的内容。过度优化的内容不会让读者和搜索机器人都满意。此外，你的内容必须回答关于其主题的所有可能的问题。投资制作没有语法、拼写和标点错误的精彩内容。

使用吸引人的图片、gif 和视频来留住你网站的访问者。此外，你也可以使用其他视觉媒体，如信息图和播客来增加观众的停留时间。

允许用户与你、你的内容和其他用户互动。鼓励访问者留下他们的评论并回答其他人的问题可能是建立强大关系和积极用户体验的好策略。

## [](#you-are-not-using-google-search-console-aka-google-webmaster-tool)你没有使用谷歌搜索控制台(又名谷歌站长工具)

谷歌搜索控制台原名谷歌站长工具，是 SEO 的必备工具之一。如果你作为一个网站所有者不使用这个工具来管理你的网站或博客，那么你肯定会错过很多关于你的网站的重要数据。所有这些数据都应该详细分析，以增加一个网站的整体搜索引擎优化。

首先，您需要添加您的网站，并在谷歌搜索控制台工具上验证它。

接下来，你应该把你的网站的 sitemap 文件添加到谷歌搜索控制台，每当你想出一个新的页面或者在你的网站上发布一篇文章的时候，重新提交它。

定期检查该工具是否列出了任何与 DNS、服务器、robots.txt 和您网站的损坏 URL 相关的爬网错误。如果你看到这些，请尽快采取必要的措施。

使用“获取为谷歌”功能通知谷歌你最近在你的网站和它的网页上所做的改变。这有助于更快地抓取和索引“提取”的 URL。

检查谷歌是否一直在“索引状态”部分索引你的页面。如果您看到索引页面的数量大幅下降，那么您应该执行一个完整的站点分析，包括 onpage、content 和 backlink 审计。

如果你在谷歌搜索控制台工具的“移动可用性”部分发现任何错误，你应该尽早修复这些错误。

仔细检查“手动操作”部分，以确认您的网站是否没有被谷歌采取任何类型的手动操作。

使用搜索控制台分析您获得的反向链接，以验证它们的相关性、权威性和毒性。如果你看到一个相对较差的域名链接到你的网站，你可能会否认使用谷歌搜索控制台工具。因此，从下一次开始，只要你把这个链接保留在你的否认文件中，谷歌将开始把它算作一个非关注链接。

养成每天使用搜索控制台工具检查关键字数据的习惯。这一部分列出了不同关键词搜索下你的网站及其页面的印象、点击、点击率和平均位置。

除了谷歌搜索控制台，还将您的网站与谷歌分析工具集成，以详细测量所有与流量相关的数据。
