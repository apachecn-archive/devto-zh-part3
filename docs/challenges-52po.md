# 挑战

> 原文：<https://dev.to/laviku/challenges-52po>

因此...我们从二月开始，今年的目标呢？已经把它们留给下一个像我一样的人了😅？

嗯，这不是让你感觉不好或者打击你的积极性，恰恰相反，我在这里告诉你一些你可能不知道的挑战。

每个人或者至少我们中的大多数人都知道 Codepen，这是一个很棒的地方，人们可以在这里分享 HTML、CSS 或 Javascript 代码形式的作品，我们可以在这里搜索客户希望在他的网站上看到的非凡的东西，或者不是程序员的设计师认为不错的东西。

Codepen 每周都会发布一个挑战，我不确定他们是多久前开始的，因为直到今年我才注意到我的通知栏里的消息😩。每个月都有一个特定的主题，每个星期他们都会给你一些元素，然后你就可以尽情发挥你的想象力了，一月是色彩，二月是形状。

由于我在视觉方面，即设计方面，不是一个有创造力的人，我决定认真对待这些挑战，我真的重申我在这些方面是多么缺乏创造力，但我会继续在每个挑战中尝试它们。你可以进入我的 [codepen 个人资料](https://codepen.io/laviku)，你不会发现任何令人敬畏的东西，但我正在努力，这里的[是我收集的年度挑战，这里的](https://codepen.io/collection/XMoaZz/)[是这个月的第一个挑战。](https://codepen.io/challenges/2019/february/)

尽情享受吧！！
