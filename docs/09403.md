# 找不到新工作

> 原文：<https://dev.to/protium/failing-at-getting-a-new-job-1dlg>

自从我告诉自己“是时候找一份新工作，搬到国外去了”以来，差不多一年多了 还能出什么差错？

[![-](img/cb0e87a3a8518beca70819e0437c8499.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--qospH1a5--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/qgph2ere9agww0229d6c.jpg)

更新:我在 5 月获得了德国公司的签证，该公司在 1 月联系了我(我们审查了现行法律并申请了正确的文件)。自八月以来，我一直住在慕尼黑。

### 简要介绍

我是一名自学者，从事 JavaScript 和. NET 的全栈开发，但我也使用过 Python 和 Ruby。20~21 yo 开始职业生涯。现在我 25 岁了，我为我目前所取得的成就感到骄傲。

### 第一次尝试:亚马逊

> **日期:2018/10**

是的，亚马逊。不知何故，他们觉得我的简介很有趣，告诉我他们将于 11 月在布宜诺斯艾利斯举办一场招聘活动。工作描述是:**有 5 年经验的软件工程师**和一长串所需技能。
为了有资格参加这次招聘活动，我必须通过一项代码挑战。我已经知道我会失败，但我至少需要尝试一下，并了解大公司是如何寻找候选人的。
代码挑战由两个问题组成。一个**k-最近点**问题，有一些变化和某种**回溯**问题。问题之后，我必须回答一些关于时间复杂性的问题。

*   **结果:**失败
*   问题:我没有好好练习
*   吸取的教训:多去练习。阅读更多关于数据结构、动态编程等内容。开始看一些关于**竞技编程**的书，在[hackerrank.com](//hackerrank.com)解决问题

### 第二次尝试:一家瑞典初创企业

> **日期:2018/12**

我积极地在 stack overflow 找工作。我在寻找一家公司，在那里我可以使用我感兴趣的技术，而且从我的角度来看，主要产品非常棒。我主要搜索拥有环境或医疗保健相关产品/解决方案的年轻初创企业。

我遇到了一家很酷的瑞典公司，我申请了一个高级软件架构职位。这一次我和 HR 进行了第一次面试。然后他们发给我一个代码测试，我必须用一个客户端应用程序构建一个 API，负责 TDD 和 CI。我非常喜欢的东西。
然后我参加了一个技术面试。但是，在面试的前一天，我决定用**打动**他们。我安装了他们的 **Android** 应用并反编译。我做了些逆向工程，读了他们的代码。我还找到了一个 **Firebase API 键**，并证明我能够访问一个没有相关数据的小 db，但是我能够添加条目。数据库没有任何限制规则。在面试之前，我知道很多关于他们的架构和技术堆栈的事情，所以我列出了我将在应用程序中改进的事情，并告诉他们关于 API 的关键。这次采访真的很棒，他们对我的回答很兴奋。

*   **结果:**失败
*   问题:我的英语口语不够好，不适合这个职位，因为我必须与公司的每个团队进行交流。
*   **经验教训:**多说话！我说西班牙语，因为编程我学了英语，但是我没有很多机会和说英语的人交谈。所以，我雇了一名英语老师，并开始参加一些人们用英语交谈的会议。

### 第三次尝试:一家德国初创企业

> **日期:2019/01**

这次是 CEO 联系的我。他认为我非常适合公司的全栈职位。我们安排了一些采访。我用一个非常简单的任务进行了结对编程。然后被**产品负责人**面试。最后又是和 CEO。我进去了！他们给了我一个好价钱。

*   **结果:**失败
*   问题:没有大学学位，要获得德国(或欧盟)的工作许可很难，但还是有可能的
*   **经验:**搜索更多关于工作许可证和移民法、要求等的信息。我发现，如果**的工资足够高**，并且你有资格成为**高技能专业人士，就不需要大学学位。**我还在 [Edx](https://www.edx.org) 和 [Udemy](https://www.udemy.com/) 上了一些网络课程，拿到了**证书。**

### 第 I 次尝试:一家德国初创企业和一家荷兰初创企业

> **日期:2019/03**

我已经提高了我解决问题的能力和我的英语水平(仍在两方面努力)，我读了很多关于工作许可和移民法的书。与此同时，我也学会了如何谈判我的薪水和福利(我不清楚该怎么做)。这一次，我首先向人事代理人提到了我对工作许可证、签证担保和期望薪资的担忧。他们告诉我，他们肯定不会有任何问题，因为我有多年的经验。

我已经通过了两家公司的代码测试，下周我会有更多的面试。我申请了两家公司的**全栈**职位。

*   **结果:**待定

### 奖金:如何辞职

这不是一项容易的任务。有些人可能会说“我不干了”，他们准备继续前进。我考虑移居国外已经快一年了。去年五月，我决定和我的项目经理谈谈。作为项目经理，他试图与我重新谈判，甚至给我在任何地方工作的可能性。作为一个朋友，他告诉我，我正在做正确的事情，我还年轻，我有所有这些机会；我必须这么做。

所以，如果你正在考虑辞职，试着回答以下问题:

*   为什么会有这种感觉？
*   你和你的家人、朋友和/或同事谈论过这件事吗？

我想退出的主要原因是:

*   我喜欢欧洲的语言、文化和风景
*   我开始觉得自己陷入了职业生涯，我想从事像机器学习这样的新事物。

我要带着一个真正热情的告别离开公司。我认为这是可能的，因为在我做出最终决定的几个月前，我就和他们谈过并表达了我的感受。

### 最后的话

整个过程我学到了很多。作为一个成年人(有时我会忘记自己是一个成年人)和一名专业软件开发人员，我觉得自己更加明智了。我学会了如何接受采访，不仅是技术问题，还学会了如何和 T2 谈论我这个潜在的候选人；如何告诉他们为什么我非常适合这个职位和公司。我学会了如何**发现自己的弱点**以及如何克服它们。

有时候接受坏消息可能很难，得不到你渴望的工作。但是不要放弃。如果你向他们展示你的潜力，会有很多公司愿意雇用你。如果你在面试中失败了，友好地请求**的反馈**。

不断学习。尝试每天解决问题，阅读关于新技术的文章(顺便说一句，我爱这个社区)，尝试新的框架或语言，开始兼职项目。走出你的舒适区。
