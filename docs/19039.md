# 技术术语:终端命令别名

> 原文:[https://dev . to/redun can/tech-speak-terminal-command-aliases-2 of](https://dev.to/reduncan/tech-speak-terminal-command-aliases-2olf)

你曾经厌倦打字吗../通过终端导航时，在文件夹结构中向上移动一级？mkdir 怎么样？或者甚至清晰？如果使用终端来浏览你的文件结构应该是快速和容易的，为什么不通过别名化某些命令来使它变得更容易，这样你就可以使用更少的击键和更容易记忆的术语来达到同样的效果。

要做到这一点，您需要确保您设置了永久的别名，否则每次您打开一个新的终端窗口，您将不得不再次设置别名。要永久设置别名，您需要创建一个文件，终端每次加载时都会读取该文件。

在您的终端中键入以下命令...

```
cd ~ 
```

这将带你到你的根目录。

```
touch .bash_profile 
```

这将创建一个存储您的别名的文件，每次加载时您的终端都会读取该文件。

```
open -e .bash_profile 
```

该命令将在默认的文本编辑器中打开新文件。

现在我们已经在文本编辑器中打开了文件，我们可以开始设置别名了。在文本文件中，您将通过键入 *alias* 创建一个别名，后跟您想要的快捷方式，后跟等号，最后是实际的终端命令。

```
alias create=mkdir
alias clr=clear
alias .=../ 
```

一旦你有了别名，保存文件并关闭它。重新加载您的终端并测试您的别名。

这就是你如何给你最常用的命令起别名，使通过终端导航更容易。
