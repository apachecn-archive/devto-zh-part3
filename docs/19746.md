那么什么是 DNS 呢。我们都听说过 DNS 查找失败以及各种各样的事情。这是怎么回事？

DNS 或**域名服务**是一种特殊的服务，它将 IP 映射到地址。我们再深入了解一下。

假设你输入 www.gooogle.com。现在，请求必须从你的计算机发送到谷歌服务器，以获取谷歌页面。对吗？

是的。但是你的电脑怎么知道谷歌在哪里呢？这就是 DNS 的用武之地。

每台连接到互联网的服务器或设备都有一个 IP 地址。这就是我们如何确定他们在哪里。

> 跟找一个人差不多。如果有人告诉你去找杰克，你可能会搞不清楚杰克究竟住在哪里，但如果有人告诉你杰克住在印度孟买，你就清楚多了。

那么在 DNS 服务器中会发生什么呢？

您的计算机接受该请求并转到最近的 DNS 服务器。DNS 服务器是全球分布的，因此任何地方的任何人都可以随时访问它们。

DNS 服务器将为您的地址提供相应的 IP 地址。这种从 DNS 服务器获取信息的方法被称为**查询**。

一旦获得，你就可以自由地去谷歌服务器，因为你现在有了确切的 IP 地址。

DNS 有不同的记录，即

1.  **一个**或者 **AAAA**
2.  CNAME
3.  前进速度
