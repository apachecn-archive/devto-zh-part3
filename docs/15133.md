# 替代文本中应该包含颜色吗？

> 原文:[https://dev . to/knpfletcher/should-color-be-include-in-alt-text-1a6d](https://dev.to/knpfletcher/should-color-be-included-in-alt-text-1a6d)

关于如何为图像元素编写有意义的 alt 属性描述，对于那些使用屏幕阅读器或者连接速度很慢却无法加载图像的人来说，已经写了很多。相对于空的 alt 文本属性，什么时候使用向 alt 属性添加描述是有基本规则的。我在任何地方都找不到的是在 alt 属性描述中包含颜色是否有用。甚至 [W3 开发者指南](https://www.w3.org/TR/WCAG20-TECHS/H37.html)和 [WebAIM](https://webaim.org/techniques/alttext/) 都没有在 alt 属性描述中提到颜色。我一直在观察许多网站的图片替代文本，从博客到公司，以及利用相对较新的替代文本功能的 Twitter 用户。很多次我看到一幅图像被描述为“一个穿红色衬衫的人在读一本蓝色的书。”或者“明亮的蓝天下的一大片绿地。”

如果有人有任何见解，我很乐意学习更多来写最好的 alt 属性描述。
