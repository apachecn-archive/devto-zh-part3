# 社交网络-开发者

> 原文:[https://dev.to/tomavelev/social-networks-开发商-5e24](https://dev.to/tomavelev/social-networks---developers-5e24)

在二十一世纪，如果你不“在线”，你就是一只恐龙。你必须做一些与众不同的事情——对这个世界有益，能够“扩展”“你自己”，你的想法，你的遗产。以当前顶级平台接受的形式种植你的精华——视频、文本、音频等，可以让你的祖父母和更多人有机会了解你，即使你已经离开人世。更重要的是，它们允许你在活着的时候接触到别人。

脸书是(理论上)接触到很多人的平台之一。但最近脸书变得昂贵，同时在其他几个方面很穷。

它变得昂贵是因为——有机接触(不用付钱给 Zucks 做广告)越来越小。而且即使是付费的时候，触达也不如以前了。可能这就是为什么许多营销人员转向利基市场——教人们关于社交媒体。最终，不管使用何种媒介，它的核心原则都是一样的。如果你听过加里·维纳查克的歌，你就会知道原因是什么——当很多人试图吸引同样的听众时，注意力的价格就会上涨。因此，需要应用新的平台/更好的定位/或一些其他策略，以便更有效地获得资金或工作的投资回报。

与此同时，我发现越来越多的人不再使用社交网络。为了虚荣心——为了“装酷”,为了喜欢，为了戏剧化，为了单纯的存在，或者在暂时的趋势上冲浪，点击标题，利用人们对某个特定主题的无知或不了解——一般地和具体地——用[不那么令人愉快的]内容或愤怒的政治内容玩弄情感，这并不总是正确的，即使在某些部分——再多的情感也不会带来任何结果——除了冻结公众，用无意义的内容填充头脑。

我以前使用过一些策略，我也不是圣人，但我越来越觉得这不那么有尊严和冷静，尤其是从哲学和纯技术的角度来看。也许是我的原则阻碍了我在社交媒体上的成功，但这就是我。
**编程和软件的问题在于它是布尔型的——真或假，1 或 0。例如，当软件 X 只是通过网络传输文件或打开或关闭灯时，你不能说它发射和着陆火箭。没有信仰，没有相信，没有主观的观点。它按照程序做，不考虑第三方的意见，它要么工作，要么不工作。**

由于上述原因，作为一个博客作者和开发者，我对我的内容的关注转移到了其他地方。就像我前段时间在一篇文章中写的那样——把所有的鸡蛋放在一个篮子里不是一个好主意。因此，我收集了一份清单，上面列有我已经去过的网站和平台，以及我打算研究、调查和去的其他网站和平台:
