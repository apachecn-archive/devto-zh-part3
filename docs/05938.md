# dbForge Studio for Oracle 现在有了新的外观& Oracle 19c 支持

> 原文:[https://dev . to/devart software/db forge-studio-for-Oracle-now-has-a-new-look-Oracle-19c-support-4do](https://dev.to/devartsoftware/dbforge-studio-for-oracle-now-has-a-new-look-oracle-19c-support-4do)

Devart 为 Oracle v.4.1 推出了新版本的 **dbForge Studio。该版本获得了一系列可靠的更新。更新后的工具支持 Oracle 19c，具有自动提交模式、改进的模式比较性能、事务提醒和执行通知。随着整体设计的革新，模式比较性能也得到了显著提高。**

以下是该产品更新版本中实现的功能的完整列表:

**Oracle 19c 支持** —支持使用最新的 Oracle 数据库；
**自动提交模式** —自动提交文档中最近的更改；
**改进的模式比较性能** —使使用该工具的工作更加高效和愉快；
**事务提醒和执行通知** —允许用户获得关于最近执行的查询和未提交事务的状态的最新信息；
**新皮肤** —使工作室的工作更加视觉愉悦，并提供改进的用户体验。

dbForge for Oracle 产品线中的其他工具，如 dbForge Data Compare for Oracle、dbForge Schema Compare for Oracle、dbForge Compare Bundle for Oracle、dbForge Data Generator for Oracle 和 dbForge Documenter for Oracle，也更新了对 Oracle Database 19c 的支持。
