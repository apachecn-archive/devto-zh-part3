# 在新公司介绍自己

> 原文：<https://dev.to/airercode500/introducing-yourself-485o>

在新公司的第一天总是有点尴尬；尤其是如果你是第一次。我自己通常在新的环境中保持沉默，保持安静(或者，有时大声得令人讨厌)。你会遇到很多人，花很多时间试图找出他们是如何组合在一起的。

我发现自己告诉学生和应届毕业生的最重要的事情之一是:**谈论你的激情**。

如果你认为你周围的人能在你生活或职业的某些方面进一步帮助你，告诉他们。如果你对某些技术或开发的某些方面充满热情，和人们谈论它。

> 如果我知道**你的名字**和**你的爱好** :
> 我可以给你带来机会。

## 向你的团队介绍自己

你每天都要和你的团队一起工作，和他们搞好关系很重要。许多公司更关心文化契合度，而不是技术能力。不管你有多聪明，经理们都不想雇佣那些会让他们现有团队痛苦不堪的人。所以如果你通过了面试，他们可能会认为你很适合。不要让他们失望。

当我刚开始目前的职位时，我意识到的第一件事是，与我习惯的相比，我的团队真的很安静。我是一个超级喜欢社交的人，喜欢聊天，但我的团队大多不喜欢聊天。如果我从第一天起就决定尝试改变这一点，那就真的会把事情搞砸。为了提供社交机会，我举办了一些社交活动，包括棋类游戏午餐和其他团队郊游。

找到你在团队中的位置并找到你的合适人选需要一段时间，但这将会发生。可爱狗狗的 gif 通常是一个很好的起点。

## 向导师介绍自己

如果你在一家规模不错的公司工作，很有可能会有人帮你起步。很多时候是你团队中的某个人，但有时也可能是你组织中的某个人。

**如果你被指派的导师在你的团队中:**我建议在你的团队附近再找一个导师。有时会出现问题，让一个不直接参与你的团队的人来讨论问题是很好的。

与你的导师的谈话应该不那么正式，但不一定是无组织的。很多地方都有入职清单，如果你的公司没有，要求一份也无妨。对你应该做的事情有一个共同的理解是很好的。

诚实也是你和导师关系的一个重要方面。如果你正在努力寻找文档或有其他困难，那么与上游沟通是很重要的。我不知道有哪家公司认为自己完善了入职流程，总有改进的空间，而这种改进依赖于诚实的反馈。

导师是很好的资源，因为他们可以给你指导，帮助你成功。赞助商是一种特殊类型的导师。导师通常会回答你的问题并给予指导，当你不在的时候，赞助商会为你做所有的事情并为你游说。他们给你带来机会，而你不一定会问。不是每个导师都会主动成为赞助人，但是如果你能成功地找到这些人，他们会带你走得更远。

## 向经理介绍自己

你的经理是少数几个把你的成功写进工作描述的人之一。你应该定期与你的经理会面，充分利用这段时间很重要。

如果你对加入一个团队有顾虑，而你的经理不知道，他们对此无能为力。同时，如果有些事情你感兴趣而你的经理不知道...他们对此也无能为力。

与你的经理保持良好的关系会让你更容易解决出现的问题。在以前的工作中，我见过员工之间互相抱怨的情况，并认为经理出于恶意(而不是无知)忽略了这些情况。

友好和诚实，并希望得到同样的回报。
