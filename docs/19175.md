# 介绍 Laravel Links 电报频道

> 原文：<https://dev.to/adi/introducing-laravel-links-telegram-channel-2iab>

大家好，我还是 Adi，我想向大家介绍一下我大约一周前的一个想法。所以，大概一个星期前，我在找一个每天分享 Laravel 和 PHP 相关资源的时事通讯或者 facebook 页面。我找不到任何积极维护或张贴随机链接，我不想看到的。

经过几天令人沮丧的寻找这个完美的 Laravel 链接流，我决定[创建我自己的](https://telegram.me/laravellinks)。于是就出现了哪个“社交媒体”的问题。在考虑了易用性、可伸缩性、自动化发布流程和其他方面之后，我决定采用电报渠道。电报频道勾选了所有必要的复选框。它很容易使用，许多开发人员都在使用它，对成员没有限制，最棒的是，我可以用机器人自动完成发布过程。

2018 年 11 月 28 日，我创建了一个电报频道，并发布了一条关于这将是什么的消息。我给它的描述是- Laravel Links 是一个电报频道，致力于分享伟大的 Laravel / PHP 资源、教程、代码片段、提示、一般信息、官方更新、视频、问题、论坛帖子等等。截至 2018 年 12 月 3 日，共有 41 名成员，平均消息浏览量为 36，发布了 23 个链接。
