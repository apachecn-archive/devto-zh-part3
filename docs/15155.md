# 你最喜欢的 Markdown 编辑器是什么？

> 原文:[https://dev.to/dechamp/whats-your-favorite-markdown-editor-70](https://dev.to/dechamp/whats-your-favorite-markdown-editor--70)

## [](#the-discussion)讨论

所以我多年来一直在写 Markdown，但从未想过使用官方编辑器。我确实使用了来自 Jetbrains 套件(PHPStorm，WebStorm，Intellij，...)但是插件和默认的 markdownn 编辑器在 UI 模式下似乎有点问题，所以我把它们放在文本视图中。

我的同事使用名为 [LightPaper](https://getlightpaper.com/) 的官方编辑器。我决定今天去寻找我自己的。我为 mac 找到了一个名为 [MacDown](https://macdown.uranusjr.com/) 的免费软件，到目前为止我很喜欢它。很简单也很管用。

我确实想念我的 VI/M 快捷方式(想学 vim 吗？结帐 [Marc 的帖子在上面，有 9 个部分！](https://dev.to/webdevchallenges/a-brief-introduction-to-vim-o2)，他们太厉害了！但是我想没有它们我也能生活，因为我可以看到我的 raw markdown 和 UI 版本。

所以列出你的编辑器，或者如果你只是坚持原始的。对题目的想法？编辑器，插件，或者任何对 Markdown 来说很酷的东西。

很期待你的想法。

-说吧
