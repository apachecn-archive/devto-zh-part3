2018 年，Typeform 的工作人员很高兴在 Nordic APIs、JS Heroes 或 hackathons(如巴塞罗那的 HackUPC)或伯明翰的 hack conference 等会议上见到你们中的一些人。在这些活动中见到用户和合作伙伴总是很棒。您帮助我们构建了 Typeform 及其平台的未来🤩

我可以告诉你，对于 2019 年，从我们所在的地方，我们有一个新的奇妙的观点，满脑子的想法，没有人告诉我们不。有成千上万的东西可以看，还有很多我们从来不知道我们想去的令人眼花缭乱的地方…

唯一清楚的是，现在我们和你们——我们亲爱的开发者社区——一起进入了一个全新的世界！

所以，让我们开始新的一年，一起踏上魔毯之旅，看看我们的下一个目的地。

我们将从去博尔德参加几项活动开始我们的冒险:

#### [](#boulderjs-meetup-boulder)🇺🇸博尔德聚会，博尔德

2019 年 2 月 20 日

对于我们博尔德之旅的第一步，我们将在博尔德 JS meetup 停下来，会见当地的 Javascript 社区。

我将分享我使用 Vue.js 构建第一个应用的经验，当时我创建了 [https://🌚🌞。ws](https://%F0%9F%8C%9A%F0%9F%8C%9E.ws) 追踪 2018 年日食事件。

🎟️ [注册](https://www.meetup.com/Boulder-JS/events/xmkdvmyzdbbc/)(免费)

#### [](#writethedocs-boulder)🇺🇸WriteTheDocs 巨石

2019 年 2 月 21 日

WriteTheDoc 是一个由科技作家组成的全球社区。去年，我们参加了他们在巴黎和伦敦的姐妹活动 API docs。很高兴受到博尔德本地团体的欢迎，来交流文档和开发人员经验最佳实践。

我将在 Webhooks 上展示关于开发者体验的最佳实践。

🎟️ [注册](https://www.meetup.com/Write-the-Docs-Boulder-Denver/)(免费)

#### [](#typeform-office-hours)🇺🇸Typeform 办公时间

虽然我们将在博尔德，我们认为我们想满足我们可爱的用户。这就是为什么我们在周四 21 日和周五 22 日安排一些办公时间。这是您与我们见面并带着您关于 Typeform 产品的问题前来的机会。我们将在这里帮助你，特别是如果你有兴趣使用我们的 API 来构建东西。

🎟️ [在这里注册](https://developerplatform.typeform.com/to/azugOg)(免费)

#### [](#hackcu-boulder)🇺🇸HackCU，博尔德

2 月 23 日至 24 日

我们没有赞助很多学生黑客马拉松，但当我们赞助时，是因为我们与他们的团队有着特殊的联系。我们很高兴地拜访了 HackUPC 的创始人 Gerard Sans，他现在是 CU-Boulder 的硕士生，我们在同一所大学四年级的暑期实习生 Shubha Sharma。他们是组织 HackCU 的优秀团队的一部分。

迫不及待地想看看黑客将建立什么！

#### [](#api-meetup-bcn-barcelona)巴塞罗纳🇪🇸API Meetup BCN

2 月 28 日

巴塞罗那被选为每年一周的世界移动之都已经超过 10 年了。在一周的时间里，来自世界各地的 10 多万人将讨论技术的最新趋势。

利用所有来到巴塞罗纳的人的机会，当地的 API meetup 将举办他们的年度 MWC meetup，我们很高兴能在我们的总部举办！加入我们的 API nerding 之夜吧！🤓

🎟️ [注册](https://www.meetup.com/API-Meetup-Barcelona/events/258461375/)(免费)

#### [](#women-techmakers-barcelona)🇪🇸Women 科技城，巴塞罗纳

4 月 6 日

你还有一点时间来准备，但是我们非常兴奋能够成为巴塞罗那第二届女性技术标记会议的一部分。

🔗[了解更多信息](https://womentechmakersbcn.github.io)
