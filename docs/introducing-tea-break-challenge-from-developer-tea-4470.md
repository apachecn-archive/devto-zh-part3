# 介绍来自开发商 Tea 的茶歇挑战

> 原文：<https://dev.to/jcutrell/introducing-tea-break-challenge-from-developer-tea-4470>

今天，我很自豪地宣布茶歇挑战，这是开发者 Tea 的一项日常软技能练习。

### 什么是茶歇挑战？

每天都可以做一些提升自己的事情。你可能不会每天都学习一种新的编程技术或技巧，但无论你在哪里，你都可以每天花 *5 分钟*来提高你的思考和正念习惯。

自从我在 2015 年 1 月开始开发 Tea 以来，我们一直以每周 3 集的速度发布播客。现在，我决定将这些内容扩展成更简单的书面格式。

2019 年的每一天，你都会看到新的挑战。我们到处都在张贴这些照片——insta gram、Spectrum.chat、Twitter (@developertea 几乎无处不在)...甚至有一个免费的基本 JSON 端点可用:[https://www.teabreakchallenge.com/today.json](https://www.teabreakchallenge.com/today.json)-很想看看用这个构建的一些很酷的东西！请继续关注更多地方，因为我们正在寻找新的方式来分享这些内容。(我们可能很快就会建立电子邮件服务。)

开发人员的额外收获:我还开发了一个短信机器人。直接给我发消息，如果你想要每日短信，我会给你订阅的号码。目前，这条消息在东部时间上午 9 点发出。

我很想听听你对此的想法！感谢那些已经听过开发者茶的人。我希望茶歇挑战能成为你成为更好的每日开发者的一个可操作的方法！
