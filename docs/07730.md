这是来自我的博客[我的网络开发历程](https://mywebdevelopmentodyssey.dev)的帖子

这篇文章强调了这个博客的另一个目的。这就是精神疾病，在这个特殊的例子中，抑郁症，如何影响学习。我大半辈子都在和抑郁症打交道。我最近一轮的抑郁症已经持续了几年，并在去年秋天达到顶峰。去年 12 月和今年 1 月，我开始感觉好些了。

我认为学习 web 开发是帮助我走出黑暗的因素之一。然而，从大约五周前开始，我开始有点下滑。动力更少，目标更不明确。当情况变糟时，我开始感到绝望和无助。洗碗、倒垃圾或平衡预算是我无法理解的任务。学习像 web 开发这样复杂的学科几乎是不可能的。它仍然没有接近去年年底的峰值，我正在努力确保它不会。

我开始注意到我学习 JavaScript 的动力在下降。不过我不确定，这是因为沮丧还是因为我似乎被高阶函数和函数式编程难住了。这可能会导致我的抑郁症恶化。哪个造成了哪个？是“被难住了”导致我的抑郁恶化，还是抑郁导致我缺乏学习的动力和明确性？事实上，这可能是两者的结合。

不管是什么原因，我都没有像我希望的那样尽快发帖。这个博客诞生不久，我希望它能健康长寿。这可能意味着有时要解释精神疾病和生活是如何阻碍我们的最佳意图的。

生活可能会很混乱，但我们可以重新回到正轨。所以，如果我们需要踢，抓，尖叫来摆脱生活中的冲突。全力以赴地战斗。我们能做到！

以下是我正在做的几件事。我希望用我的 Aeon Timeline 应用程序很快就能有我预期的学习网络开发时间表的第一个版本。你将能够在我的网页上与它互动，以检查更多的细节。我期待着尽快发布。我希望在那里见到你。

我最近从 Udemy 购买了 Maximilian Schwarzüller 的[NodeJS——完整的指南(包括 MVC、REST APIs、GraphQL](https://www.udemy.com/nodejs-the-complete-guide/) 。我喜欢视觉学习，Max 喜欢展示图形和图表。他也很热情。我期待着完成这个课程，也期待着他的[反应课程](https://www.udemy.com/react-the-complete-guide-incl-redux/)。使用 Gimp，我将尝试为我的博客创建一个自定义标志！
