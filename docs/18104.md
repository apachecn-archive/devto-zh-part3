# AoC 第 18 天:北极的定居者

> 原文：<https://dev.to/rpalo/aoc-day-18-settlers-of-the-north-pole-9ek>

那个该死的康威很固执，不是吗？！没错，[第 18 天](https://adventofcode.com/2018/day/18)看起来像是康威的回归，但在二维空间中，有 3 种可能的状态(开阔的地面、树木繁茂的英亩和木材场),与我们几天前看到的不太复杂的活着或死了相反。

看看我们在正常的 2D·康威的生活游戏中看到的[重复的半稳定形状](https://en.wikipedia.org/wiki/Conway%27s_Game_of_Life#Examples_of_patterns)是否在这里出现，或者有 3 种可能的状态如何影响它们，这将是很有趣的。
