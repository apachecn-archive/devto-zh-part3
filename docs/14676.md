# 寻找意义..

> 原文:[https://dev.to/kodero/in-search-of-significance-3n5d](https://dev.to/kodero/in-search-of-significance-3n5d)

让我们来谈谈`feeling`和`emptyness`，在一个人的生活中没有任何重要的事情发生。我暂时忽略了这一点。不过最近，到了我想做点什么的时候了。

### [](#a-little-bit-about-my-background)稍微介绍一下我的背景。

大约在 2006 年，我在大学期间开始编程。我最初很讨厌它。语法规则真的让我心烦。那是 C/C++。然后我学了 Java，有点喜欢它(这可能是因为有一个很好的 IDE 可以开发。当语法关闭时，我可以得到提示，并且只需要处理逻辑错误)。大学快结束时，我在一家开发店找到了一份(初级)软件开发员的工作，这家店以移动服务产品为核心。我记得我做过一个订阅服务，允许足球迷订阅足球比赛的比分。在后面，我们从 live score 获得了实时反馈。

从那以后，我在许多公司以不同的身份工作过。最初在小型开发商店(最多 20 人)工作，心情很复杂(有一天会写一篇关于这个的帖子)，目前是肯尼亚顶级银行的企业架构师。这是一次疯狂的经历，我喜欢我现在的工作，因为它让我可以鸟瞰公司的技术。当然，每份工作都有其微妙之处，但我们学会了生活并超越这些。这不是故事的全部，希望它足以让你明白我来自哪里。

我是一个狂热的读者，我关注很多技术新闻，包括新工具(有时，我会玩它们，制作玩具产品)。我喜欢 DevOps，一直在阅读，并设法在 docker 上主持了一些东西。一直在阅读 kubernates，但它似乎需要体面的环境来有效地学习，所以这可能是以后的事。我做前端工作(angular 等)，也做后端工作(主要是 Java，嗯，还有 python)。虽然我做了很多兼职，但我的日常工作并不涉及这些。

我有一个家庭，一个可爱的妻子和三个孩子(两个儿子和一个女儿)。在我三十五六岁的时候

但是每天，尤其是最近，我无法摆脱这样一种感觉，那就是生活一定比我每天所做的事情更重要。上周末，我开始思考并列出一些我想做的事情来缓解这种情况。我知道不能只有我一个人经历这些。所以我的问题是，你有同感吗？你的解决方案是什么？我真的很想听听你的想法。
