# 一致性是有回报的

> 原文:[https://dev.to/eekayonline/consistency-pays-off-1c7h](https://dev.to/eekayonline/consistency-pays-off-1c7h)

[![Consistency Pays Off](../Images/67d2161212af0e54efe7465fe57ae1d4.png "Consistency Pays Off")T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--ZEacCXVe--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/vf7mz7d803ezcroj6tuy.png)

如果你有任何想要达到的目标，一致性是关键。你想培养观众，销售产品，打造自己的品牌吗？坚持不懈会有回报。真的。

通过持之以恒(以一种频繁的、可预测的方式进行同样的努力)，你将能够让你的听众增长。这是推动稳定增长的因素。

但是不要期望这是一个快速的破解。这是你需要在中长期做的事情。

持续的努力让你的目标受众适应你的信息。这是让他们知道你的产品和服务。

我的建议是最大限度地提高您持续努力的效率:

*   保持你的信息真实
*   不要广播，联系你的观众和目标市场
*   不要去做销售，而是提供价值
