[![](../Images/66c0e2cc3424dd2d9201422c81c79faa.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--YGPMZl6L--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://cdn-images-1.medium.com/max/480/1%2AKkLopdciBRMu5FZTyyLTuw%402x.jpeg)

2018 年是个好年景。我经历过起起落落。我有一些非常大的失望。

目标是不存在的。我从来没有真正写下目标并计划好的目标。我从未深入思考过我想做或想完成的事情。他们来的时候我就照做了。随着时间的推移，我开始思考我想做和学的事情。

所以，我在 Asana 做了一个项目，2018 目标，列出了今年剩下的几个月。我把事情放在我想做这些事情的月份下面，从那里我开始做这些事情。

有很多我想做的任务。有时候，我似乎没有足够的时间去做所有的事情。

现在这一年还有不到两个小时就要结束了，我想这一年的总结会让我看到我做了多少。

### [](#2019-goals-completion)2019:目标完成情况

#### [](#coding)编码

**目标:JavaScript 应用和后端**

我设计或编写的许多程序和网页都发生在今年的前 8 个月。我快结束时停下了。我没有方向。但我希望这种情况会在 2019 年有所改变。这一次，我对自己想做什么以及如何实现这个目标有了更多的想法。

100 天的代码将继续我的第二轮，但我在第二轮的第五次尝试。我有它，所以如果这个月没有更大的项目，我必须每个月做一个 JavaScript 项目。我有网站设计的想法。所以当我用完 JavaScript 项目时，我可以只使用这个站点列表。

自由代码营将会回归。我只想完成 JavaScript 和前端框架证书。

#### [](#blogging-and-social-media)博客和社交媒体

我只想做得更多。我想要一个月至少 1 个职位。我想分享我创建的网站的图像和视频。
