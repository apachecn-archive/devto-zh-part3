## [](#rounding)四舍五入

`Math.round`返回舍入到最接近的整数的数值。

```
Math.round(4.2); // 4
Math.round(4.7); // 5
Math.round(4.5); // 5\. Half-way values are always rounded up
Math.round(-4.5); // -4 
```

## [](#incrementing-)递增(++)

`++`将其操作数增加 1。

```
// postfix: returns the value before incrementing
let a = 4, // 4
  b = a++, // 4
  c = a; //5 
```
