# 塑造数据安全未来的技术

> 原文：<https://dev.to/brettclawson75/the-technologies-shaping-the-future-of-data-security-4bd0>

[![data security](img/09bfbd0541762d4d0de541e8f0f68b0d.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--02OXG0af--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/4mvpssq4yscxrihhj212.jpg)

没有一篇文章或新闻不关注当今世界的安全漏洞。无论是在医疗保健提供商领域、商业层面，还是在消费者层面，关于预防和安全攻击的数据似乎铺天盖地。随着时间的推移，勒索软件和网络钓鱼等术语已经变得家喻户晓，几乎每个人都知道有人受到了安全攻击和数据泄露的影响。有两个方面会使保护网站和在线数据变得复杂。首先，在线工作的组织可能需要在访问他们的关键数据和他们的 [安全措施](https://www.huffingtonpost.com/entry/9-technology-mega-trends-that-will-change-the-world_us_5a2e79b0e4b04e0bc8f3b69a) 之间取得平衡。该政策适用于医疗保健和电子商务领域。虽然在线零售商仍有可能锁定他们的应用程序，但底线是贸易商将需要能够让他们在快节奏的环境中提供服务的解决方案。因此，找到与组织的风险承受能力保持一致的正确平衡，以确保其安全措施与其威胁相匹配，变得至关重要。

用户群体本身是第二个方面。在许多组织中，用户既是最后一道防线，也是第一道防线。随着威胁空间不断变得复杂和增长，对抗勒索软件和病毒攻击的工具也在不断成熟。组织正在投资更多工具，如 [电子邮件保护软件](https://www.symantec.com/products/email-security-cloud) ，并投入更多资金聘请安全信息专家，但系统和网络用户仍然是最易受攻击的。无论是意外的还是恶意的，组织只需要一名工作人员打开错误的页面或单击可疑的链接，就可以引发一系列可能导致安全漏洞的事件。然而，网络安全专家正在研究几项技术，这些技术可能会在未来几年内改变行业，使组织更容易管理与敏感数据安全相关的任何风险。

**信息共享和有效协作**

这是最近开始发生的一种威胁方式，但未来学家预计随着技术的不断发展，这将变得更加常见。然而，快速清晰地分享安全攻击信息的能力将有助于在线交易者保持略微领先于竞争对手。 [安全信息](https://www.theguardian.com/technology/data-computer-security) 的共享最终将扩展到网上交易者之外，因为其他服务提供商也共享各自行业的风险格局和漏洞。

**专注于培训和教育用户**

与运输和机场安全措施一样，组织需要培训和教育他们的员工如何预防和应对安全攻击。随着员工越来越意识到自己的责任，他们可以避免可能导致安全漏洞的代价高昂的做法。意识和培训还将使员工在工作场所和家中更加安全有效，并提高他们在这个数字时代的移动性。此外，几个代际问题可能会影响在线零售商未来应对威胁的方式。年轻一代似乎更愿意对自己的生活保持透明和开放。例如，无论年轻人是在聊天还是在推特上谈论他们在哪里，他们看起来都乐于分享他们的任何信息。专家认为，这可能会改变范式，并确保只有少数事情可以通过安全和隐私控制来解决。

**转向基于云的解决方案**

随着时间的推移，组织将会把更多的信息安全工具迁移到云中。因此，在线交易者将能够动态更新他们的 [工具](https://www.forbes.com/sites/theyec/2018/11/28/five-trends-that-will-shape-it-in-2019/#20e79ac03f39) 来应对任何恶意软件或病毒。向云的迁移似乎比其他存储选项更经济，并且将使所有员工无论身在何处都可以使用这些工具。报告的大量数据泄露可能会改变组织如何投入资源和资金来保护其关键系统和网络的游戏规则。组织将需要针对某些情况的解决方案，例如他们的大部分记录将在某个时间点被破坏。
