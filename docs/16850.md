# React 库生成器

> 原文:[https://dev.to/jalalazimi/react-library-generator-5438](https://dev.to/jalalazimi/react-library-generator-5438)

React library generator 是一个 CLI，可以帮助您构建库的框架。它是完全可定制的，你可以决定你想要什么。此外，该 CLI 为您库创建文档，并提供各种测试实用程序。

[![](../Images/52e27d6fb874734831d593d8d954929d.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--6WduPdlg--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://unpkg.com/react-library-generator%400.1.5/assets/demo.svg)

## [](#features)特性

*   简单易用的 CLI 和完全可定制的
*   ES6 和流语法支持
*   测试(杰斯特、艾娃、酵素)
*   支持不同的风格语言(Sass，Less，PostCSS，CSS)
*   文档(Docz)
*   捆绑的汇总
*   捆绑 es 和 cjs 模块格式
*   运输通天塔
*   源地图创建
*   通过 esLint 自动编码林挺
*   支持对等依赖
*   用哈士奇保护你的 git 树枝
*   生成之前，请检查 NPM 上的库名可用性
