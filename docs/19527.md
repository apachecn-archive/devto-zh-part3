# [](#new-features)新增功能

兰花 [0.13.0](https://github.com/JavaEden/Orchid/releases/tag/0.13.0) 和 [0.14.0](https://github.com/JavaEden/Orchid/releases/tag/0.14.0) 在感恩节前后发行，一起标志着迄今为止最大的兰花发行！这包括:

*   Kotlin 1.3 更新
*   Wikis 现在可以为离线文档生成 pdf。在这里见文档[。](https://orchid.netlify.com/plugins/OrchidWiki#offline-documentation)
*   新的资产管理器缩短了构建时间，并智能地渲染您的图像资产。它甚至包括自动缩略图生成的基本图像操作！点击了解更多关于媒体管理的信息[。](https://orchid.netlify.com/wiki/user-manual/content-management/media)

# [](#whos-using-orchid)谁在用兰花？

Orchid 现在为两个不可思议的开源项目的文档提供了动力，去看看吧！

*   Pebble 是兰花使用的主要模板引擎， [pebbletemplates.io](https://pebbletemplates.io/) 于 11 月 11 日正式上线。它使用的一些最显著的兰花特征是:
    *   [社论](https://orchid.netlify.com/themes/OrchidEditorial)主题
    *   [变更日志](https://orchid.netlify.com/plugins/OrchidChangelog)和[自动化 Github 发布](https://orchid.netlify.com/wiki/user-manual/publication/github-releases)
    *   [Wiki](https://orchid.netlify.com/plugins/OrchidWiki) 离线文档
    *   [Javadocs](https://orchid.netlify.com/plugins/OrchidJavadoc) Java 源代码文档
    *   部署到 [Github 页面](https://orchid.netlify.com/wiki/user-manual/publication/github-pages)
*   Strikt 是 Kotlin 的一个新断言库，strikt.io 于 10 月 19 日上线，使用了以下兰花特性:
    *   一个完全自定义的主题，包括自定义模板标签
    *   [变更日志](https://orchid.netlify.com/plugins/OrchidChangelog)
    *   [维基](https://orchid.netlify.com/plugins/OrchidWiki)
    *   Kotlindocs Kotlin 源代码文档
    *   部署到 [Github 页面](https://orchid.netlify.com/wiki/user-manual/publication/github-pages)

此外，我在[caseyjbrooks.com](https://www.caseyjbrooks.com/)的个人网站进行了重大改版。它之前使用了[future implement](https://orchid.netlify.com/themes/OrchidFutureImperfect)主题，但我一直在努力创建一个基于布尔玛的定制主题，并从 [@aspittel](https://dev.to/aspittel) 的[个人资料建议线程](https://dev.to/aspittel/portfolio-advice-thread-56g9)中获得了一些非常有用的建议。当主题完成后，它将被合并到主兰花回购，所以你也可以使用它为您的兰花网站！
