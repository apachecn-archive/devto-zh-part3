# 您希望在您的开发人员个人资料中看到什么？

> 原文:[https://dev . to/aspittel/what-you-like-to-see-on-your-dev-profile-31md](https://dev.to/aspittel/what-would-you-like-to-see-on-your-dev-profile-31md)

当你进入 DEV 上的个人资料页面时，比如说，mine 上有很多东西:你的特色 GitHub 库、社交链接、你的帖子和你最近的评论。

个人资料有两个方面:你想在自己的个人资料上展示什么，以及你想在别人的个人资料上看到什么。

**对于您的个人资料**:您最理想的用途是什么？你想把它寄给未来的雇主吗？把它作为你博客文章的作品集？或者，作为自己在网络上的登陆页？

**查看其他人的个人资料**:你访问他们的个人资料是为了什么？你是雇主吗？或者，有兴趣了解某个作者吗？

在每种情况下，该档案中有哪些信息会有所帮助？在这些事情中，哪一件对你来说最重要？

大胆思考，勇于创新！提前感谢您的反馈和想法！😊
