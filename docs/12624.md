# Hello World (Dev.to)！是我，乔希。

> 原文：<https://dev.to/joshbruce/hello-world-devto-its-me-josh-190p>

这似乎是一个有趣的平台。挖掘旧的学校电脑界面。从字体选择到颜色。感觉我正盯着我爸爸大约 1984 年的旧 Mac 电脑；或者说，后来我们得到的后续 IBM。

自学:我在 1998 年建立了我的第一个网站。与其说我想成为一名开发人员，不如说我想成为一名用户体验人员，因此我的思维模式是“人向下而不是金属向上”。不幸的是，UX 的概念并没有真正流行起来，我没有语言，也不知道去哪里寻找答案。我也没有办法付钱给别人来实现我的愿景(或实验);所以，我学了开发。

有人批评我的 HTML 没有以前那么好；因此，我将自己进一步投入到发展的土地上，同时仍然跟上人类的心理以及我们如何与笨重的计算器互动。我记得我第一次有作品集，在一家小商店申请 UX 的职位时，老板说，“你看起来更像一个开发者。”

所以，我做了几年自由作家。我发现，如果我能做每个人都认为很难而且他们做不到的事情，那么我就能继续做我更感兴趣的事情。因此，软件开发成了我的敲门砖，也是我做其他事情的一种方式。

将近 20 年后，我开始将编码问题视为 UX 问题。在这种情况下，用户是下一个开发人员。当然，这使得我对某些事情的看法更加严格，但也减轻了我被贴上“开发者”标签的感觉。:)
