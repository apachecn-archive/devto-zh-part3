# 远离谷歌

> 原文：<https://dev.to/m1guelpf/moving-away-from-google-5c05>

[https://app.blogcast.host/embed/293](https://app.blogcast.host/embed/293)

出于对谷歌的垄断和隐私保护方式的担忧，我决定采取一些措施来解除谷歌对我生活的封锁。以下是谷歌产品的一些替代品:

## 浏览器

这是最容易的部分，但也是最困难的部分之一。如果你像我一样一直使用 Chrome，那么切换到 Firefox 可能会非常困难。幸运的是，还有一个选择。 [Ungoogled Chromium](https://github.com/Eloston/ungoogled-chromium) 是 chrome 的一个分支，从 Chrome 中移除了所有谷歌的东西。它有一个缺点，你需要遵循一个特定的程序来安装新的扩展。

## Google+

对关闭声明的反应让我很惊讶。原来有人在用！不管怎样，如果你是这些人中的一员，我会建议你使用推特或者 T2 乳齿象。

## 谷歌搜索

使用[达克达克戈](https://duckduckgo.com)。如果那些“你的数据不应该被出售”的弹出窗口让你烦恼，使用 [DuckDuckGo StartPage](https://start.duckduckgo.com/) 。遗漏了一些结果？在你的查询前加上`!g`,你会得到和谷歌一样的结果。

## Google Drive，Docs，Sheets &幻灯片

对于驱动器，你可以使用 [Dropbox](https://dropbox.com) ，或者 [Nextcloud](https://nextcloud.com) ，如果你有一个服务器来托管它的话。我发现谷歌文档的最佳替代方案是[概念](https://notion.so)。对于幻灯片，您可以使用[slides.com](https://slides.com)，而[气动工作台](https://airtable.com/)是一个很好的替代选择。

## Gmail

又一个难题。我发现最好的选择是 Zoho Mail T1(它也允许免费定制域名)，但是你也可以试试 T2 的 proton Mail T3。

## 谷歌分析

我在我所有的网站上使用 [Fathom](https://usefathom.com) 已经有一段时间了，我很喜欢它，尽管我也听到了关于 [SimpleAnalytics](https://simpleanalytics.io/) 的好消息。

## 其他

Pieter Levels 使[成为一个网站](https://nomoregoogle.com/),列出所有谷歌产品的替代品。
