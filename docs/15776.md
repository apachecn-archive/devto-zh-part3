# 面向 JavaScript 开发人员的 DevReach

> 原文:[https://dev . to/progress telerik/dev reach-for-JavaScript-developers-2pec](https://dev.to/progresstelerik/devreach-for-javascript-developers-2pec)

JavaScript 继续主宰着开发者的世界。DevReach 精选了一些来自世界各地的最佳演讲者-与 JS 分享现在和未来。

JavaScript 正在吞噬世界。它无处不在——在网络、桌面和手机上。它是当今最广泛使用的框架、库和平台的首选语言。每天都被数百万开发者盯上。它可以用来建造几乎任何东西。因此，对于开发人员来说，不仅要学习 JavaScript，还要有效地使用它。

参加 11 月 13 日至 14 日在美丽的保加利亚索菲亚举行的中欧和东欧顶级开发者大会。2018 年是 DevReach 成立 10 周年，我们不遗余力地让它成为开发者难忘的体验。

参加 DevReach 的开发人员受到了一群国际专家/演讲者的款待——这些人对 JavaScript 略知一二。这篇文章概括了 DevReach 上所有深入研究 JS 生态系统各种细微差别的分组会议。

> 所有会议都被记录在全高清-现在你可以重温他们！

## [](#cherish-the-memories)珍惜回忆

JavaScript 生态系统变化很快。今天，开发人员可以在 web/桌面/移动设备上获得美妙的体验，而工具的成熟只会有助于开发体验。丰富的框架提供了特性的完整性，知道如何正确使用 JS 的开发人员可以统治软件世界。

DevReach 很自豪地提供了来自世界各地一些最优秀的演讲者的 JS 内容。所有的分组会议都是全高清录制的。需要查找某个演讲者提到的东西吗？还是想坐下来喝杯咖啡学习？我们希望您喜欢录制的 DevReach 会议。
