# 在 AnyChart Qlik 数据可视化扩展中引入自定义主题上的自定义绘制

> 原文：<https://dev.to/andreykh1985/introducing-custom-drawing-over-custom-theme-in-anychart-qlik-data-visualization-extension-4b8o>

AnyChart 在其 [JavaScript 图表库](https://www.anychart.com)的 [Qlik 扩展](https://www.anychart.com/technical-integrations/samples/qlik-charts/overview/)中添加了一个新的数据可视化功能，现在允许 Qlik Sense 用户为他们的图表设置自定义渲染器，只需在需要时在自定义主题上应用自定义绘制即可。

阅读对这一新功能的介绍，了解其工作原理，并在 AnyChart 博客上查看其在交互式仪表以及饼状图和柱形图[中的实现示例。](https://www.anychart.com/blog/2019/01/16/qlik-data-visualization-custom-drawing/)
