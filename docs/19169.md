# 我使用 YouTube 的第一步

> 原文:[https://dev.to/bakedbird/my-first-steps-with-youtube-1a37](https://dev.to/bakedbird/my-first-steps-with-youtube-1a37)

这一切都是从大约一个月前我买了一台新相机开始的...

我是一些伟大的程序员的超级粉丝，我正在考虑开办我自己的频道，因为 1)这种频道不多，2)为什么不呢？

老实说，我不得不承认一开始我很害羞，不知道该说什么或如何开始，但随着 12 月的临近，我想我可以从 YouTube 上这种叫做 Vlogmas 的流行视频开始。我从我的妻子那里了解到这些，我每天花几个小时和她一起看 YouTube 视频，我注意到没有其他程序员 YouTube 做这种视频！我不知道为什么，但我看到了一个差距，并考虑开始！

这一切都是关于编程的，所以我认为 Vlogmas 不是一个好名字，因为它是由时尚和化妆 vloggers 标记的。所以我想了想，决定把它们命名为 Codemas！:D

我不知道这次尝试是否会成功，但我肯定会尝试一下！不过，我真的希望这对其他程序员来说是一个很好的推动，让他们开始制作 Codemas 视频。毕竟，这是一个更好地了解他们/彼此的好机会。

下面是我到目前为止制作的前两个视频。如果你看了，我真的很想得到一些反馈！这是一个开始，所以，我想一些方向！

此外，欢迎为接下来的视频提出想法。你会订阅吗？你想看什么？

我真的很享受这个全新的开始，这是一个美好的未开发的领域，我希望你能和我一起探索！
