# 你做什么来练习新的编程语言和/或框架？

> 原文:[https://dev . to/leoat 12/what-do-you-do-to-practice-new-programming-languages-andor-frameworks-1122](https://dev.to/leoat12/what-do-you-do-to-practice-new-programming-languages-andor-frameworks-1122)

这个问题我已经纠结了一段时间了。我很好奇，我喜欢尝试新事物，不管是语言还是框架，但是有一些编码的想法是很重要的。仅仅做教程对我来说是不够的，它通常太容易了，所以我需要一些更真实的东西来练习。然而，我对项目没有太多的创造力。我试着去思考一些我可能会遇到的问题，并想把这些问题自动化，或者思考一些与我的爱好相关的问题，这些问题很容易被创造出来。我确实有一些项目在慢慢发展，但是因为它们稍微复杂一点，我倾向于使用我已经相当熟悉的东西，所以不算。我也试着在 GitHub 上寻找我可以贡献的开源项目，但是保存良好的项目对于想练习基础的人来说通常太难了，这很吓人。

所以，问题是，当你想实践一些东西，但没有一个项目的想法去尝试时，该怎么办？有网站可以帮助你？前端、后端或两者都有。你在这种情况下会怎么做？
