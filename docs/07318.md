# 深度学习是死路一条吗？

> 原文:[https://dev.to/cheetah100/is-deep-learning-a-dead-end-2i0l](https://dev.to/cheetah100/is-deep-learning-a-dead-end-2i0l)

虽然深度学习使机器学习成为可能，但它需要大量的数据集和大量的计算资源进行适当的规范化，才能成功地训练它们。相比之下，你的狗只看你藏了一次，就能知道你把他的狗食藏在哪里。

这是我们在机器智能方面尚未跨越的鸿沟。虽然反向传播算法让我们摆脱了起跑线，数据集和原始计算资源的数量让我们能够进行强力训练，但它远远无法接近自然大脑的学习性能。

通过经验或观察来学习并不是人类独有的。章鱼拥有独立于人类发展的大脑，能够观察其他章鱼的行为并模仿它们。在看到另一只章鱼摘下帽子去抓螃蟹后，它们可以重复同样的动作来获得自己的奖励。人类和章鱼大脑的进化融合给了我们一条理解自然学习神经网络的重要方面的途径。

人类和章鱼都没有庞大的数据中心来运行反向传播算法，以确定神经元之间的权重。自然大脑中的重量训练是通过反馈路径进行的。自然大脑的运作基于离散的电压尖峰，它触发突触间的神经递质。权重根据输入尖峰的时间来修改，这反过来修改了突触化学。人工神经网络使用反向传播来模拟尖峰的概率，并修改中间神经元连接的权重。

在自然大脑中，训练的反馈连接是网络的一部分。另一方面，人工神经网络只有前馈连接。增加反馈连接会产生环路，使反向传播算法难以处理。当前的深度学习指定了一组特权输出神经元。参考输出神经元训练网络，反向传播算法修改中间连接权重，使得输入神经元将触发正确的输出神经元。

天生的大脑没有特权神经元。权重仅通过经由直接连接的交互来确定，而不是根据一些远程特权神经元来修改权重。

尽管自然大脑与机器大脑相比有缺点，与每秒运算数十亿次的机器相比，每秒仅运算数百次，但在学习性能方面，自然大脑仍然远远优于机器大脑。

虽然神经网络显然是机器智能的前进方向，但如果我们要接近自然大脑的学习效率，我们需要从根本上改变它们的训练方式。

自 2012 年以来，我们看到的机器学习的快速进展证实了我最初的估计，即我们离真正的通用机器智能有多近。随着对机器智能投入的资源和主要技术公司对它的关注，我相信发展的步伐只会加快，在短期内导致通用机器智能。

但这种明显的进步是欺骗性的，它是基于一种机器学习算法的成功，这种算法有效，但非常糟糕。现在几乎有一种正统观点认为，机器学习需要大量的结构化数据和巨大的计算资源。有多少人停下来问为什么相对脆弱的自然大脑学习能力更强？

为了实现机器学习系统的实质性改进，我预计我们将需要遵循自然规律，使用反馈和局部神经连接来训练它们，而不是目前流行的算法方法。反向传播方法的成功可能会导致许多机器学习社区徒劳无功。如果投入反向传播方法来训练神经网络的大量资源没有取得成果，我们可能会看到另一个人工智能冬天。

另一方面，已经有许多成功的应用被利用来从深度学习的背后赚钱。人工智能现在面临如此多的风险，很难想象科技公司会放弃。

在研究更接近自然大脑的方法方面继续取得进展。神经科学家正试图以更高的分辨率对工作中的大脑进行成像，以更好地了解其结构。计算机科学家正在试验更接近自然大脑工作方式的脉冲神经网络。

我对机器实现一般智能的可能性的乐观估计，已经被对当前基于深度学习的系统的局限性的新理解所缓和。会有新的应用可能会造成重大的社会影响。但我相信，在我们达到一般智力之前，我们需要从根本上提高学习成绩，我不再确信这将在我最初认为的短时间内实现。
