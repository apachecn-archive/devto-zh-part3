[![puritanic profile image](../Images/72ceb5379632ab0f47609b397150f57d.png) ](/puritanic) [ Darkø Tasevski ](/puritanic) • [<time datetime="2018-11-24T17:18:57Z" class="date-short-year"> Nov 24 '18 </time> • Edited on <time datetime="2018-11-24T17:35:34Z" class="hidden m:inline-block date-no-year">Nov 24</time>](https://dev.to/puritanic/comment/752c) 

其中最讨厌的就是 web 开发很容易，每个人都可以创建一个网站。当然，也许我的祖父可以创建一个网站，但是一个网站需要很多因素，包括性能和可访问性。还有数不清的框架，这个领域正在以一周为单位向前发展，并且总是有新的东西需要学习，这样你才能跟上时代。所以你应该准备每周抽出一些时间来学习新东西。除此之外，对于 web 开发，你只需要有扎实的 Javascript 和 CSS 知识，就目前而言，你不需要知道十种编程语言就可以开始找前端工作。

另一个常见的误解是，当你完成了网站，你的工作就完成了🙃如果我们谈论的是生产应用程序，这是远远不正确的。

要成为一名优秀的网站开发者，你不需要成为数学专家。这对于数据科学和机器学习来说是强制性的，但是在 web 开发中，你只会在极少数情况下使用你的数学技能。作为一名网站开发人员，你不需要有计算机科学学位，这是数据科学和 ML 的必修课(尽管这里也有例外)。我是自学的(没有学位)，我已经做了近 3 年的前端开发工作😎
