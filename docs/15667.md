# 当想法不流动时该怎么办？

> 原文：<https://dev.to/imronlearning/how-to-unblock-creativity-block-338j>

当我们面临创造力障碍时，这从来都不容易。取决于你对什么感兴趣或者你在做什么，是写文章、编码、制作视频等等。当你完全不知道要做什么的时候，做东西从来都不容易。我们尝试做不同的事情，只是为了让创造力流动起来，但有时它从来没有“点击”,我们最终对自己感到沮丧。

作为创造者(程序员就是创造者)，我们喜欢让事情发生。我们喜欢用自己的双手制作东西，用自己的眼睛去看，并与他人分享。作为一名创造者，最棒的一点就是看到人们享受你制作的产品，甚至是你享受你为自己制作的产品！

所以我想打开这个讨论，这样就可以分享那些你面临创造力障碍的时候，以及你做了什么来摆脱那种情况，让创造力再次流动。
