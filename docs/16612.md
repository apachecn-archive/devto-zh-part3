## [](#books)📚书籍:

*   [干净代码:敏捷软件技术手册](https://www.amazon.com/Clean-Code-Handbook-Software-Craftsmanship/dp/0132350882/)
*   [头脑优先设计模式:一个对大脑友好的指南](https://www.amazon.com/Head-First-Design-Patterns-Brain-Friendly/dp/0596007124)
*   热情的程序员:在软件开发领域创造非凡的职业生涯
*   [设计模式:可重用面向对象软件的元素](https://www.amazon.com/Design-Patterns-Elements-Reusable-Object-Oriented/dp/0201633612)
*   [重构:改进现有代码的设计](https://www.amazon.com/Refactoring-Improving-Design-Existing-Code/dp/0201485672)
*   [软技能:软件开发人员的生活手册](https://www.amazon.com/Soft-Skills-software-developers-manual/dp/1617292397)

## [](#blogs-amp-sites)📌博客&网站:

*   《粉碎杂志》:针对网页设计者和开发者的教程、观点、终极指南、案例研究。
*   [SitePoint](https://www.sitepoint.com/) :了解 HTML5，CSS3，JavaScript，PHP，响应式网页设计的教程，课程和书籍。
*   [HackerNoon](https://hackernoon.com/) :来自真正的科技专业人士(约 7000+投稿作者)的关于软件开发、AI、区块链的文章。
*   FreeCodeCamp:值得一读的来自我们开源社区的关于编程和技术的故事。
*   CSS-技巧:使用 CSS 的技巧、诀窍和技术。
*   [源码制作](https://sourcemaking.com/):设计模式、反模式、重构文章和指南。
*   Thoughbot:一个关于设计、网络、iOS、Android 的很棒的博客。Thoughbot 的一个著名产品是 up case——现在是免费的。
*   [网飞博客](https://medium.com/netflix-techblog):了解网飞世界级的工程成果、公司文化、产品开发等。
*   [ScaleScale.com](https://www.scalescale.com/):分享关于教程，关于 DevOps(Nginx，AWS)的小技巧。
*   nixCraft : Linux 技巧、黑客、教程和想法。
*   马丁·福勒(Martin Fowler):马丁·福勒是一名作家、演说家，也是企业软件设计的代言人，为 ThoughtWorks 工作。这个博客关注帮助你在职业生涯中提高软件开发职业的技能和技术。
*   软件方面的乔尔:乔尔·斯波尔斯基是 StackOverflow 的首席执行官(也是联合创始人)。分享软件开发、管理、商业和互联网的博客作者。
*   [编码恐怖](https://blog.codinghorror.com/):这篇博客的作者是杰夫·阿特伍德(Jeff Atwood)，他和乔尔·斯波尔斯基(Joel Spolsky)是 StackOverflow 的联合创始人。博客分享了许多关于软件开发的高质量文章和讨论。
*   GeeksforGeeks :极客的计算机科学门户。它包含了写得很好的、经过深思熟虑的、解释清楚的计算机科学和编程文章、测验和练习/竞争性编程/公司面试问题。
*   [GoalKicker.com](https://goalkicker.com/):免费的 C、C#、CSS、HTML5、iOS 开发、Android 开发、Java、JavaScript、PHP、Python、Ruby、SQL Server 等编程书籍。

最后，保持并发展你的阅读习惯，因为它不仅能磨砺你的技能，让你的思维更深入，还能帮助你在职业道路上更上一层楼。

如果你知道任何适合开发人员的好书或博客，请不要犹豫，在评论中与我和所有人分享。

感谢您的阅读！
