# 冥想、思想和神经系统

现代人类面临的一个难题是，我们有一个古老的神经系统，在许多方面，它不太适应我们今天生活的环境。

因为大脑要处理的信息来源比以往任何时候都多，所以现代人的大脑处于超速运转状态。

几乎每一个我教冥想的人都觉得他们的大脑比一般的大脑更忙碌！这是一种近乎普遍的感觉。

其中一个重要原因是——我们生活的环境已经迅速改变，与我们思维进化的环境大相径庭。我们的大脑仍在适应这些变化。

我们通过电子邮件、社交媒体、电视、工作等形式接收如此多的信息，以至于超出了我们处理这些信息的能力。感官输入将被“储存”在我们的大脑和身体中，以供以后处理。

我们的思维在加速，我们觉得我们不能清晰地思考，我们跟不上节奏或者看不到最清晰的前进方向。都是因为我们想得太多了。

大多数人在冥想中遇到的最大困难之一就是过度思考。认识到这一点很容易解决:

*   在冥想中，思想从来不是问题；
*   我们可以通过认识到它们只是“精神事件”来改变我们与思想的关系——在我们的意识中，我们可以根据需要给予更多或更少的关注。

随着时间的推移，头脑最终会“安定下来”——事实上，这是人们开始冥想的主要原因之一！

## 冥想如何帮助过度思考

**思维的本质**
你有没有考虑过我们为什么会有思想？作为人类，我们如此强烈地认同我们的思想，以至于可能在你的一生中都不会想为什么会这样！

思考是相当神奇的。我喜欢能够思考。但直到几年前，我才开始思考，为什么我会这样想，为什么有时会有点失控。

从进化生物学的角度来看，思考能力是一种优势，它让我们能够解决我们在世界上面临的问题。这反过来让我们能够确保作为一个个体以及作为一个物种的生存。

根据进化心理学，思维及其微妙之处是帮助我们在环境中生存和发展的适应性。你精神世界的每一个特征都是一种适应，在基本层面上与保持你的生存有关。

你的大脑做到这一点的最明显的方式是给你致命的解决问题的能力。你有超能力想办法出去(和进来！)棘手的情况。你可以利用大多数其他物种所不具备的抽象思维能力来做到这一点。

在过去，这可能很简单，就像找到一条安全的路穿过峡谷到达另一边的水或食物。

在更近的时代，我们已经用更复杂的方式运用我们的思考能力，例如:

*   进行允许飞机飞行的计算；
*   弄清楚如何最好地部署公司团队的资源；
*   规划重大建设项目；
*   试图理解自然和人体的复杂性；
*   开发支持我们最喜欢的网站的软件。

所有这些都促进了个体和物种的存在。

思考。确实是强大的东西！

如果运用得当，思考的能力对人类来说是一大益处。但有时我们计划、预测、计算和创造的能力会对我们不利。尤其是当我们觉得这就是我们所做的一切的时候！

**过度思考** 有时候，在一个环境中进化出来的特质在环境改变时会变得不那么有效。这就是所谓的适应不良。在某些情况下，思考的头脑会以一种不适应的方式行动。

当今世界有三个特征让我们的思维超速运转:

1.  我们面临的威胁在很大程度上是想象出来的。我们大多数人都过着安全的生活，食物、住所和水等问题大多都得到了解决。然而，我们的大脑仍然渴望将其解决问题的能力指向某些东西。它全神贯注于我们日常的困难，这些困难通常没有生命危险，并将其全部能力用于这些困难，导致过度思考；
2.  我们被比过去多得多的信息轰炸，所以我们的思维、解决问题的头脑(它喜欢信息，喜欢通过创造思想来处理信息)被过度刺激了。这种持续的刺激意味着我们可以放大我们问题的严重性，把日常情况变成压力和焦虑的来源。我们被过度的思考淹没了。一切都变成了需要解决的问题，或者需要实现的目标。这可能会很累；
3.  我们的生活方式不像我们的神经系统那样频繁地通过身体活动来放松。压力反应的过度激活导致紧张情绪在大脑和身体中积累，我们不给自己时间和空间来释放它，导致压力的长期积累。

思考就好。需要做多少就做多少，做好，做清楚。

另一方面，过度思考通常是压力的表现。它经常被体验为快速的想法、最坏的情况、完全活在你的头脑中，或者消极的自我对话(例如，我是一个糟糕的程序员——经常表现为冒名顶替综合症)。

通常，我们会用“应该”、“必须”或“不得不”来思考，给自己施加巨大的压力，让事情变得与现在不同。

一点都不好玩，而且毫无进展！

想象中的威胁和信息超载使我们容易过度思考。既然思考是为了解决问题而设计的，那就可以感觉一切都是问题！

进化来解决如何让我们生存的问题——帮助我们确保我们的安全——的头脑却什么也没做！事实上，它会让人觉得我们总是不安全——即使客观上我们是非常安全的！

沉思和过度思考
对生存的专注常常意味着我们的头脑被过去和未来所困扰。因为它与我们的生存息息相关，我们的大脑喜欢预测威胁将来自哪里。它还喜欢反思过去，分析事件，看看下次类似的事情发生时能从中吸取什么教训。

你可以很容易地通过观察自己的头脑来验证。花一点时间观察你现在在想什么。很有可能，你正在从事某种形式的对未来(预测)或过去(学习)的思考。

担忧、焦虑、恐惧、后悔、糟糕的记忆——最坏的情况下，这些都充斥着大脑。它们让人不舒服，让人分心，而且并不总是有用的。

标准冥想算法给了我们一种练习，帮助我们更多地与当下的体验联系起来。

如果我们面临着被过去和未来所困扰的问题，练习我们经验中被忽视的方面——现在到底发生了什么——是有意义的。

重要的是要强调，在练习标准冥想算法时，没有必要主动消除想法和思考。我们没有采取积极的、基于任务的方法来解决过度思考的问题，而是决定将注意力转移到其他地方，这让我们的思想缺乏燃料。

大多数人发现，在某种程度上，思想会自己安定下来，不需要自己采取任何行动，只需在冥想期间选择聚焦并重新聚焦于当下。
