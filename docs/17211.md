# 花了几年才实现的目标

当我开始我的软件开发职业生涯时，我不得不把家搬离我们居住的地方——我们的“家”——因为这里没有技术工作。

所以多年来，我一直试图进入一个偏远的位置。经过**年**的尝试，今年我终于成功了！

## [](#how-it-happened)事情是怎么发生的

在 2017 年和 2018 年，我决定专注于我的博客，并将有用的内容发布到不同的简讯等。以获得更多的关注。在得到一篇关于。NET 工程团队的博客我意识到其实是可以做到的！所以我只是继续前进！

我还用我在这个主题上做的一些研究彻底检查了我的职业简历。这使得他们得到了一份工作机会，可以在一家医疗行业的初创公司远程工作。

听起来很棒——但是从工作时间和创业带来的一切来看，这会给我的家庭带来更大的压力...

长话短说，这份工作与我之前工作的公司非常匹配(一个很好的工作场所)。我认为这是一笔更好的交易，因为我喜欢为这家公司工作。我们进行了一些坦诚的交谈，他们理解我在别处寻找工作的决定是基于远程工作的需要，他们也愿意提供便利。

所以，我和我的家人立刻搬回了我们的“家”,感觉棒极了！

以下是我学到的一些技巧，它们帮助我获得了一份远程工作，也可能帮助其他处于类似职位的人:

*   如果可能的话，把你的博客放到流行的时事通讯和网站上。然后在你简历的顶部强调这一点。换句话说，抛名😎
*   查看[栽培文化](https://cultivatedculture.com/)作为一个伟大的全面指导
*   努力学习你所在的专业领域的所有知识。这不容易，需要做很多工作

# [](#why-cant-net-be-as-easy-as-laravel)为什么不能。不会像拉弗尔那样容易吧？

我已经写了 Coravel 是如何实现的。

Coravel 成立初期的许多统计数据也可以在那篇文章中找到。

今天， **Coravel 在 GitHub 上已经接近 600 星了！**

对我来说，这太疯狂了。为他人创建一套有用且受欢迎的工具。NET 开发人员使用从来不是我曾经期望或试图做的事情。

Coravel 实际上只是解决我的一个问题的一个有趣的尝试。我只是想看看我能不能做些简单的东西来建造。NET 应用程序对我来说更容易。

其他让我震惊的统计数据(全部在项目的前 6 个月内):

*   平均每天有 100 次点击回购(阅读文件)
*   它已经在微软的[上被特别报道。NET Foundation 单口相声第 150 期秀](https://youtu.be/XHofNIqEDFY?t=2325)
*   它还出现在微软的 ASP.NET 官方网站上
*   在 NuGet 上下载了 4000 多次

除了 Coravel，我还开始了我的第一个付费产品——[Cora vel Pro](https://www.pro.coravel.net/)。我知道我必须掌握技巧才能做好*这样的事情*，但实际上*做这件事*工作量很大。😓

但是-我做到了！当然是在一些好朋友的帮助和鼓励下。

## [](#some-takeaways)点外卖

在这方面，我能给别人一些建议吗？

### 解决你正在经历的合法问题，让世界知道你的问题和你创造的解决方法。

“修复”可能是软件。它可能是一本书。什么都有可能。

真的是这样。你可以在需要知道的基础上了解的其他一切(管理回购等)。)

### [](#get-involved-in-the-community)参与社团。

上推特。关注一些博客。开始评论。鼓励作者。

关注社区中你所敬仰的“男人/女人”。

你知道那些你所敬仰的人是人吗？你能与之互动的人？

在过去的一年里，通过简单地参与社区活动，我结交了一些好朋友。

一个特别的人是[builtwithdot.net](https://www.builtwithdot.net)-[科尔斯蒂安](https://twitter.com/fluxmatix)的创造者。我们以各种方式互相帮助，他是一个非常积极和令人鼓舞的朋友。

# 2019 年的计划？

我有很多可以做的“事情”的想法。书籍。Saas 产品。你说吧。

但是我有 8 个孩子。和一个妻子。我全职工作。我也兼职承包。

然而，照顾家庭的一部分是将时间投入到我的事业和职业中。

就这一点而言，我本身没有任何坚实的“目标”。

现在我想重点关注几个方面:

*   Coravel + Coravel Pro
*   为我的博客、个人品牌和电子邮件简讯找到一个特定的位置。不知道...)
*   努力变得更健康(我最近有一些健康恐慌！😨)

除了这些基本原则，我没有其他期望😎

我正计划浏览朱利安·夏皮罗的人生规划指南《做什么》。

我想那会 100%让我更专注。也许我会把我的结果写在博客上？
