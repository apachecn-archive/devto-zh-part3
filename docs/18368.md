# 发布支持 RAD Studio 10.3 Rio 的 Devart dbExpress 驱动程序

> 原文:[https://dev . to/devart software/devart-db express-drivers-with-support-for-rad-studio-103-Rio-released-4cc 6](https://dev.to/devartsoftware/devart-dbexpress-drivers-with-support-for-rad-studio-103-rio-released-4cc6)

Devart 是一家公认的数据库连接解决方案供应商，为 Embarcadero Delphi 和 Microsoft 提供服务。NET 开发平台，以及数据库开发和管理软件，发布了新版本的 dbExpress 驱动程序。dbExpress 驱动程序是简单的数据访问层，可以快速访问各种数据库。

**一次更新包括以下前沿功能:**
-RAD Studio 10.3 Rio 支持
-最新的服务器版本 18c 现在在用于 Oracle 的 dbExpress 驱动程序中得到支持
-现在用户可以在直接模式下使用操作系统身份验证
-新的驱动程序版本
支持 PostgreSQL 11 和 PostgreSQL 10.1 在用于 SQL Server 的 dbExpress 驱动程序中支持 Windows 身份验证和直接模式下的 MARS
-加载用于 64 位应用程序的默认客户端库是 对 InterBase 和 Firebird 的 dbExpress 驱动程序进行了显著改进
——自从 SQLite 的 dbExpress 驱动程序的最新更新以来，直接模式基于 SQLite 引擎版本 3.20.0

**dbExpress** 是一个独立于数据库的层，它定义了一个公共接口，用于在 Windows 和 macOS 上提供从 Delphi 和 C++Builder 到 SQL 数据库服务器的快速访问。对于每台受支持的服务器，dbExpress 都提供了一个驱动程序作为独立的库，它实现了用于处理查询和存储过程的通用 dbExpress 接口。

要了解有关 dbExpress 驱动程序新版本的更多信息，请访问[https://blog . devart . com/meet-db express-drivers-with-support-for-rad-studio-10-3 . html](https://blog.devart.com/meet-dbexpress-drivers-with-support-for-rad-studio-10-3.html)
