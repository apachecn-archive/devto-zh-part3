# 我如何用戈丁的桶来实现长期目标

我开始了一个新的实验，让我的长期进步更加明显，我称之为戈丁的水桶(见下面的引文)。如果你想跳过背景故事，直接进入食谱，[点击这里](#godin's-buckets)。

# 隐形水桶

> “装满一桶水可能不会很快或很容易，但你可以很容易地衡量你的进展。[...]然而，大多数对我们来说重要的东西并不会以这种方式表现出来。”
> [这不是水桶——塞思·戈丁](https://seths.blog/2018/11/its-not-a-bucket/)

但是如果我们可以呢？

我成年后总是想知道我十几岁的孩子怎么会花无数的时间在游戏上，例如在《我的世界》花太多的时间，而且几乎从不厌倦砍树或做一些无意义的事情。

我试图总结出长期坚持的机制是如何运作的:

1.  我想建造一些东西/到达一些地方/改进一些资源收集；
2.  在我的脑海里，我粗略地收集了我需要的资源清单；
3.  有了游戏工具，我开始收集资源，一个一个的；
4.  我可以看到直接的进展，我的工作证明就在我眼前，我砍的每一棵树，我开采的每一块石头，或者放在某个地方的每一块石头；
5.  每当我有点无聊的时候，我可以后退一步，想象周围的情景；有一个大画面，每个行动如何影响整体；
6.  如果环境改变了，或者我需要做别的事情，因为我已经做了足够多的事情，我可以很容易地改变任务，下次我路过的时候，我会看到还有多少工作要做。

这似乎是我可以在游戏之外使用的东西，我拿着大量的水桶，一直装满它们，却从来没有对结束有一个清晰的愿景，并且有一个公式让我自己坚持更长时间。

# 真正的木桶🚰

我的第一个想法是找一个[你需要一个预算](https://www.youneedabudget.com)的时间。失败。

除了[一些准备实施的笔记](https://gingkoapp.com/app-timebudget)(如果你实施了这个，我很感兴趣😬).

出于某种随机的原因，我当时在测试第二个计时，以了解我所有的时间都去了哪里，没有太多的预先考虑。

但我最初在 Timing 2 上没有太注意的是，它有一个项目/任务功能，我错误地认为这是一个自由职业者/小时计价功能。原来我可以用它来做戈丁的水桶。

从我沉迷的游戏习惯中，我提取了这个列表，并把它们应用到我的生活中:

1.  我对自己想要去的方向有一个模糊的认识；
2.  我有一个计划，但不完整，我不知道确切的数字，只是我需要做什么；
3.  我使用一个工具来执行手头的下一个明显的任务；
4.  我可以看到直接的进展，我的工作证明就在我眼前；
5.  每次我退后一步，我可以观想周围，看到大画面，我的任务的每一次执行如何影响整体；
6.  如果我需要做别的事情，因为我已经做了足够多的当前的事情，我会继续前进，也许，如果愿景仍然是最新的，我会继续下去。

# 戈丁的木桶

1.  保留一份你想做的项目的简单清单；
2.  为这些项目中的每一项保留一份明智的待办事项清单。我把我的放在[到](https://todoist.com)；
3.  相信你自己。如果你不认真对待自己的职业，你就无法继续前进。你确实用你的工具带来灵感，并完成工作。
4.  从这种信任中，你可以暗示任务**的即时进度与花费的时间**直接对应(对我来说是游戏规则的改变者)。无论你在我的生活中要做什么，你都不能欺骗这唯一的东西，你需要花时间在它上面。不管对你正在做的事情有什么其他的进步概念，接受任务上的时间意味着进步。
5.  为了后退并观想，你需要记录时间，否则你会看到虚空。不知何故，做任务，花费时间的行为，应该是保持记录的行为，这就是戈丁指出的隐形水桶的问题。所以，你需要的不是主动去做，而是被动为你做的事情，同样的，不管你是否在观察，时间都会流逝。我发现 Timing 2 已经很棒了，可以跟上这个任务了。
6.  其中最难的是，你需要从第一个项目的列表中创建项目，在时间 2 中，并在上面标记你在它们上花费的时间。我是这样做的，开始做一些有时间限制的项目，比如 25 分钟的番茄大战。比如这篇文章，去了“写作”项目/桶。因为我信任前面的项目，所以我可以在第二阶段进行一周回顾，并验证我在哪些事情上花了时间(取得了进展)，哪些事情停滞了，以及我是否需要继续做其他事情。我在第一个项目上写下的所有东西，因为我想做的项目会清楚地显示为继续进行或只是停滞不前。不再有隐形桶。

# 下一步

第一个是继续使用基本的跟踪和观察时间(进度)的走向。第二件事，我仍然怀念“你需要一个预算”的预算系统，做一些实际的时间预算。我的想法是，预算将避免和改善我在第三项中提出的花费时间和进度的联系。有些项目可以永远持续下去，耗尽无限的时间，却从来没有带来价值，你应该在这些项目上花费非常有限的时间，这样你就可以从更清晰的项目中带来进展。
