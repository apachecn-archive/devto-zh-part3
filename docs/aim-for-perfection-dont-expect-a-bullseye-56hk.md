# 追求完美，不要期望成为靶心

> 原文：<https://dev.to/imronlearning/aim-for-perfection-dont-expect-a-bullseye-56hk>

我是那种有计划的人。我喜欢事情完美的时候。我的东西整理得像图书馆一样。我所有的东西都摆放得很有策略。甚至我笔记本里的文件。我讨厌杂乱和混乱。

在我所有的项目中，我希望它是最好的。我觉得如果我全力以赴，如果我付出的不够，如果不是最好的，那也没关系。当涉及到团队项目时，当我的团队成员跟不上我说的话时，下一次我宁愿自己做。我变成了这样一个人，他宁愿自己做事情，并完全按照自己的想法去做，而不愿和那些有更大风险让想法变得不完美的人一起工作。它变得孤独。我压力非常大。我努力实现我的期望，却一次又一次地失败。

我开始重新评估我对完美主义的观点。我仔细观察了这个世界是如何围绕完美主义运转的。然后我突然想到。**没有什么是完美的**。

所以我不得不改变主意。我仍然希望事情能尽善尽美。我希望我的项目能做得最好。我想让自己变得优秀。但是怎么做呢？

我学到的四件事:

1.  力求完美，但不要期望结果完美。
2.  事情永远不会完美，接受它吧。
3.  不完美让你成长。
4.  追求完美迫使你变得比以前更好。

所以要追求完美，但不要期望击中靶心。
