# 什么构成前端开发？

> 原文：<https://dev.to/tusharprajapati/what-constitutes-front-end-development-2l4a>

[![Front-end development](img/0bd27af26e361c4620e2c9c8ae1bc4dd.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--BAw9iVTw--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/9fli4hkormfbhrk9pm0b.jpg)

前端开发是客户端脚本框架和标记语言以及其他尖端技术的结合，提供了一个用户可以体验内容、功能和布局的设计环境。

[前端开发](https://www.techavidus.com/blogs/front-end-web-development)的起源是优秀的前端开发人员，他们用自己的知识和技能弥合设计和实现之间的差异，并提供令人印象深刻的交互和用户体验。他们擅长定制用户体验，制作和修改响应性和面向搜索引擎优化的网站和网络应用程序，工具创建增强和严格的测试。

有一些特定于前端语言的框架，带有内置库、可共享代码和插件，为快速开发铺平了道路，如 Foundation、Node.js、express.js、Bootstrap 等。
