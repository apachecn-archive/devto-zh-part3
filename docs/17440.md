# 我意外出现在播客上的那个

> 原文:[https://dev . to/bencodezen/the-one-where-I-incidentially-end-up-on-a-podcast-3g F2](https://dev.to/bencodezen/the-one-where-i-accidentally-ended-up-on-a-podcast-3gf2)

## [](#prologue)开场白

我刚刚成为我的第一个播客的嘉宾:[我的 JavaScript 故事#082](https://devchat.tv/my-javascript-story/mjs-082-benjamin-hong/) 。我的神经非常紧张，甚至在我挂断电话后，我的声音仍然在颤抖。尽管我的一部分完全害怕这一集会很糟糕，但能够从我的开发遗愿清单中检查出来还是令人振奋的。

*   ✅做客 dev 播客
*   ⬜创造了一个收入超过 1000 星的回购
*   ⬜被邀请在一次会议上做主题发言人
*   ⬜和更多...

在我们结束录制后， [Charles](https://twitter.com/cmaxw) 提到我应该在新的播客上注册一个位置:[对 Vue 的看法](https://devchat.tv/views-on-vue/)。对于那些没有长期关注我的人来说，我已经在 Vue.js 社区做了很多工作，它看起来确实非常适合。因此，在兴奋之余，我去登记了 10 月份下午 2:30 的一个时段。

呜！年底前我会上两个播客？太好了。

## [](#a-few-weeks-later)几周后

当我查看电子邮件时，我看到了来自[dev chatv](https://devchat.tv/)的提醒，我的会议将在几周内举行，我应该准备好一个主题之类的东西。尽管这对我来说不应该是一个巨大的惊喜，但突然间我开始恐慌。出于某种原因，我的印象是，主题已经决定，我会尽我所能作为嘉宾小组成员。但是哇哦，我还要选题引集？

[![Octopus running away](../Images/51529fe71f98e32d1cc259b613a3567f.png)T2】](https://i.giphy.com/media/6h4z4b3v6XWxO/giphy.gif)

在这一点上，我内心的自我怀疑恶魔正在全力以赴:

*   “但我什么都不是专家！”
*   "我怎么能和这些我尊敬和尊敬的小组成员站在平等的立场上呢？"
*   “如果我说了一些愚蠢的话，整个社区都躲着我怎么办？!"

所以我对自己说，我只是还没有准备好，当我觉得自己“更有资格”的时候，我会安排另一个时间。就这样，我拒绝了日历上的邀请，继续我当时的工作。

## [](#the-day-of-reckoning)清算之日

现在是周三早上，我的时间表看起来很清楚。我在想今天会是安静的一天，我可以集中精力完成一些工作。对于那些不了解我的人来说，我对我的日历很认真，并把它作为我计划的唯一来源。所以你可以想象当我在下午 1:41 收到下面的邮件时，我有多震惊。

> 嗨，本杰明-
> 
> 除了约翰爸爸，我们所有人都在犹他州帕克城的框架峰会上。我们可能会晚几分钟加入，并且都将在一台计算机上进行记录。我们会看到它如何发展。当我们加入通话时，我希望您有点耐心。

[![Cat hiding](../Images/ac212c9b5597d4682761b4fd6093ee20.png)T2】](https://i.giphy.com/media/b0ebYAjHpFqZG/giphy.gif)

现在记住。电话原定于下午 2:30，所以我只有不到一个小时的时间来准备一个电话，我以为我完全是因为害怕而取消了这个电话。如果我说我不考虑简单地回复邮件通知他们我已经取消了，那我就是在撒谎。

然而，在这一点上，大脑的理性方面意识到了以下几点:

*   很多人已经在这一天留出了时间，所以取消将是一个混蛋的举动
*   尽管时间不多，但仍有时间准备
*   我的任何怀疑都是冒名顶替综合症的证据，我需要战胜自己

信不信由你，在不到一个小时的时间里，我做好了准备，我们最终录制了 Vues #037 上的[View:Vuex，VuePress，和 Benjamin Hong 的 Nuxt】！](https://devchat.tv/views-on-vue/vov-037-benjamin-hong/)

## [](#aftermath)余波

虽然这是绝对可怕的时刻，但我真的很感激取消没有通过。这并不是因为我在“Vues 的看法”上录制了一集，而是因为它让我从自我怀疑中震惊了。更重要的是，这又一次提醒人们，你头脑中的事情几乎总是比现实中更糟糕。

所以，如果你发现自己怀疑自己是否值得上播客，记住这个故事，知道**你也绝对值得上播客**。因为说到底，播客是一群人聚在一起谈论他们感兴趣的话题。只要你对正在讨论的话题感兴趣，这就是你所需要的。
