练习

1.  JavaScript 和 ECMAScript 有什么区别？ECMAScript 是一种标准。JavaScript 是该标准的一种实现。

2.  谁是布兰登·艾希？Brendan Eich 是 JavaScript 的创始人。

3.  如何隐藏和显示 Chrome 控制台？
    要打开控制台，在 Mac 上按`Option + Command + j`(在 Windows 和 Linux 上按`Control + Shift + J`)。另一种到达控制台的方法是右键单击网页，选择 Inspect，然后转到名为 console 的选项卡。要隐藏控制台，请按下控制台右上角的 X 图标。

4.  创建一个带有脚本标签的简单页面。在脚本标记中声明几个变量，然后将它们的值记录到控制台。
    在这里查看我的答案:[https://codepen.io/flosal-bccs/embed/mvPGgQ/?height=600&default-tab=result&embed-version=2](https://codepen.io/flosal-bccs/embed/mvPGgQ/?height=600&default-tab=result&embed-version=2)

5.  研究`prompt`和`confirm`——他们做什么？
    `prompt`显示一个带有可选信息的对话框，提示用户输入一些文本。`confirm`显示一个带有可选信息和两个按钮(确定和取消)的模式对话框。
