## [](#eli5-protein-structure-prediction-problem)ELI5 蛋白质结构预测问题

蛋白质是生命的基本组成部分。它们本身是凌乱的球状物体，很像不小心卷起来的鞋带。一根鞋带是一串氨基酸。氨基酸是对生命非常重要的更小的物质。有趣的是，氨基酸组在不同的条件下形成不同的形状。它们就像活的乐高积木，这些积木的某些组合往往会粘在一起，就像磁铁一样；几乎所有的人都是这样，这取决于他们是否能在水中放松自己。我们知道蛋白质可以做很多不同的工作，我们可以通过观察它的形状来判断蛋白质可以做什么类型的工作。对我们人类来说，蛋白质的形状就像一套工作服装:我们可以分辨出一个人是警察，因为那个人穿着警服。然而，它们是很多很多的蛋白质，所以生物和化学专家使用一种叫做“测序”的技术来记录它们:一个一个地记下每个氨基酸:想象一下神奇地拉直一根卷起来的鞋带，而不必担心打结。这项技术是如此之快，它允许我们记录大量的蛋白质，但同时我们失去了蛋白质的形状。结果，只有一小部分已知蛋白质的形状被绘制出来。仅根据伸直的鞋带找出卷起的形状，我们称之为“蛋白质结构预测问题”。

## [](#hold-your-breath-terminology-and-structural-physics-of-the-protein)屏住呼吸:蛋白质的术语和结构物理学

首先，我们将使用氨基酸的缩写 **AA** 。结合 AA 的*磁铁*被称为**肽键**。我们有时也说**构象**(AA 亚序列或整个链)，而不是形状。

这些键结合一个小的 AA 序列，不同的排序和不同类型的 AA 形成不同的局部形状。我们称这些局部形状为**二级结构** ( **SS** )，它们主要分为两类:$ \α$-螺旋和$ \β$-折叠。此外，SS 通常对于它们各自类型具有一定的长度范围。$ \α$-螺旋通常有 3 到 5 个氨基酸长，而$ \β$-折叠平均可以伸展到 7 个氨基酸长。

SS 的“串”的构象(**三级结构**)主要由链中所有 AA 的复合键力决定。在更小的尺度上，AA 的碳原子(＄C _ { \ alpha } $)充当枢轴点，链在其上旋转:每个 AA 的正负极性电荷相互作用(吸引和推动),并根据一系列旋转动力学定义“稳定”的构象。更“有趣”的是，给定相同的子序列，所述序列的一些已知构象不会在链的其他部分以相同的构象结束，因为该部分不与水相互作用。

总而言之，蛋白质的构象很重要，因为结构揭示了蛋白质的功能。很多人把蛋白质等同于机器，这个类比相当贴切:在分子层面上，一个蛋白质的工厂分解并组装基因，就好像它们是生产线上不同部分的工人。

不失一般性，严格的物理规则集描述了蛋白质的机制，因此原则上生命本身是可以理解的，如果我们能够获得动力学及其功能映射的全部知识。

## [](#methodology-a-tale-of-two-cites)方法论:双城记

显然，由于问题的复杂性，传统的结构预测方法涉及与已知结构的比较。假设是，进化已经做了足够多的实验，让我们可以说，如果一个稳定和工作的结构以前发生过(“发生”是被证明，而不是灭绝)，它应该在其他地方再次发生。事实上，比较法是有效的。然而，主要的问题是，我们没有足够的已知模板来解决其余的蛋白质序列。

对于几乎没有匹配模板的序列，我们不得不寻求几乎不做任何假设的解，这些解来自第一原理。这就是我们(计算机科学、计算生物学)称之为**从头**或**从头**的方法。

**从头**或**从头**是“从零开始”的拉丁文同义词。因为我们至少知道物理键的性质和约束，我们应该能够根据能量来计算(一个构象发生的可能性或最小能量)。

从早期手工制作的基于规则的模拟到最近的机器学习方法，**从头开始**方法继续获得牵引力。例如，使用双向递归神经网络的 SS 预测可以达到远远超过 80%的精度，并且每次迭代似乎都没有变慢，继续接近 90%精度的理论极限。

## [](#pipeline-for-ab-inito-or-de-novo-protein-structure-prediction)管道进行 **ab inito** 或**从头**蛋白质结构预测

机器学习是数据驱动的，更多有意义的数据总是比更少好。在机器学习方法中，我们不是将序列单独作为训练的输入，而是使用一系列预处理步骤来丰富它们。虽然我们没有模板来查找，但我们仍然可以从序列相似性中获得洞察力。在计算生物学中，我们可以通过基于相似性比对序列来获得更丰富的蛋白质图谱。蛋白质序列是氨基酸的编码序列。我们有 20 个已知的 AA，因此我们使用字母 A-T 对它们进行编码。相似性包括但不限于序列的字母相似性、亚序列 AA 替换表和由概率模型产生的相似性。通常，主要的对齐和搜索方法是 PSI-BLAST 和 HHblits。由此产生的更丰富的数据集然后被进一步编码并为训练做好准备。

与第一部分中的说明类似，我们设计了一个多阶段机器学习模型:首先预测 SS 和其他性质，使用预测的 SS 特征和原始轮廓来预测三级特征的距离/接触图，最终结合所有先前的信息来预测最终的折叠结构。
