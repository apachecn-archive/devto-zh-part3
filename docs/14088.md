# URL 的纯文本文件

> 原文：<https://dev.to/tedmasterweb/plain-text-file-of-urls-3e73>

在搜索了相当长的时间而一无所获之后，我决定将我自己的 URL 列表作为一个纯文本文件发布，每行一个 URL。

当我进行开发时，我有时需要测试数据的纯文本文件。我可以找到[Lipsum galory](https://www.lipsum.com/)、占位符[Bill Murry 的照片](http://www.fillmurray.com/)、[单词列表](https://github.com/first20hours/google-10000-english/blob/master/google-10000-english.txt)、[不同大小的纯文本文件](https://sample-videos.com/download-sample-text-file.php)，甚至还有一个地方[处理你的数据](https://devnull-as-a-service.com/code/)(是的，垃圾桶是一种服务)，但如果我能找到一个充满 URL 的单个[文本文件，我会被诅咒的。所以我制作了自己的 URL 纯文本文件](https://www.chicagoitsystems.com/wp-content/uploads/2019/02/URLS-200.zip)[。我希望你觉得这很有用🙂](https://www.chicagoitsystems.com/wp-content/uploads/2019/02/URLS-200.zip)

帖子[的 URL](https://www.chicagoitsystems.com/plain-text-file-of-urls/)的纯文本文件首先出现在[芝加哥 IT 系统有限公司](https://www.chicagoitsystems.com)上。
