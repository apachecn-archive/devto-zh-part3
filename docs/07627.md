<p>在这种情况下，延迟加载是指延迟加载某些内容，直到用户接近页面上的内容。如果他们从来没有滚动到内容，它永远不会被加载。</p> <p>loading 属性有三个可能的参数:</p> <ul> <li>惰性:惰性加载的候选对象。</li> <li>急切:马上装。</li> <li>auto:这相当于没有属性。</li> </ul> <p>你可能见过延迟加载库，它在加载整个图像之前显示图像的“预览”。从某些方面来说，这是一种非常有效的在网络上发布内容的方式，但是由于这是通过 JavaScript 完成的，所以这是一种不太好的方式。我从来不想在用户和基本的网络体验之间插入 JavaScript。</p> <p>这种新的属性让这种情况发生，而不会碍事。</p> <p>它将在 Chrome 75
