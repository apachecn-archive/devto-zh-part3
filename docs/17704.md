# dev.to 是最好的新话题网站

[https://TechCrunch . com/2018/12/23/the year-social-networks-was-no-longer-social/](https://techcrunch.com/2018/12/23/the-year-social-networks-were-no-longer-social/)

Tech Crunch 有一篇关于 FB 和大概所有非特定主题网站没有相同的感觉，通常不适合交流或社区建设的文章。

dev.to 是软件开发人员最好的新私人社区，我是在阅读这篇文章时第一次想到它的。我从来没有参与过作者所写的 phpBB 论坛的讨论或糟糕的用户界面。太多了。

感谢 dev.to 成为最好的新主题网站，并填补了一个急需的漏洞。
