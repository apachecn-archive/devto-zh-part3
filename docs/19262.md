# 编程语言如何在不到两分钟的时间内完成

> 原文:[https://dev . to/the Venice/how-programming-language-works-in-less-2 mins-2j5l](https://dev.to/thevenice/how-programming-language-works-in-less-than-2mins-2j5l)

嘿，我是 Prakash Pawar，今天我将试着向你解释编程语言是如何在计算机内部工作的。

计算机只理解机器语言代码中的指令。

[![](../Images/ccba126e8241851759fc34912ada84fe.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--VLPH13RE--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/qusb5yrzouqyqgwuidgl.jpg)

但是有一个问题，用机器语言代码写程序更难。所以我们必须用更高级的语言编写程序，比如 Java、C++、Ecmascript、Python 等等。
用更高级语言写的程序叫源代码。
但是我们不能在计算机中直接执行这些源代码。

[![](../Images/1ff2df272381bec0cbd468663427c504.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--x6otffbF--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/ryczomgnrsgizbkah86i.jpg)

所以我们必须把它转换成机器语言代码。我们需要一些特殊的翻译器，它们基本上是用机器语言代码编写的程序。而这些翻译者被称为语言处理者。

有三种语言处理器。

1.  编译器
    它用于高级语言。
    一次性读取并执行整个程序，如果出现错误，抛出错误。
    使用编译器的高级语言有:C，C++，C#，Pascal，Java 等等。

2.  汇编程序
    它用于汇编级语言(助记码)。
    它从给定输入中读取汇编级语言指令。

3.  解释器
    它用于高级语言。
    它逐行读取并执行源代码，如果出现错误，则在每行后抛出错误。
    使用解释器的高级语言有:Ecmascript、VB Script、Python、Perl 等等。
