有一个完整的框架列表，其中大部分是 php 框架，这些框架有助于快速应用程序开发。然而，这些框架中的一些确实需要入门，具有复杂的范例，有些对于高级 php 开发人员来说不够好。这就是 yuga 框架派上用场的地方。

在本教程中，我将向您介绍 yuga-framework，这是一个适用于初级和高级开发人员的 PHP 框架。宇迦框架可以用来开发 API 或者成熟的网络应用。

## 什么是宇迦框架？

宇迦是一个 php 框架，是快速 web 应用程序开发的理想选择，它可以用来开发 API 或前面提到的整个 web 应用程序。它支持所有 HTTP 动词或方法(GET、POST、PUT、DELETE、PATCH 和自定义方法)。它有自动的表单验证，自动的跨站点路由请求伪造保护，并且如果路由方法不是来自后台，它不会强迫用户使用路由方法。你可以用框架来学习和进步。

### yuga-framework 入门所需步骤

#### 第一步:安装作曲家

从[这里](https://getcomposer.org/download/)安装 composer (php 包管理器),如果你的机器上还没有的话

#### 步骤 2:使用 composer 从 Packagist 安装宇迦框架

要开始，你只需要一个命令

```
composer create-project yuga/yuga Blog 
```

#### 第三步:运行应用程序

要使用 composer 运行我们刚刚创建的应用程序，我们 **cd** 到*博客*并在终端或控制台中打开项目，键入下面的 **yuga 命令**，哦，我忘记提到框架自带控制台应用程序了吗，我刚刚提到了。

```
php yuga start 
```

#### 第四步:查看输出

要查看 yuga start 命令做了什么，请进入您的浏览器并访问 [http://localhost:8000](http://localhost:8000)
