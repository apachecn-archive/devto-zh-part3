# 网络开发

> 原文:[https://dev.to/lucasalt/web-dev-24m8](https://dev.to/lucasalt/web-dev-24m8)

我的博客上的法语原文，由 [Deepl](https://www.deepl.com/translator) 翻译

由谷歌开发并完全开源的, [web.dev](https://web.dev) 让你发现如何优化你的网站，使它们更快、更友好。你也可以审核你的网站，以了解他们不执行。

为了改进您的网站， [web.dev](https://web.dev) 包含了六个不同类别的教程和技巧:

*   **快速加载时间:**一系列讲解如何提高你的站点加载速度的教程，
*   **网络弹性:**一系列教程解释如何在全球范围内提供相同速度的服务，
*   **安全可靠:**解释了如何简单地保护您的站点以防止数据被盗，
*   **容易被发现:**小教程解释如何提高你的网站的 SEO，
*   可安装:一个简短的教程，解释如何创建一个 PWA(渐进式网络应用程序)
*   所有人都可以访问:一系列教程解释了如何让每个人都可以访问你的网站(颜色对比，文本大小等。).

审计结果分为四大类:

*   **性能:**让你知道需要修改什么来加速你的站点的加载，比如调用更多不必要的 CSS。
*   **可访问性:**给你如何优化网页的建议。例如，您的图像缺少 alt 属性，或者颜色没有很高的对比度。
*   **良好做法:**让你知道自己是否遵守了某些基本规则，比如图像的正确尺寸(无拉伸图像)。
*   SEO: 给你建议，让你的网站在搜索引擎上被更好地引用。

目前，我的网站做得还不错。剩下的就是提高可访问性和优化一些文件，一切都会好的😉。
