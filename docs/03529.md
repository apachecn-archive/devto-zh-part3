# [](#introduction)简介

我曾经看过很多文章或教程，都是关于为什么选择 X 而不是 Y 作为开发人员的开始。

甚至还有一大堆你必须学会的技术路线图咳咳...成为一名开发者。

我发现它经常令人望而生畏，而且没有涵盖求职问题的核心。

也就是你自己，就你喜欢什么、不喜欢什么、组织偏好、社会使命、技术、行业和一堆其他非技术性的东西而言。

有了自知之明，你就能更轻松地找工作，努力工作，产生影响。

以下是我曾经用过的一些建议，也是我的导师在帮助我引导我寻找自我时给我的一些建议。

# [](#odyssey-planning)奥德赛策划

[奥德赛规划](https://youtu.be/wnU5DaIPr2Q)是我从一本名为《[设计你的生活:如何打造美好快乐的生活](https://www.amazon.com/Designing-Your-Life-Well-Lived-Joyful/dp/1101875321)》的书中读到的术语。

这基本上是一个头脑风暴会议，当你挥动魔棒时，T2 时间和 T4 金钱将不再是问题，它将帮助你在 5 - 10 年内描绘出 3 条截然不同的人生道路。

这对我很有帮助，当我即将大学毕业时，我正在思考我人生的“下一步是什么”。

在那段时间里，我有一个模糊的想法，想成为新加坡科技创业生态系统的一部分，成为一名科技销售员、软件开发员或创业创始人。

# [](#create-an-event-life-line)创建事件生命线

我的导师教我创建一条**生命线**，它记录了总共 **10** 个难忘的**高点**或**低点**点。

这样你就能更好地理解和发现你用来克服挑战的技能/能力，从而成为现在的你。

这将有助于你在找工作时发现潜在的技能或能力，这些技能或能力是在对你可能从事的行业或职业感兴趣的情况下开发的。

[![my lifeline](../Images/8533c2927680148dd887d31513dd1fcf.png)T2】](//images.ctfassets.net/ly2f59p4unnn/3bmXd6ZK2B8joAIER4sy7F/4f55cc37f178292a6772bbbb5d8f9aec/my_lifeline.png)

一旦你创建了你的生命线，你就可以继续把它填在一个表格里，这个表格包括了你过去经历过的每一件值得纪念的事情的下列问题:

*   事件-事件的摘要描述
*   高/低——这是人生的高潮还是低谷
*   使用的兴趣/技能/能力——你使用的**技能**是什么，你的**意图/兴趣**是什么。
*   采取的行动-你采取了哪些行动来克服这一事件。

# [](#whats-your-unique-selling-point)你的独特卖点是什么？

在销售和市场营销中，总有这样一句话“如果每个人都是你的客户，那么没有人是你的客户”。

这意味着你应该始终专注于吸引或寻找特定类型的雇主/客户，让你在拥挤的市场中脱颖而出。

这样你就可以吸引或让你的朋友、同事、家人为你推荐，因为他们知道你在做什么。

以我为例，我把自己定位为喜欢为初创公司工作的 Django (Python Web Framework)开发人员。

因此，我的重点是利用我的 Django 技能的创业空间。

请注意，技术技能只是等式的一部分，它可能是你的软技能或个人品牌。

可能你是一个**超级接头人**，可以组建一个复仇者团队，在一个月内完成一个项目。

也可能是因为你以前在某个行业的经验，你可以创建软件或服务来解决该行业的一个痛点。

# [](#conclusion)结论

我希望这篇文章可以帮助你寻找开发人员的工作或者进入这个行业。

通过对自己有自知之明，你有时可以发现需要你特殊能力的新工作。

这也可以让你**将你已经获得的现有技能和经验**重新包装成以前不存在或不在你权限内的新角色。

一旦你做了以上任何一件事，和你的朋友或同事一起喝茶或喝咖啡。他们可以在你计划的失误中帮助你，或者在有效利用你的时间中找到捷径。

我强烈推荐任何人去“奥德赛计划”栏目，通过这三个计划，你可以一窥自己的生活。
