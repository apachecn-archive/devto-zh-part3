# [](#prepare-to-travel)👉🏻准备旅行

你的演示可能在走廊尽头的办公室里。也可能在世界的另一边！准备你的演讲已经够紧张的了，还没有额外的开销来确保你拥有完成演讲所需的一切。在这篇文章中，我将与你分享我曾经用来随时随地“准备好演讲”的技巧和方法。我希望你觉得有用，并期待你的反馈和建议！

## [](#computer)💻计算机

这个很明显:别忘了你的电脑！这应该是你练习演示的那台电脑。一些会议提供笔记本电脑，并提供加载您的演示文稿。如果是这样的话，一定要早点到，这样你就可以接触到机器并确认一切正常。你可能会惊讶于一些简单的事情，比如没有相同的字体，会使一个演示失去效果！

一些演讲者喜欢带一台备用笔记本电脑。这是他们在主机器出现故障时可以求助的第二台机器(这不是时*的问题，而是*时*的问题)。我冒着只有一台机器的风险，只是因为我的第二台电脑是一个巨大的游戏机，如果我试图拖着它到处跑，会导致颈部受伤。*

> ⭐ **提示**:我把所有东西都保存到了云端。我的 deck 在一个云存储设备上，我的代码被签入 GitHub。如果我的主要机器出现故障，我只需进行几次拉动和安装，就可以构建一个可供演示的新环境。我不能过分强调这有多重要。我使用支持版本控制的云存储选项。我曾经遇到过 PowerPoint 演示文稿被破坏的情况，无法回滚到以前的版本意味着要进行数小时的意外工作。现在，如果有问题，我很容易选择以前的版本来恢复，并从我的最后一个保存点继续。

## [](#adapters-and-cables)🔌适配器和电缆

我总是在我的笔记本电脑包里放几样东西。我没有只在需要的时候才把它们打包，也没有冒着忘记东西的风险，而是养成了随时带着它们的习惯。事实证明，拥有适配器和电缆是在办公室和“野外”交朋友的好方法。

### [](#always-available)随时可用

这是我一直随身携带的物品清单。我带了自己的指针、电缆和适配器，从不认为场地会提供这些东西。这样我就不会措手不及。我几乎所有的商品都是从亚马逊订购的。

*   我选择的笔记本电脑包是 eBags 专业超薄笔记本电脑背包
*   BUBM(做唯一的自己)有线电视组织者
*   激光笔/幻灯片遥控器(我使用 Kiartten 无线演示器)
*   迷你显示端口适配器:HDMI、VGA 和 DVI(我保留了 2 个 HDMI，因为这些容易很快出现故障)
*   蓝牙曲面弧形鼠标
*   笔记本电脑的电源线
*   我手机的 USB 线
*   适用于我的健身手表的 USB 连接线
*   USB 插头
*   USB 转以太网转换器
*   RAVPower Turbo 系列 USB 3.0 快充充电砖
*   USB 3.0 4 端口集线器
*   多适配器(迷你到微型到 USB-C 等。)
*   HDMI 线缆(这非常适合在酒店电视上看电影和在房间里练习讲话)
*   Bose 降噪入耳式耳机
*   Jabra 耳机，适用于网络研讨会/旅途中的团队通话

### [](#international-travel)国际旅行

我为国际旅行增加的唯一东西是变压器和适配器。我推荐 BESTEK 200W 电源转换器 3 出口和 4 端口 USB 变压器...这有世界上任何国家的每一个适配器，一个插头我就有多个电源插座和 USB。这些插头都是通用的，这意味着你不必使用变压器，而是可以将笔记本电脑直接插入适配器，并将适配器插入墙壁。

## [](#food)🥑食物

为什么我会提到食物？因为重要的是你不能饿着肚子去做你的演讲！我遵循植物性饮食，所以我并不总能吃到聚会和会议上提供的食物。我试着打包零食，以防饿了，并在旅行时增加额外的物品。

[![Various food items](../Images/cb3658546c22585500e89d60b1dad3cb.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--Ky3fvrrn--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/e2x5hvwe5c4503xbs3kw.jpg)

这些包括蛋白质棒、干果和坚果。即使在不饿的时候，我也会每隔几个小时吃点东西，以确保我有足够的精力进行生动的演讲。在长途旅行中，我还会把蛋白奶昔装在单独的拉链袋子里，放在一个更大的袋子里，以最大限度地减少粉末沾到所有东西上的风险，并带上一个摇杯。

## [](#vitamins-and-medication)💊维生素和药物

在我看来，演讲者应该随时准备好三种关键药物(即使你只是去参加当地的聚会):

*   布洛芬治疗头痛和/或发烧
*   碱式水杨酸铋(一种常见的品牌是 Pepto-Bismol)用于胃部不适
*   洛哌丁胺(常见品牌为 Imodium)用于应对意外的胃肠挑战

前两个是为了帮助你感觉更好，最后一个是权宜之计，这样你就可以发表演讲而不必跑下台。

### [](#prescription-medications)处方药

我不喜欢带着我的全部处方旅行，也不喜欢把大部分瓶子都倒空，所以我手头上只有足够的缓冲剂。例如，如果我要旅行 5 天，我会在原来的处方瓶中保留大约 7 或 8 粒药丸，并随身携带。如果它丢失或被盗，我在家里还有剩余的供应，如果我在海关遇到问题，我可以出示瓶子上的原始处方。

### [](#sleep)睡眠

褪黑激素对在路上睡觉很有帮助。我与许多其他发言者讨论过这个问题，他们一致认为，这对旅行后安顿下来并帮助适应新的时区非常有用。

### [](#vitamins)维生素

我每天服用多种维生素，并且相信它们在旅行时更加重要。我不是医生，当然也不认为我的轶事经验是一个科学的案例研究，但事实是:我在旅行时感冒或其他病毒感染已经超过十年了(我一直在旅行)。我有过几个胃虫，但仅此而已！我听说很多人为不可避免的事情做准备，“我坐了 6 个小时的飞机，所以我要赶点东西”，我根本不相信这是不可避免的。

我把我的成功归功于定期锻炼、健康饮食和服用多种维生素。在我看来，维生素 C 是最重要的。我每天服用 1500 毫克，当我出国旅行时，每天增加到 2-2.5 克。

我的策略是购买足够大的小袋来装一天剂量的维生素，并且每天预先包装。然后，在路上，就像抓起一个包放在口袋里，直到我把所有东西都拿走一样简单。

如果你想知道我吃什么维生素，这是我的基本日常养生法。我故意省略了剂量，因为这些因人而异，你应该真正研究你在服用什么。

*   鱼油(是的，这是植物性的例外)
*   镁-有助于肌肉恢复和睡眠
*   维生素 C -免疫系统
*   男士复合维生素——我一整天都在服用，因为一天一片是个骗局

## [](#clothes-and-shoes)👔衣服和鞋子

如果是休闲活动，我倾向于穿一双跑鞋。我把这些松了，所以它们很容易脱下来穿上。如果活动更正式，我会带一双更正式的鞋子。

我把我的衣服分成四个基本类别:

1.  **活动服装** -适合我出席的展示和其他日子的服装
2.  **当地服装** -适合在休闲场所穿着的服装(例如，如果场地在佛罗里达，我会带短裤和短袖衬衫)
3.  **锻炼和游泳衣** -我总是试着在路上锻炼，并带一双游泳衣以防有泳池或水上活动
4.  **旅行服装**——在乘飞机进出时穿着宽松、柔软、舒适的衣服

记住，你可能会在活动中获得额外的衣服。

## [](#toiletries)🚽化妆品

这里唯一要分享的，因为每个人都是不同的，是我的系统，以确保我打包所有东西。在我旅行的前一天，我像往常一样处理我的事情，但是把我用的所有东西都放在我的柜台上。这最终成为牙线，电动牙刷等。把它放在一边，我就可以在旅行的早上很容易地拿到它，最后打包。这样我就知道我带了我需要的所有东西。

> ⭐ **提示**:除非你非常喜欢你的牙膏品牌，否则当我在登记入住时询问，几乎我住过的每家酒店都有牙膏。

## [](#understanding-airport-security)👮‍♀️了解机场安全

如果你坐飞机，你会遇到安检。我想以一些简短的提示来结束我的演讲，以帮助您加快这个过程:

*   安全着装！避免穿很多层，穿容易穿脱的鞋子。我通常把外套放在行李箱里，直到通过安检。你可以随时添加层后！
*   如果你幸运地进行了预检，你不需要从包里拿出笔记本电脑，脱下鞋子，在大多数情况下，你可以系上腰带。
*   没有液体！确保你的水瓶是空的。如果你有带液体的小件物品，确保它们放在一个单独的、容易拿取的袋子里。
*   你唯一需要隔离的是你的身份证和票。一旦你被转移到安全的地方，你就可以把它们收起来。
*   我通常把我的手表、手机和钱包放在我的笔记本电脑包里，有一个方便的口袋可以通过扫描仪。
*   对于国际旅行，假设您需要从包中取出笔记本电脑。在某些情况下，你还需要移除电线，这就是为什么我把它们放在一个易于移除的收纳件中。
*   此外，在国际航班中转时，始终假设您将再次通过安检！我会在着陆前把我大部分的东西打包在我的包里，以加快我转机的过程。
