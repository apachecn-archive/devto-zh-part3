# 区块链 vs 传统数据库

> 原文:[https://dev . to/ish war/区块链 vs 传统数据库 3p81](https://dev.to/ishwar/blockchain-vs-traditional-database-3p81)

在传统数据库中，客户端可以对数据执行四种功能:创建、读取、更新和删除(统称为 CRUD 命令)，而在区块链中，每个用户都可以确保他们正在检索的数据自记录以来未被破坏和更改
