我曾经认为软件开发是我的激情所在。人们常说这是我的激情所在。他们看到我喜欢做软件开发，并认为我必须追随我的激情。我认为我擅长软件开发，我必须追随我的热情。

但有时我会想，这真的是我的激情吗？我开始学习软件开发，作为我解决数学作业的一种方式。

做软件开发的时候还是很享受，也很兴奋。但当我写文章、诗歌和歌曲时，我也是如此。当出现新问题时，我的大脑会在我所有的知识和经历中徘徊；寻找解决方案。

但这种兴奋不会永远持续下去。有时候我不想写一些代码。有时候我只想追寻我头脑中随机冒出的想法。有时候我只是想重写或模仿我遇到的随机软件。

# [](#does-it-have-to-be-one)一定要一个吗？

我曾经相信我们一生中只能有一种激情。如果我的激情是软件开发；我对其他的事情都没有激情。我不得不去追，爱上它。

但是现在我在想，也许我对我做的所有事情都充满热情。也许写作是我的爱好。也许软件开发是我的激情所在。也许艺术是我的激情。

也许，我对一切都充满热情。

但是有时候，我经历了一些我并不热衷的事情。我对认识新朋友不感兴趣。我对搬到一个新地方不感到兴奋。我对尝试新食物不感兴趣。

我相信，我们可以拥有多种激情。

# [](#my-search-of-passion)我对激情的追寻

我尝试了很多东西。

*   做铁匠，爱漂亮的刀剑。锤钉子做一把小剑。
*   做“深奥”的事情，我研究星体投射；经历过清醒梦；
*   做软件开发时，我喜欢构建东西，这是最简单的，因为不需要太多硬件(与做铁匠相比)。
*   做音乐，弹吉他，打鼓，键盘，贝斯，甚至唱歌(我以前有个乐队，叫金梦)，写了几首歌。
*   做艺术，我喜欢看美丽的艺术；我会画画，结果可能会很难看，但我仍然喜欢这样做。
*   做硬件开发时，我焊接了几个电路；太神奇了，但我数学很差。不是因为我对数学没有热情，只是我学习不够努力。
*   写书的时候，我写了阿瓦加的几章；但最终做了很多重写，它从来没有完成。
*   做社交，我以前控制欲很强，权力饥渴；为此我运用了社交技巧。但是现在我不再对此感到兴奋。我寻找，我的激情是什么。我可以称之为我的激情的最接近的东西是，“解决问题”。

# [](#to-solve-a-problem)解决问题

我做铁匠是因为我想玩剑，但小时候没钱买剑。

我进行星际旅行，因为我想看更多的世界，然而作为一个青少年，没有钱和时间这样做。

我做软件开发是因为我想更快地解决我的数学作业，这是我正在努力解决的问题。

我做音乐是因为我想变得流行和出名，如果你来自一个偏远的村庄，这是一个快速的方法。

我做艺术是因为我想在附近有漂亮的东西，我喜欢以后看(我还记得我的第一个网站有多丑)。

我做硬件开发是因为我想有一个智能家居，那时候有一个很酷。

我确实写了一本书，因为我想表达阿瓦加的故事，这个故事直到现在还埋藏在我的脑海里。事实上，在 Avaga 的构思阶段，我发明了一种虚构的语言和角色；我曾经通过用那种语言写公式来欺骗我的学校考试，我会在考试时读到。

我确实操纵人们来解决我的问题，实现我需要的东西。

也许，我真正喜欢的是解决问题。当我开始写重复的代码时，我开始感到无聊和不兴奋。问题已经解决了，我只是不得不重复一遍；所以我很无聊。

# [](#passion-without-purpose)没有目的的激情

仅仅追随你的激情是危险的建议。没有目的的激情就像没有容器的火。它会迅速生长并开始燃烧东西。你必须为你的激情找到一个地方，一个目标。

对我来说，我的软件开发生涯是我激情的容器。软件开发热情和解决问题的热情。我找到了我的目标，但我仍在寻找。

我希望这篇文章能帮助到仍在寻找激情的你。也许你已经找到了一个，只需要找到目的。也许现在你认识到你有很多激情，找到很多目标可能很难，你可能想找到一个真正符合激情和目标结合的目标。

我也是，仍然在寻找我的激情和目标。这是一个永无止境的旅程，找到一个。
