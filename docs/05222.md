# 🐶ModdoC:一种阅读节点模块文档的新方法

> 原文:[https://dev . to/xenox dev/moddoc-a-new-way-to-read-documentation-of-node-modules-3ok 4](https://dev.to/xenoxdev/moddoc-a-new-way-to-read-documentation-of-node-modules-3ok4)

我们都在项目中使用节点模块，我们必须一遍又一遍地阅读他们的文档。但是在 npm org 上搜索一个节点模块来阅读它的文档会浪费很多时间。所以这就是我们制作 **MODDOC** 的原因。这是一个容易阅读节点模块文档的简单工具。

### [](#some-features)一些特性

***1。一次打开多个项目***

***2。显示依赖关系和开发依赖关系***
