## [](#virus)病毒

用户运行病毒，当任何程序启动时，它都会影响你的磁盘。通常，病毒会保持低调，不会被任何其他程序检测到。该病毒的代码将会感染任何新的程序或您附加的外部磁盘。这种病毒是一种软件，它会自我复制，并与其他计算机一起传播以破坏它们。

这些病毒大多像寄生虫一样，损害你的计算机的性能。在大多数情况下，病毒(如宏病毒)是罪魁祸首。因为宏病毒是嵌入在数据文件中的一段代码，然后允许可执行程序运行来影响病毒。

让我们通过例子来学习；如果你收到一封含有病毒的邮件，那么在打开这封邮件之前，你将无法感染病毒。但是一旦你打开了那封邮件，你就是在邀请病毒来危害你的电脑。不仅是电子邮件，还有附件，电子邮件链接也是一种病毒。当您打开该文件并在您的客户端之间共享时，这将从一台计算机传播到另一台计算机，并且这一链条将继续下去。计算机病毒有多种类型，它们会以不同的方式影响您的计算机，例如病毒和蠕虫。其中病毒是可执行程序，病毒是可以从一台计算机转移到另一台计算机的计算机程序。蠕虫是一种通过搜索安全更新来传播的病毒。

## [](#spyware)间谍软件

间谍软件是一种收集信息并将其传输到其他文件中的软件。有不同类型的数据，包括网站访问，浏览器，系统信息，还有你的 IP 地址。间谍软件帮助黑客收集信息，并给他们从主机隐藏的信息。该信息很容易被检测到，因为它包括组织中有意安装的键盘记录程序

这是一个跟踪你所有互联网活动的软件。这可能会对您的计算机的性能产生负面影响，因为您可以安装附加的软件搜索，更改计算机的设置等等。一般来说，有四种不同类型的软件，我们可以使用各自的目的和功能。他们是广告软件，木马，跟踪 Cookies，系统监视器。
