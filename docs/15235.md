# 截止日期与生命线

> 原文:[https://dev.to/koolamusic/deadline-versus-lifeline-54fk](https://dev.to/koolamusic/deadline-versus-lifeline-54fk)

在现代社会，我们被目标和里程碑卡住了，一切都在快速发展，千禧一代被期望达到非常高的期望，但这不是我们的观点

你有没有给开发人员一份工作让他们用 ETA 为你做？这适用于大多数创意人员、艺术家、音乐制作人、木匠等。

你是一个最后期限还是**生命线**的人:这个比较将帮助你找到答案

### [](#deadline)截止日期

*   坚持时间表，提前交付
*   坚持特定的流程/策略，并优雅地坚持到底

### [](#lifeline)生命线

*   擅长紧急交货，严格的交货要求
*   黑客程序产生最佳结果，大多数情况下是短期的。

这让我想到了穆里尼奥和瓜迪奥拉，两个很棒的人，他们都是赢家，但是我会把一个归类为生命线的人，另一个归类为最后期限的人

在创业的世界里，你需要两者的结合，立即推出一个功能，这样风投和用户都会很高兴，体验这个功能的潜力，然后逐步迭代整个过程，优化和微调交付和结果。

你是谁？截止日期:生命线
