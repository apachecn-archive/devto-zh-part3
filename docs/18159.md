# 什么是黑客？

> 原文:[https://dev.to/moderndaytech/what-is-hacking-2kpc](https://dev.to/moderndaytech/what-is-hacking-2kpc)

黑客是一个最常与恶意和非法活动联系在一起的术语。可能你在新闻和社交媒体上已经听说了很多，有人的电子邮件帐户被黑了，或者银行的网站被盗了，很多信息被盗了。

这一切都是完全可能的，因为我们处在一个每个人都使用互联网的世界，没有任何信息是私人的。

如果是出于好意，黑客行为本身并不违法。比如帮助一个公司，它的系统或者网络可以检查安全性，黑客攻击并不违法。

黑客攻击中的整洁非常重要。如果没有理由伤害任何人或任何人的财产，黑客行为就不违法。

一个心怀恶意的黑客，造成了严重的伤害，他被称为黑客，他不是黑客，因此，黑客和破解是非常不同的。

用简单的语言来说，如果有人黑进网络指出系统的缺陷，那么就叫黑客攻击。

即使一个人未经许可破坏了系统，即使这样也不被认为是无效的，因为这种黑客行为背后的主要目的是指出安全漏洞，以便可以修复它们。

黑客是进入系统偷窃或造成损失的人。这是出于恶意，在大多数情况下，黑客会在系统中留下恶意软件，制造问题并窃取信息。
