FeathersJS 使得编写实时 web API 变得非常容易。我想谈谈如何从钩子中发出数据。我有一个应用程序，它有两个服务组和 group-access，如果用户在 group-access 集合中有记录，他们只能从组中获取项目。

但这意味着当用户通过组访问服务被授予访问权限时，他们必须重新加载页面，以便重新获取组来反映新的访问权限。
