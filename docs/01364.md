## 为什么要用奶油蛋糕![Butter Cake logo](img/aa26f607a798f3cc33faeab3065a7ac2.png)

*   ✈️轻量级 59kb (css)和 4kb (js)
*   🗃模块化组件
*   🏠简单结构
*   📜使用 Sass(SCSS)构建
*   📲应答的

> ***不需要用 Gulp 等设置开发环境...因为酥饼用 [Prepros(免费 APP)](https://prepros.io/?ref=getbuttercake.com) 编译 SASS(SCSS)。***

可以在几秒钟内开始一个新项目😻！！

## 包括什么![Butter Cake logo](img/aa26f607a798f3cc33faeab3065a7ac2.png)

```
Butter Cake/
│
├── dist/
│    │
│    ├── css/
│    │   ├── butterCake.min.css (59kb)
│    │   └── butterCake.min.css.map (33kb)
│    │ 
│    └── js/
│        ├── butterCake.js (Non Minified) (9kb) - (Complete components Js)
│        ├── butterCake.min.js (Minified) (4kb) - (Complete components Js)
│        ├── alert.js 
│        ├── accordion.js 
│        ├── dropdown.js 
│        ├── modal.js 
│        ├── tabs.js 
│        ├── side-navbar.js (Side Navbar)
│        └── slideDown-navbar.js (Slide Down Navbar)
│
├── src/ (Includes SASS|SCSS Files)
│
├── prepros-6.config
│
└── index.html (Starter File) 
```

## ***入门*** ![Butter Cake logo](img/aa26f607a798f3cc33faeab3065a7ac2.png)

#### NPM

```
npm i buttercake 
```

#### CSS (jsdelivr)

```
<!-- BUTTER CAKE CSS -->
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/buttercake@2.5.2/dist/css/butterCake.min.css"> 
```

#### JS (jsdelivr)

```
<!-- BUTTER CAKE JS -->
<script src="https://cdn.jsdelivr.net/npm/buttercake@2.5.2/dist/js/butterCake.min.js"></script> 
```

## 浏览器支持![Butter Cake logo](img/aa26f607a798f3cc33faeab3065a7ac2.png)

Butter Cake 使用 [autoprefixer](https://github.com/postcss/autoprefixer) 使(大部分)Flexbox 特性与早期浏览器版本兼容。根据我可以使用，黄油蛋糕兼容最近的版本:

*   铬
*   火狐浏览器
*   边缘
*   旅行队
*   歌剧

仅部分支持 Internet Explorer (10+)。
