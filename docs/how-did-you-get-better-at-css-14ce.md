# 你的 CSS 是怎么变好的？

> 原文：<https://dev.to/aizkhaj/how-did-you-get-better-at-css-14ce>

我已经做了 2 年的开发工作，并且已经做了 5 个月的全职软件开发工作(不包括我的合同工作)，我在后端和前端分别花了 70/30 的时间。

我所做的任何前端工作都主要集中在用少量 CSS 和大部分 Bootstrap 将新组件放在 Angular 应用程序上。

虽然我的工作还不需要这样，但是我想加强我的 CSS 技能，这样我就能自信地做出我能想象的任何东西；而不是使用大量的基本 CSS 来使事情看起来足够好(一些位置调整或颜色/格式的变化)。我想少依赖 Bootstrap。

在你学习的时候，你个人使用了哪些资源来提高你的 CSS 技能？我知道有很多教程(我也接触过一些)，但出于某种原因，我似乎不能从这些教程中获得最大的收益，因为你需要练习 CSS 足够多的时间来获得一个好的窍门。

你有网页、图形等的特定列表吗？你想复制并试错你的方法到最后的结果？是否涉及结构化路径？有没有你觉得特别有用的实用/在线资源？

我的动机/原因:我主要做后端，但我希望能够更自信地为我的个人努力/兼职项目制作模拟。
