# [](#start-sending-plaintext-emails)开始发送纯文本邮件

这篇文章最初出现在我的项目 [Growthhacklist](https://growthhacklist.com/methods) 上。

发送纯文本邮件可以提高你的时事通讯的回复率。牛顿公司的 Karthik Suroju 在一次采访中分享了这个小想法。通过发送纯文本邮件而不是 HTML 邮件，他们的邮件回复率提高了 156%。

[![](../Images/68712e10f485b967e8e031d82aae777f.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--ummJ03Uh--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/1orbz3y97x5tebixjfzn.png) 
[另一个邮件例子](https://explore.reallygoodemails.com/stop-missing-user-feedback-use-plain-text-emails-to-boost-response-rates-6fccb4ad2157)

对于每封电子邮件，他们还总是遵循三条重要规则:

*   他们从不发送“不回复”的电子邮件
*   他们鼓励用户回复所有的邮件，包括时事通讯。
*   他们会在几分钟内回复所有查询，即使在节假日也是如此。
