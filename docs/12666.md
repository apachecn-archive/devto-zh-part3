# 我喜欢的 5 个播客

> 原文：<https://dev.to/dowenb/5-postcasts-you-will-love-32gd>

我远程工作，大部分时间独自在我的家庭办公室。这就出现了一个问题，房子太安静了！和一个机会:消费真棒播客！

这只是我喜欢的播客中的 5 个，我希望你也喜欢。

测试会谈
我喜欢测试会谈对测试自动化的清晰关注。这个播客制作精良，易于理解，强烈推荐。

我一直以来最喜欢的播客之一，Saron 非常幽默，并充分利用了她的嘉宾。这里的重点是探索人们的编码之旅。

[base cs](https://www.codenewbie.org/basecs)
Base 在 [base.cs](https://medium.com/basecs) 博客系列的基础上，Saron 和 Vaidehi 以一种非常消耗性的方式涵盖了一些经典的计算机科学话题。用这个我总能学到东西！

[T3】。网石！/ >](https://www.dotnetrocks.com)
一个非常向上的节拍和内容丰富的播客，这里关注的是世界上的发展。网

[本周科技](https://twit.tv/shows/this-week-in-tech)
这是我的选择，我想去更广阔的地方，听听科技行业的新闻。这里没有太多直接关于软件开发的内容，但是了解更广阔的技术世界是很好的！
