# 地理速成班

> 原文:[https://dev.to/ice_lenor/geo-crash-course-5fdj](https://dev.to/ice_lenor/geo-crash-course-5fdj)

作为一名开发人员开始在 Geo 工作时，您需要了解什么？继续读下去就知道了！

很长一段时间以来，我计划在 Geo 开始写这个指南。在本指南中，我将:

*   告诉你地球不是平的(震惊！)以及为什么那很不方便；
*   定义常用术语(什么是投影？还有一个坐标系？..OGC，OSM，WKT，WGS，ESPG？);
*   教你成为 geo 开发者需要知道的所有事情，为 Geo 或使用 Geo 构建东西；
*   解释在与 Geo 合作时的常见注意事项-该做什么和不该做什么；
*   给你一个地理技术，工业标准，数据库，图书馆的概述；
*   给你进一步阅读的参考。

我为对 Geo 感兴趣的同事们举办了这个研讨会，它非常成功！我希望你也喜欢它。

祝你环球旅行好运！
