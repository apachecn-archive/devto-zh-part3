# [](#video-games-from-my-childhood)小时候的电子游戏

我一直对编程充满热情。这一切都始于我小时候第一次玩电子游戏。我清楚地记得以前在我爸的笔记本电脑上玩[蜗牛邮件](https://en.wikipedia.org/wiki/Snail_Mail_(video_game))、 [Tropix](http://www.tropixgame.com/tropix.html) 、[喂食狂潮](https://store.steampowered.com/app/3390/Feeding_Frenzy_2_Deluxe/)、[汽车](https://en.wikipedia.org/wiki/Cars_(video_game))、[仓鼠球](http://www.raptisoft.com/index.php?game=hamsterball)有多开心。我特别喜欢*汽车*游戏，花了几个小时在开放的世界里漫游(或者说是比赛)。

当我父亲买了一台 PS2 时，我对游戏的热情增长了。我玩得最多的游戏是《史酷比》!首先惊吓、[海绵宝宝:比基尼底之战](https://en.wikipedia.org/wiki/SpongeBob_SquarePants:_Battle_for_Bikini_Bottom)、 [G-Force](https://en.wikipedia.org/wiki/G-Force_(video_game)) 、[撞车:介意变种人](https://en.wikipedia.org/wiki/Crash:_Mind_over_Mutant)、[极品飞车:地下 2](https://en.wikipedia.org/wiki/Need_for_Speed:_Underground_2) ( *向风暴上的所有[骑手喊出](https://www.youtube.com/watch?v=0VAF-4Er7_A)！*)。

几年后，我爸爸又买了一台游戏机；这一次，是 Xbox 360。我玩了一个[NBA 2K](https://en.wikipedia.org/wiki/NBA_2K)[刺客信条](https://en.wikipedia.org/wiki/Assassin%27s_Creed)[孤岛惊魂 4](https://store.steampowered.com/app/298110/Far_Cry_4/)[正义事业 2](https://store.steampowered.com/app/8190/Just_Cause_2/)[GTA V](https://store.steampowered.com/app/271590/Grand_Theft_Auto_V/)[sky rim](https://store.steampowered.com/app/489830/The_Elder_Scrolls_V_Skyrim_Special_Edition/)。后来，我短暂地回到了电脑游戏。在这方面，我想加上《我的世界》作为一个荣誉奖。

# [](#my-motivation)我的动力

我一直想知道人们是如何在电视/电脑屏幕上创造出如此广阔的世界的；他们是如何无中生有的；他们是如何告诉计算机做什么的。所有这些视频游戏都让我想追随游戏开发者的脚步。我也想创造一些对人们有持久影响的东西；会引发怀旧情绪的东西；对经历艰难时期的人来说，可以起到暂时缓解作用的东西；有价值的东西；让人开心的事；能让*我*开心的事情。结合我对学习和数学的热情，这个舞台为我开始编程之旅做好了准备。

为了确保我理解编程的基础，我从 web 开发开始，以保持简单。我可以说我一点也不后悔那个决定。也许我甚至可以说这是我当时所能做出的最好的决定。在许多方面，JavaScript 帮助我理解了什么是真正的编程以及它是如何工作的(我现在可以说这不仅仅是快速打字和记忆语法)。现在我正在学习 C++来开始我的游戏开发之旅，我所有的 JavaScript 知识(以及作为荣誉奖的 TypeScript)使学习过程变得更容易、更快、更直观。

无论如何，重要的是要知道我们为什么首先编码，以防止可怕的*倦怠*。忘记我们的根源肯定会让我们迷失在单调的编程中。也就是说，**是谁或什么给了你编程的灵感？**当然，我的兴趣是电子游戏，以及对学习、数学和创造事物(对他人有用的事物)的热情。
