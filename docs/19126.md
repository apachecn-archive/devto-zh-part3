### JavaScript 库

*   [牛逼 Javascript](https://github.com/sorrycc/awesome-javascript) :牛逼的浏览器端 Javascript 库、资源和亮闪闪的东西的集合。
*   微 js:一个微框架的微站点
*   ZeptoJS : Zepto 是一个面向现代浏览器的极简主义 JavaScript 库，具有很大程度上兼容 jQuery 的 API。
*   [charist . js](http://gionkunz.github.io/chartist-js/):简单响应图表。
*   Nivo : nivo 提供了一组丰富的 dataviz 组件，构建在令人敬畏的 d3 和 Reactjs 库之上。
*   Sizzle :一个纯 JavaScript CSS 选择器引擎，旨在方便地放入主机库中。
*   [DynamicsJS](http://dynamicsjs.com/) : JavaScript 库，创建基于物理的动画。
*   [拟像](https://simulacra.js.org/):DOM 的数据绑定功能。
*   [NProgress.js](http://ricostacruz.com/nprogress/) :一个纳米级的进度条。用逼真的细流动画告诉你的用户有事情发生了！
*   Turbolinks : Turbolinks 让你的网络应用导航更快。
*   [Clusterize.js](https://clusterize.js.org/) :轻松显示大型数据集的微型插件。
*   Rivetsjs :轻量级和强大的数据绑定+模板解决方案，用于构建现代 web 应用程序。
*   hyperdom :一个快速、功能丰富且简单的框架，用于构建动态浏览器应用程序。
*   在你的网站页面之间创造出流畅的过渡。
*   Vivusjs : Vivus 是一个轻量级的 JavaScript 类(没有依赖关系),它允许你制作 SVG 动画，给它们一种被绘制的感觉。
*   [Selectize](http://selectize.github.io/selectize.js/) : Selectize 是 textbox 和 box 的混合体。它是基于 jQuery 的，对于标记、联系人列表、国家选择器等等非常有用。
*   [Popmotion](https://popmotion.io/pure/) :一个功能强大、灵活的 JavaScript 动画库。
