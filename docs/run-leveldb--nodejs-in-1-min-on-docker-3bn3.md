# 在 Docker 上用 1 分钟运行 levelDB + Node.js

> 原文：<https://dev.to/sot528/run-leveldb--nodejs-in-1-min-on-docker-3bn3>

我在 [Docker](https://cloud.docker.com/repository/docker/sot528/leveldbtest/general) 上用[node . js implementation](https://github.com/Level/levelup)尝试了 [levelDB](https://github.com/google/leveldb) 。

# 用法

```
docker run -it sot528/leveldbtest 
```

# 运行在 Docker 容器上

```
node index.js 
```

你可以尝试编辑`index.js`中的任何东西。
