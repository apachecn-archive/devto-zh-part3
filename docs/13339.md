在众多面向网络开发者的优秀时事通讯中，我发现这 10 个给了我一个全面的视角，以最小的冗余和最大的娱乐价值展示了我们领域的当前趋势和必须知道的发展。

## [](#hacker-newsletter)黑客快讯

如果你只订阅一份时事通讯，这就是了。都在这里了。

💌[黑客简讯](https://hackernewsletter.com/)

## TL；速度三角形定位法(dead reckoning)

*TL；DR* 是为网络开发者策划的每日科技新闻。当然有大故事的链接，但也有有趣的 GitHub repos 和项目。

💌[TL；博士](https://www.tldrnewsletter.com/)

## [](#pony-foo-weekly)小马喷火周刊

Pony Foo 的时事通讯很棒，因为它交叉聚合了 web 开发的最新进展，并且比其他人更早到达。

💌 [Pony Foo 周刊](https://ponyfoo.com/weekly)

## [](#javascript-weekly)JavaScript 周刊

除了关于快速发展的 JavaScript 生态系统的更新之外，这份时事通讯还包含了各种技能水平的教程以及新的有趣项目的链接。如果你真的想成为书呆子，你可以注册它的姐妹出版物， [Node Weekly](https://nodeweekly.com/) 和 [React Status](https://react.statuscode.com/) ，但请注意，最重要的故事会出现在 *JavaScript Weekly* 上。

💌 [JavaScript 周刊](https://javascriptweekly.com/)

## [](#python-weekly)Python 周刊

如果你是一名 JS 开发人员，想学习第二种语言，我可以推荐 Python 吗？这份时事通讯一直都很好，如果你对机器学习感兴趣，你会找到各种水平的资源。

💌 [Python 周刊](https://www.pythonweekly.com/)

## [](#pointer)指针

*指针*倾向于解决与管理相关的问题，包括个人之间和个人内部。不管你是一名网页开发人员，在你职业生涯的哪个阶段，每一期都会有一些观点的改变。

💌[指针](http://www.pointer.io/)

## [](#history-of-the-web)网络的历史

没有必要急着阅读这份简讯，因为它都是旧闻，但它绝对属于需要知道的类别。每一期都是对“被遗忘的”网络历史的深入探究。这些问题总是引人入胜，总是能提供信息。

💌[网络的历史](https://thehistoryoftheweb.com/)

## [](#mozilla-developer-newsletter)Mozilla 开发者简讯

MDN 时事通讯列出了上面列出的内容，并提供了 noobs 和 pros 在可访问性、隐私、性能等方面的资源链接。额外收获:每周开发者工具提示(当然是针对 Firefox)。

💌 [Mozilla 开发者简讯](https://www.mozilla.org/en-US/newsletter/developer/)

## [](#programming-digest)编程文摘

这封低调的信函在精心策划高水平的、语言不可知的文章方面表现出色。每个星期天早上我都期待着它的到来。

💌[编程摘要](https://programmingdigest.net/)

## [](#weekend-reads)周末读着

最后但并非最不重要的是，周一给我提供迷因的时事通讯。它是与 web 开发相关的推文、视频和 gif(大部分)的幽默组合。教育性*和娱乐性*！

💌[周末阅读](https://labnotes.org/)

你最好的时事通讯是什么？下面让我知道。

制作和发布专业时事通讯是一个耗时的过程，需要仔细考虑内容和布局。如果你成为了以上任何一个网站的订阅者，通过查看他们的赞助商提供的内容来表示感谢，或者考虑捐款。
