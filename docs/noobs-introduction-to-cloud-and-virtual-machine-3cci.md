我见过很多初学者对术语云和虚拟机感到困惑，但最重要的是，我们为什么首先需要它们？因此，这是理解云和虚拟机术语含义的最简单方法，包括一些使用案例。

## 云

互联网的覆盖范围不断扩大，每天都有新的用户加入，并与世界其他地方连接。

云计算的概念也来自于同样的需求。提高您的应用程序的处理能力，以便它可以轻松地到达世界每个角落的最终用户。

不管你是开发者、企业主、工程师还是独立博主。随着用户群的增加，您将需要更可靠、更有效的系统来处理不断增长的流量，并为您的产品提供顺畅、快速的访问。

### 那么云计算机在这方面有什么用呢？

正如你可能知道的那样，互联网上的所有东西都存储在各种各样被称为服务器的设备上。

现在，当你想推出一款数码产品时，不管它是大是小，你都需要把它放在高度可靠的服务器上，以获得更好的性能。

#### 让我们以杰克为例

杰克开了一个个人博客，在那里他谈论股票市场的投资。他在自己的服务器上运行这个博客。

他刚刚开始这个新的博客，他的博客的访问者数量被限制在 50 到 500 人。在这种特殊情况下，jack 自己的服务器足以处理这些负载。他做得非常好。

几个月后，他的访客数量从每天 1000 人增加到 10000 人，现在他的服务器因为流量增加而耗尽，所以 jack 决定增加他的硬件(硬盘、Ram、处理器)，他购买了新的东西并升级了他的服务器。

几个月后，猜猜会发生什么？

是的，你猜对了。博客的访问量大幅上升，杰克又有麻烦了。

#### 你看出这里的问题了吗？这里有几个。

1.  杰克将不得不每隔几个月购买更多的东西并手动升级他的服务器。
2.  电脑硬件不是芯片。他将需要相当多的资金。
3.  杰克将不得不独自照看服务器。你知道定期清理，软件更新，维护等..
4.  第三点是很难的工作，因为它占用了很多时间，所以杰克必须在写博客和维护他的服务器之间进行权衡。
5.  在某个时候，如果 jack 决定出售/关闭他的博客，那么他会如何处理他手动设置的所有服务器？他卖吗？和他在一起，以防将来他决定重新开始写博客？

现在杰克可以自己做以上所有的事情，或者让别人来做。通过这样做，他将节省大量的资源和时间。

#### 对于杰克来说，更好的选择是转向云计算

所以让我们看看云计算将如何拯救杰克。

> 云计算是一种信息技术(IT)范式，它支持对可配置系统资源和高级服务的共享池的无处不在的访问，这些资源和服务可以通过最少的管理工作(通常通过互联网)快速供应。云计算依靠资源共享来实现一致性和规模经济，类似于公共事业。—维基百科

## 换句话说就是“你为你使用的东西付费”

借助云计算，jack 将能够添加或删除硬盘、处理器、ram 等资源..根据他的需要。这些资源将与云服务提供商一起存在，这样他将能够以更少的管理工作继续工作，并将能够更加专注于他的产品。

### 你知道这些服务是谁提供的吗？

业内许多公司都专注于云计算，如亚马逊、微软、谷歌、阿里巴巴等

您可以向该服务提供商购买服务器空间，他们会负责剩下的部分。你的基本需求得到了满足，你可以自由地专注于你的产品。如果需要增加硬件..那么你只需点击几下鼠标..

## 虚拟机

### 你知道什么是虚拟机吗？

通俗地说，虚拟机是对计算机的模拟。让我们看一个例子来理解它的意思。

假设你现在使用的是 Windows 系统，突然你需要一个 Ubuntu 操作系统。现在你要做什么？你会移除 windows 并安装 Ubuntu 吗？还是买个新系统装在那里？

简单的回答是**不**两个选项。

你要做的是安装一个类似于虚拟盒子的软件，它允许你同时运行多个操作系统，一旦安装了软件，你就可以在那个环境中安装和使用 Ubuntu，而不会影响你当前的系统架构。

这个环境叫做 ****虚拟机**** 。
