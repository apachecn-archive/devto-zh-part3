# Flappy Bird 是如何让我开始从事软件行业的

> 原文：<https://dev.to/healeycodes/-how-flappy-bird-got-me-my-start-in-software-453m>

看着我的 flappy bird 蹦蹦跳跳，回应我的按键，我的内心燃起了一团火。22 岁时，我刚刚完成了 Unity 教程。开发软件是童年的梦想——成为一名工程师。但它从未真正发生过。所以我决定实现它。

[![](img/6dfa852dd74ade10e1b61e53ce152f4d.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--ypCMebPk--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_66%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/3f3s6qbcg68m4lreuxav.gif)

视频游戏教程似乎很容易上手，所以我坚持了几个星期。关于函数、类型和类的强力学习。我远离说明书，在我能做的地方修修补补，但感觉我并没有真正创造任何东西，只是在复制。我在等待我的创意写作学位的结果。我在大学期间一直在从事一些网络技术的工作，但与我想去的地方和我现在的位置相比，这是微不足道的。

我看过一篇关于计算机科学硕士转换的文章。这些是完整的硕士课程，旨在让充满热情的学生达到学士学位研究生及以上的水平，包括一篇沉重的论文。这些课程真的会推动你，许多机构已经在学期开始前“建议”学习。(如果你想出类拔萃，它们是必不可少的)。如果这些课程听起来很激烈——甚至可能很可怕——那是因为它们确实如此。很多时候，我发现自己研究概念是为了学习其他概念，以便查阅课堂笔记！

我申请了很多，但最终选择了远程教育硕士。我尊重这个机构，校友们的议论吸引了我的注意力。第一个模块是用 Java 编写的，所以整个夏天我都在准备用 Java 完成流行的 Java MOOC [面向对象编程的两部分。我知道我会选择任何可选的网络相关模块，所以我也完成了大部分的](http://moocfi.github.io/courses/2013/programming-part-1/)[免费代码营](https://www.freecodecamp.org/)。除此之外，我还用 C#开发了游戏堵塞和[小游戏](https://healeycodes.com/games/)。这整个旅程只有一条规则是可能的:制造东西。小程序，CLI 程序，傻逼游戏，傻逼机器人。

学生们活跃的不和谐聊天室，教授们快速回答问题的速度，以及互动的直播讲座，缓解了我对在线授课的担忧。长达一年的强化课程正是我所需要的，来支持我从构建自己的应用程序中学到的一切。模块选择是灵活的，但仍然迫使我们学习基础知识，如算法和数据结构、数据库和实际的软件开发。

不久，我不得不开始考虑求职。这就是事情变得可怕的地方。到目前为止，我所做的一切都没有风险。如果你努力学习，你通常会取得好成绩，如果你不断尝试建立玩具项目，不要放弃，那么你通常会成功。但是求职呢？面试？失败是过程的一部分。

我远离了安全的有趣的应用程序和课程，建立了三个组合项目。每一个都旨在解决我或我所在社区的其他人面临的问题。它们都经过了完整的堆栈和单元测试，连接了数据库，在真实的服务器上以真实的主机名托管——我想要所有的修饰。我最终建立了一个网站分析服务，一个低延迟的配对编程解决方案，以及一个让游戏开发者能够跟踪用户高分的网站。全部开源。

求职信似乎在网上受到了很多批评，但我认为我的大多数面试都与我为每家公司写的个人段落有关。如果我觉得我写下的东西不真诚，那我就重新开始。通常，我谈到的事情都是通过电话或面对面的方式提出来的。我在全国范围内申请，电话面试的回复率只有 10%。我在房间里踱来踱去，与招聘经理交谈，并出人意料地通过了所有人。我熬夜做编码测试。一点一点地整理文件。定时算法测试也让我汗流浃背。

面对面的面试更令人兴奋。经过这么长时间，我终于可以和真实的人谈论我的编程激情了！我在房间里呆了很长时间，可能给人的印象是太兴奋了。我让他们大多数人失望了。这在当时是错误的想法。有些面试你就是过不了。我走出每一个房间的时候都在想:这就是我要找的。每个人都觉得很完美，直到电子邮件终于来了。

我最终通过的面试(今天早些时候回家的那份工作)感觉和其他任何一次都一样。我笑着走出来，给我的未婚妻打电话，告诉她这个*和这个*不一样。她笑了，充满希望但又谨慎乐观。令我惊讶的是，我对 HTTP 状态代码的最后一分钟审查得到了回报——这部分感觉有点不真实。

从那以后我就一直在学习和编码。在我的上一篇帖子中，我谈到了我在这间办公室的日常生活。在我的下一篇文章中，我可能会写下我认为怎样才是一个好的投资组合项目🕹️.评论里的建议或者 [@healeycodes](https://twitter.com/healeycodes) ！
