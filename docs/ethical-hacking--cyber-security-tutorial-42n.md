**为什么是网络安全？**

一家名为剑桥分析 a 的公司最近在 2018 年 3 月发生了 5000 万个脸书账户的巨大数据泄露，该公司为美国大选预测收集数据。数据泄露事件曝光，脸书被处以罚款，因为他们没有保护用户信息，也因为他们的系统缺乏透明度。想象一下，像脸书这样的大公司很容易受到这种数据泄露的影响，因此网络安全是每个组织的重要措施。网络安全用于避免数据泄露，保护网络和系统的完整性免受攻击。
