## 干净的代码

项目中干净代码的扩展。

*   [内聚](https://github.com/mschwager/cohesion#flake8-support) -测量 Python 类内聚的扩展。
*   [flake 8-2020](https://github.com/asottile/flake8-2020)——检查`sys.version`或`sys.version_info`误用的插件。
*   flake8-alfred - Alfred 是一个 flake8 插件，用于警告不安全/过时的符号。
*   [flake 8-按字母顺序排列](https://github.com/tlocke/flake8-alphabetize) -检查`import`语句和`__all__`列表的顺序。
*   [flake8-black](https://github.com/peterjc/flake8-black) -运行 [black](https://pypi.org/project/black/) 检查 Python 编码风格的插件。
*   [flake 8-折线](https://github.com/sobolevn/flake8-broken-line) -禁止换行符反斜杠(`\`)的插件。
*   寻找你的程序中可能的错误和设计问题。
*   [flake8-builtins](https://github.com/gforcada/flake8-builtins) -检查 python 内置的变量或参数。
*   给 flake8 增加编码魔法评论检查(编码:)。
*   在 python 中强制尾部逗号。
*   第八课-理解 -帮助…
