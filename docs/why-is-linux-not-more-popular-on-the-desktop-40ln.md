# 为什么 Linux 在桌面上没有更受欢迎？

> 原文：<https://dev.to/aghost7/why-is-linux-not-more-popular-on-the-desktop-40ln>

*图片取自[维基百科](https://commons.wikimedia.org/wiki/File:Penguins_walking_-Moltke_Harbour,_South_Georgia,_British_overseas_territory,_UK-8.jpg)。*

当谈到 Linux 在桌面上被更广泛地采用时，存在一些问题。你认为的主要问题是什么，如何解决？

我认为 Linux 最大的问题是碎片化。对于在 Linux 上发布游戏的开发者来说，这尤其是一个问题。

这个问题的解决方案是像 [Snap](https://snapcraft.io/) 、 [AppImage](https://appimage.org/) 和 [Flatpak](https://www.flatpak.org/) 这样的交叉分发安装程序。每个应用程序都有自己的一套库，这意味着它们不依赖于特定于发行版的包。

我认为另一个问题是势头。关于 Linux 在桌面上的表现有一些过时的观点。

挫败这股势头需要在公关方面付出更多努力。我认为这不会随着时间的推移而自行修复。
