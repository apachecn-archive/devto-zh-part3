# 选择性学习的重要性

> 原文:[https://dev . to/amrrbakry/the-importance-of-selective-learning-13m 6](https://dev.to/amrrbakry/the-importance-of-selective-learning--13m6)

我们很幸运生活在一个每天都有新技术出现的时代，帮助我们建造各种各样的东西，让我们的生活变得更容易。总有一些令人敬畏的工具或语言是你想要学习和使用的，因为它引入了一种新的思维方式或不同的范式。

随着今天对信息和学习资源的访问，如果你知道如何进行谷歌搜索，你会发现大量的资源来帮助你学习这种工具或语言，以及大量更多的东西要学习。这可能会让人不知所措。

学习本身就是一种技能，我们当中能够快速学习的人是幸运的，但是记住这不是一场比赛也很重要。按照自己的节奏学习；只要你在学习，你就应该抓紧时间。但是在你开始这个学习之旅之前，你应该花一点时间考虑你是否真的需要投入所有的时间来学习这个和/或那个工具。因为*你可以学到任何东西但是你学不到所有东西*；这就是为什么你应该特别**选择**你所学的东西。

总是会有开始使用这种新语言的诱惑，因为它是多么的棒和革命性，或者它将如何取代另一种语言，然而，这并不意味着你应该放下一切，开始投入时间学习它。退一步，彻底思考你是否真的需要学习它。

对你所学的东西有选择性是你区别于其他人的方式。本质上是关于你知道多少，而不是你知道多少。

记住永远不要停止学习！
