Java 编程语言是世界上最常用的编程语言之一，有成千上万的书籍可以学习各种主题。

这里是我为初学者学习 java 编程的前 3 本书。这些书中的每一本都有自己的风格。

***1-如何像计算机科学家一样思考。*T3】**

这本书实际上是麻省理工学院的计算机编程入门课程。这本书有两个版本。一个用 python，另一个用 Java。

我一般会给计算机专业的学生推荐这本书。

***2-穆拉希的 java 编程***

如果你想通过更多的实践而不是理论来学习如何迈出 java 编程的第一步，这就是你需要的书。

穆拉希的所有出版物都遵循同样的模式。左页解释了一个具体的点，右页显示了如何完成这一点。所以，如果你翻开书的任何一页，你会发现理论在左边，代码在右边。

***3-梁***Java 编程入门

我认为这本书是一件艺术品。它包含了大量的信息。对于像我这样的非英语母语者来说，它的语言非常简单。这本书的内容组织得很好。这本书还包含一些中级和高级主题。

我最喜欢这本书的地方是它的摘录。每章都有一套练习，从简单的任务开始，难度逐渐增加。
