我做了几个现有的回购，包括[“jackrabbitmvc”一个 PHP 的微框架](https://github.com/dexygen/jackrabbitmvc)，但也删除了几个，现在增加了 6 个，虽然公平地说，我没有对 3 个分支中的 2 个做任何重要的事情。我做的一个分支是浏览器端的静态内容生成器，我将其重命名为[抢了](https://github.com/dexygen/upstaged)的风头，这也是我在[自己的 github 页面上使用的](https://github.com/dexygen/dexygen.github.io)

最后也是最重要的是 [withMethods](https://github.com/dexygen/withMethods) ，它允许从 React 中的其他组件调用方法。我已经在 Reddit 上被告知这违背了 React 的哲学。很好，我希望它能流行起来。
