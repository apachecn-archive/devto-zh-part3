# 如何帮助你的孩子学习编码

> 原文：<https://dev.to/jeancarl/how-to-help-your-child-learn-tocode-j3g>

我看到越来越多的青少年开始编程并参加黑客马拉松。令人兴奋的是，技术不仅促进了青少年之间的交流，而且青少年也认识到他们也可以自己构建这些应用程序和解决方案。

我记得十几岁的时候，我在拨号连接上学习编写 HTML 和早期的(非常有限的)JavaScript 代码，并上传到 GeoCities。有人记得 FrontPage Express 吗？糟糕，我刚和自己约会过。我保证我今天手写的 HTML 不再包含冗长的标签。

今天，技术已经进步到让任何人都更容易开始做这件事。从教你一切的在线课程，从[设计思维](https://www.ibm.com/design/)到[区块链](https://www.ibm.com/blockchain/)到 [Docker 和 Kubernetes](https://www.ibm.com/cloud/container-service) ，教育唾手可得，而且大部分是免费和自定进度的。

有这么多的在线服务为我们使用的服务提供 API，有无数的组合可以放在一起。一旦您开始编写代码并接触到 API 参考，您通常可以找到包含足够代码片段的 Swagger 文档，以便复制和粘贴在一起。

代码片段到处都是。当你发现一个引起你兴趣的框架或项目时，浏览 [GitHub](https://github.com) 可以发现大量的项目和它的使用方法。如果这也可以…是一个很好的询问和完成的提示。

最后，当你冒险自己编写一个解决方案并偶然发现一个错误时， [StackOverflow](https://stackoverflow.com) 可能是一个很有帮助的地方，可以找到有时神秘而令人困惑的错误消息的真正含义的问题和答案。似乎你可能有的每个问题都已经被回答了。

偶尔会有家长问我，他们希望鼓励自己的孩子开始编程，尽管他们自己并不从事编程职业。有趣的是，看到这些父母有动机鼓励他们的孩子，但却不知道如何去做。

我相信父母可以做几件事来帮助他们的孩子找到对编程的热情和兴趣。第一，可达性。我通过使用一台废弃的电脑和拨号上网学会了如何编码。今天，iPads、云以及简单的网络浏览器都可以成为工具，让人们不仅可以使用技术，还可以定制技术来做你想做的事情。

就像父母鼓励孩子对他们周围的世界提出问题并寻求答案一样，在编程中使用这种技术。你听过一句话吗，用问题回答问题？得不到答案的直接满足是令人沮丧的，但是如果你教孩子“如何钓鱼”，他们将学会自己解决问题的技巧。

最后，坐下来和孩子一起学习会很有帮助。即使你没有使用该技术的第一手经验，你也有生活经验。用技术建造东西不仅仅是让计算机执行一组指令。事情远不止如此。

有了用户界面，我们如何设计直观、易用、高效的工具？什么类型的问题真正重要？现有的解决方案已经在做同样的事情了吗？你如何使它更独特，并使自己与竞争对手区分开来。

在结对编程中，两个开发人员一起编写代码，互相交换想法和功能。使用同样的技术，孩子们可以从父母那里获得想法，并在不同的环境和心态下获得有用的反馈。

我有时开玩笑说，最后一步是给你的孩子你的信用卡和消费限额。例如，在构建物联网解决方案时，35 美元的 Raspberry Pi 可以以非常合理的价格发挥巨大的作用。有许多价格合理的传感器可以实现更复杂的解决方案。这可能比买那些会导致蛀牙的糖果要好。不过，有相当多的传感器显示，你的银行账户会很快对这个新发现的兴趣感到后悔。
