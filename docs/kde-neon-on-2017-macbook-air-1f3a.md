# 2017 MacBook Air 上的 KDE 霓虹灯

> 原文：<https://dev.to/fundatillus/kde-neon-on-2017-macbook-air-1f3a>

我是基于 Linux 的操作系统的超级粉丝，但我总是被苹果创造的硬件和软件的苗条结合所深深吸引。虽然我还不能让自己放弃我的 iOS 小工具(消费设备)，但我最近从我的 MacBook Air(创作设备)上删除了 macOS，并安装了 [KDE 霓虹灯用户版](https://neon.kde.org/download)。

**剧透预警:**太牛了！

## 安装

关于如何在你的机器上安装 Linux，尤其是 Ubuntu，有很多教程，包括大多数 MAC 电脑，所以我就不说了。在撰写本文时，当前版本是基于 LTS Ubuntu 18.04 的。如果你在为 2017 年的 MacBook Air 寻找一台电脑时遇到问题，请搜索“MacBook Air 7，2”或 2015 年版本(也是型号 7，2)的任何教程。

唯一真正的安装问题是无线连接，这是意料之中的，因为许多笔记本电脑在基于 Linux 的操作系统上存在无线问题。幸运的是，在这种情况下很容易补救。

要在现场环境中激活无线，请打开终端并键入:

`sudo apt install bcmwl-kernel-source`
`sudo modprobe -r wl`
T2】

这将安装内核模块以支持 Broadcom BCM4360(或类似的)无线芯片组。第二个和第三个命令重新加载内核模块(可能需要，也可能不需要，但不会造成伤害)。然后，您将能够从网络图标中选择一个无线网络，并继续进行类似 Ubuntu 的正常安装，包括下载更新和安装第三方软件、驱动程序等。

一旦安装完成，你启动到你的新的 KDE 霓虹灯环境，你不会再有无线。您需要执行与上面相同的命令，并且有几个选项:

1.  插入安装媒体并配置 apt 以识别它；或者
2.  使用 USB 或 Thunderbolt 以太网适配器获得有线连接；或者
3.  通过 USB 连接你的手机。

无论您选择哪个选项，接下来执行您在真实环境中执行的相同命令，您将拥有无线连接。

## 回顾

KDE 是我在 Linux 上首选的桌面环境，所以请原谅我的偏见。安装后，你会看到一个令人敬畏的 KDE 体验 OOTB。一切正常，包括 Fn 键、波浪键...一切！电池寿命不如 macOS 提供的好，但也还过得去，可以使用。安装和校准`powertop`提供了接近 macOS 的电池寿命。虽然我没有运行到零，但在电池使用 4 小时后，我还剩 60%。

没什么可报道的。很好用！如果我有任何重大问题要报告，我会更新这个帖子。

## 为什么？

我为什么要这么做？因为我可以...但主要是因为我在 BestBuy 以 750 美元的价格买了一台全新的 2017 款 MacBook Air。它拥有双核 i5、8GB 内存和 128GB 固态硬盘。当我在购物的时候，一台类似配置的 KDE Slimbook II 大约是 1100 美元。我喜欢 Slimbook 提供的扩展自由，但我不愿意为此付费。
