# 软件开发之后，下一步是什么？

> 原文：<https://dev.to/koolamusic/after-software-development-what-s-next-6j>

我最近遇到了一个熟人，他在软件开发领域有大约 20 年的经验，我想知道在他的职业和生活的这个时候他下一步应该做什么，尽管我没有机会和他分享我的答案，但在我脑海中，这些是我的想法

*   管理
*   师徒制

### 管理

拥有 20 多年的从业经验，你应该有机会注意到潮流是如何形成、改进和变化的，并走向产品开发周期的顶端，这个周期可以是 **(idea = >设计)**或 *(design = >实现)*或 **(design = >用户采用)**你应该能够将这些与你的个人能力和优势相匹配。

### 师友

回馈社区，这些年来你学到了什么，你可以与年轻一代崭露头角的软件开发人员和爱好者分享，相信我，这对那些刚起步的人来说是一个非常孤独的旅程，与有长期经验的人交谈可以减轻压力和安慰，附带的好处是，它可以帮助你跟上最新的技术趋势，也让你跟上行业趋势。

然而，在任何领域工作了 20 多年后，我倾向于认为，经过这么长时间，任何人现在都应该知道自己的优势和长处，知道他们可以将剩余的时间和精力集中在哪里。
