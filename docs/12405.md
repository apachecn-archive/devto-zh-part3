# 他们起来了吗？用于检查站点是否启动的节点 cli

> 原文:[https://dev . to/perkinsjr/are-they-up-a-node-CLI-for-checking-if-a-site-is-up-15n 8](https://dev.to/perkinsjr/are-they-up-a-node-cli-for-checking-if-a-site-is-up-15n8)

是一个可以全局安装的 npm 软件包，它可以检查是否有任何域启动。

## [](#why-did-i-create-this-node-cli)我为什么要创建这个节点 cli。

我过去一直在开发一个特定领域的 api 框架，遇到了一个问题，就是他们宕机了，或者我不知道是我的互联网宕机了还是一个网站宕机了。

## [](#how-to-install)如何安装？

```
npm install aretheyup -g

//or this for single project

npm install aretheyup 
```
