# 12 月项目问责:Rizzo

> 原文:[https://dev . to/kaelscion/December-project-account ability-build-and-ship-Rizzo-1 daf](https://dev.to/kaelscion/december-project-accountability-build-and-ship-rizzo-1daf)

所以对于那些不知道的人来说，我需要 Dev 社区来帮助我在经历了人生中最糟糕的打击后重新站起来。

大约三周前，我在我的厨房里昏倒了，并且癫痫发作。现在，虽然我的生活中有很多糟糕的健康问题，但这是迄今为止最可怕的一次，而且完全是突然发生的。我整个时间都是清醒的，基本上被锁在我的身体里，对垃圾邮件没有任何控制，尽管我几乎完全意识到我的周围。坐救护车去了急诊室，做了几次检查后，我没有任何理由就离开了。显然，我的器官、血液和心脏异常强壮健康。然而，我花了几个星期才达到这样的程度，我可以背着 10 磅的东西到处跑，而不会头晕和昏倒。

现在，我一直在与严重的 TBI 的影响作斗争*和*已经经历了重大的脑部手术。由于这个原因，我与慢性疼痛和非常奇怪的健康问题作斗争。但是当我被身体症状淹没时，我总是能够忽略或忍受疼痛，并拒绝停止，因为我确信这些只是我的大脑向实际上没有受伤的身体部位发送疼痛信号。然而，这一最近的事件虽然尚未解决，但已导致医生假设我的身体只是因为近六年来几乎持续忽视疼痛信号、压力、严重焦虑和抑郁而关闭。你可以想象，最终面对一个你根本无法克服的问题会改变你的生活。这让我重新思考我对待残疾人的整个方式。这会导致抑郁和完全失去动力。

经过深思熟虑，我决定利用开发社区来帮助我找回动力。我设计了一个叫 Rizzo 的 web 应用程序，并制作了框架。该应用程序旨在向用户网站发送网络爬虫，下载图像、js 和 css 文件，压缩和/或优化它们，并根据请求通过电子邮件将它们发送给网站所有者，从而自动化“缩小”网站媒体的过程，而不必将所述媒体上传到服务。只需键入您的网址，确认您的电子邮件，您将在几分钟内收到针对网络性能优化的媒体，同时以最低的质量损失保存在 zip 存档中。最棒的部分？这个应用程序将没有任何数据库。整个过程将保留零数据。这一切都基于自动合成和自动修剪 docker 图像，这些图像可以旋转、完成任务、旋转和删除。

我计划在 12 月底推出我的 MVP，但由于我的精力和体力有限，我需要你的帮助来完成这件事！我呼吁 Dev.to 的力量来帮助向世界推出我的第一个真正的项目。你，我的朋友和同事，所能提供的任何动力、鼓励或反馈，都会给我我所能提供的，暂时不再召唤我自己。提前感谢，我期待你的爱，支持和评论！
