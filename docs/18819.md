# 为动物福利建立开源的想法

> 原文:[https://dev . to/kevanantha/an-idea-to-build-open-source-for-animal-welfare-56kf](https://dev.to/kevanantha/an-idea-to-build-open-source-for-animal-welfare-56kf)

我不擅长写作，但我有一个关于动物福利的开源想法(这只是一个粗略的想法)，特别是在印度尼西亚，很多人扔掉和抛弃他们的宠物，因为他们(宠物)已经长大了，人们只爱他们的宠物，当他们还是小狗的时候。甚至有宠物被交易只是为了被人类消费。

幸运的是，现在人们已经开始意识到，因为许多动物保护者给了他们教育，我想帮助他们(动物保护者)，我只是一个热爱动物的前端开发者 *p.s 大多数动物保护者都有自己的庇护所*。

重点是我想建立一个与动物保护者相连的应用程序，如果人们看到被遗弃的动物，人们会发送他们的位置或聊天或其他东西，应用程序会向最近的动物保护者发送警报或推送通知。

该应用程序可能有一些功能，以采用宠物(与条款和条件)和其他。

同样，这只是一个粗略的想法，但我真的想建立这个项目

如果有想法，请提出来。

如果我错了，请纠正我。
