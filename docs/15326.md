# 你以前可能没有听说过的 VS 代码扩展

> 原文:[https://dev . to/the Geoff Stevens/vs-code-extensions-you-may-not-heard-before-5ed 3](https://dev.to/thegeoffstevens/vs-code-extensions-you-may-not-have-heard-of-before--5ed3)

VS 代码最好的特性之一是它的扩展的巨大市场。现在有近 10，000 个扩展可用，可以调整和定制 VS 代码来增强您的工作流。

我总是留意新的和被低估的扩展。因此，如果您正在寻找更多的工具来添加到您的代码编辑器中，我已经编译了一个 VS 代码扩展的列表。

## [](#create-impressive-images-of-your-source-code)为你的源代码创建令人印象深刻的图片

[Carbon-Now-Sh](https://marketplace.visualstudio.com/items?itemName=ericadamski.carbon-now-sh) 使用网站 [carbon.now.sh](https://carbon.now.sh) 为你的代码创建漂亮的截图。如果你想在博客文章或演示文稿中使用代码片段，Carbon 是一个非常棒的工具。

[![Carbon screenshot](../Images/7246d626077216578ea5c8879d59bc87.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--fzkwTDrY--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/iu1u6clo5bdruwr30ybn.png)

要使用 Carbon-Now-Sh，突出显示 VS 代码中您想要转换为图像的一段代码，然后键入 Alt-Cmd-A 或打开命令面板来访问 Carbon 命令。该扩展在 carbon 站点上打开一个浏览器窗口，其中突出显示的代码经过适当格式化，可以导出为图像。

## [](#control-your-music)控制你的音乐

如果你喜欢边编码边听音乐，vscode-spotify 可以让你在 vscode 中控制 spotify。

状态栏显示当前正在播放的歌曲，并允许您播放、暂停和更改歌曲。该扩展甚至有一个按钮，可以找到你正在听的歌曲的歌词，并显示在一个新的标签中。

[![vscode-spotify screenshot](../Images/ce990e5c48e4aeacbf0adb8e7f31e46f.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--MP5ks1Z0--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://github.com/ShyykoSerhiy/vscode-spotify/raw/master/media/screenshot.png)

除了状态栏控件，各种命令使从编辑器控制 Spotify 成为无缝体验。

[![vscode-spotify screenshot](../Images/769eada52e6ce14e039ac5230a4ff4f4.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--Dd-SFh3Z--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://github.com/ShyykoSerhiy/vscode-spotify/raw/master/media/screenshot2.png)

## [](#send-a-message)发送消息

[Slack](https://marketplace.visualstudio.com/items?itemName=sozercan.slack) (不直接隶属于公司 Slack)帮助你通过你的 Slack 工作区快速给同事发消息。通过使用命令面板来键入您的消息和选择您的收件人，Slack 使发送快速消息变得容易。

[![Slack screenshot](../Images/456ee93b4c398fc51c9a1219050b5019.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--tNFTCZtC--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_66%2Cw_880/https://github.com/sozercan/vscode-slack/raw/master/slack-send.gif)

如果您需要更健壮的消息传递功能， [CodeStream](https://marketplace.visualstudio.com/items?itemName=CodeStream.codestream) 允许您将消息和评论附加到特定的代码块。团队成员可以在源代码中讨论和注释他们的代码。

[![CodeStream screenshot](../Images/de67b1d40182e05540e82765f832ffaa.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--yAISd-Qr--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://raw.githubusercontent.com/TeamCodeStream/CodeStream/master/images/CodeStream1.png)

CodeStream 可以免费使用，但最终也会推出一个付费层。

## [](#color-your-windows)给你的窗户上色

[窗口颜色](https://marketplace.visualstudio.com/items?itemName=stuart.unique-window-colors)根据根目录名称的散列给每个新的 VS 代码窗口一个唯一的颜色。

[![Window Colors screenshot](../Images/61f2c11186799f726032d9091258d8fa.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--OHRiT4cc--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://raw.githubusercontent.com/stuartcrobinson/unique-window-colors/master/img/live_light_screenshot.png)

窗口颜色为您的工作区增加了一点颜色，可以帮助您跟踪不同的窗口和项目。

## [](#color-your-code)给你的代码上色

[颜色高亮](https://marketplace.visualstudio.com/items?itemName=naumovs.color-highlight)是一个小扩展，用相应的颜色高亮显示十六进制代码和 RGB 值。

[![Color Highlight screenshot](../Images/7fa370fadf5ada76d6f9e5cfb2aaf22e.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--tDSH78cg--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://naumovs.gallerycdn.vsassets.io/extensions/naumovs/color-highlight/2.3.0/1499789961213/Microsoft.VisualStudio.Services.Icons.Default)

在做出设计决策时，看到代码中显示的颜色非常有帮助。

## [](#stop-typing-consolelog)停止键入 console.log

[Turbo Console Log](https://marketplace.visualstudio.com/items?itemName=ChakrounAnas.turbo-console-log) 将 console.log 片段快速插入到代码中。选择您想要调试的变量，键入 ctrl + alt + l，扩展将自动插入一行来记录变量。

[![Turbo Console Log screenshot](../Images/355a530655fc1408e225336ac96ae74f.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--Du40Na5X--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_66%2Cw_880/https://image.ibb.co/dysw7p/insert_log_message.gif)

您还可以使用快捷键 alt + shift + c 注释掉由扩展插入的所有日志消息。

## [](#take-a-break)休息一下

如果你需要快速转移注意力，视觉蛇代码可以让你在 VS 代码中启动一个快速的蛇游戏。

[![Visual Snake Code screenshot](../Images/a8ed998c4e0e299d33923a18a86a8b27.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--_G2ThpIe--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://github.com/ZhouYou528/visual-snake-code/raw/master/screenshot/gameplay1.png)

只是要确保在一个空标签中打开游戏，否则它可能会覆盖你现有的工作。
