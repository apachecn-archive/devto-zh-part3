# 如何不搞砸求职面试

这篇文章对一些人来说可能会有争议，甚至可能是惊人的大胆。亲爱的朋友们，我鼓励你们‘说一堆谎言！’“用羊毛蒙住他们的眼睛！”在工作面试的时候。来吧，伙计们，尽情推销自己吧！让他们买你。不要！让他们为你讨价还价吧！

良好的...你撒谎的时候应该看起来很自信！如果你只是结巴，没有人相信你，即使你说的是实话。以一种快速而有趣的方式去做。这些人在你身上度过的时光，应该是他们美好的回忆。谁知道呢，也许他们不仅想在面试时有同样的情绪，而且还想在接下来的两年里有同样的情绪...

> 对于那些良心不允许他们说谎的人，这里有一个小小的旁注:结束你的顾忌。广告就在我们身边，众所周知，任何广告不是谎言就是夸张。公司宣传自己，在网站上向你展示想要的职位。当你申请工作时，它会自我宣传。你用你的时间来支付，这是你唯一有价值的东西。你用自己的生命来偿还！

我希望我安抚了那些道德高尚的人的灵魂。我说到哪了？没错。好吧，如果你决定推销自己，出卖你的时间，那就专业地做，换句话说，撒谎！

### [](#how-to-pass-the-technical-interview)如何通过技术面试

警惕谎言专家！那些是团队领导和高级开发人员。在技术面试之前，他们会和人力资源专家开会，会上他们会说这样的话:

“谢尔盖·彼得罗维奇，别让他撒谎。”“哦，别担心，玛莎，我不会的。”

你应该谨慎对待这些人。你最好保持沉默，不要离题，更重要的是，要表现出你有能力。如果你能在面试中保持积极的态度，那将是一流的。

如果团队领导很聪明，他不会用任何给定系统的所有细节来询问你技术问题。当然会有一些，但是他会对你的思维方式更感兴趣。采访一个‘士兵’，这样的人就想着这个士兵可能会成为一个什么样的‘军官’。

那些喜欢问技术问题的人也很容易逃脱惩罚。完成几次面试后，你会发现问题几乎都是一样的。你可以预先研究它们。令人惊讶的是，这些问题通常与你将要在工作中完成的任务无关。唉，事情就是这样的。

### [](#surface-knowledge)表面知识

没有什么比你的表面知识更能迷惑你的对手，让他认为你很聪明。它在面试中的重要性不可低估！

如果你被问及某项特定的技术，如果你以前从未使用过，这没什么大不了的。但是如果你知道它是什么，为什么要用它，那么工作就完成了一半。如果你能在你的电脑上运行它，并使用它一点点，你可以说它对你来说很熟悉。熟悉是不是意味着别的？

如果你知道一些常见的 IT 话题，无论是网络理论还是敏捷、设计模式或当今流行的 messenger Slack，那真的很酷。你在说什么并不重要，重要的是你‘跟上 IT 世界’。这是招聘你的关键因素之一。

所以，多看看维基百科，你就有机会给人留下专家的印象，甚至成为专家。

### [](#deep-knowledge)渊博的知识

我在上面提到过，一个开发者可以比作一个士兵或者一个军官。文章的这一部分解释了原因。这个想法很简单:如果你申请一个初级职位，你不应该限制自己不知道中级或高级职位要求的东西。如果你什么都懂一点，那就太好了。即使是最难的题目也要一点一点地学。

我们每个人都有自己特定的技能。很可能一些复杂的主题对你来说很容易，而一些简单的主题可能很难理解。这就是为什么我认为 100%的高级开发者并不具备初级开发者所需要的全部知识。每个人都有自己的盲点。有时候你不够幸运，面试的大多是那些不熟悉的领域。但有时恰恰相反，你已经能给人留下某个领域专家的印象，现在是你“提高价格”的机会了。

还有一个原因可以解释为什么你应该马上尝试研究你想要的领域中的复杂话题。想象一下这样一种情况，一位技术专家在进行面试时推迟了他“超级困难”的日常任务。还有你，需要一些简单的问题。当然，他会低估你，因为他可能有很多其他更重要的任务，他很无聊，他觉得自己好像在浪费宝贵的时间。然后你的谈话转移到一些更复杂的话题。他变得更加兴奋。如果你能够转移到他现在正在做的主题上，并表现出哪怕是最轻微的专业知识，那将是一个完全的成功！

### [](#all-the-possibilities)所有的可能性

那么如何找到面试官感兴趣的话题呢？这不是一个简单的问题，通常我不能在面试前回答这个问题。但是值得一试。

你一定要浏览职位要求，阅读所有列出的技术。如果你对其中一些有所体会，那就更好了。即使你现在没有在找工作，跟踪网站上的职位也是有帮助的。

每天早上，我都会从一些知名网站上收到一些 Python 职位空缺的精选。我从头到尾看了一遍。我检查他们使用的技术，看看是否有我还不知道的新东西。如果有，我一定会打开维基百科，甚至更彻底地研究它。

抓住一切机会！做一个聪明的骗子！向所有职位发送你的简历，参加所有你被邀请参加的面试。在秋千上失去的，在旋转木马上得到的...这家公司一开始可能看起来真的很高端，但最终会是一家乏味的公司。但在一个绝对荒谬的职位背后，也可能有一个非常热情和活跃的团队。嗯，没有什么比多次面试更能让你为面试做好准备了。

所以我们去骗他们所有人吧！
