# 然而，丹妮尔编码了

> 原文：<https://dev.to/dsuurlant/nevertheless-danille-coded-53i4>

## 我的故事

[![Image: Daniëlle, a woman with shoulder-length red hair, posing next to a cut-out of a Saturn V rocket engine.](img/3186d4fdd92f7249bd5ba193d7883eba.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--npTasgFF--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/clhidxn4g2afbkvbdz3o.jpg)

我叫 danille Suurlant，今年 35 岁，是一名全栈开发人员。我专攻 PHP，Symfony 和 API 开发。我从 2013 年开始和 Symfony 合作，从 2003 年开始和 PHP 合作，从 1999 年开始和网站合作。2011 年，我从鹿特丹应用科学大学毕业，获得了 ICT 学士学位。

作为一个 IT 界的女性，女性在 IT 界的地位对我来说既是专业问题也是个人问题。薪酬差距是真实的，工作场所的歧视也是真实的——我曾经因为不适应“男性文化”而被拒绝求职。我考虑过提交正式投诉，但后来选择了不提交，因为我不想在荷兰的网络开发小世界里被冠以“难搞”的名声。

但是即使我努力工作，学得很快，只想构建令人敬畏的软件而不用担心我的性别，偏见还是找到了我。这些年来，我不得不忍受关于三明治、厨房、清洁、家务、驾驶技能、性吸引力甚至性侵犯的“笑话”，每次都与自己辩论这是不是我说话的那一天。我通常不会。然而，我这样做的时候，我经常遇到翻白眼，令人不安的沉默，以及对我如何“开不起玩笑”的评论。

但即使在这种“笑话”并不常见的地方，性别歧视也更加隐蔽。男人们凌驾于我之上，重复我说过的话，就好像他们自己想到的一样，并且不相信我的专业意见。我的声音很少或根本听不到，令人沮丧和气馁。大多数时候，和我一起工作的人甚至没有意识到他们在做什么。他们是真正聪明、勤奋、富有同情心的人，只是很少或根本没有意识到制度化的性别歧视如何影响他们每天与女性的互动。

我编码是因为这是我最喜欢用大脑做的事情。无论是创造新的东西——甚至在我输入之前就想象抽象层和应用程序逻辑——还是通过跟踪数据和错误的路径来追踪 bug。我喜欢创造东西；在我的个人生活中，我写作，画画，编织。我喜欢做东西。而编码可以让你无中生有。这对我来说是真正的魔力，也是我擅长的。我已经找到了一份职业，做我喜欢的、我擅长的事情，而且报酬很高——很棒，对吧？

作为一名女性，我已经适应了更加关注、合作、体贴和同情。作为一名开发人员，这很适合我。我在团队中运作、处理反馈、以最符合我的反馈对象的沟通风格的方式构建反馈方面没有任何问题，并且能够将自己置于客户的位置，以最大程度地将他们的需求与我们为他们构建的内容相匹配。

作为一名女性，我被社会化地认为我不太有能力做数学、算法、逻辑、理性推理、抽象、科学和物理。尽管世界上第一个程序员是女性。如果我从事了近 20 年的计算机工作可以说明什么的话，那就是我在逻辑、理性和抽象方面做得很好。不要相信你的性别与你的电脑编程能力有任何关系。

尽管如此，我还是想鼓励女性进入编程领域。正如我刚才解释的，这并不容易，也不是没有障碍的。但这正是这个领域需要你的原因。在计算机科学的开端，女性就在那里。我们是第一批程序员。让我们夺回我们的地盘。
