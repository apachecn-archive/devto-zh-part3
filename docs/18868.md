# 初始化

> 原文:[https://dev.to/pfbyjy/init-3a74](https://dev.to/pfbyjy/init-3a74)

我最近决定，我需要扩大我的数字足迹。我做出这个决定的理由很简单:在不确定的未来的某个时刻，我可能需要制作*重要内容*，我希望人们参与其中。然而，如果我不花时间建立一个追随者，那么这个潜在的*重要内容*将会像互联网上的许多内容一样:我在 Twitter 上的三个追随者将会形成一个充满内部笑话的简短帖子，以回应包含链接的推文。当森林中的树木总是倒下时，我们很快就会产生听觉疲劳。如果一棵树想被听到倒下，它必须培养一个听众，所以我们在这里。

然而，建立数字足迹也有其自身的挑战。其中最主要的是，它需要大量的人感兴趣的内容。我发现预测很多人会感兴趣的东西几乎是不可能的。

排除了一般吸引人的内容，我的另一个选择是我会感兴趣的内容。在这方面，有三件事困扰着我。首先，我要在对话中补充什么，别人还没有以更好的方式说过？第二，我可以懒到麻木的程度，而且我非常善于为自己的懒惰找理由。最后，关于*在线*的唯利是图和交易性质，有一件事深深困扰着我。此外，我对表现给自我带来变化的方式保持警惕。

一个比我更愤世嫉俗的人认为，所有的互动最终都可以归结为交易绩效。相反，我将满足于不那么愤世嫉俗但仍然辛辣的观点，即所有的在线互动最终都是交易性的表现(我知道这种说法和在网上发布这篇文章的决定之间的紧张关系，但我没有创造在线*的规则，我只是遵守它们)。我也不要求意识形态的纯洁；如果你想要兄弟，就给我涂上伪君子的颜色，管他呢 **\ *(ツ)* /** 。*

在我列出的三件事情中，最大的障碍是我的懒惰，所以这篇文章是我在此时此刻解决这个问题的方式。我不能说接下来会发生什么，这很可能是我的最后一篇文章。至于我能在对话中加入什么新的见解，我已经决定了，至少为了这些帖子，这无关紧要。只要我对产品足够满意。这就引出了最后一点，雇佣兵之类的废话。我这里的解决方案是我没有，但是我会分享(哈！看吧！)随着我对这个问题的思考不断深入。

我会写任何我觉得我应该写的东西。大部分是温和的，但我希望能适应

善良的人们，作为对我永远感谢的交换，我恳求你们所有人:赐予我这个网站所能提供的任何参与量子。
