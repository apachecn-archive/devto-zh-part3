# 一个管理你的种子的电报机器人

> 原文：<https://dev.to/ray/a-telegram-bot-that-manages-your-torrents-mcn>

这个机器人解决了一个简单的问题:它允许你从 Telegram 管理在你的 raspberry pi 或者 linux 下的 pc 上开始的种子下载。从手机上使用很方便。这个机器人很容易使用。它是用 Python 构建的。

[![TeleTor Telegram Bot](img/01e1822a946a4ced57802674699ca64b.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--0pYumFua--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_66%2Cw_880/https://www.dropbox.com/s/vj9liavfx9xph4x/TeleTor.gif%3Fraw%3D1)

它可以帮助你管理你的种子在传输 BitTorrent 客户端。安装相当容易。

你可以列出，开始，停止，删除你的种子。可以设置一个路径别名，用于下载。
实时显示下载进度。
