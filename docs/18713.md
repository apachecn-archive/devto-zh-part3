# 加密记事本

> 原文:[https://dev.to/sigmanor/crypto-notepad-4g7e](https://dev.to/sigmanor/crypto-notepad-4g7e)

[`Official website`](https://crypto-notepad.github.io)|[`Download`](https://github.com/Sigmanor/Crypto-Notepad/releases/latest)|[`Source code`](https://github.com/Sigmanor/Crypto-Notepad)|[|`Documentation`](https://github.com/Sigmanor/Crypto-Notepad/wiki/Documentation)

类似记事本的简单文本编辑器，用于保存和加载 AES 算法加密的文件。

[![screenshot](../Images/2b7677b238c6ac30a595db1d0f209da2.png)T2】](https://github.com/Crypto-Notepad/Crypto-Notepad/wiki/Documentation#screenshots)

## [](#features)特性

*   包括默认 Windows 记事本的所有功能
*   与 windows 上下文菜单集成
*   拖放支持
*   检测编辑器中的链接
*   с可定制的编辑器
*   с可定制的加密选项(哈希算法、密钥大小、密码迭代)
*   将自己的文件类型与应用程序相关联
*   轻便的

## [](#system-requirements)系统要求

*   Windows Vista SP2 和更高版本
*   。NET 框架 4.5
