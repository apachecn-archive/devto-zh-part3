## [](#firstly-who-am-i)首先，我是谁？

我是一名一年级的信息技术硕士学生，职业道路略有不同。在获得经济学学士学位并工作了一年多之后，我决定改变我的职业道路，攻读 IT 学位。我一直倾向于它，但不知何故最终为我的本科生选择了一条不同的道路，但即使在那时，我总是发现自己回到它，这就是为什么去年我终于决定不再做一个回旋镖，并追求同样的硕士学位。作为一名初学者，我明白转行不是一件容易的事，所以我想分享一些对我来说有意义的建议，我觉得这些建议可能会帮助你更顺利地完成职业过渡。

## [](#1-get-inspired)1。获得灵感

我觉得如果我们有合适的灵感，我们可以做得更好。这并不意味着整天阅读励志名言会让你成为一名出色的程序员。我的意思是，它会让你一开始就有正确的心态。

如果我们有像我们这样的人值得尊敬，我们就更容易理解并看到自己在那个位置上。尤其是如果你是一名进入科技行业的女性新手，你可能会质疑自己的技能与性别的关系，主要是因为当你考虑科技领域时，你可以仰视的女性比例很低。不要让这阻止你！

深入挖掘，你会发现女性在编程刚刚成为一个领域的早期阶段发挥了关键作用。事实上，第一个计算机程序员阿达·洛芙莱斯是女性。那么，为什么你会有哪怕一分钟的时间认为自己不适合呢？

## [](#2-accept-the-steep-learning-curve)2。接受陡峭的学习曲线

**编码不容易。**经常沮丧，怀疑自己，自己的能力被赋予。你需要明白，你正在学习一种语言，以便与机器对话，如果你不能形成它们理解的连贯句子，它们就无法读懂你的面部表情，也无法猜测你想说什么。它会完全按照你的要求去做，这似乎是一个显而易见的声明，但你会惊讶地发现你把事情搞砸了多少次，因为你没有考虑机器如何理解你所说的话。我觉得当你开始意识到并开始这样思考的时候，就是你*开始爱上编程的时候。*

所以，你越早接受你不会在一夜之间写出神奇的代码，越早接受学习任何语言都需要时间和练习才能精通，这个过程就越顺利。

## [](#3-be-part-of-the-community)3。成为社区的一部分

我已经记不清有多少次我开始学习编码并中途退出，主要是因为我是一个人在做，没有一个支持系统。所以，如果你没有注册一个学位，也没有一群和你有相同经历的人，那么当情况变得艰难时，你很容易就放弃了。

在这种情况下，参加当地的科技活动，在网上出现，会给你更多的信心，让你负起责任，无论如何，当有人一起学习时，学习会更有趣。

在 Twitter 和 Instagram 上查看*# women itech # 100 days ofcode # code newbie # women whocode*，你会发现有这么多鼓舞人心的人与你一起踏上这一旅程。此外，在 *Meetup 应用*上注册，我相信无论你在哪里，你都会找到你喜欢的科技活动，如果没有，为什么不开始呢？！

## [](#4-follow-a-curriculum)4。遵循课程

技术是一个相当广阔的领域，每天都有新的闪亮的东西出现，这可能会吸引你，并最终说服你暂停你开始做的事情，尝试市场上不同的，可能“更好”的东西。尽管抵制诱惑很难，但这绝对是一个明智的选择，尤其是在你刚刚起步的时候。

不仅仅是新技术，甚至当你开始学习第一种语言时，很容易陷入谷歌搜索和研究最好的/初学者友好的语言的恶性循环，我想我已经读过一篇关于几乎每种语言的帖子，证明它是最容易和最好的语言。

所以，在这种情况下，最好的方法可能是首先弄清楚，你想用网站、应用程序等做什么。，并学习最适合的技术。例如，如果你的最终目标是全栈开发，FreeCodeCamp 是一个不错的选择。最终，你学习的第一种语言，不管是什么，都将是最难的。与其试图成为一个万事通，不如坚持下去，高效地学习它。

## [](#5-accept-your-achievements)5。接受你的成就

如果你曾徘徊在科技行业考虑一段时间，你可能会遇到术语*‘冒名顶替综合症’*。维基百科将其定义为“一种心理模式，在这种模式中，个人怀疑自己的成就，并对被揭露为‘欺诈’有一种持续的内在恐惧”。这是一种很常见的感觉，尤其是刚开始的时候。

每次，我解决一个算法难题或编写一个有效的功能，我都有这种感觉，这一次只是碰巧发生了，我可能不会再做了。

在这种情况下，和自己讲道理，接受自己的成就，相信自己的能力，相信自己知道如何做到这一点是非常重要的。这是你需要经常处理的事情，我不确定它是否会消失，我还没有真正遇到过不偶尔面对冒名顶替综合症的人，但是你会在处理它的过程中变得更好。*(我觉得在这一点上，我比在座的任何人都更想说服自己)*

## [](#6-keep-going)6。继续

最后，就像所有事情一样，关键是坚持不懈。有那么多人在这个旅程中取得了成功，只是因为他们坚持不懈地努力。

就我今天所处的位置而言，就我希望达到的编码技能水平而言，似乎已经到了尽头，但这比我一年前的水平又近了很多。
*所以，相信自己，坚持走下去！😊*
