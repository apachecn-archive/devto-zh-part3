# 了解公司

> 原文:[https://dev.to/pablomarti/find-out-about-companies-2749](https://dev.to/pablomarti/find-out-about-companies-2749)

一家公司试图招聘你，你会怎么做才能发现这是否是一个好的选择，而不进入可能会浪费你时间的面试过程？

选项:

1 -在网上搜索并提问。公司声誉好吗？多久了？多少钱？如果是创业公司，创始人的过去是怎样的？

[https://www.glassdoor.com](https://www.glassdoor.com)

[https://www.crunchbase.com](https://www.crunchbase.com)

[https://angel.co](https://angel.co)

领英、脸书和推特。

2 -与朋友交谈，他们知道些什么？

3 -在面试之前，不要问太多关于初创公司的问题(如果是这样的话)。

你还做其他什么事情？
