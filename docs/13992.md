# 你好世界！

> 原文:[https://dev.to/fetz/hello-world-jke](https://dev.to/fetz/hello-world-jke)

你好，世界！一条简单而经典的信息，然而背后却蕴含着许多个人意义:

```
// It's the first thing I use
// when learning a new programming language
// before doing a TODO application :P
console.log('Hello world!');

/* ... */
// some times I use it
// for doing a quick debug
console.log('Hello world!'); // Why is this not working?! (╯°□°）╯︵ ┻━┻
/* ... */ 
```

我开始了写这个博客的冒险，我想分享一些东西:

1.  提高我的英语写作/交流技能。
2.  写一些小的和增量的帖子，稍后我会把它们浓缩在一本我想写的免费游戏编程书里。
3.  一种回馈别人的方式，就像别人给我的一样。
4.  提高我对所学学科的理解。
5.  最后是与他人联系。

在以后的文章中，我会更详细地探讨我正在学习的东西，我想写的书，以及为什么试图向他人解释一些东西，可以帮助我们了解更多。

PS:如果你看到一些错别字，语法错误等，请在评论中告诉我！
