# 实用二元分析书评

> 原文:[https://dev . to/nuculabs _ dev/practical-binary-analysis-book-review-1l6i](https://dev.to/nuculabs_dev/practical-binary-analysis-book-review-1l6i)

我从 11 月份开始读这本书，花了大约两周时间才读完。如果你打算尝试一下，你应该对 Linux 和编程有点熟悉。以下是我对此的想法。

[![BookCover](../Images/b4616c7f422444f8639fdf2ff3d7a38e.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--fN-srwMn--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/4jg1oods6dq3dlig48wx.png)

我最喜欢这本书的地方是，它用简单明了的方式解释了这个主题。作者是一个非常有知识的安全研究员，他的工作是最先进的！

这本书帮助我填补了很多关于二进制分析是如何完成的，代码混淆，线性反汇编程序，递归反汇编程序，中间语言和许多工具和库的空白。它还帮助我学习了一些我不知道它们存在的东西，比如:代码注入、二进制插装、动态污点分析和符号执行分析。

每一章都为下一章奠定了基础，在每一章的结尾，你会被邀请去解答那些帮助你强化和理解信息的练习。代码和例子可以在该书的[网站](https://practicalbinaryanalysis.com/)上找到。

我觉得附录很有用的书不多。这个是个例外！有一个附录指导你进一步阅读，还有一个讨论了书中使用的反汇编器和工具。作为这个领域的新手，我非常喜欢得到像丹尼斯·安德烈斯这样的专家的推荐。

我很喜欢这本书，我希望你也会喜欢！🙂
