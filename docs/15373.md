在计算机科学中，人工智能(AI)是由机器展示的智能，与人类和其他动物展示的自然智能形成对比。

我们仍然处于人工智能时代的开端。事实上，像谷歌和微软这样的全球科技巨头正在考虑在人工智能方面迈出更大的一步，但他们仍然对后果持谨慎态度。人工智能的例子有 Siri、谷歌搜索引擎、类似相关视频的推荐、你可能喜欢的电影(它是人工智能的一部分，被称为推荐系统)。

**1-学习 Python**

Python 是开发人工智能机器的最佳语言之一。它很容易学习和理解，因为它非常接近英语。

**2-线性代数**

AI 没你想的那么容易。事实上，它需要对数组和矩阵有深刻的理解，以便教授和构建您的代理(AI 机器)。

精通线性代数不需要学位，这些资源可以帮助你学习、可视化和理解矩阵和数组:

khanacademy.org
youtube.com

在那里你可以搜索线性代数并开始学习。

**4-配备高 GPU 的笔记本电脑**

人工智能需要大量的时间来训练，你的笔记本电脑越快越好。tensor flow(AI 需要的 Python 库)运行在 CPU 或者 GPU 上。当使用后者时，你会得到更快的结果，因此，如果你真的想从事人工智能，就去买一台超快的电脑吧。
