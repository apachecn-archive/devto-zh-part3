# 采访出错了！

> 原文：<https://dev.to/hannah_omu/interview-gone-wrong-51ja>

雇主经常忘记面试不是单方面的事情。受访者评估你是否适合做雇主，就像你评估他们是否适合做雇员一样。”

我走进面试环节时，已经感觉到这可能不会有好结果。当我回头看时，这应该是我的第一个**巨大的红旗**，我会避免这整个不必要的经历。

我很早就到了会场(15 分钟)，我通常用这个时间来评估当前员工的情绪和互动，我在远处静静地看着。办公室没有接待员，所以我没有机会介绍我自己以及我为什么在那里。

当我的采访开始的时候，他们让我又等了整整 15 分钟，当我走进房间的时候，小组成员似乎并没有被他们的迟到所困扰。

这次面试到底出了什么问题？我真的应该走开，当:

1.  小组成员做了自我介绍，然后“老板”让我背诵他们的名字——我猜是为了测试我的记忆能力？

2.  他们问了我一些个人生活的细节，我坦率地回答了，却得到了一个恼人的问题，“既然你是妻子和母亲，你还有社交生活吗？”

3.  我要求澄清我的 JD，并被告知它可能包括擦桌子，甚至为团队做午餐-我猜这是一个轻松的笔记？

面试结束时，我感到非常沮丧地离开了。我希望他们永远不会联系我，但唉！他们同意了，并要求我下周去做一次技术测试。我拒绝了这一请求，解释说我的法学博士在最初的面试中没有清楚地说明，并祝他们一切顺利。

为什么面试如此令人疲惫？
