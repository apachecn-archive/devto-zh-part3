# 我知道你累了

> 原文:[https://dev.to/ashleemboyer/i-know-you-re-tired-2la5](https://dev.to/ashleemboyer/i-know-you-re-tired-2la5)

生活有时真的很艰难。我们都在经历一些事情。你并不孤单。

你正在做的是艰苦的工作。它占用了你大量的精力。你觉得自己的努力没有价值。我知道这不公平。但是，如果你正在读这篇文章，并且有强烈的共鸣，很可能你还不能停下来。只是时机不对。

那么，现在怎么办？你必须继续前进。全力以赴。如果不多也没关系。不惜一切代价。我们都有自己的日子，需要一点时间。有些人比其他人需要更多的时刻。

如果你的生活中现在还没有与这篇文章联系起来，请确保让其他人来分享他们的时刻。我们不知道其他人在他们的个人生活中经历了什么。有些人过得很糟糕。不要让任何人的生活变得更艰难。
