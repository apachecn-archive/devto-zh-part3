## 什么事？

JavaScript 易学易用，至少我们都同意它有一个吸引人的渐进的学习曲线。
事实上，任何人都可以运行它的第一个“Hello World！”只需打开浏览器的控制台并键入“console.log('Hello World！')即可在 1 分钟内编写脚本)".

这是为什么它是被一个非常大的社区支持的最广泛使用的编程语言之一的原因之一。

尽管有这些毋庸置疑的最伟大的属性，JavaScript 也有一个坏名声:我听说它是“世界上最糟糕的语言”，或者“它完全不可靠”，“它只是用来制作按钮动画的愚蠢的解释语言”。

我知道所有这些担忧来自哪里，但我真的相信 JavaScript 是世界上最容易被误解的编程语言。

这就是为什么我决定开始收集所有关于 JavaScript 的笑话、批评和误解，并试图揭开它们的神秘面纱。
