# 对象设计基础-简介

> 原文：<https://dev.to/olivermensahdev/basics-of-object-design-series-28ol>

1 月 3 日，我重访了一些我过去参与过的项目。几乎每个项目的每个代码都遵循面向对象的范例。尽管如此，我还没有真正意识到我实际上是如何为软件项目编写代码的。大多数情况下，我会选择一些模式，然后按照这种方法进行设计和实现。然而，有一件事贯穿几乎每个项目，那就是，每个应用程序都是围绕对象设计的。

为了更好地在软件项目中设计和创建对象，我从[马蒂亚斯·诺贝克](https://twitter.com/matthiasnoback)那里得到了这本书[对象设计风格指南](https://leanpub.com/object-design)。我这个星期开始读它。现在，我可以坦率地说，这是一本了不起的书。我将通过一系列博客帖子与你分享我从这本书中学到的东西。一旦我们看完了这一系列的文章，我们将对如何为简单和复杂的应用程序设计和创建对象有了正确的认识。
