# 也是戴夫·多特还是戴夫·多特·蒂哦？

> 原文：<https://dev.to/_adam_barker/is-it-dev-dot-too-or-is-it-dev-dot-tee-oh-54do>

[![Man, freaking out. circa 2019](img/d89f8e57a7ea7acdcd433dd4d2fca9d2.png)](https://res.cloudinary.com/practicaldev/image/fetch/s---fMUflp---/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/4xc4b8ei79l6m7w0hnp3.jpg)

因此，实际上我整个一生都在把这个网站称为“dev dot too”，但我刚刚听到有人把它称为“dev dot tee oh”。

现在我吓坏了。当我意识到我实际上没有在任何人面前说出这个网站的名字，从而避免了我只能想象的相当尴尬的时候，我感到很害怕。

但这让我想到:我上一次这么做是在很多年前，当时人们无法决定 gif 是硬 G 还是软 G。硬的或软的 **G** 像制表符还是空格？

所以现在我想听听人们在最长的时间里把事物称为错误事物的例子...

...或者有那么一瞬间，当他们听到有人在发某个音时会想:我从来没想过我应该那样说。
