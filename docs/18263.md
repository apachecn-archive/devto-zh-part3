# OOP 太棒了！这就是原因！

> 原文:[https://dev.to/fpuffer/oop-is-great-and-here-is-why-1o05](https://dev.to/fpuffer/oop-is-great-and-here-is-why-1o05)

我对 OOP 的第一印象，很多年前，来自 C 编程:酷，我们把所有东西都放到一个类里，声明一些属性。这些就像全局变量，但是没有坏味道。不再需要传递太多的争论。

我花了几年时间才发现，这并不是 OOP 的真正好处。

不幸的是，当今天看 OOP 代码时，许多开发人员似乎仍然坚持这种思维模式。

不幸的是，这就是 OOP 仍然在学校被教授的方式。
