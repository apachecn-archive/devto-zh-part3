# 程序员的进步:在梦的模拟下交付

> 原文:[https://dev . to/jamesmh/the-programmers-progress-delivered-under-the-similarity-of-a-dream-2h2b](https://dev.to/jamesmh/the-programmers-progress-delivered-under-the-similitude-of-a-dream-2h2b)

这是一个有趣的部分重新讲述经典的“天路历程”作为一个寓言的程序员和软件开发人员的旅程。

在帖子 [@helenanders26](https://dev.to/helenanders26) [发表了将数据库比作芭蕾](https://dev.to/helenanders26/why-a-database-is-like-a-dance-class-1dmi)之后，我脑海中出现了一些有趣的想法。

这只是抓痒😋。

# [](#a-similitude-of-a-dream)梦的相似之处

当我在一整天的会议和代码审查后躺下休息时，我做了一个梦。

我梦见了——我看见一个男人(程序员)穿着牛仔裤和公司提供的帽衫。他手里拿着一本书，背上背着沉重的负担。

他打开书(我看得出马丁叔叔把它叫做《干净的密码》)，读着读着，他哭了，颤抖着。他再也控制不住自己，发出一声悲泣:“我该怎么办！”

他回家了，很苦恼。

几个小时后，他再也控制不住自己，向家人解释说:“我是因为我身上的负担而累垮的。我当然被告知，这个城市将被居住在 Reddit 的那些纯粹主义者的社会抗议之火烧毁！”

一天，当他走在田野里，痛苦不堪，不知所措的时候，我看到一个名叫开发商福音传道者的人向他走来，问道:“你为什么哭？”

程序员(P)回应，*“从这本书可以看出，我注定是一个初级开发者。然后死去。”*

开发者传道者(DE)回应，*“为什么不愿意死？既然大多数代码库都充满了幻数、10 级继承、名字像‘x’的变量和主键上缺少索引的数据库？”*

P: *“因为我害怕我背上的这个未提交代码的负担肯定会把我沉得比坟墓还低。”*

“那你为什么还不动？”

我不知道该走哪条路

德:*“你看见那闪亮的光了吗？跟着它走。”*

程序员尽可能快地跑。

# [](#leaving-the-city-of-destruction)离开毁灭之城

当他跑的时候，许多人嘲笑和戏弄他。他们要求他回到毁灭之城:程序员在那里积累了无尽的未提交代码和可怕的编码实践，如继承过度使用(以及其他无法形容的邪恶)。

其次，特别是固执和顺从。

固执的问程序员要去哪里，但他一看到这本书就恶心。他不相信用空闲时间学习更多编程概念的想法。

然而，柔韧的，留在程序员身边，问他什么样的事情等待着他。

程序员解释了没完没了的会议赠品，所有你可以喝的咖啡/浓缩咖啡，以及无限的假期。

柔韧道:“闻其言，足以夺其心！”

# [](#the-slough-of-technical-debt)技术债务的泥沼

当他们继续前行时，他们遇到了一个沼泽，深深地陷入其中——技术债务的泥沼。

没想到旅途如此艰难，柔韧回头了，把程序员留在了后面。

程序员到了沼泽的另一端，但是他出不来了。

一个名叫“有益导师”的人伸手把他拉了出来。

有益的导师说:*“这个泥沼是一个无法修补的地方:它是一个堕落的地方，伴随着“小快速修复”的浮渣和污秽不断运行，因此它被称为技术债务的泥沼；尽管如此，当软件开发人员意识到自己迷失的处境时，他的灵魂中会产生许多恐惧、怀疑和令人沮丧的忧虑，所有这些都聚集在一起，并在这个地方定居下来:这就是这块土地变坏的原因。”*

# [](#the-ivory-tower-architect-wiseman)象牙塔建筑师怀斯曼

程序员独自继续。

随着时间的推移，他遇到了另一个旅行者——象牙塔建筑师怀斯曼(ITAW)。

谁向你解释了你现在的生活方式？

P: *“一个叫开发者福音传道者的绅士。”*

我希望诅咒他的建议！没有比他让你走的路更危险的路了！你不是已经遇到技术债的泥沼了吗？”

*“听我说；我比你大:在你走的路上，你会遇到疲倦、痛苦、饥饿、危险、赤裸、刀剑、狮子、龙、黑暗，一句话，死亡，等等。这些事情肯定是真的，已经被许多证词证实。”*

P: *“但是这个负担比我一年前写的任何未提交的重构和代码都要糟糕得多。”*

ITAW: *“你是怎么知道的？”*

鲍勃叔叔写的这本书——干净的代码

ITAW: *“我也这么想；它发生在你身上，就像其他弱者一样，他们干涉对他们来说太高的事情，突然陷入你的分心……”*

看，在那个村子里有一个叫合法性的人，他会帮你摆脱负担

# [](#corrected-by-developer-evangelist)由开发者传道者纠正

他奋力向前爬下一座小山。

开发者福音传道者再次出现，这使得程序员对他接受象牙塔建筑师怀斯曼的建议感到羞愧。

Developer Advocate 解释了合法性如何从未解除任何人的负担——他强加给每个人的过于严格的林挺配置，他的应用程序的强制 5 层企业分层，等等。

程序员除了死亡什么都不想，开始悲伤地哭泣——甚至诅咒他与象牙塔建筑师怀斯曼先生会面的时间，仍然称自己是一千个听从他建议的傻瓜。

他再次听从开发者福音传道者的建议，继续缓慢前进。

# [](#the-interpreters-house)译员之家

然后，程序员来到一个名叫口译员的人家里。

程序员解释了他是谁，而开发者传道者告诉他解释者将向他展示许多精彩东西。

## [](#a-room-consumed-with-dust)被灰尘吞噬的房间

首先，翻译带他看了一个满是灰尘的房间，这些灰尘从未被清扫过。然后他叫了一个人开始打扫。

程序员开始咳啊咳——差点呛死。

P: *“这是什么意思？”*

I: *“这个客厅是一个遗留代码库的核心，它从未因采用敏捷工作流而变得神圣。这些灰尘是他的数据库第一个代码造型，它们玷污了整个人。他那一开始横扫一切的，是符合 IEEE 标准的[软件审计](https://standards.ieee.org/standard/1028-2008.html)；但她带来了水，并洒了水，是领域驱动的设计。”*

## [](#passion-amp-patience)激情&耐心

然后他被带到第二个房间。我看见两把椅子，每把椅子上都坐着一个孩子。他们的名字是激情和耐心。

激情坐不住，耐心很平静，愿意等待。

然后我看到有人在激情之前拿来一袋财宝，倒在他脚边。他拿起宝贝，开始嘲笑平儿。

但我看了一会儿，他的宝贝很快就变成了破烂。

“给我详细解释一下这件事。”

我:*“这两个小伙子是人物；激情将拥有现在的一切，这一年，也就是说，在这冲刺中；这些人现在必须有他们所有的新特征；他们不能等到下一次冲刺时才得到他们的那份好处。“双鸟在林不如一鸟在手”这句谚语对他们来说比所有可维护代码库的神证更有权威。但是当你看到他很快挥霍一空，目前留给他的除了破衣烂衫，所以这将是所有这样的人不采用持续的整合。”*

## [](#the-man-in-the-iron-cage)铁笼子里的人

接下来，他被带到一个黑暗的房间，人被关在一个铁笼子里。

这个男人看起来很悲伤。他坐着，眼睛向下看着地面，双手合十，叹息着，仿佛要心碎。

P: *“这是什么意思？”*

翻译鼓励他和那个人交谈。

然后程序员问，*“你是什么？”*。

那人回答说:“我现在不是从前的我了。”

在我自己和其他人的眼里，我都曾经是一名优秀的教授。

“我现在是一个绝望的人，被关在里面，就像被关在这个铁笼子里。我出不去；哦，现在我不能！”

“但是你是怎么变成这样的？”

“我已经把自己关在所有的承诺之外，现在我只剩下威胁，可怕的威胁，某些判断的忠实威胁和强烈的愤怒，它们会把我作为敌人吞噬掉。”

“你为什么会让自己陷入这种境地？”

M: *“为了试图掌握和维持对所有前端框架的掌握的私欲、享乐和利益；我当时确实承诺过要尽情享受，但现在这些东西都像燃烧的虫子一样咬着我，啃着我。”*

然后翻译对程序员说:“让这个人的痛苦被你记住，并成为你永远的警告。”

P: *“这太可怕了！上帝帮助我警醒和清醒，祈祷我可以避开这个人的痛苦的原因。先生，我现在不是该上路了吗？”*

# [](#freedom)自由

他继续前行，来到了一座小山上——在那座小山上矗立着敏捷宣言。

所以我在梦里看到，就在程序员走向敏捷宣言的时候，他的负担从肩膀上松开了，从背上掉了下来，开始翻滚，如此继续下去，直到再也看不见为止。

然后程序员高兴地跳了三次，继续唱歌！
