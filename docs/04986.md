# 从结束、开始、中间 Android/IOS 反应本地省略号文本

> 原文:[https://dev . to/skp tricks/react-native-ellipsis-text-from-end-start-middle-Android-IOs-1k C1](https://dev.to/skptricks/react-native-ellipsis-text-from-end-start-middle-android-ios-1kc1)

来源:[从 End、Start、Middle Android/IOS 反应原生省略号文本](https://www.skptricks.com/2019/04/react-native-ellipsis-text-from-end-start-middle.html)

本教程解释了如何在 React Native 中创建省略号剪辑文本。我们将向您解释和展示如何在 react 本地文本组件的不同位置添加省略号。省略号是 CSS(缓存样式表)的一种属性，用于剪切特定区域中的文本。当文本穿过该区域时，省略号会剪切文本，并在文本末尾显示…(三点)格式。在 CSS 中，它是文本溢出的一个属性。因此，在本教程中，我们将在 React 原生 iOS Android 应用程序完整示例教程中创建省略号剪辑文本。在 React Native 中，我们可以使用 numberOfLines = { 1 } prop of Text 组件来设置默认省略号。这个道具允许我们在文本上实现省略号剪辑效果。
