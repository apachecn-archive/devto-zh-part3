# 你的“程序员/语言适合度”是什么

> 原文：<https://dev.to/ben/what-is-your-coderlanguage-fit-nn7>

在科技创业行话中，你会经常听到这样的约定:

### 产品/市场契合度

### 创始人/初创公司契合度

### 产品/方正合体

### 等。

它的基本思想是，一个产品不会天生成功，但如果它找到了合适的市场，事情就会变好。或者说没有一个创始人天生伟大，但是有了合适的项目，他们真的可以成功。

那么你的程序员/语言适合什么？适合你的语言是因为你是一个怎样的人——而不一定是因为它内在的伟大。没有一种语言是没有瑕疵的，也没有一种语言是完全糟糕的。这是关于与项目的契合度，或者在这个问题中是关于人的契合度。
