# 寻找工作-学习-训练的平衡...

> 原文：<https://dev.to/konfistador/searching-for-work-study-training-balance-47da>

亲爱的 Dev 伙伴们，
在过去的几个月里，有件事一直困扰着我。就我 18 岁的年龄来说，我认为自己是一个非常活跃的人。
我在高中读高三(不幸的是，我们国家的教育系统太差了，所以我们在学校不会无所事事，**实际上什么都不做**)，除此之外，我还做兼职，我正试图通过举重和节食来保持健康的生活方式。

从我五年级开始，技术和编码就是我热情和未来计划的重要部分。大约一年前，我开始在 TeamTreehouse 在线编码学校上课。在那里，我从 iOS 开发开始，后来转向全栈 JavaScript。

所以，长话短说，我觉得从那里获得的所有知识都是没有实践的知识。说到实践，我不知道如何将所有的知识付诸行动。

所以这里是第**个问题** - **你能在学习时提出有效的练习方法吗？**

我的第二个问题是，由于我有很多事情要做，只是因为我选择了积极的生活方式，所以我经常会没有精力(有时是时间)或可能专注于编码。一个很好的例子是学校——因为我们不是整天无所事事，每个人都在教室里大声说话和大笑，这让我很难也不可能专注于一个问题。

那么第二个问题来了——酷人们，你们能不能建议一种方法来提升这种能量，用于真正的学习，在这种环境下有没有一种方法可以集中注意力？*

这是我第一次接触社区，所以我真的希望有人能帮我解决这些问题。

感谢您的宝贵时间！

宋旻浩:)
