# 工具

## Web 框架加固

*   阿帕奇·希罗——一个强大且易于使用的 Java 安全框架，执行认证、授权、加密和会话管理。
*   Java JWT:用于 Java 和 Android 的 JSON Web 令牌。
*   OWASP ESAPI Java-Enterprise Security API 是一个免费、开源的 web 应用程序安全控制库，使程序员更容易编写低风险的应用程序。
*   [PAC4J](https://github.com/pac4j/pac4j)-Java 安全引擎，用于验证用户身份、获取用户资料和管理授权，以保护 web 应用和 web 服务。
*   Spring Security——一个强大且高度可定制的认证和访问控制框架。
*   [Spring Security Oauth](https://github.com/spring-projects/spring-security-oauth)——支持为 Spring web 应用程序添加 OAuth1(a)和 OAuth2 特性(消费者和提供者)。
