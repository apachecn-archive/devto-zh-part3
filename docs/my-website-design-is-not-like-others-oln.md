# 我的网站设计不像别人

> 原文：<https://dev.to/mehprog/my-website-design-is-not-like-others-oln>

当我创建一个网站或类似小脚本的东西时，我总是问自己这个网站的代码是好的还是只是我的想象。我的网站看起来很好，对我来说很神奇，就像我在创建它之前所想的那样，但我总是认为其他网站是如此之好，正是他们的代码...有时当我在网站上看到一个设计时，我知道我能做到，但当我看到他们的代码时...这与我的代码非常不同，他们用另一种方式编写类 div 和其他元素。
我怎么知道是我的代码好？这一定像他们的代码？
当我练习的时候，我可以做很多事情，但是当我看到一个网站用另一种方式编码并且有很多细节的时候，我对自己有问题...
