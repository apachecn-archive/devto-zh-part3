## [](#run-the-http-server)运行 HTTP 服务器

我们可以运行以下命令来安装该工具:

`dotnet tool install --global dotnet-serve`

之后，可以执行命令来运行 HTTP 服务器:

`dotnet serve -o -S`

`-o`参数触发默认浏览器，`-S`参数提供 HTTPS 连接。
