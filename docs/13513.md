# 模因

一个用 JavaScript 编写的带有迷因、视频和 gif 的 PWA 应用程序。(Reddit 解析器)

现场版[此处](https://victorribeiro.com/memes)

# 关于

一个 PWA 应用程序，解析 Reddit 的内容，没有广告或评论。
由于该应用运行在客户端，服务器几乎没有任何成本(流量和存储)。

# 如何添加 subs

修改 main.js 文件上的 *subs* 变量，以添加或删除您想要的任何 sub。

```
let subs = [
    "funny",
    "dankmemes",
    "videos",
    "gifs"
]; 
```
