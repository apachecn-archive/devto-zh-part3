# 有人还在用 Emacs 吗？

> 原文:[https://dev.to/slick3gz_/anyone-still-using-emacs-13b1](https://dev.to/slick3gz_/anyone-still-using-emacs-13b1)

我使用过一些代码编辑器和 ide。过去几年一直在断断续续地使用 VSCode。这些年来我用过的其他一些工具包括 DEV-C++、Eclipse、Atom、代码块、Notepad++和 Visual Studio，还有一些我不记得了🤔。最近，我对 Emacs 产生了新的兴趣。我以前尝试过，但并没有真正投入时间去学习如何使用它。出于某种原因，我昨天早上醒来，决定跳上 Emacs🚂。我知道人们选择站在哪一边，并且非常专注于他们的代码编辑器或 ide。真的想知道有多少开发人员在他们的工作流程中一直使用 Emacs？
