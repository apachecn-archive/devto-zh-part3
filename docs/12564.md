# 以身作则

> 原文:[https://dev.to/craignicol/leadership-by-example-3h19](https://dev.to/craignicol/leadership-by-example-3h19)

你是经理还是领导？

你为你的团队树立了什么样的行为榜样？你会在下班后发很多邮件吗？你在会议中会一心多用吗？你是否对高管层有挫折感和异议？

你的团队会模仿你吗？

如果你发邮件发得太晚，除非你清楚原因，否则每个人都需要检查。告诉他们你每天必须提前一个小时出发去学校，这样你就可以在睡觉后发邮件给大家，让他们早上来取。更好的是，晚上写，早上再发。有很多工具可以帮你安排。

如果你不专注于会议，你怎么能指望你的团队呢？如果你不需要去，那就不要去。[如果你信任你的团队](https://dev.to/craignicol/becoming-a-technical-lead-trust-your-team-dff)并且你不会增加价值，那就退后一步。[如果你能改变会议，让它不无聊，那就去做吧。](https://craignicol.wordpress.com/2017/09/08/flightplan-how-to-run-a-meeting/)开会时，要在场，即使只是注意肢体语言，参与其中，利用反馈。

[你的团队是否对决策感到沮丧，并经常责怪他人](https://craignicol.wordpress.com/2015/11/17/handling-disagreements-as-a-team/)？那是因为你也这么做吗？你不会同意每一个决定，但是如果做出决定的时候你在房间里，或者你的工作是传播它，平静地接受这个决定。如果你不能忍受，还有其他渠道，包括离开，但是你的团队决定应该得到尊重。当环境变化时，他们可以被辩论、审查和改变，但是[如果决定不能被集体尊重，你就没有团队](https://craignicol.wordpress.com/2016/09/27/everyone-has-a-voice/)。
