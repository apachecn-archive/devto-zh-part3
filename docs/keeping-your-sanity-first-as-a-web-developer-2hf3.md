# 作为一名 Web 开发人员，首先要保持头脑清醒

> 原文：<https://dev.to/flippedcoding/keeping-your-sanity-first-as-a-web-developer-2hf3>

在快节奏的环境中工作会让你付出代价。当你同时处理多个敏捷项目时，不知所措是必然会发生的。不管你有多有才华或者有多少经验，每个人在需要休息之前，他们能做的工作都是有限的。

不要让你的工作损害了你的心理健康，因为他们不是必须解决这个问题的人。web 开发中的倦怠是一个巨大的问题，没有足够的人在到达那个点之前休息一下。如果你觉得你需要把你的电脑(或其中一个用户)从屋顶上扔下去，也许是时候尝试一些事情了。

## 请几天假

你休假是有原因的。我知道要放弃那些宝贵的时间有多难，但有时你必须为自己这么做。你不需要计划一个奢侈的假期。你不需要任何计划。去家里坐一两天，放松一下。

有时候你只是需要在快节奏的生活中休息一下。尤其是如果你一直从事短跑运动的话。好像它们永远不会真正结束。放慢速度，把时间从工作中抽出来，真的会帮助你充电。

这里的关键是**而不是**利用这段时间来强调你可能会落后于什么。将自己与 sprint 以及与之相关的一切(和所有人)完全断开。花时间关注与冲刺无关的事情。我知道这说起来容易做起来难，但是你能做到。如果你在一个不喜欢休假的地方工作，那么是时候开始寻找其他地方了。

## 下班后不要回到电脑前

我们基本上都是科技迷。我们很多人生活和呼吸都离不开电脑，也许真的如此。🤷‍♀️:但是当你濒临崩溃的边缘时，看电脑是你最不需要做的事情之一。这意味着不做最后一件事，不研究你即将解决的问题，什么都不做。

关闭笔记本电脑并后退。你有工作时间是有原因的，这个原因不包括延长工作时间。他们的存在是为了帮助你和你的雇主设定界限。一旦你投入了一周的时间，你就完成了。

在午夜解决这个复杂的问题可能感觉很好，但当你已经工作了 9 个多小时时，这并不总是对你的心理健康有好处。你需要自己的时间为自己做些事情，放松一下。当你觉得你可以“只做一件事”的时候，可能是时候停下来了。无论你在冲刺的哪一部分，早上都会在那里。

## 不要害怕对你的经理说不

这可能会令人生畏，因为对你的经理说不很奇怪。不过你必须习惯这一点。如果你不说点什么，人们会尽可能地增加工作量。不幸的是，对于 web 开发人员来说，同时处理多个项目或速度惊人的冲刺并不罕见。

如果你很难对一般人说不，这对你来说可能真的很难。当你的工作接近你的精神极限时，有一件重要的事情你需要记住。你是唯一一个对你的生活负责的人。大多数公司都吹嘘他们的工作/生活平衡，但真正做到这一点取决于你。

如果你热爱自己的工作，那就太好了。我知道我绝对喜欢做这些事情。这不是重点。一天结束时，工作仍然是工作，你需要能够拥有工作之外的生活，而不要把自己的极限推得太远。当你注意到工作越积越多，他们又找你要更多的时候，是时候说不了了。

## 做一些完全与科技无关的事情

当你沉浸在代码的世界中时，很容易忘记你可以做的所有其他事情，甚至不涉及计算机。想想你成为 web 开发者之前的爱好，不要让它们消亡。当然，学习新的框架和代码是很棒的。这并不是生活的全部。

你必须时不时地把自己拖出科技世界，以确保你仍能像一个相对正常的人一样运作。你喜欢做饭，那就多做做。学习如何演奏尤克里里，或者只是坐在外面看起来很疯狂。

写代码很有趣，是一个很酷的爱好，但是当你感到精疲力竭时，是时候离开了。在你恢复了失去的理智之后，你就可以回到你的代码中去了。尽管现在，你的大脑需要休息。做一些对你大脑不同部分起作用的事情，并在这个过程中学习一些很酷的新技能。

## 走到外面

一般来说，阳光和新鲜空气对 web 开发人员或程序员来说是难以捉摸的。我们中的大多数人在任何有黑暗主题的东西上工作，而且由于我们的工作和家庭环境，我们大部分时间都在室内。你不必做任何剧烈运动或跑酷来获得户外运动的好处。

绕着你准备扔掉电脑的大楼走一圈(在一楼)。坐在人行道上或公园的长椅上。呆在户外会让你暂时忘却烦恼，让你意识到你盯着屏幕看了多长时间，而你的眼睛却永远无法适应阳光。

## 保重自己的身体

一旦你进入了编码的最佳状态，你就可以到处修正错误，时间很容易就过去了。这就是为什么你在同一个地方坐了 5 个小时，不动，不吃，不喝。我们都知道这有多糟糕。

至少每小时起来走动一次。你不必去做全面的锻炼，即使那会很棒。只是不要忘记偶尔搬家。试着在你的办公桌上放一些健康的零食和其他我们知道应该做但没有做的东西。

在这个领域，你的大脑是你最大的财富，但是如果你的身体没有得到很好的照顾，它就不会一直这样。另外，你会觉得自己像个废物。

筋疲力尽是人们离开 web 开发的一个重要原因。这些是我在过于频繁地筋疲力尽后，不得不艰难地学会的一些事情。请不要让自己走到那一步。还有谁想补充什么吗？
