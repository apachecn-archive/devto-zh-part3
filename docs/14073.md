# [](#fyre-not-the-festival)Fyre(非节日)

除了课程之外，我这周没有做太多事情，所以这里有一个关于 [Fyre](http://fyre.navi.cx/) 的简短帖子——一个我在浏览 pamac 时发现的小型计算艺术应用。它允许你探索彼得德容地图方程的不同混沌版本。它有一吨不同的设置和数字玩。它甚至有一个动画窗口，所以你可以制作 gif 或移动形状的短视频。在过去的几个星期里，我用它玩了一会儿，我发现它们可以制作有趣的个人资料图片。这里是我用这个应用程序制作的一些图片。
