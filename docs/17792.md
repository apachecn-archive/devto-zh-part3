# 允许忘记

> 原文：<https://dev.to/m1guelpf/permission-to-forget-6p9>

我是一个对想法着迷的人。每当我有一个项目想法，我就不能停止思考它几天。例如，我非常清楚如何实现逻辑以使 Ship 或 Source 工作，如何设计着陆以及如何营销。这还是在开始编码之前。

这对我的生活产生了一些影响。有时候我上床睡觉，然后花三个小时思考项目，最后才睡着。这也让我很难专注于不相关的事情(比如学校)，除了我的想法之外，我对任何事情都没有生产力。

我最近找到了解决这个问题的方法。如果我一想到我的想法就马上写下来，我就给了我的大脑“忘记”我的想法的许可，我就可以继续思考了。

编码也是如此。如果你在编码的时候写下你的想法，而不是把它们记在脑子里(例如，你看到一些代码，然后想 hmmm 我应该以后再重构它。不要只是记住并记在脑子里，而是把它写下来，以后再回过头来看)你减少了精神上的努力，甚至可能减少了精疲力尽的可能性。
