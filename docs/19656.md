# 是什么让公司文化变得伟大？

> 原文:[https://dev . to/kosnocorp/what-makes-a-company-culture-great-ndb](https://dev.to/kossnocorp/what-makes-a-company-culture-great-ndb)

企业文化对企业的成功起着巨大的作用。无论产品有多酷，问题有多有挑战性，如果工作环境是有毒的，员工会找到另一份工作。企业可能会尝试使用财务动机，但这不是一个秘密，这是创造性职业最糟糕的动机。吸引和留住顶尖人才的唯一方法是提供一个允许他们实现生活和工作平衡的工作环境。

但这是什么意思呢？我决定找到这个问题的答案，所以我建立了[文化契合度](https://cultural.fit/)。这是一个人们喜欢或不喜欢的企业文化特征的协作列表。
