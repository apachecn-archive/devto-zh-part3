# 这是我的第一个帖子

> 原文:[https://dev.to/jgaguiar16/this-is-my-first-post-15de](https://dev.to/jgaguiar16/this-is-my-first-post-15de)

嗨，我的名字是耶稣，我来自委内瑞拉，目前我住在乌拉圭学习软件工程。现在我的职业生涯已经过半，4 个月前我开始寻找我的第一份工作，我找到了一份，作为一名 web 开发人员，你猜怎么着，我对 web 开发没有任何概念，只有基础知识，html，css，bootstrap，但是我不知道后端，节点，express，mongo 等等，什么都不知道。

所以，我在公司的第一个月开始了，我紧张得要命。起初，我从学习我所需要的课程开始(他们对我的意图是让我成为一名 MERN 堆栈开发人员)，所以整整一个月我学习了所有必要的东西。

在我第二个月的时候，我开始了我的能力培养项目，一个将我从课程中学到的东西付诸实践的简单项目。事情发生了，我花了很长时间才完成，几乎 2 个月，这并不好，我的主要问题之一是我在编码时犯了很多错误，这些错误花了我很长时间来调试它们，其中大部分是在后端，但我学到了很多，这对我来说是重要的。我的老板们对我的处境很不满意，直到今天他们一直以某种方式告诉我这一点。

我即将开始我的第四个月，上周我和一个客户开始了一个项目，一个简单的项目，我尽了最大努力，但我的老板上周告诉我，在他们看来我还没有准备好参加一个更大的项目，我应该更加努力地工作(我正在做的事情)，所以，我不知道，这是我的第一份工作，我不害怕失败， 我希望我能完成这个项目并按时完成，但是如果我做不到，我不知道我会有什么感觉，我希望我能从那种情况中学到很多。

好吧，这是我的第一个帖子，如果 a 犯了语法错误，我很抱歉，我希望你读了它，我不知道，是快乐和快乐的编码！
