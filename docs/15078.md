# 寻找投稿人的项目:ClassicPress

> 原文:[https://dev . to/nylen/project-寻找贡献者-classicpress-511h](https://dev.to/nylen/project-looking-for-contributors-classicpress-511h)

ClassicPress 是久经考验的 WordPress 4.9 分支的一个分支，有经典编辑器，没有古腾堡。你可以在[我们的路线图](https://www.classicpress.net/roadmap/)上找到更多关于我们为什么决定放弃 WordPress 以及我们未来几个版本计划的信息。

我们目前的版本是`1.0.0-beta2`和**,我们正在寻找开发者的帮助**,以便尽快推出`1.0.0-rc1`和`1.0.0`版本。你可以在 GitHub 的 [`1.0.0-rc1`里程碑上看到我们当前的任务列表。](https://github.com/ClassicPress/ClassicPress/milestone/5)

如果您有任何问题，请随时就 GitHub 的个别问题或我们的 Slack group 的[发表意见。`#core`频道是谈论 ClassicPress 自身发展的好地方。](https://www.classicpress.net/join-slack/)
