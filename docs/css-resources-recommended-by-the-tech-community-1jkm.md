在过去的 3 年里，我和 CSS 一起工作了很长时间，并且对许多我以前没有见过的资源感到惊讶。这里是技术社区共享的资源列表。

### 没见过的:

[日常 CSS 设计](https://dailycssdesign.com/)由比约恩·弗杰尔斯塔德在[运球](https://dribbble.com/bear-bjorn/)和 [Instagram](https://www.instagram.com/dailycssdesign/)
这是一个华丽的 CSS 灵感集合。比约恩的目标是在一年的时间里每天创造一个设计。

[科迪豪斯](https://codyhouse.co/) [@codywebhouse](https://twitter.com/CodyWebHouse) 在推特上。
这是一个 CSS 框架和可访问的 HTML，CSS，JavaScript 组件库。网站上有一些不错的文档和例子。

杰瑞米·托马斯 [@jgthms](https://twitter.com/jgthms) 在推特上。
关于 CSS 所有东西的综合参考。它有很多很棒的视觉效果。

亚当·马斯登 [@AdamMarsdenUK](https://twitter.com/AdamMarsdenUK) 在推特上发布的 CSS 备忘单。
CSS 善良的参考。

GitHub 上 [atomiks](https://github.com/atomiks) 的 30 秒 CSS 。
一个精选的有用的 CSS 片段集，你可以在 30 秒或更短的时间内理解。

### 更多精彩资源提及

我的最爱之一，也是我在课程中经常用到的东西。CodePen 对于开发者来说就像 Dribbble。它是一个面向任何技能的设计人员和开发人员的在线代码编辑器，特别适合学习编码的人。

[CSS 视觉词典](https://www.amazon.com/CSS-Visual-Dictionary-Greg-Sidelnikov/dp/1983065633)作者 [@js_tut](https://twitter.com/js_tut) 在 Twitter
上它有超过 250 个视觉图，解释了目前常用的所有主流浏览器支持的所有 CSS 属性和值。

Greg 在网上有很多很棒的资源，他经常免费与社区分享。你可以在 Twitter 上关注他，了解他的最新作品。

[CSS 网格生成器](https://cssgrid-generator.netlify.com/)作者莎拉·德拉斯纳[@莎拉 _ 埃多](https://twitter.com/sarah_edo)推特
它是一个为你生成 CSS 网格代码的可视化小工具。您可以指定行、列、间隙和单位，然后拖动以创建子 div，从而轻松地制作动态布局。

由 [@playcodepip](https://twitter.com/playcodepip) 在推特
上发布的 Flexbox Froggy 一个游戏化学习的网站。Flexbox Froggy 是一个游戏，你通过编写 CSS 代码来帮助 Froggy 和朋友。

一个致力于为网页设计者和开发者提供有用的教程、有见地的文章、创作灵感和免费资源的网站。

他们还有一个[代码游乐场](https://tympanus.net/codrops/category/playground/)，里面有实验、插件和与 web 开发和编程相关的文章。

FreeCodeCamp 是一个非营利组织，帮助数百万人免费学习编程。课程中有几个部分是专门针对 CSS 的。

Lea Verou 是一个非常鼓舞人心的人，值得在 Twitter 上追随。她有丰富的信息，包括她在自己的书 [CSS 秘密](https://www.amazon.com/CSS-Secrets-Lea-Verou/dp/1449372635?tag=leaverou-20)中分享的东西

如果你真的关心 CSS 网格，你应该跟随瑞秋。查看她的网站[网格示例](https://gridbyexample.com/)

Jen Simmons 如果你喜欢网格或网页布局，这是另一个值得关注的人。她在她的网站[实验布局实验室](https://labs.jensimmons.com/)和她的 YouTube 频道[布局土地](https://www.youtube.com/channel/UC7TizprGknbDalbHplROtag)上有很多惊人的例子

[韦斯·博斯的课程](https://cssgrid.io/)
[韦斯·博斯](https://twitter.com/wesbos)有很多精彩的在线内容，从课程到他的播客[句法 FM](https://syntax.fm/) 和共同主持人[斯科特·托伦斯基](https://twitter.com/stolinski)

Jonas Schmetmann Jonas 在 Udemy 上有一个很棒的[高级 CSS 和 Sass](https://www.udemy.com/advanced-css-and-sass/?couponCode=TWITTER4) 课程。当人们要求学习 CSS 课程时，我总是看到他被提起。我亲自上过他的课，非常喜欢。
