## 文章不再可用

我只做过一次“五大”面试，我更像是一个创业者，我无法想象自己会在一个面试中工作！话虽如此，[这里](https://github.com/aspittel/coding-cheat-sheets)是我两年前准备那次面试时做的一些笔记！我也强烈建议你对图遍历问题感到非常舒服，因为这些问题现在看起来非常流行。也就是说，这是我第一次经历技术面试(直接进入了最深处！)而且没做太过分！

我也被几个五大公司招聘过几次。一个(我认为)是因为我在谷歌的[代码堵塞比赛](https://codingcompetitions.withgoogle.com/codejam/)中表现不错(那是我唯一一次接受面试)，我还通过搜索 Python 的东西被邀请到 [FooBar](https://medium.com/magentacodes/things-you-should-know-about-google-foobar-invitation-703a535bf30f) ，所以如果我想的话，我可以完成挑战并通过面试。由于我的博客和开源贡献，我也被其他五大巨头通过我的 LinkedIn 招募。

再说一次，我可能不是这里给出建议的最佳人选，但我会尽我所能，并希望其他人也能参与进来！
