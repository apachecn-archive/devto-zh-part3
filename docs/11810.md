# 把我的职业生涯押在 JAMstack 身上。

> 原文:[https://dev . to/liltechnomancer/betting-my-career-on-jam stack-4d ef](https://dev.to/liltechnomancer/betting-my-career-on-jamstack--4def)

新年伊始，我的发帖量有所下降，但那是因为我一直在忙着做一些事情..那些事就是这篇博文的主题！在过去的几个月里，我在 JAMstack meetups 的凤凰城分会教了一群人如何使用 Gatsby 和 Netlify，你可以通过访问 JAMstack 社区页面来看看你所在的城市是否有这样的网站！我还接受了一些客户的委托，用 Gatsby 制作了他们的网站！我甚至将 NetlifyCMS 包含在其中，这样客户就可以更新他们自己的内容，这样做是如此令人愉快，我觉得我期待已久的梦想终于实现了。我一直希望有一个静态网站的移交体验，好到足以让整个机构围绕着它，我感觉那个时代终于到来了。

说到这里，我想宣布几件事！

首先，如果你想建立一个网站或应用程序，你希望它快速、可靠，最重要的是，提供坚如磐石的用户体验，你应该通过 [jamstack.consulting](http://jamstack.consulting) 与我联系。我特别关注对 JAMstack 开放的客户端，不会接手使用传统 CMS 或服务器端 monolith 的项目。展望未来，我认为有足够的工作来建立一个专门从事这类工作的机构，我真的希望把 [jamstack.consulting](https://jamstack.consulting) 建设成一个伟大的机构。

第二，如果你想学习如何以这种方式建立网站和应用程序，你很幸运！我还在为你制作 jamstack.courses ！第一门课将是围绕 Gatsby JS 的最令人惊叹的 web 框架的全面介绍！你将把个人博客和作品集放在一起，它们将以光速加载，让你大吃一惊。一定要留下你的电子邮件，这样我可以在有空的时候通知你。

如你所见，我已经将我的努力集中到一个焦点上。詹姆斯塔克。我渴望在未来的博客文章、课程和产品中向你传授这一切。我认为网络正在发生根本性的变化，我想记录我对更好、更现代的网络建设方式的探索。
