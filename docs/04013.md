# 我的 TBR 排名前五的书

> 原文：<https://dev.to/ashleemboyer/the-top-five-books-on-my-tbr-3972>

我讨厌一次读多本书，但我现在读了三本。我想？我甚至不能确定。这就是为什么我要练习自制力，一次只做一个。

自从发现我有多喜欢研究用户体验和可访问性之后，我遇到了几本不得不添加到我的书单上的书。有些是关于 UX 和设计，有些是关于创造和生产力。

对于每一个，我都链接了亚马逊列表，还粘贴了概要。我爱，爱，爱阅读，所以如果你有任何自己的推荐或清单，请*分享它们！我已经满满的书架不会欣赏它，但我肯定会。😊*

# 1。迈克·蒙泰罗的 [*毁于设计*](https://www.amazon.com/Ruined-Design-Designers-Destroyed-World/dp/1090532083/ref=sr_1_1?keywords=ruined+by+design&qid=1557279824&s=gateway&sr=8-1)

世界完全按照设计运行。内燃机正在摧毁我们星球的大气层，并迅速使它变得不适合居住，它完全按照我们的设计工作。导致如此多死亡的枪支，完全按照它们的设计发挥作用。每次我们“改进”他们的设计，他们就更擅长杀人。脸书的隐私设置，向他们保守的父母揭露了同性恋青少年，完全按照设计的那样工作。他们的“实名”倡议，使跟踪者更容易重新找到他们的受害者，正如设计的那样工作。Twitter 的毒性和缺乏文明的话语正如其设计的那样发挥作用。世界完全按照设计运行。而且效果不是很好。这意味着我们需要更好地设计它。设计是一门拥有惊人力量的手艺。选择的权力。影响力。作为设计师，我们需要把自己看作是我们把什么带入这个世界，以及我们选择不把什么带入这个世界的看门人。设计是一门有责任的手艺。帮助所有人创造一个更美好世界的责任。设计也是一门双手沾满鲜血的手艺。每一个香烟广告都在我们身上。所有的枪都指向我们。选民看不懂的每一张选票都在我们身上。每次社交网络的界面允许跟踪者找到他们的受害者，那都是我们的责任。我们释放到这个世界上的怪物将以你的名字命名。这本书会让你明白设计是一种政治行为。我们选择设计的是一种政治行为。我们选择为谁工作是一种政治行为。我们选择与谁合作是一种政治行为。最重要的是，我们从这些决定中排除的人是我们作为一个社会所做的最大的(也是最愚蠢的)政治行为。如果你是一名设计师，这本书可能会让你生气。这应该会让你生气。但它也将为你提供做出更好决策所需的工具。你将学会如何评估你正在做的事情的潜在好处和坏处。你将学会如何表达你的担忧。你会学到建立和与不同团队合作的重要性，这些团队可以从多个角度处理问题。你将学习如何使用数据和良好的故事来制作一个案例。你将学会用一种让别人倾听的方式说不。但最重要的是，这本书会让你充满信心，以你一直想要的方式去做工作。这本书会帮助你理解你的责任。

# 2。Erika Hall 的[T3 刚够研究 T5】](https://www.amazon.com/Just-Enough-Research-Erika-Hall/dp/1937557103/ref=sr_1_fkmr0_1?crid=225WHYZ7RCHMQ&keywords=erika+hall+just+enough+research&qid=1557280061&s=gateway&sprefix=erika+hall+just+enou%2Caps%2C165&sr=8-1-fkmr0)

设计研究是一项艰苦的工作，需要花费数年的时间来学习，并且远离真正的设计工作，对吗？不对。好的研究是问更多更好的问题，并对答案进行批判性思考。这是你团队中的每个成员都可以而且应该做的事情，而且每个人都可以很快学会。如果做得好，它将通过减少未知因素，为以最有效的方式构建正确的事物奠定坚实的基础，从而节省您的时间和金钱。在足够多的研究中，Mule Design 的联合创始人 Erika Hall 将她的经验浓缩成一本关于研究方法的简明食谱。学习如何发现你的竞争优势，发现你自己的盲点和偏见，理解和利用你的发现，以及为什么你永远不应该，永远不要举行焦点小组。你会比计划下一次推销更快地开始做好研究。Erika Hall 自 20 世纪末以来一直从事网页设计和开发工作。2001 年，她共同创立了 Mule 设计工作室，在那里她指导研究、交互设计和策略实践。Erika 经常谈论和写作跨学科合作以及自然语言在用户界面中的重要性。在业余时间，她与空洞的企业行话作斗争。她还共同主持了每周一期的商业法和耐力健身播客《逃离法律》,她可能会跑得比你快。

# 3。史蒂文·普莱斯菲尔德的 [*艺术之战*](https://www.amazon.com/War-Art-Steven-Pressfield-ebook/dp/B007A4SDCG/ref=tmm_kin_swatch_0?_encoding=UTF8&qid=&sr=)

把艺术之战想象成严厉的爱...为了你自己。从 2002 年开始，艺术的战争激励着世界各地的人们战胜“反抗”；认识并打破阻碍梦想的障碍，让我们内心的反对者闭嘴。反抗踢了每个人的屁股，打败它的欲望同样是普遍的。艺术之战确定了我们每个人都必须面对的敌人，概述了征服这个内部敌人的作战计划，然后指出如何取得最大的成功。虽然它是为作家写的，但它已经被世界各地的企业家、演员、舞者、画家、摄影师、电影制作人、军人和成千上万的其他人所接受。

# 4。劳拉·克莱因的[T3【UX】精益创业 T5】](https://www.amazon.com/UX-Lean-Startups-Experience-Research/dp/1449334911/ref=sr_1_2?crid=1ZYI5TIDLFFFJ&keywords=ux+for+lean+startups&qid=1557280161&s=gateway&sprefix=ux+for+lean+start%2Caps%2C177&sr=8-2)

伟大的用户体验(UX)对今天的产品来说是必不可少的，但是设计一个用户体验是一个漫长而昂贵的过程。通过这本实用的实践书籍，您将了解如何使用精益 UX 技术更快、更智能地完成这项工作。UX 专家劳拉·克莱因向你展示了如何从客户那里收集有价值的信息，打造他们真正喜欢的东西，以及缩短你的产品上市时间。

没有 UX 或设计的经验是必要的开始。如果你是一个企业家或创新者，这本书让你正确地使用成熟的技巧和工具来研究、识别和设计一个直观、易用的产品。

*   在你生产产品之前，确定人们是否会购买你的产品
*   在产品的整个生命周期中倾听客户的意见
*   理解为什么在设计产品之前应该设计一个测试
*   获得九种对设计产品至关重要的工具
*   辨别必要功能和理想功能之间的区别
*   了解最低可行产品如何影响您的 UX 决策
*   结合良好的 UX 实践使用 A/B 测试
*   加快您的产品开发过程，而不牺牲质量

# 5。杰瑞米·贝恩斯的&克莱夫·霍华德的[T3】UX 的生命周期 T5】](https://www.amazon.com/UX-Lifecycle-business-implementing-experiences/dp/1522768300)

这本书是为开始 UX 之旅的组织而写的。这将有助于解决一些基本问题，如界定 UX 是什么；研究的重要性；UX 是一个过程，而不是一个职位；商业价值来自于提高效率、有效性和满意度。对于建立业务案例和确定人员和流程中所需的关键投资，将带来实现成功所需的变革，有实用的指导。实施 UX 是通过 UX 生命周期实现的，这是一个源于现实生活成功的方法框架。UX 过程的核心是最重要的利益相关者——用户！Clive 和 Jeremy 帮助从初创企业到大型企业的企业从 UX 获得价值。在经营自己的 UX 咨询公司时，他们创建并应用了 UX 生命周期。从那以后，他们都通过实施 UX 解决方案和向他人学习来不断增长自己的知识。这本书将回答寻求拥抱和利用 UX 的企业一再向他们提出的问题。
