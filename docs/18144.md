# Android 的 HTTP 库

> 原文：<https://dev.to/shubhadeepb14/http-library-for-android-4kk9>

各位开发者好，

我开发了一个小的 Android 库来处理 HTTP POST 和 GET 请求。任何正在寻找执行 HTTP 调用的简单方法的人都可以使用这个库。它支持带有 JSON 和多部分表单数据的 post 请求、文件上传和带有 URL 参数的 Get 请求。

希望对你们有所帮助。
