# 再给我一次机会

> 原文：<https://dev.to/detunized/giving-go-another-chance-k85>

我决定开始记录我花在任务上的时间，这比用一支笔和一张纸要好一些。经过一番搜索，我发现了这个列表。我尝试了上面所有的工具，并坚持使用了一段时间[的 timetrap](https://www.linuxlinks.com/timetrap/) 。但是有些事情困扰着我。

首先，我希望能够添加标签，而不需要额外的解析或依赖`grep`。我还希望能够将数据库备份到一些集中的地方，或者更好地使用 git repo 作为后端。最重要的是，我想用一种新的语言编写一个简单的工具。

不久前我开始研究 Rust，我认为它会非常适合我。但是在找到我需要的箱子(图书馆)后，我决定再给 Go 一次机会。我以前在工作的时候用过它，它有点效果，非常适合那里。我也记得当时用了之后嘴里有一股苦味。所以我要再试一次，看看几年后是什么感觉。

我要记录我的步骤，并跟踪我在任务上花费的时间。开始吧！
