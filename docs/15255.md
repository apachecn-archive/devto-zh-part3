# 敬年轻的自己

> 原文:[https://dev.to/rinisimon/to-my-younger-self-4ola](https://dev.to/rinisimon/to-my-younger-self-4ola)

回顾过去的几年，我已经走了多远，从这个开发者世界的一个新手到现在能够把自己当成一个开发者，这段旅程给我上了人生的一课。

这里有几样东西是给刚开始工作的人的；那些我希望在我走进工作场所的第一天就能告诉年轻的自己的事情，当时我没有什么经验，一颗对代码的心，满脑子都是？？？

***态度决定一切。***
总的来说，我们对环境或生活的态度会塑造我们的人格，要么我们超越自我，要么屈服于环境。我们总是有一个选择，我们选择了什么就是什么，特别是在职业生涯的开始。

***把自我留在门口。***
记住没有什么工作是有失身份的。所以，不要害怕从底层开始。无论我们做什么或被要求做什么，都要做得很好；即使这意味着分拣邮件。

***我们都在不断地学习。***
嘿！错误时有发生，错误时有发生，不是每个人都知道世界上所有的事情。

那么，我们该怎么办？

承认错误，改正错误，从中吸取教训，不要再犯。不要陷入指责的游戏中。错误是学习的关键，我们如何对待错误才是最重要的。

***做好准备***
我喜欢这种陈词滥调

> ***“运气就是准备遇到机遇的时候”。
> 出自罗马哲学家塞涅卡。***

在接下来的一千年里都适用。

准备就是学习。学习并掌握基本知识，了解我们在不久的将来需要的下一个技术或工具。

人类是好奇的生物；想知道一切，但是

*   对我们有什么好处？
*   它将如何帮助我们提高技能？

具有洞察力并对它们进行优先排序是很重要的。从我们第二天需要知道的事情开始，然后建立更长远的目标才是要走的路。

***“不管你认为你能还是不能，你都是对的”。*T3】**

*   我够好吗？
*   我能做到吗？
*   他们会解雇我吗？

诸如此类，这是一条永无止境的怀疑之路，不断地在我们的大脑后面运行，我们只是在浪费我们的精力。

想想在健身房锻炼，一个人付出的努力越多，效果越明显。所以，继续努力；努力总是早晚一枝独秀，不断学习；没有人能夺走我们的学习。

其次，公司不会只为了解雇员工而雇佣员工。为了找到合适的人，我们付出了大量的努力和时间。唯一重要的是，“我们对公司有益吗？”因为归根结底，这是公司的事，而不是你我的事。

***不要纠结于别人怎么想。***
我们都有有限的时间和精力，花在这些上面的任何时间都不可能帮助我们实现目标，也不会让我们更快乐。

但是你可以做一些事情，例如

***问问***
我们都是来学习和完善自己的，你怎么知道*自己是变好了还是变坏了？*

自我反省是我们都需要发展的一种特质，一种真正评估自我的能力，但我们同时可以做的是问。要求一个诚实的反馈，在此之前，你要乐于接受它，无论是恭维还是批评。通常，我们抱怨批评，却没有把它视为反馈，而是积极地看待它，而不是针对个人。这只会帮助你成为一个更好的人。

最后，学会热爱自己所做的事情，你会享受其中的每一部分。认真学习，停止抱怨，因为抱怨不是前进的策略。T3】
