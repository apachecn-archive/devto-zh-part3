# 我的旅程

走进计算机科学中的离散数学课程，我感觉自己已经穿上了大男孩的裤子，正在超越简单的编码项目。我打算深入研究计算机科学概念，成为一名巫师程序员。我坐下来，设法通过一些集合论，几个星期后，我们开始谈论图形，迪克斯特拉的算法，归纳法证明，我看起来像这个家伙:

[![wtf meme](img/cf08f8d37a74226c9076d9df743fdd24.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--9L4WDRKt--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_66%2Cw_880/https://i.gifer.com/1K6U.gif)

我最终做了一个糟糕的决定。我把专业转到了信息技术(IT)。这不是一个糟糕的专业，只是它不是我真正想做的，所以对我来说这是一个糟糕的决定。对学习困难的热爱让位于生活的压力。

10 年后我在这里，所以帮助我，我决心理解我放弃的一切！多年来，我一直在构建实用程序和自动化。我了解系统、权衡、容器、Linux、架构、设计模式、测试、安全、项目管理、管理更大规模、设计模式、跨团队计划等等。我很感激这样的经历，但是我在编程能力上还有很大的差距。

最后，我回到了最初吸引我的技术，那就是深入研究计算机科学，理解幕后的数学，充分了解它以教授他人，提高我的能力，并构建更大更好的东西。对你们中的一些人来说，这可能听起来很无聊，但是我喜欢！今晚我正在实现 Dijkstra 的算法，以确保我理解它，我很激动！

[![excited gif](img/e189c7cc60bc3cd2b0a4f8b99c6853fe.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--lEf1B_FU--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_66%2Cw_880/https://media1.tenor.cimg/1f84b096cbe1cc9f3763c803bb17e10e/tenor.gif)

我的观点是，你应该专注于让你兴奋的事情。困惑是可以的。有些资源可以帮助你，我想分享一些。我所做的一切都不是浪费，我的注意力只是偏离了我热爱的事情，因为我放弃了一段时间。不再有了！我希望你也能专注于你热爱的事情！

外面的资源比以往任何时候都多。没有理由一个概念应该太难理解，因为在某个地方，有人已经用一种可能对你有意义的方式解释了它。你只需要找到它。我经常开玩笑说，我的工作是通过一本书或一个搜索引擎了解我的方法，而不是工程。

# 一些有用的学习资源

## 了解我们的大脑是如何工作的很重要

1.  《深度工作》——关于如何真正学习概念和想法以及如何有效利用时间的书。
2.  “[学会如何学习](https://www.coursera.org/learn/learning-how-to-learn)”是 Coursera 上的一门课程。我拿的时候是免费的。或者，《数字思维》这本书是同一个人写的，涵盖了大部分相同的观点。

## 编程和计算机科学资源

1.  《数据结构和算法常识指南》一书。如果你对这些概念比较陌生，或者很难理解它们，那么这本书就是为你准备的。我强烈推荐。
2.  [100 天代码计划](https://www.100daysofcode.com/) -好主意，我自己刚刚开始！如果你做到了，让我知道，我会跟踪你的进展。
3.  免费的代码校园 -如果你负担不起代码校园的费用，并且想学习 web 开发的基础知识，那就太棒了。
4.  [黑客排名](https://www.hackerrank.com)——对面试准备、学习一门新语言、加深对概念的理解有用。解决问题后，您可以切换到“讨论”选项卡，查看其他人是如何解决问题的。我重新实现了其中的一些解决方案，作为更好地学习我可能错过或忽略的一些概念的实践方法。
5.  Meetup -一个当地的社区或一群人可以提供更多的工作、实践和社交机会。谁知道你附近可能会有一个技术小组，或者你自己也可以成立一个！你不一定要成为一个专业人士才能开始或加入一个。只要愿意学习，让事情发生。
6.  当然还有[Dev](https://dev.to)——这是一个很棒的社区！

外面还有数十亿的其他资源。如果你迷路或困惑，要求澄清或尝试其他资源。在某个地方你会发现一些东西。

你最喜欢的资源是什么？哪些对你帮助最大？
