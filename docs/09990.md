各位开发者好！

我想向你介绍我最近开通的博客 [My Web Development Odyssey](https://mywebdevelopmentodyssey.dev) ，在这里我写下了我自学成为一名 Web 开发人员的个人经历。

我是马文。大约三个月前，我开始断断续续地学习 web 开发，不过现在我已经有了一个计划，并一直坚持下去。到目前为止，大部分是。这个网站是关于我在几个非常重要的背景下学习 web 开发的思考和过程。

虽然我有科学学位，但不是计算机科学。这里没有正规的编码教育。知道了这一点，我作为一个职业改变者寻找信息也就不足为奇了。我也是作为一个四十多岁的中年人进入竞技场的。作为一个大半辈子都在和抑郁症打交道的人。最近的一次衰退导致我“残疾”，好几年没有工作。尽管大多数人不会注意到，我也是自闭症谱系中的高功能者。虽然这导致了一些复杂的情况，但它也有许多优点。

所以，我的牌似乎非常不利。这让我担心吗？是的。我不可能看到有利的就业前景吗？一点也不！你可以在这里找到更多关于我的[。](https://mywebdevelopmentodyssey.dev/about-me/)

## 我将会写些什么

我将谈论全栈 web 开发的许多事情。我打算学什么。我将如何着手去做，以及它是如何工作的。我会写一些我理解的和不理解的东西。我将分享关于一般学习技巧的想法，以及它们如何帮助我。

我将包括的其他主题将与职业相关。随着我预计的就业日期临近，需要记住的事情。在这一点上，我作为初级网站开发人员的暂定就业日期是 2019 年 12 月 31 日。只剩不到 10 个月了！当然，这是可以改变的，因为生活以一种有趣的方式干扰着即使是最好的计划。

我会写一些我用来学习的方法，比如付费和免费的课程、教程、编码挑战和项目、书籍和其他资源，比如博客。还有更多。我们将调查我使用的软件和应用程序，以及我认为有益的软件和应用程序。

## 我为什么要这样做？

我想分享这个正在发生的过程。原始的而且大部分未经过滤。最重要的是，我会在我之前提到的那些环境中分享这个。你们中有很多人有着相似的经历，我希望我能尽可能地帮上忙。即使只有一个人也是值得的。我的目标不仅是帮助和启发读者，而且对我来说也是一个胜利。将想法写在纸上有助于我们巩固、划分和巩固我们所拥有的知识。更深层次的理解可以帮助我们更有效地利用它。

## 最后的想法

我的网络开发之旅正在进行中。它会随着时间的推移而改变。当我说改变时，我指的是学习的过程和网站本身。特别是在开始的时候，我会添加更多的资源、页面和链接。

我以前有过一些小博客，但都没有达到我认为这个网站有潜力成为的规模。随着时间的推移，质量和实用性都会提高。

希望这里有对你有用的东西。由于我打算学习一些后端编程语言，我希望这个网站对那些对 web 开发感兴趣的人和那些对一般编程感兴趣但不一定对 web 开发感兴趣的人都有吸引力。

当我创建新的帖子时，我也一定会把它们添加到这里。
