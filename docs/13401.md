# 故障项目

> 原文:[https://dev.to/wkmn/glitch-projects-91e](https://dev.to/wkmn/glitch-projects-91e)

再来说说 Glitch！Glitch 是一个在社区支持和丰富灵感下开发 NodeJs 项目的地方。

[![Glitch.com home page](../Images/1b058f9664719e237a1cebf9f04a2f4e.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--N2OemMvk--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://www.weckmann.me/assets/images/glitch.png)

Glitch 有*几个*易于使用的特性:

*   饭桶
    *   读写支持
    *   从 Git Url 导入
*   导出文件
*   协作编辑
*   嵌入项目
*   自定义域
*   日志
*   控制台访问
*   倒带(Git 版本控制)
*   内置 SQLite
*   社区帮助
*   资产存储
*   用户配置文件
*   私有代码库

所有这些功能都是免费的！

### [](#community)社区

随着一个大型的和不断增长的社区 glitch 有数以千计的项目供您浏览和从中汲取灵感。它们的特色包括 Hello Worlds、游戏、积木、学习编码、工作工具、社区选择等等。
