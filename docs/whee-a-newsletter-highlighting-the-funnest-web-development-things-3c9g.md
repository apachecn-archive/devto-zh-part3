# Whee！，这是一份介绍最有趣的网络开发的时事通讯

> 原文：<https://dev.to/michael/whee-a-newsletter-highlighting-the-funnest-web-development-things-3c9g>

几年前，我进入 web 开发的原因是因为它很有趣。我喜欢在我的电脑前花上几个小时，试验、调整和重新加载，看看代码在我的屏幕上改变事物的魔力。

我的新简讯，哇！试图通过试验和玩代码来激起那种有趣和兴奋的感觉。Whee！是一份时事通讯，收集了互联网上更有趣的 web 开发内容——无论是 CodePen 上很酷的视觉实验，还是向新 CSS 库中的 8 位图形致敬，或者是使用最新流行歌曲的编程概念。

我和 Whee 的目标是每周三与你分享我的惊人发现，让你度过难关，在这周余下的时间里说 wheeeee！🙌

为了在每周三将互联网上最有趣的网络开发内容发送到你的收件箱，请注册 Whee！。
