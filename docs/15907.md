# 重新访问您的旧 Procurve switch 界面

> 原文：<https://dev.to/c33s/regain-access-to-your-old-procurve-switch-webinterface-579o>

旧的 procurve 有一个很好的安全特性，你可以定义一个管理端口。只有当您的连接到此端口时，您才能访问 web 界面。

如果您自己编写的文档说您使用端口 2 作为管理端口，并且您的内存告诉您，您没有配置特定的管理端口，但是您的浏览器无法连接到 web 界面。如何调试那个？

这花了我一些时间，但有时这不是你的错，有时只是交换机的网络服务器崩溃了。

只需重启交换机即可解决问题:)
