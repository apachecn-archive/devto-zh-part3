# 神话 Angular 正在衰落

> 原文:[https://dev . to/Steve blue/the-myth-angular-is-in-decline-4hab](https://dev.to/steveblue/the-myth-angular-is-in-decline-4hab)

去年，NodeJS 基金会的首席运营官劳里·沃斯(Laurie Voss)在 JSConf 的舞台上宣布，Angular 的 npm 下载量已经达到顶峰，并在下降，这得到了[这篇博客文章](https://blog.npmjs.org/post/180868064080/this-year-in-javascript-2018-in-review-and-npms)中数据的支持。NodeJS 基金会对 1000 万用户进行了调查，并将这些数据与他们从流行框架下载中收集的指标相结合。

我当时认为这是一派胡言，现在仍然如此。这就是为什么。angular/core 和@angular/cli 这两个 angular 开发最基本的包的下载量随着时间的推移在不断增加！

Angular 工程师依赖的开发包(@angular/core)每周下载量超过 220 万次，并且还在增长。

[![](../Images/d7f29d70dc1cb397e9c849e940909246.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s---O6ZsIHS--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/dglka9rfdxqrncluhvzk.png)

@angular/cli 包每周的下载量减少了约 220，000 次，但每周的下载量也在增加。对于大多数 Angular 开发人员用来构建 Angular 的工具来说，这是一个令人印象深刻的采用率。

[![](../Images/19f91ccb66b841656851dd8e3360501f.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--wzOAOXG6--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/ew46rpxs195w4dhz15tp.png)

自 2018 年 4 月以来，@angular/core 包的每周下载量增加了约 150 万，一年内大约增长了 360%。

值得注意的是，React、Angular 和 Vue 的周下载量在过去一年中都出现了健康的增长。

React 是最受欢迎的 JavaScript 库，每周下载量超过 600 万次。自 2018 年 4 月以来，React 的周下载量增长了约 260%。

[![](../Images/a95384ecfedfbf39e4693b0579956f2f.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--byCUGIFN--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/510urwna3lo7gtqgs3is.png)

Vue 的人气在 2019 年 3 月攀升至超过 100 万的周下载量，自 2018 年 4 月以来，周下载量增长了约 320%。

[![](../Images/eb29280057cfac5c8dddd2103332383c.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--3Ow1PeUJ--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/rtf7s4wgdyo9spbgaq7y.png)

如果说前端框架有什么趋势的话，那就是 JavaScript 社区正在围绕 3 个主要框架进行整合。

似乎 NodeJS 基金会对传统 AngularJS 项目的下载量进行了权衡，这可能会导致 Angular 的采用出现问题。

[![](../Images/d6ee0c6edc9b586a9420298356637448.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--wKAr7gV7--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/dwk9l26stgqng3b79qdi.png)

对于在开发社区中有如此大影响力的组织来说，获得正确的数字是非常重要的！

我觉得很讽刺的是，在过去的一年中，@angular/core 包的每周 npm 下载量的增长超过了 React 或 Vue，考虑到 JavaScript 社区中的一个主要参与者站出来说 angular 正在衰落。React 确实是 web 开发中最受欢迎的库，应该因其模块化设计而受到欢迎，但是现在还不要把 Angular 排除在外！Angular 对于前端 web 开发来说仍然是一个非常受欢迎的解决方案，并且在可预见的未来可能还会如此。

P.S .祝贺 Vue 在 3 月份在 npm 上的下载量超过 100 万次！
