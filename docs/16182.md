### 特征

*   在新标签页中显示您的书签
*   书签过滤/搜索
*   键盘导航支持
*   书签文件夹可见性切换
*   主题支持
*   拖放书签排序

### 堆

*   React(使用创建 React 应用程序)和挂钩
*   以打字打的文件
*   样式组件
*   Redux、Redux-Saga 和 Typesafe-Actions
