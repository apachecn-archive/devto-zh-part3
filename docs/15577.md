# 在 Azure 上配置 OpenVPN 访问服务器

> 原文：<https://dev.to/evgenijrenke/configure-openvpn-access-server-on-azure-1akj>

VPN 或虚拟专用网络是两台或多台设备之间的安全隧道。VPN 用于保护私人网络流量免受窥探、干扰和审查。

[![alt text](img/fe837b803fe83636de946f56234742e2.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--8FEifg0h--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/mse3ij0drfu499x6sld7.jpg)

为什么我们应该使用 VPN？一方面，为了保护我们免受 ISP 的监控，另一方面，我们可以在像咖啡馆这样的开放 WLAN 热点中保护我们的互联网流量。

有了 Azure，我们可以很容易地建立一个 OpenVPN，并私下使用它。
