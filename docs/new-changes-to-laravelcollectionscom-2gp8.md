# LaravelCollections.com 的新变化

> 原文：<https://dev.to/adi/new-changes-to-laravelcollectionscom-2gp8>

距离我最近的副业-【LaravelCollections.com】的[发布](https://simplestweb.in/blog/launching-laravelcollections?utm_campaign=post-update-1)已经过去两周了。我从用户那里得到的反应总体上是积极的。浏览用户邮件和反馈促使我投入更多的时间和精力来改进网站。

上周我有一些时间，我研究了一些来自用户的常见建议。我想告诉你这里的最新变化。

### 新的变化

*   你现在可以搜索 400 多个相关链接
*   最近添加的链接显示在主页上
*   登录页面已得到改进
*   添加了一个关于页面的[来解释这个项目](https://laravelcollections.com/about?utm_source=simplestweb.in&utm_medium=referral&utm_campaign=post-update-1)
*   用户可以[提交链接](https://laravelcollections.com/submit?utm_source=simplestweb.in&utm_medium=referral&utm_campaign=post-update-1)到 Laravel 资源

### 即将推出的功能

*   标记链接，这样用户可以看到与标签相关的所有链接
*   显示最近添加的 10 个链接的页面
*   主页重新设计
*   收藏页面中的分页
*   收藏页面中链接的更好排序
*   提高搜索效率
*   将路线图添加到[关于页面](https://laravelcollections.com/about?utm_source=simplestweb.in&utm_medium=referral&utm_campaign=post-update-1)
*   每个收藏的 RSS 源
*   推广链接——一位拉勒维尔的作者就此事找过我，但我仍在考虑中

### 结论

这就是我要分享的关于 LaravelCollections.com 的全部内容。当项目有重大变化时，我会及时更新。如果您有任何反馈或建议，请发送电子邮件至-【laravelcollections@gmail.com[。](mailto:laravelcollections@gmail.com)

谢谢你。
