# 如何开始教学？

> 原文:[https://dev.to/ashleemboyer/how-to-start-teaching-4hng](https://dev.to/ashleemboyer/how-to-start-teaching-4hng)

稍后我要参加一个长达一小时的期末考试，所以今天的帖子会非常简单。

开始教学时，你的注意事项是什么？我已经享受了每一个向人们教授代码或技术的机会，并且想做得更多。现在这只是我的一个广泛的梦想，很难削减它向前迈进。

我对这里的所有建议和故事都持开放态度。无论你在哪里教过书，无论你教了多少年，我都想听听你想分享的东西。
