# 你听过比这更美的短语吗？

> 原文：<https://dev.to/ben/have-you-ever-heard-a-more-beautiful-phrase-than-this-2fe>

昨天，我们办公室的几个人注意到，我们已经一个多星期没有在 Slack 的#monitoring 频道收到警报了。每次[设备到](https://dev.to/)设备的错误率超过某个阈值时，我们都会收到警报。

[@maestromac](https://dev.to/maestromac) 调查过。过了一会儿，当我就这个问题找到他时，他是这样预测的:

> 监控一切正常。原来这个网站更稳定。

你听过更美妙的话语吗？

我们降低了一点阈值，应该会在新的级别上时不时地收到警报。

快乐编码

我确定我已经倒霉了，所以请期待一些重要的停机时间。
