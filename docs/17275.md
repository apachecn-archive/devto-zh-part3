# 多份拷贝

> 原文:[https://dev.to/trishul/multicopy-1f72](https://dev.to/trishul/multicopy-1f72)

一个可用性扩展，允许你复制多个文本，并显示在右键菜单中。粘贴任何输入字段中的任何文本。

要复制:右键单击选定的文本->多拷贝->复制这个！
粘贴:右击文本区- >多拷贝- >选择条目
