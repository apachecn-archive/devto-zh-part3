# Typescript 方法重载

> 原文:[https://dev.to/kreuzerk/typescript-method-overloading-43c8](https://dev.to/kreuzerk/typescript-method-overloading-43c8)

方法重载是 Java 或 C#等传统编程语言中一个熟悉的概念。它基本上允许一个类有多个同名的方法，如果它们的签名不同的话。

但是 Typescript 只能在编辑和编译时帮助我们，而不能在运行时帮助我们。因此，方法重载的工作方式不同。执行的代码仍然是纯 Javascript。

由于 JavaScript 的动态特性，我们不能像在其他语言中那样使用重载。[了解 TypeScript 中方法重载的工作方式](https://medium.com/@kevinkreuzer/typescript-method-overloading-c256dd63245a)
