# DDD 布里斯班——度过了一个美好的周末

> 原文:[https://dev.to/ankitvijay/ddd-brisbane-一个值得度过的周末-2p8o](https://dev.to/ankitvijay/ddd-brisbane--a-weekend-well-spent-2p8o)

这个周末(也就是 2018 年 12 月 1 日，星期六)，我和办公室的几个同事获得了参加 [DDD 布里斯班](https://www.dddbrisbane.com/)会议的机会。这都要感谢我的雇主 Youi，他也是这次活动的金牌赞助商。(我们正在[招聘](https://www.seek.com.au/Youi-jobs/full-time?classification=1209%2C6281%2C1220))。如果你不知道 DDD 会议是什么，它是开发者、开发者、开发者的简称。这是一个关于…的会议。开发商。

在周末参加一个会议对配偶来说是一个挑战性的讨论。很难解释为什么我会在周六早上 5 点起床去布里斯班，然后在深夜返回。真的很感谢老婆的理解和支持。我答应过她，我会补偿她的 [![🙂](../Images/1bb31e891282bfa40812655c9c9ace9e.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--DvXCaIjD--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://s.w.org/images/core/emoji/11/72x72/1f642.png)

这是我第一次参加 DDD 会议，但肯定不是最后一次。我非常享受这次经历，听着优秀的演讲者、食物、免费咖啡和好吃的东西。

[![](../Images/29da4fc85d9a98130e3c379bfd088190.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--ONAaay4n--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://i2.wp.com/ankitvijay.net/wp-content/uploads/2018/12/IMG_20181202_055650.jpg%3Fresize%3D255%252C340%26ssl%3D1)

以下是我从这次会议中得到的一些重要收获:

*   会议组织得非常好。注册过程很顺利，组织者很友好并且乐于助人。
*   美味的食物，免费的咖啡，很多好吃的东西，还有很多奖品要拿(不幸的是，我一个都没拿)。

[![](../Images/5f7e17ef6aaa7e7475eba56d21de29bb.png) ](https://res.cloudinary.com/practicaldev/image/fetch/s--x41yN-EQ--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://i2.wp.com/ankitvijay.net/wp-content/uploads/2018/12/IMG_20181202_055451-1.jpg%3Fresize%3D225%252C300%26ssl%3D1) [ ![](../Images/a539c719f3900e547fba66941a5a3e9e.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--utVN23nI--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://i0.wp.com/ankitvijay.net/wp-content/uploads/2018/12/IMG_20181202_062934.jpg%3Ffit%3D768%252C1024%26ssl%3D1)

*   这是一次真正意义上的“绿色”会议。与会者被鼓励自带咖啡杯。咖啡杯，包括盖子，都是可回收的。食物盘子，餐具是由竹子制成的，因此可以分解。可重复使用的袋子(或者他们称之为“赃物袋”)是由当地社区的废旧布料制成的。

[![IMG_20181202_062901-1.jpg](../Images/fc7aa39ee6e3580b9f2ebcd487371c22.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--7AN7WCkm--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://i0.wp.com/ankitvijay.net/wp-content/uploads/2018/12/IMG_20181202_062901-1-3707210842-1543705421553.jpg%3Fssl%3D1%26w%3D956)T3】我的赃物袋

*   演讲者阵容由新手和有经验的演讲者组成。我喜欢我参加的所有会议。

[![](../Images/4923a3a06c15c7ab8858404ef04efcb5.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--aSKCGHxG--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://i0.wp.com/ankitvijay.net/wp-content/uploads/2018/12/Speakers.jpg%3Fw%3D810%26ssl%3D1)

*   两次会议之间有足够的间隙——没有信息过载。
*   每节课都有东西要学。没有一个是浪费时间的。
*   少数会议还验证了我们在工作中遵循行业范围和推荐的开发实践。事实上，我们可能在某些领域领先于其他行业:)。特别是来自 clean architecture 的代码片段。NET Core 2.2 会话看起来直接来自我们的私人回购。
*   这是与其他开发者交流的好机会
*   特别提及主题演讲人杰西卡·克尔。但对我来说，这一天的亮点是尼尔·福特关于“支持不断变化”的 locknote。在他 45 分钟的演讲中，有太多东西需要学习。

[![](../Images/a4bfeb684dd439c8634ae1895555db6b.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--rjIsA2dU--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://i2.wp.com/ankitvijay.net/wp-content/uploads/2018/12/IMG_20181201_171313.jpg%3Fresize%3D300%252C225%26ssl%3D1)T3】一针见血..

总而言之，这是一个学习新事物的好周末。
