JavaScript 是新的后 LAMP 派中最大的一块。它不仅是一种前端脚本语言，还是一种软件堆栈游戏改变者。

#jskongress 2019 第二天开幕主题演讲的目的是让人们认识到软件栈中的(r)进化。其次，让他们理解 JavaScript 作为一个平台在这一(r)发展中的关键驱动力和重要性。

最重要的是，让他们能够利用谈话中分享的材料，将变化融入到他们的工作/个人项目中。

* * *

今天，我们很高兴向您介绍**主题演讲人 Maria Shoaib** ，她是将于 2019 年 3 月 11 日至 12 日在慕尼黑举行的为期两天的#jskongress 的最新成员:

Maria 是巴基斯坦科技界的一名女性，目前在瑞典爱立信公司担任软件工程师，她获得了美国 RIT 的富布莱特奖学金，获得了计算机科学硕士学位。她还领导了源于她的的倡议[，以赋予妇女权力并弥合](https://www.facebook.com/stemmingfromher/) [STEM](https://en.wikipedia.org/wiki/Science,_technology,_engineering,_and_mathematics) 领域的性别差距。
