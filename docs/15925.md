好了，这么多信息。现在让我们在系统上安装 React。

**<u>项目结构</u>**

1.  创建一个名为 **reactJs** 的文件夹。
2.  在里面创建一个 **myApp** 文件夹。
3.  在里面创建一个**公共**文件夹。
4.  在这个文件夹中创建一个**index.html**文件。

**<u>CDN</u>**

 <u>React 是一个 JS 库，所以它可以作为 CDN(内容交付网络)使用。

```
<html>
<head></head>
<body>

    <script src="https://unpkg.com/react@16.0.0/umd/react.development.js"></script>
    <script src="https://unpkg.com/react-dom@16.0.0/umd/react-dom.development.js"></script>
</body>
</html> 
```

React 需要两个库才能工作。

1.  我们添加的第一个[ [react@16.0.0](mailto:react@16.0.0) ]包含了 react 的核心语法，如 createElement()函数等。
2.  第二个[ [react-dom@16.0.0](mailto:react-dom@16.0.0) ]是 ReactDOM。这基本上用于渲染元素。
