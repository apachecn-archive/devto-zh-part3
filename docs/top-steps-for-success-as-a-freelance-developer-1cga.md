# 自由开发人员成功的首要步骤

> 原文：<https://dev.to/officialsilb/top-steps-for-success-as-a-freelance-developer-1cga>

我们许多人梦想成为自由职业者，希望这能给我们发号施令的自由。然而，摆脱全职工作带来的许多确定性，找到自己的客户可能会令人望而生畏。自己创业意味着自己寻找客户，并能够有效地管理自己的时间。虽然一开始可能会很艰难，但相对较快地站稳脚跟并获得稳定的收入是有可能的。在这里，我们将看看作为一名自由开发者取得成功的一些最佳步骤。

建立一个线索列表

在你决定自己出去之前，重要的是[建立一个你可以从中工作的线索列表](https://www.forbes.com/sites/abdullahimuhammed/2017/11/24/55-free-ways-to-find-leads-for-your-freelance-business/#274acb4b19d1)。这可以通过在活动中建立关系网、口口相传或通过电话或电子邮件联系来实现；一个开始的方法是建立一份你希望为之工作的公司的联系人名单，然后整理一份文件，记录你联系过的人以及他们的回复。在你接触任何潜在客户之前，你会想决定你的价格和你愿意接受的最低价格。

签订合同

作为一名自由职业者，最常犯的一个错误就是没有正式签订合同。合同不需要太复杂，但最好有书面的交易和协议，并由双方签字，以防以后出现任何问题。你需要在合同中加入支付条款，比如你将获得多少报酬，何时获得，以及终止合同的权利和终止方式。确保你与客户的任何协议或变更都是书面的。如果你正在努力，你可以在网上找到自由职业者的合同模板来帮助你开始。

明智地使用你的账户

当自由职业者工作时，很容易陷入管理客户关系和寻找工作中，以至于你无法保持准确的账目。然而，保持准确和定期的帐户将节省你一个巨大的头痛时，填写你的税务自我评估。

在你开始工作之前，你需要注册成为一个个体商人或者一个公司；如果你刚刚起步，除非你预计你在纳税年度的利润超过 35，000 英镑，否则注册为个体商人可能更适合你。除非你一开始就有很多客户，否则自己记账可能更容易——有很多在线记账软件可以让你走上正轨。有一些应税费用将有助于减少你的税单，所以请确保在提交你的账户之前研究这些费用。

备份您的数据

备份你所有的文件、账户和工作将确保你的业务不会受到电脑故障或恶意软件的负面影响。作为一名自由职业者，投资云计算是备份数据的最佳且最实惠的方式之一。云计算还可以让您轻松扩展业务，而无需投资昂贵的 it 基础设施。

有重点

作为一名自由开发人员，你会知道自己喜欢什么样的项目，不喜欢什么样的项目。专注于你承担的和想做的工作，不仅能帮助你推销自己，还能帮助你成为你想成为的那种开发人员。创建一个你希望成为专家的工作的文件夹，并且知道你可以对你不感兴趣的项目说不。
