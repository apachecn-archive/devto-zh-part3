# 博客:博客直播液体标签现已上线

> 原文：<https://dev.to/m1guelpf/changelog-blogcast-liquid-tags-now-live-llo>

我们有了一个新标签，Blogcast 标签！

Blogcast 是一个为你的文章生成音频版本的服务，这个标签可以让你嵌入一个播放器来在 Dev.to 上收听音频版本

玩家长这样:

```
{%  blogcast  26  %} 
```

Enter fullscreen mode Exit fullscreen mode[https://app.blogcast.host/embed/1](https://app.blogcast.host/embed/1)

要将它嵌入到你的帖子中，[创建一个博客账号](https://blogcast.host)，然后[按照官方指南在开发者上嵌入到](https://docs.blogcast.host/embeds/dev-to.html)。
