# 茶歇挑战#22

> 原文：<https://dev.to/jonmajorc/teabreak-challenge-22-3o8i>

> 专注的能力对开发人员的成功至关重要。确定你今天注意力最大的敌人，并在你工作的时候意识到这一点。~茶歇挑战#22

**敌人**

尽管“专注对开发人员的成功至关重要”，但它可能会破坏您的生产力。仅仅因为你很专注，并不意味着你很有效率！今天，我“注意力最大的敌人”是有太多的注意力。我有一个问题要解决，我在转动轮子，但我太专注了；我不愿意休息...如果你像我一样，你经常这样做。你需要学会什么时候休息。

**解决方案**

1.  多休息一下，这样我就习惯了不整天坐在键盘前。
2.  在转动我的轮子后，启动一个计时器，继续工作直到它结束。(然后休息一下。)
3.  再要一双眼睛。
4.  通过休息和做元任务来保持生产力，让我从我试图解决的原始问题中分心。(例如，休息、电子邮件、洗手间、午餐、小吃等。)

也许，只是也许...我需要更多的咖啡。那么我应该可以用蛮力找到解决方法！🤔

今天是什么敌人阻碍你集中注意力？此外，请帮助我，提出任何可能的解决办法来打击我的敌人！
