# 到目前为止，我一直在找工作，为了正确的理由，我再次爱上了编程

> 原文:[https://dev . to/bcampos 103/job-search-so-far-and-learning-to-love-programming-again-for-the-right-reasons-10j 3](https://dev.to/bcampos103/job-search-so-far-and-learning-to-love-programming-again-for-the-right-reasons-10j3)

自从我获得计算机科学学士学位的四年大学毕业已经一年了，尽管我不得不承认课程有点乏味，但我仍然很感激我能从中学到什么。虽然从那以后没什么进展。

我没有钻研任何项目想法，只是漫无目的地探索不同的技术，比如 React、Go 等。在我毕业后的一年里，我没有真正把它们付诸实践，主要是因为我不顾一切地想引起招聘经理或诸如此类的人的兴趣。正是这种绝望让我开始讨厌编程；讨厌它只是因为它不能让我找到工作，因为我认为这样我就有理由想出解决问题的办法，因为我会简单地等待解决问题。当然，这不是一种健康的心态，它极大地影响了我。

2018 年剩下的时间里，我一直在憎恨自己的人生定位，因为我目前的状况不是最理想的；我仍然在大学二年级后半学期开始工作的杂货店工作，勉强维持生计，几乎没有重温我学过的编程语言，感受着曾经想出解决方案时的快乐，但很快就放弃了，让它们独自呆了几个月。沮丧、沮丧和焦虑成了我生活中的现实。我决定多出去认识新的人，看看这是否能缓解我的这种心态，我设法在那里认识了一小群人，并很快成为了朋友。

在结识新朋友、参加新活动、与他们一起闲逛之后，我意识到他们都有自己擅长的爱好，并结识新朋友，因为他们喜欢这样做，但最重要的是，他们坚持这样做，因为他们约束自己坚持做自己喜欢的事情。他们描述说，他们不得不努力让自己做自己喜欢的事情。

当然，在自我反省之后，意识到我的位置和我在这里的原因是很有意义的。我有动力，但动力只是解决方案的一半；我需要约束自己。我意识到，我需要不断提醒自己，当我学习一门新的编程语言并想出解决问题的方法时，我的感觉就像在大学里一样。只是这一次，我必须约束自己按照自己的意愿去做，而不是依赖别人告诉我该怎么做。进入 2019 年，我决定就这样做。坐下来，做我做的事情，做我喜欢的东西，然后再去担心其他的事情。为我的生活状态感到高兴，因为这是改善它的第一步。

我可能不是最熟练的作家，但我觉得我必须写这个帖子来发泄，并看看是否有其他人和我一样。希望你一切都好。
