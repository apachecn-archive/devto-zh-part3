# Web Dev 学校还值得吗？

想从事 Web 开发的新手总是雄心勃勃，缺乏耐心。他们想在他们梦想的公司里找到一份高薪的工作，享受所有的福利和津贴。他们不知道，在被招聘者和雇主注意到之前，掌握一项技能是需要时间的。
大部分新手走的都是“自学”的路线。他们注册在线技术机构，观看多个 YouTube 教程视频，阅读书籍，听播客，阅读来自不同专家的大量编码博客。他们一遍又一遍地重复。冲洗和重复，直到他们掌握了他们想被雇用的技能。不幸的是，这需要几个月(甚至几年)才能完全实现。大多数新手都等不起那么久。

自学的问题在于没有人指导你。你完全依赖的只有你自己，因此在这个过程中会犯更多的错误。会占用你更多时间的错误。当没有人指导你时，会有很多障碍和阻碍(更不用说在编码问题上遇到很多困难)。大多数时候，你会对如此多的信息来源感到困惑，以至于不知道该找谁。可悲的是，这些导师、导师和作者中的大多数都忙于回应他们成千上万的追随者。他们无法回答所有的问题...你认为他们能抽出时间回答你的问题吗？祝你好运！

这就是技术培训学校被发明的原因。它主要是为像你这样的人创造的。他们通常有较少的学生(不像有很多追随者的导师)，他们为你制定了固定的时间表和课程。直到你毕业的那一天，一切都是有组织有计划的。你的每一步都会被指引。

这种类型的学校是如此的有效，以至于突然之间它就像一个“东西”一样在各地迅速发展起来。好吧，那你能去什么学校...我可以推荐的一所学校是里提姆学校。
这是一个为期 16 周的网络开发训练营，由训练营教练创办。总部位于旧金山的机构，专注于小班教学和个人成长。他们如此专注于一个利基市场，以至于他们的班级只有 18 名学生。

Rithm 是一个为期 4 个月的全栈 Web 开发项目，其课程以 Javascript 和 Python 为中心。它的特点是让学生在海湾地区的公司实习 3 周，让他们的学生建立和创建真实世界的项目(这是成为一名优秀的 Web 开发人员非常必要的一件事)。在所有这些之后，他们有超过 85%的学生在毕业后六个月内被录用，平均工资为 11 万美元。不算太寒酸，不是吗？

里提姆学校与合作伙伴公司紧密合作，确保课程的高质量和相关性，使毕业生从第一天起就能立即投入工作。与其他训练营相比，学生与教师的比例较低，教师经验丰富，学生们可以走得更快，学习更多的内容。

如果你缺钱怎么办？不在话下...学生可以选择“延期学费选项”，如果你在课程的前六周没有以成功所需的速度学习，Rithm 会提供全额退款和申请后续课程的机会。这意味着在你找到一份工程师的工作之前，你不必支付任何学费。

该公司提供机会，使其学生能够联系并参与风险投资支持的公司的项目，或为开源做出贡献。以下是他们合作的一些公司...伙伴公司

上课时间为周一至周五，上午 9 点至下午 6 点，地点在里提姆的旧金山校区。你通常会在一天的大部分时间里独自或两人一组地编写代码。讲师将在上午和下午讲授更多概念性的主题。
在下午和晚上，你将有材料阅读、观看和复习，以巩固你的理解并为第二天做准备。您将通过参与开源和/或合作伙伴公司的应用程序来练习。
经过 13 周的指导，你将专注于行为和技术面试技巧。该校的招聘合作伙伴网络使其毕业生在求职中处于迅速成功的有利地位。

获得“真实世界的经验”是成为编码大师的关键。当你成为这方面的大师时，公司肯定会找你。在这种经历中，你获得了完美，你会惊讶地发现，编码挑战对你来说不再是一个挑战。让里提姆学校给你这种体验。相信我...一旦你毕业，并在一家有声望的公司工作，你会对自己说...“这是一次难忘的经历，我永远不会后悔”！
