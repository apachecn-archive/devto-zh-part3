```
const connection = mysql.createConnection({
  host: 'localhost',
  user:'root',
  password:'myjs123',
  database:'simple-react-sql-db'
}); 
```

应该更改连接字符串的主机用户密码和数据库地址。现在测试你的应用程序，确保它能正常工作。
