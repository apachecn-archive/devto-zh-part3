# JavaScript RPG 好玩又开源

> 原文：<https://dev.to/naknode/javascript-rpg-fun-and-open-source-4273>

你好，我正在用 JavaScript 创建一个有趣的 2D 中世纪角色扮演游戏“德拉福德”。它拥有(传统)RPG 的所有现代特征，从库存系统到关卡到寻路等等。

很多还没写，但是你可以帮忙写！也许可以加下一个技能？(我刚发布挖矿！)也许你可以帮助创建怪物类？这么多事要做。

[https://github.com/delaford/game](https://github.com/delaford/game)

它自 2018 年初以来一直在开发中，但只有在过去几个月我有时间的时候才认真工作。

我们有(某种程度上)可用的文档。但是，如果你愿意的话，你可以下载下来随便看看。它也有 Node.js 服务器！
