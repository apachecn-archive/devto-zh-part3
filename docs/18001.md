每次我需要在 OSX 上安装 Python 或者有同事向我寻求帮助时，我都必须在谷歌上搜索最新的说明，每次我都能找到做同样事情的不同方法。

厌倦了这些，我决定写下自己的笔记。请注意，我并不认为这是在 OSX 上安装 Python 的最佳方式。它对我来说很好，所以使用它在你自己的风险。

## [](#requirements)要求

要遵循这些说明，你至少需要在 OSX 上安装 **brew** 。请按照官网上的说明: [https://brew.sh](https://brew.sh)

## [](#installing-python-37x-and-python-27x)安装 Python 3.7.x 和 Python 2.7.x

即使我强烈建议用 Python 3 启动每个新项目(由于 Python 2 只支持到 2019 年底)，也可能会有仍然需要版本 2 的用例，所以我会给你安装这两个版本的说明。

### [](#installing-python-37x)安装 Python 3.7.x

```
brew install python 
```

这将默认安装 Python 3。
