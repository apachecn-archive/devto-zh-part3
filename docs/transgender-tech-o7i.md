# 变性人和技术

> 原文：<https://dev.to/calikat68/transgender-tech-o7i>

我一生都知道，我的身体属性和我的大脑所说的应该存在的东西之间存在着内在的脱节。就像人们失去肢体时的经历一样。他们的大脑告诉他们那个肢体还在，但现在不在了。

我清楚地记得我第一次发现科技的情景。1981 年 4 月，哥伦比亚号航天飞机进行了它的处女航。发射时我在肯尼迪航天中心。我记得参观游客中心时，对帮助实现这一点的机器感到惊讶。我不关心在月球上行走的人，对我来说意义深远的是帮助他们到达那里的技术。

从发布会回来的第一个周末，我花了一个下午在图书馆阅读所有关于计算机和技术的资料。我找到了一本我非常喜欢的关于 Basic 语言的书。我决定拿出我当时所有的积蓄，大约 9 美元，给出版商写了一封信，问他们我是否可以买一本。当时我 11 岁。他们被深深打动了，把我的钱退了回来，还免费送了我一本书，并祝我旅途好运。

六个月后，我说服养父给我买了一台 TRS-80 可可电脑。OMG！！我真的有一台电脑！我真的可以用代码来构建东西！这是我一生中经历过的最不可思议的事情。我从来不觉得我和人们有联系。很大程度上是因为我从来不喜欢我的身体，以及它让我感到多么不舒服。再多的治疗也改变不了这一点，这就是我的大脑在胎儿发育期间的运作方式。电脑不在乎我的长相和声音。它只关心我写的代码。

我的 TRS-80 是一台 16k 的机器，配有磁带机和连接电视的电缆。我写的第一个应用程序是一个小版本的吃豆人。我至今还记得基本的。

我的运气变得更好了。我的家人认识一些在博卡拉顿开发最初的 IBM-PC 的团队成员。我们得到了一个早期的模型。OMG！！这对我来说就像中了彩票。一个 8088 - 4.7MHz 的 CPU？！？神圣处理器蝙蝠侠！！64k 内存？我能做的事情- Basic 和现在的 Pascal？未来是如此的光明。

在我高中的最后一年，学校开设了一门 Pascal 课程。我已经使用这种语言两年多了。我知道的比老师和班上大多数其他孩子都多。除了这个孩子，他是典型的混蛋之一，比房间里的其他人都聪明，他也知道这一点，他也喜欢贬低别人。我家里有一个非常严重的问题，整个夏天都和我的叔叔婶婶住在一起。长话短说，那个夏天我做了真实的自己。几个孩子发现了，我帕斯卡课上的那个混蛋发现了。他利用我的爱和我的快乐，一台电脑，作为在全班面前羞辱我的一种手段——一条信息在我们学校有 25 台机器的计算机实验室的所有计算机的屏幕上闪烁，让每个坐在一台计算机旁的人知道我想成为一个女孩。

我停止使用电脑，直到几年后我上了大学。我在极度痛苦中离开了家。我在没有家庭帮助的情况下完成了学业。我拿到了奖学金和贷款，拼命工作。我唯一一次接触电脑是在玩泥巴或者写论文的时候。我讨厌电脑。我也决定在大学里活出真实的自己。可悲的是，另一场悲剧将会发生，我不再活出真实的自己。

离开学校几年后，我和一个家伙成了好朋友，他辞去了他在惠普的高级主管的轻松职位，开发打印机驱动程序，去开了一家 ISP。我开始和他一起做一些兼职工作，他教了我很多关于 Unix/Linux 和网络的东西。我如饥似渴地学习所有我能学到的科技知识。我住在波特兰，或者当时，我想有一段时间人们以为我实际上住在鲍威尔的技术书店。我开始在科技领域建立一个伟大的职业生涯。

你好吗我不知道

我脑子里一直在想一件小事。我以一个不是我的人的身份活着。这是精神上的负担。关于我是谁，我一直有这个“秘密”。我总是害怕有人会发现，并认为我是一个完全的怪胎。隐藏我生活中这一方面所花费的精力导致了多次抑郁发作。这导致了三次自杀企图，其中两次几乎致命。随着年龄的增长，抑郁期开始持续更长时间。它们对我的工作能力产生了非常真实的影响。

在我快 30 岁和 40 岁出头的时候，我终于到了想冒险和我的一个朋友一起建立一家软件公司的时候了。这是一次失败，但我学到了很多。然而，进入 2012 / 2013 年，我患上了严重的抑郁症，住院 6 个月。我不再相信自己。我讨厌过去的自己。我认为我学到的一切都毫无价值。我没想到会有人雇佣我这样的人。我真的以为我在科技界的日子结束了。

直到最近，这听起来仍然是真的，无论我多么努力地工作，我永远不会得到一个席位，帮助建立东西。

2014 年，我决定不再浪费生命中的任何时间来憎恨自己的出生。我决定做真实的自己，努力追寻我对科技的热情，或者至少说我努力过。

当你是跨性别者，并且在以后的生活中开始你的旅程，这是很难的。伙计，这太难了。荷尔蒙可以影响你体内的 300 多个系统。是不是很神奇？但是要扭转这些影响需要很长的时间和大量的金钱。然而，在你开始这个过程之前，你必须活出真实的自己。这意味着你可能会看起来或感觉起来像个变装皇后。糟透了。很难。这很尴尬，你告诉自己结局才是最重要的。

因为我的职业生涯已经被火车撞毁了，我已经放弃尝试学习/保持技能好几年了，我在各种工作类型中跳来跳去，只是想回到游戏中。我记得有一次面试项目经理的工作，那个人对我说，'如果你是正常人，我会雇用你'或者另一个家伙，'这是一个笑话吗？'或者人们甚至懒得给你回电话。你必须工作，所以你要不断往下爬，直到你找到一个人，他会雇用你，尽管你资历过高，但你必须养活自己。这对你的职业和自尊伤害更大。

2015 年，我被一家初创公司聘用，被聘用几个月后，我告诉他们我是变性人，想变性。我以为他们会解雇我，这在过去的 18 个月里已经发生过两次了。不，他们说太好了，我们如何支持你。厉害！这正是我所希望的。这是我需要的能够回到我的最佳状态。他们没有告诉我的是——他们不可能让我离开我受雇的角色。当我努力进入一个我有资格胜任的 QA 职位时，6 个月来我被告知如此多的废话，以至于他们甚至不允许我开始训练，我的眼睛开始变成棕色。在一家有 130 名员工的公司，技术副总裁说，在得到首席执行官的许可之前，他不会允许我开始与质量保证团队一起培训。好吧，好吧，随便了。

和他们在一起两年后，我终于辞职了。我可以看到墙上的字迹。我搬到了加利福尼亚。我后悔搬到这里。我不建议没有工作就来加州。在这里很难。我在圣莫尼卡的一家初创公司找到了工作。我为一些糟糕的公司工作过，但我一生中从未在比这更糟糕的地方工作过。我也再次体验到男人对变性人的偏见。为了进门，我扮演了一个出租人的角色。我努力让自己进入工程团队。他们在团队中有空间，而我有必要的技能，至少在初级/中级水平上可以加入并做出贡献。

事实上，我帮助首席技术官制定了 SRE 面试的问题。我旁听了面试，并在招聘过程中给出了反馈。我不是工程团队的正式成员。有一次，我提交了一些 Ruby 代码来展示我在学习这门语言中取得的一些进步。

问题是——在不使用任何数学库或外部库的情况下找到给定数字的平方根，并舍入两个位置。我的解决方案是使用二分搜索法:

def sqrt(数字，精度)

```
return 0 if number == 0
startv = number.to_f
endv = number 

loop do
  middlev=(startv + endv / startv)/2
  return startv.round(precision) if startv <= middlev
  startv = middlev
end 
```

我为自己找到了这个解决方案感到无比自豪。我承认我花了大约 4 个小时才想出如何解决这个问题。但是他们认为“软件工程师”不能解决这个问题。然而，没有机会进入工程团队。

我被解雇的那一天是我这些年来过得最好的一天。它引导我到了现在的位置。但是即使我喜欢我现在工作的公司，即使有很多 LGBT 人在那里工作，据我所知，我是唯一一个公开自己是谁的变性女性。

关于变性人，我想和科技界的朋友们分享一些事情:

1)我的友好并不意味着我要向你献殷勤。我只是想表示友好。如果你觉得我丑陋，令人讨厌，不是你想与之成为朋友的人，这没关系，但是请试着克制你的表情。你给人的那种“啊，这太恶心了”的小表情，他们会戴在人身上。生活艰难的人，试着不要让人们更难。

2)我的胸部可能是假的，我的声音可能比一个“cis”女孩的声音更低沉，但这并不意味着我是哑巴。仅仅因为我没有工程师的头衔并不意味着我对技术一无所知，或者如果你花额外的 5 分钟来解释这件事，我也不容易理解。

3)我害怕和你一起呆在房间里。是的，我真的很害怕。我在努力融入社会和职业。我试着不去搞砸任何事情。如果我犯了一个错误，不要表现得好像这是世界末日。

4)我想加入团队并不会因为我的存在而减少团队成员。我带来了我的才华，我的激情，我通过独特的视角看到的想法，这将给你更好的想法。

我不会在浴室里跟踪你。永远不会。我才不管你在福克斯新闻上听到了什么。我要去浴室，进入一个摊位，做我的生意，离开摊位，洗我的手，然后 GTFO。变性并不意味着我是他妈的性侵犯者。说真的，不要把这两件事等同起来。

6)我想学。如果你不想帮助人们学习，或者你是反 LGBT(跨性别者)，那是完全可以的。请不要忽视我。只是说一些好听的话，比如-我真的没有时间或精力去支持任何人。伙计，之后就没什么了。

在工作场所变性是很难的。在科技行业变性更难。当你患有冒名顶替综合症时，我每天都不得不与放弃去做其他事情的冲动作斗争。

我所能要求的就是你试着让你的偏见偏向变性人。你可能会找到一个新朋友，发现我们只是普通人，过着自己的生活，努力做些酷的事情，就像你一样。
