# 初级开发者的挣扎

> 原文:[https://dev . to/unicorn dough _ 59/junior-developer-struggles-41pp](https://dev.to/unicorndough_59/junior-developer-struggles-41pp)

所以，在被拒绝了这么多次之后，我最近一直在努力让自己保持动力，坚持下去。

我被一家相当大的咨询公司聘为初级 IT“专家”。他们对我的承诺是，我将获得前端跟踪，以了解更多关于前端的知识，并有可能找到一份工作。一旦我开始工作，我就意识到他们对我没有跟踪，基本上是做一些更多的自学，并试图在公司的不同业务部门找到一些项目。

我从 10 月份就在这里了，我一直在努力保持学习一切的动力。招聘开发人员的公司通常会找那些吃饭、睡觉、呼吸代码的人，尽管我喜欢编码和学习等等。当我在家的时候，我真的不想再编码了。

我学习 Angular(2+)已经有一段时间了，我也对 java 感兴趣(但也因为公司寻找 Java 开发人员而不是前端开发人员)，我只想从任何地方开始。我更喜欢前端，但是我非常渴望进入大世界，在 scrum 团队中工作，开始我的职业生涯。

我了解 HTML、CSS 和 Javascript。我确实需要更好地学习 OOP，但我正计划用 Java 来学习，因为我发现 Javascript 有时很难。

还有人经历过和大三一样的尝试入门吗？有什么建议吗？
