# 你可以从最好的渐进式网络应用范例中学到的 6 件事

> 原文：<https://dev.to/maxchurilov/6-things-you-can-learn-from-the-best-progressive-web-app-examples-1j8f>

手机 app 还是手机 web？渐进式网络应用程序吸取了两者的长处，创造了独特的用户体验。许多世界知名的公司已经推出了渐进式网络应用程序，以下是他们的成果。

简而言之，你可以将渐进式网络应用程序(PWAs)视为一个使用网络技术构建的网站，它作为一个应用程序与用户进行交互。浏览器技术、缓存和推送界面的发展使我们能够直接从浏览器在主屏幕上安装 PWAs，接收推送通知，甚至离线工作。
