# 你的领导类型是什么？

做一个领导者并不是世界上最容易的事情。如果你是一名领导者，你将面临很多挑战:让你的团队参与进来，确保每个人都被听到，衡量成功，将你的团队引向正确的方向，代表应该遵循的原则，最难的是，当你错了的时候承认。作为一名技术领导者，除了常见的挑战之外，还有更多挑战，你必须了解你的团队正在使用的堆栈以及你为什么要使用它。

但是不同的情况可能需要不同的领导。在我大学期间与各种推广小组一起工作的三年中，我熟悉了领导者和被领导者的角色，在这些经历中，我可以将这些领导者分为三种类型:

## [](#idea-leaders)思想领袖

这种类型的领导者有追随者，因为他/她代表着一种理想，他/她的追随者希望像他们的领导者一样。领导者是其所在领域的理想人物，他/她拥有该领域的广泛知识。这些知识对他们非常有用，因为即使在他们没有 95%确定他们应该做什么的情况下——没有人 100%确定他们下一步应该做什么，但这是另一篇文章的主题——他们也会提出各种可能的解决方案，并与他们的团队讨论最佳方案。这些领导者是跳出框框思考的人，他们是天才。

技术领导更经常地表现出这种类型，因为他们中的大多数人都是他们团队所做工作的专家。他们了解应用程序的每一个细节，他们知道如何优化它的每一点，以及在紧急情况下你应该做什么。

## [](#kindness-leaders)仁慈的领导者

这种领导的主要特点是善良，除了我开头列举的常见的。他们成为领导者是因为他们以一种特殊的方式关心他人、他人的感受和幸福。这些是最人道的领导者，他们的追随者觉得他们可以在任何情况下，个人或专业，领导者会帮助它，或至少安慰和关怀。

在科技行业，这种类型会让你的生活更愉快，他/她会让你感觉像在家里一样安全和健康，从而帮助你提高工作效率。他们让工作变得愉快，他们知道如何创造最好的工作环境。

## [](#courage-leaders)勇气领袖

这些人出现在危机和巨大的不确定性时期，他们是有勇气解决任何问题的人，无论问题是大是小，简单还是复杂。勇气型在行动中有它的力量，人们追随它，因为它的效率和不屈不挠。他们不被视为一个要实现的目标，不像想法类别，他们被视为带领其他人投入战斗、解决重大危机和重新建立平静状态的人。如果我们用中世纪的术语来说，“思想领袖”将是战斗的战略家，预测战斗的每一种可能的结果，而勇气领袖将是带领战士投入战斗的人。

这种类型的领导者是做出艰难选择的最佳人选，他们是科技行业的先驱，他们喜欢处于知识的前沿。对他们来说，这感觉就像一个战场，在那里他们可以展示他们真正是谁，带来的不仅仅是结果，而是可能的最好结果。

## [](#conclusion)结论

一个好的领导者应该能够看穿危机，帮助他/她的团队克服障碍，同时关心这个团队。如果你是领导，告诉我:你是什么样的领导？你认为人们可以在不同类型的领导之间转换吗？我确实相信，但通常情况下，他们同时表现出三种类型中的两种。
