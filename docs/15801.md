# 你有哪些用例？

> 原文:[https://dev.to/kayis/which-use-cases-do-you-have-21ge](https://dev.to/kayis/which-use-cases-do-you-have-21ge)

我正在研究我的新书*“MVPs with AWS”*，并与一些 AWS 技术专家进行了交谈。

我想为软件产品的创造者创建一个学习资源，尤其是在他们的 MVP 阶段。

我假设创建者希望快速构建 MVP，并希望以后在特性和利用率方面对它们进行扩展。此外，创作者希望有一个基于代码的移动/网络和一个后端。

我认为 AWS 有许多服务，尤其是无服务器的服务，可以解决这些问题，但目前还不是很容易获得。

我的想法是写一本面向用例或业务用例的书，在那里我解决了最常见用例的技术问题。

我已经从专家那里获得了*技术*输入；人们想要实时和离线应用，人们想要一个 android/ios/web 的代码库，人们想要一个这些应用的后端，等等。

现在我正在搜索用例，所以如果你能写一点关于你需要解决的用例以及你面临的问题，那将会很酷。
