我刚刚为 GitHub 库创建了一个 pull 请求，它向人们介绍了一种晦涩但迷人的编程语言，叫做 Common Lisp。摆弄它很有趣，因为它是一种多范式编程语言，允许您以许多不同的方式处理和解决问题。CL 创造了许多人们一直用来解决编程问题的概念，比如一级函数(可以将函数作为参数的函数)，这是一个越来越常见的技巧，今天在 Haskell 和 JavaScript 中都有使用。此外，它是解释的，所以你可以不断调整你的代码来试验和修改你的代码，而不需要像 Java 一样一次性运行整个程序。能够在运行中进行更改使编程变得更有趣，更少挫折和耗时！

它只有三行 HTML，但却是人们在 [Articulate Common Lisp](http://articulate-lisp.com/) 明确要求的东西。它详细介绍了如何通过两步快速安装 Steel Bank Common Lisp，这是 Lisp 的一个流行版本。他们有一堆很酷的材料来帮助人们开始用 Lisp 写很酷的东西。

除了常见的 Lisp 之外，主要的收获是，对开源做出贡献对于您的程序员职业生涯来说是非常好的一步，即使这是非常小的事情，比如在页面上更改或添加几个单词！那些开始做小拉请求的人最终可以成为成千上万人每天使用的开源软件的维护者。试试看！你永远不知道今天的小小贡献明天会对你自己或别人产生巨大的影响！
