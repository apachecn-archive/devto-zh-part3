# 即将推出 Magento 2 的维护模式

> 原文:[https://dev.to/simonwalkerfme/coming-soon-维护-模式-对于-2-n6e](https://dev.to/simonwalkerfme/coming-soon--maintenance-mode-for-magento-2-n6e)

使用这个扩展，以一种体面的方式向你的访问者展示你的网站的当前状态。这个扩展允许你显示一个即将到来或维护的模式，清楚地告知访问者网站的当前状态。

您还可以启用电子邮件订阅表单、社交链接、添加背景媒体并允许 IP 地址加入白名单用户。白名单中的用户将能够绕过并浏览网站进行测试。

## **主要特征**

1.  3 种网站模式-实时、即将推出、维护模式
2.  启用带有用户注册表单的即将推出的页面
3.  使用倒计时定时器激活维护模式
4.  将媒体添加到即将推出和维护模式页面
5.  关于 IP 地址的白名单网站流量
6.  展示社交链接以吸引客户
7.  允许访问者访问 CMS 页面的选项

### **3 种网站模式-实时、即将推出、维护模式**

这个扩展允许你在 3 种网站模式之间切换。您可以通过单击来启用以下任一选项:

*   主动实时模式
*   启用即将推出的页面
*   启用维护模式

### **启用带有用户注册表单的即将推出的页面**

将你的网站访问者转化为注册用户或 E-main 订户。网站上线后，您的订户将收到电子邮件通知。

### **用倒计时器**激活维护模式

当你的网站处于维护模式时，定制维护模式页面以防止让你的访问者失望。显示倒计时计时器，让他们知道网站何时将被激活。

### **将媒体添加到即将推出的&维护模式页面**

您可以将媒体添加到显示在背景中的“即将推出和维护”页面。您可以通过以下方式自定义媒体:

*   编写自定义标题
*   选择媒体类型
*   上传文件或输入 URL
*   将媒体分配到网站模式
*   添加新媒体或删除现有媒体

### **关于 IP 地址的网站流量白名单**

您可以通过从后端添加某些用户的 IP 地址来授予他们访问权限。

### **展示社交链接以吸引客户**

在您即将发布的页面上显示社交媒体链接，让他们参与社交媒体页面。

### **允许访问者访问 CMS 页面的选项**

当你在你的网站上工作时，让你的访问者访问你的 CMS 页面。
