## [](#async-is-not-quite-promises-yet)async 不是挺承诺的吗(还？)

在我们的代码库中，我们已经从承诺转移到了异步/等待，在很大程度上，这是相当顺利的。代码行更少，表达能力更强等等等等。

但是我们有一行代码检查传递的变量是否是一个承诺:

```
if (promise instanceof Promise) ... 
```

事实证明这对于异步是失败的...有时候。

```

// -> false 
```
