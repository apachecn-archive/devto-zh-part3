# 二分搜索法算法- C 编程实例

> 原文：<https://dev.to/hakan/binary-search-algorithm---c-programming-example-47cg>

二分搜索法算法是一种查找搜索值在数组中的位置的搜索算法。在二分搜索法算法中，每次都要检查数组中间的元素，以找到要搜索的元素。如果中间的元素不等于被搜索的元素，则在被搜索元素的另一半中重复搜索。这样，每步搜索空间减半。

二分搜索法算法适用于排序后的数组。对于未排序的数组，必须首先通过任何排序算法对数组进行排序，才能生成二分搜索法。

二分搜索法算法的步骤:

1-选择数组中间的元素。

2-将选定的元素与搜索到的元素进行比较，如果它等于搜索到的元素，则终止。

3-如果搜索的元素大于所选元素，则在所选元素的主要部分重复搜索操作。

4-如果搜索的元素小于所选元素，则在所选元素的较小部分重复搜索。

5-重复这些步骤，直到搜索空间中的最小索引小于或等于最大索引。

比如说；2,3,4,5,6,7,8,9,22,33,45.

将遵循以下步骤，在这些数字的顺序数组中查找带有二分搜索法的数字 4。

选择数组的中间元素为 7，并与搜索的元素 4 进行比较。搜索的元素(4)不等于中间元素(7)，即位于数组中心的小于 7 的部分。

我们新的搜索序列:23456。

新搜索数组的中间元素是 4，搜索完成。

使用二分搜索法算法，可以在一个 N 元素数组中找到要进行 log2N 次比较的搜索值。

如果我们尝试在一个顺序数组中实现二分搜索法算法，示例 C 代码如下。

c 示例:

```
 #include <stdio.h> int main()
{
    int array[11] = {2,3,4,5,6,7,8,9,22,33,45};
    int low       = 0;
    int high      = 10;
    int flag      = 0;
    int s         = 4;

    while(low <= high){
        int index = low+(high-low)/2;
        if(array[index] == s){
            flag = 1;
            printf("Founded: %d \n",index);
            break;
        }
        else if(array[index] < s){
            low   = index+1;
        }
        else{
            high = index-1;
        }
    }
    if(flag == 0){
        printf("Not Found!\n");
    }

    return 0;
} 
```
