## Airbnb 的 JavaScript 风格指南

这是最著名的风格指南之一。

我应该用哪一个？“var”、“const”或“let”，为什么？你可以找到这类问题的答案。引用使用的一个例子。

```
// bad
const bad = {
  'foo': 3,
  'bar': 4,
  'data-blah': 5,
};

// good
const good = {
  foo: 3,
  bar: 4,
  'data-blah': 5,
}; 
```
