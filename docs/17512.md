# Angular 9|8|7 Firebase 教程–在本地存储中保存用户数据

今天，我们将创建一个简单的应用程序，其中我们将使用 Angular 7 和 Firebase 实时数据库在本地存储中保存用户数据。我将向您展示如何在 Angular 和 Firebase 中刷新页面后维护用户状态。

我将使用一个简单的逻辑，当用户登录应用程序时，我们将**在本地存储中保存用户数据**。当用户从 Angular 应用中退出时，我们从本地存储中删除用户数据，并将其设置为 **null** 。
