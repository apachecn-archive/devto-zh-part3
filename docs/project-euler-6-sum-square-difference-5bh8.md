# 欧拉项目#6 -平方差和

> 原文：<https://dev.to/peter/project-euler-6-sum-square-difference-5bh8>

将精彩的社区解决方案延续到[项目 Euler](https://projecteuler.net) 。

这是[题 6](https://projecteuler.net/problem=6) ，平方差求和。

> 前十个自然数的平方和是，
> 
> 1² + 2² + ... + 10² = 385
> 
> 前十个自然数之和的平方是，
> 
> (1 + 2 + ... + 10)² = 55² = 3025
> 
> 因此，前十个自然数的平方和与和的平方之差为 3025385 = 2640。
> 
> 求前一百个自然数的平方和与和的平方之差。
