Ruby 是最简单易学的编程语言之一。其中一个关键领域是可读性。然而，随着代码库的增加，复杂性也会增加。

那么你到底用 Ruby 做什么呢？

以下是一些常见的使用案例:

*   使用 Rails 的 Web 应用程序(Ruby on Rails)
*   系统实用程序
*   图书馆
*   服务器

Ruby 已经成熟了很多。它已经发展了 20 多年，并将继续发展。如今，Twitter、Airbnb、Shopify 和其他网站都在使用 Ruby。

Rails 仍然是最流行的用例之一。
