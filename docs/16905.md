# 你讨厌它，然后你爱它。

> 原文:[https://dev.to/diek/you-hated-it-and-then-you-love-it-1d7a](https://dev.to/diek/you-hated-it-and-then-you-love-it-1d7a)

不是新年的事，是我 2018 年的启示。我是一个自学成才的 devop，然后在大学当工程师，现在我又是一个自学成才的人。我一直讨厌前端，它一直是我的痛苦...然后嘭，`react`和`angular`几个月前来到我身边，就像海啸一样，它们抹去了我所有的感觉。

现在，我开发`angular`应用程序并指导`typescript`，我爱上了前端，我可以构建应用程序而不是页面，委托 angular 和 bootstrap 来完成我讨厌的工作:)

所以，我想问，你对任何技术都有这种感觉吗？也许社区改变了你讨厌/不喜欢的东西，然后它变成了你的主要资源或技能之一？

Angular 给我做的。
