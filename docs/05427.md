# API.RSS

你能找到的最好的 RESTful RSS 体验。

API。RSS 将 feed 数据变成了一个很棒的 API。该 API 简化了处理 RSS、Atom 或 JSON 提要的方式。您可以使用一个简单、快速且干净的 REST API 添加并跟踪您最喜欢的提要数据。所有条目都通过机器学习和语义引擎来丰富。
