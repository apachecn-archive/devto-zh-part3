# C#。NET 和 Visual Studio - 2018 年回顾

> 原文：<https://dev.to/calvinallen/c-net-and-visual-studio-2018-year-in-review-2ldp>

> 我对今年 CSAdvent 的第二个贡献(也是最后两个帖子之一！)，由我的好友马特·格罗夫斯主持。访问[https://crosscuttingconcerns.com/The-Second-Annual-C-Advent](https://crosscuttingconcerns.com/The-Second-Annual-C-Advent)查看完整的帖子日历！

对于 C# /来说，过去的一年是美好的一年。NET / Visual Studio 开发人员社区，所以让我们快速回顾一下来自微软的一些伟大的事情(以及它们在这一年中的进展)！

## 2018 年 1 月

2018 年 1 月主要是关于“预览”，因为我们得到了:

*   Visual Studio 2017 版本 15.6 预览版 2
*   Visual Studio for Mac 版预览
*   ASP.NET MVC 5 . 2 . 4、Web API 5.2.4 和网页 3.2.4 的预览版 1

除了预览版，我们还获得了:

*   。NET 框架 4.7.1

## 2018 年 2 月

二月份发布了一些路线图(随后发布了预览版)，例如:

*   。网络核心 2.1
*   EF 核心 2.1
*   ASP.NET 核心 2.1

二月份还发布了 ASP.NET MVC 5 . 2 . 4、Web API 5.2.4 和网页 3.2.4 的完整版本，以及一些预览版:

*   ASP.NET 核心 2.1.0 预览版 1
*   Visual Studio 2017 版本 15.6 预览版 4(我们是不是跳过了 15.6 预览版 3？)
*   。NET Framework 4.7.2“早期访问内部版本 3052”
*   。NET Core 2.1 预览版 1

在“更令人兴奋的”类别中，我们第一次尝到了 Blazor 作为一个实验项目的味道，该项目承诺在浏览器中运行 C#和 Razor！

## 2018 年 3 月

在 3 月份，我们获得了 Visual Studio 2017 版本 15.6 和 Visual Studio for Mac 7.4 的完整版本，紧接着是 Visual Studio 2017 版本 15.7 的第一个和第二个预览版以及 Visual Studio for Mac 7.5 的第一个预览版！

还有，随着 VS 2017 15.6 的发布，我们看到了 F#语言和工具的更新！

还有，更多预览！

*   。NET Framework 4.7.2“早期访问内部版本 3056”
*   ASP.NET MVC 5 . 2 . 58、Web API 5.2.5 和网页 3.2.5 的预览版 1
*   布拉索的首次“公开”预览！

## 2018 年 4 月

Visual Studio 2017 版本 15.7 预览版 3 掉了，还有一个 Visual Studio 2017 路线图(我要说，有点晚了)。

其他预览也开始下降，例如:

*   ASP.NET 核心 2.1.0 预览版 2
*   。NET Core 2.1.0 预览版 2
*   英孚核心 2.1.0 预览版 2
*   吹 0.2.0

最后，一个完整的版本。NET Framework 4.7.2，并发布了一个新的“库管理器”,用于管理 web 应用程序的客户端内容。

## 2018 年 5 月

五月见证了一系列激动人心的发布:

*   Azure 存储浏览器
*   ASP.NET MVC 5 . 2 . 5、Web API 5.2.5 和网页 3.2.5
*   Visual Studio 智能代码
*   Visual Studio for Mac 版
*   RC 1 of。网络核心 2.1
*   EF 核心 2.1 的 RC 1
*   ASP.NET 的 RC 1 2.1
*   Blazor 0.3.0“实验性”
*   ML.NET
*   Visual Studio 2017 版本 15.7
*   Visual Studio 2017 版本 15.7 的 F#语言和工具更新
*   C# 7.3

在五月结束之前(但仅仅是勉强！)，我们得到了更多-

*   。网络核心 2.1
*   EF 核心 2.1
*   ASP.NET 核心 2.1
*   Visual Studio 2017 版本 15.8 预览版 2

梅也向我们宣布了。NET Core 3 与 WinForms 和 WPF 的支持！

## 2018 年 6 月

一开始(嗯，差不多)，我们就得到了 ML.NET 0.2。NET Framework 4.8 早期接入 build 3621，Blazor 0.4.0“实验”。

快到月底的时候，我们看到了 Visual Studio 2017 版本 15.8 预览版 3，以及 Visual Studio 2019 的第一个公告(我能找到的)。

今年的第一份停产(简称 EOL)声明也是在 6 月发布的，当时宣布。2018 年 10 月 1 日网芯 2.0 EOL。

## 2018 年 7 月

七月没什么事，但我们确实得到了一些东西:

*   ML.NET 0.3
*   Blazor 0.5.0“实验性”
*   。NET 框架 4.7.2
*   。NET Framework 4.8 早期访问内部版本 3622
*   F# 4.5 预览

## 2018 年 8 月

八月是七月的重要组成部分，有一些大人物:

*   Visual Studio 2017 版本 15.8
*   库管理器(Visual Studio 2017 15.8 的一部分)
*   Visual Studio for Mac 版
*   库管理器 CLI
*   ASP.NET 核心 2.2.0 预览版 1
*   ML.NET 0.4
*   F# 4.5

## 2018 年 9 月

九月有很多预告！

*   Visual Studio 2017 版本 15.9 预览版 2
*   ASP.NET 核心 2.2.0 预览版 2
*   英孚核心 2.2.0 预览版 2
*   。NET Core 2.2.0 预览版 2
*   ML.NET
*   。NET Framework 4.8 早期访问内部版本 3646

## 2018 年 10 月

10 月份发布了许多预览版，与 9 月份非常相似:

*   ML.NET
*   ASP.NET 核心 2.2.0 预览版 3
*   。NET Core 2.2.0 预览版 3
*   英孚核心 2.2.0 预览版 3
*   Blazor 0.6.0“实验性”
*   Visual Studio 2017 版本 15.9 预览版 3
*   。NET Framework 4.8 早期访问内部版本 3673

我们收到了 Visual Studio 2019 for Mac 的第一份公告，以及一份 Visual Studio for Mac 的路线图！

## 2018 年 11 月

随着假期的临近，预告片的数量正在减少(因为有几部电影会全面上映):

*   预览版中 VSCode 的 Razor 支持
*   Blazor 0.7.0“实验性”
*   。网络标准 2.1
*   ML.NET
*   。NET Framework 4.8 早期访问内部版本 3694

最后我们收到了 Visual Studio 2017 版本 15.9，以及 Visual Studio for Mac 版本 7.7！

## 2018 年 12 月

一旦微软的员工从火鸡昏迷中恢复过来，我们就收到了一些提前的节日礼物:

*   ML.NET
*   。网络核心 2.2
*   ASP.NET 核心 2.2
*   EF 核心 2.2

这个月我们还得到了一些不错的预览，比如:

*   Visual Studio 2019 预览版 1
*   。网络核心 3 预览版 1
*   C# 8.0(注意:并非“所有”功能都可用)
*   。NET Framework 4.8 早期访问内部版本 3707

* * *

伙计们，2018 年就要结束了。我们收到了来自微软的大量更新——从新的实验框架(Blazor)到“核心套件”的快速迭代。NET、EF 和 ASP.NET)和 Visual Studio。

随着新的一年的到来，我们也有一些令人兴奋的事情值得期待，例如 Visual Studio 2019 的发布(尽管预计会先有一些预览)。NET Core 3，C# 8.0，还有。NET Framework 4.8(可能暂时是最后一个了)。
