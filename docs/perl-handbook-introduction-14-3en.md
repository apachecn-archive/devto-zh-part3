# Perl

* * *

Perl。每个人都在谈论的 Perl 是什么？快速浏览一下语言语法，您可能会明白为什么它与 MD5 加密相比——在加密之后，以及为什么最受欢迎的 Perl 书籍之一有一幅怪异骆驼的图像。但是 Perl 对大多数开发人员来说有很多好处。作为一种简短快速的脚本语言，它不需要编译，这意味着一旦你习惯了它，日常工作的自动化只需几分钟。梦想制作一些脚本，运行在世界另一端的服务器上，与其他大陆的计算机交流？PHP 是最流行的 web 脚本语言之一，几乎完全是 Perl 的衍生物。意味着一旦您建立了自己的 Perl 知识库，并训练自己在别人讲 Perl 笑话时开怀大笑，那么提高 PHP 对您来说将是小菜一碟。

在本手册教程的最后，你将学会用脚本语言编码。“但是这和我所知道的脚本语言有什么不同呢？”对你来说也是个好问题。

除了我们已经提到的所有东西之外，Perl 还有一个非常独特和巨大的优势，其他编程语言没有这个优势。(你现在可以带着温和的笑声开始训练)。Perl 有一个独特的 Regex 语法，这使得日志文本解析任务的速度提高了 5 倍。尽管有时只需要几秒钟，但您会发现，在用 Perl 而不是任何其他编程语言编写简单任务时，令人不安的停顿要少得多。无论您是愿意成为 UNIX 迷还是已经成为了 UNIX 迷，Perl 都是适合您的语言，因为当您解析 Android 日志文件和用连接的电话调试代码库时，从 Perl 正则表达式转移到 UNIX 正则表达式的感觉要好得多，您会惊讶于以前没有掌握它。
