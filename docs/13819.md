# 作为刚毕业的计算机工程师的观点 part1

> 原文：<https://dev.to/burakaykan/point-of-view-as-just-graduated-computer-engineer-20p6>

我刚从学校毕业，大约 3 个月前找到了一份工作。我只想分享我目前为止的观察，我目前为止注意到的检查点。最初几周，我试图学习公司的技术(例如语言、框架)和我们的产品。首先，我试图检查项目的代码来理解他们，后来看了一些公司提供给我的课程。在我开始工作的三周后，我被分配去修复一个主要产品的小缺陷。我非常非常害怕，这是我作为计算机工程师的第一次职业经历。我在队友的帮助下修复了这个错误(谢谢大家！).

后来，我开始阅读这些语言和框架，尽可能多地寻找资源。

我想和你分享的观点。请击鼓...我发现每个人都有自己独特的学习方式，所以我的愚见请**停止**和**思考**。
