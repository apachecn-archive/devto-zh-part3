# 让人有情绪

> 原文:[https://dev.to/ashleemboyer/let-people-have-emotions-4i5](https://dev.to/ashleemboyer/let-people-have-emotions-4i5)

工作(希望)不会耗尽我们的全部生活。我们每一个人都来自不同的背景，我们都处于非常不同的阶段。尽管这个事实显而易见，但人们在工作中表达情绪仍然会被打上一个可笑的烙印。我们不是机器人，有什么关系？

让我们首先思考人们基于性别偏见对“情绪化”行为的看法——我认为这很重要。如果有人说一个女人在工作中情绪化，他们可能会认为她在不同意一种说法或纠正某人时生气或有侵略性，如果他们打断她的话。同样是这些人认为这是男人的正常行为，并不认为他们因此而情绪化。想想看，男人会因为他们在工作中的任何行为而被称为情绪化吗？在我自己的亲身经历中从来没有见过。

但是，如果我们都能在工作中表现出一些情绪，而不被视为一件坏事，会怎么样呢？我们的工作令人沮丧。软件令人沮丧。客户令人沮丧。如果有人因为一个他们花了几天或一周研究的 bug 仍未修复而在办公桌前暴跳如雷 30 秒，那又如何？只要他们不扰乱他们的队友，我看不出有什么问题。

如果有人正在经历个人生活中的一些事情，比如失去一个所爱的人，让他们有一段时间的空间。他们可能会安静几天。他们可能会趴在桌子上哭，或者躲在浴室里。如果你看到他们这样做，只要支持他们。这并不难，也不必尴尬。同样，如果他们信任你，你也要保守秘密。你没资格闲聊别人的情况。

所以，让我们把工作中的人正常化。我知道这是一个全社会的问题，但我想不出比在你的工作场所更好的开始解决这个问题的地方，与你每天打交道的人在一起。
