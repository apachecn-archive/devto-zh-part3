# 你最喜欢的关于软件工程历史的书籍有哪些？

> 原文：<https://dev.to/mpermar/what-are-the-books-about-software-engineering-history-that-you-have-enjoyed-most-486>

我最喜欢阅读的类型之一是关于软件历史的书籍。告诉你软件公司的真实故事和经验的书，讲述他们如何成为现在的样子，或者他们如何发布你经常使用的产品。

我把它们定义为青少年怀旧书籍。描述那些你花了无数时间制作的游戏的书。如果你问我的话，《毁灭大师》和《波斯王子》是金子。但是肯定还有！《在软件中做大》是一组关于著名软件领导者的文章，讲述了他们的故事。

所以，我请求这个社区。有哪些关于软件历史的书籍推荐阅读？
