# 船只或来源

> 原文：<https://dev.to/m1guelpf/ship-or-source-foe>

[https://app.blogcast.host/embed/295](https://app.blogcast.host/embed/295)

昨天，正当我要睡觉的时候，我有了一个主意。我在想像 Peter Levels 的*[Go Fucking It](https://gofuckingdoit.com/)，*这样的网站，你给自己设定一个目标，如果没有实现就支付 100 到 1000 美元。具体来说，我在想这个网站如何提高生产率，以及我如何能做类似的事情(不用付钱，因为我是一个青少年，没有太多我可以失去的东西)来帮助我实际上完成我所有的代码项目，而不是让它们不完善。

然后，我想到了一个主意。我称之为船或源。挑战在于为你希望项目启动的时间设定一个明确的截止日期。如果到那一天产品还没有在 ProductHunt 上发布，那么你的存储库就公开了。

这不仅会激励你构建和发布你的产品，而且还会确保你对这个世界有所贡献，即使你最终离开了这个项目，因为其他人将能够挖掘代码并有希望学到新的东西。

具有讽刺意味的是，这又给我的想法列表增加了一个项目(每天都在增长)，所以我现在将手动完成这个项目(而不是为它创建一个平台)，但是如果你正在寻找一些东西来构建，请随意使用这个:)
