# 编程:初学者指南

> 原文:[https://dev.to/kwmcminn/programming-a-beginners-guide-4kl9](https://dev.to/kwmcminn/programming-a-beginners-guide-4kl9)

你决定要学习如何编码。也许你想学习一项新技能，创建一个你想了多年的应用程序，或者也许想转行。这些都是深入代码世界的好理由。但是你从哪里开始呢？有这么多不同的编程语言、概念、学校、学习平台、教程等等。如果你不知道从哪里开始，我整理了这个指南，提供一些不同的途径/资源，你可以学习如何编码，以及如何实现你的目标。

在我们开始之前，关于最佳编程学习路径有各种各样的观点。该指南包括信息和学习资源，如果您想了解更多信息，还提供了链接。学习编程没有“正确”的道路可走，但是希望这些资源可以帮助你开始你的编码之旅。在一天结束的时候，选择你认为最适合自己的道路是你的责任。

## [](#introresources)简介-资源

哈佛和麻省理工学院提供的这两个课程(在线)对任何想学习计算机科学基础的人都是免费的，是学习编程基础的良好开端。

*   [CS 50](https://youtu.be/5azaK2cBKGw)
    *   “CS50 是哈佛大学对计算机科学和编程艺术的智力企业的介绍。任何人都可以考 CS50，即使不是哈佛的学生。”
*   麻省理工学院:计算机科学和 Python 编程简介
    *   “计算机科学和 Python 编程入门”面向很少或没有编程经验的学生。它旨在让学生了解计算在解决问题中可以发挥的作用，并帮助学生，无论他们的专业如何，对他们编写小程序以实现有用目标的能力充满信心。”

## [](#languages)语言

在你开始写第一行代码之前，你需要确定你将要学习的语言。如果你还没有到需要你的程序执行特定任务的阶段(在某些方面，语言比其他语言更好)，我会推荐下面列出的一种语言。Python 是最受初学者欢迎的语言，因为它是一种可读性很强、非常灵活的语言。Javascript 是整个技术行业最流行的语言，但是如果你是从 JS 开始的，我建议你在进入 JS 之前先学习 HTML/CSS。然而，如果你不是从 JS 开始，我建议从 Python 或 Ruby 开始，因为 HTML 和 CSS 不是编程语言，这可能会让新的程序员感到困惑。

*   [Python](https://www.python.org/doc/essays/blurb/)
    *   “Python 相对简单，所以很容易学习，因为它需要一种专注于可读性的独特语法。开发人员可以比其他语言更容易地阅读和翻译 Python 代码。反过来，这降低了程序维护和开发的成本，因为它允许团队在没有明显语言和经验障碍的情况下协同工作。”
*   [红宝石](https://en.wikipedia.org/wiki/Ruby_(programming_language))
    *   “许多 Ruby 开发人员发现使用 Ruby 很有趣，在学习编码时，有趣是一个很大的动力。Ruby 是一种非常高级的语言，这意味着 Ruby 抽象掉了机器的大部分复杂细节。因此，您可以用更少的代码快速从头开始构建一些东西。Ruby 因 Ruby on Rails framework 而变得流行，这是一个全栈的 web 框架，使原型开发变得轻而易举，使它成为许多初创公司和编码初学者的首选 web 框架。”
*   [HTML/CSS](https://www.w3.org/standards/webdesign/htmlcss)
    *   超文本标记语言(HTML)是创建网页和 web 应用程序的标准标记语言。它与层叠样式表(CSS)和 JavaScript 一起构成了万维网的三大基石技术。”
*   [Javascript](https://www.w3schools.com/whatis/whatis_js.asp)
    *   “它[Javascript]是一种编程语言，其特点是动态、弱类型、基于原型和多范例。与 HTML 和 CSS 一样，JavaScript 是万维网的核心技术之一。[9] JavaScript 支持交互式网页，是 web 应用程序的重要组成部分。绝大多数网站都使用它，[10]而且主要的网络浏览器都有专门的 JavaScript 引擎来执行它。”

## [](#personal-projects)个人项目

大多数程序员都会认同的一点是项目的价值，尤其是你想要构建的个人项目。我可以自信地说，我的大部分学习和理解都可以归功于我所构建的项目。项目不仅有趣、刺激，而且最终比书本和教程更有效。使用书籍、教程和视频来学习基础知识...然后当你觉得你有足够的基础去创造一些东西的时候，就过渡到项目模式。即使当你开始一个新项目时，你也不会知道完成项目所需要知道的一切，但是有了这个基础并利用网上资源(下一节)可能会让你有一个良好的开端。

如果你正在努力思考初学者项目，[看看这个列表](https://knightlab.northwestern.edu/2014/06/05/five-mini-programming-projects-for-the-python-beginner/)，它包含了五个 Python 小项目。

## [](#bootcampsprep)训练营/预备

写这篇文章的时候，我是华盛顿州西雅图熨斗学校的软件工程学生。如果你不熟悉训练营，他们被定义为
*...教授雇主需要的编程技能的技术培训项目。它们使不太精通编码的学生能够专注于编码的最重要方面，并立即应用他们的新编码技能来解决现实世界的问题。关于训练营编码有不同的观点，因为训练营教授的大部分内容都可以免费自学。根据学校和地点的不同，训练营的费用从 5000 美元到 15000 美元不等。对我来说，这是一个让我沉浸在编程世界并加速我的学习的绝佳选择。你可能会发现训练营非常适合你，或者你可能更喜欢自己学习，两者都是很好的选择。我对训练营的建议是，把重点放在那些有着成功就业历史的学校上，并提供专门的职业服务，帮助你在毕业后找到理想的工作。*

下面的三个编码训练营都有免费的预备课程(我已经链接到了),你可以在网上参加，它们都是很棒的学习平台。我强烈推荐利用这些免费课程开始你的学习。熨斗学校的免费预备课程甚至包括与真正的教师进行即时聊天，能够在工作时帮助你。

*   [熨斗学校](https://flatironschool.com/free-courses/coding-bootcamp-prep/)
*   [全堆叠学院](https://www.fullstackacademy.com/bootcamp-prep)
*   黑客攻击反应堆

## [](#interview-prep)面试准备

[![](../Images/d6a5c43bcb796a2b936c4d96a33e70c7.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--2zjRLq2q--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://cdn-images-1.medium.com/max/2000/1%2AvLadxMniF0KW0HWeGkipJw.jpeg) 
如果你为了转行或首次进入就业市场而学习代码，你可能需要练习面试风格的问题，并学会谈论你的思维过程和代码。以下是准备面试的一些重要资源。

*   [破解编码面试](http://www.crackingthecodinginterview.com/)

    *   破解编码面试可以说是最受欢迎的面试准备资源。“破解编码面试给了你获得顶级软件开发工作所需的面试准备。这是一本技术性很强的书，侧重于在面试中胜出的软件工程技巧。这本书超过 500 页，包括 150 个编程面试问题和答案，以及其他建议。”
*   [牛逼的面试问题](https://github.com/MaximAbramchuck/awesome-interview-questions)

    *   按类别/语言分类的技术面试问题精选列表
*   [Leetcode](https://leetcode.com/)

    *   “LeetCode 的目的是为您提供真实编码面试问题的实践培训。在线法官会对你的算法的正确性和效率给出即时反馈，这有助于获得良好的学习体验。”
*   [黑客排名](https://www.hackerrank.com/)

    *   “HackerRank 是一个技术招聘平台，是评估全球 1000 多家公司开发人员技能的标准。通过使技术招聘人员和招聘经理能够在招聘过程的每个阶段客观地评估人才，HackerRank 帮助公司雇佣熟练的开发人员，并更快地进行创新。”

## [](#toolsresources)工具/资源

*   [堆栈溢出](https://stackoverflow.com)
    *   Stack Overflow 成立于 2008 年，是最大、最值得信赖的在线社区，供开发人员学习、分享知识和发展职业生涯。堆栈溢出一直是我自己编码之旅中的一个主要问题。几乎你的每一个问题都可能成为讨论堆栈溢出的话题。它将是你学习代码时最好的朋友。
*   [Repl.it](https://repl.it/)
    *   Repl .这是一个简洁的网站，你可以选择一种编程语言，写一些代码，它会立即在右边显示代码的输出。这是我开始时最喜欢的工具，因为我可以测试代码，而不必在我的计算机上保存/打开文件。
*   CSS 工具
    *   Flexbox Froggy 一款通过编写 CSS 代码来帮助 Froggy 和朋友的游戏！
    *   [格子花园](https://cssgridgarden.com/)
    *   [数码色度计](https://support.apple.com/guide/digital-color-meter/welcome/mac) *Mac only*
*   [Rubular](https://rubular.com/)

    *   在浏览器中测试正则表达式
*   文本编辑器(编写代码的地方)

    *   [Atom](https://atom.io/)
    *   [VScode](https://code.visualstudio.com/)
    *   [升华](https://www.sublimetext.com/)
