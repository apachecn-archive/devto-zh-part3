IOS 众多出色的辅助功能之一是**缩放**。

[![](../Images/738dda0cdfec6a274a2cdb3b4d7bf167.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--b2RbuEQp--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/qtch05uzhg6y8lph6zo7.PNG)

您可以用三个手指双击来激活和停用它。似乎用苹果的 UIKit 构建的应用程序可以很好地处理这一点，但我经常在使用其他 UI 库的应用程序中意外激活按钮或执行其他不想要的操作。在游戏中也是如此，我喜欢放大阅读小文本，或者更详细地查看游戏元素。

我知道可访问性并不总是最令人兴奋的部分，尤其是在游戏中，但是它真的很重要！
