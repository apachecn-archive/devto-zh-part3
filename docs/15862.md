## [](#engines)引擎

*   [YaneuraOu mini](https://github.com/yaneurao/YaneuraOu) 是一个 shogi 引擎(AI 播放器)，比 Bonanza6 更强，教育性和微型代码(约 2500 行)，USI 兼容引擎，能够通过 VC++2017 编译
*   gunyarakun/python-shogi 一个纯 Python shogi 库，具有 move 生成、验证和通用格式处理功能。
*   [niklasf/python-chess](https://github.com/niklasf/python-chess) 一个纯 python 象棋库，具有棋步生成和验证、PGN 解析和编写、多语言开卷阅读、Gaviota tablebase 探测、Syzygy tablebase 探测和 UCI/XBoard 引擎通信

## [](#protocols)协议

*   [CSA 通信协议](http://www.computer-shogi.org/protocol/tcp_ip_server_112.html):从第 17 届世界计算机将棋锦标赛开始该协议是必须的。 有使用审判服务器的服务器协议和审判服务器不可用时的 1 对 1 协议。
*   [USI 协议](http://hgm.nubati.net/usi.html):连接将棋 GUI 和将棋引擎，进行棋谱分析、对局
