# 从 IT 管理转向网络开发有多难

> 原文：<https://dev.to/frcharry/how-hard-is-to-switch-from-it-management-to-webdev-4eb1>

[![From IT to WebDev](img/5b36f955e98ad81499522af922e4db30.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--0KJdeZje--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/fznpeut5vvbb910h5jvl.png)

如今，在一切都在飞速发展的情况下，转行有多难？我想和你分享我的经验，我是哥伦比亚波哥大的一名系统工程师，在 IT 管理方面有大约 15 年的经验。从数据中心、网络到特定的公司应用程序，如 ERP、CRM 等。
几年前，我决定转行到 WebDev，我开始阅读 HTML5、CSS、JS、Bootstrap 和 Ruby on Rails 等框架，但是正如你所想象的，这还不够。要真正进入 web 开发世界，需要很多承诺，也要走一条路，不管是什么路。开始做。
这就是为什么我报名参加了由 FEDESOFT 赞助的第一个训练营，FEDESOFT 是一个非营利实体，其目标是促进哥伦比亚软件领域的公司发展。在那里，我了解了 Ruby on Rails 和混合应用程序，以及如何用 Heroku 这样的工具开发和部署它们。
目前，我注册了一个新的 Bootcamp，其重点是 Ruby on Rails 的 FullStack，它涵盖了 Javascript、Git、Bootstrap、HTML5、CSS 等主题。
但是，要回答我以前发帖的问题，有多难？嗯，在波哥大，如果你能认可一些经验，展示你的作品，或者接受技术面试的评估，似乎并不难。

在我看来，你必须在一定年龄以下(我 36 岁)才能被认为是大三学生，但就我而言，我有大约 15 年的 IT 经验，但不是软件/网站开发，要想找到一份收入不错的工作，这真的是一个挑战。方法是在你自己的下面走。尝试在 UpWork 或 Freelancer 之类的地方获得自由职业者的工作，以增加你的投资组合的经验，并保持学习/实践/学习，专注于新的框架/语言，以转向没有足够技能人才的新领域，并能够被视为有趣的候选人。现在，我继续尽我最大的努力，希望很快就能用我的第一份网页开发工作写一篇新文章。
