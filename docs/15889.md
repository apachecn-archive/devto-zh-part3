# Changelog:导出注释

> 原文:[https://dev.to/rhymes/changelog-export-comments-56mf](https://dev.to/rhymes/changelog-export-comments-56mf)

几个小时前，由于一个优雅的[合并拉请求](https://github.com/thepracticaldev/dev.to/pull/1471)，除了你自己的帖子，你还可以导出你自己的评论。

要做到这一点，你去[https://dev.to/settings/misc](https://dev.to/settings/misc)并选择“导出内容”
