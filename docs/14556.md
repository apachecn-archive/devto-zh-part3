# 移动开发者

> 原文:[https://dev.to/kodekage/the-mobile-developer-1ece](https://dev.to/kodekage/the-mobile-developer-1ece)

我在 2017 年开始用我的移动设备(Android)学习编码，这真的是我作为一名 web 开发人员的旅程的决定性时刻(我正在逐渐过渡到软件开发)。

我有足够的热情开始用我的移动设备学习编码，克服伴随着用移动设备学习编码的所有困难和不适。

这些年来，我注意到大多数初学者向我咨询如何学习编程的问题，就像我刚开始时一样，没有笔记本电脑或台式电脑来学习。所以我总是鼓励他们开始使用他们的移动设备。

但是我决定更进一步，写一本教初学者如何通过学习 web 开发开始编码的书。我几乎完成了这本书的写作，并为最终出版的时候喜欢读这本书的读者创建了一个等候名单。

## [](#what-to-expect)期待什么

*   对 HTML、CSS 和 JavaScript 等 web 技术的基本和深入解释
*   构建肌肉记忆的示例代码
*   学习编码的秘诀
*   继续成为 beat 开发人员的动力。
