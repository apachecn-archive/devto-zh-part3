# 介绍文章

> 原文:[https://dev.to/cw00dw0rd/intro-post-1623](https://dev.to/cw00dw0rd/intro-post-1623)

我很高兴成为 DEV 的一员，并希望我能够与社区一起学习！

在过去一年左右的时间里，我一直在认真地追求成为一名开发人员的梦想。我离开了以前的职位，把所有的精力投入到发展中，这是一个“要么行动，要么闭嘴”的时刻。我已经有了一些 Java 和 Python 的入门经验，并决定尝试一下 web 开发。我用 Python、Django、Bootstrap、HTML、CSS 和 Linux 创建了我的第一个网站。[https://www.newfangledcoffee.com](https://www.newfangledcoffee.com)

如今，我完全是 JavaScript，尽管我仍然喜欢 Python。我曾在 React、Vue 和 AngularJS 工作和/或创建网站。我摆弄过电子之间的所有东西。JS，Android，Unity，Blender，还有最近的 react-native。一路走来，我发现了自己对面向文档的数据库的热爱，学习了，然后通过了我的 MongoDB 4.0 开发者认证考试！我的旅程中最有收获的部分是发现所有我不知道的事情，并试图用新发现的知识提高我的技能。我的目标不是“全部学会”，而是学会如何更好地做我正在做的事情。如果我每天都能做到这一点，那就一定会有好的结局，对吗？

我总是很乐意交朋友或帮助一个项目，所以请随时伸出援手！

我在坦帕，正在积极寻求当地或偏远地区的职位，所以如果你知道任何或正在寻找一个充满激情的开发人员，让我知道！

感谢阅读，我们得到了这个！
