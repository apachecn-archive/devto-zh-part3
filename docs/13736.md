## TL；速度三角形定位法(dead reckoning)

Travelport Smartpoint SDK 用于增强 Travelport Smartpoint 的功能，这是一种更智能的预订方式，如航空公司、酒店、汽车等。我使用 Smartpoint SDK 创建了一个插件来获取 Smartpoint 控制台中的第三方数据。
