# 权威不是领导

> 原文：<https://dev.to/jesperreiche/authority-is-not-leadership-4pip>

如果你管理工具箱中唯一的工具是权威和你行使权威的能力，那么你就不是一个真正成熟的领导者。通过权威的方式让人们追随，就像用枪指着脑袋领导人们一样生硬、单一——这不需要太多的天赋。

伟大的领导力不是来自等级、资历或权力。伟大的领导力来自行动、结果和信任。

行动是你行使领导力的方式。你如何让人们执行任务，这些任务如何组合产生结果。通过行动和结果，你建立信任。当人们看到你的行动以一种有意义、关怀和热情的方式进行。当他们看到这些行动是如何结合起来，从愿景变成结果。然后他们会开始信任你。然后他们会开始尊重你这个领导。

如果他们看到的只是你在行使你的权力时没有同情心，没有激情，那么他们可能会因为你的权威而照做。但是他们永远不会尊重你，他们永远不会刻意去追随你，或者付出额外的努力来实现目标。

太多的管理者和领导者只会用生硬的武力让人们执行他们需要的任务。太多的管理者和领导者没有付出足够的自我，对他们领导的人或他们为之奋斗的事业/愿景表现出同情。

如果你想让人们付出额外的努力，并感受到同样的承诺，那么你必须带头，向人们展示你既关心他们，也关心这项事业。他们可以信任你的领导能力，相信你能够提供实现最终目标所需的结果。

用权力或权威来领导已经成为常态，而用同情、信任和行动来领导则是例外。为了交付更好的结果，让被管理的人过上更有意义的生活，我们需要反过来。

这并不容易，也不可能一蹴而就，但我们需要提高标准。我们不应该只以结果来衡量管理者和领导者，而这些结果可以通过蛮力获得，或许我们应该以他们所管理的人有多有意义、他们对自己所做工作的感觉以及他们对经理的信任程度来衡量他们。

毕竟，管理人员是经理/领导者工作的一部分。如果我们只通过这些人的结果来衡量他们的表现，而不是通过这些人对他们所做的工作或管理他们的人的感觉。那么我们永远也不会知道他们是伟大的领袖还是迟钝的当权者。
