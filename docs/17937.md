# 我如何让人们相信隐私的重要性？

> 原文:[https://dev . to/shri pathy/how-do-I-confident-people-on-the-importance of-privacy-eie](https://dev.to/shripathy/how-do-i-convince-people-about-the-importance-of-privacy--eie)

围绕脸书读取人们的信息和谷歌追踪他们所做的所有这些争议，我如何让人们相信他们的隐私很重要？我的朋友们认为这些公司可以使用人们的数据，因为我们使用他们的服务。

我删除了脸书，我不经常使用 Instagram，我从谷歌上删除了我的活动，但最终我看起来像个怪人。为什么人们把隐私看得那么低？
