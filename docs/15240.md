当我看到我的同事和网友获得“Mojaved”后，我就开始拖拖拉拉了，Mojaved 是升级到最新、最棒的 MacOS 后遇到的问题的总称。没人有时间做这个。

到目前为止，一切都很好，但是我马上注意到我的菜单栏有所不同。

<figure>[![](../Images/bb7dde3bf7ac2e94a337f79e77be2b15.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--7kdRZ9ai--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://cdn-images-1.medium.com/max/1024/1%2ApQSLTuPkr42j2eleLVubqw.png) 

<figcaption>【抓取】现为“截图”</figcaption>

</figure>

我一直在抓图。全屏，窗口，部分，以上所有。我从大学开始用 MacBook Pro 就开始用 MacOS 的“Grab”了。不过现在看起来我有了一个新的 app——“[截图](https://support.apple.com/en-us/HT201361#mojave)”。

截图有所有相同的功能，抓取，再加上更多！你不仅可以拍照，还可以录制整个屏幕或部分屏幕，这是我自己在 QuickTime Player 或 Webex/Zoom 会议上经常做的事情。

如果您正在寻找截图，它可能在您的应用程序>实用程序>截图。

再见，我会想你的。❤️
