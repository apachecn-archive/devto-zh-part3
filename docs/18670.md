# 网络应用开发成本:你的网络应用预算终极指南[2018]

> 原文：<https://dev.to/maxchurilov/web-application-development-cost-your-ultimate-guide-to-web-app-budget-2018-4p30>

每个月都会创建数百个 web 应用程序。问题不再是一个公司是否需要一个 web 应用程序，而是构建一个适合公司预算和满足业务需求的应用程序的价格。

如果你在互联网上查找一个网络应用程序的费用，你不会找到一个明确的答案。每个人都希望能这么简单——获得瞬间的估计和时间框架——但事实并非如此。

我们打个比方。

想象一下，你带着一个问题来到一家建筑公司:“盖房子的价格是多少？”。我相信大多数人都会同意，价格取决于很多因素，如房子的类型、房间数量、总建筑面积、外部和内部装修、建筑规划、许可、土地使用权、员工数量、他们的专业知识……基本上，很多细节。
