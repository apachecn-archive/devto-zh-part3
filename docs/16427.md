# 5 个重要会谈的来源

> 原文:[https://dev . to/lethyglistic/5-sources-for-talks-that-matter-4kc 0](https://dev.to/lethargilistic/5-sources-for-talks-that-matter-4kc0)

尽管我知道我不应该这样，但我通常在工作的时候会听这样或那样的演讲。在很长一段时间里，这给我带来了一个问题，因为我能找到的新演讲的唯一可靠来源是 ted 演讲。别担心，这又不是一篇*关于*批评 TED 的帖子——[这样做的 TEDx 演讲](https://www.youtube.com/watch?v=Yo5cKRmJaf0)是有史以来最伟大的 TED 演讲。这是一篇因推荐算法将最肤浅的、表面层次的、夸张的问题讨论推到顶端而产生的沮丧情绪而产生的文章，不幸的是，TED 就是一个例子。

这是我最喜欢的为自己提供新的、有意义的讨论的 5 种方式，主要是通过作者直接让他人参与他们的工作。对于每一个，我都链接了与我们这些从事软件工作的人特别相关的内容的例子。每一个都为我们提供了更多，探索它们是我最喜欢的消磨时间的方式之一。

# [](#1-politics-and-prose)1) [政治与散文](https://www.youtube.com/user/politicsprose)

[https://www.youtube.com/embed/Yc9g8RPcyEg](https://www.youtube.com/embed/Yc9g8RPcyEg)

这个讲座系列由华盛顿特区的[政治和散文](https://www.politics-prose.com/)书店主办，它们吸引了令人眼花缭乱的一系列作者来谈论他们的作品，通常是与 Q & A。我一直在努力解决的一个问题是想读太多的书——这是一个很好的问题——这些源源不断的作者总结了他们自己的作品，帮助我在寻找新的阅读内容时搔搔痒处。

# [](#2-cspan)2) [C-SPAN](https://www.c-span.org/)

[https://www.youtube.com/embed/KbAPtGYiRvg](https://www.youtube.com/embed/KbAPtGYiRvg)

(C-SPAN 不允许嵌入他们网站的内容，所以这里有一个链接，链接到[大卫·格雷贝尔在*图书电视*](https://www.c-span.org/video/?446716-2/bullshit-jobs) 上谈论他的书*狗屁工作*。它更能代表现代节目。)

是的，[“小孩看 C-SPAN”](https://twitter.com/search?q=%22kid%20watching%20cspan%22&src=typd)有点像模因，[美国人把它定性为“呆呆的”](https://twitter.com/search?vertical=default&q=cspan%20dork&src=typd)。不，这不再仅仅是众议院和参议院大楼里的摄像头了。

C-SPAN 始终如一地报道美国国家政治，没有广告，致力于新闻诚信，拒绝党派有线新闻频道所接受的浮华、魅力和错误信息。令人惊讶的是，人们哀叹福克斯新闻频道的党派问题和类似的消息来源，同时指责一个已经解决了这些问题的消息来源*，因为它已经解决了这些问题*。他们有几个节目公正地分析新闻——他们通常更擅长不突出怪人。大多数时候，我听的是他们更多的固定内容，包括政治和散文的书籍讨论，如 [*BookTV*](https://www.c-span.org/series/?bookTv) 或 [*After Words*](https://www.c-span.org/series/?afterWords) 以及他们的一系列电视转播的大学美国历史课，创造性地命名为 [*历史讲座*](https://www.c-span.org/series/?lecturesInHistory) 。他们在课堂上关注的内容通常近乎于讨论性的视频文章，就像有趣的历史内容一样。C-SPAN 的存档内容可以追溯到几十年前。

C-SPAN 移动应用程序提供他们的几个视频系列和所有三个电视频道的音频流点播。(C-SPAN 3 最好。不要@我。)

# [](#3-ipse-dixit)3) [*伊普斯迪克西*](https://shows.pippa.io/ipse-dixit)

[https://www.youtube.com/embed/Gpiu45Hx-Zg](https://www.youtube.com/embed/Gpiu45Hx-Zg)

布莱恩·l·弗莱是肯塔基大学法学院的教授，我很自豪能和他一起在推特上聊天，在 T2 互动。去年秋天的某个时候，他开始休假，并决定持续做一个关于法律奖学金的播客，他花了很多时间。每一集都是对一位法律学者关于他们工作的令人难以置信的采访，它涵盖了一系列疯狂的问题，通常这些事情每天都影响着许多美国人，而大多数人都没有意识到。因为是关于学术的，如果你有兴趣深入讨论，通常有免费的、详细的资源可供参考。他还发布了来自老式录音的“存档”片段，这些片段以同样的方式变化，从信息丰富到怪异到滑稽，甚至它们的存在。

这个 3 个月大的播客有 100 多集常规节目和 30 多集存档节目。

生产这个庞然大物并没有阻止他继续[发表新论文](https://papers.ssrn.com/sol3/cf_dev/AbsByAuth.cfm?per_id=646621)并以其他方式挫败休假的目的。我最喜欢他的论文叫[《抄袭不是罪》](https://papers.ssrn.com/sol3/papers.cfm?abstract_id=2752139)。(下载 SSRN 报纸不需要账号。如果它提示你注册，寻找“匿名下载”按钮。)

# [](#4-librivox)4) [LibriVox](https://librivox.org/)

[https://www.youtube.com/embed/6pPwB1rTJts](https://www.youtube.com/embed/6pPwB1rTJts)

2019 年标志着作品 20 年来首次进入美国的公共领域，因此这是一个欣赏一个免费为人民振兴那些作品的团体的好时机。LibriVox 有声读物和有声戏剧完全由志愿者制作，所有的书籍都通过公共领域属于我们所有人，这意味着大多数(*不是*所有)这些书籍都是在 1924 年之前出版的。有些人认为旧作品与现代观众不太相关，因为我们有太多来自四面八方的新信息，但这是一种不必要的偏见。旧书有很多东西可以教给我们，而[林迪效应](https://fs.blog/2013/08/choose-your-next-book/)告诉我们，保持相关性更长时间的东西更有可能在未来保持相关性。

你可以在这里听听我为他们念过的章节[。](https://librivox.org/reader/12667)

如果我可以站在我的讲台上一秒钟，国际版权法的现状是不可接受的。它已经封锁了整整一个世纪的作品不让公众接触。这在人类历史上第一次剥夺了人们对自己有生之年创造的文化做出反应的任何机会，无论是通过传播他们喜爱的故事，改编他们的故事，还是将他们与其他东西混合在一起。这是不自然的，必须抵制和改革。詹妮弗·詹金斯和我最喜欢的版权学者詹姆斯·博伊尔写了一部名为《盗窃:音乐史 的漫画小说，描述了艺术对文化的反应如何构建了现代文化，以及版权至上主义如何抑制了这种反应。这本书在知识共享许可下可以作为[免费下载](https://law.duke.edu/musiccomic/download/)。([我博客上的作品](https://github.com/lethargilistic/blog/)都是 CC-0。)

# [](#5-recode-decode-and-pivot)5) [*重编码解码*](https://www.recode.net/recode-decode-podcast-kara-swisher) 和 [*枢*](https://itunes.apple.com/us/podcast/too-embarrassed-to-ask/id1073226719?mt=2)

[https://w.soundcloud.com/player/?url=https://soundcloud.com/recode-decode/how-to-fix-the-problems-caused&auto_play=false&color=%23000000&hide_related=false&show_comments=true&show_user=true&show_reposts=false&show_teaser=true](https://w.soundcloud.com/player/?url=https://soundcloud.com/recode-decode/how-to-fix-the-problems-caused&auto_play=false&color=%23000000&hide_related=false&show_comments=true&show_user=true&show_reposts=false&show_teaser=true)

如果你从事技术工作，并且没有关注卡拉·斯威舍在 Recode Media 的工作或者她的专栏文章...你在跟踪谁他妈的*？斯威舍长期以来一直是科技新闻的主要评论员。近年来，她转向更直接地解决美国的文化“技术冲击”,因为我们一次又一次地受到新例子的打击，这些公司兜售他们软件的乌托邦式的未来愿景，而不是他们声称的捐助者。在 *Recode Decode* 节目中，她采访了各种各样的科技专业人士、高管和作者，梳理出我们要去哪里，以及我们如何在不破坏一切的情况下到达那里。Pivot 是她和 Scott Galloway 主持的一档每周新闻讨论节目，Scott Galloway 是 NYU 大学的教授，他写了《四个》( T10、T11、T12、T13 ),探索最大的公司是如何渗透我们的生活并从中获利的。*

她很搞笑。每个人都害怕她，每个人都喜欢她。卡拉·斯威舍是个伟大的。
