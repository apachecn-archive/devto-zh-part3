为什么要学习 Ruby 编程？

Ruby 编程语言既有趣又简单，并且有一个庞大的社区。世界上许多初创公司都在使用它。

[![](img/74804fedfc7f7c0a0247e1ba51117da6.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--0wnItgLn--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://i.imgur.com/cGIMIrr.png)

Rails 框架通常与 Ruby 一起使用，用于制作 web 应用程序。Rails 使得创建 web 应用程序变得很容易。

因为 Ruby 有一个简单的语法，它有助于提高代码质量。
