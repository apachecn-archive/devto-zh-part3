# 就做那个应用吧

> 原文:[https://dev.to/ashleemboyer/just-make-that-app-59gl](https://dev.to/ashleemboyer/just-make-that-app-59gl)

作为一个网页开发游戏的新手，我发现很难接受我所有的想法都是好的。是的，你没看错，*所有的*想法都是好想法。想法和问题类似，都是时机的问题。你的每一个想法都会带来学习的经历。

[![ashleemboyer](../Images/6f3821f9981d9ff93e1077fb2301e842.png)](/ashleemboyer) [## 不要害怕问问题

### Ashlee(她/她)5 月 17 日 192 分钟阅读

#learning #questions #confidence](/ashleemboyer/don-t-be-afraid-to-ask-questions-5g99)

要让某个东西变得很酷，或者让 T2 变得非常有用，让 T4 变得有利可图，压力很大。你不必现在就这么做，尤其是如果你正在学习一个新的框架。你可以做任何你想做的应用程序，无论你出于什么原因。如果没人受伤，就去做吧！

没有愚蠢的想法。有些想法显然比其他的好，但是所有的想法都让我们的创造力源源不断。一旦开始，就很难停下来。最终，你会找到最适合你现状的想法。你所要做的就是坐下来，不加判断地倾听你的想法。让他们尽情地倾诉。

现在，让我们听听那些应用程序的想法吧！也许你会找到有类似想法的人。也许联系一下，考虑一下拼贴。:)
