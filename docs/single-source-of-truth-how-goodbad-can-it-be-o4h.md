# 单一来源的真相，能有多好/多坏？

> 原文：<https://dev.to/vonheikemen/single-source-of-truth-how-goodbad-can-it-be-o4h>

具体来说，在 web 应用程序的上下文中，如果我们将所有东西(是的，**所有东西**)都存储在一个巨大的对象中，会有多好或多坏？任何地方都没有私人政府。

像榆树架构这样的东西和基于这种想法的大量国家管理事物让我充满希望。我喜欢他们允许你写一个“视图”作为一个只依赖于它的输入的函数。但是 javascript 世界中的人们只是把它当作在彼此不“接近”的组件之间传递状态的一种便捷方式。

另一件让我无法停止思考的事情是，firebase 数据库是一个 json 对象。把很多东西放在对象里也没那么疯狂。
