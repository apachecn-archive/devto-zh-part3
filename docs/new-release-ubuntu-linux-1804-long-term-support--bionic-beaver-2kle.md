#### *如何升级你以前的版本。*

Canonical(开发 ubuntu Linux 的公司)发布了期待已久的 Ubuntu 1804 lts——标签为“仿生海狸”。(注:LTS 代表[长期支持](https://wiki.ubuntu.com/LTS))。新的 Ubuntu 18.04 LTS 已经看到[比它的前任有了一些重大的改进](https://wiki.ubuntu.com/BionicBeaver/ReleaseNotes?_ga=2.123539531.1716714468.1552333779-1204732084.1533608967)。

[![](img/03d8c8c3b5438bca095d6ca8c1690534.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--FKzYq_uh--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://i1.wp.com/fossnaija.com/wp-content/uploads/2019/03/ubuntu-18-04-lts-bionic-beaver-released-fossnaija.jpg%3Ffit%3D665%252C414%26ssl%3D1)

canonical 将支持 Ubuntu 18.04 LTS，直到 2023 年 4 月(通常为 5 年)，并提供安全和性能更新。

如果你想试用 Linux，你可以从官方仓库下载 18.04 LTS 版本。但是如果你的电脑上已经安装了 ubuntuinstalled，你可以升级到这个最新版本。

### 升级到 LTS UBUNTU 18.04

要从 ubuntu 版本 [16.04LTS](https://fossnaija.com/linux-installation-dual-booting-with-windows-7/) 或 17.10 迁移到新的 ubuntu 18.04，请遵循以下列举的步骤:

*   打开系统设置中的**软件&更新**设置。
*   选择第三个名为**的选项卡更新**。
*   **对于 16.04 LTS:** 将“**通知我有新版本**”下拉菜单设置为“**任何新版本**”。
*   **对于 17.10:** 对于长期支持版本，将其设置为**。**

*   按下 *Alt + F2* ，在命令框中键入“**更新-管理器-c** ”。

*   更新管理器应该会打开并告诉你 ubuntu 18.04LTS 现在可用了。如果没有可以运行:*/usr/lib/Ubuntu-release-upgrader/check-new-release-GTK*

*   单击升级，然后按照屏幕上的说明进行操作。
