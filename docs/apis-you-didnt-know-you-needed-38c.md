# 您不知道自己需要的 API

> 原文：<https://dev.to/mkrl/apis-you-didnt-know-you-needed-38c>

随着最近一切都成为“服务”，有大量不同的数据源可以提供...东西。当然是作为服务的东西。请看一个你可能听说过或者没有听说过的 API 的小集合。

## 是还是不是？

https://yesno . wtf/# API

做决定很难。为什么不将这项工作外包给第三方微服务，以确保他们尽可能独立？每第 10.000 次返回“也许”作为奖励。

## 玉米饼 API

[https://github.com/evz/tacofancy-api](https://github.com/evz/tacofancy-api)

你爱上玉米卷了吗？决定不了今天做什么？一个 HTTP 请求将为您提供一份完整的食谱(包含您选择的可选成分和类别)。

## 企业扯淡

[https://github.com/sameerkumar18/corporate-bs-generator-api](https://github.com/sameerkumar18/corporate-bs-generator-api)

您是否发现很难与高度企业化的客户交谈？你经理的经理是否希望你在每周的“团队效率提升 0.5%”会议上提出一个新主题？这会让你得到掩护。

## 是这样对那样的！

[http://itsthisforthat.com/api.php](http://itsthisforthat.com/api.php)

你的另一个朋友辞掉工作，把所有的钱和时间都投入到智能袜上？

## 全部大写为服务

[http://shoutiloud . io/](http://shoutcloud.io/)

当事情很重要时，大喊大叫是 100%的沟通方式。

还提供 Ruby gem、npm 包和 go 库。

## 将重要的事情自动化

[http://www.foaas.com/](http://www.foaas.com/)

高度先进和灵活的 RESTful API，完成一个告诉人们滚蛋的常见任务。

自定义名称，签名和其他选项的负载是可用的。和不安的外国人交流有困难？支持 i18n，以及上面列出的 SHOUTCLOUD 集成。

## Lorem Ipsum

如果你厌倦了填充你的模板布局的枯燥的 Lorem Ipsum，看看[Bacon Ipsum](https://baconipsum.com/json-api/)——一个丰富的虚拟文本 API。如果你是素食主义者，[滑板主题的 Ipsum](http://skateipsum.com/) 或[棒球](http://baseballipsum.apphb.com/Home/RestApi)一个就能满足你。

这里有一个[减价风味的](https://github.com/jaspervdj/lorem-markdownum#http-api) Ipsum 生成器作为奖励。

## 以动物为役。

著名的 [Cat API](https://thecatapi.com/) 搭配 [Dog](https://dog.ceo/dog-api/) 和  API，以及更加轻量级的随机 [fox](https://randomfox.ca/floof/) 、 [cat](https://aws.random.cat/meow) 和 [dog](https://random.dog/woof.json) 服务，每次你需要的时候都会得到暖心的 JSON 响应。

## 唐纳德·特朗普 API

[https://docs.tronalddump.io/](https://docs.tronalddump.io/)

作为现代美国最受认可的政治人物，你总是可以期望人们将你所说的话记录在一个可搜索的数据库中，该数据库具有易于解析的输出格式。

如果你不想搞政治，还有一个简单的[罗恩·斯旺森](https://github.com/jamesseanwright/ron-swanson-quotes)引用 API。当然，大量的[查克·诺里斯的事实](https://api.chucknorris.io/)肯定会派上用场。

## 一切你曾经想卷曲的食物

[https://spoonacular.com/food-api](https://spoonacular.com/food-api)

食谱，配料，菜单项，营养数据。在你的冰箱里安装一个网络摄像头，给你的食物贴上二维码标签，并根据你的库存自动提出烹饪建议。

## 轻松处理数据

[https://devnull-as-a-service.com/code/](https://devnull-as-a-service.com/code/)

云中的/dev/null。通过 POST 请求删除不需要的字节。每月高达 25GB 的免费空间。
