# IPLocation 2 -为您的 IP 进行地理定位

> 原文:[https://dev.to/valeit/iplocation-2-为你的 ip-20p2 地理定位](https://dev.to/valeit/iplocation-2---geolocalization-for-your-ip-20p2)

这次重大更新是迄今为止最好的 IPLocation，是数月开发和设计的结果。希望你会喜欢！

我制作了一个简短的视频，向新老客户展示了 I location 以及 IPLocation 2 的所有新功能。

一些新功能:

*   一个全新的历史视图，查看您的 IP 随时间变化的完整时间表，以及您关心的所有信息(如位置和日期)。

*   具有更多细节的改进地图视图

*   更新的用户界面支持 macOS Mojave 上的黑暗模式

*   能够隐藏不想显示的部分 IP 地址
