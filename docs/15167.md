# 你想达到什么目的？

> 原文:[https://dev.to/josegonz321/rephrase-a-simple-question-4f62](https://dev.to/josegonz321/rephrase-a-simple-question-4f62)

而不是问

> 你想要什么?

要求

> 你想达到什么目的？

它有助于提问方(可能是你自己):

*   增益**清晰度**
*   重新措辞他们的问题
*   (大概)拿出自己的解决方案

从你身上，可以看出:

*   神入
*   在乎
*   支持
*   信任
*   耐心

试试看，看人家反应。

那么，你想达到什么目的呢？:)
