# th3 翻转器(翻转外壳工具)

> 原文：<https://dev.to/elpsyc0p4the/th3reverser-reverse-shell-tool-bg>

Th3Reverser 是一个反壳工具，会让你更容易得到反壳。它与 python 上的一个名为 nclib 的模块一起工作，该模块基本上代替 NetCat 完成这项工作。
它支持的❇有效载荷:
🔹Python (Linux)
🔹Perl (Windows/Linux)
🔹Ruby (Windows/Linux)
🔹Bash (Linux)
🔹PowerShell (Windows)
🔹PHP (Windows 服务器/Linux 服务器)
