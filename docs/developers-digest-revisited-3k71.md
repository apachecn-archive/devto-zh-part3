# 开发者文摘...ive 系统

如果你错过了最初的帖子，你可以在这里找到它，在这里我给了读者一个关于我的饮食的概要。

从那以后发生了很多变化，看到更多关于作为开发者保持健康的帖子，我受到启发，再次写下我的饮食。我仍然是一个“懒惰的咀嚼者”(如果这个词存在的话)，但是我已经在我的饮食中加入了更多的餐，并且我服用了更多的维生素。以下是我平均每天的工作餐明细:

### 早餐

我每天早上服用大约 250 毫升的螺旋藻粉和水。然后，我通常在早上 5:45 左右吃早餐，早餐包括一些麦麸麦片和热牛奶(不含乳糖)。然后，我服用我的维生素 B 复合物，欧米茄维生素，以及我的铁补充片。每样一个。

### 小吃一份

这种点心是我在早餐后 3 小时吃的思慕雪，它是由燕麦、牛奶、酸奶、蜂蜜、浆果混合物、鸡蛋和一些大麻粉制成的。我知道有些人一想到饮料里有生鸡蛋就感到恐惧，但我几乎尝不出它的味道。我也试着每天改变我的水果混合物。

### 小吃二

绿色沙拉，鸡胸肉配一些沙拉酱或蛋黄酱。我通常在喝完奶昔一两个小时后吃这个。

### 午餐

与小吃相比，午餐通常是更大的一份，我通常吃一些红肉或鱼，旁边有一两份沙拉，还有一些肉汁米饭。

### 小吃三

一包薯片。

## 晚餐

晚餐和我午餐吃的差不多。

是的，我整天都在喝水。水是非常重要的。改善我的饮食带来了很大的不同。我感觉不那么累了，而且变得更有效率了。我确实有过吃快餐的日子，节食并不意味着不吃某些食物，而是要吃得适度。
