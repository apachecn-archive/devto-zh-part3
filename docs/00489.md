# 它是如何工作的🏗

PyMarvelQL 是用 5 个工具构建的——Graphene 2.1、Python 3.7、Flask、flask-graphql 和 Marvelous。烧瓶和石墨烯是这个项目的主要工具。flask-graphQl 用于托管本地 graphQl 服务器，Graphene 用于创建允许用户从漫威数据库查询 JSON 数据的 GraphQL 解析器和对象类型。

# 烧瓶和烧瓶-GraphQL 🧪
