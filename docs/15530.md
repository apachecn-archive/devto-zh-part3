# 盲蝽亚科发展

> 原文:[https://dev.to/microbitcoinorg/mirinae-development-1fko](https://dev.to/microbitcoinorg/mirinae-development-1fko)

开发人员您好，MicroBitcoin 开源项目的开发人员希望获得尽可能多的帮助，以准备部署新的 Mirinae ASIC 抵抗算法。

支持采矿分散化的团体，因此需要开发新的散列函数。

迄今为止，Mirinae 的支持已增加到:

*   核心守护进程
*   CPU 挖掘器
*   ElectrumX (python3 模块)
*   Yiimp 池

他们正在寻找能够帮助 Nvidia 和 AMD 矿工(如 ccminer 和 sgminer)实现 Mirinae 的人。

具体的细节和讨论可以在他们的[开发者论坛](https://mbc.wiki/)
