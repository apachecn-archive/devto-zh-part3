## 从小做起，按照自己的速度成长

**11 月 27 日，我们发布了统一软件 AG 云平台，这标志着我们的云战略和旅程中的一个重要里程碑。Software AG Cloud 是一个开放和独立的云平台，是 Software AG 提供的所有云服务的一站式商店。**

**[![](img/6cd54189347f7b0daf4ebb8db56d13a7.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--KBEKB5Qh--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/http://techcommunity.softwareag.com/documents/10157/10221817/Cloud_1.png/8c434de5-7a30-4a90-bb3c-4e1a388af5f6%3Ft%3D1548169183132)**T6】

我们市场领先的云功能是**集成的**，使您能够按照自己的节奏**连接**、**扩展**和**发展**您的**数字业务**。并在您需要时订阅所需的功能。Software AG Cloud 不仅仅是一个门户网站。Software AG Cloud 是我们的云基础，为我们的云产品提供所有必要和常见的云服务:

*   单一身份管理
*   单一租户管理
*   一次订阅管理
*   跨产品使用案例和连接

此外，Software AG Cloud 是一个真正的多云平台，支持部分不同的基础设施提供商，包括:亚马逊网络服务(AWS)、微软 Azure 等。这种独立性允许您选择最适合您需求的基础设施提供商。我们提供对开放标准的全面支持，并且基于我们最先进的集成技术，您可以添加您选择的 SaaS 和内部 IT 系统。
