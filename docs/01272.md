## 介绍

这个列表最初是克隆自 Karan Bhangui 和 George Stocker 的免费编程书籍列表。

该列表由 Victor Felder 转移到 GitHub 进行协作更新和维护。它已经发展成为 Github 最受欢迎的仓库之一，拥有 160，000 多颗星，6000 多条提交，1600 多名贡献者和 39，000 多个分支。

免费电子书基金会现在管理回购，这是一个致力于促进免费电子书的创作、发行、存档和可持续发展的非营利组织。[给免费电子书基金会的捐款](https://ebookfoundation.org/contributions.html)在美国是免税的。
