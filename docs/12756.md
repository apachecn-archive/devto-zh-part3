# Java 的未来——琥珀计划

> 原文:[https://dev.to/auth0/the-future-of-java-琥珀-308e 项目](https://dev.to/auth0/the-future-of-java---project-amber-308e)

最近，Java 将其发布周期改为更快的节奏，影响了语言随时间的演变。有了这种新的节奏，功能现在可以更快地进入市场。除此之外，这种节奏使社区能够致力于(并交付)更小的功能，而不是将所有的精力花费在大的变化上(例如，lambda、泛型和模块)。

在接下来的部分中，您将了解 Amber，这个项目孕育了小型的、面向生产力的 Java 语言特性。
