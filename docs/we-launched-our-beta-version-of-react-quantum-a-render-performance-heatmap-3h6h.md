# 我们发布了 React Quantum 的测试版，这是一个渲染性能热图！

> 原文：<https://dev.to/wjp241/we-launched-our-beta-version-of-react-quantum-a-render-performance-heatmap-3h6h>

reactjs 开发社区的各位好！

我们的团队很高兴地宣布，我们已经推出了测试版的 **React Quantum** 。目前正在接受 Chrome Store 的审查，这是* *一个 React 组件性能测量工具* *，它可以可视化组件树层次结构，用户可以单击并悬停在节点上，以查看它们各自的渲染时间、提交频率以及触发提交的属性和状态。

如果没有 reactjs 成员在 DEV、Reddit 和 Hashnode 上对我们几周前发布的寻找项目灵感的帖子的深思熟虑和积极主动的贡献，这个项目就不可能实现。非常感谢！

如果您对我们的项目感兴趣，或者希望了解我们的项目的最新进展，或者为我们的项目做出贡献，我们欢迎大家关注我们的 **@reactQuantum** 并查看我们的 [Github 资源库](//github.com/ReactQuantum/ReactQuantum/tree/dev)！
