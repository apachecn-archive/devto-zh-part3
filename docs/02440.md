# 如何决定学习哪种编程语言

> 原文：<https://dev.to/flippedcoding/how-to-decide-which-programming-languages-to-learn-4ed9>

如今，一切都有一种编程语言。这不一定是坏事。虽然确实让想学编程语言的人很困惑。几乎任何你能想到的专业都有这么多选择，那么你怎么知道该挑什么呢？

你要投入时间和金钱来学习这个，所以你要确保你对自己的决定有信心。当你考虑学习一门新的编程语言时，这里有几个你可以问自己的问题。

## 你喜欢什么？

你是在寻找挑战还是想快速学会一门语言？有些编程语言比其他语言需要更多的时间来学习，所以花一些时间来研究它们是值得的。有函数式编程、面向对象编程、分析式编程以及其他各种各样的变体。

你想挑一些和你喜欢的相匹配的东西。如果你想相对快速地学习一些东西，你可以考虑一下面向对象的编程语言。如果您希望对控件的执行有更多的控制，您可以考虑使用函数式编程。为了确定你可能会喜欢什么，尝试几种不同的语言。

你可以在网上找到大多数编程语言的文档，你可以通读一下，看看你学起来有多快。这是找出你喜欢什么的最快方法。另外，它会让你接触到不同风格的编程和多种语言。了解一点不同的语言不会有什么坏处。

## 你所在地区的市场怎么样？

大多数人都在学习如何编程，这样他们就可以转行或者开始自由职业。一个确定你应该从哪种语言开始的好地方是查看你所在城市的网站开发人员的职位描述。大多数时候，同一城市的公司倾向于使用相同的技术，所以这将有助于你增加找工作的机会。

你可能会发现你所在城市的一些令人惊讶的趋势，这些趋势与这个国家或世界的其他地方不一样。你有机会找到懂 FORTRAN 的人的帖子。对你喜欢的公司或你想要的工资范围做一些调查，找出他们想要什么。然后开始学习。

## 出现多久了？

新的编程语言和框架一直在涌现。你可以赌一把某个特定的编程语言成为后起之秀。或者你可以谨慎行事，使用一种更稳定的、有历史背景的语言。当你看一门语言或框架的年龄时要小心。

一个东西还存在，并不意味着很多人还在用。看看这些年来 jQuery 发生了什么。当你考虑它已经存在了多久的时候，想想它还有多少还在被使用。你也可以关注趋势，把赌注押在学习一门前沿语言或框架上。

很难说五年后哪种语言会领先，因为事情变化太大了。如果你想稳扎稳打，就去找那些有固定用户群的知名品牌。

## 你为什么学它？

并不是所有的编程语言都是实用的。看一下空白。这是一种不可能用于实际应用的编程语言。所有代码都是用空格、制表符和其他空白字符编写的。虽然是为了好玩而学的有趣的东西。

为了乐趣而学习编程语言会让你接触到一些你在工作中永远不会用到的奇怪的东西。尽早决定你是想把它作为一种爱好，还是打算在某个时候用它来赚钱。这会让你更容易缩小你真正想学的语言的范围。

## 有没有适合它的好社区？

错误会发生，因为没有人写出完美的代码。不过，有可能其他人以前也经历过你的错误。你能找到他们并得到他们的帮助吗？当你在构建应用程序和网站时遇到问题，这一点很重要。

当你可以找到多个活跃的论坛，人们在那里谈论这种语言时，投入时间学习它可能是安全的。社区也推动语言的变化，使他们更好地使用。当您开始与其他开发人员交流时，您可能会发现有必要对整个语言进行修改。

## 是否定期保养？

您通常可以通过所能找到的文档质量和社区的活跃程度来判断一种语言或框架何时得到了良好的维护。即使是编程语言也是可以改进的，所以看到这一点非常重要。

你不会想花时间去学习一门过时的编程语言，而且很难做到这一点。查看版本发布历史。虽然您希望您的语言得到很好的维护，但您不希望每个月都有变化。但是请关注发布的更新，并查看一些已修复的错误。

## 哪里可以学到？

你能想到的几乎任何编程语言都有教程，但是数量和质量各不相同。对于像 JavaScript 或 C 这样的大型语言，很容易找到大量的教程。但是对于更抽象的语言，可能更难找到学习资源。确保你能找到足够的资源来至少学习你所选择的语言的基础。

学习一门新的编程语言可能很有趣，它可以帮助你改变职业生涯，但这些快速检查可以帮助你选择正确的语言。我一直在寻找一些奇怪的编程语言，相信我，我已经找到了一些。但是你学过的最疯狂的编程语言是什么呢？
