# 你的批评文化是什么？

> 原文:[https://dev . to/sublime geek/what-is-your-crisis-culture-4nj5](https://dev.to/sublimegeek/what-is-your-criticism-culture-4nj5)

不幸的是，发展不仅仅是⛅和🌈。就像任何与人类互动的活动一样，它永远不会完美。事实上，人们会就“完成”的定义和“足够好”的原则发表演讲。

我们谈得不够的是那些尴尬的事情。你知道，当事情不是独角兽放屁和闪光漩涡。

因为我是一名老兵，所以我有一些独特的观点。在军队里，作为一个团队你会赢，作为一个团队你会输。如果有什么不对劲，你就大声说出来。你不指名道姓。人们抓住问题并解决它...作为一个团队。这是我们成长的方式。

在敏捷软件开发中，我们使用回顾会议来找出不尽如人意的地方。我们记录。我们复习。也许...只是也许，它不会出现在下一次回顾展上。

那么，你们是如何互相批评的呢？你的流程是什么？一路走来，你学到了什么？
