## 如何解决基于字符串的编码问题？

字符串数据结构的一个好处是，如果你知道数组数据结构，你可以很容易地解决基于字符串的问题，因为**字符串只不过是一个字符数组**。

> 所以你通过解决[基于数组的编码问题](https://javarevisited.blogspot.com/2015/06/top-20-array-interview-questions-and-answers.html)所知道的所有技术也可以用来解决字符串编程问题。
