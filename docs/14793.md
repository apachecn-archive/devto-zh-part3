### [](#6-raise-the-signaltonoise-ratio-in-your-life)6。提高你生活中的信噪比

你的时间很宝贵。你知道它值多少钱。

我曾经见过一个每小时收入超过 100 美元的顾问花了一个小时争论多收了 5 美元。我完全理解这背后的原理，但是当你可以主动多开一个小时的账单时，这 5 美元真的值得吗？就我个人而言，我会把那记为 95 美元的损失。

[![](../Images/acbbf30c9bb87f5da3ebcd3d7cb3287c.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--mLNpJVLA--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://cdn-images-1.medium.com/max/800/1%2AMN2hrSlscPr9zJn7YqAg2g.jpeg)

当我决定通过吃得更健康和锻炼来改变我的生活时，我学会了珍惜时间。像许多人一样，我有一种印象，就是没有足够的时间。后来我知道了真相:

> 你没有时间问题。你有优先权的问题。

当我遇到一个做多份工作，上夜校，还能抽出时间准备饭菜和锻炼的人时，我明白了这一点。我们在谈论健身，他们随机问了我一个关于我正在看的一部电视连续剧的问题。我回应，他们点头，然后用这个打我:

“真有意思。我以为你刚才说你没时间健身。那个节目不是一个小时长吗？”

原来我有很多东西在我的时间碎片。我看了电影。我看了电视。我玩游戏。我在社交媒体上互动。我没有照顾好自己。

为了改变，我不必放弃一切。相反，我不得不改变我的优先事项。我不再每天晚上看完整部电影，在周末的时候把它们作为奖励。我只在训练结束后允许看电视。优先安排时间不仅对健康很重要，对你的职业生涯也很重要。

你允许生活中有多少戏剧性？我曾经沉迷于网上的争论。我不得不插上最后一句话，回应每一个回复。我的一位商业教练指出了我浪费了多少时间。他喜欢分享一句话，“你是对的。我会发财的。”他的意思是，他没有时间去争论正确的一方，而他可以利用这段时间来建立和发展他的业务。

你有没有计算过你在那些与职业发展毫无关系的线程上花了多少时间？你读的书是有趣的，还是有助于你学习新的工作知识的？当你通勤去办公室时，你听收音机还是播客？如果你乘坐公共汽车或火车，你会花时间喜欢 Instagram 帖子或阅读技术文章吗？

如果你决定专注于每小时有多少信号与你让多少*噪音*进来，你可能会惊讶于你赢回了多少时间。小的调整可以走很长的路。

一些具体的与电子邮件相关的技巧帮助我提高了工作效率:

*   当电子邮件进来时，停止阅读。相反，每天挑选几次来赶上进度。
*   考虑创建一个文件夹和一个收件箱规则来转储那些在*到*行没有明确显示你名字的邮件。这些通常是群发/群发邮件，不太重要，所以不要经常查看那个文件夹。
*   关闭电子邮件提醒。如果有人真的实时需要你，让他们给你发短信或其他紧急信息。

### [](#7-network-for-real)7。真实的网络

我大部分时间都在网上度过。我没有夸张。20 世纪 80 年代，我使用[公告板系统(BBS)](https://en.wikipedia.org/wiki/Bulletin_board_system) 与世界各地的人们合作。20 世纪 90 年代是[互联网中继聊天(IRC)](https://en.wikipedia.org/wiki/Internet_Relay_Chat) (在那里我也碰巧遇见了我的妻子)，接下来是 [Friendster](http://web.archive.org/web/20031015211349/http://friendster.com/index.jsp) 、 [MySpace](http://web.archive.org/web/20041001082041/http://www.myspace.com:80/index.cfm?fuseaction=splash) 、脸书和 Twitter。我有网上社区。

它不能代替面对面的交流。

我了解内向者的感受。我知道是因为很多年前，我确信我生来就不善于社交，永远不会“融入”我讨厌拥挤的人群，当我坐在候诊室时，我会把鼻子伸进一本书或其他设备里，以避免互动。当被介绍给别人时，我最害怕的是“接下来我该说什么？”

我转型的第一步与技术或工作无关，而是与我的健康有关。我很肥胖，在各种饮食和计划中挣扎。最终，我找到了一个我感觉不错的系统，但仍然严重怀疑它是否能帮助我。该节目以“之前和之后”的照片为特色，这些照片包含了鼓舞人心的成功故事，但我仍然心存疑虑。一次工作活动把我带到了迈阿密地区，我知道一个名叫杰夫的人改变了他的生活方式，在那里经营着一家健身房。我走出自己的舒适区，安排了一次约会。遇见他是我需要的催化剂，让我相信什么是可能的，并改变。书籍、网站和网上图片是不够的:我必须亲自联系。

我达到了我的目标。

这些变化激励我开始自己的健身业务，当我离开科技行业专注于全职个人培训时，我被迫再次转型。突然间，我不得不销售自己的服务，提供客户支持和指导客户。这帮助我进一步突破了我的舒适区，在与人交往时获得了更多的自信。在我卖掉我的健身业务并回到技术领域后，我有目的地专注于培养“面对面”的关系。

这并不像看起来那么难。它可以简单到参加本地用户组会议、出差参加会议或参加与技术相关的活动。拥有这样的经历可以带来巨大的回报，产生协同效应，因为你的投资回报大于你投入的精力。

> “我是一个内向的人”可能是真的(这绝对没有错)，但是“我不喜欢和人在一起”是你可以改变的想法之一。

例如，有人在网上联系我成为一本书的技术编辑。当出版商来镇上参加一个会议并邀请我边喝咖啡边见面时，我已经在这个项目上工作了几个月。我们进行了面对面的交流，这导致了我写第一本书的邀请。这导致了其他几本书和一段持久的关系。

在我的第一次会议上，我在演讲室遇到了几个人，我只在路上见过他们。虽然我们一年才见一两次面，但我们已经建立了几十年的友谊。其中一些发言者后来组织了自己的会议，并寻求支持。这带来了更多的机会。

任何时候，只要你能花点额外的努力亲自联系，都会给你的职业生涯增加价值，无论是机会、推荐人，还是在你需要的时候支持你的朋友。这并不意味着在线互动不算数，但现代技术太容易避免老式的人与人之间的互动。记住之前的建议:这仍然是关于做真正的你，即使你是内向的，也不是强迫自己成为外向的人，而是建立信任和承担一些风险。

如果感觉像政治，*不要做！*应该是关于关系的。

我使用[Meetup.com](https://www.meetup.com/members/10127361/)与当地开发商联系，然后跟进[LinkedIn.com](https://linkedin.com/in/jeremylikness)以维持长期的业务关系。

### 8。暂停！暂停！总结一下。

据我估计，在我的职业生涯中，我已经进行了 100 多次面试，看了数千份简历。在过去的 25 年里，我也修改了我的答案，并多次坐在受访者的椅子上。不幸的是，简历通常是这个过程中的第一个接触点和过滤器，一份糟糕的简历可能会阻止一个完全合格的候选人进入这个领域。有很多网站承诺改善你的简历，所以我不想把它作为一个全面的“如何做”的指南，而是想分享一些亮点，我认为这些亮点可以成就或毁掉一份简历。

首先，你的任务是什么？我很难理解这一点:

> 极其合格和经验丰富的专业人士，拥有惊人的技能，已获得数十个行业奖项。

好的，那太好了…但是你为什么要向我提交你的简历呢？自私地说，我想知道你是否只是想在一长串项目中再选一个，想接触新技术或挑出我们公司，并真的想加入。我劝你考虑一下*你真正想要的是什么*,并在简历中写明。不是“这就是我”，而是“这就是我在寻找的。”

> 寻求一个令人兴奋的开发人员关系职位，该职位将利用我丰富的经验和技能，通过在开发人员社区建立深厚的关系来推动价值。

为便于说明，这里有另一个例子:

> 拥有三项 Azure 认证的优等生。

怎么样……

> 热情的开发人员寻求与愿意利用我的热情和能力来学习新技术以推动成功采用云的公司合作。

不要担心我用了太多的词，那是我的风格，但是试着从“这是我是谁”到“这是我正在寻找并愿意做的。”会让你的简历脱颖而出！

永远把你的优势放在简历的最前面。如果你是这个行业的新手，而教育是你的强项，那就去做吧。就我而言，没有大学学位，我的经历和荣誉会放在最上面，关于我的教育背景的说明会放在最下面。想象一下有人一口气翻看了 100 份简历，你想从中找出什么。

将你的技能融入到你的经历中。没有什么比读到有人有丰富的 Web API 经验，然后发现在他们的工作经历中没有提到它更令人沮丧的了。当 XML 和 XSLT 成为重要技能时(现在我又开始和自己约会了)，我不只是列出“3 年的 XML 经验”…我还展示了我是如何使用它的:

*   构建了一个移动设备管理系统，该系统利用 XML 定义和 XSLT 转换来驱动能够快速适应自定义设置的动态 UI

试着把你的技能和你的经历结合起来。说到经历，当你在列举历史的时候，请列举你做了什么，而不是你应该做什么。站在招聘经理的角度想想。你会看到这样一份简历:

> 负责设定编码标准、指导初级开发人员和探索新技术的高级开发人员。

另一个是这样的:

> 作为一名高级开发人员，成功地建立了编码标准以加速交付和提高质量。指导了几名初级开发人员，其中一名后来被提升为高级开发人员。带领团队成功从 WCF 过渡到 REST。

最后，不要觉得这一定只是你的主要工作职责和成就。我包括出版物、视频、演示文稿和我参与的项目。面试团队对我的经验、热情和能力有着全面的了解。

### [](#9-just-say-no)9。就说“不”

学会说“不”可能是你可以添加到你的武器库中的最有力的工具。

我第一次发现这项技能是在我的第一份技术工作中。我最初是按小时计酬的。夜班“奖金”和加班加起来对我来说是一笔不小的收入。就在那时，老板打电话给我，告诉我，由于我的敬业精神，我被“提升”到了一个有薪职位。有人向我保证，我的新薪水非常高，需要公司总裁的特别批准。我对谈判的艺术并不熟练，我微笑着说:“是的！”留下失望的感觉。他们给我的薪水比我按小时计算的工资少了 30%。

经过思考，我把“是”变成了“不是”，并开始在其他地方寻找工作。我很感激这家公司给了我第一次编程的机会，但它不再适合我。我最终接受了亚特兰大的一份工作，并在此后不久搬了家。在亚特兰大，很快就发现这个新角色也不太合适。我再次说“不”，然后开始了另一个对我职业生涯整体成功至关重要的角色。

说“不”的一部分是理解你的优先事项，并按照它们生活。我的家庭几乎是我最优先考虑的事情(在工作之前)，所以我尽量不让工作干扰家庭。有时候，这意味着一个公司，当我被要求去旅行或承担项目，将削减宝贵的时间，“不”。这并不是说我不是一个团队成员；相反，我总是愿意在紧急情况下介入并提供帮助，但我对一种在紧急情况下蓬勃发展的文化没有耐心。

我很早就学会了珍惜时间，关注投资回报的最大化。这意味着我寻找奖励生产力而不是活动的机会。比起那些总是开夜车、错过睡眠来完成工作的人，我更钦佩那些学会了超高效、在正常工作周内完成工作的人。这是我个人的偏好，我已经学会对那些更关心工作时间而不是产量的公司坚决说“不”。

问问我在微软的任何一位同事。我不怕在适当的时候说“不”。

### [](#10-its-not-only-about-the-benjamins)10。这不仅仅是本杰明的事。

你听说过爱让世界转动吗？那可能是真的，但是钱支付旅行。没有什么比一个有很多钱的人骄傲地宣称“钱不重要”更让人讨厌的了。这是现代社会的核心部分，但人们经常忘记…

> 钱不是**的东西**。它是让你得到**的东西**。

了解事情是什么很重要。

在我开始我的技术生涯之前，我曾想过成为一名职业台球运动员。我在找工作，没有时间吃饭，所以我在台球厅找了份工作。我每天工作 8 小时挣 20 美元。我花了 10 美元在车上加油，省下了 5 美元房租和 5 美元食品费(我通常会走到 7-11 便利店，买一份压制的古巴三明治和薯条)。我这样生活了一年。你猜怎么着？我的工作让我可以免费使用台球厅，所以在我下午 4:30 下班后，我经常呆到午夜做我喜欢的事情:玩一架又一架的一袋球和九球。事实证明，我不需要很有钱就能玩无限量台球。

我一直想环游世界。在我早期的一个角色中，我开发了一个软件，将大型机与仓库传送带集成在一起，以传送和分类库存。我们的许多客户都是国际客户，因此我能够访问荷兰、德国、比利时、卢森堡和法国。那时，我不得不省吃俭用一段时间来支付我自己的旅行费用，但是这份工作支付了我的旅行费用。这个*东西*是旅行，所以我不需要“富有”才能得到这个*东西*。

理解这一点很重要，因为有时你必须后退才能前进。这是我的薪水和职业生涯的曲线图。

注意第一次大幅度下降。就在那时，我冒了很大的风险，开始了自己的事业。那段职业生涯让我接触到了辅导和指导的力量，并帮助我在公开演讲时变得自信。我学到了如何建立和扩大一家公司的重要经验。你可以看到，第一年我不知道自己在做什么，但到了第三年，我赚的钱已经超过了我之前在技术岗位上的收入。

第二次衰退发生在一位成功的企业家找到我，问我是否能帮他建立一家新的软件公司。随着时间的推移，我的收入稳步提高，但我的生活质量却下降了。我在创业时间工作了五年，结果我的健康受到了影响。有一个我没有圈出的小倾角代表着生活方式的改变。该公司后来以 15 亿美元的价格出售，但我开始在家工作，成为一名顾问，与家人在一起的时间比过去几年都多。那就是*的事情*。

上一次薪水下降是我为了有机会进入开发人员关系部的当前职位而做出的牺牲。我个人的使命是让开发人员成为他们最好的，这是这个角色的本质，也反映在我们公司的整体使命中。做我喜欢的全职工作是值得的，作为奖励，我在一年内就得到了提升。

最初几年，我专注于钱，钱来得很不情愿。在过去的几年里，我专注于我想做的事情，以及我想如何平衡我的生活，钱似乎到位了。你的里程可能会有所不同，但请记住，这不仅仅是关于本杰明的*。弄清楚事情，然后专注于它。*
