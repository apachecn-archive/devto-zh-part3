# 正念冥想可以帮助你进入心流

> 原文:[https://dev . to/mrpowerscripts/正念-冥想-能帮助你进入心流-6hp](https://dev.to/mrpowerscripts/mindfulness-meditation-can-help-you-get-in-the-flow-6hp)

难以集中注意力？你觉得没有以前那么有效率了？你被小事困扰了吗？如果你对这些问题中的任何一个回答是肯定的，我真希望我有一个药丸卖给你。真的。我现在可能已经是百万富翁了。我认识的人几乎都有这些问题。不过，我不是来提供任何医学建议的。如果你有严重的疾病，你真的应该去看医生。如果没有，这里有一个巨大的帮助你的意见。

我注意到许多朋友和同事总是遭受类似的战斗。专注，专注，动力，甚至内心的平静。我们生活在这样一个时代，人们只需按几个按钮，你就能看到发生在地球另一端的灾难的图像。要担心的事情太多了！啊啊啊啊啊啊！

这就是在这个信息时代的感觉。然而，你仍然要醒来，做好你的工作，处理所有直接影响你生活的疯狂。沉重的时代哟。我仍然记得当我没有随身携带一个设备，可以让我直接把世界上所有的问题注入到我的血管里，只要我有数据信号。不一定是坏事。当我们走在街上时，我们总是打断生活中的一些时刻，以便被朋友的通知或广告分散注意力。

当我们习惯了这些日常消费模式，或者我们生活的社会将这些行为正常化时，就很难脱离了。从当前对我们没有帮助的信息流中分离出来，可以让我们集中注意力，找到自己的心流。我们只是花哨的多肉电脑，一次只能处理这么多。如果你的大脑一直在 100%运转，你最终会筋疲力尽。你必须在生活中找到一个好的方向。

我不是说“流动”是一个空洞的术语。[心流实际上是一个被研究过的术语](https://en.wikipedia.org/wiki/Flow_(psychology)),它描述了一种精神状态，在这种状态下，你在做某事时处于最佳状态。让我们花点时间来欣赏一些维基百科的文章:

> 在任何给定的时刻，每个人都可以获得大量的信息。心理学家发现，一个人的大脑一次只能处理一定数量的信息。根据 csikszentmihályi 2004 年的 TED 演讲，这个数字大约是“每秒 110 比特的信息”。[5]这看起来像是很多信息，但是简单的日常任务需要相当多的信息。仅仅解码语音每秒就需要大约 60 比特的信息。[5]这就是为什么在谈话时，一个人不能把注意力集中在其他事情上。
> 
> 在很大程度上(除了像饥饿和疼痛这样与生俱来的基本身体感觉)，人们能够决定他们想要关注什么。然而，当一个人处于心流状态时，他们会完全专注于手头的一项任务，而不会有意识地决定这样做，从而失去对所有其他事情的意识:时间、人、干扰，甚至基本的身体需求。这是因为处于心流状态的人的所有注意力都集中在手头的任务上；没有更多的注意力分配。[6]
> 
> Csikszentmihályi 将心流状态描述为“最佳体验”,即一个人从体验中获得高度满足。[7]获得这种体验被认为是个人的，取决于个人的“能力”。[7]一个人克服挑战以实现其最终目标的能力和愿望不仅会带来最佳体验，还会带来总体的生活满足感。[7]

你只是在处理你需要处理的事情，以便做你需要完成的事情。这就是伟大的来源。那是你想去的地方。你如何通过这种让你做大事的意象聚焦肌肉达到新的高度？和其他肌肉一样。你必须锻炼它。

现在，进入篮球运动员的心流和进入程序员的心流当然是不同的。但是任何类型的心流的共同点是有能力控制你的思想，从而让你的思想沉浸在当下。如果你一边烤蛋糕，一边想着上周弄砸的蛋糕，你很可能也会弄砸这周的蛋糕。当前的滴答时刻永远是心流状态中最重要的时刻。不要把它浪费在你现在无法控制的事情上。

我确实发现自己在某些活动中经常处于这种心流状态。当我玩电子游戏的时候。当我在球场上打篮球时。当我解决问题和帮助别人的时候。我能够把其他的事情都放在一边，这样我才能真正欣赏眼前发生的事情。我觉得对我来说，实现这一目标的主要驱动力已经练习正念冥想好几年了。

关于正念冥想，你会在谷歌上看到无穷无尽的信息，所以我在这里不打算深究。我宁愿只是分享我的想法。每当我提到任何一种冥想时，我都会听到一些这样的短语:“我做不到。”，或者“我试过了，没用”。

我认为冥想是一种类似锻炼的练习。除非你有什么令人困惑的健康问题，否则少吃多运动会让你身材更好。事情就是这样的。你的思想真的需要同样的治疗。不要让它负荷过重，也要训练它保持良好的状态。我发现“我做不到”通常意味着“我不想做”。这很好。你做你的。而“我试过了，没用”往往更像是“我做了一个星期，没看到什么效果所以放弃了”。就像你走进体育馆，一周后出来就像终结者一样。集中你的思想就像收缩肌肉。你必须通过锻炼来获得效果，这需要时间来发现巨大的变化。如果你找不到锻炼的方法，你将永远找不到你的心流。

正念冥想并不是锻炼这块肌肉的唯一方式。但这是我发现的唯一一个免费的，可以在任何地方坐下或躺下的。当我练习的时间足够长的时候，我开始注意到我一生中的许多好处。我比以往任何时候都更有耐心。我不会让外部因素经常控制我的情绪。我没有那么多“糟糕的日子”。我有比别人更有挑战性的日子。作为一个曾经经历过并仍在经历相当动荡的生活的人，这对我来说是一个相当大的声明。不过，我真的相信。尽管发展这块肌肉很有挑战性，但我觉得正念冥想教会了我更好地了解自己。通过这种理解，我已经能够做出改变，帮助我在日常生活中更有效地运作。这种更强的自我感知和专注的力量也许也能帮助你找到你的心流。
