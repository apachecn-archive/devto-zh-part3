# [](#basics)基础知识

让我们快速后退一步，只考虑一些基本问题。

当我第一次开始使用`querySelector`时，它是在 HTML 中挑选出单独的标签，然后从那里开始。

一个基本的例子是选择页面上的第一个或所有段落标签。

`let paragraphs = document.querySelectorAll(“p”)`。
