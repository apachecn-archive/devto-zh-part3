# 经理 Ugh 字段

> 原文:[https://dev.to/ejames_c/the-manager-ugh-field-587o](https://dev.to/ejames_c/the-manager-ugh-field-587o)

[![Pouring coffee into an ugh cup](../Images/5b2b0f542d913ccfdcaa31ecea41693f.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--cQ00Pyf1--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/dkvmozssv05iqb7i5r0m.jpg)

你有没有做过一些你必须做的事情…却没有去做？我知道你在想什么:“哦，上帝，他要谈论拖延症了。”但是这不是关于拖延症——不完全是。相反，我想谈谈如何避免情感上痛苦的事情。

这篇文章将会讲述作为一名经理令人难以置信的不舒服的事情。

大概是那次你推迟了一次糟糕的指挥。

是关于你一直拖延的那个艰难的工作对话。

这是你大脑中的痒痒，它提醒你，你将不得不向你的老板提出一个不舒服的问题…而且你*知道*你们两个都不会喜欢它。

像这样情感上痛苦的活动引发了我喜欢称之为“经理恶场”的东西。恶场是如此强大，以至于你的大脑甚至不敢朝坏想法的方向思考。当你发现自己在自省时，你的注意力就跳过了这个问题。当你思考这个问题的时候——当吃饭时或洗澡时，你的想法无情地转向这些痛苦的情况之一——你会立即发现自己想出借口来避免做那件痛苦的事情。

“哦，我不必解雇约翰……他上周按时完成了那个小任务。也许我会再给他一个月的时间来证明自己。”

“我没必要在周五向玛丽提出交货缓慢的问题，这没什么大不了的。”

“我不必和老板谈论我们的工作文化。我可以处理。真的！”

经理 ugh 字段使管理变得如此困难。在每一种情况下，你大脑中的痒痒感是一个信号，表明你不能推迟这些事情。在很多情况下，你*已经*知道你需要做什么。你只是试图避免处理这件事时的情感创伤，所以你试图尽可能地拖延。你在脑子里吐出一个呃场。

但是...

但是让我们诚实面对自己，好吗？

你知道你迟早要解雇约翰——在过去的三个月里，你已经给了他足够的机会来证明自己。

你知道你必须在星期五向玛丽提出交货缓慢的问题。你每周都在伤害你的团队，因为你没有提出这个问题。

最后，你知道你有时间和你的老板谈论每周加班的文化。你知道这是你团队成员精疲力尽的一个原因。如果你不这样做，你将无法坚持太久。

那么当你面对 ugh 场的时候应该怎么做呢？你如何更好地处理这个问题？

## [](#the-dirty-little-secret-of-ugh-fields)恶场的肮脏小秘密

我发现了一个小秘密:ugh 领域是一个伪装的机会。

人类的大脑是一个强大的工具:当你的大脑抛出一个恶场时，它正在试图保护你远离生活中的一些痛苦领域。但是伴随着这种力量而来的是一致性——这种一致性也意味着你可以利用它为自己谋利。如果你训练自己去发现‘ugh’字段，你可以用它来触发*富有成效的自省会议*——也就是说，会帮助你正面解决问题的会议。

你现在有一个 ugh 字段吗？你有没有发现自己忽略了某个话题？如果是这样，那真是天大的好消息！你刚刚给自己找到了一个练习的机会。

富有成效的自省会议的目标是让你将全部注意力集中在隐藏在一个 ugh 领域后面的问题上。这将允许你方得阻止你处理问题的痛苦。

进行这项活动需要持续的注意力，这比听起来要困难得多。在典型的一天中，你的注意力分散在各种事情上，比如工作任务、家务和脸书。你会发现你的注意力，如果不加控制，就像一盏大灯发出的光一样扩散开来。在开始富有成效的内省之前，你需要先集中注意力。这很像通过透镜将光线聚集在一个点上，然后将光束导向你想要聚焦的地方。

做到这一点的诀窍是*冥想。*这里有两个我发现对我有用的技巧，一个比另一个简单。

第一种更简单的技术被称为“[箱式呼吸](https://www.medicalnewstoday.com/articles/321805.php)”——这是一种在高压力情况下为地面专家开发的技术，如警察或海军海豹突击队队员。要执行箱式呼吸:

1.  用鼻子吸气，同时慢慢数到四。感觉空气进入你的肺部。
2.  屏住呼吸，慢慢数到四。不要紧闭你的嘴或鼻子。简单地避免吸入或呼出四秒钟。
3.  开始慢慢呼气四秒钟。
4.  呼气后，在没有呼吸的情况下再等四秒钟，然后再吸气(也就是回到第一步之前)。
5.  重复上述步骤至少三次。你可以选择执行上述步骤四次(大约一分钟)或根据需要尽可能长的时间，直到你感到平静和集中。

第二，稍微难一点的技巧是正念冥想。这是这样的:

1.  找一个安静舒适的地方坐下。如果你无法进入一个安静的空间，使用耳塞或在耳机上播放一些舒缓的声音。设置一个五分钟的计时器。
2.  当你闭上眼睛时，把注意力集中在你的呼吸上。注意空气是如何进入你的鼻子的，以及当它从你的鼻腔进入你的肺部，然后再出来时的感觉。
3.  你会开始注意到你的大脑在思考各种各样的事情。晚餐你打算吃什么？你忘记倒垃圾了吗？你没有什么需要为明天的会议准备的吗？注意这些想法，接受它们，轻轻地将你的注意力带回到你呼吸时的感觉上。
4.  目标是在五分钟结束时，除了呼吸的感觉之外，你的头脑没有任何想法。那是你知道你的注意力足够集中的时候。
5.  如果你失败了，在计时器上再设定五分钟，然后再来一次。

一旦你完成了这个活动，把你的注意力集中到 ugh 领域。这是更痛苦的第二步。问问你自己:为什么我会感到这种精神上的痛苦？我在这里想避免什么？这种痛苦的来源到底是什么？

每次当你感觉你的注意力在流失，或者如果你注意到你的大脑很快给出了让你感觉更好的理由，就把这些放在一边，把注意力集中在让你感到痛苦的事情上。任何由 ugh 字段保护的问题都可能由多个棘手的小问题组成。像侦探在犯罪现场走来走去一样检查每一个。

这个练习应该感觉像在伤口上撒盐——这很好！盐抹得越多，以后的疼痛感就越小。(我知道这听起来像是自虐，可能是错误的医疗建议……但我袖手旁观。有用！)

当这个领域永远消失的时候，你会知道你已经成功了。

## [](#a-quick-story)一个小故事

两个多月来，我一直在推迟拍摄一部我称之为“杰克”的电影。然而，在某个时刻，我后脑勺的痒痒变得无法忽视。

我知道我的团队中没有人对杰克满意，这要归功于我最近与我的服务器团队经理进行的一对一谈话。我也知道，他们中的一些人认为我*不*知道杰克表演得如此糟糕。他们感到愤怒，因为他们需要为他打掩护。每当他们不得不花额外的时间来修复他的糟糕代码时，他们都会感到不高兴。

一天下午，我坐在办公室里，戴上耳塞。杰克不在——他午休时间很长——我关上了公司会议室的门，设置了五分钟的计时器，开始冥想。

然后我开始审视我的 ugh 领域。

我为什么要回避这个？这很简单:我避免这样做是因为解雇某人是痛苦的——对我和他们都是如此。我的大脑立即开始提供替代方案:“哦，不，你可以再给他一次机会！”“但是杰克是如此的可爱！你不能这样对他！”

我没有理会这些不同的声音，而是再次关注这个问题。恶场还在。我的大脑偏离了我需要的结论:我不可能让杰克留下来。一个月前，我已经和他坐下来谈过了，并给他制定了一个绩效改进计划。负责我的服务器团队的经理仍然不高兴。他认为无论杰克做了什么改进都不够好。

我坐下来，再次审视我的选择。我已经给了杰克第二次机会。我知道他是一个好人——这就是为什么我过得如此艰难。所以我抛出了这个问题:如果我不解雇杰克，会发生什么？

答案是:我会锁定一个本可以给别人的位置。我们公司的资源有限，这意味着工程团队的席位有限；我浪费了一个座位给一个表现不佳的人。

最重要的是，我意识到我在向我工程团队的其他人传递信息。我告诉他们:“表现并不重要。我不在乎你因为杰克的无能而受苦。替他打掩护就是了。”

我意识到我不能让这种情况持续下去。我给别人带来的痛苦太大了，让杰克无法留下。此外，我减少了整个团队的产出——这是任何经理都不应该有意识地做的事情。

当我再次睁开眼睛的时候，我感觉到我的视野有所后退。它还在，但我大脑中能迅速提出替代建议的那部分被抑制了。当我决定采取行动解雇杰克时，它只是微弱地抗议。

那天晚些时候，我离开了房间，做了必要的事情。

## [](#takeaways)外卖

经理 ugh 字段是阻止经理做他们工作中必须做的事情的字段。通常你会犯的最严重的管理错误是疏忽——你没有进行痛苦的谈话，或者你推迟了困难的行动，直到为时已晚。

ugh 领域的存在通常有一个很好的理由:这些痛苦的对话和行动可能会非常糟糕。解雇表现不好的员工会让你面临被报复的风险。告诉你的老板他或她不想听的事情可能会让你陷入困境。

但是解决这些困境的方法不是避免去想它们。答案是用冷静、理性的头脑来考虑它们。当你让 ugh 领域恶化时，你并没有给自己带来任何好处，而且在这个过程中，你很可能伤害了你的整个团队。

这里的好消息是，ugh 场是一个*一致的*机制，也就是说你可以把它变成一种实力。像使用早期预警系统一样使用它:训练自己发现何时存在 ugh 场，并以此为契机反思你正在回避的问题。这很难，但绝对值得:一旦你能够做到，你就会成为一名更好的经理。
