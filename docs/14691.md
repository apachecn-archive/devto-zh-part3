# 我的“灯泡”时刻！

> 原文:[https://dev.to/jonmajorc/my-lightbulb-moment-32cj](https://dev.to/jonmajorc/my-lightbulb-moment-32cj)

> 无论你是如何开始编码的，在学习的时候你都有形成的时刻。回想一下你的某个“灵光一现”的时刻，那时事情开始“咔嚓”一声了~茶歇挑战，第 29 天

差不多六年前...我职业生涯的开始，在大学期间，我被聘为兼职移动软件开发顾问。我被录用时，对如何做这份工作知之甚少，甚至一无所知——但是，我下定了决心...我第一年没有真正理解如何阅读 API 文档；然而，在这第一年里，我能够为一个客户构建我的第一个生产移动应用程序！尽管这款应用很成功，但我仍然不确定自己是如何开发它的。我当时工作的公司雇佣了一名新的开发人员，他也渴望成为一名软件开发人员，我被指派教他“如何构建移动应用程序”。——小崽子，我就是这么干的！我教他如何阅读 API 文档，你知道吗...？就在那时，它突然出现了！我不会忘记那个“灯泡！”瞬间，因为我看着他的样子...****我终究知道我在说什么！

教别人，你就会学到东西。

你的一个灯泡时刻是什么？
