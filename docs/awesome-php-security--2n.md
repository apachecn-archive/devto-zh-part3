# 工具

## Web 框架强化

*   [Snuffleupagus](https://snuffleupagus.rtfd.io)——PHP 7/8 的安全模块， [suhosin](http://suhosin.org/stories/index.html) 的继承者。
*   [Secure-Headers](https://github.com/BePsvPT/secure-headers) -向 HTTP 响应添加安全相关的报头。

## 静态代码分析

*   Enlightn - Enlightn 是一款静态和动态分析工具，用于提高 Laravel 应用的安全性。
*   [Exakat](https://github.com/exakat/exakat) - *Exakat* 是一个 PHP 静态代码分析，带有严重的[安全审查](https://exakat.readthedocs.io/en/latest/Rulesets.html#security)。
*   phpcs-security-audit 是一组 PHP_CodeSniffer 规则，用于查找 PHP 代码中与安全性相关的漏洞和弱点
    *   `docker pull guardrails/phpcs-security-audit`
*   用于安全目的的静态分析器。
*   [Parse](https://github.com/psecio/parse)-*Parse*扫描器是一个静态扫描工具，用来检查你的 PHP 代码中潜在的安全相关问题。
