*   [注册一个免费的 Azure 账户](https://azure.microsoft.com/en-us/free/?wt.mc_id=academic-0000-chnoring)要像使用私有注册表一样使用云中的容器，你需要一个免费的 Azure 账户
*   Docker 第一部分-基础知识这一部分介绍了 Docker 是什么，以及为什么我认为你应该使用它。它提出了图像和容器等概念，并带您构建和运行您的第一个容器
*   [Docker part II - volumes](https://dev.to/softchris/docker-from-the-beginning---part-ii-5g8n) 这是关于卷，以及我们如何使用卷来持久化数据，以及我们如何将我们的开发环境转变为卷，并使我们的开发体验显著改善
*   [Docker 第三部分-数据库、链接和网络](https://dev.to/softchris/docker-from-the-beginningpart-iii-2h51)这是关于如何处理数据库，将它们放入容器，以及如何让容器通过网络使用传统链接和新标准与其他容器对话
*   [Docker 第四部分-介绍 Docker Compose](https://dev.to/softchris/docker-from-the-beginning-partiv-mi6) 这是我们如何使用 Docker Compose 管理多个服务(这是 Docker Compose 的 1/2 部分)
*   [Docker 第五部分-深入了解 Docker Compose](https://dev.to/softchris/dockerfrom-the-beginning-part-v-n2c) 这一部分是 Docker Compose 的第二部分，也是最后一部分，我们将介绍卷、环境变量以及如何使用数据库和网络
