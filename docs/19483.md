# 改变是好事。你先走。

> 原文:[https://dev.to/kymiddleton/change-is-good-you-go-first-186 p](https://dev.to/kymiddleton/change-is-good-you-go-first--186p)

作为一名新兵训练营的学生，再次坐在教室里有点超现实。据我目前所知，如果有人正在考虑从事 web 开发，但不喜欢解决问题的复杂性，或者整理大量代码发现故障与打字错误有关，那么 web 开发可能不是最好的职业选择。在某些方面，web 开发就像解开一大堆乱糟糟的绳子。对于一些人来说，他们会看一眼巨大的混乱，要么恐慌，要么走开。对其他人来说，拼图的吸引力是令人兴奋的，追逐单股绳子找到第一端的行为是令人满意的。

在传统的学习环境中，一个特定的学科在一个确定的时期内建立了精通。新兵训练营环境在快节奏的学习环境中建立了广泛的知识基础。简单地说“新兵训练营”这个词就会让人想起军事题材的电影，里面有教官和新兵，训练的强度显而易见。web 开发训练营的环境类似于快节奏的学习环境，需要大量的关注、时间和精力。学习任何新的东西都会让人产生挫败感、自我怀疑或不适，这可能会让一些人措手不及。

学习就是改变。你可能对传统的“变化曲线”很熟悉

[![change curve](../Images/ef1aceac0ccac221b68e117c39d512c4.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--XLHogvm8--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/76t9nlhr16aijm7ghk69.jpeg) 
根据威廉·布里奇斯的观点，管理转型，改变是情境的，然而转型是心理的。转变有一个结束阶段，即放弃旧的方式和旧的身份。中性阶段是探索和学习。“旧的”已经过去，然而“新的”还没有到来。最后是新的开始，一个新身份、新能量和发现新目标的阶段。每个人都有不同的体验，反应不同，速度也不同。让我们面对现实吧，任何类型的改变都是困难的，但如果有合适的工具，会变得更容易。

我的非技术背景的一部分是通过 Prosci 变更管理作为变更管理的认证从业者度过的。他们的变革模型 ADKAR 简直棒极了，可以应用于任何专业或个人情况，甚至是 web 开发！
