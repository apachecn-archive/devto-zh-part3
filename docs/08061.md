# 什么是用户测试，为什么它很重要？

> 原文:[https://dev . to/sarahatzing/what-is-user-testing-and-why-it-important-4on 2](https://dev.to/sarahatzing/what-is-user-testing-and-why-is-it-important-4on2)

用户测试是一个误导性的术语。这意味着被测试的是用户，而实际上被测试的是产品或服务。在任何关于用户反馈收集的讨论中，记住这一点是很重要的，因为反馈是你的项目在用户眼中的感知价值和有用性的指标——不仅仅是理性的，也是感性的。

诸如此类的指标可以为团队提供切实可行的方法来与现有的和潜在的用户建立共鸣。反过来，团队可以使用这些见解迭代地改进他们的项目。

目标应该是解释收集到的反馈，以提供人们希望使用的更好的产品或服务，无论是因为它令人愉快、有帮助、有用、需要等等。用户测试是在受控环境下收集用户反馈的一个好方法。如果你已经明确了你的目标受众，这一点尤其正确。如果你还没有，那么用户测试可以作为帮助你开始这样做的起点。

一般来说，用户测试包括设计场景，在期望的环境中从相关的受众群体中收集反馈。更简单地说，这是倾听用户意见的一种方式。尽管用户测试可以在许多不同的设计环境中找到，但我将重点放在 web 用户测试上，因为我在一个数字产品团队工作。

## [](#general-to-specific-user-testing)一般针对特定用户的测试

用户测试可以作为探索性的起点，用于发现需要优先考虑的摩擦和痛点，或者可以根据特定的用例或问题进行设计。这种通用的/特定的用户测试方法可以在一个范围内使用(例如，你从通用的开始，随着时间的推移进展到特定的用户测试)，或者根据需要使用(例如，你从特定的开始，以回答预先存在的问题或者解决明确的问题)。

影响用户测试动机的一个因素是项目的开发阶段。例如，较新的项目可能会受益于更通用的用户测试策略，而更成熟的项目可能会受益于更具体的侧重于特性发布的用户测试策略。从一般到特定的用户测试过程对于以前没有尝试过用户测试的团队来说也是有意义的。

在我的例子中，当我第一次开始探索用户测试时，我为我的产品和团队选择了前一种更通用的方法，原因有二:1)这是我和我的团队第一次体验用户测试，2)我希望用户反馈能够帮助指导我的团队的开发计划以及我的设计和营销计划。第二个原因对我来说尤其重要，因为我们的产品还是新产品。

## [](#user-feedback-formats)用户反馈格式

记录的用户反馈通常以视频、音频和/或文本的形式出现。为了确定哪种反馈形式适合您的团队，考虑以下因素会有所帮助:

1.  你用来管理考试的工具
2.  参加测试的用户的位置(远程与现场)
3.  你的测试目标
4.  审查和评估测试反馈的可用带宽
5.  谁需要访问和审查测试反馈

不同的反馈格式为测试人员提供了不同的价值，值得考虑哪种格式最符合您的需求。如果您的用户测试是远程管理的，这可能特别重要。以下几点说明了考虑反馈格式的重要性:

*   **书面反馈**
    *   支持:鼓励用户在回答之前进行更多的思考，这意味着反馈通常更清晰、更有凝聚力
    *   *Pro:* 可以更好的捕捉小细节
    *   测试人员查看反馈的最快方式
    *   *缺点:*不太能代表用户当时的情绪反应
*   **视频反馈**
    *   屏幕录像向你展示了用户在他们的机器上，在他们的浏览器上，到底经历了什么。
    *   *Pro:* 直接视频可以展现肢体语言和面部表情
    *   反对意见:即使加快速度，也要花很长时间来检查所有的测试片段
*   **音频反馈**
    *   正面:更容易抓住用户最初的反应，不带任何感情色彩
    *   *Pro:* 变调，语调等。可以指示用户的情绪反应以及他们的反馈内容
    *   反对意见:即使加快速度，也要花很长时间来检查所有的测试片段

## [](#determining-production-value)确定产值

用户测试可以是正式的，也可以是非正式的，只要你需要。我一方面在付费平台上设计和管理用户测试，另一方面使用共享的 Google doc 和 Quicktime 录音。最终，付费工具可以提供的主要价值在于找到符合你的目标人群的合格人员来测试你。有许多次要功能，如视频转录、注释和录制，也可以使付费服务变得值得。

到目前为止，我用来管理用户测试的唯一付费服务是 [UserTesting](https://usertesting.com) ，但是我鼓励你探索 user testing 的[替代页面](https://alternativeto.net/software/usertesting-com/)，以获得类似用户测试工具的更广泛的概述。
