# 职位晋升是一个陷阱

> 原文：<https://dev.to/eduardsi/job-promotion-is-a-trap-4lg0>

我们喜欢升职。晋升给了我们职业发展的感觉:

初级开发人员→开发人员→高级开发人员→首席开发人员→架构师→高级架构师→首席架构师→ CTO → CIO(职业生涯结束)。

随着每一项未被锁定的成就，我们会赚更多的美元，我们的社会地位也会提高。我们正朝着我们的梦想前进。我们很快乐，生活也很美好。

> 但是职位晋升是雇主用来让我们在公司呆得更久的工具。
