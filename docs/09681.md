## [](#1-repl-read-eval-print-loop)1。REPL(读取评估打印循环)

*   Python:

您将通过 python 的安装获得它，并根据安装调用不同的 python 版本。

*   JavaScript:

Javascript 没有内置的 REPL，因为大多数时候开发者都在浏览器中运行，但是你可以通过安装 node.js 来使用 REPL

## [](#2-mutability)2。易变性

*   Python:

Python 有可变和不可变的数据类型，比如 set(可变)和 list(不可变)。

*   JavaScript:

Javascript 没有可变和不可变的概念。

## [](#3-strings)3。用线串

*   Python:

在 python 中，默认情况下源代码是 ASCII，除非您指定任何编码格式。

*   JavaScript:

Javascript 应该编码为 UTF-16，并且没有对处理原始字节的内置支持。

## [](#4-numbers)4。数字

*   Python:

在 python 中，有不同的数字类型，如整数(int)、浮点、定点小数等。

*   JavaScript:

Javascript 只有浮点数。

## [](#5-hash-tables)5。哈希表

*   Python:

Python 有内置的哈希表，称为字典、集合等，可以用在键和值的哈希中。

*   JavaScript:

Javascript 没有内置的哈希表支持。
