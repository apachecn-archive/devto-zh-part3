你好，我是穆罕默德，我是一名老程序员。(人群的声音:你好穆罕默德)，(程序员的代言人匿名:现在告诉我你的故事)。

好吧，好吧，不是这样的，但事实是我曾经是一名研究员，突然当我成为自由职业者时，我到处都是。

理论原理，过程编程，面向对象，SDLC(旧学校婴儿)。尽管如此，但当我坐在 ide 旁边时，我却愣住了，为什么？简单。他们不会告诉你真相。
编程是一个想法、积木、经验积累和聚合而成的脚本。

但是他们剪辑了这一切。

事实上，我和 Ruby、Python、Raspberry pi 等有过几次小小的接触，但是在我的脑海里，它们都是空的，因为我无法完成它们。

多动症可能是问题的一部分，但它的自我形象。程序员不是开发人员。
开发人员有一种将理论转化为代码的方法。
无论如何，这是我故事的一部分。
现在我又骑上了我的平台 Github。
祝我好运吧，伙计们，在我一个人的未知之旅中。
