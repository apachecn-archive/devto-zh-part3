# 安全空间& WoSEC

> 原文:[https://dev.to/shehackspurple/safe-spaces-wosec-587b](https://dev.to/shehackspurple/safe-spaces-wosec-587b)

大约一年前，我第一次去了以色列，并有幸见到了网络女士聚会小组的几位女士。我看到了她们之间美好的友谊，这种友谊是从她们行业中其他女性的聚会中建立起来的，我想“我也想要，这是一个很棒的主意”。我和我的朋友唐娜在加拿大的渥太华发起了一个类似的聚会，后来这个聚会变成了一个名为 WoSEC: Women of Security 的国际组织。

这篇博客的目的是告诉你我们的社区，如果你是女性(或以任何方式表明自己是女性)，邀请你参加。也欢迎有成人陪同的女孩。

#### [](#wosec-women-of-security)WoSEC:女性的安全感

*   WoSEC 是一个让女性在安全领域结交新朋友的社区，这样女性不仅会加入我们的行业，而且会留下来。
*   我们用我们的 twitter 账户来关注女性在 InfoSec 中所做的伟大事情，并努力创造一个持续的积极流。
*   我们“早午餐和婊子”(就像一针和婊子，但有食物)，这样我们可以交朋友，建立网络，并创建指导关系。
*   我们拥有安全的学习空间，有女性专用的研讨会、课程或讲座。我们在聚会和会议上这样做。
*   我们“撞车男孩事件”；通过参加一个团体，我们中没有人是“那里唯一的女性”。
*   我们正努力为所有在 InfoSec 工作(或想在 InfoSec 工作)的女性创建一个社区，希望她们能有更多的机会、更多的支持和更多的乐趣。
*   我们不收会员费，欢迎所有女性，即使有人没有参与 WoSEC，我们也很乐意分享她们的成就并为她们加油。
*   WoSEC 的成果包括:女性一起创业、指导关系、工作面试、实习、能够从我们的社区中招募人员、出现在女性演讲者面前给她们惊喜的啦啦队、相互支持、无价的建议和友谊。

*   我们可以互相支持，我们也这样做了。我们分享酒店房间或让对方搭车，我们组队迎接大挑战，我们介绍需要认识的人，等等；我们正在一起建立一个社区。

我邀请您加入我们在世界各地的 meetup(只需在 meetup 上搜索“WoSEC ”,看看我们是否在您附近有这样的聚会),或者联系我们，看看您如何能够参与其中或创建自己的分会。

感谢您的阅读。

加入我们的在线社区吧！
