2019 年 5 月 29 日，出于自私的原因，我发了一个帖子。我正在为一个图书馆工作，这个图书馆得到了我很多的爱。Tidal Node，你不需要了解太多，除了它允许你在同一个平台上运行 2 个虚拟机，更快的 Lua 和 Node.js v8 JavaScript 通过神奇的 WebAssembly。

TLDR
我用了大约一个小时的时间完成了 logo 1.0，你有什么想法或者主意来改进它或者它的发展吗？这个想法是 Node.js 徽标和 Lua 徽标以及我自己的想法的混合(因为无论如何它们都是丑陋的徽标)。

这些标志中的任何一个会让你多看一眼吗？请在评论中告诉我。🦄
