# 博主看别人的博客得到了什么？

> 原文:[https://dev.to/bassemibrahim/blogging-question-3fm](https://dev.to/bassemibrahim/blogging-question-3fm)

博主看别人的博客得到了什么？

我最近才开始写博客，科技博客是我的人生目标，我想做得更好。

我给自己定了一个目标，每天读 5 篇文章，让自己熟悉写作，并从其他作家那里获得经验和灵感。

有时我不想阅读，我只是浏览 5 个博客来说服自己我仍然在正轨上。

我只是想知道它有多重要。

我想我必须读它们，即使我不喜欢它来提高写作水平。

不管怎样，你们觉得这个怎么样？
