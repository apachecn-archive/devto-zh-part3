# 字体大小侦探

> 原文:[https://dev.to/n8kowald/font-size-detective-n8d](https://dev.to/n8kowald/font-size-detective-n8d)

在一个网站上很容易使用太多的字体大小。
特别是在使用`em`字体大小时，因为这些是相对于它们的父字体大小的。

我想要一种简单的方法来查看网站上使用的字体大小。我创建了这个 JavaScript 书签。

1.  在浏览器中添加新书签
