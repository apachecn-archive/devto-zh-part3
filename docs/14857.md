# Faqtor:基于承诺的任务执行人。易于使用，易于扩展。

> 原文:[https://dev . to/代数大脑/FAQ tor-promise-based-task-runner-易用-易扩展- 3lj7](https://dev.to/algebraicbrain/faqtor-promise-based-task-runner-easy-to-use-easy-to-extend--3lj7)

Faqtor 是 NodeJS 生态系统的基于承诺的任务运行器/构建自动化系统。现有插件支持 [rollup](https://www.npmjs.com/package/faqtor-of-rollup) 、 [uglify-js](https://www.npmjs.com/package/faqtor-of-uglify) 、 [browser-sync](https://www.npmjs.com/package/faqtor-of-browser-sync) 、[handler](https://www.npmjs.com/package/faqtor-of-handlebars)等一些工具。我将感谢任何反馈或支持。
