# 做了一段时间的网站开发人员后会发生的 4 件事

> 原文:[https://dev . to/flipped coding/4-things-than-happed-after-you been-a-web-dev-a-while-2d gp](https://dev.to/flippedcoding/4-things-that-happen-after-youve-been-a-web-dev-for-a-while-2dgp)

做任何事情足够长的时间都会导致一些…有趣的副作用。在你做了一段时间之后，Web 开发对你有它自己的影响。看看你能不能认出这些。

## [](#burnout)倦怠

好吧，任何事情都会发生这种情况，但我们在这里谈论的是 web 开发。当你有了第一个关于代码或者一个特别难对付的客户的梦想/噩梦时，恭喜你！你已经筋疲力尽了。走到这一步并不需要很长时间，尤其是在科技行业。

当 4 个项目中的 2 个有重叠的冲刺时，同时处理这 4 个项目会让你付出代价。你已经到了不再关心代码的地步。你只是想在进入下一个项目之前完成当前的冲刺。

有这种感觉很正常，因为发展速度很快。在你需要休息之前，你只能跑这么快，这没关系。当你可以的时候，退一步想想别的事情。

## [](#unexpected-job-requests)意外的工作请求

他们可以突然出现。你可能对你的工作或你正在做的任何事情非常满意，然后招聘人员突然给你发信息。老实说，这种感觉很棒。这是一个开发人员的好时机。

这个我再补充一点道理。这些随机的要求只有在你有了几年的经验，并且你处于适合你技能的领域之后才会真正开始出现。即便如此，如果你在一个足够大的城市，这种几率还是很高的。

确保你机智地处理这些请求，因为如果有一天你决定是时候换一份新工作了，你可能需要这些招聘人员来帮助你。至少回复他们的邮件，感谢他们联系你。

## [](#the-inability-to-not-think-in-code)无法不用代码思考

这个需要一段时间才能有意识地注意到。慢慢地，你更多的决策会变成 if 语句和 switch 语句。这些日常事务变成了你在一天的特定时间根据不同的条件触发的功能，你会开始怀疑你能处理多少异步任务。

编程能与你的生活联系起来，这是不可思议的。尽管这可能会让你周围的人感到恼火。朋友和家人会开始告诉你生活并不是绝对的，也不是每件事都需要一个符合逻辑的答案等等。

不过说真的，试着听听他们的意见。他们会帮助你走出你很久以前进入的与世隔绝的世界，帮助你重见天日。

## [](#the-pursuit-of-other-hobbies)追求其他爱好

当你写完一整天的代码后，有时你最不想做的事情就是回家盯着另一个电脑屏幕。你会开始产生以前从未考虑过的新兴趣。也许你会拿起尤克里里或者开始学日语。

这就是你如何战胜精疲力竭的感觉，并作为一名开发人员提升到下一个级别的方法。做一些不相关的事情可以帮助你学会如何在人们通常看不到的地方建立联系，因为你会有更多的机会接触不同的事物。能够以不同的方式思考是成为一名优秀的 web 开发人员的要素之一。

所以去参加即兴表演小组或者去外面散散步。无论你决定做什么，只要确保它与 web 开发无关。如果你真的喜欢网络开发，这可能比听起来要难。

当了一段时间的网站开发人员后，你还经历过哪些事情？我听说过一些…关于人们所做的奇怪的故事。请在评论中告诉我你的战斗伤疤，因为我确实读过它们。
