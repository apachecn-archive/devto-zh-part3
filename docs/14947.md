# 代码应该像诗一样吗？

> 原文:[https://dev.to/mortoray/should-code-be-like-poetry-i49](https://dev.to/mortoray/should-code-be-like-poetry-i49)

> "像对待诗歌一样对待你的代码，把它带到最低限度的边缘."-伊柳

在本周的节目中我强烈反对这个想法。诗歌是有意模糊的，让读者形成自己的印象，让自己的情感引导他们所看到的。代码应该是清晰和等价的。不过，我确实有点同意极简主义。

倾听我们所有的争论。此外，我的美食博客编程的状态更新和夏羽的追求得到一个完全匿名的网站。
