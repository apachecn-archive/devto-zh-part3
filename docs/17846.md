# 我们如何分配服务器资源来管理数百个网站和更多的网站？

> 原文：<https://dev.to/oelbaganwg/how-do-we-allocate-server-resources-for-managing-hundreds-of-websites-and-more-to-come-18cc>

我在寻求一些建议。我们为我们管理的 100 多个网站提供专用服务器，并为每个网站托管电子邮件。服务器是几年前用灵动网络安装的。我们已经超越了这台服务器，现在已经到了极限。

随着服务器达到极限，现在有了新的服务器技术，我想知道如何开始分配资源。所有网站也需要电子邮件。一些得到大量的交通和其他人没有。

我在考虑三个选择:

1.  只需为新网站安装第二台大型专用服务器。
2.  开始把每个网站放在自己的数字海洋水滴上？但是然后邮件呢？我想我们把电子邮件放在它自己的云上。

我正在想办法管理所有的服务器资源，当然也要考虑预算。

我很想听听其他开发者的想法。你会怎么做？非常感谢你们。
