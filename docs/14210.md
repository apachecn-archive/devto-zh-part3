# 大学在我们这个行业被低估了吗？

> 原文:[https://dev . to/michaeljota/is-collage-undertised-in-our-industry-4 CFG](https://dev.to/michaeljota/is-collage-underrated-in-our-industry-4cfg)

我是一名软件工程师，毕业于我的出生地委内瑞拉。我不得不说，由于大学是免费的(自 1960 年以来)，选择学习是一个优势，但即使它是有偿的，我可能最终还是会学习。我对此并不后悔，因为我学到了太多编程之外的东西，以至于我学到的还不如我自己学到的多。但是，其他科目，许多人可能会发现不相关，不足以证明拼贴的成本，但对我来说却是不同的。

你怎么想呢?拼贴是否被低估？你能真正自学成为一名优秀的专业人士所需要的一切吗？
