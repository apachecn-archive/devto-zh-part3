# React 钩子的大集合

> 原文：<https://dev.to/stereobooster/big-collection-of-react-hooks-2bab>

我越来越多地使用钩子。大部分时间我自己写钩子，公平地说，`useState`、`useEffect`、`useRef`在 80%的时间里都能工作。有时，当我编写自定义钩子并且对实现有疑问时，我发现检查其他人对类似钩子的实现是有用的。我认为大量引用钩子会很有用，这样我就不需要在不同的 GitHub 库之间跳转了。

我给尼克·格拉芙的收藏增加了 150 多个挂钩和[改进了搜索功能](https://github.com/nikgraf/react-hooks/pull/56)。你可以在这里 查看**最终结果[，而 PRs 没有合并。](https://stereobooster.github.io/react-hooks/?search=%23GraphQL)**

它对我来说似乎是一个有用的工具，也许对你也会有帮助。

如果你想做一些分析，比如跟踪大多数重新实现的钩子或者最大的库，所有的数据都可以用 JSON 获得。
