## [](#teck-stack)技术堆栈

这个模板内置在 Bootstrap 4 中，图标使用的字体是 Awesome 5。
我还决定使用`SASS`对样式进行编码，通过使用变量、函数和混合来更容易地分配颜色和字体系列。

## [](#purpose)目的

这个模板实际上是我上周构建的 2 部分演示项目的前半部分。我被安排在本周早些时候做一个关于 CraftCMS 的报告。这是工艺方用来显示输入记录的模板。

## [](#idea)想法

我想出了创建一个假公司的主意。这家公司专门为客户开发定制的 web 和移动应用程序。演讲的听众是当地的 PHP 社区。因此，我想出了公司的名字 *"Comphanny"* ，这是对单词" company "、" php "的一种玩法，而" ph "在" elephant "中读作" f "，大象是一种通常与 PHP 联系在一起的动物。

## [](#reflection)倒影

我最初想从头开始构建这个模板，来练习我的`CSS` &网页设计技能，以及探索 Flexbox 和 CSS grid。但是由于我日常工作的时间限制，而且仍然有项目的工艺部分要做，我觉得这太雄心勃勃了。所以我决定构建 Bootstrap，因为这是我所熟悉的。
