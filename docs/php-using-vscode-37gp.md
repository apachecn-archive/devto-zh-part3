# 使用 VSCode 的 Php

> 原文：<https://dev.to/jel111/php-using-vscode-37gp>

我正在学习 Php，并试图让 VSCode 格式化和编码完成我的文件。我有 VSC 推荐的扩展，并且四处看了看，但是看起来他们推荐的我都有了。我错过了什么场景吗？对我来说，问题是保存时不能正确格式化或标记任何内容。(甚至当格式化菜单中的文件时)我在设置中设置了 on_save 而不是 on_type。它甚至没有 Php 块的自动完成功能。
我错过了什么吗？我设置了自己的代码片段，这很有效，但是当你输入<时，我想知道为什么？VSC 不会自动完成剩下的部分。任何帮助都是令人敬畏的。

```
{

    "Php IF Statement": {
        "prefix": "pif",
        "body": [
            "<?php"
                "if ( $1 ) : ",
                " $2;",
                " endif;",
            "?>"
        ],
} 
```
