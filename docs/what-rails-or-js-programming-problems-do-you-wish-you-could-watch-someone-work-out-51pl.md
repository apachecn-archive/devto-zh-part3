# 你希望看到有人解决哪些 Rails 或 JS 编程问题？

> 原文：<https://dev.to/cohere/what-rails-or-js-programming-problems-do-you-wish-you-could-watch-someone-work-out-51pl>

朋友们好！

2019 年，我们将重心从客户工作转向无障碍编程教育。

我们处于一个幸运的位置，我们有一些现实世界中开放或接近开放源代码的生产应用程序，这些应用程序给我们带来了许多微妙的问题。我们在想“为什么不花时间把其中一些打包成对我们以外的人有用的东西呢？”

如果你可以让两个有着 30 年左右编写生产 web 应用程序经验的人一步一步地教你如何解决视频形式的特定问题，你会怎么做？

它会在几个小时内向现有的代码库添加一个特性吗？诊断和解决性能问题？执行复杂的多步重构？

也许是我们从未想过的完全不同的东西？不管是什么，我们都想听听，这样我们就可以找到一种方法来展示我们将如何一起解决这个问题。此外，如果你愿意，你可以在[的一个非正式 twitter 投票](https://twitter.com/zspencer/status/1074724243235950592)中投票，我们很抱歉没有提供 CodeyMcCodeFace 选项。
