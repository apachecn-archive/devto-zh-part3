# 从教师妈妈到程序员妈妈

> 原文：<https://dev.to/alexandrabryan/from-teacher-mama-to-coder-mama-4ea4>

介绍太吓人了！这和我在中学舞会或尴尬的第一次约会时产生的麻痹恐惧是一样的。什么是酷？我应该和谁交往？会给人什么印象？

你好。我叫亚历山德拉，30 多岁，有两个孩子。我不知道什么是酷，也没什么联想，但我一直被对学习的强烈热爱所驱使——所以如果要留下什么印象的话，那就是它。

最近，我是一名高中英语老师。我说的“最近”是指我从 2010 年开始从事教育工作，直到去年秋天，我发现教中学变得越来越累，而不是越来越令人振奋。在我递交辞呈，面对“那你下一步打算做什么？”，我转向编码。回来了，真的，因为我花了两年时间设计和编写电子邮件通讯。这里有一条漫长的道路，进出技术，我将进入另一天。但是很长一段时间，我从来不认为自己足够聪明或者足够技术，能够成为一名成功的开发人员。那些日子已经过去了。

所以现在我参加了一个各种各样的训练营，但不是典型的编码训练营。相反，作为 MomsCan:Code Develop Camp 的第一批成员，我磨练了许多开发人员的软技能——个人品牌、远程工作、同行编程等等——同时还学习了编码教程和构建产品。而且很神奇。我充满活力。我正在学习。我有责任感，有同伴的支持，还有一个了不起的技术导师。

从长远来看，我想在我的生活中收敛这些不变的线程。学习，技术，教学，设计，编码，故事。我还没想好我该怎么做，但是这个过程已经是一半的乐趣了，对吗？

只要我不让那种令人麻痹的恐惧挡了我的路。
