# 周末黑客-这个周末你在做什么项目？

> 原文:[https://dev.to/chasestevens/weekend-hacking-这个周末你在做什么项目-53jb](https://dev.to/chasestevens/weekend-hacking----what-projects-are-you-working-on-this-weekend-53jb)

星期五快乐，德夫图！

我们都有自己想探索的兴趣和领域，我们在一周内也有义务。周末是给自己留出时间、学习新东西、探索新想法或做兼职项目的好时机。

这个周末，我在写一篇博文，用一家餐馆作为类比，解释 Ruby on Rails 的模型-视图-控制器架构。我还报名参加了周六上午的 Python 公开黑客会议。我现在的想法是做一个类似聊天客户端的东西，但不是和朋友或同事交谈，而是和一个聪明的指导橡胶鸭交谈，他会帮助你调试你的代码。

我想接触一下，看看其他人都在做什么——这个周末你打算黑什么？
