# 开发日志#1:登陆者游戏

> 原文:[https://dev.to/niorad/dev-log-1-lander-game-238k](https://dev.to/niorad/dev-log-1-lander-game-238k)

我喜欢时不时地做游戏。我发布的最后一款游戏是 2016 年的 [Dubiduh](http://dubiduh.com) ，这是一款简短的益智游戏，你可以通过关卡控制一个蜿蜒的幽灵。

那已经是几年前的事了，我想再次发布一些小东西！

我不知道这将会如何发展，或者是否会有第二篇开发日志，我只是想写下一些话。

想法如下:

*   玩家负责使用太空机库的着陆平台。机库有几个这样的平台。
*   每隔几轮/迭代就有一艘船到达。
*   玩家决定它可能落在哪里，或者是否把它送走。
*   船只支付登陆费，每轮都在机库。
*   每艘船在一轮或几轮后离开。
*   每艘船都属于一个分支:警察，公司，卡特尔。
*   玩家拥有每个分数的声望。
*   把一艘警察船放在一艘卡特尔船旁边会提高警察的声誉，但会降低卡特尔的声誉。太空中的石头剪刀布。
*   所有的组合都会有一些类似上面的效果。
*   声誉越好，该分数为登陆/停留支付的费用就越多。

因此，主要目标是战略性地放置/发送即将到来的船只，以达到特定的财务目标，或者构建额外的平台(但不是以 cookie-clicker 的方式)

平台可以是一排 4 个，或者甚至是不同船大小的 2D 网格。

可能会有一些事件，比如一艘卡特尔船承诺给你一大笔钱，因为你不允许警察巡逻 n 圈。

但是现在让我们保持低调。我正在开发一个原型，玩家应该能够放置即将到来的船只，获得住宿费用，并根据描述的效果获得不同的声誉。

因为我已经迷恋仙丹有一段时间了，所以我选择它作为初始代码。我现在将接口保持为 ASCII，因为我想在绘制单个像素之前弄清楚规则。我想这也适用于纸和骰子。

我把它叫做“NEORAID ”,因为我打错了“niorad ”,而且喜欢这个词。
