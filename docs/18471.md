# 根据最佳实践做出反应

> 原文:[https://dev.to/metamn/to-react-with-best-practices-4a9o](https://dev.to/metamn/to-react-with-best-practices-4a9o)

WordPress 和 Gutenberg 在一起，是时候学习 React 了。

在过去的两年里，我一直在做一些远离边缘的项目，没有时间去了解网页设计和开发的最佳实践——现在是重新学习的时候了。

我习惯了在荒野中感到舒适。2015 年，我用 Gulp 创建了自己的[风格指南。然后我出现在](http://metamn.io/styleguide/)[野兽派网站](http://brutalistwebsites.com/metamn.io_gust/)上。我还经营着[的另一个博客](http://metamn.io/beat/)，关注行业前沿。

我仍然觉得被落下了。

我最大的问题是，我的旧框架建立在 BEM / SCSS / HTML5 和普通 Javascript 之上，除了 Javascript 部分，一切都很好。我努力让屏幕上的东西移动。是的，我需要找到比 Javascript 更好的东西，这就是我第一次去 Vue，然后现在做出反应的方式。

有了这些框架，你就不用编写普通的 JS 代码了。您用最少的原始 Javascript 编写奇怪的标记和样式来声明和控制 UI。只是把框架提供的东西连接起来。那是我真正需要的。

我对 Vue 充满热情。这是与反应式 UI 世界联系的最好和最快的方式。

一年后，现在，我开始反应。我真的很喜欢它。在一周的时间里，我设法建立了一个完整的故事书，以我想要的方式创建了 T2 的新组件，并且在一个下午的时间里完成了这个博客。谢谢[盖茨比！](https://www.gatsbyjs.org/)
