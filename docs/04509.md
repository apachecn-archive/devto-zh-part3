# Gmail 键盘快捷键

> 原文:[https://dev . to/therealdarkmage/Gmail-keyboard-shortcut s-20kc](https://dev.to/therealdarkmage/gmail-keyboard-shortcuts-2okc)

管理我的收件箱一直是争论的焦点，尤其是在我过去担任安全顾问的时候。当我应该做 pentests 时，我被迫手动一个接一个地删除电子邮件，因为微软 Outlook 365 的 web 客户端是狗屎，我拒绝使用本地邮件客户端(我相信能够在尽可能舒适的环境中工作的哲学)，并且当时的 Outlook 365 web 客户端，或者至少我们特定安装的它没有“全选”能力，或者甚至没有选择单个电子邮件的能力，以便我可以，例如，一次选择几封电子邮件来删除。

当时我甚至不需要这种能力，但我的工作场所只给我分配了 500mb 的磁盘空间来存放我的电子邮件。他们本来可以使用另一种解决方案，但微软是我们的客户之一，客户真的很喜欢你说你在使用他们的产品。

我以前为使用 Gmail 的地方工作过，使它们比使用微软产品的地方更加智能和优越。你得到更多的磁盘空间，使管理过程更容易，压力更小。

此外，最近我开始熟悉 Gmail 的键盘快捷键。我经常归档邮件，用双手的鼠标手动选择每一封邮件变得非常痛苦。我有药物可以帮助你，但是你不想简单地...用药然后继续伤害你自己。那很糟糕。你需要采取行动来变得更加安全和高效。

输入 Gmail 键盘快捷键。作为一名编辑，我喜欢 vim 的原因和我喜欢 Gmail 键盘快捷键的原因相似:效率！

以我的简单用例场景为例:电子邮件存档。我每天检查我的收件箱，归档那些只是随机通知或者我已经读完的邮件。我会在收件箱里保存一些东西作为提醒，但一般来说，我会把所有收到我收件箱或它的标签(主要、社交、推广)的东西都存档...大部分都是注册的。社交活动和促销活动每年都会被藏起来几次。我应该修理那个 lol。

不管怎样，我的用例场景是:你可以完全用键盘来完成这些，而不是选择单个的电子邮件或点击按钮。

首先，您需要在 Gmail web 客户端的“设置”中启用键盘快捷键。启用后，返回收件箱主选项卡。现在，您可以使用箭头键在邮件列表中上下导航。

```

Navigate to all-mail: g+a...like, you press and hold g then press a
Toggle star: s

Mark as read: Shift+i
Mark as unread: Shift+u
Undo last action: z
Reply all: a
Delete: #
Report as spam: !
Move focus to toolbar: ,
Mark as important: = or +
Mark as unimportant: -

Tried this one, could not get it to work
-----
Switch between Calendar/Keep/Tasks sidebar and inbox: Cmd/Ctrl+Alt+, and Cmd/Ctrl+Alt+. 
```

考虑到我现在的交流量，适应这些快捷方式就像是天赐之物。我不再被诅咒用鼠标点击我的邮件选择，或者用鼠标点击存档，或者用鼠标点击...几乎所有与电子邮件相关的东西...我没有被迫在我的智能手机上使用 Gmail 应用程序(公平地说，这对于左手滑动存档来说很好，这是 def。我手腕上的伤口)。
