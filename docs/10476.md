# 宣布 Aybolit:用 LitElement 构建的轻量级 web 组件库

> 原文:[https://dev . to/web Padawan/announding-ay bolit-lightweight-we B- components-library-build-with-litelement-302 f](https://dev.to/webpadawan/announcing-aybolit-lightweight-web-components-library-built-with-litelement-302f)

昨天，我发布了 Aybolit 的 0.1.0 版本，这是我最近一直在做的宠物项目。简而言之，它是关于编写 web 组件和寻找流行的 CSS 框架之间的“共同点”。

Aybolit 旨在解决的问题是:具有自定义元素的粒度 DOM 结构、由于 Shadow DOM 而具有作用域的 CSS，以及具有自定义 CSS 属性和 calc()的灵活主题化 API。它还利用了 Chrome 中可构造样式表的优势，因为对它的支持已经集成到 LitElelment 中。

最初的版本 0.1.0 只包含 5 个组件:按钮、复选框、进度、范围和开关(更多组件即将推出)。同时，它为所有这些提供了引导，布尔玛和材料版本，加上“白色标签”的样式最少的变体，易于覆盖。

我感谢任何反馈，并希望在社区的帮助下使 Aybolit 变得更好。自从我开始用聚合物做第一步以来，已经快 3 年了，我把我所有的经验和热情都投入到这个项目中。
