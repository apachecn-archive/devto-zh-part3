# 失去了个人计算的潜力

> 原文:[https://dev . to/progrium/lost-potential-of-personal-computing-5d1i](https://dev.to/progrium/lost-potential-of-personal-computing-5d1i)

你还记得探索计算机有趣的时候吗？也许你一直认为探索计算机很有趣，但在互联网出现之前的个人计算时代，人们对计算机的潜力感到兴奋。当然，它们可能已经超出了我们今天的大多数预期，但与此同时...不一样。我们得到我们所希望的了吗？我们现在还能从计算机中获得希望吗？

今天，我们认为我们了解计算。我们见过它，用过它。它就在我们周围，对吗？今天的一个区别是，我们不去想电脑能为我们做什么，而是去想公司能卖给我们什么。计算的商业化使我们接近无处不在的计算，但与此同时，总的来说，它慢慢侵蚀和模糊了计算作为人们集体和个人进步工具的真正生产潜力。

应该说我说的大多是软件。软件正在吞噬这个世界。我们已经关闭了硬件。我们口袋里的计算能力比我们登上月球所需的还要多。我们用那种力量做了什么？

即使作为软件开发人员，你上一次能够编写这个工具——可以变成任何工具——来改善你的生活是什么时候？甚至可能只是你的工作生活！如果你有一点点创造力，你可能已经想象出了有趣的项目或者理想的工作流程或工具。你试过建造它吗？容易吗？或者也许你甚至没有想到这样的改进...为什么不呢？一个很大的可能性是:你还没有得到启发你*可以*足够容易地建造这样的东西的积木。

还记得那句话“有一个应用程序可以做到这一点吗？”这经常是真的，但不总是，当它是真的时，它真的是你想要的吗？它真的解决了你的具体问题吗？如果是的话，它还做了什么你不需要的事情？毫不奇怪，我们使用的最流行的应用和工具是社交。我不反对社交软件，但我认为社交软件把社交作为拐杖。让你与人联系和互动的软件总是有价值的。你甚至不用很努力，记得 MySpace 吗？

但我不是在说技术成就或者代码有多漂亮什么的。我说的是改善你生活的工具。帮助您完成更多工作的工具。帮助你更好地利用他们。Twitter 真的改善了你的生活吗？

道格·恩格尔巴特以发明鼠标而闻名，但他实际上是大多数现代计算的先驱。他的工作相当深入，尽管有些人开玩笑说当他的想法用完时，硅谷会做什么，但在我看来，他们甚至没有触及表面。

从 1960 年直到几年前去世，道格一直在说，“人类面临的问题越来越复杂，超出了我们解决这些问题的能力。为了提高我们解决世界问题的集体能力，我们必须利用技术的巨大潜力和力量。”

他相信计算是实现这一承诺的手段。我的观点是，我们认为我们了解计算，但我们并不了解，真的。我们知道它的一个版本已经被打包卖给我们了。我相信，如果我们能够想象，我们可以从现有的东西中获得更多。一个可以变成任何工具的工具，不亚于一个想象力编译器。
