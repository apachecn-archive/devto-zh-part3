# 可穿戴设备和物联网推动实时监控

> 原文:[https://dev . to/getset capture/wearables-and-IOT-drive-real time-monitoring-2ff 8](https://dev.to/getsetcapture/wearables-and-iot-drive-realtime-monitoring-2ff8)

事实上，互联网是一个巨大的实时数据流海洋。数据流也来自不同的其他来源，如连接到传感器和可穿戴设备的物联网。只有对这些数据进行实时监控和分析，这些设备及其产生的数据才能发挥最大效益。

### [](#wearables-and-iot-data)可穿戴设备和物联网数据

谈到可穿戴设备，首先想到的是智能手表和健身追踪器，它们配备了可以监测温度、心率和其他参数的传感器。此外，还有其他可以监测呼吸频率的可穿戴设备和具有冲击检测功能的运动头盔。想想医疗保健场景，物联网和可穿戴设备开始发挥作用。配备可持续传输数据的可穿戴设备的患者可以实时监控自己的状况。其他地方的医生可以根据可穿戴设备传输的数据接收警报。如果出现治疗需求，医生可以借助配备物联网传感器的医疗设备提供的数据来定位他们。当这样的设备就位时，接受治疗的患者可以一直被监控。

除了健康数据，带 GPS 的可穿戴设备还可以精确定位个人位置，让其他人知道他的准确位置。例如，卡车司机可以配备可穿戴设备，监控他们的状况，并将数据传输到中央办公室。卡车可以有连接到中央办公室的传感器，中央办公室监控其位置、进度、速度以及发动机、轮胎和其他部件的状况。例如，物流可以发现可穿戴设备和物联网大大提高了服务质量。

同样的逻辑可以移植到制造过程中。设备中的基本自动化包括使用 PLC/CNC 来自动执行步骤，使用各种指示器来显示压力和温度以及其他参数。它仍然需要人来观察这个过程。传感器向中央控制单元馈送数据的简单实现消除了对人工监督的需要。同时，物联网提供了实时监控的双重优势。一个是过程本身和正在处理的材料发出数据流，这些数据流可以被记录下来供以后分析。数据流也可以被标记，以在故障或停止时触发警报，并减少昂贵的浪费。运行这些过程的设备可以有自己的一组传感器，用于传输关于其性能和状态的数据。这大大增强了预防性维护，延长了使用寿命，降低了成本。同样的原理也可以应用于汽车，其中物联网流数据提供关于汽车及其各个部件的信息。以农业和物联网实施为例，它可以帮助远程监控泵以及化学物质含量、pH 值、温度和水分含量以及其他参数，特别是在温室中。给在这些环境中工作的人佩戴可穿戴设备会产生额外的数据，如他们的行踪和健康状况。

此外，还有令人恐惧的奥威尔式前景:智能城市为所有公民配备可穿戴设备和 IOT 连接，让他们的生活时刻受到追踪。然而，这是另一个讨论的话题。

### [](#beyond-monitoring-and-collection-of-data-control)超越监控和数据收集-控制

在互联的世界里，监控是数据流中最简单的部分。通过人工智能和机器学习的事件流处理来利用它，这种数据可以用于做出基于事实的决策，从而降低成本，改进流程和提高质量。例如，时间、温度、压力和湿度可能是工业过程中的主要参数。传感器感应到这些，并将其传输到监控系统。然而，当将这些数据与具有机器学习算法的后端控制程序一起使用以管理设备中的控制器并确保过程的一致性时，可以将这些数据投入实际使用以确保过程的最佳性能。这些数据成为复杂的自动化控制系统的输入。为了实现这一目标，该行业需要保留一家开发公司的服务，该公司熟悉物联网、传感器及其工作原理、制造流程以及人工智能机器学习的实施。当然，人们可以简单地分析数据，以了解如何改进流程，但实施基于物联网数据实时流的更好、高度复杂的自动化控制系统是一个更好的选择。
