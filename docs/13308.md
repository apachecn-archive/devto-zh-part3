# 使用消息对时差进行主题化

> 原文:[https://dev . to/designer 023/theming-slack-using-a-message-3hm 4](https://dev.to/designer023/theming-slack-using-a-message-3hm4)

Slack(桌面)客户端允许你创建自己的主题。但是你知道吗，你也可以通过信息来设置主题。只需发送一条内容为 8 个逗号分隔的十六进制代码的消息。你也不需要把信息发给任何人——只要发给你自己就行了！

```
#DCDCDC,#ED6970,#ED6970,#FFFFFF,#FFFFFF,#291d1f,#1c7c5a,#3a4a8ff 
```

这些代表以下元素:

*   侧边栏和顶栏背景
*   ？？？？
*   突出显示的频道背景
*   突出显示的频道文本
*   突出显示的频道背景(悬停)
*   侧栏文本
*   在线指标
*   消息指示符

一旦你把这个发给你自己，一个小按钮就会出现，让你能够触发主题改变。
