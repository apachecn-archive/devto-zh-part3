# 意大利 PyCon X:提交意大利 Python 大会(佛罗伦萨，5 月 2 日-5 日)的报告

> 原文：<https://dev.to/andreagrandi/pycon-x-italy-submit-a-talk-for-the-italian-python-conference-florence-may-2nd---5th-16f0>

甚至在今年，意大利 Python 大会也将在**佛罗伦萨**举办，并将于 5 月 2 日至 5 月 5 日在**举行。**

这是一个很棒的机会，可以认识很多 Python 开发者，听非常有趣的演讲，参观世界上最好的城市之一(我可能对这最后一点略有偏见:P)！

主题范围从纯粹的 **Python** 、 **PyData** 、**机器学习**、 **web** 、 **DevOps** 、**安全**和**社区**相关的演讲，并且将会有大量(通常是大多数)以**英语**提供的演讲。

没有我们出色的发言人，这个活动就不可能进行:不仅仅是专家，尤其是第一次发言的人都被热情地鼓励为会议提交提案。

如果你有兴趣，你可以在官网找到所有的信息[https://www.pycon.it/en/call-for-proposals/](https://www.pycon.it/en/call-for-proposals/)

请**抓紧**，因为**的截止日期是 1 月 6 日**！
