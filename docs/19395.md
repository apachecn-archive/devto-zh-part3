# 我为什么参加编码训练营

> 原文:[https://dev.to/bechev/why-i-attended-a-coding-bootcamp-49hc](https://dev.to/bechev/why-i-attended-a-coding-bootcamp-49hc)

经过 8 个月的努力工作，我终于到了——即将从我的编码训练营毕业。那现在呢。

### [](#flashback)**闪回**:

大约 18 岁时，我在法国的一所工程学校上学。这是我第一次体验编码的地方。我以前从未真正接触过它。我至少可以说，经验并不十分确凿。那是差不多 15 年前的事了，大多数记忆都有点模糊了，但我记得的是它有多抽象。
我们被要求构建一些将返回一些值的函数。*哼...好的。那又怎样？发生了什么事？*如你所知，当你返回一个值时，什么都不会被打印出来， ***你看不到函数的结果*** 。这正是我迷路的地方。我需要 ***看看*** 发生了什么。几堂物理课和几个月后，我做出了决定。我将学习经济和金融而不是工程学。

### [](#fast-forward-3-years)**快进 3 年:**

我现在正在攻读金融硕士学位，作为所有金融专业的学生，我需要精通 Excel，我课程的一部分是延伸到 VBA 和 c 的专门课程。VBA 实际上非常视觉化。无论你写什么样的代码，你都能在电子表格中看到变化。真是 ***牛逼*** 。我喜欢！( ***我非常喜欢看到这些变化，所以我最初使用电子表格作为数组，这样我可以看到更多的变化...*** )在我的朋友群里，我成了“**编码佬**”。负责项目的编程方面(期权定价、股票市场模拟等)。

### [](#fast-forward-11-years-roughly-8-months-ago)**快进 11 年(大概 8 个月前):**

我现在做财务方面的顾问，几乎不做任何编码。这些年我拿了一堆网上教程和 MOOC 自学。我了解面向对象编程的基础， ***因为在 Excel*** 上编写了一个带变色装饰的圣诞树而差点被解雇，最近搬到纽约工作。
来自法国的我在纽约发现了一个全新的世界。人们有 20 个副业，他们不断学习新事物，其中一个非常流行的学习是编码。为了满足这种需求， ***训练营*** 就在这里，这正是我所做的。

### [](#current-times)**当前次数:**

已经 8 个月了，我完成了我的在线训练营，我“ ***知道了*** ”如何建立一个网站。那么现在， ***又是什么呢？***
