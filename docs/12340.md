# GraphQL 资源:面向初学者的顶级工具、扩展和教程

> 原文:[https://dev . to/graphql editor/graphql-resources-top-tools-extensions-resources-for-初学者-2f13](https://dev.to/graphqleditor/graphql-resources-top-tools-extensions--resources-for-beginners-2f13)

自从脸书在 2015 年推出 GraphQL 以来，它的受欢迎程度不断增长。如果你想学习 GraphQL，这里列出了最好的工具、扩展和教程。

* * *

### [](#knowlege-base)知识库

1.  [GraphQL 官方文档](https://graphql.org/learn/)绝对是你开始 graph QL 冒险的最好地方。
2.  如果你正在寻找有关 GraphQL 项目的原始知识，脸书官方 RFC 规范草案是一个好去处。
3.  GraphQL 周刊是一份时事通讯，包含新闻、文章以及与 GraphQL 相关的一切
4.  [How to GraphQL](https://www.howtographql.com/) 是一个网站，它将带你进入一个从 GraphQL 新手到发布你的第一个 GraphQL 项目的旅程。
5.  Explore-tech.org 是收集 GraphQL 工具的有用资源，尤其是库。
6.  由 hasura 团队为前端开发人员制作的 2 小时 GraphQL 教程系列

* * *

### [](#tools-amp-extensions)工具&扩展

1.  GraphQL Docs 是一种在几秒钟内为 GraphQL 模式生成静态文档的简单方法。
2.  GraphCMS 一个允许你定义 GraphQL API 的工具。创建结构、关系、强大的权限并轻松管理您的内容，一般来说，为您的 web 项目构建一个托管的 GraphQL 后端以及管理其内容的工具。
3.  [GraphQL 编辑器](https://graphqleditor.com/)让理解 GraphQL 模式变得更加容易。通过组合可视块来创建它，编辑器会将它们转换成代码！从 URL/文件或虚假后端数据加载模式等功能可以加快应用程序的原型制作。
4.  [Apollo Launchpad](https://launchpad.graphql.com/new) 允许你用 JavaScript 编写一个 GraphQL 模式，使得 GraphQL 代码非常容易访问。
5.  GraphQL 是一个简单的 IDE 工具，用于测试和记录 GraphQL APIs，具有漂亮的图形界面。
6.  GraphQL 游乐场一个 graph QL 的替代品。
7.  [Apollo Elements](https://github.com/apollo-elements/apollo-elements) 一个用于构建无 UI 框架的 GraphQL 支持的 UI 的库。
8.  牛郎星 -一种图形的替代品
