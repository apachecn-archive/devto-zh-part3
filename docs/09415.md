今天，我在写我在 Lambda 学校 7 个月的课程中学到的东西。从 2018 年 7 月份开始，感觉自己在这么短的时间里成长了这么多，一路走来遇到了这么多不可思议的人。为了让你更好地理解我的出身，我辞去了药剂师这份轻松的医疗工作，尝试了不同的商业冒险。长话短说，我发现编码真的很酷，真希望我 8 年前就学了这个。我参加了编程课程，开发了一些小应用程序来学习，并加入了弗吉尼亚当地的开发小组。在一年的自学之后，我决定要完成我的网络开发教育。在大约一年的时间里，我设法自学了 javascript、reactjs、nodejs 和 mongodb。我知道在我的 web 开发之路上还有很多东西要学。因此，我决定报读拉姆达学校。我在 Lambda 学校学习了其他语言和计算机科学。这也将使求职过程变得更加容易，因为他们有一个很棒的职业团队。

在 Lambda 学校学习期间，你是如何克服困难的？你个人是如何成长的？专业上？

最初，我没有太多的程序前端模块的斗争，因为我有以前的经验。后端也很熟悉，但我还是设法学到了一些新东西。当我们进入这门课程的计算机科学课程时，事情变得尤其具有挑战性。进度更加自主，主题尤其具有挑战性。一路走来，我们也学会了像 Python、Django 和 c 这样的新语言。

我通过坚定的逻辑思维克服了困难。我不会告诉你这很容易，但我会告诉你，如果你坚持下去，这是可能的。如果挑战很大，我会把它分解成更小的部分，并试图清楚地理解它们。一旦理解了，我会把拼图拼在一起，得到一个更大的画面。浏览相关主题的视频和话题也需要极大的耐心。最坏的情况是，我知道我的站立团体和同伴可以通过简单的直接或渠道获得信息。

我觉得我在个人发展方面取得了巨大的进步。我明白了，只要我保持冷静，我就能处理好问题。挫折会限制进步，但有时会帮助我寻找更多的信息。就职业成长而言，我觉得在经历从令人惊叹的老师到雄心勃勃的学生的课程时，我建立了很多联系。当我们做 Lambda Build Week 和 Lambda Labs 的时候，我学会了更好地与其他工程师合作。我了解了自己的缺点，并努力提高这些技能。

向我们介绍一下你在 Lambda 的经历。你遇到了哪些考验，你是如何克服的？

Lambda 的大部分体验都是由讲师指导的，并随着计算机科学课程的进行而慢慢减少。我必须经历的考验是学会以更好的方式自学。在这一点上，我习惯于自学，但当你有时间编写代码时，故事就变了。我克服了这一挑战，确保我很好地理解了这个话题的基础，并花了我的时间。当新的变化和替代解决方案出现时，彻底的理解帮助了我。如果我不能带走任何东西，我会试着至少记住问题的基础和实现。

什么最有助于你坚持每天上班并尽最大努力？

我把我的大部分成功归功于休息好和课外的恢复时间。最重要的是要保持冷静、营养和清醒，以便进行更多的学习。就像勒布朗·詹姆斯在比赛后花了很多时间恢复一样，我不希望我的表现在比赛时不受影响。我不是全押的坚定信徒，因为这是一种很快筋疲力尽的方式。我至少从我的医疗生涯中学到了这么多。

就尽力而为而言，看到我的同学每天都在工作是一种激励，这进一步激励我也去参加。当我遇到问题时，有人依靠感觉很好，因为我们都在艰难地解决同一个问题。这是我去年自学时与参加 Lambda 学校相比的主要区别。这个社区给了你很多帮助，帮助你坚持工作。
