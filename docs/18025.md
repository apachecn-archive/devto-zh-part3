# 新代码将 v5.0 与 C# 6.0/7.0 和 VB 的结构比较进行了大幅改进

> 原文：<https://dev.to/devartsoftware/new-code-compare-v50-with-greatly-improved-structural-comparison-for-c-6070-and-vb-2ajd>

Devart 是一家公认的数据库连接解决方案和数据库管理与开发工具供应商，它发布了新版本的 Code Compare，实现了许多新的增强功能来提高 Devart 用户的工作效率。

代码比较的更新提供了以下新功能:

*   大大改进了 C# 6.0/7.0 和 VB 的结构比较*改进了最新版本的с#和 VB 语言的结构比较。
    *改进了注释和处理器指令与代码中结构元素的绑定。

*   VS 中对 JavaScript 和 TypeScript 编辑器的支持现在，Visual Studio 的 Code Compare 中的可用语言列表提供了 JavaScript 和 TypeScript 等流行语言。

*   大文件的初步比较
    运行比较后，用户会看到一个对话框，显示每个文件(大小超过 10 Mb)的大小、日期和上次修改的信息，以及文件差异的信息。

*   改进的对话框风格
    *改进了对 4-5K 显示器和可用性的支持。
    *为了在独立应用程序中更方便地滚动，支持 Shift +鼠标滚轮快捷键。

*   文件夹比较改进

代码比较是一个免费的比较工具，旨在比较和合并不同的文件和文件夹。Code Compare 集成了最流行的源代码控制系统:TFS、SVN、Git、Mercurial 和 Perforce。代码比较作为独立的文件比较工具和 Visual Studio 扩展提供。
