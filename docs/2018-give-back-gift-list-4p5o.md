# 2018 回馈礼单

> 原文：<https://dev.to/davefollett/2018-give-back-gift-list-4p5o>

每到圣诞节，我总是害怕为我想出家人可以用的礼物点子。那太贵了吗？是不是不够贵？这种非常特殊的电子器件容易买到吗？这些只是我每年都会问自己的一些问题，但今年不会。我会的💀🐦🐦⚫.今年，我有了一个完美的方法来建立一个清单，它也核对了我一直想做的另一件事。通过购买他们的赃物来支持我每天使用的免费内容制作者和开源项目。所以我向你介绍:

### 🎄🎄🎄戴夫·福利特 2018 回馈礼物清单🎄🎄🎄

### [t1】1️⃣](#codingblocks-podcast)[coding blocks 播客](https://www.codingblocks.net/)

[CodingBlocks 播客](https://www.codingblocks.net/)是我听过的第一个播客，它让我迷上了使用播客作为学习媒介。《T2》播客主持人、[艾伦·安德伍德](https://twitter.com/theallenu)、[乔·扎克](https://twitter.com/thejoezack)和[迈克尔·奥特洛](https://twitter.com/iamwaltuo)，将复杂的编程主题分解成易于理解的解释，一路上充满乐趣。他们还运营着一个非常活跃和支持性的[松弛渠道](https://www.codingblocks.net/slack/)。所以我把这个漂亮的双色咖啡杯放在我清单的首位。

我最喜欢的另一个播客是愤世嫉俗的开发者。主持人[詹姆斯·斯图达特](https://twitter.com/JamesStuddart)每周都会制作播客，内容包括分享他的经验或就最新发展技术进行采访。我几乎像喜欢他的播客一样喜欢这件醒目的蓝色 t 恤。差不多了。

### 3️⃣ [开发到](https://dev.to/)

如果你还没有检查过 [dev.to](https://dev.to/) ，你真的应该检查一下。这是一个开发者友好的发布软件开发文章的平台。我在自己的网站上发表文章，但也会交叉发布到 dev.to 上，因为它们提供了原始资料的规范链接。在[本·哈尔彭](https://twitter.com/bendhalpern)、[李佳薇](https://twitter.com/jessleenyc)和[彼得·弗兰克](https://twitter.com/PeterKimFrank)的带领下，它已经迅速成为我阅读软件开发的首选资源之一。他们还有一个大商店，里面有很多非常好的商品。我真的希望我最后能戴上开发帽！

### 4️⃣ [Ubuntu Linux](https://www.ubuntu.com/)

我已经用了很长时间的 Linux，T2 的 Ubuntu 是我最喜欢的发行版。对我来说，它在稳定和提供最新软件包版本之间取得了平衡。此外，他们刚刚宣布 [Ubuntu 18.04 将支持 10 年](https://www.zdnet.com/article/mark-shuttleworth-reveals-ubuntu-18-04-will-get-a-10-year-support-lifespan/)，所以这款 t 恤应该会流行很长时间。

### [vista . js](https://vuejs.org/)

在我的日常工作中，我不是一名 JavaScript 开发者，所以当我决定学习它和 JavaScript 框架时，我选择了 T2，因为我听说它的门槛很低。哇，我完全被震撼了。对于非 JavaScript 开发人员来说，它不仅易于掌握，而且还通过其 CLI 和 DevTools 提供了一种[的优秀开发体验](https://www.youtube.com/watch?v=UW6YycsYzHY&t=3884s)。尤雨溪 (Vue 的创造者)和他的团队真的创造了一些特别的东西，我很乐意穿着这件 t 恤在镇上逛逛。
