# 从新的云平台 GCP 开始

> 原文:[https://dev . to/manas1v/starting-with-a-new-cloud-platform-GCP-3o2b](https://dev.to/manas1v/starting-with-a-new-cloud-platform-gcp-3o2b)

在 AWS 工作了 4 年多之后，我开始学习 GCP。云或任何基础设施的主要构造总是相同的。重要的是理解不同之处，这些不同之处允许您以更简单的方式构建比您在早期云上实现的更强大的东西。我很高兴找到了这个课程-[AWS 专业人员的谷歌云平台基础](https://www.coursera.org/learn/gcp-fundamentals-aws)

它给出了 Google Cloud 功能的比较，让了解 AWS 术语和概念的人非常容易理解。

事实上，这样的课程也有助于加强你对 AWS 的理解。现在，您能够以全新的视角看待您所了解的云。它还谈到了可移植性解决方案。

云的发展如此迅速，以至于我们大多数人都有过某种形式的云体验。对于 Y 一代，他们很早就知道云，他们的照片、信息、文档甚至大学作业都存储在那里！

每个主要的云提供商都应该为有一些云经验的开发者准备一个入门文档。

学完这门课程后，我可以继续在 qwiklabs 中实践其他 GCP 课程和概念验证。

快速提示:

1.  为你学习的每门课程/辅导课做笔记
2.  动手:假设你知道，不要跳过动手练习
3.  浏览你的笔记后，两周后再次访问该课程
4.  请参考 GCP/AWS 的主要文件，了解最新版本和所需的详细信息
5.  计划一个练习/小解决方案，以便在课程结束后开发
