# 我在 Lambda 学校的头十周

> 原文:[https://dev . to/lafriedel/my-first-ten-weeks-at-lambda-school-MCA](https://dev.to/lafriedel/my-first-ten-weeks-at-lambda-school-mca)

上周，我完成了 Lambda School 全栈 Web 工程和计算机科学课程的第十周。这意味着我现在已经完成了 30 周全日制课程的三分之一。

我想总结一下在这段时间里我学到了什么，完成了什么，不仅是为了我自己，也是为了所有考虑申请 Lambda 学校的人。虽然我的经历只是众多经历中的一个，但我希望这能让你对这个项目有所了解。

## [](#what-brought-me-to-lambda-school)是什么把我带到了λ学校

2017 年 7 月，我决定要转行做网页开发。我当时的时间表是不可预测的(我是一名自由职业者，担任活动设计师和风景艺术家/壁画家)，所以我一有时间就自学基础知识。因为很难提前计划，有时学习会间隔几周。

这是一个缓慢的过程，我知道如果我想很快成为一名专业开发人员，我需要学习更多的东西。我于 2018 年 6 月申请并被 Lambda 学校录取，我开始与 2018 年 11 月的队列一起学习。

现在，经过 10 周的全职学习和实践，我惊讶于我已经能够真正塞进我的大脑了！

## [](#the-first-five-weeks-deep-dive-into-fundamentals)前五周:深入研究基本面

第 1-4 周都是关于 web 开发的基础知识:语义 HTML、普通的和预处理的 CSS、响应式设计、Javascript 的基本概念(包括闭包、回调函数、像`.map()`和`.reduce()`这样的高阶函数、原型、组件和 DOM 操作)，以及版本控制。到第四周结束时，我已经完成了将近 20 个小项目。

在 Lambda School，每隔五周就是构建周，在这一周，我们会花四天时间与一个由来自不同群体的学生组成的小组一起构建一个更强大的项目。刚刚完成前四周课程的学生将担任构建周项目的 UI 开发人员。

我为一款名为 Stylogue 的产品设计了一个完全响应的登陆页面，这是一款使用类似 Instagram 的图片馈送将角色扮演者和造型师联系起来的网络应用。我还设计了网页的布局和视觉效果，因为我们没有一个 UX 设计的学生和我们一起做这个项目。你可以在这里查看我的作品: [Stylogue](http://carlosv.me/hair-care-landing-page/index.html)

总的来说，我对前五周教授的内容感到满意。Lambda 的预科材料提供了一个很好的基础，我所有的自学主题都集中在这些基础上。

如果你很快就要开始学习 Lambda，我强烈建议你在课程开始前尽可能多的学习。如果你不这样做，你会没事的，但是这会给你更多的时间去理解这些概念，让你走在潮流的前面。

## [](#lambdas-winter-hackathon)拉姆达的冬季黑客马拉松

Lambda School 每年两次在半年的课间休息时间举办 48 小时黑客马拉松。按照 Lambda 的结构，任何学生在那里的时候都应该能够体验至少一次黑客马拉松。

我在 2019 年的第一周参加了冬季黑客马拉松，并作为一名 UI 开发人员加入了一个金融项目。我和我的团队最终创建了 [Compound](https://compound.careers/) ，这是一个 web 应用程序，它可以可视化你当前工作的预期未来收入以及在 Lambda 学校毕业后的潜在未来收入(或任何加薪)。

## 接下来的五周:做出反应，重新开始

在为期 5 周的单元的前几天中，我们讲述了 React 基础知识。我们还讨论了 React 组件生命周期，利用了高阶组件、CSS-in-JS、React 路由器、使用 axios 的 CRUD 操作、中间件和使用 Redux 的状态管理。

五周前，如果我看到上面的 React 相关主题列表，我不可能告诉你它们是什么。现在我知道如何利用它们来创建一个有条件呈现的单页面应用程序。

在 React Build Week 期间，我作为前端架构师参与的项目是一个直接连接服务人员和顾客的数字小费应用程序。我还没有准备好分享它，因为我想添加一些额外的功能，并进一步完善它，但构建它是一次有趣、有教育意义、令人沮丧和谦卑的经历。

## [](#my-thoughts)我的想法

Lambda 引入和教授的概念的广度和深度，以及我用这些新知识解决的其他编程问题，都让我感到惊讶。

我很感激 Lambda School 没有手把手地教我:我不得不自己或在同学的帮助下解决许多编码问题，但如果需要的话，会有很多人站在旁边帮我指引正确的方向(项目经理、部门领导、讲师)。学习如何学习和解决问题是一次很棒的经历。

那么，仅仅过了十周，我是前端开发专家了吗？肯定不是。如果说我的 React 构建周教会了我什么的话，那就是:我已经学会了很多关于 React 的东西，并且可以用它来构建东西，但是还有很多我不知道的。Lambda 为所有涉及的主题提供了一个坚实的基础，使我能够被雇用，我将继续通过每个新项目自学新的、先进的概念。

## [](#whats-next)接下来是什么？

今天早些时候，我开始了该计划的后端工程部分。除了在 React Build 周和一个后端学生一起编程之外，所有的材料对我来说都是全新的。

第 20 周后，下次我写总结时，我将完成后端部分和 Lambda 实验室，在那里我将花 5 周时间与我的同学团队一起构建我的顶点项目。我很兴奋地看到这将把我带向何方，以及在接下来的 10 周里我将获得多少知识和经验。
