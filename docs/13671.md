# 将第三方 SDK 添加到您的应用中？

> 原文：<https://dev.to/assender/adding-3rd-party-sdk-to-your-apps-l1f>

我知道，现在已经有很多关于在你的应用中加入第三方软件开发工具包的说法，因为谷歌最近开始禁止很多应用或开发者。但是 tbh，我见过的大多数情况，都是开发商的错。

据我所知，如果你的应用因为有第三方软件开发工具包而被删除，基本上是因为你没有在 TOS 中恰当地提到它。你必须更改你的服务条款，提供你添加的信息，以及你正在收集的信息，然后你可以重新加载应用程序，而不用担心被禁止。大多数开发者没有这样做，只是重新加载应用程序，没有清理任何东西，这导致了商店的禁令。

TL；博士，如果你把所有的事情都讲清楚，一切都“按章办事”，你认为这样做还有利可图吗？
