# [](#what-is-radix)什么是基数

Radix 是一个新平台，就像比特币或以太坊一样，但可扩展且易于构建。我们没有使用区块链，而是从零开始进行全新的设计，这样世界上的每个人和设备都可以使用 Radix，而不会出现集中化或妥协。我们创建了构建模块，使应用程序、令牌和硬币尽可能易于部署。它很快，很有效，我们的测试网已经上线了。

# [](#how-is-radix-different)基数怎么不同？

Radix 是区块链和 DAGs 的快速替代品。它使用逻辑时间的推移和数据库分片来创建一个非常安全和可伸缩的系统，用于数据的共享存储和访问。这是 6 年来对许多尚未解决的计算机科学问题进行研究和开发的结果。它是冗余复制的，但没有大量电力使用效率低下的缺点，或区块链无法扩展到数百万个同时使用的用户，没有重大的妥协或集中。

# [](#how-does-it-work)它是如何工作的？

Radix 基于 Leslie Lamport 的逻辑时钟理论，使用数据库分片理论来线性扩展和向量时钟来生成事件的部分排序，作为主要的共识机制。
