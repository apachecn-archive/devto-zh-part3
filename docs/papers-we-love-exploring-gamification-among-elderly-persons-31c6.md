# 我们喜爱的论文:探索老年人的游戏化

> 原文：<https://dev.to/thejessleigh/papers-we-love-exploring-gamification-among-elderly-persons-31c6>

欢迎回到我们热爱的报纸！很抱歉上周没来——假期让我有点扫兴，我落后了。但是我选了一篇关于年老体弱的病人之间的游戏化的论文，我认为这将会引发一场很好的讨论。

我有点自私地选择了这篇论文。在我成长的过程中，我父亲在老年诊所工作。因此，我花了很多时间和他一起去养老院，和病人交流。我对老年人护理、医疗保健和技术的交叉领域非常感兴趣。就像这张纸是为我量身定做的一样。

我认为这篇论文在概述可能的收益以及技术、文化和可访问性挑战方面做得很好。然而，我有点惊讶和沮丧的是，这篇论文没有触及这种数据收集可能产生的伦理和隐私后果。当然，你也许能够观察到下降，并及早进行医学干预。但是这些信息是如何被收集和存储的呢？谁有权限，患者是否自愿参与？这将与电子医疗记录整合吗？在游戏化活动中的表现会影响保险费率或治疗方案吗？

我非常警惕那些认为老年患者比其他人享有更少权利的政策和技术。这是一个陷阱，它可以剥夺患者的尊严，同时剥夺他们的行动能力，感觉和自主生活的能力，我认为在将技术与老年人护理相结合时，必须记住这些问题。
