# Codementor 上的自由职业如何让我成为一名更好的开发人员

> 原文:[https://dev . to/perkinsjr/how-freelancing-on-code mentor-make-me-a-better-developer-32ep](https://dev.to/perkinsjr/how-freelancing-on-codementor-made-me-a-better-developer--32ep)

我几乎整个职业生涯都在做两件事情中的一件，要么为大公司工作，要么在家提高自己的技能。去年年底，我决定要指导其他学生或开发人员。

起初，我想制作一个网络系列，并上传到 YouTube 或 udemy，但我决定放弃，因为不能保证我会有足够的业余时间来编辑和开发等。

后来出现了 Codementor，一个我可以用最少的启动和最大的影响教学生或自由职业者的地方。在跳入教学之前，我从简单的自由职业工作开始(在 WordPress 上添加一个地图，为什么我的 Java 应用程序不像我想要的那样)。我让自己在美国东部时间每天下午 5:30-9:00 有时间做任何涉及我技能的工作。请求一个接一个地涌来，所以我戴上耳机开始工作。

起初，我收到了一些很棒的问题，比如“我有这个 Java 用户输入控制台应用程序，但循环似乎不工作”，我打开了他们的屏幕共享，看到了一个简单的解决方案。我详细解释了一下，然后他们高兴地离开了。

然后更难的问题来了，比如解释红黑树是如何工作的，以及如何用一个足够简单的例子来实现一个红黑树，这个例子是一个大学生花了 15 个小时看着同一个问题，却一无所知。虽然我明白 RB 树是如何工作的，但我必须努力用最简单的形式来解释它，这样才能更好地理解这个算法是如何工作的。

我在 Codementor 兼职工作了 6 个月，现在对开发人员到非开发人员的翻译有了更好的理解，这是我多年来专业编写代码的新视角。这 6 个月的旅程让我成为了一名更好的开发人员。
