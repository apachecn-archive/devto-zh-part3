# 自签名证书有多差？

> 原文:[https://dev.to/k4ml/how-bad-is-self-signed-cert-222d](https://dev.to/k4ml/how-bad-is-self-signed-cert-222d)

当涉及到验证证书时，你有多糟糕？人们使用来自 CA 的证书，因为浏览器已经信任用于签署证书的根证书。因此浏览器可以进行验证。使用自签名证书，您必须自己进行验证。如果只有你一个人访问这个网站，也没那么难。您有证书，所以您可以将它添加到您的浏览器中，使其可信。

但是如果 Joe、Anna 和 Foo 也需要访问站点，您必须“安全地”将证书交给他们。可能还是没什么问题，因为你可以亲自去见他们。但是想象一下，如果有 100 个或 1000 个以上的人需要访问你的网站，而你甚至不认识他们。现在您开始看到使用自签名证书的问题。
