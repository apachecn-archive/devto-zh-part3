# 为什么我在这里创建一个帐户，为什么我这样做

> 原文:[https://dev . to/sven/why-I-diving-with-creating a account-here-and-why-I-did-it-5ah](https://dev.to/sven/why-i-struggled-with-creating-an-account-here-and-why-i-did-it-5ah)

今天，我将告诉你我在这里创建一个帐户的故事。你会问，什么样的挣扎？良好的...让我告诉你。

# [](#the-technical-struggle)技术斗争

## [](#chapter-1-commuting)第一章:通勤

当我发现一个朋友在这个网站上发了一些很酷的帖子时，我想；哇！我要喜欢那些帖子来支持他们！”。

当时我正在用移动设备上下班。于是我在移动设备上打开“dev.to ”,按下“通过 GitHub 登录”按钮。

和..！

什么都没发生。我无法通过 GitHub 登录。

顺便说一句,“通过 Twitter 登录”按钮确实有效。但当时我不知道你可以用这两种方法登录。因为我不经常使用 Twitter，所以我宁愿用 GitHub 登录。

“一定是因为我的手机浏览器”，我想，“我们回家再试试吧”。

## [](#chapter-2-at-home)第二章:在家

所以在家里，我又试了一次。使用多种浏览器。没有任何成功。

我迷路了。我不确定该做什么。
我只是想喜欢一些帖子。❤

没有任何动力，我上床睡觉了。

[![Sunrise](../Images/3a50a62d73d4fff4de3316afa13eb56b.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--cEGW4DsZ--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://unsplash.com/photos/xg8z_KhSorQ/download%3Fforce%3Dtrue)

第二天，我仍然有点沮丧，我不能创建一个帐户。但是我又下定决心了。我会喜欢那些帖子的！

所以我用 Twitter 登录。它成功了！

我很开心。另一方面，我仍然对 GitHub 登录按钮不起作用感到沮丧。*一点也不。*

在喜欢了那些我想喜欢的帖子后，我很满意，于是我又去了那个网站。

这是我看到它的时候。
我看到了**解决方案**。

## [](#chapter-3-the-solution-kind-of)第三章:解决方案(种)

在设置界面，我看到你也可以链接你的 GitHub 账户。和...这个按钮真的有用！

我很开心。这样，我可以使用两种方法登录！

直到今天，我仍然想知道为什么“使用 GitHub 登录”按钮在注册时坏了。只有我一个人面临这个问题吗？

# [](#the-personal-struggle)个人奋斗

我个人也为创建一个账户而纠结。

## [](#committing)提交

不，不是那种。我害怕如果我在这里写了一个账户，我将不得不一直写评论和写作。这可能非常可怕！这也不容易，因为英语是我的第二语言。再说一遍，练习的最好方法就是走出去，获得反馈。所以，我做了。现在我们在这里。

# [](#but-i-made-it)但我做到了

最后，我确实创建了一个个人资料！如果一切顺利，这将是我的第一篇文章。我希望我能与你们分享我的一些知识，或者至少用我的帖子娱乐你们。😃

## [](#being-part-of-a-community)成为团体的一部分

我费尽周折创建一个账户的一个重要原因是，有时我觉得自己不像是“开发者社区”的一员。这当然不是真的，但也许如果我更多地参与社区，我会感觉不那么陌生。做一个潜伏者很好，但我希望这能帮助我接地。

## [](#a-friend-inspired-me)一个朋友启发了我

我的一个朋友在这里也有账户。我看到他们发了一些很棒的文章！我想知道我是否也能分享一些我的经历。我意识到，找到答案的唯一方法就是一头扎进去，亲自尝试。

# [](#my-story)我的故事

这是我目前为止的故事。你呢？我很想知道你报名的原因。你也有过挣扎吗？还是觉得开始很容易？让我知道！
