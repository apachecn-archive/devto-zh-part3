# 确保您的开发业务符合规范

> 原文:[https://dev . to/official silb/确保您的开发业务符合规范 4f0n](https://dev.to/officialsilb/ensuring-your-dev-business-is-up-to-code-4f0n)

随着开发行业的快速发展，我们中的许多人选择冒险开始自己的事业。虽然许多专业人士知道如何建立联系，以有机的方式发展他们的新企业，但他们中的许多人在经营办公场所的法律要求方面没有经验。

当你经营自己的企业时，无论是大企业还是小企业，你都需要考虑很多方面，以保持企业的运营，并确保所有政策都符合法律。这些领域中的一个领域是健康和安全，如果你雇用员工或与公众打交道，那么作为企业所有者，你要对你的团队的健康和安全负责。

创建健康和安全程序并不像你想象的那样漫长或复杂。对于许多小公司来说，所需要的只是运行一些基本的检查和任务，以确保符合健康和安全法。以下是你需要知道的。

首先，进行一次[风险评估](http://www.hse.gov.uk/simple-health-safety/manage.htm)来评估你在工作场所必须观察和管理的风险水平。简而言之，风险评估记录了所有可能被视为危险的潜在区域，以及针对每个记录的实例的适当步骤和预防措施。虽然法律并没有要求你消除这些风险，但是有一个适当的检查来帮助控制这些风险。它只是需要一个日志，记录可能发生的事情，以及你将如何去预防它。

最佳做法是在你的办公室周围走一走，寻找任何风险，并[将它们记录为高、中、低](https://www.phsc.co.uk/health-and-safety-in-the-workplace/)。如果你有五个以上的员工，那么你需要写下你的发现和注意事项。将此保存在一个文件中，以便您的员工可以方便地访问，并定期与他们交流。即使在用于开发工作的办公室环境中，这仍然是至关重要的。

那么什么样的[被认为是风险](https://www.safetyandhealthmagazine.com/articles/14054-common-workplace-safety-hazards)？通常情况下，您需要针对电气安全、消防安全、人工搬运、滑倒和绊倒以及高空作业的健康和安全措施。每天的风险都是假设的，你不需要记录这些，因为尽管你运气很好，但你永远无法预测或控制随机事故。如果你的工作场所被客户等非员工拜访，这一点尤为重要。

你还需要考虑你的[员工的需求和要求](https://startups.co.uk/health-safety-guide/)。你的健康和安全政策需要考虑残疾人。你需要确保你的消防安全控制考虑到轮椅使用者，并且你可以自由进出你的工作场所。此外，你还需要考虑到 18 岁以下的员工和[孕妇的额外风险评估](https://www.maternityaction.org.uk/advice-2/mums-dads-scenarios/pregnant/health-and-safety-during-pregnancy-and-on-return-to-work/)。

为了帮助你实施必要的健康和安全程序，看看一些[有用的在线指南](https://www.axa.co.uk/business-insurance/business-guardian-angel/quick-small-business-health-and-safety-check/)，帮助你的企业遵守法律。他们建议这样的事情:员工培训和风险沟通以及他们在自身和访客安全中的作用。你应该保留一份培训记录，以确保新员工和现有员工了解所有的安全措施。还建议您在相关的地方展示这些预防措施，以进一步提高健康和安全意识。即使在办公桌前工作也需要健康和安全措施。

通过实施这些程序，您可以确保您的企业得到保护，并继续发展。
