# 用于 web 和软件开发的 Linux 发行版

> 原文:[https://dev . to/vignes h 0025/Linux-distribution-for-web-and-software-development-1fo 7](https://dev.to/vignesh0025/linux-distribution-for-web-and-software-development--1fo7)

最适合 web 和软件开发的 linux 发行版是什么？

### [](#ubuntu)Ubuntu

开箱即用，美观明快。但是我觉得它的 gnome 3 桌面界面超载了，而且由于那些不必要的动画，速度非常慢

### [](#kububtu)Kububtu

KDE 确实感觉非常慢。它提供了很多定制，但太多的定制总是会破坏一些东西

### [](#debain)德巴因

目前为止我用过的最好的发行版。但是他们回购的软件都是旧的。我总是要修修补补加一些 PPT 来获取新的东西

### [](#mint)薄荷

完美的 Windows 拷贝。很好用。感觉不错。充满了不必要的废话。我觉得没人用他们给的那么多程序。

### [](#arch)拱门

如果你真的在学习 Linux 系统管理，那就太好了。开始时，Aur 看起来很棒，但这为 bug 让路。几天后，系统出现问题。

伙计们，有没有其他发行版在外观、性能、稳定但又不会太过时的软件 repo 上平衡得很好？如果我分享的某个发行版有错误，请纠正我。
