# dbForge Studio for PostgreSQL 2.1 获得了一个新的查询分析器

> 原文：<https://dev.to/devartsoftware/dbforge-studio-for-postgresql-2-1-gained-a-new-query-profiler-54m4>

Devart 是面向开发人员和 DBA 的专业数据库管理软件的公认供应商，它发布了 dbForge Studio for PostgreSQL v2.1。更新后的版本具有新的连接选项、用于优化查询的全新功能、重新设计的执行大型脚本向导，以及数据库浏览器和代码完成改进。

**新版本包括以下增强功能:**

**Greenplum DB 支持** — dbForge Studio for PostgreSQL 拓宽了其数据库连接性。从现在开始，开源的大规模并行数据库管理系统 Greenplum 正式列入受支持的数据库列表，这大大扩展了应用程序的功能。

**新的查询分析器** —新的查询分析器功能有助于定位瓶颈并直观地优化缓慢的 SQL 查询。分析结果整齐地显示在几个选项卡中，提供了以下分析统计信息和数据:查询执行的计划树和图表、服务器上发生的顶级操作列表以及 XML 格式的执行计划。

**数据库浏览器改进** —更新后，数据库浏览器可以显示域、视图的规则&触发器、物化视图的索引等数据库对象和节点。

**代码完成改进** — SQL 编码通过这些有用的代码完成特性变得更加方便:
在刷新物化视图中提示现有物化视图的名称；
SELECT 查询的‘FROM’列表中数据库对象的上下文提示；
对真空语句的语法支持；
提示功能关键字；
在选择查询中提示关键字。

**重新设计执行大型脚本向导** —向导自动选择连接设置中默认指定的连接和数据库。也可以刷新数据库列表。

dbForge Studio for PostgreSQL 是一个用于数据库开发和管理的 GUI 工具。PostgreSQL 的这个 IDE 允许用户在一个方便且用户友好的界面中创建、开发和执行查询，编辑和调整代码以满足特定需求。
