### 新年快乐！！

今年我的决心不同于往年，今年我有编码和编程的决心。这对我来说很重要，因为我觉得我没有投入我应该投入的时间，所以这是一个很好的时间让我重新考虑和重新组织我的日常活动，特别是与编程有关的活动。

**1。不要偷懒**

在我空闲(不编码)的时候多编码。

**2。更加熟悉新程序**

留意并练习新的编程语言、框架或库——走出舒适区总是有帮助的。

**3。耐心…**

当编码没有按预期呈现时，对自己有耐心，一步一步地、毫不犹豫地查看不同的变量。并且，对其他需要你帮助的人要有耐心，因为当我的代码没有按预期工作时，我有幸得到了比用户更多的关注。

**4。注意糟糕的代码实践并避免它们**

这是一个解决方案，也是将我从无效代码中拯救出来的最佳实践。

**5。网络、导师和助手**

对扩大你的人际网络很重要，你永远不知道什么时候你会需要你所在领域的人的帮助，指导和帮助他人，因为这不仅仅是支持他人的一种方式，而且会让你的技能保持更新和最新。

这是我的 2019 年编码分辨率，因为我有保持分辨率的糟糕历史，我将从五个开始。

我能做到。

你的决心是什么？新年快乐
