GethDesk—是一个用 JavaScript 编写的开源应用程序，用于简化以太坊 Geth Go 控制台客户端的工作，并在不使用控制台命令的情况下向您显示有关 Geth Go 节点当前状况的数据。

Github 项目页面

任何反馈都将不胜感激。如果觉得有用就在 Github 页面上加个星。

它解决了什么问题？

当我开始使用以太坊 Geth Go 时，收集关于我的节点状态的数据非常不方便，所以我决定编写一个应用程序，通过运行 Geth Go node 来简化使用和数据可视化。
