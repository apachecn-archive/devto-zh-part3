我第一次接触编程是出于偶然。我的叔叔和我差不多大，在我们年轻的时候，他已经尝试过编程，在他父亲的 Tandy-1000 上写简单的游戏(今天是一个过时的硬件，但在爷爷得到它的时候，那东西对我们来说是令人兴奋的……就像是一个发现了未来入口的穴居人)。我对他印象深刻，但当时还不足以让我自己接受……我更感兴趣的是磨练我的艺术技巧绘画。直到几年后网络成为消费者熟悉的东西，我才真正开始注意到它。我搬去和叔叔一起住，并开始涉猎 Macromedia Flash，因为它面向视觉的工具对我很有意义。然而，我很少接触任何支持代码，而是关注事物的图形端。这种调情被我加入海军陆战队打断了，直到我入伍一年后买了我的第一台电脑(一台便宜的电子机器)，我才开始认真努力学习有用的东西。然而，那时候互联网还处于青春期，谷歌刚刚起步，我花了很多时间通过反复试验来学习。

我的第一堂真正的课是当我决定学习一些关于我一直听说的“Linux”的东西。我对 UNIX 比较熟悉，知道很多重要的系统都运行它，所以我想学习一个类似的操作系统可能会有用。我从我们基地的 PX 购买了一本关于 Linux 的书和一份 RedHat 的硬拷贝，然后开始在我的机器上安装操作系统……却惊恐地发现，我的电脑自带的 56kbps 拨号 winmodem 不支持 Linux。我失去了上网的能力，在不到一个小时的时间里，我的电脑就失去了作用。我花了一晚上的时间重新格式化我的硬盘并重新安装 Windows ME(是的，那是很久以前的事了)。这是第一次打破东西，然后我自己学习如何修理它，这将成为我今后几年学习的正常模式。

<figure>[![](img/d20a10c01b9d3129f11822fc7f802582.png)右上方那个大耳朵的傻瓜？那就是我。](https://res.cloudinary.com/practicaldev/image/fetch/s--0PPU0fly--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://cdn-images-1.medium.com/max/1024/1%2AH_5wA4z9hwRjI9HltzrOYQ.png)</figure>

几年后，我在当地的社区大学学习了一些编程课程，并结婚了。面对即将从 EAS 军团退伍，我从一家临时机构找了一份工作，在基地做建筑工作，承诺完成后会有更多的工作……一个没有实现的承诺。然而，临时代理公司有一个技术总监的职位空缺，他们看到我有一些计算机经验，所以给了我这个职位。我当时不知道的是,“技术总监”实际上是指“员工中唯一懂电脑的人，他需要安装电脑并保持一切正常运行，还要为公司网站设计和编写所有内容”……遗憾的是，我对这个职位毫无准备。虽然我尽了最大努力，但不出所料，大约六个月后，我被解雇了。尽管我失败了，但我决心在这个行业取得成功……我的下一份工作来自我的一位老同事的推荐信……他有一个朋友开了自己的科技公司，需要一些帮助。事实证明，他的公司负责网络布线。这并不完全是我想要的，但它仍然涉及大量的计算机工作，所以我花了一年的大部分时间在仓库和办公楼里运行网络电缆。业余时间做公司网站，尽我所能磨练 HTML 和 CSS 的技能。我原以为这样做很好……后来发现我的雇主把公司所有的资金(包括我的工资)都花在了他的新女友身上。一天早上我发现了这一点，在存入我的薪水支票并支付了当月的账单后，我醒来时发现有几千美元的负结余，因为我的雇主在发给我薪水支票后取消了它，然后逃离了城镇。沮丧之余，第二年的大部分时间里，我都在与前老板进行旷日持久的法律诉讼。我获得了 12000 美元的和解金，但实际上我一毛钱都没见过。在接下来的几年里，我也没有从事技术工作。然而，我在当地的必胜客找到了一份助理经理的工作。在那里，我与两位思科认证的工程师一起工作，他们也很难找到工作，因为“网络泡沫”刚刚破裂，南加州现在有大量技能相似的技术工人，他们现在发现自己处于同样的位置……失业，并想知道他们为什么要花这么多钱进行培训和认证。我想，如果这些比我更有资格的人，在他们选择的领域找工作都这么困难，那我就没什么希望了。我之前提到的那个叔叔救了我，他建议我回到德克萨斯州的家，并向他现在工作的公司提出申请。

跳过一年左右，我现在在休斯顿的那家公司工作，同样，我的大部分职责包括拉电缆(以及为老板的家人提供 IT 支持和计算机维修，唉)，但我有更多的时间来磨练我的 web 开发技能，因为我还负责管理公司的各个网站。这一切都是为了 10 美元/小时的低工资。我再次发现自己是公司唯一的开发人员，我在互联网上的大部分学习也是如此。起初，我通过几所在线大学做到了这一点——这两所大学都重组了他们的《退伍军人权利法案》( GI Bill)项目，以便让那些试图在上学期间支持一个六口之家的人完全无法实施这些项目，迫使我最终辍学。在那之后，我依赖于各种留言板、教程网站、书籍等等，拼命地消耗任何和所有我能消耗的知识，努力跟上不断流入的新信息。我去了几个不同的方向，但最终学会了足够的 C#来编写一些门房软件，以控制我们服务的一个高端公寓的自动化系统。然而，此时我还没有学到任何关于版本控制系统的东西……这并不重要，因为最后一刻对基于 Linux 的平板电脑平台的硬件更改需要我用 PHP 从头开始重写整个软件。就在这个时候，我得到了一份编写 Flash 应用程序的工作，薪水是我当时职位的两倍。

在这家新公司，我发现自己处于一个奇怪的位置，在一个主要基于微软的开发店里，我是一个孤独的 Flash(后来是孤独的 OS)开发者。虽然我也在这家公司做了很多设计工作，但我的职责再次包括支持公司依赖的各种网站，以及做大量的应用程序架构和 UX 工程……基本上，如果公司提供了一份不需要认证 C#或 SQL-Server 工程师的工作，这份工作就落到了我的肩上。我继续尽我所能地学习，从所有我能得到的资源中学习，但是仍然很漫无目的……我开始被这样的想法所困扰，尽管我得到了所有的经验，我将永远处于一个我什么都不懂，什么都不懂的境地。我决定深入 Ruby 开发，因为 Ruby 的语法一直吸引着我，而且它的包生态系统相当成熟。我得到了一份工作，为一家支持石油行业的物流公司开发 Ruby 应用程序，这也让我的薪水翻倍。哇，这真是太棒了，我又一次想……事实也的确如此，直到不久后石油行业的底部掉了下来，项目被取消了。

我又一次失业了，但这并没有持续多久。在过去的十年里，我积累了足够的经验，最终被我现在所在的公司认真考虑了一个很棒的职位。我现在在家工作，整天写 Ruby、SQL 和 Javascript 代码，而且收入颇丰。

我刚刚分享的故事，虽然看起来很长，但实际上是简短的版本。我把这些都告诉了你，所以我可以告诉你……我走上今天的这条漫长、崎岖、曲折的道路，在很大程度上是可以避免的，只要有人能指引我走上正确的道路。一个导师。一个已经确立了自己地位的人，知道我需要付出什么才能达到他们的位置。当我第一次发现编写代码的兽医时，我立刻对他们的项目印象深刻，并希望以任何可能的方式做出贡献。这里有一个社区，通过给刚进入这个行业的老手分配已建立的导师，能够在几个月内传授价值数年的正确知识，并允许初出茅庐的开发人员实现我自己花了十多年才完成的事情。对于一个没有导师的人来说，有导师的区别是显而易见的。在成长过程中，没有人提醒我可疑的雇主或定义不清的职位，甚至没有人告诉我一个开发人员的合理工资是多少。找工作时，没有人给我指出哪种技术最适合我，或者我在使用这种技术时需要的最佳实践和工具。即使我在接近完成计算机科学学位的过程中学到的东西也没有让我为商业的现实做好准备，也没有给我任何方向感。我有基本的知识，但没有重点。VWC 项目为我们的军队提供了我所缺少的一切(除了我们期望他们投入的工作之外，他们不需要任何费用)…使用行业标准工具和技术的真实体验，以及将这些技能转化为高薪职位的指导。虽然大多数付费的编码训练营会给你知识，但很少有人会给你社区和指导。对我来说，这就是我们的项目真正提供价值的地方。如果这个项目在我刚开始我的旅程时就存在，我毫不怀疑我的职业道路会完全不同。我看到过这个项目的毕业生在刚开始的时候获得了我梦寐以求的演出，如果我完全诚实的话，现在我有点羡慕他们。作为这个项目的导师，我为我们所有的士兵和他们持续的成功感到骄傲，并强烈鼓励任何有兴趣成为开发者的兽医尝试一下这个项目。如果你正在阅读这篇文章，并且已经在这个行业立足，我邀请你自愿成为一名导师……帮助老兵们更容易地走向成功；他们面前的道路已经够艰难的了。
