# 创建新的分区

请忽略**如何切换分区**部分

# 在新分区上安装最新的“官方”macOS

重启 Mac，启动 Mac 后立即按 Command+R。然后你可以看到“重新安装 macOS”选项。请确保选择新的分区进行安装。

# 安装 macOS 卡特琳娜测试版

1.  从 [Beta 软件下载-苹果开发者](https://developer.apple.com/download/)获取简介
2.  安装下载的 pkg
3.  然后我们可以优先获得更新，继续安装
4.  🎉
