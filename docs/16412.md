长期以来，我一直听人们滔滔不绝地谈论 Vim。我一直很尊重他们的意见，但最后，我还是拒绝给 Vim 一个尝试。

我和 Vim 的经历太痛苦了。想象它如何提高我的生产力是超现实的。

听了另一个 Vim 爱好者的话，我终于撸起袖子，决定试试 Vim。💪猜猜看，这是一个非常好的决定。

不要误解我。

我不是一个知道所有命令并且安装了很多插件的 Vim 铁杆用户。我也不想用普通的 Vim 编码。

但是结合您选择的 IDE，Vim 是一个完美的工具！
