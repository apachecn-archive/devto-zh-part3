# 你应该建立个人网站吗？

> 原文:[https://dev.to/exbe/should-you-build-personal-website-2n80](https://dev.to/exbe/should-you-build-personal-website-2n80)

在过去的 1000 年里，我一直在敲打个人网站的事情。发展还是不发展——这是我的问题。

社交网络几乎涵盖了每个人的需求:专业的、个人的、纯图像的、纯声音的。我相信你现在可以找到任何网络。

个人网站需要各种各样的技巧来使它感觉良好。设计、托管或云、编程、搜索优化、辅助功能、结构化和链接内容。你知道的越多，就越觉得这是一个预算很重、耗时的项目。哦，我还错过了数据收集、条款和条件等法律方面的东西...你说吧。

你到底要告诉人们什么？最常见的个人页面是作品集，可能是博客或滑稽表演。无论如何，这是与品牌建设有关的事情。但是有没有高于和超越的东西呢？

另一方面。

个人网站是你控制的东西。这就像你手中握有一小部分互联网。你的数字领域，没有广告，没有谎言。

拔出来就能成为珍贵的宝石。

想法？

请随意分享你的个人网站，我很乐意看到它。

TL；我想这是我年轻时的梦想，那时科技还没那么先进。不幸的是，对我来说，技术超过了我的梦想。

然而，我再次站在停止域名更新的边缘，让它自然消亡。
