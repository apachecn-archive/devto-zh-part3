# 通过网络同步程序启动

> 原文：<https://dev.to/computersmiths/synchronize-program-start-over-network-4l1b>

我正在尝试使用 Python 中的网络连接让两台计算机同时启动一个程序(播放视频)。

我开始在一台机器上打开一个套接字服务器，在另一台机器上连接到它，并发送某种信息说“现在！”，这是可行的，但似乎不必要的复杂。

我错过了什么简单的事情吗？理想情况下，我会有一个名为“Block.py”的程序，我会从一台机器上的 bash 脚本启动它，它会一直等到我在另一台机器上运行它，然后它们都会继续执行 omxplayer 命令...

两台机器都有静态 IP，如果这有助于简化事情的话。
