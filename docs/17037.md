### 常见问题

还需要答案？看看我今天已经回答过几次了。随着更多人加入，我甚至会更新这个列表。

#### 这是否会影响已经包含 SDK 和/或 VR 语音沙盒的现有项目？

没有。但是如果你想要一个更新的版本，你需要从 GitHub 下载。

#### 资产还会维持吗？

是的。没有计划反对 SDK 或 VR 语音沙盒上的维护。

#### 我可以提交反馈吗？

请随时报告问题或在相应的回购中提交 PR。

#### 哪里可以找到 SDK 的发行说明？

在[回购 wiki](https://github.com/watson-developer-cloud/unity-sdk/wiki) 上！

这是一个很好的问题，最终为什么我认为 SDK 应该只存在于 GitHub 上，以使它更接近发行说明。我们的 IBM 云服务和 SDK 以敏捷的方式进行维护和增强，这意味着变化经常发生。最亲爱的开发人员，让代码更接近发行说明可以让你了解更多信息。

#### 我从未使用过 GitHub，如何将 SDK 拉入我的 Unity 项目中？

您可以下载 SDK 并将其解压缩，然后通过拖放到项目窗口或目录中，将其移动到 Unity 项目文件夹中。

或者，您可以使用命令行 an `git clone`到您的项目目录，但是您需要安装 git。
