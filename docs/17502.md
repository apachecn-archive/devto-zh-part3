# 来自 Ashesi 大学的黑客马拉松的关于 Cloudinary API 的宝贵一课

> 原文:[https://dev . to/whizzy _ ellah/a-valued-lesson-on-the-cloud inary-API-from-a-hackathon-at-ashesi-university-528 a](https://dev.to/whizzy_ellah/a-valuable-lesson-on-the-cloudinary-api-from-a-hackathon-at-ashesi-university--528a)

[Cloudinary API](https://cloudinary.com) ？我第一次听说这个是在 Ashesi 大学的黑客马拉松之前，可能是因为我是编程新手，这并没有让我失去兴趣，但是我很好奇并渴望了解更多。

为了满足黑客马拉松的要求，我必须在 Hackathon.io 网站上注册，创建一个 Cloudinary 帐户并激活我的 Github 帐户。然后我通读了网站上的信息。等一下！我是否在 Cloudinary 上看到了几乎所有正在使用的编程语言？是啊！PHP，Python，。Net、Ruby——事实上，几乎任何你能想到的语言。这足以鼓励我报名参加黑客马拉松的 API 竞赛。“我的团队没有赢，但我想探索想法，肯定会学到很多东西，”我对自己说。我发誓，我曾经。

比赛当天，负责搭建场地和确保 WIFI 正常运行的技术人员将所有东西都按时准备好了，嘭！我们的团队，叫做 Pro。看来，立即开始按照指示进行黑客攻击了。我们的一个明显优势是，我们的许多成员都擅长探索 Cloudinary 平台上的工具。我借此机会了解了 Cloudinary，哇！你不仅可以将图片和视频从浏览器上传到云端，还可以在事后无缝访问它们？凭借这些令人惊叹的功能，您可以随时随地无缝处理您的图像或视频。那确实令人印象深刻。

关于 *Cloudinary API* 我喜欢的另一件事是它从文本创建图像的能力——这对前端开发人员来说是一个巨大的帮助。Cloudinary 还提供了许多其他功能来管理您上传的图像和视频。据我所知，Cloudinary API 是一个理想的地方，在这里你可以积极地管理上传的数字资产——一个虚拟的家，在这里这些资产通过 Cloudinary 的集成和插件不断优化。

考虑到所有的知识，其他团队提出了他们的想法。其中一个团队提议开发一个名为“M.T”的应用程序，其中的视频和教程将对学生有所帮助。还有一个团队主张用微软 Word 将语音捕获为文档，然后可以编辑上传到云端。这难道不是一个强大而有用的工具吗？

另外，我的团队“Pro。view”([https://github.com/Khophi-And-Khode](https://github.com/Khophi-And-Khode))想出了开发一个 web 应用程序的主意，该应用程序用于显示外国和本地制造的食品(包括软饮料和食品补充剂)的图像，以供查看者查看和评论，查看者包括认证的系统权威，包括 FDA 和其他相关权威。在这种情况下，观众也可以通过填写平台上的表格来请求产品评论，该表格要求上传产品图像以供评论。
