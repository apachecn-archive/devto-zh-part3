# 面向 ChromeOS 和 Chromebook 用户的开源项目

> 原文：<https://dev.to/bitsrfr/open-source-project-for-chromeos-and-chromebook-users-3jng>

我做了一个开源(MIT 许可)的截屏/录音 ChromeOS 应用。它可以工作，但是我确信在后端仍然会发现很多错误。还有，前端在这一点上非常难看。

我创建它是因为我找不到任何其他免费的 Chromebooks 屏幕记录器。付费的通常有免费版本，但它在视频上加了水印，我不想这样。

我从中得到了我的用处，所以我想我应该开源给其他人使用或贡献。

我很想看到有人贡献一个更好的用户界面(只是一些 HTML，CSS，JS 的东西)，但真的任何贡献将被超级感谢！

这里是项目存储库:[https://github.com/joekreydt/Captis](https://github.com/joekreydt/Captis)。

如果有任何意见或问题，请在 Dev.to、GitHub 或 Twitter 上联系我。

和平！
