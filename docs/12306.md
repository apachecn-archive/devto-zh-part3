本教程解释了如何用 html、css 和 javascript 自定义创建上下文菜单。上下文菜单是 GUI 中的菜单，在用户交互时出现，例如鼠标右键操作。上下文菜单提供了在操作系统或应用程序的当前状态或上下文中可用的有限选项集。通常可用的选项是与所选对象相关的操作。
