# 你为什么开始写关于 DEV 的文章？去哪？

> 原文：<https://dev.to/solbiatialessandro/why-did-you-start-writing-articles-on-devto-a3h>

我最近加入了 DEV。我认为这是一个伟大的利基社区，我几乎每天都在阅读大量的文章，正如其他人在其他地方提到的“阅读清单”是一个伟大的功能。

我在琢磨要不要开始写一些文章，也不完全确定社区是否还足够成熟。 ***是什么让你决定开始写关于 DEV 的文章？去哪？*T3】**

即使我在一些文章中偶然发现了一些有趣的信息，我仍然没有找到那些真正吸引你的伟大文章。在我看来，现在的社区有点过于以网络开发为中心，或者我不擅长处理我的标签？我可能会在接下来的几周尝试一下。
