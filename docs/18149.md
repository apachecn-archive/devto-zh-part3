# 从终端打开 Linux 中的文件管理器，而不知道它的名字

> 原文:[https://dev . to/programmer rush/open-file-manager-in-Linux-from-terminal-without-known-it-name-4i8o](https://dev.to/programmerrush/open-file-manager-in-linux-from-terminal-without-knowing-its-name-4i8o)

使用命令 **xdg-open** 直接从终端打开文件管理器。

如果你不知道文件管理器的名字，那么运行命令 sudo
ex: **sudo xdg-open。**

直接从终端知道默认文件管理器的名字是一个不错的方法。
