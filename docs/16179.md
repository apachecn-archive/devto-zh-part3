# 我用 Python 创建了我的第一个电报机器人

> 原文:[https://dev . to/amr3k/ive-created-my-first-telegram-bot-with-python-27bn](https://dev.to/amr3k/ive-created-my-first-telegram-bot-with-python-27bn)

你可以直接从电报`@randomovie_bot`或者通过这个直接电报[链接](https://t.me/randomovie_bot)来尝试

这个机器人基本上可以让你设置你喜欢的电影规则，如最低发行年份，最低 IMDB 评级和最喜欢的类型。接下来，机器人会根据你的规则向你推荐一部更详细的电影。
这么简单！！
