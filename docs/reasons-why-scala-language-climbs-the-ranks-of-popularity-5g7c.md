# Scala 语言越来越流行的原因

> 原文：<https://dev.to/apium_hub/reasons-why-scala-language-climbs-the-ranks-of-popularity-5g7c>

Scala 被认为是一种复杂的，但是非常高效和苛刻的语言。在今天的文章中，我将向您展示一些关于 Scala 语言流行的事实，使用它的好处，并列出一些有用的 Scala 博客，在那里您可以经常找到有用的理论和实践示例信息。

## Scala 语言

好吧，首先我们先说清楚，Scala 是什么？Scala 是一种通用编程语言。它被设计得很简洁，许多人说它的引入是为了弥补 Java 的不足。它的源代码旨在被编译成 Java 字节码，以允许结果代码在 Java VM 上运行。

还有，顾名思义，Scala 是 scalable and language 的缩写，它被设计成随需求而增长。此外，它是一种类型安全的 JVM 语言，将面向对象和函数式编程结合到一种极其简洁、逻辑严密且功能强大的语言中。

在软件开发领域，Scala 被认为是一种高端语言，并且有着初级开发人员学习复杂语言的名声。世界上很少有开发者真正知道如何使用它，也很少有公司使用它。它被越来越多的人采用，稍后我们会看到一些关于它的事实，但只是告诉你一些名字，像 LinkedIn、Twitter 和 The Guardian 这样的大公司在他们的代码库中使用它。那些了解 Scala 的人享受着伟大的职业生涯，因为 Scala 开发人员的空缺职位肯定会越来越多。

## Scala 语言日益流行的原因

当今技术的核心是编程语言。今天，开发人员对进步的重要性是前所未有的。据 Jaxenter 称，自 2017 年 5 月以来，Scala 在排行榜上跃升了 11 位，排名第 20。[事实上](https://stackify.com/popular-programming-languages-2018/)显示，Scala 在需求最多的语言中排名第 12 位。

为什么？Scala 是专门为了成为更好的语言而创建的，摆脱了 Java 中它认为对开发人员来说限制性的、过于乏味的或令人沮丧的方面。因此，Scala 编程有点困难，但结果是一种更干净、组织更好的语言，最终更易于使用，并提高了生产率。

此外，在 Scala 中，编译器非常聪明，所以这避免了开发人员需要显式指定编译器可以推断的内容。Scala 非常适合某些问题领域。它在定义抽象方面非常灵活。

当需要在一个复杂的领域中导航时，Scala 绝对是最棒的。仅举几个例子，分布式和并发编程都可以。此外，并行性是非常棘手的，Scala 提供了大量的库，通过构建抽象来简化并行性。

另一个巨大的好处是不变性。Scala 使得使用不可变数据编写代码变得容易。这包括诸如一级 val、case 类、高阶函数等结构。但这也是因为标准库是如何设计和编写的；所有的“默认”数据结构都是不可变的。不变性使许多事情变得更简单。

更重要的是，Scala 对类型构造函数、更高级类型和类型类的支持，通过隐含，使得处理类似包装器/容器的类型更加容易，比如承诺、未来或任务。
