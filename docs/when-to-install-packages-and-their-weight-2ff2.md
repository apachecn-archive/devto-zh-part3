# 何时安装软件包及其重量？

> 原文：<https://dev.to/bartude/when-to-install-packages-and-their-weight-2ff2>

在 web 开发中，无论是否使用框架，安装包的需求几乎是不可避免的。有一些对每个人来说都是不可或缺的。

我的问题的第一部分是，决定何时安装 X 或 Y 包的因素应该是什么(并且自己开发必要的逻辑)。

我问题的第二部分是，已安装的软件包会影响很多网络应用程序的性能吗？例如，我曾经看到一家名为 Spatie(开发 Laravel 软件包)的公司为客户创建了一个 web 应用程序，他们正在安装的软件包列表非常庞大(我的标准是大约 30 或 40 个)。
