# 克服恐惧

> 原文:[https://dev.to/ilonacodes/overcome-fear-15if](https://dev.to/ilonacodes/overcome-fear-15if)

有哪些事情是你如此害怕，以至于你无法拥抱，也无法克服的？

例如，对我来说，它是把我自己放在那里，在每个人或所有事情面前。那一刻，我产生了最大的社交焦虑。

是的，我正试图确定我的恐惧，然后勇往直前，直面它们，因为我们的恐惧只是我们的极限。所以，我每天都在努力变得更好。

通常，我们有两种选择。要么我们忘记一切，接受恐惧，要么我们面对一切，奋起！选择权只在我们身上。

然而，通过回避任何挑战，我们阻碍了我们的成长。当我们挑战自己时，我们的生活质量会成倍增长。不仅我们的生活质量会提高，而且我们会在这个过程中更多地了解自己。挑战自我会让我们感觉更强大、更聪明、更快乐。

传统上，当涉及到职业，关系或任何特定的恐惧-恐惧症-如恐高症，幽闭症，蜘蛛恐惧症等，它会发生很多。在这种情况下，最好的解决办法是挑战自己，解决让我们如此不开心的情况。

糟糕的工作？换个工作。
没有个人成长？离开你的舒适区。
有毒关系？不要把它作为礼物送给自己。
有恐惧症吗？向医生求助。

最终，当涉及到我们生活的所有领域时，与这些情况相关的消极会限制我们的潜力。

我相信一旦我们解决了这些情况，结果会给我们一个极好的机会去学习和点燃我们的灵魂🔥

你知道你的恐惧吗？挑战他们！
