# 反应本地共享 API 以共享 TextInput 消息

> 原文:[https://dev . to/skp tricks/react-native-share-API-to-share-textinput-message-66a](https://dev.to/skptricks/react-native-share-api-to-share-textinput-message-66a)

Post 链接: [React Native Share API 分享 TextInput 消息](https://www.skptricks.com/2018/12/react-native-share-api-to-share-message-in-android-ios.html)

本教程解释了如何实现 React 原生共享 API，以在 android 或 ios 应用程序中共享 TextInput 消息。React 本地共享组件，用于共享任何内容。我们通常在应用程序中添加一个分享按钮，通过任何社交媒体或任何其他媒体与其他用户分享一些东西。我们可以直接从我们的应用程序向 Whats App contacts、Hike Contacts、Groups、SMS 共享文本消息，复制到剪贴板，共享到 Google drive、Gmail、Hangouts、One Drive 和其他已安装的应用程序。
