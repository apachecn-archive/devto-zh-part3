# 独立制作人还是青少年创业者？

> 原文：<https://dev.to/m1guelpf/indie-maker-or-teenpreneur-5c8i>

[https://app.blogcast.host/embed/296](https://app.blogcast.host/embed/296)

在我最近做了所有这些疯狂的事情之后(比如在不到一个小时内推出了两个 Chrome 扩展)，我觉得我终于找到了一个代表我的“数字空间”:制造商空间。

这与我现有的个人品牌(也就是你投射到互联网上的形象，你的个人“品牌”)相冲突，这是我在今年早些时候推出我的第一个大产品时建立的。那么，有什么区别呢？

对我来说，青少年创业者是一个已经走上企业化道路的青少年，他把自己的产品标榜为大公司，并在常见问题中写下“我们”，而创客则展示自己，不等待事情变得非常完美才推出它们，并在登录页面上使用表情符号。

我想成为什么？嗯，我是一个人，我是一个在互联网上做疯狂事情的青少年。是的，我犯了罪，在山洞里建了几个月，躲在“我们”后面，试图成为一个人公司的首席执行官。但现在不是了！

从今天开始，[我的网站](https://miguelpiedrafita.com)上写着“*我 16 岁，喜欢做东西*”，而不是“*我是一名 16 岁的开发者&青少年企业家*”，我喜欢这样！
