0。为什么我想创建一门课程

嗯，我会说——荣誉、金钱、未来财务可能性评估以及提高我在 IT 市场的竞争力。

此外，我知道黄金法则:“想学更深层次的东西——去教它”。就我而言，我对 RxJS 有很强的了解，但像往常一样，你在日常工作任务中只利用了部分可能性。所以深入学习它也是我的动力点，因为我是 RxJS 的超级粉丝。

此外，我有一个梦想，作为演讲者参加一些 Angular (ng-conf，angular-connect 等)或 JS (ReactiveConf 等)国际活动。录像是一个很好的练习，尤其是如果英语不是你的母语的话。到目前为止，我作为一名[演讲人](https://www.youtube.com/watch?v=d7zO9fzXbfM&t=4s)只有全国[JavaScript fwdays’18](https://fwdays.com/en/event/js-fwdays-2018)参与。

**1。我是如何被雇佣的。**

在我的空闲时间，我在 [codepen.io](http://codepen.io) 上担任а JS/Angular 导师。

来自 packtpub.com 的猎头公司在这个网站上联系了我(通过对每个感兴趣的人的指导请求),并提议创建一个课程大纲。在我完成大纲后，花了大约 1.5 个月的时间来协调议程和编辑委员会的期望。我签了合同(远程，数字)，开始创作和录制视频。

我被指派了一名个人编辑来帮助创建课程。我真的很感谢她的帮助，因为我有很多问题甚至恐惧——但她以非常专业和耐心的方式消除了我的担忧。

嗯，我可以告诉你，我超过了合同中规定的时间限制。但是我被告知要专注于质量，不要担心时间。非常感谢编辑的支持。

**2。创建课程是另一项全职工作。**

如果你已经有了一份全职工作——恭喜你——你又找到了一份工作

课程创作中最大的过程是在我生病呆在家里的时候完成的。我可以分配更多的时间来创作课程，每天或几天上传一次视频。我指的是一天——24 小时，因为你不能像平常工作时间那样集中精力工作 8 小时。有时(可能是上午 8 点或晚上 11 点),你会得到一些好主意，并继续制作幻灯片来表现主题。

一些视频需要长达 2 周的时间才能完成。实际上，产生主要课程的想法是最困难的任务。一旦完成，所有其他信息都可以成为这个基本准则的一个字符串。

您准备了幻灯片、代码示例并编写了演示文稿(实际上是您要讲的内容)——现在录制时间到了。

第一次录像(只是记录活动)花了 7 个小时...一个视频。嗯，不是太激励)帮助我不被未来要做的大量工作压垮的主要方法是忘记其他视频。每个视频都是一个产品，你只需要专注于如何尽你所能创作和录制它。一旦记录下来——万岁！用红叉标记完成，开始另一个)每一个视频结局都变成了一场小小的盛宴。

*筋疲力尽*是一个非常轻量级的意思，用来反映我的状态，有时当主题对你来说不是很熟悉，你必须处理大量的文章、调试示例和库源代码，才能找到一个好的主题解释方式。

当我完全无能为力时，我的经验法则是“小步前进”。这意味着—一次创建一张幻灯片，一次为一张幻灯片编写脚本，为下一个视频创建空的 codepen.io playground，创建空的幻灯片文件，等等。一步一步的过程更进一步，它激发了很多。如果你不能走，你应该爬。

**3。家庭时间**

五个多月来，我每个星期六都在工作。这意味着这些天我的妻子和两个儿子变得离我而去，没有我也能散步，没有我也能开心。如果你有小孩，有一份全职工作，没有经济困难——三思而行……不……三次。

我利用每一分钟空闲时间工作——在工作时间(如果我有空闲时间的话),在孩子们起床前的早晨，以及下班后。

我的建议是——如果你有小孩——最好帮助你的妻子，视频课程可以在其他时间录制。如果我在签合同的时候知道就好了:-)我想对我的妻子说‘谢谢’，她对我很有耐心，直到这个课程结束。爱你，亲爱的:-)。我想把这门课献给我的妻子和孩子:如果没有你，我肯定会做得更快(:-D)，但从那以后也不会有太大收获。

**4。虚荣。**

当然，对我来说，这绝对不是最后的动力点，也不是评估课程创作是否值得投入时间和精力的可能性。当你有足够多的关于某个图书馆的文章时，你只需要把它们收集起来，创建一门课程。对我来说，情况正好相反。我创建了一个课程，它为我提供了一个关于[文章](https://medium.com/@alexanderposhtaruk)的好主意。但是我花了几天，有时甚至几周的私人时间，试图更深入地了解我以前不使用或很少使用的领域(当然，熟悉的主题会更快)。

值得吗？也许吧，但我认为，如果只是你的媒体(文章、演讲、视频教程)活动和丰富工作经验的延伸，创建一门课程会容易得多。我试图加快普通的过程——这是驾驶乐趣，但非常累人。缺少家庭时间是代价。

**5。PacktPub 课程创建支持怎么样？**

在我的活动中，我总能感受到和我一起工作的人的专业精神。我被提供了操作系统和显示设置来配置视频录制，以及用于屏幕和麦克风录制的特殊软件。此外，所有的编辑过程以及出版和营销、质量评估都由 PacktPub 处理。每次录像后，我都会得到一些提高课堂质量的建议。这对我来说是一次非常有价值的经历，因为我以前从未制作过任何视频教程。

**6。是什么让我对开设这门课程如此着迷？**

开始时，您可能会错误地配置环境(大多数情况下—在显示设置中设置了错误的分辨率),您必须重新录制整个课程。唷！

有时你的编辑很忙，几天后就开始审查你的新视频。这意味着在遇到麻烦的情况下——你必须重新录制几个视频，而不仅仅是最后一个——有奖游戏！:-D

另一个问题是麦克风。特别是这门课，推荐我买了“[蓝雪球](https://www.amazon.com/Blue-Snowball-Microphone-Textured-White/dp/B000EOPQ7E)mic。有时我会忘记在录音设置中选择正确的选项。想象一下:你录了一段视频(浓缩的)，然后发现你用了笔记本电脑内置的麦克风。啊啊！！！所以，又一次尝试来了)

[![](../Images/1de765d23e06e7a73503f4e83b5ff9c1.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--YY99DLF7--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://cdn-images-1.medium.com/max/1600/1%2A9PktyiDMUON_EhyfN3SrIA.png)

创建视频后，编辑会进行第一次审核，并要求您重新录制一些片段(回声、发音不清的单词等)。所以做好准备。但这还不是全部。质量团队也可以要求你对课程做一些改进——将课程一分为二，或者添加更多动画来更好地解释主题。通常，当你已经切换到另一个部分时，它会晚一点发生。

还有一点比较有意思——录音环境:应该是一个小房间(赢家:衣柜)，地板上(可能的话墙上)铺着地毯，窗帘紧闭。有一次，我忘了拉上窗帘——录像中出现了回声。但是重录是我的爱好，所以我都没有不爽)))

7 .**。值得吗？**

**加号:**

1.  我对 RxJS 6 的了解比我以前知道的要深得多。我在 GitHub repo 里复习了很多问题，甚至调试了它的源代码，了解它的工作原理。不需要表扬——就叫我‘主人’:-d .开玩笑！
2.  准备带幻灯片的 20 分钟演讲？轻松点。(创建了 36 个之后)
3.  一些有趣的文章作为课程创建的副产品出现了。
4.  在某个时候，我将能够估计它是否有利可图。
5.  对我来说，这是在一个新领域的有趣经历——相当令人兴奋。早些时候，我从[lynda.com](http://lynda.com)、[乌德米](https://www.udemy.com)和[pluralsight.com](http://pluralsight.com)那里学到了很多东西，我很欣赏这些家伙(像[特洛伊·亨特](https://www.pluralsight.com/authors/troy-hunt)、[斯科特·德里斯科尔](https://app.pluralsight.com/profile/author/scott-driscoll)、[摩西·哈梅达尼](https://www.udemy.com/user/moshfeghhamedani/)等)，他们创造了这样的课程，让困难的事情听起来清晰易懂。因此，成为课程创建者之一的可能性对我非常有吸引力！我试图把重点放在我以前作为学生观看的课程中最喜欢的方法上:不要浪费学生的时间，以简洁明了的方式说话，以便每个人都可以理解，使用许多代码示例，解决一些现实生活中的任务，解释基础知识，然后提供一些典型的边缘案例和解决方案。

**负号:**

1.  家庭时间急剧减少(当然，如果课程不是你的工作)
2.  当一课的主要思想还没有显露出来时，有些人会强调(乘以课数)
3.  由于我有两个小孩和一份全职工作，课程的创建给我带来了明显的疲惫。

我会推荐给你吗？如果你有时间，想成为一名更好的专业人士，或者只是需要视频教程经验——当然可以，但是……如果你有工作和孩子——只有在获得视频课程创作经验对你至关重要的情况下。对我来说，金钱是一个很弱的动力:来自出版商的预付版税和销售的百分比对新手作者来说是相当适度的。
