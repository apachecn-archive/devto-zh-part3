## [](#webpacker)网页打包员

Webpack 现在是官方的 JS bundler，链轮仍然是静态资产(图像)和 CSS 的推荐资产管道。

## [](#new-frameworks)新框架

它包括两个新框架:

*   行动文本:轻松为您的应用添加丰富的内容支持
*   行动邮箱:轻松路由收到的电子邮件

## [](#even-more-scalable)更加可扩展

扩展的两个主要特征:

*   并行测试:加速多核环境下的测试套件
*   多数据库:简单的 API，支持多数据库活动记录

以及许多其他的改进
