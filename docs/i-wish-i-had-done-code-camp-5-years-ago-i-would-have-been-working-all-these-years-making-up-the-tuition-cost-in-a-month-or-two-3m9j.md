# 我在全栈 JavaScript Bootcamp 的经历

我在 2018 年 8 月至 10 月参加了巴塞罗那代码学校，对体验非常满意。我以前有编程和 Javascript，HTML，CSS 的经验，并一直试图自学 web 开发。因此，我有点担心这些部分对我来说会移动得太慢。

然而，有了视频讲座作为现场讲座的补充，**我可以按照自己的速度进行**。我还发现，即使我知道很多材料，每天 8 个小时的课程也是一种惊人的“练习，练习，练习”。后来，我觉得自己踏实多了，当然也填补了一些知识空白。有经验丰富的老师帮助我学得很快。

我是后端编程的新手，尽管有很多材料，我觉得我已经达到了一个更有经验的水平。还有很多东西要知道，还有很多东西要学习，但这就是这个行业的本质——所以要习惯它！

# 我选择巴塞罗那代码学校的原因

我选择巴塞罗那代码学校是因为他们专注于 React 和 React Native。我住在巴塞罗那，我想学习这些，因为它们是波士顿(我以前的家)现在最需要的技能。另外，我知道他们很酷。

我也曾试图自学 React，所以我有了一个开始，但没有什么比得上有一个老师。其他代码营表示，如果人们感兴趣，他们在 React 有一个晚上的特别研讨会，但这怎么可能够呢？这门课程全用英语教授，老师们在回答我的所有问题时都很有帮助，所以我理解了其中的概念和技巧。

# 为什么是编码训练营？

多年来，我一直在考虑代码学校，但任何代码学校的费用似乎都很高。我试着自学，慢慢地走着，有时会被一些愚蠢的问题堵上好几天，我知道这些问题可以由有理解力的人来解决...编码]在 5 秒内。

然后今年，我看着一些孩子从大学毕业，意识到他们没有试图“自学生物学”。你可以做到，但是要多花 100 倍的时间。老师很重要！所以我最终决定入学。我希望我 5 年前就参加了代码营。我会一直工作这些年，在一两个月内赚回学费。

# 我的训练营体验成果。

巴塞罗那代码学校非常适合我，因为它不太长，也不太贵。女士们——有针对女性的[奖学金和募捐活动。另外，我喜欢接触视频和课堂笔记。我可以回去查资料和复习，这是我参加大量工作申请技术测试所需要做的。](//www.f6s.com/futurefunded)

课程结束后，学校把我的名字发给了招聘人员，我接到了很多电话。我参加过很多面试，有一些不错的机会。我目前正在和 3 家优秀的公司进行严肃的谈判。

所以，我很高兴我决定去 BCS 。
