**1-布拉德·特拉弗斯**

Brad 开始为他创建的名为 Tech Guy Web Solutions 的公司建立客户网站，后来发现他真正热爱的是以简单易懂的方式教授 Web 开发和编程。布拉德现在经营着 YouTube 上最顶尖的网络开发和编程频道之一。

你可以在这里查看布拉德的课程

2-克里斯·肖恩

Chris 是一名经验丰富的前端工程师，拥有在出版行业工作的丰富经验。他精通 Vue.js、微软 Word、销售、管理和社交媒体。此外，克里斯是一位才华横溢的优秀工程师。

3-摩西费赫·哈梅达尼

Moshfegh Hamedani 是一名热情务实的软件工程师，专门从事 web 应用程序开发，包括 ASP.NET MVC、Web API、实体框架、Angular、Backbone、HTML5 和 CSS。他拥有软件工程学士学位和网络系统硕士学位。虽然他从未学过如何创建游戏，但他有超过 15 年开发 web 应用程序的专业经验。

查看他的网站，享受多达 20 门令人惊叹的[课程](https://programmingwithmosh.com/)

4-约书亚·福禄克

约书亚是一个雄心勃勃的勤奋的人。事实上，他是在没有开发背景的情况下进入这个行业的，并且一直在想办法让自己和他的学生有机会进入这个行业。这些学生通常没有经验，或者只有很少的经验。传统教育在发展中是一个最小的因素，因此他用他的方法跳出框框思考。

他已经和超过 35 名不同年龄和背景的学生一起工作过，并且有很多学生同时收到了多份工作邀请。

他可以是你正在寻找的导师，看看他的[网站](https://grindreel.com/personal-mentorship/)

他的 Youtube 频道也提供了许多开发工具、技巧和窍门

保罗·哈利迪

最后但同样重要的是，“保罗·哈利迪”。Paul 是一名软件开发人员，他热衷于通过教学来学习。他的课程有超过 20，000 名学生，内容从跨平台移动应用程序到 web 应用程序和 VR 体验。

他在自己的[网站](https://alligator.io/author/paul-halliday)上完美地展示了自己的技能。
