# 拖延季节

> 原文:[https://dev.to/jmspace/procrastination-season-141f](https://dev.to/jmspace/procrastination-season-141f)

#### [](#i-just-spent-6-months-procrastinating)我就是这样拖了 6 个月。

我真的休息了很多时间。我不认为在那段时间里我有把握。

[![](../Images/f413e37129fba2320659b5bf5abb44cc.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--crOYOmJB--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://cdn-images-1.medium.com/max/1024/1%2AZO0EUppuWgoIef_4X52mvw%402x.jpeg)

我仍然在拖延，但是我已经克服了无所事事的感觉。我已经很久没有像以前那样编码了，所以我想我需要重新开始学习一切。我没有看过线框。

我考虑编码，偶尔我坐在笔记本电脑前，什么也没做。真的。我只用笔记本电脑编码。因此，如果我不编码，我的笔记本电脑就不会打开，如果我的笔记本电脑没有打开，它就不会像它应该的那样频繁地更新，如果它在我使用它时试图更新，我就不能在这段时间内编码，因为我只有 2 GB 的 RAM。因此，一旦我意识到我的笔记本电脑正在更新，我只需关闭它，这样它就可以更新。你赢得了 Windows 10。

我想我已经到了什么都不想做的地步了。我的懒惰开始了，我宁愿什么也不做。

我甚至还没有完成我的在线课程。对我来说幸运的是，每次我落后的时候，我可以注册一个新的班级。但这也是我没有说完的原因。没有真正的截止日期。

*   也许我变得不知所措或无聊。
*   可能当时我的事情太多了。
*   也许我失去了动力。

### 是时候回到正轨了。

[![](../Images/24bcf3f7caa384970a76935b56019e69.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--41MCby1e--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://cdn-images-1.medium.com/max/1024/1%2AwMOasR7xK1IER0LOCDCB1w%402x.jpeg)

所以我完全改变了我做事的时间表，直到它起作用并继续起作用。

我要去试试街区调度。这是我没试过的。所以我要去看看能不能帮到我。我也要承诺我的 BUJO。这是一种创造性的方式来安排我的一天，并根据目标和其他需要做的事情来思考。我认为 BUJO 是一个小型创意渠道。

一旦我找到了做每件事的最佳方式和时间，我会参考我写的一篇关于完成工作的文章。每次我觉得我开始错了的时候，我都应该读一读。
