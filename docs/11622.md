# Vue SSR 样板文件

vue-cli-ssr-example 的一个分支，增加了一些额外的功能。

## 特征

### 服务器端渲染

*   view+view-路由器+ vuex 协同工作
*   服务器端数据预取
*   客户端状态和 DOM 水合
*   路由级代码拆分

### 渐进式网络应用程序

*   应用清单

## 使用

安装依赖项:

```
yarn 
```

启动开发服务器:

```
yarn serve
```
