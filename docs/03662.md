# 如何在你的职业生涯中接受拒绝

> 原文：<https://dev.to/ashleemboyer/how-to-accept-rejection-in-your-professional-life-3pej>

拒绝很糟糕，出于某种原因，来自熟人和陌生人的拒绝*尤其是*很糟糕。也许你提交了一份在活动中演讲的提议，但没有被选中。也许你申请了一份工作，他们没有给你面试机会。也许你要求升职或加薪，却得不到你所需要的。我们如何从这样的拒绝中走出来？

首先，不要马上就往心里去。你确实很棒，但是让我们喘口气，更仔细地分析一下情况。从另一个人的角度来看，拒绝很可能有一些好的理由。你的演讲主题可能与活动的目的不太一致。另一个申请人可能更适合这份工作。你的公司可能没有空缺职位来提拔你，或者没有足够的钱来提高你的工资。

在你问任何后续问题之前，确保你真诚地**感谢**拒绝你的人抽出时间。这样做会增加他们让你记住未来机会的机会。

一旦你感谢了他们，然后考虑一些潜在的反馈问题。你想确保它们听起来不会很苦涩，所以如果你想仔细检查你的措辞和语气，看看你是否能在前进之前找到人来审查它们。你可以问什么会让你的演讲更有吸引力，你在申请中缺少什么技能或资格，或者你可以在工作中做些什么，以便在下次升职/加薪时得到更认真的考虑。

你可以在这里找到这些问题的答案，然后带着它们跑步。希望你能收到积极有益的反馈。以此为基础，让自己成为更强的候选人。然而，如果反馈没有帮助，或者你对此不屑一顾，也许是时候深入了解情况了。在这种情况下，找一个外部的视角来谈论它。他们可能会看到你看不到的东西。

无论如何，请记住，你不是唯一被拒绝的人。这种事每天都会发生在每个人身上。不要害怕把自己放在那里。你可能会遇到一生难得的机会。
