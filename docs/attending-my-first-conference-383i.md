# 参加我的第一次会议

> 原文：<https://dev.to/moopet/attending-my-first-conference-383i>

几个月前我参加了一个会议 <sup id="fnref1">[1](#fn1)</sup> : [苏格兰 CSS](http://scotlandcss.com/) 。对我来说很方便，因为旅行不远。这也是我参加的第一个科技会议。事情是这样的。

我要求我工作的公司支付钱和休假，他们答应了，多花了一点钱来帮助支付某人的多样性机票。

有很多事情是我没有预料到的。一些小事，比如出于某种原因，我没想到我们会吃午饭。

第二天，这个场地将被重新用于 Javascript 会议，很多与会者都去了，所以程序员非常多。我不知道，但我认为我期望它是代码轻量级的，因为 CSS“只是为了表示”。这里的一些谈话有助于消除这种想法。

因此，这些演讲大多是技术性的(是的，其中涉及到一些数学知识)，但也有一些更为一般性的。关于如何在这个行业取得成功，如何应对困难局面，诸如此类的事情。我不确定我对那些部分的感觉。我的意思是，他们是体面的会谈，但这不是我为什么有兴趣去。我认为，如果一个会议是关于一个非常具体的事情，那么一般感觉良好的会谈有点像填充；他们本可以在任何科技会议上表现得同样出色。我理解人们为什么喜欢它们；我很喜欢它们，我也明白为什么让它们参加会议意味着人们不必向老板要一张积极性研讨会的门票，而是要一些更“工作”的东西，但它们并没有给我个人的一天增加多少。

大厅里有几家赞助商，但他们一点也不强势。事实上，他们是如此放松，以至于我根本不知道他们是否是任何事情的官方代表，并且对接近他们感到有点奇怪。所以我没有。

我选择了挂绳的颜色，我选择了红色，这是会议上表示我不想被拍照的方式。我甚至不知道这将是一个选项，但很高兴它以如此清晰的方式完成。摄影师不需要眯着眼睛看你姓名徽章上的标记，或者——但愿不会——当面问你是否可以。他们确实这样做了，无论如何，他们走过来聊了聊，说他们会真诚地尝试不在任何照片中包括我们这些红丝带成员，但显然人群拍摄很难管理。

总的来说，组织者让这一天感觉非常舒适。他们竭尽全力让人们感到被包容。他们如何看待观众的一个例子是，他们在舞台上有一排座位，如果你想在未来的某个时候演讲，你可以站在那里，在演讲期间从人群中往外看，以了解这种感觉。没有多少人接受他们，但有一对夫妇接受了。

而我快乐的回家了 <sup id="fnref5">[5](#fn5)</sup> 。
