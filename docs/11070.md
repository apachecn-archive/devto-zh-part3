# 尽管如此，莲娜编码

## [](#i-startedcontinued-to-code-in-2019-because)我在 2019 年开始/继续编码是因为...

其实 2019 年我一次都没编码过但是 2018 年我编码过。我甚至因为对我的第一个开源项目做出了贡献而得到了一件 hacktoberfest t 恤。我不认为自己是一个程序员，但我知道 HTML，并且能够做出贡献，这很酷。

## [](#i-deserve-credit-for)我应得的功劳...

成为一个合法的开源贡献者。你可以看看我的开发配置文件，并看到贡献者徽章作为证据！我还设计了徽章，这是我应得的荣誉，因为这是我在这里真正做的事情😄

我在 markdown 写这篇文章。我觉得这很酷，我甚至知道减价是什么，我喜欢它。

## [](#i-hope-to-see-my-schoolworkdevelopertech-community)我希望看到我的学校/工作/开发者/技术社区...

我希望整个软件行业变得像 DEV 上的社区一样善良和令人敬畏。
