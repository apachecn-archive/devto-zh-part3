# 为什么网络应用比独立应用更受欢迎？

> 原文：<https://dev.to/decipherzonesoft/why-web-applications-are-becoming-popular-than-standalone-applications-2nbi>

[![](img/6c44e22fdf139dd06a70097e180d6cdd.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--7eT7EzsH--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/f4yneb7khb6pw7e16f35.jpg)

互联网大约在 25 年前出现，已经成为我们社会不可避免的一部分。我们都熟悉网站和独立应用程序，我们在系统中使用它们来执行一组不同的任务，现在这些独立软件被嵌入到网站中，人们可以通过简单的网络浏览器应用程序来使用该软件。这些作为本地应用程序运行和工作的网站被称为 web 应用程序(web 应用程序)。

什么是 web 应用开发？创建软件应用程序的过程被称为 web app (Web 应用程序)开发，该软件应用程序被存储在遥远的远程服务器上，并通过复杂的 Web 浏览器经由因特网在用户的设备上呈现该软件应用程序。经过这个过程后创建的具有动态行为的网站被称为 Web 应用程序(Web applications)。

为什么 web 应用程序比独立应用程序更受欢迎？独立的应用程序给公司和用户带来了许多问题，如果他们将应用程序转移到 web 开发平台上，这些问题就可以得到解决。
