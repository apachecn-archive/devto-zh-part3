# Cloudflare 宣布 workers.dev

> 原文:[https://dev . to/Glenn men/cloud flare-announced-workers dev-3 no 9](https://dev.to/glennmen/cloudflare-announced-workersdev-3no9)

Cloudflare [今天宣布](https://workers.dev/)他们的`workers.dev`服务。
目前处于预注册阶段，您可以注册一个自定义的`workers.dev`子域(注册是免费的)。

一些基本信息这是什么，这里的大多数人可能听说过 Cloudflare Workers(或其他一些无服务器的替代产品)。很久以前，如果您想在 Cloudflare 上托管一个无服务器应用程序，您必须已经拥有一个域，并将您的名称服务器更改为 Cloudflares 名称服务器。不是每个人现在都知道如何获得域名、管理域名服务器(或其他 DNS 设置)，因此有了这项新服务，Cloudflare 将会处理所有这些事情。您可以获得 Cloudflare Workers 的全部功能，而无需管理其周围的事情。
