# 你好世界！！这是一个初级贡献者

> 原文：<https://dev.to/feralamillo/hello-world-this-is-a-junior-contributor-45nm>

作为初级开发人员和编码训练营的 6 个月商业经验之后，我觉得前方的路还很长。我真的很喜欢分享的开发者，因为这是一种很好的学习方式。从这个意义上说，虽然我不是专家，但我觉得我仍然可以以某种方式为社区做出贡献。

[![Hello World](img/1a1335f51139d269a6f38e7a32001aca.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--TxdldCPh--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/lh7t98w0dpglgb5q8afd.jpeg)

从大学开始，我就一直参与创业，在那里我遇到了很多不可思议的人。希望对世界产生影响、改善做事方式的个人。我担任过与业务、销售、增长、项目管理、产品相关的不同角色，我一直认为编码是一项关键技能。几年前，是我关注它的时候了。我参加了一个训练营，当我完成后，我在一家高档创业公司找到了一份工作。

还有很长的路要走，我正在一步一步取得巨大的进步。我希望人们能就做出贡献的最佳方式提供反馈。与以下内容相关的提示:

*   生成内容:需要考虑哪些网站或技巧？
*   参与开源项目。
*   要考虑的其他选择。
