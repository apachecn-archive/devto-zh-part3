# 密码管理器及其使用方法

> 原文:[https://dev . to/jayhankins/password-managers-and-how-to-use-them-442m](https://dev.to/jayhankins/password-managers-and-how-to-use-them-442m)

## [](#2019-year-of-the-password-manager)2019:密码管家年

说真的，现在是 2019 年，如果你没有使用密码管理器，我的拙见是，你正在给自己设置一个 [*包伤害*](https://www.macworld.com/article/1136079/bluray.html) 。安全漏洞每天都在发生，有些被检测到，有些没有。您必须假设任何 web 服务的任何凭证都可能被泄露。为了降低这种风险，您需要:

*   使用随机、复杂的密码
*   永远不要重复使用密码

通过做这两件事，任何泄露的凭证都不能被用来访问其他帐户。你是如何完成的？使用密码管理器。苹果的 iOS 和谷歌的 Android 最近的变化使得在手机上使用密码管理器比以往任何时候都更容易。你可以在这里阅读更多关于密码管理器的内容，但是你真的应该停止质疑它。基本上，如果你使用互联网，你就有风险。

### [](#how-to-do-it-right)怎么做才对

仅仅将密码存储在密码管理器中是不够的。以下是正确使用密码管理器的方法:

您的主密码必须完全唯一。

不可能跟别的有关系。它不可能泄露出去，因为它只能存在于你的头脑中。要有人想明白可不容易。不能是生日，不能是昵称，不能是姓。

需要注意的一点是:由单词和短语组成的密码实际上可能比一些由普通数字或符号组成的密码更强。听说过“[正确的马电池订书钉](https://xkcd.com/936/)”吗？

总是尽可能随机地生成密码。

每个网站都应该有自己独特的密码。看起来越复杂越好。你不会记住它，你的密码管理员会！

您经常会发现某些网站，如政府、医疗和金融网站，不支持广泛的密码字符，或者不支持长密码。这很糟糕(合规和监管落后于实际的最佳实践)，但密码管理器将帮助您充分利用这种情况。一个好的密码管理器将有一个熵计，它将帮助您确定最强的密码，您可以适应有限的密码要求。

当然，只要你能生成复杂的密码，你就应该这样做。大多数现代网站都支持相当长且复杂的字母数字密码。

审核你的密码。

正如我们所讨论的，泄漏时有发生。所以，当你的密码在网上泄露时，你需要得到通知。首先，去[haveibeenpwned.com](http://haveibeenpwned.com)查看你不同的电子邮件地址，然后注册他们的电子邮件提醒。你被 pwn 了吗？(我也是，没什么丢人的。只是随机的！真的，如果你用过互联网，你可能已经被 pwn 了。)如果是的话，是时候改一些密码了。

好的密码管理应用程序会内置定位弱密码的功能。您应该立即更改已识别的弱密码。

你也不能有重复的密码。有一个重复的密码是一个非常糟糕的场景，因为一个漏洞会让你在多个地方容易受到攻击。密码管理器通常会提醒您不要使用重复的密码。

此外，寻找你的信用供应商提供电子邮件提醒。有一些供应商会搜索黑暗网络的一部分，在漏洞中寻找你的信息。

还有，你是万豪酒店泄密事件的一部分吗？如果是这样，注册他们付费的身份保护服务。这是你最起码应该做的。

永远不要再使用非随机密码。

**不如实回答安全问题。**

安全问题很糟糕。你知道有人知道你的生日、你妈妈的婚前姓和/或你宠物的名字有多容易吗？这些安全问题是我能想到的最弱的安全形式。不要给出真实的答案。

取而代之的是，给出假答案，或者更好的是，为每个问题输入随机密码。请注意，有些网站不接受高度复杂的安全问题答案。没关系，只是要确保观看你的密码管理器的熵计，以生成最安全的答案。

现在，任何时候有人试图重置您的密码或绕过正常的身份验证流程，他们都会遇到密码答案毫无意义的问题。

> 问:你母亲的娘家姓是什么？
> 
> 答:“9T9。FE#54R98416B "

多美的名字。

在任何需要的地方获得应用程序。

说这个好像很重要。如果您正确设置了密码管理器，除了主密码之外，您将不再知道任何密码。

这意味着你需要将这些密码与你使用的任何手机、平板电脑或其他电脑同步。因为离开电脑而被谷歌账户锁定是没有效率的。

大多数密码管理器应用程序将在一个高级服务中云同步您的密码，无论是订阅还是一次性购买。云同步是安全的，因为你的密码钱包是用你的主密码加密的——对手试图暴力破解你的密码钱包是浪费资源，除非你是一个非常重要的人，并且认为国家支持的攻击者正在瞄准你。你也可以使用 KeePass 这样的离线密码管理器，手动同步密码，尽管这非常不方便。

哦，这包括浏览器扩展。所有好的密码管理器都应该有功能扩展，使你在浏览时使用安全密码变得轻而易举。

**要勤奋。**

承诺使用您的密码管理器。

如果你在很多网站上都有很多账户，你会很快意识到设置密码管理器是一件很费力的事情。当你登录更多网站并在你的应用中保存这些密码时，你必须返回并再次审核。大多数密码将需要彻底改变(一个新的，安全的)，一些网站将有过时的网址，登录流程等。一天一天地关注这件事，因为当它发生时更容易处理。

密码管理器也可以让你知道你的账户。每隔一段时间，我都会浏览一下我的密码管理器，看看我打开了哪些不再使用的账户。减少攻击面总是好的。
