本教程解释了如何在 javascript 编程语言中实现析构。析构是一种从数组和对象中的值创建变量的强大方法，它将使您的代码更简单。

什么是解构？ES6 中引入了析构。这是一个 JavaScript 特性，允许我们从一个数组或对象中提取多段数据，并将它们赋给自己的变量。
