STO 启动流程一开始就需要启动文件和 SEC 注册。向美国证券交易委员会注册是必要的，因为在注册之前，由于美国法律，卖方不能合法出售代币。那些运行新数字经济的人应该考虑保护投资者的法律。但这只是第一步。除了管理这些手续，卖家还应该管理很多其他的事情。
选择区块链平台也是一个重要的事情，没有它就没有注册这个业务的必要，因为，它不会成功。如果你想知道如何启动 STO，打开这个网站:[http://mere head . com/blog/how-to-launch-security-token-offering-sto/](http://merehead.com/blog/how-to-launch-security-token-offering-sto/)。除了区块链平台，我们还需要一个合适的项目团队。它应该由专业管理人员、开发人员、顾问和营销人员组成。关键人物对项目非常重要，因为投资者首先关注他们。然后你要选择代币的类型。弄清楚代币的功能也很重要。费用的计算是必要的一步。一开始就做，因为首先你要考虑你是否有足够的钱来做这件事。如果你缺钱，你就会失败，浪费掉你已经投入的资金。
