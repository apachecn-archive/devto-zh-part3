# 他们是这样卖“Kubernetes”的吗？

> 原文:[https://dev . to/pavanbelagatti/is-this-how-they-sell-kubernetes-3 olb](https://dev.to/pavanbelagatti/is-this-how-they-sell-kubernetes-3olb)

如果“Kubernetes”是一个菠萝，那么“GKE”就像是在一家名为“谷歌云”的商店里卖菠萝，店主“谷歌”在上面贴上贴纸，包装精美，然后切成小块，这样你就可以很容易地吃到它。

另一方面，这个名叫“亚马逊”的家伙在“谷歌云”对面有一家水果店，他看到了这一点，他也拿了一些菠萝，在他名为“AWS”的店里出售，在那里他以他认为不错的方式包装菠萝，并以“EKS”的名字出售。

RedHat 将菠萝取出，制成果汁，放入一些防腐剂使其更加安全和稳定，包装在一个玻璃杯中以便携带，并给它起了一个名字“Openshift”。
