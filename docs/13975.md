# 创建了我的第一个 NPM 包(第一个帖子)

> 原文：<https://dev.to/amroessam/vgauge---created-my-first-npm-package-first-post-446o>

大家好，

我 2019 年的决心是更多地参与开源社区，尽我所能做出贡献。

我必须为工作创建一个仪表板，他们需要看到仪表，我找到的最简单的库是 [GaugeJS](https://github.com/bernii/gauge.js/)

问题是它太分散了，不能在我的项目中使用，所以我为它创建了一个包装器。

我经历了创建包并发布它的过程。

根据 NPM 文件，这个过程相当简单。

做

`npm i vgauge`

是一种令人满足的感觉，而且确实有效。

这里有 [github 页面](https://github.com/amroessam/vgauge)，一个现场[演示](https://jz3qoxny63.codesandbox.io/)，如果你想修补的话还有[沙盒](https://codesandbox.io/s/jz3qoxny63)
