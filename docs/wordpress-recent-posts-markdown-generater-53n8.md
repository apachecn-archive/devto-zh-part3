# wordpress 最近帖子降价生成器

> 原文：<https://dev.to/jackharner/wordpress-recent-posts-markdown-generater-53n8>

我做了另一个东西。这一次，它是一个 WordPress 插件，可以为你最近的帖子生成一个链接列表。我专门为链接回你的博客文章而建。到(或其他降价支持的)帖子。

# [结账](https://wordpress.org/plugins/recent-posts-markdown/)

或者你可以通过你的 WordPress 后端安装它。只需在添加插件屏幕中搜索`Recent Posts Markdown Generator`。

一旦安装完毕，你只需进入工具>最近的文章降价。选择你的文章类型和你想要链接的文章数量！指向您最近帖子的链接列表。
