# 通过短信向威瑞森用户发送 ngrok 链接？！

> 原文：<https://dev.to/jess/sending-ngrok-links-via-sms-to-verizon-users-1dhh>

我在《发展》杂志上发表了一些有趣的文章标题，想与我的伙伴分享。很自然的，我设置了一个 ngrok 链接，方便分享。我给他发了几次链接短信，尽管我的提供商 t-mobile 说已经发送了，但他从来没有收到过！他收到了我所有的其他信息/链接/等等，但是没有 ngrok 链接。我试过另一个 verizon 用户，他也没有收到我的链接。

这只是一个奇怪的巧合还是 verizon 屏蔽了 ngrok？在快速搜索中没有看到任何内容。注意:与任何其他移动服务提供商都没有问题！
