# 让新年过得更好的 3 个建议

> 原文：<https://dev.to/goyo/3-tips-for-a-better-new-year-51lb>

你好。

新年到了，我们都准备好至少在-th^1 时间改变我们的生活，所以谁不想要一堆提高效率的小贴士呢？

我的意思是，它们可能不会有太大的变化，但是当你仍然精力充沛、天真无邪、干劲十足的时候，读这样的书感觉不是很好吗？

介绍够了，我们现在应该都在工作，而不是读这篇文章(或者像我这样写文章)。

# 1。承认敌人

不管你今年打算做什么，不管是减肥、学语言、写书，还是其他什么，你都要和你头脑中那个试图毁掉一切的声音做斗争:

*嘿！今天不要开始！周一似乎是开始新事情的好时机...或者更好，下个月的第一天！哦，我当然是指下个月的第一个工作日。*

是啊，这是一个好的开始，所以我们今天休息一下，明天再补上，怎么样？

*没有？没有休息？嘿，面对现实吧。没有人会想读这本书/使用这个项目。*

哦，来吧。进展得如此顺利，你能不能暂时放下困难的部分，现在研究一些市场营销的东西？我是说，你总有一天会处理好的...

哦，不！这个小小的障碍打破了一切！你完蛋了！你永远也不会解决这个问题。结束了。好游戏，谢谢，再见。

你明白了。不管是懒惰、理性、自我怀疑、恐惧，还是其他什么，**阻力都在那里与你战斗，阻止你实现你想要的**。

准备好 fight^2，不管这个声音告诉你什么，继续前进。同时，做好有一天会失败的准备，第二天(或者下周，或者下个月)重新开始工作。

# 2。确保可持续性

这可能有点讽刺，但在告诉你与试图说服你少做的声音斗争后，我将鼓励(一些)你实际上少做。

当我们一直“玩”在自己的期望之下，罪恶感越积越多，很容易想出一个伟大的改变计划，但不幸的是，这是完全不可持续的。

所以，在你被所有的计划和决心搞得精疲力尽之前，问自己一个简单的问题:

这是我愿意在未来 5 年、10 年、20 年投入的 effort^3 数量吗？

我的意思是，我们所追求的(或应该追求的)是一个终身的积极改变，而不是一年中“和以前一样”的短暂爆发(或生活！).

如果你仍然不服气，让我们做一些糟糕的，不可靠的数学:

1.  一个伟大的计划，每天提高 2%,我们坚持 20 天，耗尽给我们...`1.02 ^ 20 = 1.4859...`...大约 50%的改善，随后是长时间的停滞，实际上远远低于这个数字。
2.  我们坚持一整年每天提高 1%的半伟大计划给了我们...`1.01 ^ 365 = 37.7834...`...惊人的超过 3700%的改进，我们会继续改进，因为它太简单了。四

想想吧。也许制定一个持续产生积极效果的简单计划比我们努力坚持的其他计划更好。

# 3。实验，实验，实验...

如果你像我有时倾向的那样思想封闭，你可能会发现自己在寻找对代码、生活、你自己以及更多的完美想法/愿景/理解。在你做出任何“仓促”的决定之前，你可能会试图抓住这个完美的画面，这个你应该朝着的理想。

如果上面的句子至少部分描述了你，那么我有一些坏消息。有可能你会在到达最终目的地之前死去，或者，在一个稍微好一点的情况下，你会认为你找到了理想，只是意识到你错了。

但是，嘿，有一个替代方法，上面的提示标题已经破坏:*实验*！

如果你有一个改进的想法，不管它有多不完美，“just"^5，努力去做，看看它会把你带到哪里。

如果你有一堆想法，但不知道该遵循哪一个，尝试其中的几个，挑一个你最喜欢的。

如果你没有时间尝试几个，跟随你的直觉，选择一个。

保持开放的态度。我们很少拥有做出完全知情的决定所需的所有信息，也就这样吧。如果有的话，它使我们的生活更有趣！

看在上帝的份上，别担心。除非你正在进行太空任务，否则即使失败也没什么大不了的。^6:你失败的几率可能比你想象的要小得多。

# 最后的话

我不知道为什么一个最初打算是一堆编码生产率技巧的帖子变成了这个新年自助的东西，但是它在这里。我希望你没有讨厌它，如果至少有一个人觉得它有价值，那就太好了。编码技巧将不得不等待另一次灵感的迸发。

祝你新年快乐！
