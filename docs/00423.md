## [](#so-why-write-it)那么为什么要写呢？

我想学习一点关于 Vue.js 的知识，并证明我已经有一段时间的理论。这个理论是这样的:所有的 javascript 组件框架都以几乎相同的方式运行。它们都有相似的
模式、惯例和最佳实践。它们都很容易拿起
并开始编程。我们对工具、工具的过度设计，以及缺乏 Javascript 中的大型标准库
是前端编程变得困难的原因。
非要学 Vue.js，或 React，或 Ember，或 Angular 都很容易；不得不学习这些框架各自的生态系统是困难的。在前端开发中，
所有的概念都是一样的，我们只是找到越来越聪明的方法让它们
难以理解。
