# 网络犯罪

它包括从网上银行账户盗取数百万美元用于下载非法音乐文件。

网络犯罪还包括涉及非金钱交易的交易，如在其他计算机上传播病毒或在互联网上发布任何业务的私人信息。

网络罪犯可以利用计算机技术获取你的个人信息，了解蹦极交易的秘密，或者利用互联网达到其他恶意目的。

从事这种非法行为的罪犯通常被称为黑客。
