# “云计算”也可以吗？

> 原文:[https://dev.to/vjnvisakh/can-clouds-compute-too-18hj](https://dev.to/vjnvisakh/can-clouds-compute-too-18hj)

**云计算**已经成为业界新的流行词汇有一段时间了。但是它是什么，它是如何改变工作方式的。是像听起来那么简单，还是另有隐情。让我们看看。

1.  云计算是一种管理资源的新方式。你不得不购买服务器的日子已经一去不复返了，我指的是一个物理服务器来保存你的文件，以便公众可以访问。

2.  然后是共享服务器的时代，你可以让你的邻居把他们的文件保存在你的服务器上，这样他们的网站可以通过给你象征性的一点钱来浏览。

3.  然后出现了专用服务器，每个人都可以获得自己的物理服务器，而无需实际购买并带到家中。

4.  然后 VPS 的人来了，你可以在那里购买一个虚拟环境，它将和专用服务器一样好。

那么云计算会带来什么呢？

嗯，三样东西- **可扩展性、安全性和资源。**T3】

### **可扩展性**

如果你使用过共享主机，你会知道存储文件的空间是有限的。索引节点的限制，内存的限制，任何你能想到的限制。云计算克服了这一点，在你真正需要的时候给你你所需要的。相信我，它是无限的。

 <small>你问什么意思？好吧。想象你有一个网站，也许是一个托管在物理服务器上的社交网络网站。第一天，两个人去参观。一切都很好。第二天 10。第二天 50，然后 100，然后 1000 等等。<u>但是突然你开始注意到，随着每个新用户的注册，网站开始变慢。</u>有人告诉你，你必须为服务器购买更多的空间或者 RAM。你出去买一些。它还能工作几天，然后你又开始面临网站运行缓慢的困境。你的用户越来越沮丧，你没有更多的钱来购买更多的内存。</small>

你是做什么的？你从主机提供商那里租了一台服务器。您同意为资源支付租金。它可以正常工作几天，但速度会再次变慢。主持人说你必须为更多的资源支付更多的租金。当你和你的主机提供商谈判的时候，你的用户很难接受。你转向专用服务器。那也能工作一段时间，然后噗！那也不能处理负载并关闭。你又开始谈判了。到目前为止，30%的用户已经离开了。迄今为止投入的所有资金都没有起到任何作用。

<small>然后你了解了<u>云计算的家伙</u>。他提出了一个很好的建议。他说“我会在流量高的时候通过增加资源来照看你的网站(**增加**)，当流量低的时候会减少资源(**减少**)。此外，如果没有流量，你也不必支付”。啊！好极了。正是你要找的。不会再有网站因为流量而关闭。你被分类了。</small>

 **云计算有一个随需应变的资源扩展过程。所以你的内存或硬盘等会随着需要而动态增减。你只需为你使用的次数付费。
T3】**

### **安全**

但是你想想我网站的安全性。当我拥有一台自己的服务器时，我能够以我想要的方式保护它。云计算有吗？是的，不仅仅是简单的一个。它有一个最安全的网络。

### **高端资源**

而且不止于此。如果你是一个开发者，你不需要购买一个新的软件/工具来工作。你可以租用它，像在你的电脑上使用其他软件一样使用它。你听说过 **Google Stadia** 吗？是的，下一件大事即将到来。它可以让你玩高端游戏，只需要一个良好的互联网连接。你不必为了玩 GTA 5 而做一个怪物柜子。你只需要一个良好的互联网连接。你是一个离不开 photoshop 的创意狂吗？好消息是，你不必购买它，它就在云中，所以只需租用和使用它。
