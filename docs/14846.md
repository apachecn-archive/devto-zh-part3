# React CRA + Netlify = 💯❤

> 原文:[https://dev.to/ajsharp/react-cra-netlify-2jp](https://dev.to/ajsharp/react-cra--netlify---2jp)

我已经将 static frontend 移植到 s3/cloudfront 有一段时间了，我终于给了 [Netlify](https://netlify.com) 一个机会和**wow is easy。**你连接你的 github repo，给它一个命令运行就搞定了。他们在幕后处理所有我过去必须手动配置的 cloudfront / CDN 东西。

[查看此处](https://www.netlify.com/blog/2016/07/22/deploy-react-apps-in-less-than-30-seconds/)了解更多信息。
