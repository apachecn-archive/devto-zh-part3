## Web 框架强化

*   [Secure.py](https://github.com/cakinney/secure.py) - secure.py <g-emoji class="g-emoji" alias="lock" fallback-src="https://github.githubassets.cimg/icons/emoji/unicode/1f512.png">🔒</g-emoji>是一个轻量级的包，为 Python web 框架添加了可选的安全头和 cookie 属性。
*   这个简单的扩展为 Flask 路由提供了基本的、摘要的和令牌的 HTTP 认证。
*   Talisman 是一个小的 Flask 扩展，处理 HTTP 头的设置，有助于防止一些常见的 web 应用程序安全问题。
*   [姜戈会议 CSRF](https://github.com/mozilla/django-session-csrf) - CSRF 保护姜戈没有饼干。
