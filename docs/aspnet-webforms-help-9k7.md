# Help 窗体帮助

> 原文：<https://dev.to/shortstuff81000/aspnet-webforms-help-9k7>

我利用 ASP.NET 网络表单创建可访问的、数据驱动的、移动友好的网站。尽管 WebForms 是非常古老的技术，但我更喜欢 web forms 而不是 MVC，因为:

*   在 MVC 中，对我来说，将标签 ID 与数据属性相关联比将名称属性与数据属性相关联更容易
*   我利用他们的主题系统来创建彩色主题，用户可以使用这些主题来提高网站的可读性，或者简单地选择一种有趣的不同颜色而不是默认颜色。
*   比起 MVC 站点，我更喜欢编写 WebForms 站点

我写这篇文章是希望找到一个导师——一个我可以分享我的项目并在需要时寻求帮助的人。我目前正在向实体学习 LINQ。我已经成功地创建了创建、读取、更新和删除数据的函数，但是在读取相关数据时遇到了问题。我试过用 ASP.NET 论坛和 StackOverflow 提问。如果我得到一个答案，它通常是我无法理解的，如果我用另一个问题回复，我通常不会得到另一个回复。我认为找到一个可以分享我的项目的人会更容易，这样当我有问题时，他们就知道我在做什么。如果你有兴趣帮助一位希望提高网站开发技能并获得更多客户的自雇女商人，请[发电子邮件给我](mailto:Blind.Geek81@gmail.com)。
