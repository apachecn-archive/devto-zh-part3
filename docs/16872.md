# 我的 MacBook Pro 的蛤壳模式

> 原文：<https://dev.to/peterwitham/clamshell-mode-with-my-macbook-pro-2dn6>

我在我的网站上写了一个帖子，觉得在这里分享也是合适的。

## 现在翻盖式什么？

所以，让我们先了解一下基本情况，什么是蛤壳式模式？这是指您合上笔记本电脑的盖子，使用外接键盘、鼠标和显示器时的情况。

## 这又教会了你什么？

很高兴你问了。因此，考虑到我几乎总是在笔记本电脑上使用外接鼠标和键盘，这种转换是没有痛苦的。一开始我以为我会怀念有两个屏幕的日子(记住，笔记本电脑的屏幕现在是关着的)。事实并非如此！

由于 macOS 上的桌面空间，我通常使用全屏视图来显示应用程序，所以我需要适应的只是更频繁地切换空间，而不是只看另一个显示器。

我使用插入 USB-C 集线器的戴尔 27 英寸外接显示器，以及键盘、鼠标、以太网和几个外接硬盘。我的桌面分辨率是 2560x1440，所以当我切换回 MacBook Pro 屏幕时，我不必担心 windows 会跳来跳去。是啊，那也让我很烦！

我了解到的最有趣的事情是，当盖子合上时，我的 MacBook Pro 的运行温度大约低 11 度(平均 101 度)，这让我感到惊讶，因为热量是我在盖子合上时的主要问题(MacBook Pro 2018)。现在我认为这是因为它只需要要求 GPU 为一个屏幕供电，而不是两个，但我没有这方面的科学数据。

顺便说一下，我使用 iStat 菜单来监控系统栏中的温度和风扇速度。

我的下一个想法是风扇以更高的速度运转以保持温度下降，所以我检查了这些。没有，仍然运行在一个很好的安静的 2680 转。

对于一些人来说，最后一个问题可能是“但是失去 MacBook Pro 上的 Touch Bar 怎么办？”嗯，老实说，我几乎不使用这个东西，所以除了失去解锁 1 密码的触摸传感器之外，这不是什么大问题。

## 裹

因此，一周后，我对测试结果感到非常惊讶，并彻底改变了我对合上盖子使用 MacBook Pro 的担忧和观点。我将更多地这样做，以重新获得一些桌面空间。我还想为苹果的工程师们在热量管理方面所做的出色工作而欢呼。

如果你曾经想知道蛤壳是否适合你，我现在说，试试吧，你可能会对结果感到惊讶。

我正在我的设置中添加一个垂直支架，因为我知道它对我有用。

我选择的几个原因包括可以改变间隙的宽度，以支持未来的其他设备，加上颜色与我的太空灰 MacBook Pro 相匹配。更不用说价格也很棒。
