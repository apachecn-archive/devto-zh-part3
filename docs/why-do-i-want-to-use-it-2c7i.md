# 为什么我想用它

> 原文：<https://dev.to/vjnvisakh/why-do-i-want-to-use-it-2c7i>

实际上有几个原因。虽然你现在不会承认这些。

1.  **<u>基于组件的架构</u>**
    React 使用组件。组件只不过是一段 HTML 代码，你可以在任何地方重用它。这使得一次编码有点太容易了。

2.  **<u>简单学习曲线</u>**
    反应很容易学会。如果你知道 HTML，CSS 和 JS，那么你已经成功了一半。

3.  **<u>虚拟 DOM</u>**
    React 使用一种叫做虚拟 DOM 的东西，其主要目的是将现有 DOM 与所做的更改进行比较，并使用可能的最佳方式进行更新。是的，不再有忙乱的 DOM 操作。
