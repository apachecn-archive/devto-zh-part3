# 设计模式系列

> 原文：<https://dev.to/itscoderslife/design-patterns-series-18h2>

我正在更新我的设计模式知识，这个系列将会有将近 20+篇博文。每个帖子都是关于一个设计模式的。帖子将包含何时、何地使用模式的要点，以及模式的本质，并附有简单的示例说明，以便理解。

下面是设计模式的列表，分为 3 种类型:

**创作模式**

*   一个
*   原型
*   建设者
*   工厂
*   抽象工厂

**结构模式**

*   适配器
*   桥
*   复合材料
*   装饰者
*   外表
*   轻量级
*   代理人

**行为模式**

*   责任链
*   命令
*   解释者
*   迭代程序
*   调解人
*   纪念品
*   观察者
*   状态
*   战略
*   模板
*   访问者
