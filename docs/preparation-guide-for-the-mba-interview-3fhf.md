# MBA 面试准备指南

> 原文：<https://dev.to/annaeverson/preparation-guide-for-the-mba-interview-3fhf>

当然，你的学术成就和职业经验是商学院录取的关键因素。它们是你被邀请参加面试的原因之一。但是面试本身是极其重要的。MBA 项目也对你的人际交往能力感兴趣。

你的面试让学校有机会更好地了解你自己。他们不仅看重你的价值观，还看重你的沟通技巧，以及你是否能承受压力。基本上，面试是学校评估你的领导潜力的一个机会，不仅仅是你是否能在学术上通过这个项目。

最好是亲自去学校面试，而不是在当地见代表。然后你还有机会[帮我写一篇论文](https://writemyessay.services/essay-help/)并评估它是否适合你。这也让你有机会认识一些老师，并有不止一次的面试机会。

尽最大努力为你的面试着装。这意味着一套保守的西装，尤其是如果你申请的是常春藤盟校的话。你的衣服必须干净整洁，不应该戴很多首饰。如果你有纹身的话，确保没有纹身。

面试会问很多不同的问题，包括你为什么申请这所学校，你的成长经历，你的职业生涯，你的教育和时事新闻。尤其重要的是，你要做好充分准备，解释你去商学院的动机，尤其是你要参加的面试。

你给出答案的语气和答案本身同样重要。你想听起来自信但不傲慢，自信但不自私。回答时要冷静理智。面试官希望看到你在情绪上很平衡。

准备好你自己关于学校及其项目的问题。MBA 面试也是你确定学校是否满足你需求的机会。你的问题应该针对具体的学校和项目。表明你知道自己在说什么，并且已经考虑过学校与你的关系。

像对待求职面试一样认真对待你的 MBA 面试。认真对待，但不要有压力。记住，如果你被邀请参加面试，那是因为他们对你能提供的东西非常感兴趣。
