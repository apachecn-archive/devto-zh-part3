学习 rails 路由是如何生成的，以及它们如何映射到 HTTP 方法和 SQL 查询，这是许多开发人员在开始开发时所忽略的基础知识。在本帖中，我们将探索一些有助于填补空白的基础知识，以便学生能够在最短的时间内了解铁路路线。

Rails 约定描述了一个控制器在大多数时候应该只有 7 个动作。一般来说，如果你正在创建一个控制器或定义路线，并且你觉得你需要 7 个标准动作之外的东西，你应该考虑创建一个新的控制器。
