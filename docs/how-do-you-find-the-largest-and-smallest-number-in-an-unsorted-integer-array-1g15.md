# 如何在一个未排序的整数数组中找到最大和最小的数？

> 原文：<https://dev.to/mu/how-do-you-find-the-largest-and-smallest-number-in-an-unsorted-integer-array-1g15>

Python 是完成这些操作的非常简单的语言。这就是为什么它被用在机器学习过程中

```
input = [10,20,30,11,10]

print(max(input))
print(min(input)) 
```
