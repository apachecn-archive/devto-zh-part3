# 参加了今年的比赛

> 原文:[https://dev.to/javasamurai/flat-adventures-game-off-5273](https://dev.to/javasamurai/flat-adventures-game-off-5273)

今年我们参加了年终总决赛。这是我参加过的最长的一次游戏堵塞，持续了 30 天。今年的主题是“混合”。
所以，我们想创造一种混合了两种东西的东西。所以我们决定做一个 2.5 平台游戏。Godot 是我们的第一选择，因为它很容易操作。我们决定专注于编程本身，而不是艺术方面。因此，我们借用了开放的艺术和音乐，并与游戏配合得很好。我们犯了一些漏洞和错误，我认为这是游戏堵塞的关键。
