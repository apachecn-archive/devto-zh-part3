寻找方法
给你一个整数`N`和一个值`k`。你需要找到一个最小的数字`X`，当它乘以`N`会得到一个整数形式的 N^(1/k 值。

例如，假设 N =6，k=2，那么答案将是 6，因为 6*6 =36，如果我们计算 36^(1/2，它等于 6，这是一个整数。

`Input`
第一行包含两个空格分隔的值`N`和`6`作为输入。

`Output`
打印最小数`X`模 10^9 + 7

`Sample Input`

```
12 2 
```

`Sample Output`

```
3 
```

如果你用 6 乘以 6，那么结果就是 36，这是 2 的完美幂。
