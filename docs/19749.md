# 用 Advent of Code 一起学习 Elixir 吧

> 原文：<https://dev.to/gumi/advent-of-codeelixir-4jni>

2018 年 11 月 21 日在 Plataformatec 公司的博客上写道:“[let’s learn elixir together on advent of code！](http://blog.plataformatec.com.br/2018/11/lets-learn-elixir-together-with-advent-of-code/) ”的报道被公开了。

[Advent of Code](https://adventofcode.com/) 是从 12 月 1 日到 25 日，发表各种语言和水平的编程谜题的企划。 Elixir 的开发者 José Valim 乘坐了这个企划，呼吁为了学习 Elixir 解开谜题。 在此期间，将于日本时间 23:00(UTC 14:00 )由 Valim 在[Twitch](https://www.twitch.tv/josevalim) 上现场解说前一天的谜题解法(因为是一起讨论，所以被认为是观众参与型的) 录像将在 Twitch 上上映 14 天。

准备和参加方法如下所示。

*   通过“[开始获取](http://elixir-lang.org/)”(“[Elixir 入门](https://dev.to/gumi/elixerelixir-10g2))或其他资源学习 Elixir。
*   练习用的介绍了去年的[Advent of Code 2017](https://adventofcode.com/2017) 和[exercism.io](https://exercism.io/) 。 另外，Valim 解开了 Advent of Code 2017 的[12 月 1 日的谜题](https://adventofcode.com/2017/day/1)，在[GitHub 上公开了](https://gist.github.com/josevalim/978771f4f5c948504db1e88383cac8d0)。
*   Advent of Code 企划用的[邮件列表](https://pages.plataformatec.com.br/together-on-advent-of-code-2018)在限定期间内制作，演唱会结束后会通过邮件向读者发送录像链接。
