#### CSS&SCS

##### [使用智能 CSS 基于用户滚动位置应用样式](https://pqina.nl/blog/applying-styles-based-on-the-user-scroll-position-with-smart-css/)

这是一个有趣的开始，但我认为一旦你理解了这个概念，你就可以走得更远。这个例子基本上处理了 CSS 中基于数据属性的两种情况…一个类也可以。但是如果您使用同样的想法来设置一些 CSS 自定义属性会怎么样呢？你可以处理各种各样的卷轴变化。可能性是无限的。

##### [纯 CSS 刻字，一个又烂又好玩的创意](https://kennethormandy.com/journal/pure-css-lettering-a-bad-and-fun-idea/)

从实践的角度来看，这是一个可怕的想法，作者坦率地承认了这一点。作为一种艺术探索，一个学习项目，或者将代码作为一种创造性表达的形式来使用，这是一个非常酷的项目。

##### [自定义光标效果](https://tympanus.net/codrops/2019/01/31/custom-cursor-effects/)

这是一种介于不切实际和有用之间的东西。大多数情况下，您不需要创建自定义光标效果。但是在某些情况下，当有人接近某个特定的 UI 元素时，您可以使用这种技术来创建一个很好的微妙的注意力导向动画，因此它可能是添加到您的工具包中的一个很好的工具。

##### [浏览器 CSS 支持指南](https://www.smashingmagazine.com/2019/02/css-browser-support/)

与过去的 web 开发相比，当 IE 6 和 ie7 仍被普遍使用时，今天的跨浏览器支持问题似乎很小。但是，当你遇到一个阻碍你想做的事情时，这并不会让他们少一点沮丧。然而，真正酷的事情之一是，不仅浏览器在保持最新方面变得越来越好(evergreen FTW！)但是 CSS 语言本身也获得了越来越好的机制来处理缺乏支持的问题。这篇文章给你分解。

##### [CSS 网格:Excel 电子表格](https://codepen.io/oliviale/details/rPjgmB)

在“你能做的令人惊奇的事情的例子”类别中，检查这个代码笔！通过在 SCSS 的 500 行代码中实现对 Excel 的模仿，真正展示了 CSS grid 的威力，其中大部分代码都显示在工具栏和图标中。

#### JavaScript

##### [深入 JavaScripts 对象](https://blog.bitsrc.io/diving-deeper-in-javascripts-objects-318b1e13dc12)

在“掌握基础知识”类别中，这是对 JavaScript 中对象的许多细微差别的深入研究。那些你可能永远不会注意到的事情，直到你遇到一些奇怪的意想不到的行为或者想做一些与众不同的事情。

##### [未来 JavaScript:还缺什么？](http://2ality.com/2019/01/future-js.html)

Axel Rauschmayer 博士对 JavaScript 语言仍然缺少的特性和功能进行了深入的探讨，他是这种语言的主要作者/教师之一。更让我兴奋的是…他强调的特性已经在开发中，或者至少是 TC39 的研究。JavaScript 的未来确实是光明的。

##### [反应 v16.8:带钩的那个](https://reactjs.org/blog/2019/02/06/react-v16.8.0.html)

这是官方的！钩子是生活和生产准备就绪！现在，听起来在早期原型和最终版本之间确实有一些变化，所以如果你用早期版本的 hooks 编写的东西需要一些工作来更新，不要感到惊讶，但是在这个版本中，React 团队宣布它们已经可以用于生产。

##### [钩子对 Vue 来说意味着什么](https://css-tricks.com/what-hooks-mean-for-vue/)

对 React 中实现的 Hook 模式如何应用于另一个 JavaScript 框架的有趣探讨。这一点特别有趣，因为 Vues 有 mixin 的概念，它允许一些与钩子相同类型的代码重用，但是正如本文所展示的，钩子的可组合性带来了一些主要的优势。

##### [在 VueJS 中掌握计算属性](https://dev.to/treyhuffine/mastering-computed-properties-in-vuejs-4od0-temp-slug-3575460)

Vue 的反应性模型，其中计算属性是一个关键组件，可能是主要 JavaScript 框架中对 Vue 最独特的概念。我发现，随着我越来越多地使用 Vue，我对反应性越来越适应，我的代码就越具有声明性、可读性和可重用性。

#### 其他牛逼

##### [九头蛇-直播编码联网视觉](https://hydra-editor.glitch.me/)

我是从夏威夷 JSConf 的一次演讲中了解到这一点的，那次演讲让我大吃一惊。这让你可以用几行简单的 JavaScript 代码“重新混合”各种视觉效果——几何图形、你自己的摄像机、实时反馈。你可以直接跳进去开始玩，或者查看一下【twitter 上人们分享的例子！

##### [谷歌 Play 商店现在对渐进式网络应用开放😱](https://medium.com/@firt/google-play-store-now-open-for-progressive-web-apps-ec6f3c6ff3cc)

这是一件大事，让 PWAs 进入 play store 缩小了可发现性和易安装性之间的巨大差距，并为 PWAs 打开了一扇大门，它可以访问一些超越普通 web 应用程序的本机功能(目前看起来完全是通过意图)，而不必引入 Cordova 之类的东西。非常非常酷！

##### [2018 年最佳:UX 设计案例研究](https://www.casestudy.club/bestof)

对于那些更多的前端开发设计端的人来说，这是去年发生的 UX 设计案例研究的惊人综述。侧重于软件，但也有一些非常有趣的过程案例研究。

##### [启动 SpaceX GraphQL API🚀](https://medium.com/open-graphql/launching-spacex-graphql-api-b3d7029086e0)

这既是对 GraphQL 的一个令人惊叹的演练，也是设置 GraphQL 服务器背后的一些思考，而且也是一篇非常有趣和令人愉快的博文。作者的热情刚刚冒泡，他们给你探索的数据很吸引人。
