# 我经常使用 Bootstrap，我想知道没有它我怎么能写出好的 CSS？

> 原文:[https://dev . to/nsebhastian/I-used-bootstrap-too-frequency-and-I-wonder-how-I-can-write-good-CSS-without-it-4fnn](https://dev.to/nsebhastian/i-used-bootstrap-too-often-and-i-wonder-how-can-i-write-good-css-without-it-4fnn)

大家好！我想和大家一起讨论如何正确学习 CSS 而不使用框架。

你认为哪些教程或路线图会帮助使用 Bootstrap 的开发者学习如何制作自己的 CSS？

一些背景:我是一名在 React 和 Express 方面经验丰富的 JS 开发人员，但是在过去几年中，我的所有工作都方便地使用 Bootstrap 来搭建 CSS。现在我在想，如果没有它，我怎样才能学会创造出间隔良好的表单和响应性的布局呢？

此外，Bootstrap 有免费的 CSS 代码片段资源，如 Bootsnipp。有没有类似的网站但是是针对无框架 CSS 的？

我熟悉媒体查询、边距、填充和所有 CSS 基础知识，但是把它们放在一起做一个简单的响应网站是我已经忘记的事情。

谢谢你的帮助。戴夫真棒。🤟
