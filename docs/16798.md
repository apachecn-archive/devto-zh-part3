# MyClient 非常好的 Mysql 客户端

> 原文:[https://dev . to/jean carlosn/my client-very-good-client-to-MySQL-4mn 5](https://dev.to/jeancarlosn/myclient-very-good-client-to-mysql-4mn5)

MyClient 是 Mysql 的自动补全工具，看这个:

安装 Ubuntu:

步骤 1:键入以安装 mycli

 `sudo aptitude install mycli` 

下一步 2:在客户端输入

```
mycli -uroot

Version: 1.5.2
Chat: https://gitter.im/dbcli/mycli
Mail: https://groups.google.com/forum/#!forum/mycli-users
Home: http://mycli.net
Thanks to the contributor - Ted Pennings
mysql root@localhost:(none)> 
```
