对我来说，这是一个大大小小的一年。有了新工作，我个人变大了，但公开场合变小了，尽管我现在能够腾出更多时间参加今年的 DDD 会议和 T2 会议。

我还在适应经纪公司的生活，但这可能是因为对电影媒体来说，今年是令人惊奇的一年。

看看我的统计数据，很高兴看到 web dev 和 security 在我的日志中排名很高，因为这是我的博客开始的地方，尽管这些天也有更多的技术领导内容。
