我经常读别人怎么做的事，让我很高兴，昨天我读了两个布宜诺斯艾利斯社区的人讲了他们的观点，鼓励更多的人

*   [从前途未卜的工作中得到我所热爱的](https://medium.com/@MeliHildt/de-un-futuro-incierto-a-trabajar-de-lo-que-me-apasiona-7340245f35a7)

就在昨天，我和社区里的其他人交谈时，我想到了社区给我带来的一切，而不仅仅是布宜诺斯艾利斯或阿根廷……我指的是全世界的社区。

> 这不是□我什么都没隐瞒，只是一种我很久以前就渴望做的放电

我在特定情况下有几个观点……但我希望你读完这本书后，能想到它是什么让你的社区做的，为什么你会感激，如果你鼓励写它，我会很乐意读它-什么

从谷歌的大力推动，到专家…

**gdg 是 Google Developers Group** 【但没有谷歌】jeje，意思是说谷歌帮助一群人本着分享的精神来尝试技术……这将像是一次会议，但在大 g 的世界里更加平衡。

我把它放在第一位，因为通过 GDG，我的使命是分享更多的知识(无论多少)，我明白你必须分享你所知道的，并帮助别人，因为一切总是会回来。

但不仅仅是分享你学到的东西，如果不是认识了这么多人，以至于叫朋友太多(这比学什么都好)

另一方面，这是专家计划，GDE 或 Google Developer Expert，它是一个针对 Google 的非雇员计划，意思是它只适用于那些没有直接或间接与 Google 合作的人，如果你到目前为止还没有理解间接的话□我们收取准确的费用， 我们这样做是出于对社区的热爱:…成为 GDE 并不能使你比别人(我们都是平等的)更好。它使你更有知名度，这意味着你有责任帮助更多……正如我前面提到的那样，它的好处是它让我穿上了更多的社区 t 恤(如阿根廷所说， 我把 10)和更多的时间花在社区上……正如我告诉他们的那样，它使课程在讲西班牙语的最重要的平台上，而且每次我写或制作视频都会产生有趣的影响……但正如我告诉你们的，不是所有的数字，而是如果对我来说最重要的是世界各地的朋友， 我有很多朋友，每当我看到(有时一年只看一次)让那些时刻变得美好，不睡觉，或者乘 3 架飞机，等待移徙 1 个小时，在酒吧间工作，都让它变得有意义。

会议给了我许多宝贵的朋友和时刻，相信 NG-BAIRES 有着分享多个角度的想法，今天我们是一群长期以来一直带着 NG-BAIRES 的朋友，以便能够进行交谈或安排会议，让他继续每月生活……

另一方面，这是布宜诺斯艾利斯 Javascript 会议，在 slack 已经有 1900 多人，我很幸运地打电话给几个朋友，但重要的是，无论你问什么问题，它都致力于包容和帮助…

最后，我想谈一谈 ng-conf，这个世界上最大的角度会议，它是由社区组织的……这听起来很不可思议，而且我认为这种形式更酷……理解他们采取的每一个步骤都经过了彻底的修订，并且担心每个人都可能绝对舒服，这是非常困难的，有时很难处理 5 个人， 1500 还有很多 xd…但我明白到，在这种糟糕的情况下，有一个演讲人的位置，那些去活动的人最好告诉我们大家，我们都喜欢和演讲人，组织者，所有人……今天，经过大量的帮助，给了我一个机会成为新的组织者之一，这让我非常高兴，站在舞台后面

我写这封信是为了让你在读的时候能分享一些东西，鼓励你更多的…I wrote this so when you ' re reading this，you want to share something，encourage you more…

你总是会有批评，这是很昂贵的，但你必须了解哪些是建设性的，哪些不是……所以那些不是，虽然有时会痛的批评，你必须让它们过去很长时间。现在，建设性的批评必须以极大的爱来对待，并认识到总是可以改进的。
