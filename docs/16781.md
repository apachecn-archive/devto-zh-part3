# 如何重振事业？

> 原文:[https://dev.to/hwolfe71/how-to-revive-my-career-i25](https://dev.to/hwolfe71/how-to-revive-my-career-i25)

我不确定是复活还是开始更准确。我获得了计算机科学的学士学位，大约 10 年后，我在当地的一所社区大学学习了一些课程来提高自己的技能并学习一些新的技能，获得了计算机编程的理学学士学位。我只做过一份编程工作，当时我是一名软件工程师，使用 Visual Foxpro，然后是 Java。那份工作我干了大约两年。

我知道一些 Java，当我还是软件工程师的时候用过它，然后当我回到学校的时候，又学了几门 Java 课程。我也很熟悉 C 语言，因为我在攻读第一个学位时几乎只在高级计算机科学课程中使用过它。这些是我觉得最适合编程的语言。

我也上过 Pascal、C++和 VB 的课程，自学过一点 Perl 和 Ruby，也做过一些 Unix 脚本。我使用过 HTML 和一些 CSS，足以建立一个个人网页，然而，我已经多年没有更新它了。

我目前正在使用 Visual Foxpro 进行数据处理，但是我更愿意从事应用程序开发方面的工作，最好是在 Unix 或 Linux 环境中。不幸的是，我在这里没有看到太多的招聘信息，更不用说初级职位了。许多入门级职位需要比我更多的技能或经验，而且往往更适合全栈或前端职位，这两个职位都不符合我的兴趣。

我不知道该怎么办。搬迁不是一个选项，我真的负担不起再次回到学校。我曾考虑过尝试学习 Javascript 或 Python，但两者都不吸引我，我不想最终从事另一份我不开心的工作。

我愿意接受任何建议。谢了。
