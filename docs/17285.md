# 开始新的一年(2019)#讨论

> 原文:[https://dev.to/robertfish/starting-the-new-year2019-215k](https://dev.to/robertfish/starting-the-new-year2019-215k)

我想以说新年快乐开始！

2018 年对我和许多其他人来说是疯狂的一年。我花了一年的大部分时间，作为一名自由职业的网络开发人员，开始了我自己的生意，在大部分时间里，我的生意非常好...由于收入不稳定，这无疑是紧张和疯狂的时期，但现在是改变的时候了，所以我采取了一个飞跃。一家年轻的初创公司找到我，让我成为一名 SWE，来构建他们的产品和基础设施。现在回想起来，我很感激我抓住了那个机会加入他们的团队。

现在是 2019 年，这一年从我完成所有上述产品在 Q1 上市的工作开始。我对接下来发生的事情毫无准备，但我可以说的是，我已经准备好了，并且很期待接下来会发生什么。
