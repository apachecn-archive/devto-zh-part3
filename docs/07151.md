# Android 教程:构建和保护您的第一个应用程序(第 2 部分)

> 原文：<https://dev.to/auth0/android-tutorial-building-and-securing-your-first-app-part-2-193a>

在本教程的第二部分，您将学习如何将您的 Android 应用程序与后端 API 集成。对于初学者来说，您将构建一个简单的 REST API(您将有不同的选择来实现这一点)，然后您将调整您的应用程序，用这个 API 提供的列表替换静态的待办事项列表。最后，您将利用应用程序与 Auth0 的集成，使其使用私有(也称为安全)端点来保存新的待办事项。
