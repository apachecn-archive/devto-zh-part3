BinStorm 是一个 React 本机样式实用程序，用于快速构建自定义用户界面

它不同于 Shoutem、Native Base 或 UI Kitten 等框架，因为它不是 UI 工具包。

它没有默认的主题，也没有内置的 UI 组件。

但重要的是，它包含了将帮助您构建应用程序设计的所有助手。而不是预定义的组件和主题来控制应用程序的外观。

如果你正在寻找一个带有预建部件/组件的框架来构建你的应用，BinStorm 可能不适合你。
