## [](#2-flexsearchjs)[2。FlexSearch.js](https://github.com/nextapps-de/flexsearch)

> Web 上速度最快、内存最灵活的零依赖全文搜索库。

[![FlexSearch.js Github Repository](../Images/ad0cf31480ae82557e86f53ee4727e11.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--3MhI1mbl--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://devstuffwithjpada.com/media/posts/13/responsive/Screen-Shot-2019-02-03-at-6.31.31-AM-xl.png)

FlexSearch 是一个新的 JavaScript 库，几乎包含了文本搜索功能的所有内容。它功能齐全，支持异步搜索、部分匹配、语音匹配等等！显然它也真的真的很快，但像往常一样，对所有事情都要有所保留，实际上亲自尝试一下，看看他们的测量结果是否真的符合要求。

## [](#4-graphql-designer)[4。GraphQL 设计器](http://www.graphqldesigner.com/)

> 一个开发者 web 应用程序工具，可以快速原型化一个带有 React 的 GraphQL 全栈实现。

[![GraphQL Designer](../Images/510b8c9ab1ca87bc4e8a3383d5506280.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--dz6p6-YX--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://devstuffwithjpada.com/media/posts/13/responsive/Screen-Shot-2019-02-03-at-6.42.55-AM-xl.png)

选择一个数据库，创建一个模式，导出你的代码，瞧！GraphQL Designer 是一个快速的 GraphQL API 设计和原型工具，它允许您选择一个数据库，创建一个模式，然后将您创建的模式转换成工作代码！您将下载数据库模式、GraphQL 模式和向 GraphQL API 发出请求的客户机代码。非常适合在黑客马拉松中进行原型开发，或者如果您正在构建简单或直接的应用程序。在任何情况下，如果您使用 GraphQL，您可能希望了解这一点。

## [](#5-react-testing-library)[5。React 测试库](https://testing-library.com/react)

> 简单完整的 React DOM 测试工具，鼓励良好的测试实践

[![React Testing Library by Kent Dodds](../Images/921a8ee3297ff41619b150d464072de5.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--e5ReiYkv--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://devstuffwithjpada.com/media/posts/13/responsive/Screen-Shot-2019-02-03-at-6.49.48-AM-xl.png)

多产的 React 和 JavaScript 开发人员 Kent Dodds 创建了这个库作为酶的替代品。它给了我们较少的灵活性，因为它固执己见，但这也意味着我们将通过使用它来实现最佳实践。更少的选择意味着我们上吊的绳子更少。Kent 已经在 YouTube 上有一个关于如何使用它的教程视频，但我想我会用菲律宾语为我所有的菲律宾开发人员同行报道它。绝对期待！
