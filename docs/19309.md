# 代码的来临是活的！

> 原文：<https://dev.to/rpalo/advent-of-code-is-live-3j7l>

又到了一年中的这个时候:代码的出现。该网站在美国东部时间午夜上线(这意味着，在西海岸，我在睡觉前做了一点编码)！

《代码的来临》是我非常喜欢的编码挑战来源之一。挑战是喜庆的，有趣的，有趣的。这也不仅仅是另一轮“罗马数字”或“墙上的 99 瓶牛奶”。这些都是手工制作的、手工的、编码挑战，你在别的地方看不到。他们每天出来两个，直到圣诞节！这是一项任务，你需要在接下来的 25 天内完成所有的挑战来拯救圣诞节。

这真的是一个很好的方法，可以让你在一门新的语言中获得持续的、引人入胜的练习。今年，我一直在努力学习 Rust，所以我会尽我所能完成 Rust 中的挑战，并分享我在这个 repo 中的进展[(剧透在场！).我很乐意任何想放弃一些知识的 Rust gurus 看看我的代码，并帮助我变得更好。](https://github.com/rpalo/advent-of-code-2018)

如果你正在学习 Python 或 Ruby，并且想让我看看你的代码，请告诉我。我很乐意尽我所能帮忙。😀

无论如何，对于那些庆祝它的人，圣诞快乐！节日快乐！还有快乐编码！
