Dev Chats 第二季又回来了！在这一系列中，我大约每周都会与一位出色的开发人员或技术人员交谈。你可以在这里阅读更多。如果你知道下一个我应该和谁聊天，请告诉我。

### [](#introduce-yourself-who-are-you-where-do-you-work)自我介绍！你是谁？你在哪里工作？

我叫 April，是 Vogue & Code 的首席执行官和创始人，这是一个帮助非传统背景的人过渡到科技行业的平台。我目前在微软工作，担任微软文档的高级项目经理，帮助开发人员构建交互体验。

### 是谁或什么让你开始编程的？

我在大学时学过语言学，因为我很好奇，想更多地了解人们说话的方式。随着我开始学习更多关于句法的知识，我对计算语言学产生了兴趣。本科毕业后，我梦想成为一名计算语言学家。我做了大量的研究，看看哪种编程语言可以帮助我实现这个目标，我意识到 Python 就是答案。这真的为人工智能聊天机器人打开了一个世界——这现在是我的强项。

### 你认为科技行业在未来几年会发生怎样的变化，这会对你的角色产生怎样的影响？

人们越创新，就越需要为所述技术生产内容。我的专业工作和副业都专注于产生技术内容，这在我的书中是双赢的。

### 迄今为止，在你的软件生涯中，你学到的最难的一课是什么？

有很多东西要学，因此，慢慢来是可以的。

### 对于成功的软件职业生涯，你的第一条建议是什么？

找到你的定位，了解你的定位中的相关话题。

### 工作之外，你有什么爱好吗？你认为它们对你的科技事业有任何帮助吗？

我为乐趣而缝纫，实际上也为乐趣而编码(因为我并不是为我在微软的角色而专业地做这件事)。与我在微软的工作相比，缝纫是一个截然相反的工作。但是，我对编码的兴趣帮助很大。我在熟悉开发者社区方面做得很好，这帮助我对开发者产生了很多共鸣。

### 你会向那些想走上与你相似道路的人推荐哪些书籍/资源？

老实说，媒体上的科技文章是获取他人观点的良好开端。虽然没有两个人走在同一条科技道路上，但是除了有用的建议之外，了解一个人可能期望什么是很好的。
