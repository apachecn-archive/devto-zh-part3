# 从头开始学习 R 编程:第一天在 Windows 上安装

> 原文:[https://dev . to/maheshkay/learn-r-programming-from scratch-day-1-install-on-windows-4 lad](https://dev.to/maheshkay/learn-r-programming-from-scratch-day-1-install-on-windows-4lad)

和我办公室的许多朋友一样，我决定学习 R 语言。那么为什么突然需要 R 呢？我进入前端和测试了一段时间。但是现在这还不够。我的公司已经决定进入数据集。他们正在对大量实习生和现有员工进行数据科学培训。对于社交媒体和特定分析公司中的许多公司来说，数据似乎是新的方向。

然而，我唯一的问题是，与我的大多数前辈不同，我在工作场所可以使用窗户。根据目前的人力资源公告，我们不会很快切换到 linux 或 mac。所以我迈向 R 的第一步是从在 Windows 中安装 R 和 RStudio 开始。我有两个选择。一是安装 Cran R 和 RStudio。

[](https://cran.r-project.org/)

 **这是你可以下载 r 的官方网站，你会得到一个控制台二进制文件和一个简单的编辑器。无论您选择哪种方式来运行脚本都是可行的。这是免费下载，不像 Rstudio 也有一个高级版本。如果你使用了这个，你就会明白如何从头开始下载软件包，这比使用官方的 R installer 要简单得多。如果贵公司不想使用 Rstudio，我会推荐 CRAN-R。
