## [](#learn-machine-learning-with-python)用 Python 学习机器学习

首先，你需要学习 Python。Python 广泛应用于数据科学、IOT、机器学习、Web 应用或游戏开发。

与其他编程语言不同，Python 的语法简单易懂。所以作为一个初学者，这将让你快速掌握基础知识，减少精神紧张，你可以更快地升级到高级机器学习主题。

有许多精彩的免费在线资源可以帮助你开始学习机器学习，但如果你在开始学习时遇到困难，我会建议你参加任何 Python 课程，在我看来，侧重于数据科学的入门课程要好得多。

## [](#brush-up-your-math-skills-)刷刷你的数学技能！！！
