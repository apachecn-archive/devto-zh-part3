# 10 必须有 Wordpress 插件-2020 更新

> 原文:[https://dev . to/sebtoombs/10-must-have-WordPress-plugins-updated-2020-2k5e](https://dev.to/sebtoombs/10-must-have-wordpress-plugins-updated-2020-2k5e)

## 我们列出了 2020 年将使用的顶级 Wordpress 插件

作为 Wordpress 开发者，你会很快发现有很多插件可以帮助你开发你需要的功能。然而，在构建 Wordpress 网站时，有一些插件会让你的生活变得更容易，你会发现自己会一遍又一遍地重复使用它们。我们认为我们已经列出了用 Wordpress 开发时必须拥有的 10 个插件。

这 10 个插件是我们经常使用的包，因为它们工作得很好，帮助我们以快速灵活的方式实现目标。我们不建议为了安装而安装所有的软件！不是每个项目都需要相同的工具，但是这些工具比其他工具更方便。看看他们！

请注意:我们不是这里提到的任何插件作者赞助的，也没有从他们那里收到任何佣金或回扣。我们简单地把我们日常使用的插件列表放在一起，发现非常有用。

## [](#1-advanced-custom-fields)1。高级自定义字段

高级自定义字段是迄今为止我们日常使用的最有用的插件。更重要的是插件的高级版本 ACF Pro，它提供了一些我们不能没有的功能(repeater & flexible content )!

如果你以前没有使用过 ACF，它是一个插件，可以轻松地将 Wordpress 从一个简单的博客平台变成一个 CMS(内容管理系统)。ACF 基本上做到了它在 tin 上所说的:提供定制字段。"那么，这听起来很简单，是怎么回事呢？"我能听到你在问。ACF 给了你非常容易使用的工具来添加自定义字段到你的任何和所有的文章类型，页面，甚至菜单。如果您还没有开始使用 ACF 的话，我们强烈建议您查看我们的快速教程[。](https://nimblewebdeveloper.com/blog/tutorials/how-to-use-acf-wordpress/)

查看 Wordpress 插件库中的[高级定制字段](https://wordpress.org/plugins/advanced-custom-fields/)。

## [](#2-w3-total-cache)2。W3 总缓存

缓存——现在它是任何生产站点的必备工具。谷歌和移动用户希望你的网站几乎瞬间加载！当你与有些笨重的 Wordpress 内部组件以及潜在的糟糕的插件和主题进行斗争时，你如何实现这一点呢？缓存！当然，有很多方法可以实现一个快速的站点，也有很多方法可以实现一个缓存解决方案，但是我们发现 W3 Total Cache 是最简单、设置最快速、最可靠的解决方案。我们还有一个 W3 Total Cache 快速入门指南。

在 Wordpress 插件库中查看 [W3 总缓存](https://wordpress.org/plugins/w3-total-cache/)。

## [](#3-ninja-forms)3。忍者形态

表格和表格提交糟透了。真的！处理用户输入、验证、提交错误、保存提交内容和发送电子邮件(我们稍后会谈到这两项)是让我们所有 NimbleWebDeveloper 人员在必须处理它们时都感到苦恼的任务。进入忍者形态。如果你正在建立一个需要询问表格、用户提交、注册或其他任何东西的 Wordpress 站点，Ninja Forms 确实是一个必备的插件。Ninja Forms 非常健壮，我几乎从未见过它倒下，超级容易使用，它记录提交和发送电子邮件。对于我们为企业建立的 99%的网站，Ninja Forms 是我们的首选表单解决方案。

在 Wordpress 插件库中查看[忍者形态](https://wordpress.org/plugins/ninja-forms/)。

## [](#4-post-smtp-maileremail-log)4。发布 SMTP 邮件程序/电子邮件日志

继 Ninja Forms 之后，是 Post SMTP Mailer 和电子邮件日志。如果你的 wordpress 站点正在发送电子邮件，这是一个绝对必须的。Post SMTP Mailer 记录从您的站点发送的所有电子邮件，还允许您配置您的站点以使用外部电子邮件传递服务。如果你正在为客户建立网站，这些功能真的很棒。客户*讨厌*邮件丢失！使用这样的插件意味着所有发出的邮件都会被记录，而使用外部邮件发送服务提供商意味着你的邮件很可能会到达正确的邮箱，而不是垃圾邮件。

在 Wordpress 插件库中查看[发布 SMTP 邮件/电子邮件日志](https://wordpress.org/plugins/post-smtp/)。

## [](#5-woocommerce)5。网络商务

好吧，我在这里作弊了一点。这个列表本来是要列出几乎可以在任何安装上使用的插件，但是很难不包括 Woocommerce。如果你没听说过，Woocommerce 是 Wordpress 的一个电子商务插件。它允许你的网站销售实物和数字商品，它超级可靠和灵活，并有大量的第三方支持和插件。Woocommerce 很容易配置，但仍然可以扩展以实现您需要的几乎任何结果。

在 Wordpress 插件库中查看 [Woocommerce](https://wordpress.org/plugins/woocommerce/) 。

## [](#6-code-snippets)6。代码片段

对于 Wordpress 开发者来说，代码片段是一个很棒的插件。我们经常发现自己不得不对别人构建的客户端站点进行小的修改，或者需要添加一个与主题无关的小片段。你把这个代码放在哪里？在**functions.php**？那么，如果我们改变了主题，但仍然需要功能(例如，ajax 或 REST 处理程序)。好的，让我们使用一个插件。当然，但是用一个插件来写 15 行代码似乎有点大材小用了(举个例子)。或者，您可能确实在 functions.php 丢弃了代码，但现在某处出现了冲突，您想要关闭您添加的一些代码片段。祝您在注释掉函数文件中的部分时愉快。

出于这些原因，我们经常使用代码片段，因为它允许我们添加主题之外的代码片段，可以通过单击按钮独立打开和关闭。

在 Wordpress 插件库中查看[代码片段](https://wordpress.org/plugins/code-snippets/)。

## [](#7-clearfy)7。清除

Clearfy 是 Wordpress 的一个很棒的插件，它允许你删除很多你在 Wordpress 中可能不需要的功能。它有一些简单的搜索引擎优化和安全调整，可以结合和缩小 HTML，样式表和 javascripts。Clearfy 对于加快 Wordpress 站点的速度和删除可能会降低站点速度的功能非常有用。有相当多的选项，但一键安装使安装变得非常容易。

在 Wordpress 插件库中查看 [Clearfy](https://wordpress.org/plugins/clearfy/)

## [](#8-custom-post-type-ui)8。自定义帖子类型用户界面

自定义 post 类型 UI 是另一个很好地实现了标签上所说的功能的 UI。它提供了一个方便易用的用户界面来添加和管理定制的文章类型(和分类法)。当然，你可以用几行代码实现同样的事情(一旦你找到了正确的代码片段),然后花点时间找到正确的选项来得到你需要的东西。CPTUI 为您提供了一个快速添加和编辑自定义文章类型的方法。

在 Wordpress 插件库中查看定制文章类型 UI 。

## [](#9-duplicator)9。复印机

我们发现自己需要移动网站——从一个生产位置到一个新的开发实例，从开发到一个新的生产服务器，从一个服务器到另一个服务器。有时我们需要克隆一个不是我们建立的，也没有 FTP 访问权限的站点。Duplicator 将 Wordpress 文件和数据库捆绑在一起，压缩这个包，然后提供一个简单的安装程序。下载安装程序和文件包，然后将它们上传到新位置并运行安装程序。轻松点。有很多方法可以迁移 Wordpress 安装，但是 Duplicator 是大多数时候最简单的方法。

在 Wordpress 插件库中查看[复制器](https://wordpress.org/plugins/duplicator/)。

## [](#10-yoast-seo)10。Yoast SEO

最后一个是 Yoast SEO。Yoast 旨在成为一个全方位的解决方案，帮助提高您的搜索引擎优化。显然，Yoast 无法取代真正的 SEO 专家，但它提供了您需要的工具来帮助实施一些 SEO 策略。Yoast 拥有编辑页面和文章标题和元描述、生成站点地图、洞察关键词密度、聚焦关键词和可读性分析等工具。

在 Wordpress 插件库中查看 [Yoast SEO](https://wordpress.org/plugins/wordpress-seo/) 。
