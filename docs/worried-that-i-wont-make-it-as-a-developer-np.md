# 担心我不能成为一名开发人员

> 原文：<https://dev.to/radu_p/worried-that-i-wont-make-it-as-a-developer-np>

# 第一步

这一切都始于 2018 年 7 月，当时我在 AngelList 平台上注册，一切看起来都很棒，因为我在 16 岁时获得了第一份实习。这是一份无薪实习，但它并没有困扰我，我已经有了一些自由职业者的项目，面试并不困难，因为面试官很友好。

# 担忧的最初迹象

我学到了很多东西，我仍然在实习中学习，但最近我开始考虑实习(将于 12 月结束)后我应该做什么。最理想的情况是找到一份初级前端开发人员的工作，但我有点泄气了。我不是最伟大的开发者，我仍然是 Vue 的初级/中级用户，我仍然需要练习我的算法和 UI/UX 原则。

我曾有过这样的想法，一旦我进入这个行业，就很容易获得进一步的工作，但现在对我来说似乎不是这样。实习的面试并没有那么难，主要是因为这是一个没有报酬的实习，我在想下一次工作面试者(如果我一开始就参加面试的话)不会雇佣一个几乎没有任何开发经验的孩子。

# 我该怎么办？

实习结束后，我找到工作的机会有多大，我每天都在不断学习，我不会很快停下来，我看到很多工作都需要反应，我正着眼于在周末有时间时和实习结束后学习，但我仍然担心我不会成功。我想进一步增加我的 portofolio，目前我有几个 spa 和几个 html/css 页面，但我想用 Vue 和 React 制作约 5 个项目，我将能够自豪地向任何潜在的面试官展示。

你会给我什么建议，让我去实习，或者至少让我成为一个自信的、全面的更好的开发人员？
