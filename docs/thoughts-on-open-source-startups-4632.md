## 简介

开源对我来说一直有着特殊的地位。

自从我在一所技术学校开始学习 Java 作为我的第一门编程语言。

当我 16 岁的时候，我把它作为计算机编程导论课程的一部分来学习。

那是我接触 Java 世界的时候，在那里开源被认为是我的一大福音。

当时由于我的学校与太阳微系统公司关系密切。

我们接受了 Solaris 管理(UNIX 的一种变体)的基本培训，这在当时的数据中心非常流行。

## 带着信仰的飞跃去开源

由于培训，我对它的兴趣增加了，这让我可以玩开源软件。

这让我在笔记本电脑上安装了 Ubuntu，并选择了 LAMP stack 作为我最后一年项目的一部分。

尽管缺乏精通 Java 的讲师的支持。我完成了我的项目。

由于从论坛到堆栈溢出答案的开源社区的大量知识和支持。

## 开源启动

那是开源，今天，由于云计算和创业公司的兴起，开源变得很酷。

除了创造 Ubuntu 的标志性的红帽或 Canonical，谁被认为是开源运动的老卫士。

最近几年，我在听一集软件工程师每日采访来自 **OSS capital** 的 **Joseph Jacks** 时，遇到了一个新名词叫做**开源创业**。

开源创业公司(OSS)专注于生产开源软件。他们通常驻留在企业软件空间中，以解决相关行业中的特定问题。

像 Cloudera，MapR，Kong，Redis Labs，Nginx，MongoDB，Pivotal Software，Gatsby 或者 Docker 这些创业公司都是我想到的。

近年来，OSS 在业界&在开发者中获得了牵引力。

由于商业模式的改变或最近被 IBM 或微软这样的公司收购，这些操作系统成了新闻焦点。

与开发者社区和开源创新保持联系。

## 让开源成为你职业生涯的一部分

我个人觉得只要你是开发商。你将直接或间接地参与开源运动。

传播开源技术的意识，或者将开源技术作为开发过程的一部分。

对我来说，去年由于**黑客节**，我只是直接为开源项目做贡献，比如 **FreeCodeCamp** 。

但是我参与了开源社区，帮助组织了几年的 PyCon。

当我还是一名大学学生的时候，我为新加坡的 Python 用户组工作。

## 开源创业前景光明？

对我来说，我认为未来是光明的，因为会有越来越多的创业公司。

谁将创造像盖茨比或孔这样的开源创业公司。

除此之外，我开始看到越来越多的团体参与到帮助提高对 OSS 的认识和资助中来。

从开放源代码来看，Patreon 项目如 Open Collective、Linux Foundation 的 CommunityBridge 和风险投资公司如 OSS Captial。

## 结论

由于近年来微软对开源的参与，开源已经走过了从讨厌硬核到喜欢微软的漫长道路。

我看到自己正在积极参与开源运动，并在其中发挥作用。

也许有一天，我会创建自己的开源软件创业公司。

因为随着我越来越深入地研究，我看到了大量有助于开源和创业的机会。

请留下你对开源创业公司的看法。我很想知道更多关于你的想法。
