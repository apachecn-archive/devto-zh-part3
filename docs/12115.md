# 实际的用户反馈是最有价值的

> 原文：<https://dev.to/eekayonline/actual-user-feedback-is-the-most-valuable-kind-2c47>

[![Real user feedback is the most valuable kind](img/72ef37cb5f9e4c79bc051a675fc01f16.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--YIL5rip0--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/stx7io83yiw8riv084vc.PNG)

没有比实际反馈更有价值的反馈了

这就是保罗·格拉厄姆的这句话给人的深刻印象:你需要发布你的产品，并让用户测试它。

在发货之前，你永远不会知道你的产品提供的价值是否足以解决任何问题。

当人们在生产中使用你的产品时，就会产生摩擦:

感觉不到的微小事物产生的摩擦。
屏幕摩擦或信息关闭。
摩擦，因为与用户从你的产品中获得的收益相比，感觉有点太贵了。

你需要真实的用户反馈。正是这些反馈给了你必要的洞察力来优化你的产品。

所以尽快推出你的产品。比起一开始就有自己的假设，你更有可能赢得彩票。

和往常一样，请记住你需要创造价值，并以#valuefirst 为标签。检验你是否为用户提供了足够价值的最好方法是运送你的产品。

让用户决定它是否值得为之付费。

努力编码，努力运输✌🏻
