# 你的技术演示分享太多了吗？

> 原文:[https://dev . to/jean Carl/do-your-tech-demos-share-too-more-124h](https://dev.to/jeancarl/do-your-tech-demos-share-too-much-124h)

今天，我们的个人隐私正受到多方面的挑战。从你的家庭住址、电话号码、友谊和秘密等个人信息被曝光……人们可以发现很多个人的小细节。我们不要谈论社交媒体和位置共享。

职业上是否分享太多？

我偶尔会发布一些教程，或者是截图，或者是视频，展示如何做事情。我还在集会和会议上做了数百次现场演示，面对的是那些可能会无意中为我展示的东西拍照的人。然后是非正式的屏幕共享，人们在空闲时间打电话给他们看我电脑上的东西。在大多数情况下，没有恶意...但是如果有足够多的碎片...有人可能会利用这些造成伤害。

你的屏幕显示的信息太多了吗？从 API 密匙，到账号，到关注者列表，我发现有些东西在我不希望的时候是可见的。

注意:在写这篇文章的时候，我认为包括截图会使图片完整...但是当我开始截图这些例子时，我意识到我会分享同样的信息，我不想在互联网上分享。我甚至试图模糊这些部分，但有些例子几乎模糊了整个屏幕。抱歉，我们得根据文本来做。我鼓励你打开网站自己看看。

# [](#tabs)选项卡

对于一心多用的人来说，标签是最好的发明。在标签页中打开一个页面，您可以返回到该页面。你的公开标签分享你的什么？你是 Gmail 用户吗？如果你有比这更少的标签，页面标题共享什么？

[![](../Images/59edd5fee30100a1d0e11601ca1cd99f.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--WweHGAoj--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/n7egmkmk55ulbhkf1he9.png)

仅打开您经常使用的主要选项卡。打开一个新的 web 浏览器窗口并重新开始。

# [](#email)邮件

有多少次你看到有人演示时意外打开他们的电子邮件收件箱，显示他们通信的人的名字，主题和电子邮件预览可能有点太多了？关闭标签页，稍后再回复邮件。

# [](#github)Github

哦，对了，我有些代码可以给你看看。键入[github.com](https://github.com)并按回车键。Github 仪表盘，甚至是你的 Github 回购列表都会显示私人回购。如果你正在进行的秘密项目就列在那里呢？

# [](#bookmarks)书签

书签工具栏也是另一个快速到达常见目的地的便捷工具。但是这些书签是像 Project X roadmap 一样共享内部资源吗？添加非描述性文件夹，并在文件夹中放置敏感书签。

# [](#api-keys)API 键

如果您创建云资源或使用 API 密钥，您最终将需要复制 API 密钥或证书。在截屏凭证页面之前，我进入开发人员工具并检查元素。我通过完全随机化来修改 API 键。然后我截图。它看起来像真的，但是如果你试一下钥匙…它就不起作用了。另一个技巧是在暴露密钥后立即旋转或使其无效。

# [](#websites)网站

像谷歌地图和亚马逊这样的网站会显示你的行踪信息。打开[maps.google.com](https://maps.google.com)，它会将地图居中显示你的电脑所在的位置。亚马逊分享你的邮政编码和你以前看过的产品。

# [](#files)文件

如果您在电脑上保存或打开文件，请注意您正在显示的内容。打开一个下载的文件可能意味着显示你的下载列表，显示你过去下载的内容。如果你上传文件，你会暴露你正在做的秘密项目的文件或文件夹吗？在 Mac 上，您可以点按“隐藏”来折叠左侧边栏中的列表。

# [](#notifications)通知

有多少次你一直看着有人在场，看到一个通知弹出。例如松弛时间、电子邮件、文本消息或日历提醒？如果你共享一部手机，你甚至可以共享一个新号码打来的电话。在 Mac 上，您可以按住 Option 键点按右上角的通知图标。
