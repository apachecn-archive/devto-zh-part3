## ScaleGrid:自动化云中的数据库管理

数据库即服务(DBaaS)格局在不断发展，令人兴奋的新数据库进入市场，新的云供应商不断涌现。现有的云供应商也在发布新的硬件和软件平台。服务供应商现在面临的挑战是不断发展他们的平台，以利用所有这些变化为客户提供最佳服务。 [ScaleGrid](https://scalegrid.io/) 提供全面管理的 DBaaS 解决方案，被包括 ups、戴尔和 Adobe 在内的成千上万的开发人员、初创公司和企业客户所使用。ScaleGrid 平台在公有云和私有云上都支持 [MongoDB](https://scalegrid.io/mongodb.html) 、 [Redis](https://scalegrid.io/redis.html) 、 [MySQL](https://scalegrid.io/mysql.html) 和 [PostgreSQL](https://scalegrid.io/postgresql.html) ，包括[亚马逊 AWS](https://scalegrid.io/mongodb/aws.html) 、[微软 Azure](https://scalegrid.io/mongodb/azure.html) 、[数字海洋](https://scalegrid.io/mongodb/digitalocean.html)和 [VMware](https://scalegrid.io/platforms/enterprise-on-premise.html) ，并处理任何规模的所有数据库操作。“这大大减少了跨不同平台管理数据库的时间负担，并降低了不兼容和数据库蔓延带来的问题风险，”首席执行官 Dharshan Rangegowda 说道。

ScaleGrid 面向在公共云或私有云上部署数据库的 SMB 和企业客户。“ScaleGrid 是一个多语言、多云解决方案，允许我们的客户在他们选择的云上部署他们选择的数据库。我们还可以无缝地将数据库从一个云迁移到另一个云，使他们能够防止和保护其基础架构不受供应商或云的束缚，”Dharshan 提到。

ScaleGrid 是唯一允许客户对其数据库保持完全超级用户管理控制的 DBaaS 解决方案。这使他们的客户能够使用他们现有的脚本和工具，并与他们过去合作过的其他数据库顾问合作。

### 标度网格的概念

ScaleGrid 的[开始可以追溯到 2012 年。Dharshan 曾是微软的首席开发主管，负责微软的虚拟机管理器，后来转到 Azure，负责下一代数据中心软件网络。Dharshan 对云计算充满热情，他很快就看到了企业在云中的未来。考虑到部署和管理生产数据库集群的极端复杂性，他希望扩展云模型的简单性来帮助解决这个问题。ScaleGrid 以简化公共云和私有云上的数据库管理为使命开始了它的旅程。“我们实现这一使命的第一步是推出公共数据库即服务(DBaaS)解决方案，然后最终将其扩展到不同的云提供商和私有云，”他说。](https://scalegrid.io/about-scalegrid.html)

[![SiliconIndia Cover Story - ScaleGrid Automating Database Management In The Cloud](../Images/26a32248d765bd78828db8c2e1816901.png)](https://startup.siliconindiamagazine.com/ranking/10-best-startups-in-cloud-2018-rid-541.html)

自 2012 年从 [AngelPad](https://angelpad.com/) 加速器毕业以来，ScaleGrid 已经走过了漫长的道路。“我们的部门是工程密集型的，建立一个好的工程组织需要一段时间。我们现在是一个分布在加州帕洛阿尔托和印度班加罗尔的分布式工程组织，”Dharshan 说。

ScaleGrid 致力于数据保护，并为其基础设施整合了安全最佳实践，以确保客户数据的安全性。该公司使其客户能够通过[自带云](https://scalegrid.io/pricing.html#section_pricing_byoc) (BYOC)计划，在 AWS (VPC)和 Azure (VNET)上部署他们自己的虚拟专用云中的数据库服务器。

### 前进的道路

仅在 2018 年，ScaleGrid 的新客户增长率就超过了 65%，自 2014 年以来，年均增长率一直保持在 58%。ScaleGrid 已经在 70 多个国家部署，它支持全球 52 个不同的数据中心。

在接下来的时间里，ScaleGrid 团队希望让各种规模和类型的企业能够在任何云(包括私有云)上管理任何数据库。“我们最近针对私有基础架构环境推出了一款[企业数据库即服务](https://scalegrid.io/platforms/enterprise-on-premise.html)解决方案，我们很高兴能够拓展这一市场，帮助使用内部和私有云服务器的公司优化耗时的数据库管理任务，并降低总拥有成本(TCO)，”Dharshan 总结道。

## 2018 年硅谷十大最佳云创业公司

在高度互联的企业中，很难想象一个不受网络攻击和数据泄露影响的世界。对于现代商业组织来说，一个强大、资源丰富的 IT 基础设施是当务之急。开发和维护这一系统需要巨大的成本，这也是企业尽管拥有训练有素的技术资源却落后的主要原因之一。近年来，基础设施和底层云应用的改造取得了巨大进展，被认为是下一波创新浪潮。满足不断增长的需求对于所有组织来说都变得至关重要，这也是云初创公司发挥作用的地方。这些初创公司正在转变业务，简化企业应用交付。被吹捧为业务运营方式中最重要的技术突破之一，在激烈的竞争中，这些初创公司提供了许多好处，如灵活的定价模式、轻松的可扩展性、资源的最佳利用、移动性、安全性、生产率和 ROI。毫无疑问，云计算正在推动当今世界，从火箭船技术到银行和建筑等传统领域的长期参与者，所有行业都在朝着这一变化前进。
