# [](#my-follower-counts)我的追随者计数

在这里发布和在 Twitter 上分享我的帖子真的增加了我在这两个网站的粉丝数量。我在 DEV 上有 2061，在 Twitter 上有 238。我不记得当我的挑战开始时，我在 Twitter 上有多少粉丝，但不是太多。我在四月中旬的某个时候开始开户。

# [](#the-unexpected-side-effects)意想不到的副作用

我加入 Twitter 主要是为了建立关系网。我觉得我错过了了解我所在地区的开发者的机会，而 Twitter 似乎也是一个了解正在发生的事件的简单途径。事实证明，很多非常酷和鼓舞人心的开发者都在那里。我认为我在 Twitter 上看到的最酷的事情之一是#100daysofcode 标签。我为那边的技术社区和教育环境感到骄傲。

Twitter 是一个很好的想法。在各种讨论中互动，阅读来自经验丰富的开发者的智慧，以及与各种各样的酷人 e-meeting，这些真的改变了我的生活。

*   我发现了我对网络无障碍宣传的热爱
*   今年夏天，我将参加多年来的第一次会议
*   我发布了 textua11y，这是我的第一个 React 应用
*   我在我所在的地区发现了大量的聚会机会

总的来说，这个博客挑战是我做过的最好的决定之一。如果你是一个有抱负的作家，我最好的建议就是去做那该死的事情。我不知道我是否会每天都在 DEV 网站上发帖，但我计划继续每天在我的个人博客网站上发帖，以摆脱有时的技术空间。

最后，感谢来到这里。你帮了我们一把。希望你能留下来看更多！我有很多梦想。😊
