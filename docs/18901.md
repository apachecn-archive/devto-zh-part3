# dev.to 内部的导师计划是如何运作的？

> 原文：<https://dev.to/equiman/how-it-works-mentorship-program-inside-devto-3h2h>

我在个人资料设置中看到一个**导师**部分。我认为这是一个伟大的想法，但我找不到一个选项来寻找导师？我是不是漏掉了什么类似的东西？

例如，我想交换说我的母语西班牙语的导师，同时有一个导师练习说英语。

这将是一个很酷的有活动/主题可用性的日历，人们可以在 dev.to 内部安排这些会议作为网站的一部分。

我知道游戏化不包括在 dev.to 的哲学中，但是知道一个用户在其他人身上投入的指导和时间的数量是很好的。在你的简历(履历表)上展示这些指标会很好。

像这样的工具可以给我们更多的关注和访问，并且可以成为一个很大的社区帮助。
