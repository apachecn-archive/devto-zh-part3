# 苹果应该把可用性要求放在广告上吗？

> 原文：<https://dev.to/_bigblind/should-apple-put-usability-requirements-on-ads-11ld>

网上有很多关于苹果公司如何利用他们的权力作为应用商店的看门人的讨论。但最近我一直在想，他们或许应该在一个领域发挥更大的力量，那就是广告。

我认为如果他们拒绝广告不符合这些规则的应用程序会很好:

*   插播广告中的按钮必须遵循 HIG，它们必须足够大并且具有良好的对比度。
*   插播广告中的按钮必须可以使用 VoiceOver 访问
*   广告不得中断背景中的音频播放
*   广告不得使用令人困惑的策略，如在视频播放时将关闭按钮放在左边，然后在视频播放后放在屏幕的右边。

我不确定这里的音频规则，因为它可能是广告的一个组成部分。另一方面，应该有一个自动重启广告开始前播放的音频的机制。
