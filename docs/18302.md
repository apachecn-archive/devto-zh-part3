本教程解释了如何在 react 本地应用程序中检测设备是 android 还是 ios。当构建跨平台应用程序时，您会希望尽可能多地重用代码。React Native 是一种跨平台的本地应用程序编程语言，带有特定于平台的代码。有两种类型。react 原生应用程序中的 JS 文件，第一个是 index.android.js，第二个是 index.ios.js 文件。index.android.js 文件用于编写应该专门在 android 设备上运行的代码，index.ios.js 文件的代码应该在 ios 设备上运行。

有时我们需要在 android 和 ios 应用中使用平台特定的组件，为了在移动设备中使用平台特定的组件，我们首先需要识别设备是 android 还是 ios。
