作为一名长期的开发人员，我开始时是一名提示风格的开发人员，因为在 [Visual InterDev](https://docs.microsoft.com/previous-versions/visualstudio/visual-studio-6.0/ms950412(v=msdn.10)?WT.mc_id=devto-blog-shboyer) 介绍给我之前，这就是我的全部工作。

在经历了像 Eclipse for Java 这样的工具之后，Visual Studio 的各种[迭代。网；当 node.js 进入开发领域时，我发现自己又回到了命令行。](https://visualstudio.microsoft.com/?WT.mc_id=devto-blog-shboyer)

今天，我很少在日常开发任务中使用鼠标。

*   饭桶
*   [。网芯](https://dotnet.microsoft.com/download?WT.mc_id=devto-blog-shboyer) : `dotnet`
*   Node.js，JavaScript : `node`、`npm`、`yarn`、`vue`、`ng`
*   [蔚蓝](https://docs.microsoft.com/cli/azure/?view=azure-cli-latest&WT.mc_id=devto-blog-shboyer) : `az`
*   码头工、库员:`kubectl`、`kubectl`
