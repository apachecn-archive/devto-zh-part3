# Stadia 可以为所有人提供 3D 网站

> 原文：<https://dev.to/dotnetcarpenter/stadia-could-enable-3d-websites-for-all-3ki4>

[![Stadia Icon](img/e0caff05b3cf850a4b18b0b3172a1de0.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--Pu1smagW--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://stadia.dev/statimg/favicon/touch-icon-180x180.png)

自从 webgl 问世以来，创建 3D 网站一直是我的梦想。让这项技术非常令人兴奋的是你可以用着色器而不是 3D 世界做什么。在移动优先的场景中，这是不可行的。我们需要在客户机上完成所有计算，这排除了除台式机之外的所有设备。

进入 Stadia。[https://stadia.dev/about/](https://stadia.dev/about/)借助来自 Khronos 的 Stadia tech 和 Vulcan，我们可以在谷歌云上呈现网站服务器端，并通过高带宽连接将其传输到设备上。这使得 webgl 面向更广泛受众。

即使这项技术是基于开源软件，Linux 和 API，Vulcan。它只能在谷歌云中运行，这意味着它远远不适合今天的网站，但这只是技术的早期阶段，谁知道未来我们会有多少主机。现在，我只想把这个想法说出来。我谨慎地感到兴奋。这可能会改变游戏规则，也可能永远不会发生。
