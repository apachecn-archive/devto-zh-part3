**终端:**
`Terminal`程序内置在 Mac 电脑上，使用文本命令操作电脑，不使用用户界面。GitHub 的终端命令都以前缀`git`开头，后跟命令。

**常用命令:**

*   `~`表示主目录
*   `pwd`打印工作目录(pwd)显示当前目录的路径名
*   `cd`更改目录
*   `mkdir`新建一个目录/文件夹
*   创建一个新文件
*   向上一级/目录
*   `cd ~`返回主目录
*   清除显示屏上的信息，以提供一个空白状态
*   `ls` List 提供了一个目录下所有文件的列表
*   `ls -l`垂直显示一长串文件权限、修改日期和文件名
*   `ls -la`显示所有文件

**自动补全文件名:**
输入文件名的前几行，然后按`tab`键，它将自动补全剩余的文件路径名。
