# 每个开发者都应该访问的最佳顶级开发者社区

> 原文：<https://dev.to/prathaprathod/best-top-developer-community-every-developer-should-visit-12dk>

通常你会发现网站和开发者社区，这些社区将帮助建立一个网站，并帮助开发者制作更好的网站或应用程序。

互联网上有很多社区，开发人员是如此古怪的一群人，没有普通的网站可以做到这一点，所以我收集了一些必须访问的网站，以满足开发人员的需求，让他们了解最新的技术新闻，与同行交流并讨论项目和故障排除，供初学者掌握编程的网站，当然还有一些有趣的网站，当你需要从最新的开发项目中休息一下时可以访问。

### [Dzone](https://dzone.com)

它是最好的开发者指南之一，超过一百万的开发者属于 DZone，使它成为当今编程专业人士和开发者的一个突出的编程新闻中心。DZone 涵盖了从敏捷到云、devops、大数据、集成、性能等等的所有内容，是来自行业各个角落的最新见解的宝贵来源。

### [栈交换](https://stackexchange.com)

StackExchange 是为您带来 StackOverflow 的平台(它也恰好是 StackExchange 在技术类别中最大的社区)，但它也是程序员和开发人员感兴趣的各种其他社区的所在地。

### [【Github】](https://github.com)

如果不提到 GitHub，开发者的网站列表就不完整，GitHub 是使用最广泛的代码库之一，其项目涵盖了从开源到私有协作的所有范围。

### [站点点](https://sitepoint.com)

这里是学习网站，你可以从这个网站学到任何东西。

### [Reddit 编程](https://www.reddit.com/r/programming/)

是面向开发者的 Reddit！Reddit 是讨论最新发展趋势、技术新闻和忙碌的开发人员所想的一切的首选社区。

### [斜线号](https://slashdot.org)

SlashDot 是一个关于所有与技术和编码相关的最新博客文章和新闻故事的大型交互式资源，由 Rob“cmdr taco”Malda 于 1997 年创建，但现在归 BIZX，LLC 所有。

### [t1](#bitbucket)[比特巴克](https://bitbucket.org)

Bitbucket 提供无限的公共和私有存储库，对个人和拥有五个或更少用户的组织是免费的。对于较大的群体来说，定价非常合理。

### [开发到](https://dev.to)

Dev.to 是开发者的超级网站，许多开发者在这里写有趣的话题，程序员在这里分享想法，互相帮助成长。

我希望这篇文章对开发者有所帮助。
