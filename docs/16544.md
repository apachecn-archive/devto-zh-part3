# 摘录或未编入索引的帖子

> 原文:[https://dev.to/k4ml/snippet-or-not-indexed-posts-1dil](https://dev.to/k4ml/snippet-or-not-indexed-posts-1dil)

有时我只是想添加快速笔记，但这并不符合适当的职位。这有点类似于 coderwall.com。也许我们可以添加另一个像`indexed: false`这样的标志，这个帖子不会出现在任何提要中。但它仍然是公开的，有适当的网址，或者你可以在特定的网址如 dev.to/snippets/,查看，人们仍然可以对帖子做出反应或评论。这只是为了减少一些半成品笔记对主要提要的干扰。
