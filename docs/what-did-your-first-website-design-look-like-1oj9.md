## 有些前后关系

这件作品是在 2006 年夏天完成的，当时几个朋友请求我帮忙做一个网站，作为克罗伊登辅助教育项目的一部分。我当时不懂 HTML 和 CSS，就用 web 软件把这个屏幕拼起来代替了。

这次经历的挫折让我选择了 [**Sams 在 24 小时内自学 HTML 4**](https://amzn.to/2Q79iIt)。剩下的几乎都是历史了。
