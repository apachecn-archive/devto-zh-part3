# 如何在旧磁盘上(重新)安装 OpenBSD

> 原文:[https://dev . to/ner fur/how-to-re-install-openbsd-on-old-disk label-21k 2](https://dev.to/nerfur/how-to-re-install-openbsd-on-old-disklabel-21k2)

有一次，我找到了我的好的旧三星上网本(小 NC10 型号，对于我的需求来说，这是一款令人惊叹的笔记本电脑)，并发现它有好的旧 OpenBSD(我预料到了)。

但是问题出现了...真的是旧的 5.7 版本。该更新了！但是怎么做呢？官方常见问题仅支持封闭版本升级。一步一步来？太多时间了...重装！但是我不记得了，可能家里已经有重要的东西了。

我快速查看文档，却找不到如何在安装时保留一个 disklabel 分区的信息...但我找到了实现的方法。

重要。/家我都是用分离式隔断的，推荐给你！

启动你的安装介质，进入 fdisk 部分
选择 O 使用已经存在的 OpenBSD 磁盘分区。
下一页将是默认的自动磁盘标签...

DONTPANIC)

只需选择[C]自定义布局，用“p”确认您看到了旧的磁盘标签，现在使用“n”为您要格式化的所有标签设置挂载点。

！！！DONT 触摸您要保存数据的标签！！！

为“新”标签设置所有挂载点后，使用“q”保存并退出，观察只有“已挂载”标签如何格式化:)

继续安装，不要忘记在重新启动之前为其他挂载点修复/mnt/etc/fstab。

瞧啊。
