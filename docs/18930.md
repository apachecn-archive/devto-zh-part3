在将一个小型 ruby 应用从 ruby 2.3.3 迁移到 ruby 2.5.1 时，我在用 csv 库解析一个`mechanize`文本 CSV 文件时遇到了问题。这在 ruby 2.3.3 和 2.4.4 中运行良好。
