# 自学成才？你的优势和劣势

> 原文：<https://dev.to/stel/self-taught-your-strength-and-weakness--1d25>

### 大家好！

今天我在考虑更新我的简历，我开始问自己，作为一个完全自学成才的人，我的三个优点和三个缺点是什么。我很想知道你是否和我有相同的优点和缺点:

⭐️:对于那些自学成才的人来说，你的优点/缺点是什么？⭐️:对于那些自学成才的人来说，他们的优点/缺点是什么？

而如果你是雇主，如果你有两个相似的候选人，你会倾向于哪一个？学术的还是自学的？

#### 我的强项:

*   **又快又好？！)学习者**:我不知道是不是因为我是自学的，所以我学得很快，但我知道这对我帮助很大。当我开始我的第一份工作时，我没有用 JS 编码很多，我必须在两个月内学会如何用 React JS 编码。
*   独立:自学有助于你学会自己寻找信息、文件和问题的答案。
*   **自学显示你的动力和好奇心**:很明显你是有动力和好奇心的，否则你不会在这里；)

#### 我的弱点:

*   **没有被一个庞大的开发者社区包围**:很复杂(完全不是不可实现！)找一个开发者社区来交流。
*   **没有好的词汇**:有时当我和另一个开发人员交谈时，他们问我是否知道这个或那个，我说“不知道”。但当他们向我解释这是什么时，我意识到我知道这个概念/主题，但我不知道如何命名。我认为这是因为当你自学时，你没有就你所学的内容进行足够的交流，你没有正确地记住所有的词汇(几年的专业经验有助于正确命名事物)。
*   **独立**:这也是一个弱点。事实上，你习惯于自己做所有的事情，你忘记了有时候，寻求帮助对于提高效率和增长知识是至关重要的...
