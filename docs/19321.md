# 向 Mongo +添加索引:案例研究

> 原文:[https://dev.to/stephencweiss/adding-indexes-to-mongo 案例研究-3kbo](https://dev.to/stephencweiss/adding-indexes-to-mongo--a-case-study-3kbo)

我最近一直在从事具有较大数据集的项目，并开始注意到不是所有的查询都能在一两毫秒内解决。

正如我在[索引数据库 Postgres 示例](https://wp.me/p6u9oI-gW)中所写的，添加索引可以加快查询速度，但代价是插入/更新时间。

这一次，我在使用 MongoDB，希望了解索引是如何工作的。虽然原则是成立的，但细节有所不同。

尽管如此，结论是一样的:如果你读的比写的多，并且能想出一个合理的索引目标，它可能是你工具箱里的一个有价值的工具。

让我们开始吃吧。
