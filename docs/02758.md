这篇文章摘自我的博客[我的网络开发历程](https://mywebdevelopmentodyssey.dev/time-for-a-readjustment/)

由于我上次提到的持续抑郁症，我有必要在精神健康中心呆上两周。我现在在家，感觉好多了，尽管最近关系破裂。有时候，药物治疗上的小小调整和/或在一个“安全”的地方呆上一段时间就足够了。

朋友们，这就是博客的内容。这是超越典型的斗争。它是关于在一条有许多弯道、山脉和岔路的道路上旅行。少走的路。最终我会多写一些技术类的帖子。但现在，它主要是关于不败之路。

有几个因素可能加剧了我的症状。我将只讨论一个与 web 开发相关的问题。

因为我将是入门级 web 开发人员中年龄较大的一个，所以我觉得我必须多走一步，让自己更有市场，更有竞争力。

我不仅在学习 fullstack JavaScript 开发，还计划学习 PHP/Wordpress、Python 和 Java 后端开发。所有这些都将在今年年底完成。

我仍然打算学习 PHP 和 WordPress，因为我想投身于自由职业，既做兼职，又用这些兼职建立我的职业组合。

我显然对自己期望过高。我决定缩减规模，等到明年或者工作需要的时候再学 Python 和 Java。其他一切我都延长了学习时间范围。

以这种方式进行可以让我每天花大约 3 个小时完成规定的学习(如果我想按计划进行的话),而不是 7 个小时。

我的新时间表允许更多的个人时间，如果我落后了，也允许一定的灵活性来赶上进度。

我迫使自己真正理解 JavaScript 的深度和基础。在我的世界里，理论极其重要。在我看来，知道事情如何以及为什么工作是灌输最佳实践的基础。

给自己更多的时间去真正理解基本的和高级的概念，可以确保我处于真正理解任何框架和库的最佳位置。当然还有尽我所能写出最好的代码。

老实说，我不认为它会花这么长时间，但当生活继续抛出曲线球时，我们需要暂停，反思，并重新调整。

我很快就会经历另一次重大的人生转变。我将搬到加利福尼亚的中央山谷地区。距离湾区和硅谷东部仅 2 小时车程。我希望地理位置会成为一个主要的职业优势。

总的来说，我认为这是一个正确的方向。这可能会轻微甚至中度影响我的学习，但我会根据需要重新调整。

在接下来的帖子中，我将分享我搬家的原因，以及它将如何促进我的 web 开发职业。
