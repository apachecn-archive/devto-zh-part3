# 自举！初学者前端开发的有用工具

> 原文:[https://dev . to/Joshua loran/bootstrap-a-used-tool-for-front-end-development-3g 52](https://dev.to/joshualoran/bootstrap-a-useful-tool-for-beginners-front-end-development-3g52)

所以，我要去学校学习成为一名全栈开发者。到目前为止，我们已经学习了许多有用的工具。我们已经学习了各种处理数据集的方法。我们已经学习了用 ruby 进行面向对象编程，以及如何在模型之间建立关系。我们已经学习了 SQLite 和活动记录数据库管理。此外，我们还学习了一些用户可以与视图(或网页)交互的简单方法。到目前为止，我们所学的一切使我们能够为一个网站构建一个相当复杂的后端，但是我们还需要了解用户体验。

令人敬畏的经历...

这不是我们作为开发者想要给我们的用户的，你可能不知道，我们的下一个项目是建立一个有很棒前端的全功能网站。当然，没问题。但是用户体验呢？我从未涉足 html、css 或 javascript，尽管我有充分的信心为网站开发支架，但我从未构建过好看的前端。在我之前，我有机会和一些同学见了面，他们的项目看起来很棒。它们包括交互式滑块、按钮、精美的图像和布局，可以根据用户的设备和偏好进行缩放和调整。我想要同样水平的抛光。我希望用户和我的网站是:

[![alt text](../Images/2db1b9f76d9b8157c8e86495b3e7c31b.png "Best friends forever")T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--hXrogJ-u--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_66%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/xv6rpdxips5le36m6zlr.gif)

这些和我处于同一水平的学生怎么能做出如此美妙的东西。答案是，自举。我从未听说过 bootstrap，也不知道它是什么，所以我花了几天时间研究它是什么以及如何使用它。我必须说，这是很酷的东西。

从基本意义上来说，bootstrap 是一个框架，前端开发人员可以在设计中使用它。它由 CSS、JavaScript/jQuery 和 glyphicons 组成。在这一点上，那些是我没有完全理解的神奇的事情。但是，bootstrap 的美妙之处在于，我不需要掌握这些方面的大量知识。它是一个框架，让我可以以一种简单的方式来玩页面的外观以及用户如何与它交互，这样我就可以把它作为一个工具来接受，并继续做更重要的事情。

[![alt text](../Images/426663c46fc5b542c8506b3a58d3c0d2.png "magical")T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--_9pUA02A--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://encrypted-tbn0.gstatic.com/images%3Fq%3Dtbn:ANd9GcRn_s0DI2LNNtQhXBzn87t3fO2EfqWxYLLII9rsouRL9lthcgMo)

最简单的开始方式是前往[入门](https://getbootstrap.com/docs/4.3/getting-started/introduction/)页面。但是，您不一定需要下载文件来使用它。它也可以通过利用 CDN(内容传递网络)来使用。使用 CDN 的一个优点是，bootstrap 被许多站点使用，并且通常在用户缓存中，这意味着潜在的更快的加载时间。

w3schools 有一个很好的资源可以让你体验 bootstrap 的一些特性。有交互式的“自己尝试”部分，让您在项目中使用代码之前就可以使用它。但是，也许一个更好的资源，一旦你熟悉了 bootstrap 的工作方式，就是 [bootstrap cheatsheet](https://hackerthemes.com/bootstrap-cheatsheet/) ，它有一套非常健壮的 bootstrap 交互，使用户体验更加愉快。

[![alt text](../Images/1d2a02d3a00f4fa0c7dbd095cf3fbf74.png "user experience")T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--dtXRx-6E--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_66%2Cw_880/https://public-media.interaction-design.org/images/uploads/9153fdf2373a09b5a3d95cc4e68a966f.gif)

现在你已经阅读并使用了 bootstrap 一点点，并且有了一长串的可用性特性，你想让它看起来也很漂亮。一些背景成像和设备缩放肯定会给外观和感觉一个升级。bootstrap 的一大优点是它有一个大型开源社区。因此，有了像 [Start Bootstrap](https://startbootstrap.com/themes/) 、 [Get Bootstrap](https://themes.getbootstrap.com/) (需要一些费用)和 [Wowslider](http://wowslider.com/posts/35-top-free-bootstrap-templates-2016-95.html) 这样的网站，很容易找到适合你的主题。而且，您可以随时根据自己的需要调整主题。
