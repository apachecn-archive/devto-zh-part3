# 想提升你的职业生涯吗？

> 原文:[https://dev . to/bhavaniravi/thinking-of-up skilling-your-career-45hd](https://dev.to/bhavaniravi/thinking-of-upskilling-your-career-45hd)

事业是每个人生活的主要部分。就时间和激情而言，职业是我们大部分生活经历围绕的地方。因此，做出职业选择似乎是每个人生活中至关重要也是最令人畏惧的事情。

像我这样的技术人员在市场上追逐下一个闪亮的东西是很常见的。虽然了解周围的新技术是好的，但不要忘记核心技能。与任何一个技术人员交谈，你认为你自己在哪里，他们会说“我目前正在做这件事，未来就是这样，所以我很快就会开始做这件事。”

听起来完全合理，知道有人正确地规划了他们的生活很酷？。但是等等，你想成为一切，你想成为一个无所不知的人。但是我很抱歉地告诉你，这个系统有问题。

### [](#mastering-skill-takes-time)掌握技巧需要时间

假设你是一名前端开发人员，刚刚开始学习 HTML、CSS 和 JS，几年后，你已经将 ReactJS 融入到你的技能中，生活看起来非常精彩，当你掌握了所有这些前端技能时，你将进入一个对后端一无所知的 FOMO 状态。

现在，如果你是一名大学生，你有足够的灵活性和时间去探索多个领域。但是一旦你进入公司，在 5 年的前端工作之后，尝试涉足后端并不是一个好主意。

记住你已经走过的掌握当前技能的道路。选择一个离你现在的工作有几英里远的工作将会花费你同样多的时间，而且你不确定你将会面临的障碍

### [](#specialization-gt-generalization)特殊化>普遍化

对人类来说，最大的恩惠和禁令是他们的好奇心。很难抑制想要打开那个闪闪发光的盒子，找到里面的东西的冲动。虽然它造就了我们今天的样子，但在当今世界充斥着各种信息的情况下，我们很容易感到分心和被忽视。你想成为外面每一件很酷的事情的一部分，但是要沉住气，大声说出这一点**专业化>普遍化。**这个行业希望人们真正擅长他们正在做的一件事，而不是一个什么都懂的忍者。

你不可能捕捉到一个黑洞的图像，同时赢得一枚奥运奖牌。

### [](#find-your-sweet-spot)找到你的甜蜜点

是的，我们都必须学习，我们都必须提高，这是毋庸置疑的。但是你选择下一步要掌握的技能应该与你现在拥有的相一致。

寻找互补的技能来掌握，而不是寻找让你的简历闪闪发光的技能。如果你正在经营一家服装店，扩展到配件不要试图出售船只和飞机。你在这个世界上的时间有限。因此，用正确的方式提升你的职业技能。
