# 你真的能从在线课程中学到 JavaScript 吗？

> 原文：<https://dev.to/canderson93/can-you-actually-learn-javascript-from-an-online-course-5eag>

当你在 Udemy 上看到有人承诺只用 2 周就能把你从 JavaScript Zero 带到 JavaScript Hero，很容易产生怀疑。像 FreeCodeCamp 这样的课程呢？仅仅通过在线课程，你真的能成为一名开发者吗？

答案是...*种。*

要成为一名开发人员(使用任何语言)，您需要两套技能:

1.  你需要知道一种编程语言
2.  你需要知道如何*构建软件*(或网站等)

在线课程在教你#1 方面做得非常好——如果你完成了 JavaScript 的在线课程，你会对 JavaScript 的构建模块有很好的了解。

你的弱点在于知道如何应用这些知识——即使是那些承诺教你使用“真实项目”的课程，也常常会让你复制粘贴大量的代码，但对于帮助你自己去应用这些知识却没有什么帮助。

## 学习建造

那么如何学习构建软件呢？

从一个大项目开始学习可能很有诱惑力(你可以在建立你的大商业想法时学习 JavaScript:欧洲的优步，对吗？)，我建议不要这么做- *大项目不利于好的学习*。

学习构建软件的最好方法是创建小的、一口大小的项目，你可以在一个小时到一个周末之间完成。这意味着什么完全取决于你，以及你对编码的适应程度。

总的来说，集中精力一次学一件事，把它们叠起来。编码倾向于建立在以前的知识之上，所以当你进入更难的概念时，关注更简单的概念并花时间去适应将会极大地帮助你。

最重要的是:*做工作*。
