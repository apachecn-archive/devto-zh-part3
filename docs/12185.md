# 使用 Docker 创建一个节点开发环境

> 原文：<https://dev.to/auth0/use-docker-to-create-a-node-development-environment-pcp>

在本教程中，您将利用官方 Docker 节点映像所基于的 Debian Linux 操作系统，而不是在本地创建和运行节点应用程序。

您将创建一个可移植的节点开发环境，解决“但是它在我的机器上运行”的问题，这个问题一直困扰着开发人员，因为容器是通过在任何平台上执行 Docker 映像来创建的。
