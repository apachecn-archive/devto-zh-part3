# 它会成长吗？

每当你开始投资发展你的职业生涯时，你可能会决定这样做:

*   博客
*   构建辅助项目
*   建立工作关系网
*   参与用户组和聚会
*   等等。

你应该期待立即看到你的声誉和职业发展的显著成果吗？

没有。

当考虑发展你的职业时，我喜欢用花园来比喻。

我认为这个类比的一个方面是，对于许多类型的种子来说，你必须等待几周甚至几个月才能看到任何结果。

有些植物只有在整个生长周期接近尾声时才真正开始有形的生长。

我们的职业生涯也是如此。

开始似乎很慢。

但是在你建立势头之后，事情会很快发生！

# [](#compound-interest)复利

经常用来说明这一点的类比是复利。在复利的情况下，你对银行的投资开始时是少量现金，增长非常缓慢。但是随着时间的推移，你的现金会越来越快地增长。

随着时间的推移，这种势头开始增强。

就像种植植物需要让种子一开始就吸收足够的水分和养分一样，职业发展需要一些前期投资和耐心。

是的，这的确需要努力。但是期望努力工作会在短时间内产生巨大的收益是不现实的。

建立你的职业生涯是一个长期的游戏。这就像投资并等待复利发挥它的魔力。

职业发展的第一步更多的是建立动力，而不是建立切实的结果。

# [](#my-experience)我的经历

我的经历就是这样。

在我职业生涯的大约 2 年里，我意识到仅仅“做好我的工作”不会给我带来机会。这不会把我和其他想得到好工作和好机会的人区分开来。

于是，我开始制定计划，开始学习非常有针对性的课题。

2017 年，我开始写博客。但是我没有写那么多博客。

我有一个家庭要照顾，又有一份全职工作，所以没有大多数人那么多空闲时间。

慢慢地，我试着在其他出版物上推广我的内容。

我在一个用户群发言。

一次做一件事。

大约一年后，我得到了价值超过 2000 美元的软件，用来购买我在博客上写的东西。

然后，我开始了一个[开源项目](https://github.com/jamesmh/coravel)，最终获得了一些关注。

通过这种方式，我最终被邀请在[https://builtwithdot.net](https://builtwithdot.net/)上写客座博文。

几个月前，我开始使用 Twitter 作为一个平台，帮助开发人员使用技巧，测试我的想法等。

其中一些想法似乎引起了很多人的共鸣，所以我根据我测试过的[一个话题](https://twitter.com/jamesmh_dev/status/1080455811048984576)开始了一篇[电子邮件简讯](https://eepurl.com/gdIV5X)。

这就像滚雪球一样变成了最近的机会，比如:

*   其他出版物的客座博文，如 [App Academy](https://blog.appacademy.io/tips-for-first-time-software-developers/)
*   [Developing Up 播客上的特邀嘉宾讨论职业所有权](https://www.developingup.com/episodes/41)
*   [史蒂夫·史密斯关于加快职业发展的每周发展提示播客特邀嘉宾](https://www.weeklydevtips.com/048)
*   为[兰巴学校](https://lambdaschool.com/)举办网络研讨会
*   还有更多我还不能说的东西！

就我而言，我觉得这种势头开始赶上来了。

但是有一段很长的等待和“露面”的时间。

# [](#my-advice-for-you)我给你的建议

我的建议是:

出现就好。

如果你致力于为你的博客创造内容，那就坚持做下去。

你不需要每天甚至每周都写博客。我没有。

尽你所能向前推进。

要有耐心。

抓住你遇到的任何机会。

与你尊敬的人建立联系。

试着在 Twitter(或其他任何地方)上测试想法，看看人们怎么想。

写**你**热衷的内容！

但是要理解复利的原理。通常需要*年*才能看得见你的劳动成果。

最重要的是，记住你是在播种。

最终，你会看到植物开始发芽！
