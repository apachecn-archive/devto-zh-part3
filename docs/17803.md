在本教程中，我们将讨论 javascript 纯函数。这些函数在 javascript 中被广泛使用，我们使用它们是为了一些特定的目的。

什么是纯函数？
纯函数的定义是:
如果传入相同的参数，函数总是返回相同的结果。它不依赖于程序执行过程中的任何状态或数据变化。它必须只依赖于它的输入参数。
该功能不会产生任何可观察到的副作用，如网络请求、输入输出设备或数据突变。
