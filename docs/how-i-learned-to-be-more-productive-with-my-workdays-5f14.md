# 我是如何学会在工作日提高效率的

> 原文：<https://dev.to/kelly/how-i-learned-to-be-more-productive-with-my-workdays-5f14>

我是分心女王。如果我看到一封电子邮件通知，我必须停下手头的工作去查看。我的手机发短信亮了？哦，有人想和我说话！(没有，只是美国电话电报公司让我知道我的自动转账已经成功处理。)我在 2018 年花了很多时间，试图弄清楚我能做些什么来让我的工作日更有成效。这仍然是一项进行中的工作，但我发现了六个主要的行为变化，它们帮助我在工作时更有效率。

1.  我关闭了推送通知。大约六个月前，我关闭了所有社交媒体的推送通知。结果是我对工作的关注立刻增加了。我过去常常在手机亮起来的时候拿起手机，不可避免地，我会分心，开始浏览脸书或推特。接下来，半个小时过去了。我强烈建议在短时间内尝试禁用推送通知，看看它是否会对您的工作效率产生影响。
2.  **我所有的电子邮件都是先通过 [SaneBox](https://sanebox.com) 的。** SaneBox 是一个非常有用的工具，它可以自动过滤你的电子邮件，这样你就只会收到关于需要处理的重要事情的通知。你也可以训练你的电子邮件，这样系统会随着时间的推移变得更加智能。在繁忙的一周，我现在只收到大约 75%电子邮件的通知，只是在一天结束时收到一份摘要，让我知道我还有其他电子邮件要检查。计划起价低至 7 美元/月。
3.  我再也不在周五开会了。那是我的“我”日。我现在很闲，还在查看我的电子邮件，但是会议要么必须在周五之前召开，要么必须等到下周。这给了我整整 8 小时(大部分)不受干扰的时间来完成工作。
4.  当我需要的时候，我会休息一下。每个人都需要大脑休息。开发者尤其需要脑洞大开。我们倾向于试图解决一个问题，我们尝试，尝试，尝试，直到有希望的事情发生。90%的情况下，如果我在休息 15 分钟后带着一个全新的想法回到问题上，我可以找到一个解决方案，或者至少朝着一个解决方案前进。
5.  我开始明白没有什么是真正紧急的。如果没有写下来，就没有完成。你知道这是怎么回事:你开始一天的工作，期望完成 A、B 和 C，但一整天 G、H、I 和 J 都在出现，当你结束一天的工作时，你甚至还没有开始 A。我越来越能接受不是所有的事情都是紧急的。不火的话，大概可以等到明天。
6.  我不会接没有预约的电话。我知道这并不适用于所有人，我也知道其他人不会同意我的观点。但我对电话有强烈的感觉，如果你不在 Slack 上给我发邮件或信息，问我是否可以打个电话，而是直接打电话给我，你会认为你的时间比我的时间更宝贵。我可能正在忙着做一些(希望是)重要的事情，拿起电话不仅会分散我对正在做的事情的注意力，还会在电话结束后让我分心，我需要重新整理思路，找出我从哪里停下来的。友好地安排电话。
