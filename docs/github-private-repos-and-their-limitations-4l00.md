# GitHub 私人回购及其局限性:

> 原文：<https://dev.to/10secondsofcode/github-private-repos-and-their-limitations-4l00>

优势:

*   我们可以创造任意数量的私人回购。
*   可维护性很容易
*   所以我们可以像 gitlab 和 bitbucket 等其他 repo 一样私有地管理我们的代码。等。

缺点:

*   我们不能为组织创造免费的私人回购。
*   免费的私人回购不能创建生长激素页。
*   只有 3 个合作者只添加免费的私人回购。
