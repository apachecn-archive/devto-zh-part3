# 我从对竞争对手的痴迷中学到了什么

> 原文：<https://dev.to/skcript/what-i-learnt-about-obsessing-with-my-competitors-5dif>

我痴迷于我的竞争对手。我到处跟踪他们。一丝不苟，毫不留情。

当我在 2013 年开始 [Skcript](https://www.skcript.com) 的时候，我沉迷于我的竞争，以至于我会给它很多该死的东西。几年过去了，已经是 2016 年了，那种永远无法‘暂停’(不是停止)我跑步的感觉开始下沉。一个美好的夜晚，我开车去看看如何克服这种心态，并把注意力转移到我们可以在 Skcript 做的其他事情上。

我对竞争分析的理解是不同的。

人们说竞争是好的。你必须对你的竞争对手着迷。这背后的真正原因不是让你的公司一直保持更新，而是知道你的客户真正想要什么。句号。就这么简单。

知道你的客户想要什么比密切关注你的竞争对手更重要。当你初入市场，试图创造一些好东西时，这可能是唯一对你有意义的指标。

观察你的竞争对手并对他们着迷是充分了解你的客户的一部分。

我在 2016 年经历了漫长的旅程，在压力下，我改变了一件事来打造重要的伟大产品，那就是我在哪里看待我的竞争对手。

早些时候，我曾痴迷地到处追随他们。在社交媒体上，从朋友那里获得更多关于他们的信息，当我经常听到新闻说他们的平台上增加了 2000 多名用户时，我的主要压力就来了。

今天，我所寻找的和帮助我增加收入的一个指标是分析我的竞争对手的功能集，并找出他们为什么要这样做。把我的竞争对手看作是我最大的免费用户测试提供商之一让一切都变得不同了。

最终，我们都是来解决客户问题的。不是我们自己的问题。
