# 我的软件工程之旅

> 原文：<https://dev.to/emmabostian/my-journey-into-software-engineering-15ef>

"你真的应该考虑软件工程——我认为你会很擅长的。"这不是我第一次从父亲那里听到这些话。

在一个父亲是软件工程师、母亲是设计师的家庭中长大，我几乎注定要成为一名工程师。但是和所有青少年一样，我坚持“走自己的路”

我是参加 IBM 的“带孩子来上班日”长大的。我总是喜欢参加危险挑战，并创造自己的条形码。然而，我从未想过工程学是一条可行的职业道路。

## 学院

我于 2011 年进入锡耶纳学院，主修生物学。我百分之百肯定我会成为一名产科医生。不用说，那没有发生。

当我的第一个学期接近尾声时，我离精神崩溃只有五分钟了。

我的生物和化学差点不及格，我知道这不适合我——我对它不感兴趣。于是，我向邻近的一所学校申请了音乐教育专业。

我收到了纽约奥尔巴尼圣罗斯大学的录取通知书，但我内心的某种感觉告诉我，这不是一个正确的决定。我决定坚持下去，尝试一些新的东西:精算学。

那是在我大一(2012)第二学期的时候，我选修了一门“计算机科学导论”的课程。我学习了所有关于阿达·洛芙莱斯的知识，并学会了如何将二进制转换成十六进制再转换成八进制。我被迷住了。

编码的技术和创造性之间的二分法激发了我的兴趣。我在大二(2012)的秋季学期宣布主修计算机科学，辅修商科。

我热爱我学位的每一秒钟。这的确是一门很有挑战性的课程，然而当你从这些材料中获得灵感时，它就不再像是工作了。

大三(2013)秋季学期，在伦敦城市大学留学。这是我一生中最美好的三个月。我通过不同的文化体验了计算机世界。

正是在我出国期间，我参加了 IBM 实习的面试，并被录取了。

大三(2014)后的那个夏天，我在 IBM Poughkeepsie 做合作社/实习生。我的主要目标是使用 Python 在 z/OS 上自动安装 WebSphere。

在我的整个课程中，我主要用 Java 编写代码，所以学习 Python 对我来说是一个新的挑战。

## IBM

快进到高三(2015)年末。我获得了计算机科学学士学位，并接受了位于德克萨斯州奥斯汀的 IBM 工作。

我的职业生涯始于 IBM 频谱控制。转折？我是从前端开发人员做起的。

我在大学期间没有太多的网络开发经验。虽然我确实上过网页开发课程，但那是非常高级的课程，我真的没有为职业世界做好准备。

对我来说，这是一个极具挑战性的时期。就技能而言，我远远落后于我的同事。我不得不在全职工作的同时自学前端开发。但是我坚持下来了，熬过来了。

2016 年，我担任了可访问性主管的角色，学习了所有关于网站可访问性的知识——键盘导航、屏幕阅读器兼容性、高对比度模式。我能够在 IBM Spectrum Control 上实现完全可访问性，这是一项伟大的成就。

然后 IBM 量子计算副总裁兼 CTO Scott crowd er 找到了我。他给了我一份工作，和他的设计团队一起做前端开发。

因此，在加入 IBM 一年半后，我换了团队，获得了设计过程的大量知识。

在 IBM 系统和转换设计团队工作期间，我参与了许多有趣的项目。我致力于开发支持的未来和开源能力和可用性工具。但是我最喜欢的项目是 IBM Q。

量子计算是当今工程和计算领域最酷的领域之一。能够与业内一些最聪明的人一起工作，这让我着迷不已。

作为设计团队中唯一的前端开发人员，我有机会尝试新技术。

我最大的成就是用 Vue.js 单枪匹马地建立了 IBM Q Network 网站，直到今天我都为这个网站感到无比自豪。

然后，我的人生来了一个完整的 180 度大转弯。

2017 年在 IBM 做项目的时候认识了现在的老公。唯一的问题是。他住在德国。

## [对数](#logmein)

所以在 2018 年 2 月，我卖掉了我所有的东西，带着我的两只猫和我自己飞到了德国。

不幸的是，我无法保住我在 IBM Quantum 团队的工作，所以我被迫寻找新的职位。

我参加了许多德国公司的面试，但在有大量国内候选人的情况下，许多公司都不想投资一个外国人。

然后，我采访了 LogMeIn。经过三四次面试和一次编码挑战后，他们慷慨地给了我一个职位。

我在一个开发团队工作，为 GoToMeeting 构建 UI。这太棒了，因为我学会了 React、Redux、TypeScript 以及如何用 Jest/Enzyme 编写单元测试。

最近，我担任了一个 UX 工程师的新角色(博客帖子即将发布)。我现在主要关注的是用 React、Vue 和原生 web 组件构建一个组件库和样式指南。作为 LogMeIn 的第一位 UX 工程师，有些令人兴奋的事情。我开始学习宝贵的开发和设计技能。

我的旅程才刚刚开始，我期待着未来的岁月。
