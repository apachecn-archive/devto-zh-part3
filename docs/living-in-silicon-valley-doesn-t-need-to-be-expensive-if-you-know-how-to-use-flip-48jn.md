硅谷是大多数人的“圣杯”。它是许多世界上最大的高科技公司的所在地，包括财富 1000 强企业的总部，更不用说...成千上万有前途的创业公司。
硅谷也占了当今美国所有风险投资的三分之一。
这是大多数 Web 开发人员想去工作的地方。全职工作，拿着丰厚的报酬和所有的额外津贴。

唯一的问题是，硅谷(就此而言，可能是整个北加州)因其令人瞠目结舌的房地产价格而臭名昭著。换句话说，房租会让每个人付出高昂的代价。旧金山一居室公寓的月租金中位数为 3500 美元。
这基本上是美国 Web 开发人员工资的一半。这还不包括地铁中的其他年度必需品，包括食品、交通、医疗保健、储蓄和其他可自由支配的支出。
这是完全荒谬和不可接受的。

平均而言，翻转租赁的期限通常约为 6 个月，但协议期限从 3 个月到一整年不等。
