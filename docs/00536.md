我想在谷歌地图(或其他软件)上显示某些事件，有什么建议吗？

点(实际上是线)将从数据库中动态加载，并通过 web 浏览器呈现给用户。

因此，我正在尝试做的是一个谷歌地图(最好)与我自己的自定义点(线)，这是可点击的，点击后更多的信息应该加载。我已经做了一些类似的传单，但现在的要求是使用谷歌地图或类似的东西...
