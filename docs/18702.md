# 培养良好的习惯

> 原文:[https://dev.to/lorenzoattrexin/building-good-habits-1ph8](https://dev.to/lorenzoattrexin/building-good-habits-1ph8)

培养良好的习惯是建立更有意义的生活的关键。我们从定义我们想成为什么样的人开始。如果我们想要成为更好的开发者，我们必须定义这对我们意味着什么。我喜欢我在一次参加的斯蒂芬·柯维研讨会上学到的练习。想象你自己在守灵(或一些生命结束的庆祝活动)，选择一些人发言。他们会是谁，你希望他们怎么评价你？这成为你想成为的那种人的蓝图。例如，让我们想象你选择你的孩子说话，说你是世界上最好的父母。你的目标之一就是成为一个更好的父母。

不幸的是，只有目标是不够的。你怎么去那里？一种方法是培养习惯，塑造你的行为。为了成为一个更好的父母，你可以定义的一个习惯是每天至少花 15 分钟和你的孩子交谈。只是聊聊天。没别的了。这可能看起来不多，但经过多年的时间，你的孩子会记得他们生命中的每一天都与你交谈。这将产生巨大的影响。

假设你想成为一名更好的软件开发人员。这对你意味着什么？谁会为你说话，他们会说什么？是你的经理、同事、顾客还是其他人？那个人会说你聪明、机灵、敏捷或者是一个伟大的导师吗？取决于谁说你会定义一个更好的软件开发人员对你意味着什么，这才是最重要的。我将在以后的文章中更多地讨论如何培养良好的软件开发习惯。
