有没有想过用**粗体**、*斜体*或~~删除线~~文本发推文？

Twitter 不允许其用户格式化推文。但是有一个选项*强调*或**喊出**一条推文中的一个单词或一个短语，有时可能会派上用场。

# [](#announcing-tweetfancyio)发布推文

我最近意识到 UTF 字符集有[字符和符号](https://en.wikipedia.org/wiki/Mathematical_Alphanumeric_Symbols)，看起来像**粗体**、*斜体*或~~删除线~~ ascii 字符。所以我做了一个工具，可以生成这些字符，可以复制粘贴到几乎任何网站上。

*   在 markdown 中写下你的原文
*   编辑器将生成文本供您复制/粘贴
*   直接从编辑器发推文
*   将生成的文本复制/粘贴到任何网站(FB/Insta/etc)

*注意:生成的字符是数学符号，而不是实际的英文字母。所以如果你用屏幕阅读器来运行它们，你会发现阅读器会根据它们的符号名称来读出它们。但是，所有的屏幕阅读器也很难读懂表情符号(🎉🔥🚀🦄😎)这并不妨碍我们使用表情符号。所以，我们也可以用这个。*
