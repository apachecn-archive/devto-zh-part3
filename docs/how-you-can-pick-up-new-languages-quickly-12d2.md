# 如何快速掌握新语言

> 原文：<https://dev.to/nandotheessen/how-you-can-pick-up-new-languages-quickly-12d2>

[![learning new things](img/993e22970a8c3b8bcf1f29f8767e7174.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--_DXbEeOe--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/qsiqjte92je1ryk6zeti.jpg)

关于学习一个新概念、语言或框架的最大误解之一是你如何能有效地做到这一点。我曾经和许多优秀的人一起学习和工作，他们在任何大型学习网站上都有帐户，或者有一个大型的技术课程图书馆，我自己也有一个类似的图书馆。

可悲的是——在大多数情况下——仅仅跟随一个教程或一门课程并不能让你在新事物上做得更好。
很多时候，它让你不再思考自己，从而阻碍了你的进步，而不是像你希望的那样加快了它。

## 我的情况

我最近在一家公司获得了一个新职位，该公司使用了一个我从未合作过的堆栈。

在我看来，这是一个很好的机会，我认为更多的人可以从考虑他们认为“不适合”的工作中受益。

如果你是一个好的候选人，很多公司都愿意在工作中教你语法。对他们来说，这取决于你是否是一个好的候选人:如果你理解潜在的概念，可以接受训练，并且很适合团队，他们无论如何都会雇用你。

在这篇文章中，我想向你展示我是如何学习一门新语言的，以及它是如何让我(希望你也是)快速地做出贡献的，我将以 Golang 为例。

## 1:寻找有用的资源

如今，找到好的资源令人惊讶地困难。这主要是因为大量的内容。在任何给定的主题上，大约有 15 个视频课程，带你在 4 个小时内从一个完全的初学者到精通，并且伴随着大约 200 篇博客文章。从那么多选项中做出选择是很难的(T2)。

除了基本语法之外，我喜欢关注的是如何翻译我已经熟悉的概念，并查看它们在新语言中是如何实现的。这有助于我快速建立一个正在发生的事情的心理模型。

我的下一步将围绕最佳实践展开。
以下资源对我学习 Golang 尤其有帮助。

#### 介绍去

对这种语言语法的一个很好的介绍是你可以在这里找到的官方旅游。

它带你了解这门语言的基本概念，有具体的例子和练习帮助你熟悉基础知识。

我认为这是一个很好的资源，不管 Go 是否会成为你的第一语言，或者你以前是否和其他人合作过。虽然这当然会更有挑战性。

下一步，我建议至少通读一下[如何编写 Go 代码](https://golang.org/doc/code.html)，这也是官方资源的一部分。

#### 超越基础知识

真正帮助我的一件事是阅读 Go 中的最佳实践，并将它们与我使用过的其他语言进行比较。例如，将我已经熟悉的概念转换到新的环境中，可以帮助我理解新代码库中正在发生的事情，或者如何设计我必须编写的新服务。

一个很好的资源就是[有效 Go](https://golang.org/doc/effective_go.html) 。我还发现了一些真正帮助我理解围棋生态系统的讲座，并让我了解了我应该注意的一些怪癖:[围棋说瓦特](https://www.youtube.com/watch?v=zPd0Cxzsslk&t=999s)或[围棋中我从来不用的东西](https://www.youtube.com/watch?v=5DVV36uqQ4E)。

## 2:构建熟悉的事物！

[![Simple Things](img/8a3dbb02541238a9797d44e8e06b99e0.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--vVf_poRE--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/65st8lxpd0khf5arn2jy.jpg)

对于 Golang，我的第一个项目是 REST API。这是学习该语言如何处理 HTTP 请求的一个极好的方法，什么工具对此有用，而且它足够简单，不会占用你多天的时间。此外，应用编程接口到处都在使用，很可能是你(未来)工作的核心部分。

总的来说，我认为使用你已经熟悉的东西的简单版本，并将其移植到你正在尝试学习的语言中是一个很棒的主意。如果你非常擅长编写一个 API、算法或玩具程序，你可以把重点放在关键差异上，并从中学习！
