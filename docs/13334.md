# 最受欢迎的 React 网站功能

> 原文:[https://dev . to/Anders Jr 1984/most-loved-react-site-features-57nf](https://dev.to/andersjr1984/most-loved-react-site-features-57nf)

有你喜欢去的网站吗？告诉我你最喜欢的网站功能是什么，为什么你一直回去！我尝试在我的个人网站中实现一些我喜欢的东西，我总是喜欢寻找更多。这里有几个例子:

-AirBnb.com:页面用占据整个屏幕的美丽图像向你问候。我想把这个放在我的主页上。让图片只显示在我的主页上有点麻烦，但是我想通了，因为 React 太棒了。

-instacart.com:这和我的介绍性横幅很搭。我喜欢页面在横幅上滚动的方式。这个很简单！

-zillow.com:嵌入式图像开启器。这是我写的一个教程。我想让用户留在我的网站上，并扩大图片以满足他们的浏览乐趣。这个需要一点初始设置的逻辑，但是现在它工作了，重新创建将会非常容易！

-React app 的初始页面:他们如何让 atom 旋转是非常出色的，并且只需使用 CSS 就可以非常简单地实现！我想把它添加到我的站点中，作为一个“Taco locator”API 示例。谁不喜欢旋转玉米卷呢？
