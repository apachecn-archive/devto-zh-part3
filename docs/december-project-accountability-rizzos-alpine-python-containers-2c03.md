# 12 月项目责任:Rizzo 的 Alpine-Python 容器

> 原文：<https://dev.to/kaelscion/december-project-accountability-rizzos-alpine-python-containers-2c03>

因此，我的 12 月 MVP 挑战的第一周已经完成，我已经达到了预期的目标！如果你不知道我的自我挑战，不知道我为什么要这么做，不知道这个项目是什么，请点击这里查看我的第一篇帖子。

特别感谢 [@jess](https://dev.to/jess) 直截了当地问我第一周的目标是什么，这让我保持专注。本周的目标是构建两个 docker 映像，这是我的服务将使用的关键后端功能所需要的。虽然这个过程很琐碎，但是使映像尽可能高效和快速的目标帮助我真正研究和缩小了要使用的基础映像，以及哪些依赖项是真正需要的，哪些是可以消除的。非常感谢到目前为止参与的人！

第二周的目标是构建前端和 web 服务器映像。虽然两个后端映像都非常简单，因为它们依赖于相同的基础映像，但下周会有点困难。如果有人对他们最喜欢的 Gatsby.js 和/或 Nginx 图片有任何建议，请在评论中留下。我开始对这个小挑战的成果感到非常兴奋，并期待着在 1 月 1 日与大家分享最终成果！
