# 上传 Hugo 主题分享剧本

> 原文：<https://dev.to/statnmap/blogdown-hugo-themes-to-share-scripts-22mm>

我修改了两个 Hugo 主题，以便能够使用 blogdown 共享 R 代码。Hugo-future-important 和 hugo-statnmap-theme 是多语言主题，允许代码折叠、语法突出显示、相关文章列表、引用卡、SEO 图表或联系形式。下面列出的所有选项在 Github 主题的 Readme 页面中都有详细说明。在存储库的“exampleSite”目录的配置文件中还列出了它们的配置参数。
