# 开放幻灯片(会议工具)正在寻找贡献者

> 原文:[https://dev . to/Thomas junkos/open-slides-conference-tool-is-looking-for-contributors-516 e](https://dev.to/thomasjunkos/open-slides-conference-tool-is-looking-for-contributors-516e)

[我们](https://github.com/intevation)正在基于 Django 和 Angular 7 开发一款名为 [openslides](https://openslides.org/) 的会议工具。该软件处理演讲者管理、在不同媒体投影仪上显示内容、投票等。[这里的](https://www.youtube.com/watch?v=SvZ8N2SEzEg)是一部展示工具运行的影像影片。去年我们为大赦国际进行了一次测试。

目前的版本是 2.3(具有完全移动集成和材料设计的版本 3 正在酝酿中，将很快发布)。最近我们用 WebRTC 插件实现了一个基本的实时聊天。

我们的项目托管在 [github](https://github.com/OpenSlides/OpenSlides) 上。

如果您正在寻找入门级的问题，我们有[好的第一个问题](https://github.com/OpenSlides/OpenSlides/issues?q=is%3Aissue+is%3Aopen+label%3A%22good+first+issue%22)标签。
