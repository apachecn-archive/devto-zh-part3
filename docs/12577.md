# 用于 Firebase 托管的 Short fire /自托管 shorten URL 生成器

> 原文:[https://dev.to/thangman22/short-fire-self-hosted-shorten-URL-generator-for-firebase-hosting-2dn 3](https://dev.to/thangman22/short-fire--self-hosted-shorten-url-generator-for-firebase-hosting-2dn3)

在我使用了 Google、Bit.ly 等缩短链接服务后，我总是会纠结于自定义域名和 slug 功能。有些服务提供自定义域名，但没有自定义的 slug，你也需要支付一些费用来使用自定义域名。我试图在 Firebase 主机中发现一个特性，我发现 Firebase 主机支持重定向规则，但你需要手动设置。我创建了一个工具来生成重定向规则，使 Firebase 主机成为缩短链接服务。这是**短火**。您可以通过`npm -g install short-fire`尝试**短火**，并按照说明操作。[https://github.com/thangman22/short-fire](https://github.com/thangman22/short-fire)
