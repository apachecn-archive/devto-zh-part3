# 很高兴加入开发团队！

> 原文:[https://dev . to/michaeltharrington/happy-to-be-on-the-dev-team-502m](https://dev.to/michaeltharrington/happy-to-be-on-the-dev-team---502m)

我很自豪地宣布，我最近作为社区协调员加入了开发团队。现在演出已经一个月了，可以肯定地说我很喜欢。

首先，承认:我不是一个开发者。然而，尽管我从未[完全致力于](https://dev.to/michaeltharrington/baby-steps-44fd)学习*如何*发展，但我已经*学到了很多关于*发展的东西。

在过去的几年里，我一直在软件领域工作，担任编辑和社区经理。我非常热衷于支持喜欢写作的开发人员，参加会议，在网上搜索很酷的开发内容，和开发人员交朋友。

当我偶然发现 DEV 时，我立即爱上了这个平台和社区。每个角落都有那么多超级友好和乐于助人的人，这让我印象深刻。我马上注册了，不久前庆祝了我的[一周年开发纪念日](https://dev.to/badge/one-year-club)。虽然到目前为止，我基本上是一只墙上的苍蝇，但我喜欢参加别人的集会，阅读适用于我的建议。毕竟，这里不仅仅是教程，还有很多关于在技术和一般生活指导中工作的共享智慧。🙂

那么，社区协调员到底是做什么的呢？我的混合职责主要围绕推广这个社区正在创建的所有精彩内容，支持有技术问题的人，并帮助确保每个人都遵守行为准则。

到目前为止，这是一次有趣的旅行。我正在学习很多东西，与一些优秀的人一起工作，并且很高兴成为这个超级酷的 ~~dev~~ DEV 社区的一员！
