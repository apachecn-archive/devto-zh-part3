# 描述某人职位或工作的名字。

> 原文：<https://dev.to/ijash/a-name-that-describes-someones-position-or-job-263e>

阿米巴虫，有时写为“阿米巴”，是一个通常用来描述单细胞真核生物的术语，它没有确定的形状，通过伪足移动。伪足是细胞的暂时突起，这个词字面意思是“假脚”。这种细胞利用伪足作为移动的方式。阿米巴的复数是“阿米巴”，不是“变形虫”。虽然“阿米巴”或“变形虫”一词通常用来指所有利用伪足移动的原生动物，但“阿米巴”一词(用斜体和大写字母书写)指的是一个特定的原生动物属，其中变形阿米巴是最知名的物种。变形虫的细胞质包含细胞器，并被细胞膜包裹。变形虫利用一种叫做吞噬细胞的过程来获取食物。在这个过程中，阿米巴原虫的细胞膜突起延伸并包围食物颗粒，将其完全包围。食物颗粒就这样被内化在一种叫做空泡的“气泡”中。食物颗粒可以在液泡中被消化。变形虫通过有丝分裂繁殖。

一些阿米巴原虫会导致疾病。例如，棘阿米巴原虫可导致人类阿米巴角膜炎和脑炎。

变形虫、病毒和细菌都是微生物。对微生物的研究叫做微生物学。
