# 如何构建一个(Drake) Twitter 机器人

> 原文：<https://dev.to/charlieinthe6/how-to-build-a-drake-twitter-bot-27g9>

2017 年假期，我和一群老朋友在酒吧。当我们排队的时候，其中一个人提到他听说过一个推特机器人，可以用其他 Biggie Smalls 的歌词来回复包含 Biggie Smalls 歌词的推文。

我觉得这个想法很搞笑，喝了一六杯啤酒后，我觉得这是我必须要做的事情！然而，我想做一些不同的事情:

*   我代表我住在多伦多，所以推特机器人应该是给德雷克的，也就是毛毛机器人
*   毛毛机器人应该用歌曲的下一行来回复包含德雷克歌词的推文
*   “毛毛雨”机器人应该在没有人类干预的情况下定期运行

经过一天的编程、修补和咯咯笑，毛毛机器人诞生了。
