# 通过异步提高工作效率

> 原文:[https://dev.to/m1guelpf/go-async-for-productivity-9ik](https://dev.to/m1guelpf/go-async-for-productivity-9ik)

[https://app.blogcast.host/embed/298](https://app.blogcast.host/embed/298)

在过去的一年里，我读了很多关于从手机上卸载社交媒体应用或禁用通知的故事，但直到最近，我才开始关注这些。我是一个青少年，在我这个年龄的每个人都频繁地使用他们的手机；我以为。我并没有从工作中分心，对吧？

后来我的成绩出来了，事实证明我确实有一份工作:学习并取得好成绩。问题是你需要非常专注才能记住东西，而我不擅长专注，尤其是在有网络连接的情况下😅。

我开始研究如何集中注意力和避免分心，并找到了一些很好的资源，如 [Brain.fm](https://brain.fm) (因为显然金属对学习没有好处，谁会想到呢)但我仍然无法完全专注于我的书。

有一天，CleanMyMac 不小心关闭了 Discord 和 Whatsapp。当我上床睡觉时，我注意到我已经更加专注了，并意识到我在学习期间没有收到任何消息，因为我认为我已经打开了应用程序，通知会出现，所以我没有检查它们。

那一刻，我明白了“删除你的社交媒体应用”的建议。今天的平台是即时的，你应该同步处理它们:如果我收到通知，我会停下手头的工作去回复或转发。

你必须去异步。当你工作的时候，你不需要知道是否有人给你发信息。禁用通知。当你完成后(当你想知道是否有人给你发信息时)，打开应用程序。**这不应该取决于信息的可用性，而是取决于你接受信息的意愿。**
