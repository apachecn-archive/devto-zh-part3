# Nemo 文件管理器(GNOME 文件的一个分支):恢复地点书签

> 原文:[https://dev . to/nab bisen/nemo-file-manager-a-fork-of-gnome-files-recoverying-bookmarks-of-places-2346](https://dev.to/nabbisen/nemo-file-manager-a-fork-of-gnome-files-recovering-bookmarks-of-places-2346)

[Nemo](https://github.com/linuxmint/nemo) 是一个开源的文件管理器，是 [GNOME](https://www.gnome.org/) [文件](https://wiki.gnome.org/action/show/Apps/Files)的一个分支，是 [Linux Mint](https://linuxmint.com/) 的 [Cinnamon](https://github.com/linuxmint/Cinnamon) 桌面环境的组件之一。

Nemo 有一个名为 Places 的功能，可以进行书签管理。
