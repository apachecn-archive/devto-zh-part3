# 为您的数字图书馆延迟加载图像

> 原文：<https://dev.to/jsrn/lazy-loading-images-for-your-digital-library-3m9o>

一个快速的上午(或下午，在这一点上)项目来玩懒惰加载的魔法。我之前在我的个人网站上写过关于我如何在年初[处理东西](https://jsrn.net/personal/2019/02/05/the-month-of-deleting.html)。像许多人一样，我相当傲慢地认为自己是读者*。书籍是作为战利品积累起来的。我读过的书占据了相对较小的空间，但我打算读的书威胁着要把书架直接从墙上扯下来。*

 *这在日语中叫做“T0”tsundoku

所以，很多书不得不搬走。然而，人们倾向于把书放在身边有两个原因:

1)我可能会再读一遍。我喜欢在那里吃。

第一点值得商榷。第二，更难反驳。我想收回一些空间，同时确保我不会忘记这些年来我读过和喜欢(或不喜欢)的书。

如前所述，我想玩懒惰的图像加载。
