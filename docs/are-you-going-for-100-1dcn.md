# 你 100%跑了吗？

> 原文：<https://dev.to/k1pash/are-you-going-for-100-1dcn>

几年前，我为创造新事物、重复想法和前进的动力而激动不已。我的风格包括盲目地投入一个项目，并在许多项目上碰壁...很多问题。例如，我想创建一个多人射击游戏，但结果却一团糟。唯一的结果是经验。

自从我得到第一份合同工作后，我真的停止了前进。我发现不可能在其他副业中重塑自我。唯一的例外是 game jam，你必须在 48 小时内创建一个游戏。时间有限，无忧无虑，两天免费玩乐。

我意识到那是我朋友的原因。在我做常规工作的日子里，他直接跳入一个新的问题并解决它。他实际上是被这种进步所驱动的，这是非常珍贵的。

我们在 100%运行吗？我们真的想吗？发生了什么事？

对我来说，解决办法是加强时间管理。放弃游戏等非生产性的东西。小目标，提前计划，在其他项目上与他人合作，跟踪任何事情的进展。另外，我不是一个早起的人，所以早起实际上是非常累人的，虽然我睡眠不足，但那些管理任务是无法完成的。

有时候，会有一些纯粹热爱编程的时刻，你可以参与任何项目，但这些时刻是在很多条件下发生的。例如，健康的睡眠周期，没有压力，总体来说就是快乐。但是谁总是快乐的。总是全力以赴的人。

我想我朋友的形象有点模糊，因为我只在他的巅峰时期听说过他的作品。

找一份工作，为别人开发程序或游戏，以金钱为主要驱动力，这种令人沮丧的想法只会伤害自我反省。

你觉得我该怎么办？你的第一份工作怎么样？
