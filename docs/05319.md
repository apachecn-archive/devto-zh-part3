# [](#mommy-i-dont-feel-so-good)妈咪，我不太舒服

所以，你做一两年程序员。你太害羞，太无能，不敢自称工程师。你知道一些事情，但是看起来计算机科学的领域对你的小脑袋来说太大太可怕了。你觉得自己理解不够，无法大声说出来，所以你试图保持沉默，直到被直接问到。你觉得自己入错了行，最好去给别人割草或当出租车司机。

不要。我想给你看一种有趣的生物。

# [](#birdiebirdie)小鸟-小鸟

这是一只孔雀，美丽而光荣的鸟。更准确地说，它是一只孔雀，雄性孔雀。孔雀由两部分组成- *鸟*和*羽毛*，背景的这个东西，比鸟本身大。羽毛不仅是外表上引人注目的细节，它还有一个功能。它的目的是吸引配偶，并创造一个巨大和自信的合作伙伴的印象。否则，它就被用来分散更大的食肉动物的注意力，这些动物渴望在途中从可怜的孔雀那里获得食物。这就是它们如何在它们通常居住的
肮脏危险的森林中生存下来。

# [](#be-a-birdman-show-yourself)做一个鸟人，展示自己

为了在你的森林中生存和繁荣，你必须成为一个，至少在一段时间内。要发展壮大，得到更有趣、不那么令人沮丧的工作，你必须看上去比实际情况更合格。这并不难。知道它的完美候选人是什么样子，并模仿它，长出合适的羽毛。一个完美的候选人应该对这份工作感兴趣，他应该*表现出技术优势*并采取相应的行动。人们喜欢聪明的**和随和的**知道事情应该如何运作的人，更重要的是这些知识与他们自己的相匹配。为了*恰当地传达美德*，你最好参考你所在领域的最佳实践和常识。阅读关于过程、代码、架构的高层次文章，了解他们的观点，并展示你对这些实践的了解。显示你有**风度**。我们是不久前才走出森林的类人猿，所以我们的大脑在抵制刻板思维和群体氛围方面并不那么先进。如果你表现出你和你的面试官/经理/高级工程师在如何开发软件上有共同的想法或原则，他们更有可能喜欢你。这意味着你必须是一个爱炫耀的人，站起来大声说出来，即使你可能会因为你内向的天性而感到不舒服。

积极主动也是吸引力的一部分，可以为你创造一个伟大工程师的名声。即使你不知道如何做一些困难的工作，试着去做，这是成长的唯一途径。关于这类任务的唯一警告是，确保你能够很快看到悲伤和浪费时间的深坑，并在你浪费雇主太多钱之前撤退寻求帮助。主动的另一面是
可见性。工作是伟大的，伟大的工作更伟大，尽管，从你的上司的角度来看，伟大的工作是不值得的，如果它是无形的。向你展示你的工作，解释它的价值并得到表扬。努力提高你的声誉，改变别人对你的看法，增加被提升的机会，获得更高的薪水和推荐。即使世界上所有的程序员都喜欢独自处理他们的代码和产品，而不需要考虑他们的吸引力，我们生活在一个不完美的世界，所以如果不使用这些技术，你会把你的繁荣交到命运和随机机会的手中，这是不够可靠的。

# [](#harder-than-you-think)比你想象的难

好了，简单的部分完成了，现在开始最难的部分。你光彩照人，积极主动，人们喜欢你。这很好，你不想让它停下来。但是如果你不采取行动，迟早羽毛会掉下来，只有一直装腔作势的小鸟才会面对现实世界，面对太难完成的任务，面对对你狭窄的技能来说太宽泛的工作，面对对你没有吸引力的主管。为了避免这种情况，你必须努力适应你的外表，真正成为你假装的那个人。从你得到的更好的工作中学习，感谢美德信号和给人们提供他们想听的东西，锻炼你的礼貌和你所在社区的最佳实践，以首先找到他们的核心和原因，获得你同事的每一份有价值的文件、指南、信息和工作流程，为这份工作制作你的实际框架。那么迟早你不需要展开你的羽毛来显得更大，你自己就会变大。

呀，你现在不害羞也不无能了。你已经成为一名工程师了。把这些羽毛挂在墙上，自己去喝一杯。你太棒了。一直都是。
