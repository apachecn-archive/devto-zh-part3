# 是只有我还是超级用户一直在失去电力？

> 原文:[https://dev . to/tinmanjk/is-it-just-me-or-power-users-been-lose-these-power-1knf](https://dev.to/tinmanjk/is-it-just-me-or-power-users-have-been-losing-their-power-1knf)

我不知道是我只是年龄大了，还是过去 5-7 年的趋势对于一般的电力用户来说真的那么糟糕。

# [](#1-laptop-gt-thinkpad)1。笔记本电脑- > ThinkPad

ThinkPads 曾经不仅仅是笔记本电脑。它们一直都是形式上的**函数。在外行看来，它们很丑，但对我来说，我知道欣赏内在(和外在)的东西。耐用性、可维护性、出色的键盘，当然还有指点杆。但是慢慢地，设计师们开始寻求尝试**“现代化”ThinkPad** ，让它越来越通用。
原来的键盘换成了新的岛式键盘。接下来是跟踪点下面的物理按钮。为什么不把物理按钮的便利换成基于触摸的错误按钮(并且不做任何可用性测试)？幸运的是，这项实验只持续了一代人。下一步——升级硬件太容易了，为什么不让用户拧开所有的螺丝，卸下整个底盖，让升级变得更难呢？我几乎已经忘记了 ThinkLight - gone。**

# [](#2-os-gt-windows7)2。操作系统- > Windows7

Windows7 曾经是一个操作系统。而不是像 Windows10 那样的操作系统即服务。如果我愿意，我可以无限期地禁用所有更新。没有 Windows10。我将被迫终身成为 MS **beta 用户**(如果我决定升级的话？)

# [](#3-smartphone-gt-bb10)3。智能手机- > BB10

BB10 是(现在仍然是)我体验过的最好的移动操作系统。没有 home 键。所有导航都是基于手势的。伟大的设备搜索，最好的统一通知系统，最好的股票应用程序有->文件管理器，浏览器，日历等，超级原始设备解锁机制通过图片密码(youtube it)，设备加密，Android 运行时和...物理键盘。现在，如果我想使用物理键盘，我唯一的选择(因此别无选择)就是运行劣质操作系统 Android 的劣质 Key-x 系列。

# [](#4-browser-gt-firefox)4。浏览器- >火狐浏览器

火狐曾经是完全和容易定制的。不再是了。
对我的第一个打击是全景功能被移除。为什么？似乎没人用过它。太好了。接下来，让我们干掉所有让 Firefox 成为现在这个样子的插件。为什么？*无论如何，我们都想和谷歌一样快！！！*接下来，让我们从我们的存储库中删除这些附加组件，就好像它们从未存在过一样。
为什么？我们真的想与我们的遗产断绝关系。接下来，这就是我要做的，让我们隐藏“从不更新”选项。为什么？似乎没有人在使用它。是啊，没错。
