几乎每次我想从网页中获取颜色时，我都会打开开发工具，选择任意随机元素并创建一个颜色 css 属性，将其设置为随机颜色。之后，我点击小颜色方块，它将切换吸管工具来选择我首先想要的颜色。

“但这与你的投资组合网站无关！”-让我解释！每次我在属性里放一个随机的颜色，我只要按一个键就可以得到任何颜色。这个键大部分时间是“r”，选择“rebeccapurple”颜色。随着时间的推移，我开始非常喜欢这种颜色，因此，在我的第 n 次作品集重新设计迭代中，我想使用 rebeccapurple 作为主要颜色。

我还想传达一种“古老”的感觉，而不是去追求一个过于饱和的设计。一些能让你想起旧游戏的东西？我不知道我是否达到了目标，但我对最终的结果感到非常满意。

事不宜迟，我想让我的投资组合网站由我们的真棒社区审查。[这里是](https://www.bakedbird.com)

你也希望你自己的投资组合被审查吗？我很想看一看！
