作为一名初级程序员，尽快熟悉数据类型非常重要。首先了解它们是从自下而上开始的好方法，而不是先考虑数据结构，甚至是类和类的实例，因为对于初学者来说，这些术语可能完全陌生，可能会令人不快。

所以，从最基本的基础开始，采用自下而上的方法，这里有一些 Ruby 中的基本数据类型。

#### 整数

整数是整数(没有小数的数字)。

示例:

one

seven

-21

3_000

你可以用这些在 Ruby 中做数学，使用和你在小学时一样的操作符。

1 * 7

=> 7

4 + 10

=> 14

8–5

=> 3

45 % 10

=> 5

*(注:*使用%符号时，其称为模运算符；模运算符返回余数。)

6 / 3

=> 2

3 / 4

=> 0

*(注意:*我们当然知道 3 除以 4 不是零，但是因为我们对两个数字都使用了 integer 数据类型，所以 Ruby 返回一个整数。实际的答案需要呈现为“float”数据类型，我们将在下一步讨论。)

#### 浮动

带小数的数字。

示例:

Five point five

-2.75

Nine point three

你也可以用这些做数学，哦，天啊！关于这些返回值，您注意到了什么？

4.2 + 5.3

=> 9.5

3.0–5

=> -2.0

3.0 / 4.0

=> 0.75

(*注:*还记得之前我们专门用整数来执行这个相同的函数时，返回的是一个整数吗？现在我们使用了 float，返回了一个 float，这在数学上也是正确的！)

3.0 / 4

=> 0.75

*(注意* —你不需要两个数字都是一个浮点数来返回一个浮点数。即使公式中使用的数据类型中只有一种是浮点型，Ruby 也会返回浮点型。)

#### 字符串

字符串是引号之间的字符。引号可以是双引号或单引号，但它们需要匹配。

“12345”

嗨，新朋友，我叫亚基拉。我们去买面包吧。

“没有他 d！d naW+"

你可以用字符串做一些很酷的东西，一个好的起点是插值。

为此，请遵循 Erb 或 pry 中的以下步骤:

首先，设置一个以字符串形式保存某人姓名的变量。

然后，创建一个新的字符串，并将变量#name 放入其中。

看返回值！

示例:

name = "Akira "

“嗨，新朋友，我叫#{name}”

= >“嗨，新朋友，我叫亚基拉”

很酷吧。

有各种各样很酷的事情可以用字符串来做，但那是另一个时间的另一个帖子。

#### 数组

数组对我来说，开始变得非常酷。

它们是用逗号分隔的有序列表，用方括号括起来。

它们可以包含大量数据(称为元素)，也可以完全为空。

它们可以保存任何类型的数据，数据类型可以是混合的(一些字符串，一些整数，甚至其他数组！).这通常不被接受，因为它不是数组的最佳用途，但是它是可以做到的。

注意:所谓“有序”，我是指存储在数组中的元素是按顺序索引的。它们从 0 开始(我知道，很奇怪)，然后从 1，2，3，4，5，6，等等。

数组示例:

[]

[1, 2, 4, 5, 6, 7]

[7.9, 8.98, 0.25]

["凯瑟琳"，"瑞秋"，"布莱恩"]

[[0,1], [2, 9], [6, 98], [34, 876] ]

【《十七》，17，17.0，[17]】

利用数组有很多令人惊奇的方法，我将在另一个时间写另一篇文章。
