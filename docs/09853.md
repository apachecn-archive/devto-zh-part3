我们有了一个新标签，Blogcast 标签！

Blogcast 是一个为你的文章生成音频版本的服务，这个标签可以让你嵌入一个播放器来在 Dev.to 上收听音频版本

玩家长这样:

```
{%  blogcast  26  %} 
```
