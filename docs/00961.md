# 为您的 DEV 文章引入横幅生成器

> 原文:[https://dev . to/Christopher kade/introducing-a-banner-generator-for-your-articles-on-dev-353 a](https://dev.to/christopherkade/introducing-a-banner-generator-for-your-articles-on-dev-353a)

我一直在努力为我在 DEV 上的文章寻找漂亮的图片(以及事实上我被允许使用的图片)。所以我想我应该做一个简单明了的快速横幅生成器。

[链接](https://christopherkade.com/banner-generator/)

[![gif](../Images/eaca610f33b0880b39ae4d2aa21e1acc.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--hIN7bI7F--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_66%2Cw_880/http://g.recordit.co/tYJhy9QcPH.gif)

你要做的就是:

*   编辑您自己的横幅
*   下载它
*   上传到开发，以便获得一个网址

[![screenshot-2](../Images/13ecc3f1ed7a4fbb78c73d19635788a7.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--KZxh_FIO--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/wifdi16t7ssuhsgr0a8n.png)

> 你可以通过点击文章编辑器左下角的底部来上传你的图片！

*   用上一步获得的 URL 在文章的元数据中添加`cover_image`属性

就是这样！

虽然它的主要功能已经完成，但我会在未来几天尝试添加以下内容:

*   更多主题
*   减小文件大小
*   评论中还有其他建议吗

我真的希望你们中的一些人能像我一样经常使用它😄
