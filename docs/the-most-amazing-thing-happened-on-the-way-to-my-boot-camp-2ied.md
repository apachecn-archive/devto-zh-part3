在上面的帖子中，我谈到了当我决定全力以赴加入一个全职的沉浸式训练营时的挣扎和兴奋。自从我上一次写下我在做出巨大的生活改变时的兴奋和恐惧以来，它已经成为一股旋风。当我在努力控制我的财务状况，并试图看看这些数字如何匹配时，也就是说，我将如何依靠我妻子的教师工资度过难关？完全出乎意料的事情发生了。由于我有一点开发经验，兼职自由职业者和在我的系统管理员工作中建立前端网站，我的投资组合网站在 React 中看起来足够体面。我收到了一些招聘人员的回电。我让他们知道了我即将到来的新兵训练营的开始日期，并告诉他们我希望在这一点上提高我的技能。

我从来没有想到会从我面试的三家公司中的任何一家得到一份工作，或多或少地经历了个人成长和经验的面试。我最终收到了两份相互竞争的聘用通知，其中一份通知想让我尽快通过招聘程序，让我在第三家公司(亚马逊)面试我之前入职。事实证明，他们把亚马逊视为直接竞争对手，并乐于在有机会之前让我加入。这种事情对你来说很有趣。我感觉很糟糕，不得不在开始日期前取消我的训练营，我会学到很多东西，能够建立一些很好的联系，并获得乐趣，但老实说，我怀疑我是否能够保持光明。所以一切都变好了。我觉得自己很幸运，边做边学是提高技术水平的一种非常好的方式。

我只是想记下这一点，并谈谈通向发展职位目标的许多途径。坚持下去，在面试中变得更好，并享受这个过程，是帮助你获得正确心态的一件事。喜欢了解不同的公司，在面试中问很多问题，钻研，享受，这很有趣。我希望写一些我一直在学习的有趣的事情。

从另一边，
