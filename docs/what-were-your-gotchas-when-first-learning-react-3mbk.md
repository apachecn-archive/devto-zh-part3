# 当你第一次学习反应时，你的“陷阱”是什么？

> 原文：<https://dev.to/joshua/what-were-your-gotchas-when-first-learning-react-3mbk>

我发现人们在学习反应时有一些常见的绊脚石，有时会有“啊哈！”当他们明白的时候。

*   ES2015、React、JSX 之间的区别
*   示例代码中复杂的库用法，如下划线或 lodash
*   理解 React 的整体“数据流”(以及作为概念的“流动”架构)
*   分享(但不一样！)其他框架中的概念，如 Angular、Vue 甚至 jQuery
*   一些操作的异步本质(如`setState`)

当你学习 React 的时候，有什么特别棘手的事情需要处理吗？你的“啊哈”是什么瞬间？
