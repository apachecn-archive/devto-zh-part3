# 和...我回来了

> 原文:[https://dev.to/jaysonjphillips/and-i-ve-returned-5gf1](https://dev.to/jaysonjphillips/and-i-ve-returned-5gf1)

今天是我回到三部曲教育服务公司的第一天。而且……这就像是某种“欢迎回家”。我很享受在[比特币基地](https://coinbase.com/)的时光，和一大堆才华横溢的人一起工作，他们肯定给了我“加密 MBA 速成班”，但我有机会回来担任一个我无法拒绝的角色:帮助塑造 Trilogy 在线教育平台和产品的未来。

因此，我回到了一个技术团队，这个团队在我离开后成长了很多，我回来后担任了这个团队的工程总监，这是一个非常重要的角色:让我们进入下一个阶段，而且要快，把质量和世界级的学生体验放在最前面。

作为加州大学伯克利分校扩展部的兼职讲师，我非常兴奋能够参加一个项目，在以前的形式中，我尝试进入它的异步学习的新模式。
