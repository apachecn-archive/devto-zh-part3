# 今年不要下决心。

> 原文：<https://dev.to/itsjzt/dont-set-a-resolution-this-year-1h4n>

### 这次试着设定每月的 SMART 目标。

*埃斯蒂·扬森斯在 Unsplash 上拍摄的照片*

我们又回到了一年的另一个年末，并制定新年计划。每一年我们都有新的目标和改变一切的动力。今年的大部分时间你都精力充沛，但日复一日，动力会下降，一些目标甚至在短短几周后就看起来不相关了。随着每一天的过去，你都在重新评估你的决定，并试图认为你现在无法实现目标。你的时间越来越少，工作却越来越多。

新年决心是什么样的？设定不切实际的目标，期望过高。典型的新年决心就像“今年我要写一本书”。接下来会发生什么，你写了几页，取得了一些进展，获得了一些乐趣，现在事情开始发生变化，你工作越来越累，陪伴家人的时间越来越少，你就是想不出再写更多了。你忘记你的决心，回到你正常的生活。今年不会像这样了。而不是做出不切实际的期望和不相干的目标。少做

相信“少即是多”的哲学，为更少的日子做计划，做更少的期望，少一些死板和不灵活。不要马上计划一年，要计划几个月。人们相信你可以在 21 天内养成一个新习惯，所以计划一个月似乎是一个很好的选择，同时仍然有足够的灵活性来改变和适应你的时间和生活方式。从小处着手，不要让你的决心随着时间变得无关紧要，被这个快速发展的世界的步伐所抛弃。比方说“你想今年开始关注你的健康”，现在为了健康，你可以开始做运动，管理你的饮食，冥想等。，这个月从冥想开始。制定一个像“这个月每天冥想 5 分钟”这样的小目标，并努力完成它。制定一个月的目标，并跟踪你的进展。虽然它们可能看起来无关紧要，但它们对实现长期目标特别有用。事实上，在不确定的情况下，它们更重要。

像“今年减肥”这样的目标是行不通的，因为你无法衡量它们。你不能说你达到了目标，因为你没有决定一个具体的可衡量的目标，像“本月减肥 2 公斤”这样的目标是一个更好的目标。SMART 标准是一个被广泛接受的战略目标设定框架。助记法 SMART 意味着目标应该是具体的、可衡量的、可实现的、现实的和有时限的。

除非你跟踪目标，否则你无法衡量你的目标是否成功。跟踪很重要，最简单的形式是，你只是在记录你是否完成了目标。如何记录这一切完全取决于你自己。你可以使用纸笔、Excel 表格、文本文件、应用程序、挂历——任何东西。

跟踪的第二大优势是用于分析的有形数据。我们太容易欺骗自己了！当你把一切都记在脑子里的时候，幻想和幻觉就和事实混在一起了。你可以为任何行为辩护。

### 结论

对大多数人来说，建立新的决心并不是一件很好的事情。今年不要设立不切实际的目标，而是尝试一些更好的东西。从小处着手，制定一个月的计划并跟踪你的进展。这样你可以很容易地完成你的目标，实际上取得进展。
