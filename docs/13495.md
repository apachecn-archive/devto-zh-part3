# 使用挂钩

> 原文：<https://dev.to/stereobooster/usedumbhooks-206n>

这篇文章的灵感来自于 Sara Vieira 的演讲“打造哑巴 sh*t”。

学习东西的最好方法是使用它，玩它(皮亚杰建构主义理论的原始解释)，对吗？但是不要建立另一个 todo 应用程序，因为它很无聊，当一些无聊的东西很难学习时，你需要在你的大脑中强有力地推动它。相反，找一些有趣的任务，在这个过程中你可以练习新技术。造一些好玩又没用的东西。

所以我想更好地理解 React 的钩子，我想知道你能想到的关于钩子的最荒谬的想法是什么？

我已经有了一些想法:

*   useFaviconProgress -在 favicon 中显示进度，参见 [faviconx](https://github.com/nicolasbize/faviconx)
*   显示类似 youtube 的进度条，参见 [nprogress](https://github.com/rstacruz/nprogress)
*   在 web worker 中使用 Greenlet - run 函数，参见 [greenlet](https://github.com/developit/greenlet)
*   使用弹出菜单
*   useSocket -订阅 websocket

构建有趣、简单、愚蠢的钩子。在评论中分享你的想法和/或实现。
