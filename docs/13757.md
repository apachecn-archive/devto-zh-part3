# Github stargazers 信息收集工具

> 原文:[https://dev . to/0x prateek/github-stargazers-information-gathering-tool-3bnf](https://dev.to/0xprateek/github-stargazers-information-gathering-tool-3bnf)

链接:[https://github.com/0xprateek/stardox](https://github.com/0xprateek/stardox)

Stardox 是一个先进的 github stargazers 信息收集工具。它从 Github 中提取信息，并在列表树视图中显示出来。它可以用来收集你的/某人的储存库观星者详细信息。
