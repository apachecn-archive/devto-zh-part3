# 什么是加密货币交易所？

> 原文:[https://dev.to/ishwar/what-is-cryptocurrency-exchange-aj8](https://dev.to/ishwar/what-is-cryptocurrency-exchange-aj8)

加密货币交易所允许用户购买/出售加密货币。
加密货币交易所或数字货币交易所(DCE)是允许客户交易加密货币的业务，
允许客户用数字货币交易其他资产的业务，如传统的法定货币，或不同的数字货币。它们可以是做市商，通常将买卖差价作为其服务的交易佣金，或者只是作为撮合平台收取费用。

一个加密货币交易所是如何开始的？

一个交易所要运转，需要同时有出价和要价。如果没有要求，那么没有人可以购买特定的加密货币，因此没有交易量。
