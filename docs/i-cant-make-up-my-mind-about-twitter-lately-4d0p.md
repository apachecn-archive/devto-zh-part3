# 最近我对推特拿不定主意

> 原文：<https://dev.to/victoria/i-cant-make-up-my-mind-about-twitter-lately-4d0p>

或者看起来是这样。

不久前，我创建了[短命](https://github.com/vickylai/ephemeral)，这是一个 AWS Lambda 功能，可以从我的时间线中删除超过一定年龄的推文。我给[写了这篇文章](https://dev.to/victoria/why-im-automatically-deleting-my-old-tweets-using-aws-lambda-1b81)，花在 Twitter 上的时间越来越少。

与此同时，我运行了另一个 Lambda 函数，将我的博客链接到我的时间线，希望我可以推动一些参与，而不会看起来死气沉沉。我通常从中获得良好的结果(还没有人认为我死了)，新的有趣的人发现并关注我，发表评论，作为回报，给我一个关注他们的机会。

最近我收到一些反馈，嗯，一个只包含我的博客文章链接的时间表变得有点无聊。当然，由于到目前为止我的时间和文章数量有限，开始有一些重复。坦白地说，我同意这个评估——我只是不知道该怎么办。

我继续吸引新的和有趣的评论和追随者，所以我不想完全扼杀这个功能。当然，我也不想让那些已经在关注我的更新的好心人感到厌烦。

因此，我开始更多地在 Twitter 上闲逛，一周总共花了大约 3 个多小时，这对一些人来说很正常，但对我来说很多。

我想知道是否有一种程序化的方法可以让我的推文更有用，更有趣，或者不那么无聊，同时还能分享我的链接。我想过加入一些有用的信息，比如天气或者到目前为止我的洗衣篮里有多少只袜子，或者甚至尝试用一个语言生成器来模仿我的声音。

我还没有一个好的答案。

首先，[这里是 Lambda 函数](https://github.com/victoriadotdev/author-afk),我用它来发布我的 Twitter feed 的 RSS 链接。也许在 dev.to 和开源 GitHub’verse 上向有创造力和有趣的人开放可以帮助产生一些想法。

如果你看到这篇文章是因为你是我的粉丝，感谢你的关注！你们说了这么有趣的事情。

如果你还不是我的追随者，欢迎你加入。好戏要开始了！我想。

无论如何，一如既往，欢迎投稿。
