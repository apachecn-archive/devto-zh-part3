# 不要忘记休息

> 原文：<https://dev.to/ashleemboyer/don-t-forget-to-take-breaks-3o6c>

*这篇文章也在[我的个人博客](https://ashleemboyer.com/don-t-forget-to-take-breaks)上结束了。*

你想知道当我非常努力地工作时，我最常忘记做什么吗？我忘记吃饭了。一开始这是个很搞笑的想法。比如，你怎么能忘记对你的存在如此重要的东西？(本人零想法，知道答案请告诉我。我的胃会很感激的。)

我的意思是，我明白了。很难休息，因为各种原因。我们经常感受到要尽快完成所有工作的巨大压力。你猜怎么着？永远不会结束。永远不会。如果你在食品服务行业工作，总会有人想要食物。如果你在支持热线上工作，总会有人需要帮助。如果你以写软件为生，总会有人要求更多的功能或者告诉你你需要如何做得更好。

这些例子有什么共同点？他们关注其他人。我们大多数人都乐于为他人付出时间和努力，我们不想让他们失望。如果你像我一样，你会尽最大努力让每个人都开心，甚至会把别人的幸福和安康放在自己的前面。

让我们少做那件事。你也要照顾好自己。花时间去做任何有助于你恢复精神的事情一点也不自私(只要你不伤害任何人或你自己！).你会有更好的状态来有效地完成你的工作。

以下是我自己的一些建议，当你需要休息的时候，你可能想试试:

*   在办公室外面或周围散步 5 分钟
*   站起来，在办公桌旁做几个伸展运动
*   吃点点心(或你忘记的那顿饭)😅)
*   花几分钟时间专注于你的呼吸
*   喝一杯水或一杯茶
