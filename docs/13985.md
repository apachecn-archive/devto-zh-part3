# 获得好主意的最佳方法

> 原文：<https://dev.to/hussein_cheayto/best-ways-to-get-great-ideas-884>

以下是一些简单的技巧，可以帮助你找到问题的解决方案或新业务的创意:

**1-断开**

智能手机是最大的干扰之一。你可能就要解决一个问题了，突然，电话响了，或者你收到一个通知，甚至是一条信息，把你从一个充满创新的世界传送到现实世界。

我给你的建议是训练自己每天花 5 分钟脱离现实世界。把自己隔离在一个房间里，关掉手机，把它放在另一个房间里，闭上眼睛，从盒子外面思考。想象你已经死了，你正从高处看着自己。

**2-冥想**

比尔·盖茨和许多其他成功人士每天至少冥想 5 分钟。以上建议将引导你进行冥想。冥想时，从各种信息中解放你的头脑，因此，你的大脑会有更多的空间来接受新的想法和思想。

**相关文章:** [我为什么开始编程](https://dev.to/hussein_cheayto/why-i-started-programming-35dj)

最重要的一个窍门是在你的床边放一个笔记本和一支笔。你可以把你的智能手机当作笔记本使用。在你睡觉之前，许多想法浮现在你的脑海中，因为你的大脑有时间为一个特定的问题寻找答案，因为你处于休息状态，因此你的大脑在你的无意识和你的智力中挖掘，以解决你无法解决的问题。

作为结论，我希望这些技巧能帮助你找到解决所有问题的方法，并更具创新性。
