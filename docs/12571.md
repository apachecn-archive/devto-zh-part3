# 我是个冒名顶替者

有时候我真的很喜欢照看服务器这份令人讨厌的工作

有时候，我花了半个上午写脚本来修复一些东西，我忘记了我在做什么，然后突然想到“哦，最好停下来做点工作”

然后我记得这是我的工作，*真的*这是我做的

其他时候，我和聪明的人交谈，他们说“我正在学习围棋，但我还不太理解指针”。我开始说“funcs 是通过值调用的……”然后我怀疑我是不是那个告诉他们这些的人...

我记得，*实际上*我有固定的项目，我在 Go 中有现场生产代码

当然，我不是冒名顶替者

# [](#how-to-remember)如何记忆

为了提醒自己我正在学习所有的新东西，这也是我实际上在做的事情，我在电子表格中保留了一个列表。它有 3 列:日期时间，面积，winulation

“日期时间”只是我表达时间的方式

“面积”是我一直在研究的东西。2018 年，这些领域是

*   golang
*   库伯内特斯
*   书
*   会议
*   开源代码库
*   自动警报系统
*   演讲
*   java 描述语言
*   MOOC

“winulation”是我在那个领域取得的小小成就。

以 Golang 为例，我在 2017 年末已经学会了它的绝对基础知识
我在 2018 年的一个目标是精通它。Golang 的一些 winulations 表示“对 go-spacemesh crypto 进行了 100%覆盖的测试，并提交了 pull 请求”，“使用 runtime/trace 对一个玩具 web 服务器进行了测试”，“读完了 Donovan & Kernighan 的书”以及“Soduku 程序终于工作了”。不是特别快:)”

我相信你可以从上面的列表中欣赏到，那里没有什么惊天动地的事情！但是每一件事都让我有一种短暂的成就感，这就是我试图用电子表格捕捉的成就感

下次我觉得自己是个冒名顶替者时，我只要看看单子，看看我做过的所有事情。

今年的新领域是 ML(机器学习)、松鼠(我在做一个副业项目，这是代号)和普罗米修斯(工作中的一件趣事)

我鼓励你保留这样一份清单，这是一个建立自信而不自大的好方法
