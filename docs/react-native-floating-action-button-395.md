# 反应本机浮动操作按钮

> 原文：<https://dev.to/skptricks/react-native-floating-action-button-395>

帖子链接:[反应原生浮动动作按钮](https://www.skptricks.com/2018/12/react-native-floating-action-button.html)

本教程解释了如何在 react 本地应用程序中添加浮动操作按钮。它最初是在 Android 材料设计中引入的，Gmail 应用程序是你可以看到浮动操作按钮的最好例子。这是一个圆形的图标按钮，位于屏幕右下方的应用程序 UI 上方。在本教程中，我们将使用 TouchableOpacity 和图像组件创建一个带有浮动操作按钮的 react 本机应用程序。
