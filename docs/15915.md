# 如何创建系列

如果你想知道如何创建一个系列，你需要添加

```
series: [series name]
```

to the front-matter part of your blog post header.

有关更多详细信息，请参见下面的编辑器指南。

[Ref:编辑指南](https://dev.to/p/editor_guide)
