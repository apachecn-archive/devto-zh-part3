## [](#applications)申请

我正在慢慢过渡到跨平台软件，因为我希望能够在任何操作系统上使用我的大多数应用程序。

### [](#productivity)生产力

*   Alfred 既是我的启动器、剪贴板历史管理器，也是许多令人敬畏的工作流程的入口点。

*   它有一个很棒的[社区](http://www.alfredforum.com/)和[惊人的工作流程](https://github.com/learn-anything/alfred-workflows#readme)供你使用。

下面是我用它快速运行/评估 Ruby 代码的一个小例子:

#### [](#todoist-simple-and-quick-task-manager)[Todoist](https://todoist.com/) -简单快捷的任务管理器

*   Todoist 帮助我计划我想做的任何事情，无论是短期的还是长期的(我不确定我是否追随 GTD)。
*   它也有全球快速添加热键。还有列表、优先级、强大的搜索等等。

[![img](../Images/7fd6b7a76b470c550101796e4fb19db8.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--LmRb2H77--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://i.imgur.com/QicIQmD.png)

#### [](#trello-project-management-tool)[特雷罗](https://trello.com) -项目管理工具

*   简单的项目管理工具，大量的集成。

#### [](#pipefy-automation-on-boards-and-forms)[Pipefy](https://pipefy.com) -纸板和表格的自动化

*   当我需要类似 Trello 的东西，但有更多的自动化/功能(例如，有一个公共表单，当事情移动到某一列时发送电子邮件，触发其他操作，等等)。)

#### [1 密码](https://1password.com) -密码管理器

*   用它生成我所有的密码，并通过主密码和双因素身份验证将所有东西保存在一个安全加密的保险库中。
*   我不再需要记住密码，现在我注册的每个网站都有一个唯一的密码，同时尽可能地激活双因素认证。

#### [](#mindnode-interactive-mind-mapping)[MindNode](https://mindnode.com) -互动思维导图

*   我写了很多，实际上太多了，MindNode 帮我写了我的想法的简短总结，我“连接了点”(或线)。

#### [](#pdf-expert-pdf-readereditor)[PDF 专家](https://pdfexpert.com/) - PDF 阅读器/编辑器

*   超快的 PDF 浏览器。
*   编辑 PDF 或填写 PDF 表单的最佳体验。

#### [](#contexts-window-switcher)[上下文](https://contexts.co) -窗口切换器

*   允许我模糊搜索所有当前活动的窗口。

[![](../Images/1cb3dd454e9123d098d9efe7c7f32efa.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--eifLXvum--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://i.imgur.com/vO9sDbN.png)

*   让我毫不费力地跳到正确的窗口。我可能经常有许多运行不同项目的 VS 代码实例，这让我可以在几秒钟内切换到我需要的项目。

#### [](#bartender-menu-bar-organizer)[调酒师](https://www.macbartender.com/) -菜单栏组织者

*   允许您自定义和隐藏菜单栏的内容，提高操作系统的美观度。下面是我的样子:![img](../Images/1c3f76ceca448fb9103d31b004d11993.png)

#### [](#magnet-organize-your-workspace)[磁铁](http://magnet.crowdcafe.com/) -整理你的工作空间

*   允许我使用键盘快捷键(或者更少见的，拖放到边缘)在屏幕上平铺窗口。
*   在监视器间移动窗口的快捷键。

#### [](#yoink-simplify-and-improve-drag-and-drop-on-your-mac)[Yoink](https://eternalstorms.at/yoink/mac/) -简化和改进 Mac 上的拖放操作

*   一个小的/快速的工具，我用来从多个位置/应用程序中放入文件，以便以后对它们采取行动。

#### [](#little-snitch-control-incomingoutgoing-network-traffic)[小飞贼](https://www.obdev.at/products/littlesnitch/index.html) -控制进出网络流量

*   惊人的网络工具，给你一个清晰的图片，什么连接进入你的电脑，什么是外出。
*   正确设置它需要一点时间，首先打开它，让它每隔几秒钟通知一次某个应用程序正在试图向某个服务器发送数据，以及您是否希望允许这样做，这是一个非常有见地的体验。
*   如果你想控制什么信息从你的电脑发出，什么连接有权连接到你的数据，这是很重要的。

#### [](#evernote-large-writing-scans-web-clips-and-annotations)[Evernote](https://evernote.com) -大型写作、扫描、网页剪辑和注释

*   我在这里写作。
*   如果我想从任何地方访问一个文档，我通常把它存储在这里。

### [](#utilities)实用程序

#### [](#xscope-measure-inspect-test)[xScope](https://xscopeapp.com/) -测量。检查。测试。

*   一套功能强大的工具，适用于 Mac OS X，是测量、对齐和检查屏幕图形和布局的理想工具。

#### [](#istat-menus-mac-system-monitoring-from-menu-bar)[iStat 菜单](https://bjango.com/mac/istatmenus/) -菜单栏中的 Mac 系统监控

*   伟大的系统监控工具，我用它来快速查看我的 CPU/网络/内存使用情况。
*   有一个很棒的日历弹出窗口，当你点击菜单栏中的日期/时间时就会显示出来。这是我曾经在 Windows/Linux 上使用过的东西，我很高兴又回来了。

#### [HyperDock](https://bahoom.com/hyperdock)-Dock 的窗口预览

*   当鼠标悬停在 dock 上的应用程序上时，窗口预览(缩略图)。
*   悬停在日历上时预览日历。

#### [](#paw-great-api-tool)[爪子](https://paw.cloud/) -伟大的 API 工具

*   全功能的 HTTP 客户端，允许您测试和描述您构建或使用的 API。
