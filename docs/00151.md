# 通过制作令人惊奇的游戏学习编码

> 原文:[https://dev . to/borrellidev/learning-to-code-by-making-amazing-games-bj6](https://dev.to/borrellidev/learning-to-code-by-making-amazing-games-bj6)

你还记得你的第一个编码项目是什么吗？嗯，实际上我可以…我是一个 11 岁的游戏迷，脑子里有一些疯狂的想法。**我想做一个游戏！**我要开发的游戏将会是有史以来最好的游戏。没人会想到的事情。像 Rockstar、Bethesda 和 Square Enix 这样的百万富翁公司怎么可能不考虑制作一个像我将要制作的这样伟大的游戏呢？
然后，我拿起爸爸的笔记本电脑，开始在网上搜索如何做到这一点。最终我想到了这个:

[![](../Images/aafa3df707b0a668a1d21371cd2fcbfc.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--ummFlXjN--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://d2mxuefqeaa7sj.cloudfront.net/s_0401209AE143B9813F26527CC3141D3A01D6E93FBBED38ED121AF25877AC8249_1549225629775_VisualBasicStartPage_lg.png)

一个 visual basic 速成 IDE，我听说它很容易使用，所以我想用它做一个游戏只需要几分钟。我错了。
我不仅发现为了制作一个游戏，我必须先学会编程，而且我还知道制作一个游戏是难以置信的困难。但是我不会放弃，我不会在今年创造下一款游戏的任务上失败。在大量的复制/粘贴和网络导航之后，我得出了这样的结果:

[![](../Images/1ff4da51c4baf590dcd9d0832f495470.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--EoGaZEVn--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://d2mxuefqeaa7sj.cloudfront.net/s_0401209AE143B9813F26527CC3141D3A01D6E93FBBED38ED121AF25877AC8249_1549225515558_Y2XBvF.png)

只是在几年后，我才开始通过阅读书籍、编写 C++代码和使用在线资源来真正钻研编程。我完成了很多小项目，甚至用 GameMaker studio 和其他很棒的工具制作了一些自己创作的游戏。我小时候经历，加上我做的所有项目，无论是游戏还是经典作品，都让我意识到一件非常重要的事情:

学习编程的一个最重要的因素是你能用某种语言实现什么项目，以及你能多快实现它们。T3】

也就是说:

不管你对编码有多热情，你的欲望有多强烈，如果你不实施一些很酷的项目，你很快就会失去继续下去的意愿。

这也是为什么我认为一些语言对初学者来说更好开始的原因，因为它们提供了许多你可以使用它们来创建很酷的项目和学习的方法。这就是游戏发挥作用的地方:它们是学习编程的一种有趣的视觉娱乐方式。现在网上提供的学习如何以简单的方式创建游戏的机会比我开始时高得多。你可以使用游戏引擎，在线编码工具，创建游戏并与其他玩家竞争，修改你最喜欢的游戏，提供令人难以置信的优惠，每个人都可以为他们挑选最好的选择。

而这也是本文的重点，向你解释如何通过制作游戏来学习编程！你如何使用这种媒介来创造伟大的冒险，同时学习编码并从中获得乐趣！但是首先，请记住，在展示您可能使用的所有工具之前，我想澄清几件事情:

*   制作游戏并不意味着玩游戏:我这么说的意思是，为游戏编程与玩游戏非常不同，你仍然是一名程序员，因此你仍然会面临你之前的每个人每天都面临的错误、配置问题和其他经典障碍。

*   **制作游戏不是学习编码的唯一途径:**游戏是一个很好的开始方式，但不是你可以遵循的唯一好的道路。你可以开始学习 **Html & CSS** ，然后学习 **Javascript** ，这样你仍然可以在创建酷网站的过程中获得乐趣。你可以通过制作简单的应用程序或者使用像 **Arduino** 或者 **Raspberry** 这样的系统来学习编程。

*   制作游戏必然需要学习其他东西:如果你的目标只是专注于编程，那么游戏可能不适合你。请记住，制作一个游戏需要你学习一些与编程无关的东西，比如精灵、动画如何工作、游戏循环、碰撞等等。因此，如果你只想要一种原始的、终端式的编程语言体验，那么你可能会重新考虑游戏选项。

这个指南不仅对想在 2019 年通过制作游戏开始学习编程的人有帮助，而且对已经在开发游戏并可能在这里发现一些新工具的人也有帮助。
我们开始吧！

## 游戏制作者

GameMaker 是一个游戏引擎，旨在将 2d 游戏开发带给所有愿意学习的人。

![](../Images/d08e24154e9371d689ef4b1cb6cbde14.png)

你可以使用它的拖放系统(这对学习编程的基础很有帮助)或者它的内置编程语言 GML 来构建游戏。有了 GML，你不仅能学会使用编程语言，还能随心所欲地扩展游戏。这里有很多资源，包括:

*   **游戏制作** [**学习章节**](https://www.yoyogames.com/learn)
*   **这两本使用拖拽制作游戏的入门书籍:** [**这里**](https://www.amazon.com/Game-Makers-Apprentice-Development-Beginners/dp/1590596153) **和** [**这里**](https://www.amazon.com/Game-Makers-Companion-Technology-Action/dp/1430228261/ref=sr_1_1?s=books&ie=UTF8&qid=1549224823&sr=1-1&keywords=the+game+maker+companion)
*   **本** [**本**](https://www.amazon.com/GameMaker-Language-InDepth-Guide-Cover/dp/1329419561/ref=sr_1_fkmr0_1?s=books&ie=UTF8&qid=1549224840&sr=1-1-fkmr0&keywords=learn+gml) **学习如何使用 GML**

## 代码战斗

Code Combat 是一个在线平台，遵循一个简单的目的:**通过创建引人入胜的游戏**来教每个人编码。有不同的部分，每一个都致力于开发的一个特定方面，如果这还不够，代码战斗也是一个开放的项目，允许你建立自己的水平，修改游戏行为，并帮助其他人成长。

## 一致

Unity 游戏引擎不仅是你能找到的最先进、功能最丰富的 2d/3d 游戏开发工具，它还被用于数以千计的游戏。他们中的许多人甚至成了非常著名的。

[![](../Images/f74d45e59acbd9d6d8917b226932f305.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--hhgi4X6d--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://d2mxuefqeaa7sj.cloudfront.net/s_0401209AE143B9813F26527CC3141D3A01D6E93FBBED38ED121AF25877AC8249_1549225370166_Grome_ui_objnity-engine.jpg)

有了 Unity，你将能够使用 C#编程语言来构建游戏，经过一些练习后，这种语言可以用来创建非常复杂的游戏。这个工具最好的部分之一也是你为了学习它可以查阅的资源的数量，这里有一些最好的:

## 虚幻引擎

这是另一个你可能想开始使用的杀手级游戏引擎。虚幻引擎是一个行业顶级的游戏引擎，参与创造了令人难以置信的成功的游戏。你将会编写 C++代码，以满足你对一个游戏应该如何运行的每一个雄心勃勃的想法。

[![](../Images/8389e48dbc4b1845a5470dd418ea2626.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--dLvM_ifL--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://d2mxuefqeaa7sj.cloudfront.net/s_0401209AE143B9813F26527CC3141D3A01D6E93FBBED38ED121AF25877AC8249_1549226900148_unreal-engine.png)

有了一个完整的包，将使您能够从开发的开始到最后阶段运送一个高质量的产品，虚幻引擎肯定是最好的解决方案之一，尤其是对更有经验的开发人员来说。但是不要因为它的复杂性而气馁，Unreal 还提供了一个很棒的视觉系统，叫做 BluePrint，用于创建不需要编码技能的游戏！这里有一些学习它的最佳资源:

*   **幻化引擎** [**学段**](https://academy.unrealengine.com/)
*   [**用虚幻学 c++**](https://www.udemy.com/unrealcourse/)

## 编码游戏

CodinGame 是一个基于挑战的平台，面向希望通过解决有趣且引人注目的游戏问题来提高编程技能的开发者。它有一个很棒的社区，支持超过 25 种编程语言。有了 CodinGame，你还可以在线挑战其他玩家，并以有趣友好的方式学习新的编程概念。该网站还提供了参加锦标赛的可能性，在锦标赛中，找到问题最佳解决方案的开发人员可以获得奖励，并受到重要公司的关注。

游戏开发是一种有趣的、视觉上吸引人的、友好的学习开发的方式。我希望这个指南能帮助你理解你能在游戏中找到多少力量，以及它们能在多大程度上帮助你开始学习编码。
