# 超越今年:2019 年目标

> 原文：<https://dev.to/apvarun/beyond-this-year-2019-goals-3de3>

随着一年即将结束，是时候回顾这些事件并加以改进了。2018 年是一个过山车，有很多新的经历。我学到了很多东西，从开发到设计等等。

最大的变化是我的全职工作从 React 生态系统转向 Angular。是的，我今年换了工作，我非常高兴。(**非正式**:我在[跳板](https://www.springboard.com)工作——一家教育科技初创公司。)

> **起始有角？**创建一个基本的 Angular CLI 项目并构建一些东西。你会从中学到比从文件中学到更多的东西。

在 UI 元素的设计方面，我也学到了很多。因为这是一个持续的过程，所以明年也会继续。我的另一个很棒的经历是在中国工作了一个月。那是一个完全不同的世界，我也在那里交到了一些好朋友。

除此之外，我从 Windows 转到了 Mac，尝试开始制作播客和视频系列，启动了几个个人项目，等等。

**挫折？**是的，有，也有人向他们学习。我被选为 React 集会的发言人，但由于签证问题未能成行。

因此，当我们结束 2018 年时，我想<mark>感谢所有支持和激励我</mark>实现上述所有事情的人。

### 这对 2019 年意味着什么？

门槛绝对高！但是这不应该阻止任何人去梦想😉。我的待办事项中有太多需要我关注的事情，所以我会相应地计划它。

##### 主要目标

*   🚀增加**开源**的贡献
*   👨🏻‍💻在服务器上学习和使用 **GraphQL**
*   ✍🏻每月至少写一篇博客文章
*   📹通过实时编码屏幕共享创建更多**视频**
*   📚学习 Python(Django)
*   🛫计划至少一次与家人和朋友的旅行
*   🗣在一次会议上发言
*   💪🏻定期锻炼

这些不是新年决心，而是我公开承诺的目标。我也期待着活跃在 Twitter 上，学习和支持那里令人敬畏的网络开发社区。

### 包装完毕
