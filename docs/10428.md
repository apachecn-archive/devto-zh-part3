# 软件工程基础知识:永远保持自信！

> 原文:[https://dev . to/Molly/ABCs-of-software-engineering-always-be-confidence-27de](https://dev.to/molly/abcs-of-software-engineering-always-be-confident-27de)

软件工程是一个艰难的职业。每天我们都面临着复杂的问题，有时甚至会让我们质疑自己做这份工作的能力。因此，在科技领域取得成功的关键之一就是自信。即使对自己的编码能力不自信，也需要对自己有信心。在自己身上找到自信并不容易。在这篇文章中，我想分享一些你可以用来帮助你在工作中建立自信的技巧。

## [](#1-review-senior-devs-prs)1。审查高级开发人员的 PR

高级开发人员也会犯错！没有什么比能够在高级开发人员的公关中发现 bug 或错别字更能给你信心了。最坏的情况是，你没有发现任何错误，但是你从阅读代码中学到了一些东西。当我刚开始的时候，有一个高级开发人员，他知道关于可枚举的一切，并且总是写最紧凑、最令人愉快的可枚举代码。我知道的每一个可枚举的技巧，都是从阅读他的 PRs 中的代码中学来的。

另一方面，高级开发人员，如果一个初级开发人员在你的公关中发现了一个错误，不要置之不理，给他们荣誉！一句简单的“好发现！”回应可能会比你想象的更有助于增强他们的信心。

## [](#2-teach-teach-teach)2。教！教！教！

许多人通过训练营学习，然后找到了他们的第一份开发工作，并且从未回头。在你获得一些经验后，考虑回到你原来的训练营，看看他们是否在寻找导师或导师。参加当地的结对编程聚会。帮助别人学习如何编码会帮助你意识到并关注所有你知道的东西，而不是你不知道的东西。通常，你会意识到你知道的比你想象的要多得多！

## [](#3-find-your-niche-and-be-the-champion-at-it)3。找到你的定位，并成为它的冠军

这可以是任何事情，从拥有一段代码到审查 PRs 到监控一段基础设施。你的利基是什么并不重要，只要你有一个。找到你能掌握的东西，并成为它的关键人物。每当有人需要帮助时，他们就会来找你。通过这样做，你会让自己被需要。没有什么比被需要的感觉更能灌输自信了。

## [](#4-find-your-person)4。找到你的人

[![](../Images/e852a84a988a8ede9080b8ef4eb81fb3.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--NJvQZuX1--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_66%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/2bgcqc3satpcosfukvve.gif) 
当你在一家新公司开始工作时，找到你的人。找到那个比你更相信你的人。不需要是资深 dev 可能是同行。关键是，这个人需要在专业上了解你，了解你的工作。如果是和你每天一起工作的人，效果会更好。那些你觉得自己把一切都搞砸了的日子，那个人会在那里提醒你，你还没有。当有人挑战你的时候，那个人会在你身边为你辩护。

在一家新公司开始工作可能会令人生畏，而且一开始很难找到合适的人。确保参加工作活动，白天花时间多了解你的同事。将代码与他人配对，以便更好地了解他们。这需要一些努力，但回报是值得的。

## [](#5-power-pose)5。力量姿势！

研究表明，仅仅是站在一个有力的姿势就能增强你的自信，帮助你表现得更好。不相信我？如果你需要说服的话，看看这个 TED 演讲。现在，当我做动力姿势时，我会加一点扭曲。我喜欢在镜子前做这件事，还喜欢在凉爽的跑道上背诵《少年》中的这些台词。[https://www.youtube.com/embed/_Gqwi7Y96sk](https://www.youtube.com/embed/_Gqwi7Y96sk) 在任何高度紧张的事件之前，我都会摆出力量的姿势。当我停止生产，觉得自己一文不值的时候，我会摆姿势。我会第一个告诉你这很有效。以一个充满力量的姿势开始你的一天，这可能会让你有一个好的开始。

## [](#6-envision-yourself-succeeding)6。想象自己成功了

> “你不能超越你的自我形象”——心灵健身房:运动员内心卓越指南

我读过很多关于运动心理学的书，其中一个普遍的信息是，如果你想成功，你需要想象它。你必须相信自己。当你去参加面试时，想象自己战胜了它，得到了这份工作。幻想当他们请求你为他们工作时，你会要求什么。

> “学会用你的头脑，否则你的头脑会利用你。行动跟随我们的思想和形象。不要往你不想去的地方看。”

一旦你有了能力，剩下你唯一要做的就是相信自己。这是一夜之间发生的事情吗？肯定不是！但是，如果你花时间训练你的头脑，改变你的思维方式，慢慢地你会训练你的大脑为你工作，而不是与你作对。
