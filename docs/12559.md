# 远程工作者:你的日常生活是怎样的？

> 原文：<https://dev.to/gsto/remote-workers-whats-your-daily-routine--4k8n>

我很好奇是否有人有帮助他们集中注意力并在一天中保持高效的习惯或系统，以下是我的:

*   很早(早上 6:00):起床，写作和兼职项目工作
*   清晨(7:00am):准备，早餐，遛狗
*   开始工作一天(上午 8:30):检查电子邮件，github，项目管理工具，草拟一天的计划。向队友说早上好
*   `if(Monday && 10:00am)`:每周策划视频通话。
*   `if(Friday && 9:00am)`:每周一次[咖啡协作](https://glennstovall.com/coffee-cowork/)会议。
*   `else`工作/编码
*   午餐(中午 12 点)
*   更多工作(下午 1 点)
*   下午休息/更多遛狗活动(下午 2:30)
*   当天的最终邮件检查(下午 3:00)
*   完成工作(下午 3:15)
*   关闭工作笔记本电脑，离开家庭办公室，关上门(大约下午 5:00)
*   去健身房。
*   `else`寒气逼人。

你呢？
