# 永远不要停止学习

> 原文:[https://dev.to/ashleemboyer/aim-to-never-stop-learning-38ei](https://dev.to/ashleemboyer/aim-to-never-stop-learning-38ei)

你永远不会知道一切。外面有太多的信息，没有足够的时间去了解所有的信息。这意味着，有时我们错了，因为我们不知道所有的答案。这没关系——犯错是 100%没问题的！

如果你做错了什么，没什么大不了的。这是一个做得更好、学习新事物和成长的机会。你的队友*会*比你更了解一个话题，就像你比他们更了解一个话题一样。这就是在团队中工作的好处！你们可以就这些话题互相教授，你们的技能可能会互相称赞，对每个人都有好处。

今天我的大脑相当疲劳，言语艰难，所以我想打开评论，让人们分享他们最喜欢的学习新事物的资源。你喜欢从哪里了解生产力？有哪些哲学或伦理学方面的书籍推荐？你最喜欢的 UI/UX 书籍的作者是谁？不管你有什么，我都想听。:)
