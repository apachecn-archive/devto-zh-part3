# 构建地理定位应用的正确方法

> 原文:[https://dev . to/maxchurilov/the-right-way-to-build-a-geolocation-app-KOA](https://dev.to/maxchurilov/the-right-way-to-build-a-geolocation-app-koa)

从在谷歌地图上查找方向到通过优步叫出租车，地理定位应用已经成为许多蓬勃发展的企业的支柱。但是你能开发出一个能获得类似成功的应用吗？

Foursquare 通过让用户登记入住餐厅，已经拥有了 6000 万忠实用户。优步通过连接乘客和出租车司机，形成了价值 720 亿美元的市场。星巴克通过在其应用程序中添加本地通知增加了销售额。随着每年越来越多的移动应用程序用户，这些公司的利润将继续增长。

在这篇文章中，我将分享我们在开发基于位置的应用程序 Juvo 时获得的见解。这是一款面向挪威当地社区的原生 iOS 和 Android 应用。
