# 通过示例理解 Javascript 中的 Const

> 原文：<https://dev.to/skptricks/understanding-const-in-javascript-with-examples--515h>

帖子链接:[用例子理解 Javascript 中的 Const](https://www.skptricks.com/2018/11/understanding-const-in-javascript-with-examples.html)

本教程解释了如何在 javascript 中使用 const 关键字。const 语句值只能赋值一次，不能重新分配。const 语句的作用域类似于 let 语句。

const :-它具有块级范围，但不能更改值引用。
点记:
用 const 定义的变量的行为类似于 let 变量，只是它们不能被重新赋值。
我们不能改变常量的原始值，但我们可以改变常量对象的属性。
