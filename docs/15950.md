# 在 8.1 版的 dbForge Studio for MySQL 升级版中支持 MariaDB 10.4

> 原文:[https://dev . to/devart software/support-for-Maria db-104-in-upgraded-dbforge-studio-for-MySQL-v81-2g7b](https://dev.to/devartsoftware/support-for-mariadb-104-in-upgraded-dbforge-studio-for-mysql-v81--2g7b)

Devart 是一家为数据库管理和开发提供数据库连接解决方案和工具的捷克软件提供商，它发布了一个新版本的 dbForge Studio for MySQL，支持对最新的 MariaDB 10.4 的连接。Devart 不断扩展其 MySQL 管理工具的连接选项，以确保 dbForge Studio for MySQL 的用户可以使用最新的数据库引擎。在 dbForge tools for MySQL 的更新版本中，实现了对最新的 MariaDB 10.4 的支持。在以下工具中:dbForge Data Compare for MySQL、dbForge Schema Compare for MySQL、dbForge Compare Bundle for MySQL、dbForge Query Builder for MySQL、dbForge Data Generator for MySQL、dbForge Documenter for MySQL。

dbForge Studio for MySQL 是一个通用的 GUI 工具，用于 MariaDB 和 MySQL 数据库的开发、管理和行政。IDE 允许创建和执行查询，开发和调试存储例程，自动化数据库对象管理，通过直观的界面分析表数据。MySQL 客户端提供了数据和模式比较和同步工具、数据库报告工具、带调度的备份选项等等。

dbForge Schema Compare for MySQL 是一个允许用户比较两个 MySQL 数据库结构的工具。dbForge Schema Compare for MySQL 支持 MariaDB 数据库服务器和 Percona 服务器。

dbForge Data Compare for MySQL 用于比较和同步 MySQL、Percona 和 MariaDB 数据库中的数据。该工具提供了数据之间差异的清晰视图，允许分析它们，生成同步脚本，并对目标数据库应用更改，一目了然。

dbForge Compare Bundle for MySQL 包含两个独立的工具，dbForge Schema Compare for MySQL 和 dbForge Data Compare for MySQL。

dbForge Query Builder for MySQL 是一个可视化工具，允许创建任何类型的 MySQL 查询，从简单的查找到复杂的连接。该工具提供了广泛的兼容性选项，只需点击几下鼠标就可以创建和编辑复杂的 MySQL 查询。

dbForge Data Generator for MySQL 是一个强大的 GUI 工具，用于创建大量真实的测试数据。该工具包括一个大型的预定义数据生成器集合，具有可定制的配置选项，允许用特定数据填充 MySQL 数据库表。

dbForge Documenter for MySQL 是一个方便的 MySQL 和 MariaDB 文档工具，可以自动生成 HTML、PDF 和 MARKDOWN 文件格式的整个 MySQL 数据库的文档。该工具允许用户根据自己的需要使用一系列选项和设置来定制生成的文档。
