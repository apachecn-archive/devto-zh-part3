## 你用什么在线民调工具？🤔

我正在考虑在 DEV 上添加一个投票液体标签。我认为这将为不同类型的博客帖子带来许多新的机会:

*   你会看 x 直播吗？
*   你认为这个新的插件/框架怎么样？
*   你对这个新功能感到满意吗？
*   ...

我认为选择的清单是无穷无尽的。

我想知道如果你们进行网上投票，你们会使用什么免费的投票工具？到目前为止，我们似乎不能使用 Twitter 投票，这当然是我的第一选择。

也许有些要求:

*   它必须是免费的
*   我认为内嵌 iframe 在液体标签中最容易实现

另一个选择是在 DEV 中添加本地投票支持，但缺点是投票会增加复杂性。这将(我认为)需要游客有一个帐户，以便能够投票。

你们觉得怎么样？欢迎所有的想法。

请注意，这还不是一个批准的开发功能，只是我正在投资的东西，可能会提出功能建议和公关。
