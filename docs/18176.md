# link-ga.me

> 原文:[https://dev.to/bigaston/link-game-cgn](https://dev.to/bigaston/link-game-cgn)

你好！我创建了一个新的小网站，叫做 link-ga.me。这是我去年开始的一个老项目(你可以在 [link-ga.me](http://link-ga.me) 上找到第一个版本)。这是一个受 Linkfire 网站启发的网站，但是是为游戏设计的。

在这个网站中，你可以添加一些市场链接(如 Itch.io，Newgrounds，Steam，Google Play 和其他)，一些描述，游戏名称和图片。

所有的数据都存储在我的服务器上的一个 lowdb 数据库中(这是一个本地的 JSON db 存储)。要创建一个游戏，你需要用谷歌账号登录，因为我不知道，确切地说，我不知道如何存储密码和用安全数据登录。

对于技术，我使用 NodeJS、ExpressJS、Pug、LowDB 和 Bootstrap。这不是一个完美的网站，也许我可以改进它(我必须改进它)，但我想发布这个小消息。

目前，V2 还没有开放，但可能在一两个星期后！
