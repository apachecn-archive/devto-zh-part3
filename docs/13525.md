# 我是丹尼·德·弗里斯，阿姆斯特丹大学的网络开发讲师，有什么问题尽管问我！

> 原文：<https://dev.to/dandevri/im-danny-de-vries-web-development-lecturer-at-the-amsterdam-university-ask-me-anything-1b48>

我是[阿姆斯特丹应用科技大学](https://www.amsterdamuas.com/)的讲师兼研究员[通信和多媒体设计](https://www.cmd-amsterdam.nl/english/)。

我主要在我们项目的技术课程中讲授 web 开发，如[后端](https://github.com/cmda-bt/be-course-18-19)或[项目技术](https://github.com/cmda-bt/pt-course-18-19)，这些课程侧重于 HTML、CSS、JavaScript & Node.js。除此之外，我还在次要的视觉界面设计中教授人们有关 Web 动画的知识，这些设计侧重于关键帧、过渡、处理& SVG。

除了在教室前授课，我还指导过很多学生，帮助他们在设计和代理领域导航:

*   我帮他们的作品集找到了实习机会。
*   我提高了他们的软技能，比如表达和交流。
*   我已经帮助他们选择在学习期间应该关注的技能。
*   我已经进行了一对一的会谈来设定目标并真正实现它们。
*   我对学生提交的作品给出了建设性的反馈和设计评论。

我很乐意谈论这些话题或给出任何反馈！
