# 不好意思，时间的流动突然变得可以商量了吗？

> 原文：<https://dev.to/kaelscion/im-sorry-did-the-flow-of-time-suddenly-become-negotiable-5daj>

嘿，开发世界！我 2019 的第一个帖子 woohoo！虽然我很想阐述我的“新的一年，新的我”综艺的宏伟计划，但我没有任何这样的计划。因此，我想听听你的一些关于永远不变的事情的故事:一个客户/顾客/项目经理试图“说服你放弃”一些毫无商量余地的事情，以及你是如何处理的。

你们中的一些人可能还记得我以前的一篇帖子(如果你想查看的话，位于这里)，我和你们中的许多人都有过客户试图与我谈判的经历，这些事情从奇怪到不负责任甚至是非法的。许多这样的情况可以通过简单地以清晰、冷静、简洁的方式解释你的立场或者消除你的客户可能没有意识到的误解来弥补。但是有些时候，这些要求会变得如此坚持，以至于到了咄咄逼人甚至辱骂的地步。鉴于 Dev 拥有大量的新开发人员或初露头角的开发人员，我恳请我们中更资深的人给出一些这样的例子，你是如何处理这种情况的，结果是什么。
