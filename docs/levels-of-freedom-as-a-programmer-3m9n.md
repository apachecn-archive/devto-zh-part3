# 程序员的自由度

> 原文：<https://dev.to/rlxdprogrammer/levels-of-freedom-as-a-programmer-3m9n>

作为开发人员，我们是不同的，我们喜欢不同种类的挑战。我们中的一些人喜欢只专注于开发的技术部分，其他人喜欢对管理、销售和营销主题有一个概述。根据你的喜好，你有好几个就业机会。范围很广:从绝对的雇员身份到领导自己的企业。但是在它们之间有几个步骤可能更适合你。我能告诉你的是，有很多方法可以减少自由。在这种情况下，你的任务很简单，你有固定的薪水。有很多方法可以让你更自由地就业。在这种情况下，你还负责寻找下一个要做的项目，寻找新的业务，签署新的合同，与客户沟通，你对成功交付负有更高层次的责任。通常在这种情况下，你需要处理多个主题(开发、管理、销售、营销等。)在同样的时间里，你承担了更高的风险，但你也可以获得更高的薪水。在这篇文章中，我将向你描述作为一名程序员，自由的不同步骤以及它们的优缺点。

**产品/服务开发公司的内部开发人员**

我认为对于那些想专注于纯开发的人来说，这是最舒适的选择。你是在为一家公司的产品或服务工作，仅此而已。你有固定的月薪和固定的带薪假期。独立于你的项目的成功，你得到同样的钱。你不需要处理项目合同等。这也意味着，即使产品真的成功了，并且为公司赚了很多钱，你得到的仍然只是你的工资，也许还有一点奖金，这绝对比不上公司的收益。所以在这种情况下，你有低风险，低可能的最大利润。在这种情况下，你的公司正在承担风险(你的生产成本降低的风险)。

作为产品/服务开发公司的外部人员工作

在这种工作模式中，您受雇于一家公司，该公司与一家产品/服务公司签有合同。你像公司内部一样在那里工作，所以你在他们的办公室工作，和公司内部的同事一起，有一定的任务分配给你。产品公司按时间为雇主付费，而你有固定的月薪。所以从工资和假期的角度来看，这和直接为产品公司工作是一样的。唯一不同的是，你可以根据雇主的需要，随时调到其他公司工作。这是一个熟悉不同技术、流程和工作文化的好机会。这也是扩展你职业关系网的好机会。在这里，你仍然在低风险的工作，却没有那么高的薪水。在这种情况下，与你的雇主签约的公司承担了风险，因为你的雇主是按时间支付工资的。

在一家咨询公司工作，为不同的客户开发服务

在这种情况下，你的工作是由你的雇主提供给其他公司的，但你的雇主不是基于时间，而是基于结果。这个价格可以在每个小任务之后支付，也可以在整个项目之后支付。您总是专注于一个项目，而不是每次都在客户现场。你可能在雇主雇佣的同事团队中工作。根据你公司的工作文化，你也可以参与寻找新的项目机会。你仍然有固定工资。在这种情况下，您通常会经常更改项目，为不同的客户处理不同的主题。你的风险和潜在利润仍然很低，你的雇主承担了所有风险。在这种情况下，你的雇主承担所有风险。我认为这是一个不用冒任何风险就能学习商业知识的好职位。在开始自己的事业之前，这是很好的一步。

作为自由职业者的工作时间

对我来说，做一名自由职业者意味着，你不会和一家公司签长期合同，你只是签一个短期(几个月)或一个项目的合同。您要么在现场为您的客户工作，要么在家工作。这种工作的一种方式是基于时间的支付。所以你签的是固定期限和固定价格的合同。在这种情况下，公司仍然承担风险，但你需要总是寻找新的项目，为此你需要有一个真正好的参考。这意味着你以前的客户需要对你真正满意，才能赢得高薪项目。为了达到一个好的薪资水平，你需要在你的领域做得非常好，你需要为自己建立一个好的市场。参加开源项目，在社交门户和技术论坛上建立关系网，写自己的博客和技术文章都是很好的。还要注意的事实是，在这种情况下，你没有带薪假期或带薪病假，你需要承担所有硬件和培训的费用。你需要计算这些成本。

作为一名自由职业者，以结果为基础

自由的下一步是做一名基于结果的自由职业者。这意味着你要求完成某个项目或任务的固定价格。如果你能很快实现它，你可以有很高的利润，但如果你快没时间了，你也可以有很差的利润。在这种情况下，你承担了所有的风险，但这也意味着如果你很优秀，你可以获得比普通员工高得多的利润。推销自己、建立关系网和寻找新项目也是你日常工作的一部分。在这种情况下，你通常在家或在自己的办公室工作。

作为自由职业者开发自己的产品

这里的想法是找出你自己的产品或服务，可以是例如移动应用程序或任何其他东西。开发它并尝试销售它。在这种情况下，你的风险真的很高。有可能你的产品根本卖不出去，所以你白忙活了。但也有可能一个相当简单的程序就能让你赚到数百万。记住《愤怒的小鸟》或《脸书》。在这种情况下，你需要专注于营销、市场研究等主题。

领导你自己的公司

最高的责任当然是如果你不仅对自己的工作负责，也对别人的工作负责。这意味着，如果你正在建立自己的团队或公司，公司可以按时或按结果生产自己的产品或为其他公司工作，但你有责任以获利丰厚的方式销售他们的产品。在这种情况下，通常你不再是真正的开发人员了。你需要真正专注于销售、营销和管理主题。

希望你们都能找到最合适的工作方式，如果你对目前的工作模式感到厌倦，随时可以改变你的职业。
