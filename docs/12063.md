# 今天我要发布 TAYL——一款利用人工智能将网站转化为播客的应用

> 原文:[https://dev.to/miickel/today-im-launching-tayl-应用人工智能将网站变成播客-4e72](https://dev.to/miickel/today-im-launching-tayl---the-app-that-uses-ai-to-turn-websites-into-podcasts-4e72)

1984 年史蒂夫·乔布斯介绍麦金塔电脑的视频总能让我会心一笑。从许多方面来说，这是一个开创性的工程壮举，但给我印象最深的是计算机第一次有了自己的个性。它说话了。[https://youtu.be/2B-XwPjn9YY?t=193](https://youtu.be/2B-XwPjn9YY?t=193)

自 1984 年以来，计算机、软件和人类已经走过了漫长的道路。通过站在巨人的肩膀上，我设法建立了一个捆绑了几种不同个性的小东西…

🌍它可以让你从 17 种语言的 50 多种声音中进行选择。
🔍它会自动检测文本的语言。
🎧听起来棒极了。
🤖它使用人工智能来总结长篇文章。
🤓它通过删除缓慢的难看的垃圾来使网页可读。
⚡️它超级快，超级可靠，全天候工作！

在过去的三个月里，我每天清晨、深夜和周末都在做这个项目，除此之外什么也没做。终于能够走出我的房间，并有所表现，这是一种很好的感觉。希望大家去看看！

下面是网址: [https://www.tayl.app](https://www.tayl.app)
