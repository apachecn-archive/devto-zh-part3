# 在狂野的网中看到丑陋。

> 原文：<https://dev.to/nabilahsan/seeing-ugliness-in-the-wild-wild-web-1h8j>

### 如何应对看起来很可怕的网站？

作为一名即将毕业的计算机专业学生，是时候开始找工作了。去 LinkedIn 键入前端 web 开发人员职位，选择实习或入门级，浏览每个职位发布，看看你是否有资格申请。

有时候你会想出一些和你要找的相匹配的东西。当然，这是无偿的，但在你创建自己的创业公司的同时，你可能会得到你正在寻找的导师。在你申请之前，你决定先了解一下这家公司。你肯定想知道你将在哪里实习。

结果这家公司的网站停留在 2008 年，在 LinkedIn 上，它说他们正在寻找一名网络开发实习生...你看到发生了什么...对吗？

### 那我该怎么办？

作为一名自由职业者，我应该去问他们是否需要重新设计吗？可能会有帮助。我想成为一名自由职业者。

或者我接受这份工作？我试着给他们打电话和发邮件...已经一个月了，没有任何回复。

这不是我仅有的一次经历，有好几次了。在浏览了这些网站后，我估计我可能需要 2-3 天来提出一个好的直观设计，4-5 天来建立一个完全移动响应的静态网站。

我的大学教授也有可怕的网站。我该如何接近他们？(不用担心...我在他们所有的课上都得了 a)。这可能是一个挣钱的好方法。

如果有人能帮我，那就太好了。还有，第一次发帖。耶！
