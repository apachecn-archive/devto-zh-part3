## [](#source-injecting-custom-javascript-into-react-native-webview)来源:[将自定义 JavaScript 注入 React 原生 Webview](https://www.skptricks.com/2019/01/injecting-custom-javascript-into-react-native-webview.html)

本教程讲解如何在 react 原生应用中的 webview 组件中注入自定义 javascript。injectedJavaScript 是 React 本机 Webview 组件的自定义属性。您可以将任何 JavaScript 代码(作为字符串)传递给这个 prop，React native 会将这个 JavaScript 代码注入 Webview。Webview 加载完成后，注入的 JavaScript 将被执行。
