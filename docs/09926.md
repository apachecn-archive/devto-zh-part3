这篇文章是用英语写的。为了语法正确，我必须遵循一套结构规则。

*   我，不应该，把，一个，逗号，后，每个，单词，并，声明，我的，句子，语法上，正确的。
*   。我不应该用句号来开始我的段落。
*   当" it's "的意思是"*it ' s "*时，省略撇号是一个语法错误。

上面三条都写得不好，语法上。

对于编程语言来说，语法就是这样。每种编程语言都有一个定义好的语法，它建议用户(开发人员)如何用这种语言编写代码。

不同之处在于，如果你犯了语法错误，读者可能不会完全理解你的信息，但他们可能会做出有根据的猜测，然后继续阅读。

如果你犯了一个语法错误，你的电脑将无法理解你在做什么，并且会崩溃。计算机有目的地不去猜测用户的意思，并且会崩溃以避免不可预见的副作用。

**<u>例句</u>**

如果您已经阅读了这些文章，您应该已经知道什么是[变量](https://www.codetips.co.uk/beginner/what-is-a-variable/)。每种语言都规定了如何编写变量的语法。

**Javascript**

```
var x = "Hello World" 
```

**出发**

```
x := "Hello World" 
```

**Python**

```
x = "Hello World" 
```

所有三个代码片段做的事情完全相同——它们定义了一个变量(`x`)并将值设置为字符串`Hello World`。
