# photo con/insta gram 克隆应用

这个图片分享应用程序可以让你分享图片，它有如下功能

*   张贴包含基本信息内容的图片
*   喜欢并评论
*   删除帖子
*   用于查看照片的影院视图
*   移动优先设计

## 堆

这个应用是用`React-Redux`搭建的，UI 开发框架是`Semantic ui`。

### 项目说明

该项目是使用 React-Redux 架构构建的，如果存储图像的`likes`是使用 redux store，单个状态管理器完成的。用于此响应式 web 应用程序的语义用户界面适合移动视图。照顾移动第一设计使用 axios 加载 insta.json 文件到应用程序。起初，我直接厌倦了提供的 api，但我面临着 CROS 相关的问题。

实现了用户体验的就地编辑/更新信息功能，处理了基本验证，如帖子没有评论…
