# 什么？💻

是的，我已经建立了一个 Instagram 机器人，它基本上做了以下事情:

*   打开 Instagram 登录页面
*   使用特定的**用户名**和**密码**登录
*   遍历标签的**【array】**，并在标签上循环，以获得最新的 3 幅图像
*   点击图片，喜欢它们，关闭模态
*   并重复直到停止。

# 为什么？

出于娱乐和教育目的，我不建议你使用这个代码来发送垃圾 Instagram。
