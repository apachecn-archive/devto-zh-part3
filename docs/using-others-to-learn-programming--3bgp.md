# 利用他人学习编程

> 原文：<https://dev.to/jessekphillips/using-others-to-learn-programming--3bgp>

在我的编程生涯中，我一直在从事并关注 D 的开发。我会得到课堂作业，我会做两次，一次用 Java，一次用 d。

d 还很年轻，从那以后成长了很多。我不认为 D 可以再用在这方面，但是学习该语言做出的设计选择可能是对我的编程知识最有益的。

Jai，Johnathon Blow 的语言已经在视频中体现了它的开发和设计选择。这可能不是最好的方法，因为这种语言不可用，如果是，视频中显示的版本将是旧的，而不是可用的版本。

对于 D，我已经看到了低水平的支持与高水平的糖的挑战。语言特性或库功能之战。我们可以用 5 种方法创建委托直到星期天，我们应该添加 C#的方法吗？是的，是的，我们应该。

我认为 D 是一门伟大的语言，这也是我不断回到论坛的原因，但我不禁感激我在那里为它的成长所获得的价值。
