### 针对初学者的 Git 和 GitHub

如果您是编码新手，您可能会被即将开始使用的所有新技术所淹没，不仅仅是因为它的安装，还因为它的使用本身。程序员之间使用和评论最多的一个就是 Git 和 GitHub。

但是有什么区别。Git 是一个免费的分布式版本控制系统，它被认为是现代和成熟的；它是并积极维护开源。它允许跟踪计算机文件的变化和多人的项目协作；它的一个主要特点是处理大型项目的效率。换句话说，是一个管理源代码或存储库的工具。GitHub 是 Git 存储库的托管服务，所以即使它们不相同，它们也可以一起工作:Git 是一个工具，GitHub 是使用 Git 的项目的托管服务器。

**安装 Git 并创建 GitHub 账户**

你可以按照安装链接，它会带你通过所有的安装步骤，也有通过命令行安装它的选项，但我不会包括在这篇文章中。

对于 Mac，请访问以下链接:

[http://git-SCM . com/下载/mac](http://git-scm.com/download/mac)

对于 windows，请访问以下链接:

[http://git-scm.com/download/win](http://git-scm.com/download/win)

**创建 GitHub 账户**

步骤:

1.转到 GitHub 注册页面。

[![alt text](img/05fd68cd1126c29ffbae179ed8440a05.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--wZ3KZNjS--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/cw5fhwdvcjyio04lvf5g.jpg)

2.输入用户名、有效的电子邮件地址、字母、一个数字和七个字符。

3.继续之前，请仔细阅读 GitHub 服务条款和隐私政策。点击“创建帐户”按钮，您将同时同意这些文件。

4.选择一个计划。你有两个选择:免费和付费。

[![alt text](img/8bbe9345379e53e00150f518ff57f021.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--emGqqtDo--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/p0axosp36lqk7enazjci.jpg)

5.定制您的体验。

[![alt text](img/7fcecb3dad70ce1d8b57b85810473207.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--2ySIksDp--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/hxlh3yncjl4hohe7yj55.png)

6.您的 GitHub 帐户已创建！
