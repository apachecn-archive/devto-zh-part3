下面是 github 项目
[https://github.com/EffectNode/en-node-graph-gui](https://github.com/EffectNode/en-node-graph-gui)

编辑器引擎是开源的，后端只存储数据，但出于安全原因，它是私有的。

Vuejs 太牛逼了...

1.  它可以帮助我在没有其他 UI 库的情况下轻松制作一个 SVG 图形。
2.  它可以帮助我轻松地为 iframe 构建执行环境代码，这样我就可以将用户逻辑放在 iframe 中以使事情更安全。

Three.js 太牛了...

1.  它可以帮助我使用一个场景图来组成一个树状结构的项目。
2.  它可以帮助我专注于图形编码，而不是结构设置..

我的小软件棒极了...XD lol 哈哈哈哈哈哈

[https://igraph . effect node . com/](https://igraph.effectnode.com/)

1.  它可以帮助我减少设置东西的工作量，比如调整大小处理程序，应该如何构造和组织东西。

2.  它可以帮助我用相同的场景图结构组织内容。

3.  它可以帮助我轻松地输入人类的价值观，如时间轴计时。

4.  它可以帮助我通过在线存储代码使生活变得更容易，这样我就可以很容易地分享它们。
