# 它只是 HTML

> 原文：<https://dev.to/m1guelpf/its-just-html-43po>

[https://app.blogcast.host/embed/297](https://app.blogcast.host/embed/297)

如今，互联网上充满了美丽的网站，人们很容易忘记每个网站都有一些共同之处。本质上，每个网站都只是 HTML。

受到创客运动的启发，我挑战自己，要在短时间内打造一个与这个想法相关的简单产品。虽然最初的目标是几个小时，但我设法在不到一个小时内完成了！

最终的产品是一个 chrome 扩展，可以切换网站的所有样式，所以你可以看到根 HTML 组件、表格、图标等

如果你想尝试一下，你可以从谷歌网上商店[安装扩展](https://chrome.google.com/webstore/detail/unstyler/iniahmhgmcmkimliphhbgfkmichgccjd)。而且，即使你不知道，也要记住它只是 HTML:)
