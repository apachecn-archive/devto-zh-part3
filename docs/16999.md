# 编码训练营——即将结束……或者，即将开始！

在过去的五个月里，我一直很忙，努力保持动力，不仅要尽我所能完成每一个项目，而且要真正理解我在做什么，时间飞逝，即使我在最后几周即将完成这个项目，我也意识到我不是要完成，而是要开始。

**即将开始**

…是的！即将开始在没有导师和助教的支持和指导下自己编写代码…即将开始寻找一份工作，在那里我可以应用我在过去几个月里应该学到的东西，剩下的时间我将在训练营…只是一个多月，因为我觉得身体上(和精神上)很累…但同时看起来没有足够的时间开始计划和完成我的最终项目。

**即将结束**

哇…

我不敢相信时间过得这么快，当训练营即将结束时，我感到恐慌，因为我需要准备好开始自己做事情。我知道，一旦训练营结束，我将需要拿起我的笔记，清理我的代码，重读作业并重新做一些其他的，阅读与主题相关的书籍和博客，并寻找额外的资源来保持自己的学习。

我很高兴我决定开始这个旅程，我很高兴当我感到不知所措时我没有放弃，因为我学到了一套新的技能，让我能够创造。而且，我是在我一生中最美好的时光里做到的。
