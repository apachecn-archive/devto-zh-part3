# 把你订阅的所有邮件扔进垃圾箱

> 原文:[https://dev . to/ashleemboyer/throw-all-your-email-subscriptions-in-garbage-25ah](https://dev.to/ashleemboyer/throw-all-of-your-email-subscriptions-in-the-garbage-25ah)

我知道。不是。关心。关于收件箱零。:)太多时候，它变得不那么有效地管理你的收件箱，而是因为任何原因急于没有电子邮件。我更喜欢确保我能充分处理信息，而不是清空我的信箱。但是，我对整洁收件箱的第一个建议是？

### [](#get-rid-of-all-of-your-subscriptions)去掉你所有的订阅。

是的。我指的是 ***他们当中所有的*** 。重新开始。你知道事实上你有太多了。花一个小时去做。这是一个巨大的进步，但是你将获得的是了解哪些订阅是你真正喜欢的，并且想要继续订阅。如果值得再去找一次，那么它在你的收件箱中所占的空间也是值得的。

为了好玩，下面是我目前订阅的一些我喜欢的东西:

1.  极简主义者 -生活和极简主义
2.  [Praxis 简讯](https://discoverpraxis.com/newsletter/) -专业建议
3.  我当地的针织店——我最喜欢去玩的地方之一

这个帖子很短，但是说真的，就这么简单。退订一切。全部扔进垃圾箱。一旦你想好了要重新订阅的内容，请在评论中与我们分享，或者[给我发推文](https://twitter.com/ashleemboyer)。:)
