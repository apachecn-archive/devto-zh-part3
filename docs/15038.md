# 如何在 WebView React Native 中启用和禁用 JavaScript 代码

> 原文：<https://dev.to/skptricks/how-to-enable--disable-javascript-code-in-webview-react-native-56mc>

* * *

来源:[如何在 WebView React Native 中启用&禁用 JavaScript 代码](https://www.skptricks.com/2019/01/how-to-enable-disable-javascript-code-in-webview-component.html)

* * *

本教程解释了如何使用 react 本机应用程序启用和禁用 WebView 组件中的 javascript 代码。该功能有助于 android 和 iOS 应用程序用户停止在设备的 WebView 中加载的网页上执行 JavaScript 和 jQuery。我们将使用 Android 或 iOS 应用程序中的 JavaScript enabled = { Boolean Value } prop，制作一个具有启用和禁用 JavaScript 代码功能的 react native 应用程序，并停止在他们的 WebView 组件上执行 JavaScript + jQuery 代码。
