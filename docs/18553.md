# 副业有那么重要吗？

> 原文：<https://dev.to/buphmin/are-side-projects-that-important-pk6>

所以，最近我一直试图强迫自己开始一个新的兼职项目，这让我思考拥有兼职项目有多重要。我相信学习是重要的，并且保持技术领先，但是你真的需要一堆副业来做吗？

我正好有一个主要的副业项目来制作一个梦幻足球选秀板。我首先在 Symfony 和 Aurelia 中构建它，但我对结果不满意，所以我用 Adonis 和 Vue 分别为后端和前端重建了它。在构建 API 驱动的单页面应用程序的过程中，我学到了很多东西。

另一方面，我也通过在项目之间摆弄工作中的东西和阅读诸如关于开发的文章学到了很多。

所以我想看看 DEV 的好人们对副业项目有什么想法，以及它们有多必要。就我个人而言，我认为拥有它们很好，但不能以牺牲你的幸福和安康为代价。
