# T2】👉[开发清单](https://dev.to/listings)

DEV Listings 是一个专门的区域，会员和组织可以在这里发布有关机会的信息，包括即将到来的活动、职位空缺、导师职位以及其他任何信息。

重要的是，列表获得了一个与博客空间相邻的专用空间，博客空间是 DEV 社区生态系统的其余部分。通过为服务的寻求者和提供者提供空间，我们将有助于适当的协作，同时通过减少在内容中“插入”对读者可能无用或相关的内容的压力，提高平台其余部分的内容质量。

自然，这个产品代表了一种商业模式。创建一个列表通常是免费的，但是更商业化的列表(例如工作)通常是要付费的。更多关于这将如何工作在下面。对我们来说，重要的是我们要以面向社区的方式赚钱，而不是“你就是产品”的方式。我们觉得这符合这种精神。同样重要的是，我们不要试图以欺骗、粗暴的方式将“内容”货币化，就像某个共享博客平台占据了“小”和“大”之间的空间一样。

最终，我们觉得这是平台的一个附加元素。由于我们是开源的，您可以跟踪或贡献我们在此过程中所做的任何更改。

## [](#how-to-make-a-listing)如何制作清单

创建一个列表需要“信用”。截至本周末，我们给所有现有会员每人至少五个积分，加上获得徽章的更多积分。今后，每个人每获得一枚徽章，将获得五个积分。这个系统使得它可以有效地免费为社区中的人们发布列表，但也作为反垃圾邮件措施来确保列表是有“价值”的。

也可以购买信用来创建更“昂贵”的列表，例如工作职位。
