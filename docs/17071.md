# 最佳性能的身份多样化

> 原文:[https://dev . to/jesperreiche/identity-diversity-for-optimal-performance-a5c](https://dev.to/jesperreiche/identity-diversification-for-optimal-performance-a5c)

当你想清楚了你的目标是什么，你就应该把所有的精力都用在达到那个目标上，对吗？嗯，是也不是。你应该总是努力朝着它前进，但是那些在短期内似乎不会让你更接近它的活动，从长远来看，可能会让你不会绊倒和倒退。身份多样化有助于消除充满压力的职业或努力的起伏。

如果你的目标是环游世界，你当然需要一些钱。但是你不需要的是买一栋房子或者找一份全职工作。除非全职工作是短期挣钱的方式。你可能也不需要寻找稳定的关系，除非你打算带上他/她。

另一方面，如果你的目标是开一家公司，或者试图在你目前的公司中获得某个职位，那么环游世界半年肯定不会让你更接近那个目标。

你可以争辩说，为了成功，你需要把你醒着的所有时间都用来朝着你的目标努力。这是你想要实现的目标——所以投入更多的时间和精力就等于实现更多的目标——当然不总是这样。如果你正在建造一座金字塔并砌砖，那么是的——你每天砌砖越多，你就离完成越近。但是大多数努力并不是那么直接。必须向前迈一步——然后后退两步，向另一个方向迈两步——然后后退一步，等等。等等。

这就是身份多样化发挥作用的地方。

如果你所有的快乐和成就感都来自一个来源，当你的公司在挣扎或者你的目标看起来很遥远的时候，你可能会把自己挖到一个很深的洞里。相信我，总有一天你和你的公司会陷入困境。如果你为了一天花 16 个小时在创业上而放弃了所有的休闲活动，那么你就在经历一些事情变糟的黑暗日子——再加上摆脱所有非创业朋友，那么你就让自己陷入了一个真正令人沮丧的循环，所有相似的朋友都在讨论你感觉如何糟糕和艰难。

另一方面你应该做的是身份多元化。不仅是创业先生，也可以是健身先生、舞女、画家先生或任何你感兴趣的人。当一切都按照计划进行时，这些看起来像是分心，但是一旦事情开始变得困难并且违背计划，它们可能是你的救星。

假设你度过了非常糟糕的一天，因为你今天尝试的所有销售线索都失败了。你感到有些沮丧，但仍然出门参加每周一次的游泳课，和你的伙伴们一起。你没有那么多的心情说话，因为这种失败的灼烧感。

然后是你 50 米自由泳的计时。即使你真的一点也不想游泳，你也只是在每一次划水中发泄你所有的愤怒。但是等等，什么！？最奇怪的事情发生了；你最终打破了你一生中最好的成绩，这是你一整年都在努力做到的。

今天是个好日子！

以前看起来很糟糕的一天最终变成了美好的一天。你的商业自我可能受到了打击，但是因为你的游泳是完全不相关的，这丝毫不会冲淡达到个人最佳状态的美妙感觉。

身份多样化我的朋友。

确保你所有的价值感和认同感不是来自一个来源是非常重要的。新的想法不是来自一遍又一遍地敲打同样的东西。它们源于你的大脑有时间将这些点联系起来并思考其他事情的时候。

上面的例子可能是你成为一个好配偶，短跑运动员，举重运动员或者其他什么。关键是不要从一个来源获得你所有的价值感。无论来源是什么都不重要，只要它与你的主要目标完全无关。

作为一件轶事，我差点就写下了我的体操生涯，因为在我创业的时候，这项花费会花掉我几周的现金，而那时我们正处于亏损状态，没有未来收入的直接来源。但是当我想到这件事的时候，想到我会因为拒绝体操而错过那些美好的时光和人，我感到很难过。我幸运地得出结论，一个快乐和理智的我会比一个有点抑郁的我工作得更好，也更聪明——比我说“不”所能挽回的要多得多。在体操比赛期间，我周围的人绝对不会帮助我的公司——但这在某种程度上是关键的一部分。这让我放松下来，思考其他事情，让我回来后更有效率。

无论你是初创企业的创始人、首席执行官、父亲还是运动员，你的首要任务都是保持头脑清醒、感觉良好——一旦你做到了这一点，你就能发挥出最佳水平。身份多样化有助于你实现这一点，让你更能适应挫折。
