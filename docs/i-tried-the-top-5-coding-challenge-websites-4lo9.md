我快速浏览了所有这些网站，并提供了我对它们的直接印象。很有可能他们遗漏了什么。:)如果需要更正，或者如果我错过了这些网站中的任何一个，只要发表评论，我就可以更新这篇文章。

# 1。[代码大战](https://www.codewars.com/)

*   提供 20 多种语言
*   其他人的解决方案可供审查
*   您可以练习编写自己的测试

# 2。[黑客排名](https://www.hackerrank.com)

*   提供 20 多种语言
*   许多技能类别和挑战有待探索
*   求职板似乎是一个额外的好处:)

# 3。[练习](https://exercism.io/)

*   提供 50 多种语言
*   [“指导模式”](https://exercism.io/mentored-mode-vs-independent-mode)—链接页面有大量关于其结构的信息
*   每个挑战都有很多步骤，下载代码，然后重新提交。我更喜欢互动挑战网站。

# 4。[编码游戏](https://www.codingame.com/start)

*   几种可用的语言(我没数，它们是精选的)
*   不同技能水平的多种挑战
*   让我有点不知所措的是，每个屏幕上都发生了很多事情

# 5。 [ProjectEuler](https://projecteuler.net/archives)

*   不是一个交互式编码网站，但是
*   编写软件解决方案面临许多巨大的挑战
