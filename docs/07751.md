嘿大家好！我叫尼克，是一名初级软件工程师@澳大利亚 OCTO。我完全随机地碰到了 dev.to，在简单地浏览了一下之后，community vibe 让我几乎立刻注册了帐户，继续努力吧！

通常我会在所有随机来源上发帖:媒体、我的个人博客等等。这一次，让我们来试试 dev.to！

我的职业道路非常有趣:当我在欧洲的一所高中学习时，绝对没有任何东西促使我去从事软件工程或 IT。不打算撒谎，我 18 岁的时候就学会了 Windows 机器上的热键。我的英语知识仅限于一些非常原始的短语，如“伦敦是大不列颠的首都”和“今天谁值班”。职业道路不明显，所以我不太在意。事情很快变得更糟。

毕业后不久，我成了当地一所技术大学软件开发系的学生。这是一个随意的选择，因为没有什么我特别感兴趣的。我的第一个编程语言是 Delphi 和 C++。从一开始，我就开始一点一点地对编程的迷人世界感到兴奋。我找到了志同道合的朋友，我们甚至开始一起研究个人项目。然而，这并没有持续很久:我开始纠结于材料，我的导师没有提供支持，只是让我考虑改变课程，做一些不同的事情。我的快乐很快转化为愤怒，然后——绝望。我不敢相信我对技术唯一的热情火花被认为是如此之快…

…我不相信。我觉得即使我自己的老师建议我放弃并转系，我也必须继续下去。

有一天我做了一个选择。搬家的选择。不是一所大学或一座城市，而是一个国家。对我来说，澳大利亚看起来是一个很好的地方，可以让我的后青少年生活从新开始。经过几个月紧张的英语准备和为飞机票努力工作，我已经穿越印度洋开始了我的新生活。

[![alt text](../Images/5da417cf85d478388ed7dc74b9db68ac.png "Long journey")T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--wsu8FBeJ--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://www.fly4free.com/wp-content/uploads/2017/05/m-8.png)

改变环境就像呼吸一口新鲜空气。新的语言、人、文化。太棒了。然而，最初的几年难以置信地难以适应:我不得不管理大学、职业生涯，甚至在我人生中最艰难的时刻也总是振奋我的精神。但是变化是不可思议的。

我的新大学给了我一种发展的激情。我遇到了在每一步都支持我的新老师，证明了任何人都可以编程——我做到了。演示、会议、竞赛，这一切突然开始为成功的 it 职业打下基础。新朋友让我参加了谷歌和 Atlasssian 办公室的技术聚会，在那里我遇到了一些很棒的人，他们最终帮我找到了实习机会。梦想实现了。我能感觉到。每一点。

[![alt text](../Images/7a77c0c3d30fa94fc06eb2147f42fc0f.png "Another one!")T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--ZaZkLaVc--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/9mtjxan084yj1xdm5a0o.JPG)

软件工程成了我生活中不可或缺的一部分:我开始从事多个项目，学习新技术，并和我的大学朋友分享/教授。我可以轻松地在晚上 11:00 开始开发，并在第二天早上 5:00 完成，而没有任何时间的感觉，这有多棒。考虑到我几乎每周都要参加聚会和会议，我的衣柜里堆满了谷歌 t 恤。

为国际毕业生找工作是一个真正的问题，然而我的运气和丰富的经验给了我一个机会，在一家总部设在澳大利亚的法国公司，那里的人们相信我的技能和热情。为此干杯！

现在我从事软件工程已经两年了。我有机会使用多种技术处理无限数量的堆栈:从前端到后端。从大数据到开发运营和机器学习。最棒的是，我们都对工艺充满热情，都将开发视为一门艺术。

在这篇简短/随意的帖子的结尾:

每个人都可以开始编程。你只需要非常努力地去实现你的梦想。想学点新语言吗？开始一些小项目。并推动它，直到你感到自信。我有很多朋友从艺术或商业等不同领域转到了信息技术领域。让自己加入一个对发展充满热情的人的社区。如果你在这里，看到这篇文章，恭喜你！你在正确的轨道上！
