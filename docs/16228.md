# 深入工作回顾

> 原文:[https://dev.to/horia141/deep-work-review-5591](https://dev.to/horia141/deep-work-review-5591)

我最近读完了卡尔·纽波特的《[深度工作:在分心的世界里专注成功的规则](https://www.amazon.com/Deep-Work-Focused-Success-Distracted/dp/1455586692)》。这是一本*有趣的*书。这当然很好，如果你需要并且能够采纳它的建议，这也许会改变你的生活。

现在，*深度工作*的*基本想法*对我来说并不新鲜，因为自从*永远*以来，它就一直在互联网上(可能还有许多其他地方)流传。所以至少要几年。但是很明显，这基本上是一种不受干扰的工作，有规律地进行，一次持续几个小时，这是最有助于在困难问题上取得有意义的进展的。毫无疑问，这是一种知识型或创造性工作者需要做的工作。但至关重要的是，这也是书中经常重复的一点，在现代世界中，在*那种级别*工作是极其罕见的，这要归功于现代工作场所的性质——注意力分散严重，注重环境转换。

这本书的第一部分的主要思想是，深入的工作是重要和罕见的，并为致力于它的人提供了优势。有一部分致力于深度工作如何帮助你过上有意义的生活——因为你认为尽最大能力做你喜欢的事情——你的职业——是值得努力的事情，那么尝试做深度工作总的来说将是一件好事。

这本书的第二部分重点介绍了实现*深度工作*的方法。有一个深入工作的方法纲要。这是很好的阅读和理解，因为不是每个人都可以像作者一样过学术生活。事实上，还有比作者更极端的深度作品版本——看看 Knuth 几乎完全与外界干扰隔绝。T2 的重量级人物 T3 是他几乎完全切断社交媒体的建议。这也是一件非常切中要害的事情，因为我*认为*我比我想要的和过去更频繁地使用它。这个策略也很好——不要把它变成零，只要把社交媒体放在你一天中非常严格的时间段内，不要让它溜走。其他好的建议是把重点放在重要的工作上，把肤浅的工作减到最少。同样，在他们自己的*时间*内装箱是必须的。

正如我所说，这是一本有很多好建议的好书。但是大部分需要慢慢融入，而不是简单的完成。也许我还会记录下我想用它做什么。到目前为止，放弃社交媒体是件好事——卸载 facebook、Instagram 和 Quora 非常容易，确实让我获得了一些额外的关注。然而，工作中发生的事情对其他事情没有帮助。
