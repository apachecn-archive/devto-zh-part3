# 努力获得初级前端开发人员的工作

> 原文：<https://dev.to/clam0626/struggling-to-land-the-job-as-a-junior-front-end-developer-1nno>

我已经被许多公司面试过了。职位是设计和开发角色的混合体。主要职责是建立一个网站或改善它的外观和功能。我还没有过技术面试，他们会测试我的代码或者问我技术问题。面试主要是为了了解我和我的工作。比起调试，我更热衷于用代码设计网站，但是我会接受任何能帮助我学到更多的东西。我相信我有这个技能，但是我总是被拒绝。我的经历包括 3 个月的实习和个人项目。我实习时的经验是使用 WordPress，但我主要是通过 WordPress(硬编码 HTML/CSS)来编写网站代码。我目前正在学习使用 React 建立自己的网站，但它似乎对我找工作没有帮助。我想向面试官强调，我愿意在与他们共事的过程中学习。我可以申请最简单的工作，但我会被拒绝。我如何得到这份工作？
