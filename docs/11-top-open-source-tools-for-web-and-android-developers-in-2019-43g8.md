## 安圭拉

[**AngularJS**](https://angularjs.org/) 是一个开源的免费 JavaScript 框架，用于构建动态前端 web 应用。由于其强大的功能，它已经成为全球 web 开发人员最喜爱的工具之一。AngularJS 由 Google 开发，主要用于构建单页面 web 应用程序。

它允许您使用 HTML 作为模板语言，并且您可以扩展 HTML 语法来更精确地表达您的 web 应用程序组件。其数据绑定和依赖注入系统消除了大部分代码编写部分，您可以从浏览器访问所有功能。

**特点:**
MVC 框架——这个框架基于模型视图控制(MVC)架构的概念，它帮助你构建客户端的 web 应用。您不需要编写额外的代码来将所有元素组合在一起。它允许你单独创建所有的 MVC 元素，并通过使用 AngularJS 框架自动组合它们。

数据绑定——不必编写专门的代码来绑定数据；它自动同步模型和视图组件之间的数据。

更少的编码——你不需要为不同的设备编写不同的代码来运行应用程序。一个单一的代码将为你做的工作，可用于各种平台。

单元测试——代码被分成更小的单元，这有助于你轻松地发现每行代码中的任何错误。它有一个内置的依赖注入系统，通过该系统注入和测试模拟单元，以查看输出是否产生预期的数据。

## Node.js

[**Node.js**](https://nodejs.org/) 是一款用于开发服务器端应用的开源免费 JavaScript。Node.js 建立在 Chrome 的 V8 JavaScript 引擎之上，用于构建可扩展的网络应用。它使用事件驱动的非阻塞 I/O 模型，这使得它既轻便又高效。

Node.js 的大多数基本模块都是用 javascript 编写的，它还提供了各种 javascript 模块的丰富库，简化了 web 应用程序开发过程。

**特点:**
执行速度快——由于 Node.js 是建立在 Chrome 的 V8 javascript 引擎之上的，它的库在执行不同的代码时工作速度非常快。

高可伸缩性——它的事件机制有助于服务器以非阻塞的方式响应，这使它具有高可伸缩性。

异步和事件驱动的 Node.js 库包含异步 API，这意味着服务器永远不会等待 API 返回数据，而是在调用它之后移动到下一个 API。它的事件通知机制帮助服务器接收来自前一个 API 调用的响应。

不缓冲——node . js 应用程序从不缓冲任何数据，因为它通过将输出数据分成更小的块来加速整个过程。

单线程 Node.js 应用程序使用带有事件循环的单线程模型。

## 安卓工作室

[**Android Studio**](https://developer.android.com/studio) 是另一个开源工具，它包含了构建 Android 应用程序所需的所有功能。这个工具也被称为谷歌 Android 操作系统的官方集成开发环境(IDE)。它包含 APK 分析器、快速模拟器、灵活的构建系统、可视化布局和智能代码编辑器等功能。为您提供一个流畅的应用开发环境。

它已经推出了最新的 Android studio 3.2 版本，该版本更加稳定，最适合 Android 开发者构建新的 Android 应用捆绑包。

**特性:**
智能代码编辑器——它提供了先进的代码完成特性，可以让你写出更好的代码，提高你的工作效率。当您键入代码时，它会在下拉列表中提供建议，您可以简单地选择所需的一个。

即时运行应用程序-它的即时运行功能可以理解您在代码中所做的更改，并在无需重新启动或重建应用程序的情况下交付这些更改，这有助于您立即看到效果。

测试应用——它帮助你用 JUnit 4 和功能 UI 测试框架测试你的 Android 应用。

所有 Android 设备优化-它让您可以为 Android 手机、Android 可穿戴设备、平板电脑、Android TV 和 Android Auto 构建应用程序。

## 自举

[**Bootstrap**](https://getbootstrap.com/) 是另一个免费的前端框架，用于开发网站和应用程序。使用 Bootstrap 开发的网站和应用程序兼容 iOS 和 Android 设备。

Bootstrap 是一个基于 HTML、CSS 和 Javascript 的框架，可以让你创建动态网站。它附带了一个内置代码，为大多数棘手的元素提供了解决方案，例如经常会降低 web 项目速度的响应性。

**特性:**
易于启动——Bootstrap 的启动非常简单，这是它最吸引人的特性之一。

LESS 和 CSS 文件- Bootstrap 提供了 LESS 和 CSS 文件。

易于定制- Bootstrap 设计在响应式 12 列网格、布局和组件上，易于定制。

模板——模板的集合使得初学者根据 Bootstrap 上的简单教程创建网站变得更加容易。

广泛的组件——Bootstrap 带有下拉菜单、按钮、导航栏、面包屑、提醒、进度条等组件。

## 括号

[**括号**](https://brackets.io/) 是一个开源的文本编辑器工具，可以帮助你直接从浏览器构建网站和应用程序。它是专门为网页设计师和前端开发者设计的。

Brackets 是用 HTML、CSS 和 Javascript 构建的，目前它作为一个桌面应用程序运行，可以访问您的本地文件。括号工具可在 Mac、Windows 和 Linux 操作系统中下载。

**特性:**
高亮语法——你可以高亮特定的代码，用不同的颜色、字体或背景来表示它们。

查找和替换代码——它可以帮助你找到一行特定的代码并用新的命令替换它们，而不用滚动整个代码，这样可以节省你很多时间。

跨平台支持-您可以访问各种设备类型和操作系统的数据。

语言支持-它支持各种编程语言，如 c、Java 或 Python，同时，它支持前端语言，如 HTML、CSS 和 JavaScript。

## Instabug

[**Instabug**](https://instabug.com/) 是每个开发者构建无错 android 应用的必备工具。您可以共享截图、音频视频记录、详细的错误日志来轻松识别和修复错误。

Paypal、Yahoo、Buzzfeed、Dubsmash、Scribd 等流行的 Android 应用程序使用 Instabug 来检查漏洞问题。

**功能:**
bug 和崩溃报告——接收详细的 Bug 和崩溃报告，跟踪 Bug 或崩溃前的每一步，重现崩溃和网络请求日志。

改善用户沟通——您的用户可以轻松地报告错误并提交他们的反馈。解决问题后，你可以通过应用程序回复他们。这对你的用户的保持率和参与度有很大的影响。

集成工具——你可以直接从你的仪表盘上集成 Instabug 和吉拉、GitHub、Slack、Trello 等工具。

## LeakCanary

[**LeakCanary**](https://github.com/square/leakcanary) 是 Square 开发的一款功能强大的开源内存泄漏检测工具。它试图找出应用程序中的任何内存泄漏，并提供一个泄漏跟踪系统来帮助您识别和消除这些泄漏。

**特性:**
每当发生内存泄漏时，它会自动显示泄漏信息，使开发人员能够立即解决问题并尽早修复。

一旦你设置了 LeakCanary，当你的应用程序发生内存泄漏时，它会自动向你发送通知，并提供完整的跟踪选项来帮助你解决问题。

LeakCanary 用红色下划线波突出显示泄漏的可能原因，帮助您找到是哪个引用导致了泄漏。

LeakCanary 应在调试版本中使用，在发布版本中应禁用。

## 凉亭

[**Bower**](https://bower.io/) 是 Twitter 打造的另一款流行的开源 web 开发工具，可以帮助你管理各种 web 组件，比如 HTML、CSS、Javascript、框架、字体、库、资产等许多元素。

**功能:**
Bower 帮你获取和安装正确版本的包，更新，管理所有前端开发依赖。

您可以跟踪您的所有软件包，并确保它们被更新或设置为某个特定的编程版本。

Bower 从各地获取并安装软件包，负责搜索、查找、下载和保存您需要的数据。

一般是针对前端开发者优化的。如果多个包依赖于单个包，例如 jQuery，那么 Bower 将只下载 jQuery 一次。这种现象被称为平面依赖图，它有助于减少页面负载。

## GitHub

[**GitHub**](https://github.com/) 是一个 web 开发平台，程序员和开发人员在这里一起工作，审查代码，管理项目，构建软件。这个开源平台托管了大量项目和多种编程语言的代码。

Github 简化了整个文档过程，作为开发人员，您可以直接从存储库中托管您的文档。Github 是 Windows 和 Mac 的桌面应用程序，允许您轻松地将其集成到您的工作流程中。

**功能:**
拖动&放下 Gist 代码——Github 的 Gist 允许你托管代码片段。使用 Gist 你可以直接从文件中添加代码。你只需要将文件拖放到要点上，文件中的代码就会立即被复制。

通过 web 界面创建文件夹——它允许你通过 web 界面管理存储库，这就是你如何在 Github 中创建新文件或文件夹。

文件查找器-除了创建新文件，你还可以很容易地在任何存储库中搜索文件。

链接行——Github 允许您通过在文件 URL 末尾添加行号来指定存储库文件中的行。

## 月食

[**Eclipse**](https://www.eclipse.org/) 是一个主要用 Java 语言编写的集成开发环境(IDE)，用于用 Java 以及各种编程语言如 C、C++、Python、Ruby 等开发应用程序。

**特性:**
你可以使用它的 Mylyn 特性，通过本地存储库来跟踪你自己的任务列表。你不需要保持你的浏览器打开来查看你的任务，你会通过系统托盘通知得到它们。

有了它的软件更新机制，你将获得任何框架或插件的最新版本，使开发者的工作变得容易。

Eclipse 平台可以用来开发使用插件的富客户端应用程序，包括 Ada、ABAP、C、C++等等。

## 咕哝

你可以使用 Grunt 使用数百个插件，以最小的努力自动完成任何任务。配置 Grunt 之后，task runner 将自动完成您的大部分日常工作流程，让您作为开发人员的生活变得更加轻松。

**功能:**
Grunt minifies JavaScript，CSS 文件，测试文件，CSS 预处理文件如 SAAS，LESS 等。

Grant 包括用于扩展插件和脚本功能的内置任务。

它加速了开发过程并提高了项目的性能。

它有一个简单的方法，包括 JS 中的任务和 JSON 中的配置。

该工具减少了执行重复性任务时出错的机会。
