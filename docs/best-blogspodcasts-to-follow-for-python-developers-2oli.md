# Python 开发者应该关注的最佳博客/播客

> 原文：<https://dev.to/luminousmen/best-blogspodcasts-to-follow-for-python-developers-2oli>

我想分享我偶尔阅读和收听的博客和播客。

## [Python subreddit](https://www.reddit.com/r/Python/)

对于任何与 Python 相关的新闻，这都是一个很好的新闻提要。

## [开发到](https://dev.to/t/python)

对于初学者来说，这是一个很好的信息来源，我真的很喜欢围绕这个资源的社区。低信噪比。

## [奈德·巴奇尔德博客](https://nedbatchelder.com/blog/)

这是 Ned Batchelder 的个人博客，在 Python 社区很有名。

## [尼基塔·索博列夫博客](https://sobolevn.me/)

Nikita Sobolev 的博客主要是关于 Python、工程实践和职业发展的。我总是对阅读他的帖子和想法感兴趣。

## [真正的巨蟒](https://realpython.com/)

在这里，您将主要学习该语言的基础知识和 Python 的相关概念。有时它很有趣，但内容大多是针对初学者的，但有时作者会告诉你一些细节，这样就变得有趣了。

## [用 Python 发明博客](http://inventwithpython.com/blog/)

作者的博客“用 Python 自动化枯燥的事情”。对初学者来说会很有趣。

## [跟我谈巨蟒](https://talkpython.fm/episodes/all)

Talk Python To Me 是一个播客，面向希望了解该语言和相关技术的开发人员。有时很好，有时很无聊，但值得分享。

## [播客。**init**T5】](https://www.pythonpodcast.com)

它提供了对项目、平台和实践的洞察，工程师、商业领袖和计算机科学家需要了解这些项目、平台和实践，以便在他们的职业生涯中学习和成长。另外值得一提的是作者儿童播客:[数据工程播客](https://www.dataengineeringpodcast.com)。

## [皮丹尼(丹尼尔·格林菲尔德)的脑袋里面](https://www.pydanny.com/)

丹尼尔·罗伊·格林菲尔德关于 Python 和 Django 的博客。

## [每周蟒迅](https://importpython.com/newsletter/)

这是每周的 Python 时事通讯。它包含了最好的 Python 博客帖子、开源项目、有趣的推文。至少他们是这么说的:)

## [Python 编码器](https://stevepython.wordpress.com/)

Python 初学者博客。作者提供了许多提示和代码片段。

## [新韵律](http://neopythonic.blogspot.com/)

然后，还有 Guido 自己的博客:)-虽然不定期更新，但是可以很有趣。

## [发光人博客](http://luminousmen.com)

这是你现在所在的博客。我写关于 Python、数据科学、大数据和软件工程的文章。
