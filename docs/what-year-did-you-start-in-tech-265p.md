# 你是哪一年开始从事技术工作的？

> 原文：<https://dev.to/theaccordance/what-year-did-you-start-in-tech-265p>

让我们来一次有趣的回忆之旅:回想一下你第一次开始科技职业生涯的那一年:

*   那是哪一年？
*   第一个角色是什么？
*   那个角色如何帮助你成长为今天的你？
*   包括一个象征一年的图案

这是我的:

*   我的职业生涯始于 2001 年
*   我的第一份工作是在当地学区做学徒，在服务台工作
*   问责制。具有讽刺意味的是，当我十几岁开始我的职业生涯时，我并没有太多这样的东西，最终我发现自己没有那份第一份工作。这是一个艰难但有价值的早期教训，让我带着感激的心情回顾过去。
