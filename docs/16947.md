# 2018 回顾——我的个人开发者年亮点

> 原文:[https://dev.to/lechenco/2018-retrospective-我的个人开发者年度亮点-3dma](https://dev.to/lechenco/2018-retrospective---highlights-of-my-personal-developer-year-3dma)

这是我在 Dev.to 的第一篇文章，促使我写这篇文章的原因是强迫自己记录作为一名计算机工程学生和开发人员的经历，并提高我的英语和写作技能。所以，对于我的第一篇文章，我想与你分享我在 2018 年取得的主要成就和经历的主要事件。

### [](#logicat)LogiCat

首先，我很高兴我和几个朋友能创建这个项目。LogiCat 是一个项目，它向孩子们介绍编程逻辑，然后教他们如何使用 [Scratch](https://scratch.mit.edu/) 编写程序。除此之外，主要目标是帮助孩子们发展创造性和逻辑能力，以及相互合作。这项工作是由 7 到 9 岁的孩子历时一年完成的。在上周，每个双人展都展示了一个自己的期末专题，使用了课程中提到的工具。

### [](#my-first-open-source-contribute)我的第一次开源贡献

2018 年，我决心为 [Open Genius - COSMOS](https://github.com/OpenGenus/cosmos) 知识库做出贡献，该知识库旨在成为“你一生中会遇到和使用的每种算法和数据结构的个人离线集合”。在提出拉取请求后，我变得非常紧张，因为他们可以合并我的冯诺依曼细胞自动机的[实现，我必须承认，编码很有趣。然后，当我收到一个肯定的回答时，我非常高兴，支持社区真的很棒，我希望很快能做得更多。](https://en.wikipedia.org/wiki/Von_Neumann_cellular_automaton)

### [](#operation-rondon-2018)隆顿 2018 行动

我特别喜欢参与志愿者工作，今年我有机会在 Sertaneja 镇工作两周，这是一个由近 20 名大学生组成的团队，他们参加了 2018 年隆顿奥运会。在这一天，我们举办研讨会，向当地人介绍不同的主题，如信息学概念、儿童编程逻辑(我有点像 LogiCat 的简短演示)、基础电学、健康相关主题以及许多其他主题。除了社区电影院和采访抢救当地历史。

这个行动对认识新的人和帮助那些处境不太好的人很有帮助。

### [](#home-automation-project)家庭自动化项目

随着我的毕业接近尾声，项目开始变得更加复杂。一方面，教授们要求更深入的研究和更健壮的原型，另一方面，学生们希望应用到目前为止积累的所有知识开发一个好的实用系统。

因此，对于我们班的一个人，我们小组决定实现一个能够用普通智能手机控制附近环境的系统。有了 [NodeMCU](http://www.nodemcu.com/index_en.html) 在手， [thingSpeak](https://thingspeak.com) 服务器，花了几个小时构建电路和 Android 的界面，系统运行得相当不错。这个经历教会了我如何组织一个复杂系统的开发，以及如何处理事情没有按照它们应该的方式运行的压力。

### [](#embraers-interview)巴西航空工业公司的采访

尽管 2018 年发生了一切，但最好的经历是去一个工业中心，并参加该国最大的国有企业之一的实习职位面试，这些企业开发和制造飞往世界各地的飞机。

这一天是紧张而不可思议的，从 9 个小时的巴士旅行到面试本身，面试很简单，没有人力资源的那些经典问题。我得以见到其他也在完成学位并寻找工作的开发人员。这给了我灵感，我能够知道市场期望的专业水平，当然，我会为我的下一次面试做更充分的准备。

## [](#2019)2019

2018 年真的是很好的一年，我期待 2019 年会更好。为此，作为一个新的一年的决议，我将集中精力在我的弱点上，从整体上提高我的开发人员和逻辑技能。今年，我将:

### [](#improve-my-english)提高英语水平

主要是我的写作技巧，这就是我写这篇和下一篇文章的原因。

### [](#learn-javascript)学习 JavaScript

我意识到这是无法逃避的。实际的开发人员世界呼吸 JS，所以我必须学习如何编码这一点，直到年底。

### [](#project-euler)工程欧拉

为了提高我的逻辑和数学能力，我将开始解决来自 [Project Euler](https://projecteuler.net/) 和 [HackerHank](https://www.hackerrank.com) 网站的挑战。
