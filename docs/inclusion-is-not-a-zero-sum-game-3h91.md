# 包容不是零和游戏

> 原文：<https://dev.to/craignicol/inclusion-is-not-a-zero-sum-game-3h91>

看看你的办公室。有多少人长得像你？这与你在路上或超市里看到的人相比如何？这与您的用户相比如何？你知道吗？

有多少人在与精神疾病作斗争？有多少有无障碍挑战？有多少是白人，异性恋，异性恋，健全的男人？

是否存在不匹配？如果是的话，你会做些什么来改变它？你的招聘广告是包容性的吗？在包容性的网站上吗？(提示:[在发布你的工作之前，先看看 StackOverflow 开发者调查](https://insights.stackoverflow.com/survey/2018/#demographics))你在工作之外有没有可以用来寻找新候选人的关系网？

你在为每个人创造一个安全的空间吗？公司活动对新父母、不喝酒的人、素食者和女性开放吗？人们必须外出参加家庭日活动，或者避免去某个客户所在地吗？你支持需要过渡的员工吗？需要安静空间的员工？斋戒或需要非基督教节日的员工？

你的员工能找到学习手语的地方吗？或者为语言学习者，为那些努力学习或保留语言的人写简体中文？

这些问题中哪三个是你今年最重要的目标？你会怎么做？

如果你需要从某个地方开始，看看微软包容性设计工具包中的[包容性 101，问问自己那是不是你，或者](https://www.microsoft.com/design/inclusive/)[从 Cause a Scene 播客中挑选一集来听，尤其是如果它让你不舒服的话](https://hashtagcauseascene.com/blog/category/podcasts/)。
