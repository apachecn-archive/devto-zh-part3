# 回到学校

> 原文:[https://dev.to/deciduously/back-to-school-57fd](https://dev.to/deciduously/back-to-school-57fd)

从今天早上起，这是官方消息——钱已经易手，我要回大学了。我已经报名参加了一个软件开发的在线本科项目，这个项目将适合我目前的工作日程。当行政管理尘埃落定时，大约两年半到三年后，它看起来像一个科学学士学位，我已经选择了一个方向，大约一年后将获得 C++编程证书。

已经过了...呀!...自从我接受正规学校教育以来，已经有很多年了，而且那次学校教育不是在软件方面。我很紧张——事实证明，这次以成年人的身份参赛对整个焦虑舞会没什么帮助！去想想。

如果你有机会重新获得你的软件学位，了解你现在所知道的，你会做什么不同的事情吗？在学期中和学期之间，你如何最大限度地利用你的时间？明年一月开学前，我应该准备些什么特别的事情吗？开始时什么让你感到惊讶？

如果其他人已经完成了在线学位，同时还在一个单独的领域全职工作，我也很想知道一些关于你如何处理一切的轶事！听起来会有很多工作要做——你是如何平衡学校、工作和其他一切的？
