# 英格尼斯，2015 年

> 原文:[https://dev.to/laneone/ingenius-2015-1k6j](https://dev.to/laneone/ingenius-2015-1k6j)

InGenius 是一个非常出色的黑客马拉松，关于黑客马拉松的一切似乎都是经过深思熟虑的，并且执行得非常好。这是 2015 年 10 月在 PES 南校区进行的一次公开主题黑客活动。

我们在 hack 面临的竞争真的很棒，他们做了大量的准备，并消除了他们应用程序中的缺陷，用于评估想法权重的审查流程也是经过深思熟虑的，它包括几轮，第一轮是他们听取想法的概述并评估团队的演示技巧，下一轮是在一个评委小组面前，在那里你可以解释你的应用程序的复杂性，并从评委那里获得实时反馈。下一轮包括在两位评委面前演示你的想法，他们将评估/测试你的应用，并为你提供将它推向市场的最佳方式。

我在 PES hackathons 上认识到的一件事是，他们鼓励产品原型和推向市场。我从导师那里听到的大多数建议主要是针对在现实世界中实现事情，并试图在市场上推出产品并随着时间的推移进行改进。

我们在这次黑客马拉松上构建的产品缺乏一定的理解，但我们的想法是使用 Node.js 构建一个工具，让我们能够为开发人员提供最快最简单的方法来设置初始 API。如果一个硬件开发人员拿起一个 Arduino 并制作了一个 potato 启动器的原型，我们名为 SnAPI 的应用程序将允许他快速设置一个可以从 web 服务器启动的 API，对服务器的 API 调用将被实时跟踪，并使用 D3.js 进行分析并放入图表中，定量值将放入条形图中，它会在非离散值的图表上画一条线。我们的实现依赖于两个字段，一个字段 custom.js 需要您的程序使用的函数声明，另一个字段由值组成。一旦设置好 api，custom.js 将用于启动脚本、跟踪调用、计算图表等。

我们向评委展示的演示未能向他们展示对该工具需求的相关性，他们没有看到这个想法的潜在价值，我们也必须公平地承认这一点。在这次黑客马拉松上，我意识到了用金钱潜力来支持一个产品的重要性，你在一个产品中挑选的任何东西都要向黑客推销，用相关资源来支持它，告诉他们如何将其推向市场并从中获利。向他们展示一个可行的商业模式，他们会立刻对产品产生更大的兴趣，而不管产品的范围。

这个黑客有一个基于条形码的晚餐入口，使用你身份证上的条形码实时跟踪，我觉得这真的很酷，管理得很好。他们甚至用这个分发小吃，我认为这是在黑客马拉松中管理资源的一种非常好的方式。

我们本来打算空手离开这次黑客马拉松，但是有一个比赛，给任何获得#InGenius 标签转发次数最多的人一张 5k 礼券，从而推广他们的黑客技术。所以我们认为我们可以参加这个比赛。我们在 twitter 上发布了我们想法的照片，并在社交媒体上分享了链接，直到我们有大约 62 次转发。因此，我们最终成为转发量最高的团队，这让我们获得了 5000 元代金券。于是我们把代金券分成 3 份，用 5k 买了 3 个无线键盘鼠标组合。
