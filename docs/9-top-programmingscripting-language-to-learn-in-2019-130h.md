# 2019 年要学的 9 大编程/脚本语言

> 原文：<https://dev.to/yogeswaran79/9-top-programmingscripting-language-to-learn-in-2019-130h>

还是，对 2019 学什么编程语言很迷茫？以下是基于 2018 年 StackOverflow 调查的 9 大编程/脚本语言列表。

1.  JavaScript 是一种符合 ECMAScript 规范的高级解释编程语言。它也是一种动态的、弱类型的、基于原型的和多范例的语言。与 HTML 和 CSS 一起，JavaScript 是万维网的三大核心技术之一。JavaScript 支持交互式网页，因此是 web 应用程序的重要组成部分。绝大多数网站都使用它，所有主要的网络浏览器都有专门的 JavaScript 引擎来执行它。作为一种多范式语言，JavaScript 支持事件驱动、函数式和命令式(包括面向对象和基于原型的)编程风格。它有一个 API 来处理文本、数组、日期、正则表达式和 DOM 的基本操作，但语言本身不包括任何 I/O，如网络、存储或图形设施，这些都依赖于它所嵌入的主机环境。

2.  Java 是一种通用的计算机编程语言，它是并发的、基于类的、面向对象的，并且被特别设计成具有尽可能少的实现依赖性。它旨在让应用程序开发人员“编写一次，在任何地方运行”(WORA)，这意味着编译后的 Java 代码可以在所有支持 Java 的平台上运行，而无需重新编译。Java 应用程序通常被编译成可以在任何 Java 虚拟机(JVM)上运行的字节码，而不管计算机架构如何。截至 2016 年，Java 是最受欢迎的编程语言之一，特别是对于客户端-服务器 web 应用程序，据报道有 900 万开发者。

3.  Python 是一种解释型的高级通用编程语言。Python 具有动态类型系统和自动内存管理。它支持多种编程范式，包括面向对象的、命令式的、函数式的和过程式的，并拥有一个庞大而全面的标准库。

4.  C# (C Sharp)
    C# (C Sharp)是一种通用的多范例编程语言，包含强类型、命令式、声明式、函数式、泛型、面向对象(基于类)和面向组件的编程原则。它是在 2000 年左右由微软在其。NET 倡议，后来被 Ecma (ECMA-334)和 ISO (ISO/IEC 23270:2006)批准为标准。ECMA 标准列出了 C#的设计目标，包括该语言旨在成为简单、现代、通用、面向对象的编程语言，该语言及其实现应提供对软件工程原则的支持，如强类型检查、数组边界检查、检测使用未初始化变量的尝试以及自动垃圾收集等。

5.  [PHP(超文本预处理器)](https://www.learn-php.org/)
    PHP(超文本预处理器)是一种专为 Web 开发设计的服务器端脚本语言。PHP 代码可以嵌入到 HTML 代码中，也可以与各种 web 模板系统、web 内容管理系统和 web 框架结合使用。PHP 代码通常由 PHP 解释器处理，该解释器被实现为 web 服务器中的一个模块或公共网关接口(CGI)可执行文件。web 服务器将解释和执行的 PHP 代码的结果(可以是任何类型的数据，包括图像)与生成的网页相结合。PHP 代码也可以通过命令行界面(CLI)执行，并可用于实现独立的图形应用程序。由 Zend 引擎驱动的标准 PHP 解释器是在 PHP 许可下发布的免费软件。PHP 已经被广泛移植，可以部署在几乎所有操作系统和平台的大多数 web 服务器上，而且是免费的。

6.  [C++](https://www.learn-cpp.org/)
    C++是一种通用编程语言。它具有命令式的、面向对象的和通用的编程特性，同时还为低级内存操作提供了便利。它的设计偏向于系统编程和嵌入式、资源受限的大型系统，以性能、效率和使用灵活性作为其设计亮点。C++在许多其他环境中也很有用，主要优势是软件基础设施和资源受限的应用程序，包括桌面应用程序、服务器(例如电子商务、网络搜索或 SQL 服务器)和性能关键型应用程序(例如电话交换机或空间探测器)。

7.  C 是一种通用的命令式计算机编程语言，支持结构化编程、词法变量范围和递归，而静态类型系统可以防止许多非预期的操作。通过设计，C 提供了有效映射到典型机器指令的结构，因此它在以前用汇编语言编码的应用程序中找到了持久的用途，包括操作系统，以及从超级计算机到嵌入式系统的各种计算机应用软件。

8.  Ruby 是一种动态的、解释的、反射的、面向对象的、通用的编程语言。它支持多种编程范例，包括函数式、面向对象和命令式。它还具有动态类型系统和自动内存管理。

9.  Swift 是苹果公司为 iOS、macOS、watchOS、tvOS、Linux 和 z/OS 开发的通用、多范例、编译编程语言。Swift 旨在与苹果的 Cocoa 和 Cocoa Touch 框架以及为苹果产品编写的大量现有 Objective-C 代码一起工作。它使用开源的 LLVM 编译器框架构建，自 2014 年发布的版本 6 以来一直包含在 Xcode 中。在苹果平台上，它使用 Objective-C 运行时库，允许 C、Objective-C、C++和 Swift 代码在一个程序中运行。Swift 具有解决一些常见编程错误的功能，如空指针解引用，并提供语法糖来帮助避免厄运金字塔。Swift 支持协议可扩展性的概念，这是一个可应用于类型、结构和类的可扩展性系统，苹果公司将其宣传为编程范式的真正变化，他们称之为“面向协议的编程”(类似于 traits)。

以下是 2019 年要学习的 9 大编程/脚本语言列表。记住，学习新的编程语言需要奉献、坚持和努力。希望，你会实现你的目标！
