### 介绍

Bagisto 是一个手工定制的电子商务框架，它建立在一些最热门的开源技术之上，如 T2 的 Laravel T3(T4 的 PHP T5 框架)和一个进步的 Javascript 框架 T6 vue . js T7。

**Bagisto 可以帮助您节省时间、成本和人力来建立网上商店或从实体店迁移到要求不断提高的网络世界。你的企业——无论是小企业还是大企业——都能从中受益。而且设置起来很简单。**
