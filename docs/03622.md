# JavaScript 简介

> 原文:[https://dev.to/codetips/javascript-introduction-28a5](https://dev.to/codetips/javascript-introduction-28a5)

> JavaScript 是一种轻量级编程语言。它最广为人知的是作为 Web 页面的脚本语言，但是它也用于非浏览器环境，比如 Node.js

JavaScript 在 1995 年出现，所以它不是一种新语言。它是动态类型的并且是一种[解释语言](https://codetips.co.uk/intermediate/translation-and-types/)。

多年来，它只被用作网页脚本语言。您可以在浏览器中使用 JavaScript 进行网络呼叫(当一个网站想要从另一个网站获取信息时)，编写可以在浏览器中玩的游戏，并在单击按钮时让神奇的事情发生，等等。

简而言之，如果你想写一个网站，你可能需要写一些 JavaScript。虽然也有其他语言可以在浏览器中运行，但是 JavaScript 是目前最流行的选择。

我们不会通过写网站来学习 JavaScript，而是将重点放在学习语言本身上。

幸运的是，在 2009 年，Node.js 诞生了。我们不需要深入了解 Node.js 是什么，只需要知道它是一个运行时环境，可以用来在没有浏览器的情况下编写和运行 JavaScript。
