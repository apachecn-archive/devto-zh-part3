# 作为开发人员在家工作的顶级技巧

> 原文：<https://dev.to/officialsilb/top-tips-for-working-from-home-as-a-developer--17do>

许多开发人员是自由职业者，这通常意味着他们发现自己在自己舒适的家中工作。诸如不用通勤和自己当老板之类的额外待遇经常让人们认为在家工作是一种梦想；虽然在家工作有很多好处，但也有一些缺点。在这里，我想看看我学到的一些能让在家工作更愉快的最好方法。

创建一个例程

如果你让自己脱离了常规，你很可能会变得效率更低。就像你做一份普通的办公室工作一样，[给自己设定一个开始和结束的时间，并努力坚持下去。这样安排你的日子和制定计划会容易得多。穿得正式一点也是个好主意，因为这会让你以正确的心态度过富有成效的一天。](https://blog.hubspot.com/marketing/productivity-tips-working-from-home)

允许自己休息一下

你不会接受为雇主工作时没有休息时间，所以你自己也不要接受。从最后一点继续，为自己创造一个常规，这将允许你计划喝茶/咖啡以及午餐的休息时间。休息可以让你的头脑清醒，充分利用你工作的时间。

给自己设定目标

在办公室工作时，你可能会设定目标。但是，给自己打工，当然除了自己，没有人给你定这些目标。给自己设定可以实现但仍需努力的目标。这会给你一个衡量成功的基准。

一定要建立关系网

人们在家工作时可能面临的最大问题之一是孤立感。不再有同事在你身边，一天中的大部分时间独自工作会是一种相当孤独的经历。为了克服这一点，[去参加社交活动](https://www.axa.co.uk/business-insurance/business-guardian-angel/how-to-run-your-business-from-home/)会有所帮助。这些活动不仅意味着你得到了你错过的社会互动，而且你可能还会遇到一些很好的商业机会。人际关系网也是结识导师的好方法，导师可以利用他们的专业知识引导你走向成功。

知道什么时候停下来

当你在办公室工作时，你知道一天结束了，因为你已经打卡下班回家了。然而，当你在家工作时，可能更难知道何时关闭和停止。试着让自己充分享受周末和晚上的时光，不要觉得自己应该工作或修改即将到期的项目。
