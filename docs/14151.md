# 对 DIY 微服务系统感兴趣？

> 原文：<https://dev.to/lysofdev/discuss-interest-in-a-diy-micro-service-system-4bm4>

大家好。在过去的几天里，我一直在用我能想到的任何东西来构建一个微服务系统的更常见的组件。例如，我用 Phoenix 和 Elixir 构建了一个 API 网关，并用 Python 构建了一个配置服务器。

有没有兴趣在这些课之后上一个教程或者系列？本教程的主要目的是演示如何通过组合简单的、与堆栈无关的服务来开发微服务系统。我将避免使用预构建的解决方案，而是选择用几个库或一个流行的框架自己构建每个组件。我发现这种学习技巧对于揭开任何技术更复杂的一面特别有帮助。

我将使用 Docker 和 Docker-Compose 进行开发，使用 K8s 进行模拟部署。

如果这听起来有趣，请在评论中告诉我。
