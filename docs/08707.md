当你想学习一些东西时，你总是会搜索，通常会去谷歌。它旨在解决你的管道问题，帮助你完成论文，并帮助你找到梦想中的家。
所以当我想换个职业，想从事网络开发的时候...和大多数人一样，我也去了谷歌。
有很多在线教育平台都有不错的推荐，但只有 TEAMTREEHOUSE 吸引了我 100%的注意力。

那为什么是树屋呢？首先，他们的课程主要面向像我这样的初学者和新手。他们的学校会让每个学生都明白编码的基本原理，不会被它吓倒。他们还有一个集成的工作空间(他们自己的 IDE 版本),可以在文本编辑器中进行所有典型的文件操作，比如创建新文件、重命名、移动和删除。您也可以从您的计算机上传您自己的文件和项目。

第二，和其他人相比，他们并不无聊。凭借他们在 CSS 和网页设计方面的专业知识，TEAMTREEHOUSE 能够让他们所有的视频课程变得生动，让他们的学生渴望观看更多。你可以进入他们的网站，观看他们的视频，并注意到你已经看了一整天了。这就是它们的迷人之处。

第三，他们给学生一种成就感。每当一个学生完成一门特定的课程或一条轨道，学校就会给他们分数，显示他们的成就。它给新手一个奖励和“鼓励”,因为他们付出了努力和时间。这也鼓励他们继续前进，更快地达到目标。

总之，TEAMTREEHOUSE 给了我进入 Web 开发世界的基础和基本技能。通过他们，我学会了如何为我自己和我的客户编码和创建项目。

随着时间的流逝，当我慢慢地从初学者过渡到中级时，我有机会认识了 Web 开发领域的其他导师。
这些新导师带我走上了参加高级课程的道路，以获得更深层次的技能，使我成为一名 Full Pledge 全栈开发人员。在那段时间里，我从事了更多的后端开发。沉迷于更多的 Web 框架、服务器架构和数据库管理。

尽管我已经成为一名成熟的 Web 开发人员，但只要有时间，我总是会回到母校的 TEAMTREEHOUSE。我浏览了他们的新曲目和课程，也参与了他们的社区论坛，帮助新手解决他们的编码问题。这是我对这个机构表示感谢的方式，是它给了我一把有前途的网络开发事业的钥匙。

直到现在，我还在向 TEAMTREEHOUSE 学习。也许不是很多的编码，而是更多的企业家精神。现在，我从 TEAMTREEHOUSE 首席执行官兼创始人 Ryan Carson 那里得到了积极的反馈。像每个网络开发者一样，我也对建立自己的创业公司有浓厚的兴趣。
提供有价值的东西，获得多用户&客户，并在这个过程中货币化我的网站。扩大规模和成长开辟了我的成功之路。

而能帮我这个忙的人，正是“教育之王”。这个家伙在 190 个国家拥有 286，000 名学生，年收入 1500 万美元...TEAMTREEHOUSE 首席执行官瑞安卡森。他是一个经典的例子，你可以效仿，并希望模仿他来创造你自己的成功。

我记得他说过...“我们成功的秘诀是提供高质量的视频课程”。这应该是每个人的口头禅。展现我们最好的一面，展现我们最优秀的品质。花相当多的时间去学习和研究，去创造对每个人都有价值的东西。

这是 TEAMTREEHOUSE 以前教给我的，也是 TEAMTREEHOUSE 直到今天还在教给我的。
