# 你有生意还是运气或生存工作或职业

> 原文:[https://dev . to/tomavelev/do-you-have-a-business-or-do-you-have-you-have-a-luck-or-survival-job-or-occupation-ekn](https://dev.to/tomavelev/do-you-have-a-business-or-do-you-have-luck-or-survival-job-or-occupation-ekn)

你是如何赚到你生活所需的钱的？它们是从哪里来的，我指的不是最终，而是实际的，字面上的汇款人，把钱汇到你的个人口袋，你的个人银行或加密账户？在你和你所做事情的最终用户之间的人越多，你的生意就越少，工作或职业就越多。

《中间人》有哪些变种？老板、雇主、自由职业客户、亚马逊、脸书、OLX 等在线市场、Skrill、PaySafe、PayPal 等支付提供商，有时甚至是你自己的银行。

你有没有注意到，当你打开脸书、Messenger、Instagram 应用程序、谷歌新闻或搜索应用程序，或者任何其他大型应用程序的链接时，网页会在网络视图中加载——这是具体应用程序中的嵌入式浏览器，而不是在单独的浏览器中——由操作系统提供或由用户安装？你问自己为什么吗？

答案是，在一个单独的浏览器中——用户要么退出上下文，要么完全退出当前应用程序和平台的控制——可能无法跟踪用户在打开的页面上花了多少钱，用户正在观看或点击页面的哪个部分，以及更多的统计数据，更多的数据，这些数据在大量情况下非常有价值。

做生意的秘诀是对你从中赚钱的东西拥有实际的控制，成为合法的所有者，尽可能地“拥有它”，如果没有，至少作为一个你可以真正操纵的环境，像一个(虚拟)机器，特别是如果你正在货币化或者你计划货币化你从中赚钱的环境。

你维护你的网站运行的机器和硬件吗？如果不是，你就不能完全控制。您对运行您的站点的虚拟机或容器有完全的控制权吗？如果没有，你就无法控制。授权是有意义的，这样你就可以专注于你自己的事情。

在一个电子商店平台上注册——ebay、amazon、woo-commerce、shopify 并没有控制权。它们可以为你减轻一些工作，尤其是在你刚开始的时候，但这是有代价的——如果你达到了很大的规模，并且会为你创造一种自然的需求，尽可能地迁移到一个更加个人化的环境。另外，如果这些平台做出决定，理论上他们可以开除你。

实际控制这些东西并从中赚钱需要的不仅仅是技术技能，还会给你带来更高层次的问题和困难——接触关键数量的客户或用户，调节他们的行为，过滤用户反馈中的负面评论，实现越来越多有意义的功能或销售更多热门产品，征服竞争对手的市场、市场、品牌，在世界上推销自己。这需要你外向、自信、勇气、耐心和谦逊，以及越来越多的超出纯技术领域的技能和品质。

关于营销的一个想法-我最近意识到的一个想法-为什么电子邮件营销仍然很热，即使今天大多数邮件都充满了垃圾邮件？是有原因的。当你为某件事吸引了观众，你就拥有了对它的实际控制权——电子邮件地址。因此，你可以选择——深入设置邮件服务器的技术细节，将其列为非垃圾邮件，并发送无限数量的电子邮件，你可以使用邮件提供商的 SMTP 服务器，或像 Mail Chimp 这样的第三方服务。发送邮件将更有可能到达用户那里，即使被他们忽略。许多平台提供从电子邮件列表中创建受众。当你在推特、脸书、Instagram 或其他网站上有粉丝时，你并没有完全的控制权，恰恰相反。甚至在这些平台上对你的追随者的有机影响也会逐渐消失，最终会花费金钱。

以下是更多关于控制和商业的迹象:

*   如果你只依赖一个客户，你就没有生意。
*   如果你只依赖于一家支付提供商、一台服务器、一个在第三方平台上的注册、一份数据拷贝
*   如果别人来决定你、你的团队、你的公司下一步要做什么，你就没有生意可做。
*   如果当你不在现场或虚拟现场时，整个系统停止工作超过 50%。
