# 变更日志:查看您的标签版主

> 原文:[https://dev . to/seanmfox/changelog-see-your-tag-moderators-4n 25](https://dev.to/seanmfox/changelog-see-your-tag-moderators-4n25)

有没有想过谁能帮助调节各种帖子标签？好了，现在你可以见到他们了(可以这么说)！在由[@杰斯](https://dev.to/jess)发起的[提议的功能](https://github.com/thepracticaldev/dev.to/issues/1677)中，想法是当查看一个标签的页面(如【https://dev.to/t/javascript】)时，用户可以发现谁是特定标签的版主，这将是很好的。

无论是在桌面还是移动设备上，左侧边栏都会在其他当前显示的信息下面显示一个版主列表。
