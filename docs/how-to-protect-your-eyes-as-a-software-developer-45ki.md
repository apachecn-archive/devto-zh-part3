# 作为软件开发人员如何保护眼睛

> 原文：<https://dev.to/alansolitar/how-to-protect-your-eyes-as-a-software-developer-45ki>

作为一个过去一直与眼睛问题作斗争的人，我想我应该向开发者社区提供一些关于如何作为一名软件开发者保护你的眼睛的提示。

作为一个软件开发人员，你的眼睛是极其重要的。所有那些长时间的调试会话，扫描那些不可能找到的 bug，浏览在线文档以查看如何实现某个特性——所有这些紧张的工作都需要你的眼睛处于良好的工作状态。不幸的是，大多数人没有意识到这些活动给眼睛带来的压力。我知道我没有。

几年前，由于不良习惯和过去的决定，我的眼睛有一些非常严重的问题。我不会用细节来烦你，但我只想说，有一次，情况非常糟糕，我每次只能看着电脑屏幕大约 5 到 10 分钟。作为一个开发者，这是不可接受的。

经过几年的反复试验、在线研究和行为调整，我发现了一些工具、环境变化和行为，让我能够改善和保持我的眼睛健康。以下是我的发现:

## 以下是我的发现:

### 1。使用笔和纸:

长时间坐在电脑前会对眼睛造成极大的伤害，但是因为我们的大部分工作都需要使用屏幕，所以这是一种不可避免的伤害。然而，作为软件开发人员，并不是我们所有的时间都花在编码上。我们所做的大部分都是架构性的——规划和设计我们将要构建的系统/界面。在这些阶段，我发现我并不真的需要使用我的电脑。相反，我拿起我那支可靠的旧笔，在一张纸上写下我的作品。这让我可以离开电脑休息一下，我的眼睛总是感谢我。

### 2。使用 f.lux:

f.lux 是一个调节显示器色温的程序。蓝光可能是眼睛疲劳的主要来源，尤其是在白天和太阳下山的时候。Flux 通过智能管理显示器的温度来帮助解决这个问题，随着时间的推移，慢慢减少发出的蓝光量。对我来说，f.lux 是一个真正的游戏改变者。眼睛舒适度的差异很明显，我可以慢慢增加我的屏幕时间。

##### f . lux 工作原理:

向 f.lux 提供您的位置和您早上醒来的时间——它会尽力匹配您的昼夜节律，随着一天的进行和您就寝时间的临近，慢慢降低监视器温度。您也可以手动调整颜色或从预设中选择。

### 3。调整显示器的亮度:

你是否发现自己在电脑屏幕前很快就会疲劳？可能是你的显示器太亮了。很难说什么样的亮度水平是合适的，因为设备的亮度(通常以尼特为单位)会因具体的产品和面板类型而异。我见过的一个找到最佳亮度的好方法是，在能够阅读屏幕上所有内容的情况下，尽可能把亮度调到最低。作为一个对光非常敏感的人，调低我所有设备的亮度极大地提高了我的舒适度。

注:如果您正在做任何颜色敏感的工作，通量和您的显示器的亮度水平可能会扭曲颜色。对于 f.lux，您可以临时调整色温，或者选择“色彩保真度”等预设对于亮度，您可以为不同的任务分配不同的自定义配置文件(如果您的显示器支持)。

### 4。使用深色主题扩展:

这个对我来说真的很重要。显然，大部分网页都是白色的。然而，这种白色背景，特别是当你的眼睛疲劳或烦躁时，会成为一个真正的麻烦。在我最糟糕的时候，看一个有着刺眼的白色背景的网站几乎是不可能的。我总是在所有的代码编辑器中使用深色主题，所以我开始想，如果 web 浏览器也有类似的东西，那不是很好吗？

这个问题的解决方案是一个可以改变每个网站 css 的浏览器扩展。在火狐上，我用的是 [Owl](https://addons.mozilla.org/en-US/firefox/addon/owl/) 。对于 chrome，我使用[黑暗阅读器](https://chrome.google.com/webstore/detail/dark-reader/eimadpbcbfnmbkopoojfekhnkhdbieeh?hl=en-US)。

虽然它会因网站而异，但使用这两种扩展，你通常会在深色背景上看到白色文本。

除了“有用”之外，我对《黑暗阅读器》没什么好说的 Firefox 是我的日常驱动，所以我将讨论的重点放在 Owl 上。

猫头鹰真的改变了我的生活。以上所有的技术都很有帮助，但是使用黑暗主题扩展才是真正让我重新开始在电脑上工作的原因。我从几乎不能浏览网页，到实际上能够看着屏幕一次工作几个小时。

Owl 是一个很好的扩展——你可以为某些网站启用/禁用它，它甚至可以在 pdf 上工作。

事实上，我非常喜欢 Owl，几个月前我甚至为 github 上的项目做出了贡献。

### 5。考虑照明:

任何直射光对眼睛都很不好受。我总是确保我的光源在我身后，照亮我的工作空间。我发现的另一件重要的事情是确保光线被灯罩或其他物体过滤。在我的房间里，我实际上使用了两个灯罩，因为一个不足以适当地过滤光线。

另一个好的选择是重点照明。例如，过去我有时会在显示器的正后方放置一盏灯。

### 6。使用高对比度模式(Windows 10)

这个只真正适用于 windows 用户。我遇到的一个问题是，win32 应用程序，如文件资源管理器，没有像 uwp 应用程序那样的黑暗主题。

为了解决这个问题，我将高对比度模式映射到键盘上的一个可编程键。每当我需要使用 win32 应用程序时，我都会进入高对比度模式。当我切换到另一个应用程序时，我会退出高对比度模式。有时候很烦，但是很管用。

* Update *–Windows 实际上刚刚在最新的 1809 更新中发布了文件资源管理器的黑暗主题。高对比度模式在处理其他 win32 应用程序或操作系统中不能使用深色皮肤的部分时仍然有用。

### 7。黑暗主题一切

这可能会与上面的一些技术重叠，但这是值得一提的重要事情。每当我考虑使用一个新的软件，我总是看它是否有一个黑暗的主题。这适用于我的代码编辑器、我的操作系统或我使用的任何软件。即使在我自己的个人项目中，我也总是确保使用更深的配色方案。有时我不得不使用一些奇怪的方法来应用黑暗主题，但最终我总是发现这是值得的。

## 其他我觉得有用的东西:

### 1。注意空调/暖气:

这是我过去真正忽略的东西。交流电会降低湿度，而加热会提高温度，从而降低相对湿度。这会使你的眼睛干涩，使你缺乏急需的水分。为此，我几乎从不使用暖气或空调。

### 2。眨眼:

确保眨眼对我来说非常重要。使用电脑时眼睛发炎的主要原因是眨眼频率降低。眨眼是你的眼睛获得所需水分的方式。我过去的眨眼频率真的很糟糕——远低于平均每 4 秒 1 次的眨眼频率。这花了很长时间和大量有意识的眨眼和行为改变，但我的眨眼率似乎已经恢复到正常水平。

### 3。休息一下:

我总是尽可能地让我的眼睛休息一下。过去，我非常认真地坚持 20-20-20 法则，即“每 20 分钟，看 20 英尺外的东西，持续 20 秒。”然而，正常的休息和偶尔离开我的电脑屏幕已经成为习惯，所以它不再是完全必要的。尽管它在过去确实非常有用。

### 4。远程工作:

当您远程工作时，您可以完全控制您的环境。这使得控制室内光线、室内温度以及电脑上安装的软件等因素变得容易。如果有人对远程工作感兴趣，请查看我的文章，我在文章中谈到了我喜欢成为远程软件开发人员的五个原因。我喜欢远程工作，如果没有远程工作，我将无法保护我的眼睛。
