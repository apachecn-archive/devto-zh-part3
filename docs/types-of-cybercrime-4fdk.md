# 网络犯罪的类型

> 原文：<https://dev.to/moderndaytech/types-of-cybercrime-4fdk>

1)黑客攻击
简单来说，黑客攻击就是黑客在未经你允许的情况下访问你的电脑或服务器的任务。

黑客(黑客)基本上是计算机程序员，他们了解计算机的进步，通常会因为各种原因滥用这些知识。

他们通常在特殊软件程序或语言方面拥有专家级技能。黑客以前可能有很多意图，但最常见的是——贪婪、名誉、权力等。

2)窃取 FTP 密码
这是另一种常见的篡改网站的方式。FTP 密码黑客利用了这样一个事实，即许多网站管理员将他们的登录信息存储在一台保护良好的 PC 上。

黑客在受害者的系统中搜索 FTP 登录信息，然后将这些信息发送到他们的远程计算机。然后，他通过自己的远程电脑登录到自己的网站，随心所欲地修改网页。

3)病毒传播
计算机病毒是一种感染系统或文件的程序。这些倾向于由网络上的其他网络广播。

它们会扰乱计算机操作，影响存储数据，要么修改数据，要么全部删除。

与病毒不同，“蠕虫”不需要宿主。当它们没有吃掉系统中所有可用的内存时，它们仍然会被复制。

计算机病毒通常通过可移动介质或互联网传播。已经感染病毒的闪存驱动器、光盘、磁带或其他存储设备。与此同时，电子邮件附件、网站或受感染的软件也容易受到病毒的攻击。

当病毒到达一台电脑时，它就会通过网络在所有电脑上传播。所有的计算机病毒都是由于直接或间接的经济损失。

4)逻辑炸弹
一种逻辑炸弹也叫“渣代码”，是一种恶意代码，在特殊事件触发时，故意插入软件中以 excel málcio 任务。

它不是病毒，但它通常表现得像病毒。它被秘密地插入到一个程序中，并保持不活动，直到它有一个特定的条件。它们在特定条件下变得活跃。

例如，臭名昭著的“13 号星期五”病毒只攻击特定日期；它过去常常攻击星期五第 13 天到来的那一天，使系统变慢。

5)网络钓鱼
网络钓鱼通常是通过电子邮件欺骗进行的。这封邮件看起来像是从你背后发出的，但事实上，它们是假的。点击用户中包含的链接，将被带到虚假网站。在这里，无论他们填写什么信息，他们都会去找黑客。

在单击电子邮件中包含的任何链接之前，请确保检查该链接是否安全。
