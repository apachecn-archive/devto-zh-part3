# Android 的 ColorPrefUtil 库

> 原文:[https://dev.to/akndmr/colorprefutil-library-for-android-6f8](https://dev.to/akndmr/colorprefutil-library-for-android-6f8)

一个开源的 Android 颜色偏好实用程序库。

轻松更改主题和视图/布局/图标/文本颜色或自定义背景。

[在 Github 上查一下](https://github.com/akndmr/ColorPrefUtil)

```
  **Gradle**

**Step 1\. Add the JitPack repository to your build file**

Add it in your root build.gradle at the end of repositories:

    allprojects {
        repositories {
            ...
            maven { url 'https://jitpack.io' }
        }
    }

**Step 2\. Add the dependency**

    dependencies {
            implementation 'com.github.akndmr:ColorPrefUtil:1.0.1'
    } 

```
