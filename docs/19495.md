# 降价标题现在被锚定！

> 原文:[https://dev . to/dev team/markdown-headers-are-now-anchored-18bg](https://dev.to/devteam/markdown-headers-are-now-anchored-18bg)

你现在可以链接到你文章的标题了！这是在创建标题时自动发生的。它将创建一个基于标题文本的锚链接。比如你有一个类似“实施”的标题，可以访问`https://dev.to/andy/andys-post#implementation`直接跳转到帖子的“实施”部分。
