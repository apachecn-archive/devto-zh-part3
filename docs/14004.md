# 我的第一篇帖子

> 原文：<https://dev.to/mirelescloud/my-first-post-3n28>

我目前从事几个 React/Gatsby 项目。在某种程度上，我可能会转向 NextJs，但我已经花了相当多的时间学习盖茨比，并觉得有必要继续在此基础上发展。我喜欢盖茨比的一些插件，比如用于处理图像的 Gatsby-transformer-sharp(T1 ),并允许一些很酷的效果。最近 Gatsby 发布了一个新的 StaticQuery API，允许组件访问数据。最后，在我最近的一个项目中，我已经开始对页面数据使用 markdown 文件。我的目标是创建我自己的 CMS 模板，以便在将来的项目中使用。

我会感谢任何对我的工作的反馈。提前感谢。

我通常不是一个很好的作家/博客作者——这可能对我自己不利，但我希望在这方面有所改进。
