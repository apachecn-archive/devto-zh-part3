# [](#git)去吧

Git 是一个免费的开源分布式版本控制系统，用于软件开发和其他一些版本控制任务。它可以快速、高效、数据完整地支持从小到大的各种项目。

您主要可以在命令行上使用/管理 Git，但是，有几个图形用户界面(GUI) Git 客户端，GitKraken 就是其中之一。

这是什么？

它是一个 GUI Git 客户端，可以在桌面上高效可靠地使用 Git，并提供大多数命令行操作。

访问主页:[https://www.gitkraken.com/](https://www.gitkraken.com/)

Gitkraken 通过以下特性帮助 Git 客户端提高效率:

*   内置合并工具
*   与用户的 GitHub 或 Bitbucket 帐户 GitLab 集成
*   支持单击撤销和重做功能
*   轻松适应用户的工作空间，还支持子模块和 Gitflow
*   快捷键
*   Git 挂钩支持
*   交互式 Rebase
*   Gitflow 支持
*   提交签名等

常见动作？

*   取消
*   重做
*   拉
*   推
*   树枝
*   隐藏物
*   Pop Stash 等

注意:这些都在工具栏上，只需点击一下。
