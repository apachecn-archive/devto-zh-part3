# 参与当地社区有什么好处？

我对当地社区和活动的第一次体验始于 2017 年。我自发地参加了由柏林女性科技创客社区举办的国际妇女节活动。那天晚上，我对一切都感到惊喜。从人们那里，我有机会见到了事件发生的地方。

从那以后，我开始参加越来越多与我感兴趣的话题相关的活动。我没有工作，所以我有足够的空闲时间和精力每周参加一些活动。今天，我想和你们分享我在聚会上以这种方式度过夜晚时发现的好处。

我将把它们归为一类，因为它们彼此密切相关。

### [](#networking-expand-your-network-of-likeminded-people-amp-find-new-friends)联网。(扩大你的志同道合者网络&寻找新朋友)

大型当地社区组织的活动吸引了很多人，尤其是在受欢迎的场合。面对这样一群人，即使你像我一样非常害羞，也很难不和任何人说话。它绝对是自发发生的。有人从你身边经过，扔出一个单词或短语，仅仅 10 分钟后，你注意到你毫不费力地参与了一场有趣的对话。我还发现，如果你刚到一个城市，独自搬家，或者只是来度假，这是结识新朋友的好方法。而且，你永远不知道这些熟人会给你的生活带来什么。也许你刚刚遇到的人会是你未来的邻居、同事、导师、朋友或灵魂伴侣。

作为额外的奖励，在几个月内，你会认识大多数组织活动的人和那些常客。因此，它将不再像最初那样尴尬、不舒服和孤独。此外，如果你不想参加活动，因为你不认识那里的任何人，你可以带一个朋友或同事一起去，这将有助于缓和气氛和放松。

### [](#knowledge-amp-inspiration-get-inspiration-and-learn-new-things)知识&灵感。(获得灵感和学习新事物)

有几种类型的事件可以给你这些好处。

首先，所有事件都有谈判。讲座可以是任何话题，但通常你可以在活动开始前看到讲座的标题和简短摘要，这样你就可以决定你是否对它感兴趣。可以是整个晚上的一次长谈，也可以是一连串的短谈。每次演讲后或休息时，你都有机会向演讲者提问。

有时，演讲的主题超出了你的知识范围，但你仍然会有来听这个演讲的诱惑，因为它可以给你一些尝试新工具或新技术的灵感。

我个人不喜欢谈论更新，比如 framework X 的新版本，或者库 y 的新特性。但是当谈论使用这些工具的好处或问题时，这是完全不同的事情。在这种情况下，你可以从别人的经验和错误中吸取教训。

另一种你可以学到东西的活动是与不同领域的专家聊天或小组讨论。这种类型的活动通常是由某人领导的，他正在就他们的职业道路、经验或其他事情采访专家。另外，观众也总是有可能提出问题。也许你想改变职业，对新领域的某件事感到恐惧，那么你可以问问专家他们是如何处理的。

我还建议你在听讲座或小组讨论时，身边要有纸和笔。你可能想记下提到的工具、文章、书籍、要点或想法。以后当你需要时，检索信息会更容易。对我来说，我更喜欢在谈话和讨论时用图画做一个粗略的笔记或相当详细的笔记。这有助于我集中精力听一个演讲，并更深入地理解它，因为我需要立即将它转换成一个简短的图像或短语，以便以后我能清楚地理解。与仅仅听相比，这也更令人疲惫。选择更适合你的。

回到好处，也有活动，在那里你毫无疑问地获得知识和学习新的东西，这些是课程，研讨会和学习小组。即使你熟悉一门课程或一个研讨会的主题，仍然会有一些你以前不知道或没有想过的小事情，比如捷径、另一种方法或新技术。此外，如果你已经完成了你那部分的任务，你可以随时帮助你的邻居。这给了你额外的练习向别人解释事情，这是更好地理解一切的最好方法之一。学习小组基于相同的方法。一群人聚在一起，在活动开始时，每个人都介绍自己，并简短地解释他们今天将做什么。在构思的时候，你也可以问一些问题，和其他人讨论问题，帮助他们完成项目。

### [](#sharing-teaching-amp-mentoring-share-your-ideas-experience-and-knowledge-with-others-and-give-back-to-the-community)分享，教导&师徒。(与他人分享您的想法、经验和知识，并回馈社区)

这是我今天要讲的最后一系列福利。在某些时候，你可能想和别人分享你的想法、观点和经验。它不需要很大，你也不需要很多专业知识。当然，如果你在某个话题上有着非凡的知识，这是分享的一个很好的选择，而且你在解决问题时的想法或思考可能会对周围的人有所帮助。我自己喜欢阅读个人故事，人们如何开始他们的职业生涯，他们如何建立自己的方式和成长。很鼓舞人心。

你可能认为教学和辅导只对学生或被辅导者有益，只有有经验的人才能做到，但事实并非如此。这对所有正在学习新事物的人都有帮助，在这种情况下，你们两个都是如此。你将再次回顾你所知道的事情，刷新你的知识，也许会发现一些空白，你将在向你的学生或学员解释某些事情时填补这些空白。你也将有机会看到你以前没有机会尝试的新工具和技术，这将帮助你扩展你的经验。因此，你将满足与人分享信息的需求，因为帮助他人而感到快乐，个人和专业成长，并且你将支持尽最大努力吸引更多人参与技术的社区。

去年，我们和我的同事一起组织了一个 JavaScript 课程，教人们(大部分是女性)如何编码。一开始我有很多疑虑，担心我没有足够的知识来回答学生们可能会问的问题。但是第一次讲座之后，一切都烟消云散了。我觉得我知道得足够多了，我可以提供帮助并向他人解释一些事情。你自己试试，你就会明白我在说什么。

这就是我今天想与大家分享的全部内容。希望你能从我的文章中找到对自己有用的东西。
