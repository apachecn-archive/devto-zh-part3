# 2018 年回顾

> 原文:[https://dev.to/ben/2018-in-review-18ep](https://dev.to/ben/2018-in-review-18ep)

随着 2018 年接近尾声，我肯定会计划写几篇关于今年软件行业发生的事情的帖子。一年之内能发生这么多真是令人惊讶。

我做了标签 [#yearinreview](https://dev.to/t/yearinreview) 如果有人想和我一起回顾你的特定语言/环境/社区的进步和/或分享一些你今年最喜欢的 [dev.to](https://dev.to/) 帖子。

随着今年接近尾声，我们的团队肯定会在社交媒体上分享这个标签的许多帖子。不要觉得任何一个帖子都需要是对所发生事情的完美概括。
