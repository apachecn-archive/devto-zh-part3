# 晚年改变职业

> 原文：<https://dev.to/zk433/changing-career-later-in-life-4jn8>

我是永远的学生。有一段时间，我甚至把它作为我个人简历的一部分，直到我开始觉得这样说自己可能是件坏事。当然，从一个兴趣跳到另一个听起来不是一个好的性格特征。万事通，无所不能。没人想变成那样。

然而，它完美地描述了我。我在高中时学的是商科，但在 25 岁出头的时候，我在航运业工作。在我快 30 岁的时候，我决定去上大学，但没有去学商业、运输或物流，而是以优异的成绩从心理学专业毕业。我曾与患有各种精神和身体残疾的儿童和成人一起工作，但在我 35 岁左右的时候，我偶然发现了 web 开发。我成为了一名网络开发人员。

对于其他任何人来说，这可能看起来确实像一次非常随机的旅程。我的意思是，谁会在 38 岁的时候换三次工作？

好吧，我做到了。事后看来，我在我的旅程中看到了完美的感觉。我过去的每一份工作都让我成为今天更好的开发人员。我现在的开发者是由我在以前的职业生涯中，在我的学习和与残疾人的工作中获得的经验、知识和技能组成的。

那么为什么是永远的学生呢？

回过头来看，有一条共同的线索将上述所有内容联系在一起。我喜欢学习新东西。我不害怕新的话题，事实上，我会去寻找它们。我就是那种爱写作业的学生！我喜欢发现新事物的感觉。打破理解上的障碍，挑战自我。这就是为什么，作为一个自称患有数学恐惧症的人，我在攻读心理学学位期间选修了一门数学课程。出色地完成了任务。

这也是为什么网络开发成为了我最适合的职业。

作为一名开发人员，你永远不能停止学习。web 开发环境经常变化，虽然我不认为我们需要知道所有的事情，但是我们确实需要知道外面有什么。有些人可能会觉得这很可怕，而且没完没了，但我觉得这很新鲜。

这对我们有好处:学习新东西，解决难题和挑战，让我们的大脑保持健康。

晚年转行是很可怕的。人们已经有了承诺。责任。很难改变你的生活，重新开始。

但是你没有重新开始。你是建立在你已经是谁和你已经拥有的基础上。你在成长。

我意识到我可以永远做一个学生。原来这终究是我的强项！

我不相信停滞。你要么前进，要么后退。

我在前进。
