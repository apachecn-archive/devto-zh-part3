# 感觉太舒服了👟-是时候遥控了？

> 原文:[https://dev . to/jonasgroendahl/feeling-too-comfort-time-to-go-remote-17ao](https://dev.to/jonasgroendahl/feeling-too-comfortable-time-to-go-remote-17ao)

大家好！丹麦网站开发报道。

我现在的情况是，我对自己的现状感到太舒服了。我做同样的工作已经有两年半了。这份工作允许我使用我喜欢技术设备，并允许远程工作——这是我认为的两个重要因素。此外，我的家和我工作的地方在同一个城市，开车上下班的路程很短。一切都很方便。

对于大多数人来说，这可能是一个很好的情况。然而，作为一名开发人员，我们可以选择在任何地点工作。这个选项可以开启一个新的和令人兴奋的日常生活。在完成工作的同时探索新的国家、城市、人和地方的能力可能是摆脱舒适区的一种方式。

在我看来，有几个因素需要考虑:

这次旅行可能会影响一个人的工作效率。我头顶上的几个例子可能有影响:

*   努力寻找合理的住宿地点
*   感觉像探索而不是工作

我认为时机也是一个重要因素。出国工作，在家带着妻子和孩子会导致一个复杂的情况(与我的情况无关)。

还有谁有过同样的“感觉太舒服”的感觉？如果是的话，你做了什么吗？如果是的话，你是否决定环游世界，做一些完全不同的事情来搅乱你的日常生活？

如果有人目前过着数字游牧生活，请随意与大家分享技巧，✌️将不胜感激
