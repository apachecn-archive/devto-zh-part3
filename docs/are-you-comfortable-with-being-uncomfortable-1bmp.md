# 不舒服你舒服吗？

> 原文：<https://dev.to/rdumais/are-you-comfortable-with-being-uncomfortable-1bmp>

我认为关于舒适区有很多要说的。我一直在想，有多少次我停留在自己觉得舒服的水平上，从而成长或成功了……我想不出有多少次了。

在过去的几个月里，我一直朝着让自己不舒服的目标努力。我在马萨诸塞州的一家保险公司开始了一份新工作。这个新职位意味着一种不同的通勤方式，一个全新的行业，以及一套我必须熟悉的新技能。离开我的舒适区听起来像是冒险，事实也的确如此。如果我不喜欢我的新职位，会发生什么？不管你怎么想，一旦你离开了你的舒适区，就没有回头路了。这是你做出的选择，也是你必须坚持的选择。

回顾过去，我很自豪我做出了这个选择，因为我知道很多人不会这样做。
