各位开发者好，

这是我在 Dev.to 上的第一篇文章，也是我用 React Native 开发的第一个应用。这是一个简单和基本的计算器应用程序，是的，你猜对了，简单的计算。

#### [](#what-i-learned)我所学到的:

嗯，这只是一个基本的应用程序，所以我没有深入学习 React Native。但是这个应用程序帮助我学会了一些东西，

*   反应本机语法
*   样式化组件的不同方式(不使用 CSS 文件)
*   如何在两个不同的组件之间传递值
*   如何设置组件状态

#### [](#what-i-missed-from-the-web-development-world)我从 web 开发世界中错过了什么:

我真的错过了一个使用 React Native 设计按钮样式的简单方法。React 本机**按钮**组件没有自己的`style`属性。

(不过，我了解到你总是可以使用 [TouchableOpacity](https://facebook.github.io/react-native/docs/touchableopacity) 创建一个自定义按钮)
