# 星球大战超跳跃

> 原文：<https://dev.to/annlin/star-wars-hyperjump-1mf0>

使用`translateX`重现星球大战中的超跳跃效果。

```
@keyframes hyperjump {
  0% {
    opacity: 1;
    transform: translateX(0);
  }
  1% {
    transform: translateX(-50px);
  }
  100% {
    opacity: 1;
    transform: translateX(-100vw);
  } 
```
