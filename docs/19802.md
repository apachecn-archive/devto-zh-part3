# 你知道如何{ content:attr({ data-attribute })；} ?

> 原文:[https://dev.to/rinstarskykujat/do-you-know-how-to-内容-属性-数据-属性-1 ke](https://dev.to/rinstarskykujat/do-you-know-how-to--content-attrdata-attribute---1eke)

我们可以将 deta 传递给 el::before，after { content:' '；}

```
 <div class="button">
        <span data-attribute="HOME">HOME</span>
    </div> 
```

```
 .button{
        span::before {
            content: attr(data-attribute);
            }
        }
    } 
```

这非常简单！！！

我用这个机制制作了一个简单的动画。
