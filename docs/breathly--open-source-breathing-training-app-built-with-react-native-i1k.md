## 概观

在这个库中，你会找到 Breathly mobile 应用程序的源代码。

Breathly 是我在 2018 年的空闲时间开发的一个小型 React-Native 应用程序，用于玩 React-Native `Animated` API。随着 React-Native 的钩子支持的到来，我决定重新开始，用钩子重写它，并在 Play Store 和 App Store 中发布。

希望源码对某人有用。
