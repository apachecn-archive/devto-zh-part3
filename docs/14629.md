# 如何说服初级开发者先学基础？

> 原文:[https://dev . to/arximughal/how-to-说服初级开发人员学习基础知识-first-48oa](https://dev.to/arximughal/how-to-convince-junior-developers-to-learn-basics-first-48oa)

我自己是初级开发人员，但在我迄今为止的 3 年旅程中，我有机会与一些最伟大的人一起工作。他们帮助我学习编程，工作礼仪和许多其他东西，为此，我真的很感谢他们。在工作中，我帮助初级开发人员/实习生学习基本的前端知识。

当我开始学习时，我有最好的老师。我被指示先学习基础知识，然后继续学习框架。我从基本的 HTML & CSS 开始，学了一点 JavaScript，然后开始学习 Bootstrap。之后，我转到了基础和材料设计。现在，我不做太多的 UI 工作，但是我知道很多最好的 UI 框架。但我不是一个晚上就做到的(很明显)。

这花了我很多时间。夜晚、白天和周末。很多都是。就像大多数初学者一样，我渴望一下子学会。但很快意识到这不是它的工作方式(感谢优秀的导师和客户)。

这也是我想教给那些选择我作为导师的人的。我尽最大努力用一个恰当有效的参考来解释一切。我告诉我教过的每个人，学习的东西对每个人来说是不一样的。有人可能在一个周末学会一个全新的框架，但其他人可能需要一个月甚至两个月。但是大部分初学者似乎并没有认真对待我的建议。

最近，在我给他的第一个任务中，我告诉一个人不要使用 Bootstrap。他回答说:

> 我看见你用了自举。为什么我不能使用 Bootstrap？

我告诉他先学习 CSS，但是他坚持使用 Bootstrap，最终还是离开了🙁。同样的事情发生在另一个同事身上，他正在帮助另一个被拘留者。

我想问的是:

## 如何说服初学者先学习基础知识？

...不让他们觉得我是他们前进道路上的障碍？
