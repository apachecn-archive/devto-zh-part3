# 增加一个让你的用户付钱给你的方法是保持积极性的一个非常好的方法！

> 原文:[https://dev . to/chipd/add-a-way-for-your-users-to-pay-you ' s a-really-great-way-to-stay-motive-1 gdl](https://dev.to/chipd/adding-a-way-for-your-users-to-pay-you-is-a-really-great-way-to-stay-motivated--1gdl)

我的一个副业项目 [(mydevportfol.io)](https://mydevportfol.io?ref=devto) 是一个 web 应用程序，让用户开发者/程序员基于他们的 github 档案建立一个投资组合。

从一开始，我就想看看我是否能赚些钱，但我不确定它是否能达到那个阶段。

在尝试了一些不同的定价模式后，我最终决定让人们支付他们想要的价格。这是“终止”该应用的决定的一部分，这样我就可以专注于其他项目。

几周后，我震惊地发现有人为一个投资组合捐了 49 欧元。

我真的惊呆了。

这是一个巨大的推动，让我迅速回到开发，修复错误和可用性问题，并计划更多的功能。

在过去的几周里，我收到了一些捐款(见截图)，没什么特别的，但是每天早上查看条纹仪表盘，看到(有时)我的早餐咖啡已经有人付钱了，这是一个巨大的激励！

我鼓励任何人添加一个捐赠按钮或其他一些接受金钱的方法。如果你做了一些世界上只有少数人真正喜欢的东西，他们会给你钱。他们感谢你所付出的努力。

当然，将来当你发现某人做的东西时，你总是可以支付这笔钱，你会因为给他们买了一杯咖啡而满足地微笑。

附言:那一大笔捐款来自我邮件列表中的某个人。如果你没有收集电子邮件，你应该开始了。我不会给我的列表发垃圾邮件，我只是给他们提供他们感兴趣的产品的相关更新。
