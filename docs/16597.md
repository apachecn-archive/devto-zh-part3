## [](#source-github-private-repos-are-now-free)来源: [GitHub 私人回购现已免费](https://www.skptricks.com/2019/01/github-private-repos-are-now-free.html)

免费私有库的宣布可能让 GitHub 的一些竞争对手大吃一惊，但下面是我们从 GitLab 首席执行官 Sid Sijbrandij 那里听到的消息:“GitHub 今天宣布与多达三个合作者推出免费私有库。从一开始，GitLab 就在私有库上提供了无限的合作者。我们认为微软更关注 Azure 创收，而不是对 DevOps 软件收费。在 GitLab，我们相信组织使用多个公共云平台的多云未来。”

> 每个私人回购最多可以添加三个合作者。还有一些与他们的其他定价水平相关的新闻。
