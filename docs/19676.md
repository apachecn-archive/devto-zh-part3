# [](#marvel-app)漫威 app

漫威的应用程序几乎拥有我需要的一切。它在入职方面做得很好，提供了大量有用的提示，甚至是空状态下的教程来鼓励创作。我能够将草图文档作为屏幕导入。

画画感觉很棒，而且有各种各样的工具可以使用(比如创建基本的形状，或者插入图像——甚至是库存照片)。它甚至以网格模式为特色，允许更紧密的绘图和对形状的控制。

你可以在你的草图中添加可点击的框来链接草图，当播放时，这是你的应用程序的一个活的(可共享的)原型。这是唯一一个允许真正创建“原型”的应用程序，而不仅仅是一些屏幕草图。

最精彩的部分？不需要帐户。你可以画出草图，并将 jpg 保存到你的相机胶卷中。创建一个帐户可以让你在线分享你的原型。

**弊？**自由层仅限于 2 个项目和原型品牌与漫威的标志(而不是草图 jpg)。付费版本从 12 美元/月起，用于无限制、无品牌的项目。漫威还将画布大小限制在手机屏幕的大小，这使得草图很窄，除非你总是以 1:1 的比例工作。

**判决结果？**漫威是在旅途中创建原型的最佳(也是唯一的选择)，但可能不是勾画更松散的线框、站点树或其他 UX 可视化数据结构的最佳地点。

# [](#invision-app)视觉 app

老实说，在所有其他 UI/UX iOS 应用程序中，Invision 应用程序感觉最好。绘图的感觉很棒，你可以在无限的画布上画画——这是其他应用程序无法提供的。很酷的是，我可以选择轻按一个开关，让我的画传播给我所有其他团队成员看同一个文件。没有其他 UX 应用程序有这种实时协作，只是事后分享创作。

您可以导入草图设计和原型，以便随时浏览。还有一些其他的协作功能，比如和团队成员聊天，在大型项目中可能更有吸引力。

犯人在这里来得更快。加载时间比漫威长，打开任何手绘草图大约需要 15-20 秒。不过，这款应用缺少很多关键功能，比如形状绘制工具，或者将屏幕链接到原型。在进入“徒手”草图模式之前，你需要登录一个 Invision 帐户(这意味着没有互联网就不能工作)。尽管可以导入草图文件，但您实际上不能操纵它们，甚至不能在它们上面绘图。

**判决结果？** Invision 似乎是一个很好的应用程序，可以用来画草图，甚至可以浏览你在桌面上创建的原型，但不可能在旅途中制作原型。

# [](#adobe-sketch)土坯草图

我担心一些 UX 应用程序可能没有最好的绘图技术，所以我将 Adobe Sketch 加入其中，希望它能满足我对流畅绘图体验的标准。

老实说，我能传达的唯一的优点——除了你能从 Adobe 那里期待的奇妙的绘画感觉——是它有更多的画笔？哦，令人惊讶的是，这是一个完全免费使用的应用程序(除了高级付费画笔)。

画布的大小是有限的，尽管可以将其设置为更大的尺寸和比例。但最糟糕的是有限的绘图区域，应用程序的用户界面是如此的侵入性，以至于你会因为画笔和其他工具而损失 40-50%的画布。与其他移动素描应用不同，你不能在绘画时隐藏所有面板(或者甚至旋转到横向，这将有所帮助)。画形状既麻烦又笨拙——你必须用选框工具画形状，然后用手指描出它们。

**判决结果？甚至比不上漫威或 Invision 的功能集。缺少草图导入、原型制作的屏幕链接以及简单的**

## [](#the-best-ux-ios-app)最好的 UX iOS 应用？

总的来说，我对我评测的所有应用程序的性能印象深刻。令人惊讶的是，使用这些应用程序可以方便地实现什么，以及它们工作得有多好。

然而，在一天结束时，我无法为我的工作流程选定一个应用程序。我用漫威创建原型(草图和编辑的图片)。在我找到另一个拥有无限画布和良好形状工具的绘图应用之前， **Invision 是我获得更多界面、线框等的*手绘*草图**的首选工具。
