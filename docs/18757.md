### [](#general)🌤️综合医院

**2018 年芬兰的天气非常棒！！！**从五月初到十月，大部分时间都非常温暖宜人，阳光充足。纯粹牛逼！之所以如此特殊，是因为它有点罕见，大多数夏天往往是多雨和寒冷的一面。2018 年 11 月 10 日天气。

我们也度过了安东出生后的第一个假期，然后在九月去了我在塞浦路斯的家。这是去那里的最佳时间，因为芬兰的气温开始下降，但塞浦路斯白天的气温仍在 30 度左右。我们玩得很开心！

### [](#%EF%B8%8F-double-down-on-react-and-vanilla-js)👨‍💻️加倍下注 React 和 Vanilla JS

我喜欢 JavaScript，但主要是它的普通形式和 React 项目。2018 年，我想通过一个现代框架来大幅提高我的技能，**我决定最终加倍下注[反应](https://reactjs.org)T3】。我在这个问题上犹豫不决已经太久了，伙计，当你最终做出决定并扫清障碍的时候，感觉是不是很棒！**

> 做出这个决定的主要原因之一是[盖茨比](https://www.gatsbyjs.org)！

这个博客和我的许多其他项目都是使用 Gatsby 的 React 项目。Gatsby 是一个很棒的静态站点生成器，为速度和性能而建，有很多插件，大社区和很棒的开发者体验。简而言之，建造东西是一种乐趣，结果总是令人难以置信。

这种思维模式的改变也帮助我开始去 meetups，我在 2018 年去了 3 次(尽管我希望我去得更多)。我在一月和十二月去了赫尔辛基，八月去了一次 T2。这是一次很棒的经历，我期待着明年在这些活动中的一次演讲！
