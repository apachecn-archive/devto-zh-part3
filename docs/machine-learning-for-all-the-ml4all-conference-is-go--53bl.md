## ml 4 all 是什么？

ML4ALL 代表“面向所有人的机器学习”。去年，我和阿莱娜·霍尔、T2、特洛伊·霍华德、T4、格伦·布洛克、拜伦·格拉赫和本·阿克一起工作，度过了一段美好的时光。ML4ALL 是我们举办的一次会议，旨在将对机器学习、系统、围绕机器学习的生态系统以及参与或更多参与这部分技术行业的方法感兴趣的人聚集在一起。我们的目标是组织这次会议，公开邀请所有人前来学习、展示和扩展个人的机器学习经验。我期待着我们的团队、演讲者和嘉宾今年再次为社区举办一次精彩、有趣、有教育意义的会议！

但我总是说，不要只相信我的话，这里有一组去年对嘉宾和演讲者的采访，是我们的演讲者沙皇 [@lenadroid](https://dev.to/lenadroid) 整理的。这个很棒的活动视频和一些对演讲者和与会者的简短采访。这是一块坚固的手表，花几分钟时间查看一下，就能很好地了解大会的情况。

## 想参加？救命啊！

门票正在出售，但现在有很多其他方式可以参与进来。首先，跟踪更新的超级简单的方法是关注 Twitter 账户: [@ml4all](https://dev.to@ml4all) 。第二种方式稍微复杂一点，但对你来说可能有更高的投资回报，加入 [ML4ALL Slack Group](https://ml4all.slack.com/join/shared_invite/MTk1MjUwNzgyMDgyLTE0OTY5OTgwODUtNTU1MTZhMjA0Zg) ！在那里，我们讨论会议更新，谈论机器学习，介绍我们自己，以及一系列其他讨论。

如果你在机器学习领域的一家公司工作，从事人工智能和相关业务的浪潮，你可能想通过赞助会议来参与进来。我们有一份不同级别的招股说明书可以寄给你，只要发一封电子邮件给[ml4allconf@gmail.com](//mailto:ml4allconf@gmail.com)就可以了，邮件主题是“Plz 招股说明书”。我们将向您发送计划书，然后我们可以开始讨论哪种级别最适合您的公司！
