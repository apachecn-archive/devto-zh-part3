# 我的礼貌在哪里！嗨，我是格伦

> 原文:[https://dev . to/Glenn men/where-are-my-manners-hi-im-Glenn-4f3a](https://dev.to/glennmen/where-are-my-manners-hi-im-glenn-4f3a)

在我来到这里的头几个星期里，我积极地发帖，而不是躲在暗处，我想不先介绍自己是多么不礼貌...
你好，我是格伦。我今年 24 岁，住在比利时。
我目前作为一名专业的 Android 开发人员已经有两年多的经验，也做过一些较小的 web 项目，大部分只是 PHP 后端部分。

我不记得我实际上是什么时候开始编程的(我猜这取决于你把编程归为什么)。我从编写通过 MSN 发送的小型 bash 脚本开始(对我来说是古代的)，例如，当其他人打开它时，他们的 PC 会重新启动，或者如果他们关闭它，他们的 CD 播放器会继续打开，一个年轻的十几岁男孩的小型 troll 脚本只是玩玩(没有损害接收者的财产)。

在那之后，我花了一些时间在那些在线网站建设平台上尝试制作“网站”,但很快就厌倦了，对结果也不满意。我转行用 jQuery 和 Bootstrap 做简单的静态网站(感谢 Bootstrap！)因为我没有任何托管知识和资金，我学会了如何在你的 Google Drive 上托管一个静态网站，并使用一个免费的. TK 域名。

最后，当我开始上大学的时候，我学到了更多关于网络开发的知识，最重要的是我学到了 PHP 的基础知识！我用 CodeIgniter 重建了之前制作的静态网站，学习了所有关于 cookies 和缓存的知识，并获得了一个共享的托管服务提供商。我真的看到了网站质量的提高(以及用户群)，从 3 页增加到了 17 页，我还制作了自己的基本聊天框。这个网站已经存在几年了，我试着更新它，它是我当时玩的一个在线游戏的计算器和其他工具。它成为排名第一的社区网站，人们甚至认为它是由官方游戏开发商制作的，因为数字非常准确。[链接到 GF5FCalc](https://www.gf5fcalc.com/) ，不幸的是上周游戏开发者宣布他们将退出游戏。

在我大学的最后一年，你必须在一家公司实习 3 个月，我选择了 Appwise。一家小公司(当时只有 12 个人),为公司定制本地移动应用程序。
尽管我在网络方面有更多的经验，但我还是决定使用安卓系统。我在大学时对 Android 有一点经验，但在实习期间我很快发现自己实际上一无所知。这是一次很棒的经历，我学到了很多，玩得很开心。这家公司有一种非常轻松的氛围。
我已经决定，实习结束后，一旦我毕业，我将申请那里的一份工作，然而，在我的时间结束前几周，我的老板要求和他坐下来，他问我是否想在我毕业后开始他们的工作。我当然很高兴他们给了我一份工作！

所以几个月后我毕业了，开始在那里工作。又过了两个月，我的老板再次召集了一次会议，通常你会期待一次会议或者知道会议的主题，但是这次又出乎意料。他(和我的下一个老板，我当时不知道)告诉我，他们想成立一个新的公司/部门 Appwise Consultancy，开发人员将作为顾问帮助客户完成他们的项目(大部分由我们开发，但不是一个要求)，这可能是帮助维护，添加新功能，改进，建议，教导他们的开发人员，...很百搭。他们问我是否愿意成为他们的第一个顾问。他们说这可能是一个失败，但这是一个实验，他们希望我成为其中的一部分。我想了想，我还年轻，这样的事情可以帮助我成长，所以我答应了。

所以目前我已经为我们的一个大客户做了两年左右的全职顾问。负责 5 个 Android 应用、3 个 iOS 应用和 4 个 web 应用(Laravel)。大部分的 iOS 问题，我依靠 Appwise iOS 开发者的帮助，但其余的我自己做。与客户/公司如此密切地合作是一次很好的学习经历。管理会议、不同的项目、不同的团队(外部)，...

除了我的职业生涯之外，我也是开源软件的狂热爱好者。我尽我所能帮助项目:cdnjs，DEV，CodeCopy，...
我也是 PoGo 地图 [PMSF](https://github.com/Glennmen/PMSF) 的项目负责人。但是正如你所看到的，这个项目已经过时一段时间了，我想对它做一个完整的重建，清理大量的代码，但是我目前在寻找时间和/或兴趣方面有一个问题。
我也希望开发 Android 项目将很快开始，我很乐意在这个项目上提供更多帮助。

我想这总结了我迄今为止的编程生涯，我希望你没有在阅读时睡着😂
