[复制(或分叉)本 codepen 中的内容即可开始使用。](https://codepen.io/tevko/pen/joxjZq)

规则是:

*   没有 Javascript
*   您可以添加 HTML，但不能删除任何 HTML
*   在小屏幕上，应该出现一个按钮。点击按钮应该会使导航从左边飞出([像这样](https://www.jqueryscript.nimg/Responsive-Off-canvas-Navigation-with-Gooey-Transition-Effect.jpg)
*   在大屏幕上，导航应该贴在屏幕的顶部
*   点击按钮时，使用[复选框黑入](https://css-tricks.com/the-checkbox-hack/)启用飞出导航。
*   添加 CSS 不仅使其功能强大，而且看起来也很漂亮！

一些提示:

*   不要过度设计。专注于让导航在小屏幕上工作，在其他事情之前先黑掉复选框！
*   这个练习很难。慢慢来，在开始之前确保你理解了这个练习。如有疑问，请联系我们！
*   虽然你可能想在小/大屏幕上添加两个 nav 并隐藏一个，但不要这样做！重复导航不利于可访问性！
