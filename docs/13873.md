# 作为 Web 开发人员生命线的 21 个平台

> 原文:[https://dev . to/lambdatest/21-platforms-as-a-lifeline-to-web-developers-1h J1](https://dev.to/lambdatest/21-platforms-that-serve-as-a-lifeline-to-web-developers-1hj1)

Web 开发每天都在以惊人的速度不断发展。对于 web 开发人员来说，跟踪这个领域中大量涌现的新工具、库、框架、插件和平台是一个巨大的挑战。网站开发涉及 5 个复杂阶段的复杂循环，即信息收集、规划和设计、开发、测试和交付，最后是项目维护。处理所有这些阶段是一项令人痛苦和令人畏惧的任务，即使对于一个熟练的开发人员来说也是如此。这就是为什么我为 web 开发人员策划了这 21 个基本平台的列表，以帮助他们提高生产力并保持高效的工作流程。

## [](#information-gathering-and-organizing)信息收集和整理:

在我看来，任何网站开发者的第一要求是收集网站的信息。实现目的、设计、要实现的模板、预期受众和主要目标。收集信息和跨平台的可访问性提供了便利。毕竟不像手机，你不可能随时随地都带着笔和纸。web 开发人员的 Following 平台将取代他们传统的记笔记方式。

*   **[米拉诺特](https://www.milanote.com/)** ![milnote](../Images/c204101f55af43697ca262b47a738d75.png)

这款网络应用提供了跨平台的内容组织功能。除了预定义的模板和来自品牌设计师、营销人员、开发人员和工程师的灵感，该网络应用程序还提供了各种实用工具，用于笔记、规划和共享板，具有拖放界面。免费注册，获得 100 个笔记、图片、链接和上传。对于更大的项目，查看他们的亲会员计划。

*   **[微软 OneNote:](https://products.office.com/en-IN/onenote)** ![onenote](../Images/47f837c62739390d85d503d24730f590.png)一个专业的计算机程序，用于做笔记、收集信息、分享信息，也用于协作。它允许并行编辑上传到 OneDrive 或 DropBox 上的文档。

## [](#join-a-community-help-and-be-helped)加入一个社区，帮助和被帮助:

社区是一个强大的平台，让全球的 web 开发人员相互联系，创新想法，消除与 web 开发相关的疑惑和疑问。加入关于特定领域的虚拟社区并进行讨论，提供建议可以帮助获得更多的曝光和即兴工作。以下是一些帮助我提高日常工作效率和智慧的知名社区:

*   **[Stack Overflow:](https://stackoverflow.com/)**
    [![stack_overflow](../Images/8a4d295b8794fa2f46acd8bfde9b9002.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--5STKtZpK--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://cdn.shortpixel.ai/client/q_glossy%2Cret_img%2Cw_930%2Ch_180/https://www.lambdatest.com/blog/wp-content/uploads/2018/12/unnamed-3.png)
    一个面向所有软件开发人员的全球社区，他们可以在这里联系、交流和寻找职业。对于求职者，有一个专门的部分，其中包括职位空缺和独特的功能称为工资计算器，让你知道你的价值。除了各种编程语言，你还可以在网站的搜索引擎上搜索各种标签。

*   **[Hashnode:](https://hashnode.com/)**
    [![hashnode](../Images/c0329a4c6251fb9d7b7cb5802cfbf708.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--ZmrcE2NX--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://cdn.shortpixel.ai/client/q_glossy%2Cret_img%2Cw_930%2Ch_180/https://www.lambdatest.com/blog/wp-content/uploads/2018/12/pasted-image-0-1.png)

另一个网站，开发者可以就不同的主题进行交流，包括各种编程语言，如 Java、PHP、Javascript 等。在这里，你可以学习、分享和讨论关于软件开发的各种问题。

## [](#code-editors)代码编辑:

为了让你的程序易于管理和整洁，代码编辑器已经被证明是 web 开发人员的好平台。它们使用户能够加速编码，简化他们的工作流程，进而提高生产力。源代码编辑器可以是独立的，也可以内置在 web 浏览器或 IDE 中。列出了一些为我们的 web 开发人员社区量身定制的代码编辑器:

*   **[vs code:](https://code.visualstudio.com/)**
    [![vscode](../Images/8bc73a6fa32a345a37f64ec9a5986ecc.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--ELN6buvV--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://cdn.shortpixel.ai/client/q_glossy%2Cret_img%2Cw_930%2Ch_180/https://www.lambdatest.com/blog/wp-content/uploads/2018/12/image7-1.png)
    微软开发的免费开源代码编辑器，可在 Linux、微软 Windows、macOS 上下载。它支持多种功能，包括代码重构、语法突出显示、定制和智能代码完成。

*   **[括号:](http://brackets.io/)**
    [![brackets](../Images/71d39051c9412065207169775657a21f.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--TjQPV5hF--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://cdn.shortpixel.ai/client/q_glossy%2Cret_img%2Cw_930%2Ch_180/https://www.lambdatest.com/blog/wp-content/uploads/2018/12/image5-1.png) 
    由 Adobe systems 创建，括号是一个开源编辑器，具有多种功能，包括实时预览、可扩展性、快速文档和 JsLint。这个免费的程序有一个独特的“快速编辑”功能，可以通过同时按 Ctrl/Cmd 和 E 键来实现，并允许内嵌 CSS 编辑。点击[此处](https://registry.brackets.io/)浏览并上传扩展注册表。

## [](#is-your-website-compatible-check-the-responsiveness)你的网站兼容吗？检查响应度:

有一次，我开发了一个漂亮的网站，上面有图解矢量和标志。这是一个博客网站，有着令人惊叹的图片，所有的东西都排列有序。我和它有着美好的联系；甚至模型看起来都很完美。我把作品提交给制作组，5 分钟后就被拒绝了。为什么？因为我的网站在运行旧版本安卓系统的三星 Galaxy S 上显示混乱。我是说，了解一下最新情况？从那以后，我遇到了一些响应测试平台，在不同的操作系统和设备上测试我的网站:

*   **[响应者:](http://www.responsinator.com/)**
    [![responsinator](../Images/099156300d1b03f11da2704de7c2935c.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--CAammDxq--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://cdn.shortpixel.ai/client/q_glossy%2Cret_img%2Cw_930%2Ch_180/https://www.lambdatest.com/blog/wp-content/uploads/2018/12/pasted-image-0-3.png) 
    一个免费使用的工具，您可以在其中检查您的网站对最新设备的响应能力。在他们的主页上检查 4-5 台设备的响应情况，或者附上你的网址[http://www.responsinator.com/?url=YOURURL.COM](http://www.responsinator.com/?url=YOURURL.COM)例如，[http://www.responsinator.com/?url=www.google.com](http://www.responsinator.com/?url=www.google.com)。登录以访问更多设备。

*   **[LambdaTest](https://www.lambdatest.com/)**
