我是一名***【Web Developer Convert】***的学生，在攻读国家文凭(nd)课程期间，我花了大量时间学习 java 和 c++等语言，用 visual basic 6 完成了一些项目，并在攻读国家高级文凭(HND)课程期间花了更多时间进行计算机维修、维护和升级。我希望直接进入软件开发，而完全不涉及 web 开发。但这一切都随着时间的推移而改变，现在网络开发是不可避免的，因为互联网和网络浏览器已经取代了桌面原生应用，并且仍在增长。

在我的 HND 项目之后，我一直想尝试一些真实的东西，这在某种程度上是我职业生涯的起点，也是我成为全栈 web 开发人员的关键。然后我开始学习 HTML/CSS、JavaScript、PHP 和 MySQL，这花了我大约 3 个月的时间来启动项目工作。该项目是一个云存储平台的复制，用户可以上传、下载和删除他们的文件，也可以管理他们的个人资料。我和我的团队成员实际上通过使用 WAMP 和无线路由器在本地托管应用程序来进行项目的测试运行和防御，这使我们能够在其他设备上测试运行应用程序。从那以后，我一直想继续这个项目，并且已经开始在[000webhost.com](https://www.000webhost.com)上托管它，并在 [GitHub](https://github.com/chuksjoe/cloudhere) 上上传文件。你可以在这里查看:【Cloud-Here.com[。](https://cloud-here.000webhostapp.com)

在我的 HND 课程之后，我开始更多地玩代码，提高我的编程技能，还申请了一个初级开发人员的角色，在那里我可以从有经验的开发人员那里学到更多东西，并更好地理解软件工程的行为。可悲的是，我仍在寻找。尽管如此，一个朋友给了我一个绝佳的机会，是关于建立他们公司的网站。

我被要求会见他们当时的 BDM，他问我以前是否参与过任何项目。我向他明确表示，我是一个初学者，我唯一需要展示的项目是我的在线个人资料页面。我给他看了。这是我的第一个项目，也是当时唯一一个在免费托管空间托管的项目，这是我在 Bincom 申请实习项目时提供的，在这个项目中，我被要求在不使用任何框架的情况下，开发一个包含我简历内容的单页在线个人资料。我没有参加这个项目，所以我花时间在个人资料页面上做了些改进，从那以后我一直把它作为我的个人资料页面。[你可以在这里查看。](https://chuksjoe.github.io)

当 BDM 人看到它时，他很高兴，并答应说服公司的老板。我确信，鉴于我的实践经验，我实际上可以处理好这个项目，当公司的老板同意我的提议时，我立即开始工作，即使他们没有做出财务承诺，以向他们表示我的意愿。我不是被从中赚钱所驱使，我主要是渴望接触和体验一个真正的项目。

项目的初始阶段压力很大，挑战性超出了我从一开始的预期，因为在对一个真实项目如何进行的大量研究之后，我期待着一次会议，在会上我将会见相关的利益相关者，以决定网站应该是什么样子，内容应该是什么，他们喜欢的设计模式，以及诸如此类的事情，但这从未发生。我在一个项目中成为了内容开发人员、前端开发人员、后端开发人员、SEO 工程师，甚至是图形设计师。
牛逼吧？！！！是的，这的确是一次很棒的经历，很有趣，因为它让我做了比预期更多的研究。虽然，我花了更多的时间让项目进入可托管状态，但它最终肯定是完成了。[你可以在这里查看项目。](https://Astromaticng.com)

# [](#about-the-project)关于项目

我称之为 web 应用程序的项目是一个为 Astromatic 尼日利亚有限公司设计的公司网站，它讲述了公司的情况，他们卖什么，做什么。它还有一个产品部分/页面，在那里展示他们的商品，它是员工上传他们的产品到电子商务平台的后端。

除了 jQuery 和 bootstrap(其使用非常少)之外，我选择不使用任何框架或库，以便增加我对事物如何在内部协同工作的理解，并对实际代码有更广泛的了解。即使是电子商务部分和它的后端，我也不得不用普通的 PHP 进行硬编码，以积累我的经验和对语言的理解，这真的得到了回报。

在项目过程中，我的 HTML/CSS、JavaScript、PHP、MySQL 和图形设计技能得到了提升，我还将我的软件开发和项目管理理论知识付诸实践。使用敏捷方法，我仍然会在需要的时候给项目添加更多的特性。

在所有这些项目中，获得的经验非常棒，我真的迫不及待地想接手更多的项目。尽管如此，我仍然希望和渴望在更有经验的开发人员/工程师的指导下工作，他们将指导我的成长，并且在团队中工作以提高我的协作技能。

我非常乐意与任何一家不仅为公司，也为员工有增长思维的公司合作。不过，现在我是自由职业者，准备接受任何网络项目。
