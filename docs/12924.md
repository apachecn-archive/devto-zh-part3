# 反应堆管理仪表板主题

> 原文：<https://dev.to/dashboardpack/reactjs-admin-dashboard-theme-94b>

ArchitectUI 是一个干净、时尚的仪表板 UI 主题/模板，它是为 ReactJS 支持的现代应用程序而构建的。它包括许多组件、元素和部件，这样开发者就不会在视觉/设计部分浪费时间。他们只是把一些组件放在一起，瞧，这个应用程序看起来像是由专业设计师制作的。

React 版本基于脸书流行的 create-react-app 初学者工具包。它使用 Webpack、Yarn 和 SCSS(用于模块化/可扩展风格架构)。所有组件都使用通用的 Bootstrap 4 风格指南完美集成。基本组件由 ReactStrap 标准化，并已升级到 Bootstrap 4.2.1(最新版本)
