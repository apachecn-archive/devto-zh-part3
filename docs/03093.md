# 对技术时尚说不

> 原文:[https://dev.to/iagosilva/say-no-to-technological-fad-1nf2](https://dev.to/iagosilva/say-no-to-technological-fad-1nf2)

*原发布于[媒体](https://medium.com/interage/diga-nao-ao-modismo-tecnologico-cbbfa2d4ed0a)。*

我们当中有多少人已经对了解一项新技术感到兴奋，并且不假思索地马上开始使用它？

当然，我们都知道使用技术来加快我们生活的乐趣。另一方面，投资一个项目而没有带来预期结果的风险可能会很高。

这个问题是我们必须避免的，在投资某些技术趋势之前想好，避免不满。

## [](#trend-x-technological-fad)潮流 x 科技时尚

置身于技术创新之中是重要的，但要小心其植入的可行性，因为并非一切都像它看起来那样是花海。总是走在潮流的前面是好的，但是不要混淆潮流和时尚。

术语“技术时尚”是指工具的使用，因为它们是新的，而不是必需的。这最终会产生一个学习曲线和适应过程，并因此导致生产力不足，这是由于缺乏使用新工具的经验以及依赖尚未在社区中整合的工具的风险造成的。

## [](#investment-x-result)投资 x 结果

有时，我们会看到一个新的解决方案，它似乎比我们正在使用的解决方案更好，无论是美观还是功能，甚至是因为我们已经知道了它。

我们可能在谈论网络领域的新服务器，编程领域的新语言或新框架，或者总的来说，有望彻底改变我们生活的新机器。重要的是要注意:这项技术的好处会抵消投资和风险吗？

当主题是时间投资时，事情就复杂了，因为时间就是金钱。谁知道团队已经适应的当前工具是否会给我们带来与新工具相同的结果？或者它会带来更少的结果，但它是否仍然值得继续工作，而不是投资一个新的工具来获得更少的结果？

## [](#adaptation)适应

维持一个适应某种工作方式的团队是很重要的，因为这给工作带来了舒适和安全。适应时间通常是好的，因为走出常规，学习新的东西可能是可爱的，但它也可能导致消极和不安全。

当选择一项新技术时，我们必须考虑到它的所有优点和缺点，从谁是它的创造者以及他们对未来的计划，毕竟，我们不能冒险投资于没有未来的东西，不是吗？

## [](#conclusion)结论

不落后于技术新闻和避免某些尚未在社区中整合的趋势一样重要。

在没有必要的情况下投资新趋势的风险可能很高，因为其创造者停止使用该工具，或者您的业务没有适应它希望成为的解决方案。当涉及到植入新技术时，可行性分析对于做出好的选择是至关重要的。
