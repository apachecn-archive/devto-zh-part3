GITHUB 电子邮件采购#1

找到任何你感兴趣的开发者的 GitHub commit。单击提交名称以打开提交页面。补充。修补到地址栏中页面地址的末尾。搜索(Ctrl+F) "@ "。获取开发人员的电子邮件。

FACEBOOK 采购瑞士刀

去 https://inteltechniques.com/osint/facebook.html。你会发现自己在英特尔技术公司的定制脸书工具页面上。将脸书用户名(从脸书个人资料页面的地址栏中)粘贴到 FB 用户名字段中，获取用户编号并用该编号填充所有字段。只需点击一下鼠标，你就能找到脸书的所有信息。

按国家导出所有堆栈溢出配置文件

去 http://bit.ly/2uhr6sU。您会发现自己在 StackExchange Data Explorer 的“按国家列出的顶级用户”查询页面上。在页面底部的“国家/地区名称”搜索栏中输入国家/地区。点击“运行查询”。您将获得您感兴趣的国家/地区的 Stack Overflow 用户的可点击列表，该列表根据声誉进行排名。您也可以下载 CSV 格式的列表。

按位置和技能导出所有堆栈溢出配置文件

去 http://bit.ly/2yIrP6P。您会发现自己在 StackExchange Data Explorer 的“查找您所在城市的堆栈溢出用户(带有用户链接)以及他们对某个特定标签的评分”查询页面上。在“标记名”搜索字段中输入技能，并在页面底部的“位置名”搜索字段中输入您感兴趣的城市。点击“运行查询”。您将获得您感兴趣的城市的标签分数排名的堆栈溢出用户的可点击列表。你还会看到用户的信誉和排名情况。该列表可以作为 CSV 文件下载。
