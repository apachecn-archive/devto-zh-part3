# 如何恰当地为你的(边)项目设定目标？

> 原文:[https://dev . to/eekayonline/how-to-set-goals-for-your-side project-right-2pl 3](https://dev.to/eekayonline/how-to-set-goals-for-your-sideproject-properly-2pl3)

为了明确项目的“成功”是什么样子，定义目标总是好的。就像骑自行车一样，看看你想去哪里总是好的，这相当于为你正在做的项目设定目标。

最近我一直在思考我想为 Shipharder.com 达到的具体目标。事实证明，只设定一些目标而不考虑你首先要努力建立什么是很难的。

我对这一切进行了认真的思考，发现只有在定义了 Shipharder.com 是什么并思考了它的发展方向后，我才能实现具体的目标。

这就是为什么我设定目标的方法分为以下具体步骤:

*   写下你的产品或项目的定义
*   明确任务是什么
*   根据前面的要点，决定今年需要达到的目标

就 Shipharder.com 而言，这一切导致了以下情况:

*   定义:Shipharder.com 展示了关于如何构建可行产品的思想塑造内容，并记录了开发人员在这方面的努力

*   使命:Shipharder.com 提供有趣和鼓舞人心的内容，将引导(数字)产品制造商获得一个心态，创造可行的产品

明确这一点后，我把自己的目标定义为:

*   📩Shipharder.com 邮件列表的订户达到 1000 人
*   📈每月有 5000 名访客阅读 Shipharder.com 基石博客帖子并观看视频博客
*   💵网站产生的广告收入:以收入为目标(任何！)和 2019 年的支出
*   📦至少我的一个副业变得可行，并给我带来额外的收入

这些目标并不令人震惊。我知道，为星星而奋斗会帮助你实现比预期更大的目标。
