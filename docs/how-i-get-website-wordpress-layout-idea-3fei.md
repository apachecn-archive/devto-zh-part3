# 我如何得到网站布局的想法

> 原文：<https://dev.to/joehobot/how-i-get-website-wordpress-layout-idea-3fei>

我正在为我自己的网站做一些 wordpress 设计，想真的让它看起来很好，有一些流量。成为 DevOps 并不意味着我也是一个超级网络极客营销者，寻找按钮是这种绿色还是那种绿色。

我所关心的是我的网站在 2 秒内加载，谷歌评分超过 91(目前排名 94)。

这就是我所做的，我去 pinrest，我一直认为它更适合时尚人士和 etsy 人，然而在寻找一种特定类型的模板时，我有一天登陆了 pinrest，发现有数以千计的模板可供我浏览。

看看这个 sh**！[元素或模板](https://www.pinterest.com/search/pins/?q=elementor%20templates&rs=rs&eq=&etslf=1972&term_meta%5B%5D=elementor%7Crecentsearch%7C0&term_meta%5B%5D=templates%7Crecentsearch%7C0)

好的，接下来就搜索红色主题:)..

我去的另一个网站是 [99designs](https://99designs.com) ，如果你一定要找到这个竞赛链接，只要一直滚动到底部，然后点击[设计竞赛](https://99designs.com/web-design/contests?sort=start-date%3Adesc&status=won&entry-level=0&mid-level=0&top-level=0&dir=desc&order=start-date)，在那里你会找到各种各样的东西，大部分看起来真的很酷。

所以你有我的 Devs。

顺便说一句，我最喜欢的 wp 速度模板是 Neve 与图像优化+ Cloudflare DNS。
