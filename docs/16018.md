[![ld00d profile image](../Images/3e719e2c51a08b7eab7ffa9910e515b0.png) ](/ld00d) [ Brian Lampe ](/ld00d) • [<time datetime="2019-01-13T18:38:39Z" class="date-short-year"> Jan 13 '19 </time>](https://dev.to/ld00d/comment/852m) 

要回答这个问题，首先看看为什么 Windows 和 OS X 像现在这样受欢迎。

你今天可能购买的大多数台式机或笔记本电脑都带有 Windows。要获得没有 Windows 的硬件，你必须想办法。戴尔生产他们将安装 Ubuntu 的机器，但要找到这些机器并不容易，你真的必须*寻找*它——这不是大多数客户打算做的。

OS X 在苹果电脑上出现。不管出于什么原因，这些买家中的大多数都想要一台 Mac，而且他们得到了 OS X。我在这里是一个局外人，因为我喜欢 MAC 的一个主要原因是它们运行 OS X。

大多数消费者不会说“好吧，所以我需要一台运行 OS X 的机器，甚至我需要一台可以安装 OS X 的机器。”他们购买的硬件附带任何操作系统。

有卖 Linux 机器的公司叫“Chromebooks”:)。再说一次，他们并不是在寻找一台 Linux 机器，甚至不是一台必须运行 ChromeOS 的机器。他们想要一台便宜、安全、能在浏览器中运行的机器。

我认为 Chromebooks 甚至 Android 在这里回答了这个问题。“桌面 Linux 年”到来的唯一途径是某家公司制造一种人们想买的设备，恰好运行 Linux。大多数购买者并不关心操作系统是什么，只要它能做他们需要做的事情，比如运行 MS Office，运行他们玩的游戏等。

嗨，亚夫卡里，

谢谢你。我认为你特别提出的一个重要观点是，学习一门新技能，比如一门语言，常常会被工具所掩盖。Git 就是一个很好的例子，我遇到过很多只通过 UI 工具了解 Git 的人。

这在一开始是没问题的，但是一旦他们遇到问题，他们得到的最常见的反应是切换到终端来弄清楚发生了什么。这通常会让他们陷入困境，现在他们必须从终端学习 Git，然后才能开始处理他们试图理解的实际问题。

ide 对于完成工作和提高生产率非常有用，但是我觉得只有当你掌握了这个工具的强大功能时，它才会有所帮助。

不仅是为了`const`也是为了`var`和`let`。

这使得重构更加容易，因为每个声明都完全独立于其他声明。只需删除一行就可以删除一个变量。你甚至不需要把逗号改成分号。

另一个优点是:大多数搜索工具只显示搜索字符串出现的行。如果您在编辑器中搜索所有出现的`bar`，您可能更喜欢看到

```
const bar = 20; 
```
