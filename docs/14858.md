# 界限是远程工作的关键

> 原文：<https://dev.to/smizell/boundaries-are-the-key-to-working-remotely-4b15>

到目前为止，我已经远程工作了四年多，在使它适合我的过程中，我有成功也有失败。总的来说，这是一次很棒的经历，我将分享我认为远程工作的关键——建立良好的界限。

当我和人们谈论远程工作时，他们的反应是从“我很想这样做”到“我永远也做不完任何工作”人们对远程工作带来的自由感兴趣，但他们担心自己的工作效率不高，或者无法保持健康的工作生活平衡。

好的边界是关键。好的界限是关于创造一个工作空间，这样你就能集中注意力。好的界限是关于定义你什么时候工作，什么时候不工作，这样你就不会一直工作。良好的界限是指诚实地对待远程工作的性质，这样你和你的同事就能很好地合作。

# 设置一个工作地点并只工作

如果我只能给你一个建议，那就是试着在你的家里指定一个地方工作，并且让它成为你唯一工作的地方。如果可能的话，创造一个与你的生活空间分开的工作空间，因为我发现在沙发或餐桌上工作太容易了。这导致了坏习惯，比如当你不想工作的时候工作。你的工作空间可以是整个房间，也可以是远离每个人的桌子。无论你做什么，试着找到一个你可以集中注意力的工作空间。

此外，尽量不要在你的工作场所做私事。虽然很难只为工作留出空间，但你会发现这有助于工作和生活的分离。我发现即使我在工作场所做个人任务，也很容易被工作任务吸引。

对我来说，一开始我没有把我的工作放在工作空间里，这让我很难受。我有一个单独的空间，但它并不舒适，我最终在房子周围的其他地方工作。这使得生产率难以提高。一旦我改善了工作环境，习惯了在工作环境中度过时间，事情就变得更好了。

注意，我并不是说你不应该离开你的工作场所。我建议你不要把生活区和工作区混在一起。一定要停下来看看你的家人，去散步思考，和朋友一起吃午饭，或者找一家你喜欢工作的咖啡店。我发现我需要这些时间，否则我不会离开家。

# 当你在工作时，你就是在工作

保持工作与生活良好平衡的第二个关键是安排好工作时间。在家工作很难做到这一点，如果你不能创造一个独立的空间，那就更难了。但是，如果你能搞清楚工作空间和时间的界限，你就会处于非常好的状态。

首先，重要的是把进入你的工作区域当成是去工作。如果你在实体办公室工作，你会在某个时间起床，穿好衣服，准备好，去办公室，工作一会儿，然后回家。尝试继续这种模式。即使没有人会在你工作的时候看到你，也要为这一天做好准备——这有助于创造一种分离感，有助于自我照顾。当我等到一天的晚些时候才准备好的时候，我从来没有那种开始工作的感觉。整个上午都模糊在一起。

当你在工作空间时，你就是在工作。你的家人和朋友不得不那样对待你。你不能做家务、跑腿或帮助别人。人们很容易认为你是可交往的，因为你的身体接触更多，但你必须努力设定这个界限。除了你的同事，对其他人来说，你是不可接近的。

随着时间的推移，我和我的孩子不得不一起学习这个界限。一开始，他们会闯进我的办公室打招呼，这是我喜欢的，但这会分散注意力，很多时候会打断会议。他们现在知道我在工作，他们会悄悄地把纸条塞到我的门下。这样效果更好，而且总是一个有趣的惊喜。

也有例外，你需要放下工作去办点私事。如果你有一项常规任务，比如每天上班时间接孩子，先和你的经理谈谈。他们可能会理解，如果你已经建立了远程工作的信任，你的经理应该不会有意见。

# 当你不工作时，你就是不工作

当你下班了，你离开了你的空间，上面的情况正好相反——你的同事应该把你当成不可接近的人。建立这种关系的最佳步骤是严格控制你的工作时间。你不应该被期望一周工作超过 40 小时，如果你是，那也应该是个例外。为你的工作确定一个开始时间和结束时间，并坚持下去，否则你会很难停止一天的工作。你的同事会理解你的模式，并在工作时牢记在心。但这也要求你尊重他们的个人时间，并期望他们在一天中的某个时间结束。信任是双向的。

大多数日历工具可以帮助确定开始和停止时间。例如，Outlook 让你设置工作时间。我也把吃午饭的时间划出来，因为当人们试图安排会议时，它会显得很忙。预定的休息时间让你可以不吃午饭，这样你就不用在办公桌前吃饭了。

# 列出你的时间表来完成工作

视频通话是人们远程通话的主要方式。他们工作得很好，但是如果你有太多的会议，这些会议会占据你一天的大部分时间。如果你工作的地方任何人都可以把会议放到你的日历上，如果你的工作堆积如山，划出时间来完成一些事情。如果不紧急，人们会理解并乐意安排。

# 仅在工作时使用你的工作电脑

如果你从雇主那里得到一台不错的电脑，用它来处理个人事务会很有诱惑力。试着反抗。如果你下班后在电脑上处理个人事务，工作将会是你整个时间的焦点。电子邮件、通知和打开的文档会试图从你的私人时间中偷走你的时间。即使是快速浏览一下电子邮件也能让你想到工作。

对我来说，答案是买一台个人电脑，只用于个人任务。我不在工作场所使用它，也不在上面做任何与工作相关的事情。这是额外的开支，但它有助于区分工作和生活。

注意，如果你有个人电脑，有些法律问题你不必担心。如果你在你的工作机器上建造了一些有价值的东西，你的雇主可能会要求你的工作的所有权。在你的雇主那里，你可能不担心这一点。然而，我认为如果可以的话，把事情分开会更安全。在开始远程工作之前，请咨询律师。

# 如果可以的话，不要用手机上的工作应用

我努力让工作邮件或聊天应用远离我的手机，但如果可以，尽量不要添加它们。它让你早上一醒来就很容易拿起手机，让你的大脑进入工作模式。但是当我出门在外时，我需要查看我的电子邮件或日历，所以我总是最后安装它们。这里的答案可能是，当你看着工作的东西时，练习自律，而不是完全把它拿掉。总是有平衡的。

在手机上保存与工作相关的应用程序很容易在你不知不觉中打破你的界限。下班后，你可能会发现自己在沙发上工作，拿起电话，打开电子邮件。电话会给你一种错觉，以为你不在工作——你只是在打电话。如果你这样做，你的家人会改变他们的期望，你的同事会开始认为你在不同的时间工作。尽可能避免电话陷阱。

# 成功依赖他人

这些界限是你为自己设定的。然而，要想成功，就需要别人尊重他们。当然，你也需要尊重他们的界限，并随着时间的推移建立信任。你必须提醒人们你创造的界限。你可以说一些小事情，比如，“你介意我们改变会议地点吗？就在我吃午饭的时候，”人们会理解的。即使下班后不工作，也会让人们感觉到你的界限。对我来说，这是发展远程优先或远程友好文化的关键。

# 但是如果不能设定这些界限呢？

你可能无法为你的工作创造一个单独的空间。你可能没有钱买一台个人电脑。我认为你仍然可以成功地远程工作，但是可能需要更多的训练。为了工作，你可能要做更多精神上的分离，而不是身体上的分离。不管怎样，记住健康的界限。

你也可能无法正常每周工作 40 小时。如果你经常被要求在正常工作周之外加班，而且你和你的经理关系很好，那就和他们谈谈。寻求他们的帮助，创造一个更好的工作生活分离。优秀的经理总是渴望帮助他们的员工获得成功。

# 我推荐体验

对我来说，远程工作非常棒。我不用通勤，我可以在早上见到我的家人，我可以住在一个平静的小镇上，我可以和世界各地的优秀人士一起工作。我很幸运在我的生活中有人帮助我处理这些界限——特别是我的好妻子——我和那些让工作变得容易得多的人一起工作。如果你有机会和方法让它发生，你应该试一试。
