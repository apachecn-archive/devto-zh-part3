但是...我很开心。

Ruby 似乎非常适合 Lambda，任何非本机支持的东西通常都不值得。

Ruby 的好处——简单易读的语法，非常适合编写小问题的脚本——让我觉得非常适合无服务器环境。作为一种支持 Rails 等的服务器端语言，Ruby 本身并不擅长我们要求它做的事情。

无论如何，作为一个 Rubyist，这对我来说是可爱的。我很想听听其他人的看法。
