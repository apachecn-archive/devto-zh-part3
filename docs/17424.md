# ★2018 年回顾

> 原文：<https://dev.to/freekmurze/a-recap-of-2018-58b7>

现在 2018 年即将结束，[a](https://davidllop.com/posts/year-retrospective-2018)[lot](https://adamwathan.me/2018-year-in-review/)[of](https://sebastiandedeyne.com/hindsight-2018-edition)[牛逼](https://dev.to/gonedark/2018-dev-goals-4abd) [人](https://christoph-rumpel.com/2018/12/my-coding-year-2018) [有](https://www.dieterstinglhamber.me/blog/2018-looking-back-at-an-awesome-trying-year) [写的](https://themsaid.com/thats-a-wrap-2018) [a](https://bitpress.io/2018-year-in-review) [重述](http://marcelpociot.de/blog/2018-my-year-in-review) [帖子](https://ericlbarnes.com/2018/12/28/my-2018-a-look-back)。这是我的！

个人层面最重大的变化是我和我的女朋友在根特买了房子。今年夏天，我们和两个孩子一起搬了家，现在我们住在一个非常棒的房子里，离市中心很近。对于一个城市人来说，这真的很好。它离火车站也很近，这很方便，因为我需要经常去安特卫普。

因为我经历了一些精疲力竭的症状，我试着再做一些编程之外的事情。我又开始骑自行车了。今年夏天，我骑着我的赛车从安特卫普到根特，来回跑了几次。当比利时稍微暖和一点的时候，我会再骑出去。与 2017 年相比，我在家的时间也多了很多，这非常好！

我也开始和几个朋友重新录制一些音乐。这里是其中的两个:夏洛特和 T2。

## 启动副项目

在代码方面，今年我最自豪的事情是我的副业项目的启动。和我的好友马蒂亚斯一起。我推出了[天啊！这项服务不仅监控主页，还监控你网站的每一页。](https://ohdear.app)

如果你想知道更多关于这个过程的信息，可以看看 Mattias 对独立黑客的采访。

我最喜欢在主要工作之外做一些事情，因为没有时间和预算的限制。我们没有做市场调查。没有什么大的计划。当我们觉得它对我们自己的网站来说足够稳定时，我们凭直觉推出了它。

我想长大哦，亲爱的！再来一些。马蒂亚斯和我很早就明白传统广告对我们不起作用。我们不是在潜在客户面前推销广告，而是通过提供额外价值来让他们了解我们。我们通过我们的博客吸引了很多新客户。明年，我们将尝试通过添加一些免费功能来吸引 Laravel 社区之外的受众。留意那个博客的更新。

我个人的一个亮点是 [Blade X](https://docs.spatie.be/laravel-blade-x/v2/introduction) 的创作过程。我们和整个团队坐在一起，[集思广益，一起创造了最初的版本](https://freek.dev/use-custom-html-components-in-your-blade-views#how-we-made-this)！

我们还发布了一个新的主版本 laravel-medialibrary，增加了对 T2 响应图像的支持。

在 Spatie 的开源工作之外，我与 [Marcel Pociot](https://github.com/mpociot) 合作创建了 [Laravel WebSockets](https://github.com/beyondcode/laravel-websockets) 。我们很高兴看到这个结果，也很高兴这个包裹受到了社区的热烈欢迎。我真的很喜欢和 Marcel 在一起——这是我第一次和 Spatie 之外的人结对编程——我们已经在忙于另一个合作包了。

让我们把焦点回到斯帕蒂身上。三月份，我们庆祝了 1000 万次包下载。我们为它创建了一个专门的网站，人们可以在那里赢得一件 t 恤。在写这篇文章的时候，我们开源包的下载量已经达到了 2400 万。

7 月，我们(终于)推出了我们公司网站的新版本[。它现在更加关注我们的 Laravel 专业知识和我们的开源努力。这里有一篇关于后台](https://spatie.be/)[发生的一些很酷的事情的博文。我的同事威廉做了大部分的前端工作，这是](https://freek.dev/a-new-website-for-spatie-backend-highlights)[在上面的客座博文](https://freek.dev/a-few-notes-about-the-frontend-of-the-renewed-spatiebe)。

Willem 也是更新我们仪表板的驱动力。它现在使用 Tailwind CSS。在[这篇博客文章](https://freek.dev/building-a-realtime-dashboard-powered-by-laravel-vue-pusher-and-tailwind-2018-edition)中阅读更多相关内容。

斯帕蒂的团队增加了三名新成员:沃特、阿德里安和鲁本。

我们现在承接的项目比前几年更大。即使我们有时仍然创建小的推广网站，我们大部分时间花在中型到大型的应用程序上。

2019 年，Spatie 可能会继续增长。我们已经在网站上发布了一些职位空缺。我们的办公室变得太小了，我们计划搬到一个更大的地方。

尽管我们的团队和客户工作量在增加，但我们希望继续关注团队成员的幸福，保留足够的时间从事开源项目，并将每个团队成员的利益放在心上。

## 说话之间

我仍然喜欢会议演讲。在我的博客上，你会发现[一个我今年做的很多演讲的列表](https://freek.dev/me#talks)。

2018 年，我在四个不同大洲的会议上发言。圣保罗的 PHP 经历标志着我第一次来到南美。芝加哥的 Laracon US 和预期的一样，非常酷！会议结束后，我喜欢探索这座城市本身。

拉拉康欧盟仍然是我最喜欢的会议之一，主要是因为那里有很多了不起的人。今年， [Spatie 有一个展位](https://www.instagram.com/p/BnJPy8hn5Dl/)，几乎整个 Spatie 团队都在那里。希望明年我们能重复这一点。

最后但同样重要的是，看到[拉利特](https://twitter.com/lalitvijay)、[法赫德](https://twitter.com/fahadkhan1740)和他们的工作人员回到新德里参加拉韦尔现场会议是一次奇妙的经历。在现实生活中见到[哈里什](https://twitter.com/introwit)也很有趣，他一直在远程为斯帕蒂做一些工作。会议结束后，我们参观了泰姬陵。那是难忘的一天，我希望在 2019 年回到 Laravel Live。

尽管我喜欢公开演讲，但我可能会少做一些。会议演讲意味着很多旅行，我喜欢旅行，但这也意味着我不在家。有了两个孩子，我现在不想再经常旅行了。但是你可能仍然会在美国和欧盟的 Laracon 上看到我，无论是作为演讲者还是与会者。

## 全栈用户组

我与 Dries 和 Rias 一起组织了[全栈安特卫普](https://www.meetup.com/fullstackantwerp/)用户组。2018 年，我们实现了每月组织一次聚会的目标。我们的观众增长了不少。我们的每次活动都有 60 多人参加。

2019 年，Rias、Dries 和我将启动另外两个用户组:[全栈布鲁塞尔](https://www.meetup.com/fullstackbrussels/)和[全栈根特](https://www.meetup.com/fullstackghent/)。

我正在组织的另一个很酷的活动是全栈欧洲会议。Dries 和我以前都没有开过会，所以这很令人兴奋。我们已经在幕后做了很多准备。我不能分享太多，除了它将在 2019 年秋天左右在安特卫普发生。在我们的网站上订阅我们的邮件列表[,了解最新动态。](https://fullstackeurope.com/)
