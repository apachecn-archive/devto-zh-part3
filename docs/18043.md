# 深层已学习模型的交换格式 ONNX

> 原文::t0【https://dev . to/vochiong/-onnx-2 AFL】

作为深层已学习模型的交换手段，开放格式的[ONNX](https://onnx.ai/supported-tools) 正在逐渐普及。 原本好像是 Facebook 开始的，现在由 AWS 和微软赞助的样子。

查看支持列表。

结构

*   咖啡 2
*   链条机
*   认知工具包
*   MXNet
*   矩阵实验室
*   PyTorch
*   ...

转换器

*   Keras
*   ScikitLearn
*   TensorFlow
*   XGBoost
*   ...

另外，谷歌推荐[TFHub](https://tfhub.dev/) ，在 ONNX 中没有[积极的与](https://github.com/tensorflow/tensorflow/issues/12888)相关。
