标题:开源时间追踪软件

衡量花费在工作上的时间一直是了解绩效和计算应付金额的最简单和最常用的方法之一。工业革命后，守时成为一种必须:在大批量生产和轮班工作的时代，“时间就是金钱”的陈词滥调定义了雇员和雇主之间的关系。

如今，时间是员工最宝贵的资产之一，计时不仅是计算和证明可计费金额的最有效工具之一，也是分析生产力和优化工作流程的最有效工具之一。

不仅公司发现有必要记录工作时间。自由职业者和个体创业者是时间追踪工具的一大群用户。他们使用时间费用报告为客户开账单，分析他们承担的项目的盈利能力，并研究他们的个人表现。

低价格和根据特定需求调整工具的能力通常是选择正确计时软件的关键因素。在这一选择中，我们回顾了许多技术专家肯定会喜欢的免费和开源时间跟踪软件。

Kimai
Kimai 是一个简单的时间跟踪器，可以记录你的时间，并在你需要的时候打印出结果。该工具非常适合那些需要一个没有任何额外功能的简单解决方案的人:在其中，您可以在多用户模式下记录时间，管理客户和项目数据，运行报告，以及导出时间表数据。

安装该应用程序不需要数据库设置，它有现成的数据库支持。Kimai 的新版本基于最新的技术，并有一个 JSON API 用于设置与其他应用程序的集成。

**[racho ta](https://sourceforge.net/projects/rachota/files/)**
racho ta 是一款轻量级便携计时 app，可以运行在各种平台上。它的时间跟踪技术是基于计时器的:应用程序创建任务并为它们跟踪时间。Rachota 还提供关于如何有效利用工作时间的基本报告，并就如何提高生产率提出建议。报告可以导出为 HTML、CSV 或 TXT 格式。

由于该应用程序是便携式的，所以没有必要安装在电脑上。你可以在任何机器的 u 盘上运行它。由于它是用 Java 编写的，所以可以在各种操作系统上工作——开发者在网站上指定了 Windows、Linux、Solaris 和 Mac。

**[open project](https://www.openproject.org/collaboration-software-features/time-tracking-software/)**
open project 是一个项目管理、预算和计时解决方案，有助于跟踪时间和成本。在此工具中，您可以创建项目并将其分配给用户，设置项目预算，并跟踪您的团队的表现。该工具还具有任务管理和甘特图模块，用于更详细地了解项目工作是如何执行的。

OpenProject 的社区包永远免费。它的云和企业选项是付费的，基于云，因此不需要安装。可下载选项可用于各种 Linux 发行版:CentOS、Debian、Suse 和 Ubuntu。

**[魅惑](https://github.com/KDAB/Charm)**
魅惑时间追踪器是一款比较新的、很有前途的跨平台时间追踪器 app。它允许创建任务和子任务，并跟踪它们的时间。对于那些只需要一个带有基本报告选项的时间跟踪器的人来说，这是一个简单的解决方案。Charm 提供每周和每月的性能报告，可以用 XML 导出或打印出来。

**[eHour](https://ehour.nl/)**
eHour 是一个基于网络的公司和个人时间追踪器，支持多个项目、分配给所有用户的默认项目、锁定时间表、丰富的报告和自动电子邮件提醒。虽然 eHour 比大多数其他开源计时工具具有更广泛的功能，但它的设计尽可能简单和用户友好。

eHour 可用于 Windows、Linux 和 macOS。除了可下载版本，还有一个新的基于云的版本。它是在订阅的基础上作为服务提供的。

**[项目仓鼠](https://github.com/projecthamster/)**
项目仓鼠是一款为个人计时的 app。它自动跟踪每天的活动与时间段，并显示在图表中。您可以将位置添加到活动中，例如将“工作”位置附加到“会议”任务中，或者使用标签对任务进行分类。该工具还支持向特定活动添加描述。

项目仓鼠允许运行报告，统计你如何度过你的时间，出口到不同的格式(HTML，XML，iCal)，并打印出来。

**[沙漏](https://sourceforge.net/projects/hourglass/)**
沙漏是一个供个人使用的开源时间跟踪器，它允许记录在各种任务上花费的时间，然后运行报告来分析性能。它支持多种级别的时间跟踪层次结构、日历视图和记录特定活动的时间段。你也可以使用计时器来自动计时。

沙漏允许生成基本报告(考勤卡),记录一周或任何特定日期范围内的时间。您可以使用它们来为客户开账单、提高工作效率或查看时间是如何花费的。

**[Anuko](https://www.anuko.com/time_tracker/index.htm)**
Anuko 是一个简单的时间跟踪工具，支持基本功能，如时间跟踪、报告、团队管理和数据导出。在 Anuko 中，您可以在两种模式下跟踪时间——项目，其中每个时间条目都与特定的项目相关联，或者项目和任务，其中根据构成项目的任务来跟踪时间。

Anuko 是一个 web 应用程序，可以在不同的操作系统上运行。该工具安装在一个 web 服务器上，并与一个数据库一起工作，该数据库收集并存储团队记录的时间数据。

**[Kapow](https://gottcode.org/kapow/)**
Kapow 是一个打卡出入工具，允许跟踪不同项目的时间。在其中，时间跟踪是基于时钟方法:点击开始/停止按钮，打卡上班和下班。如有必要，该工具允许自动更正捕获的时间条目。

对于那些使用计时工具为客户计费的人来说，Kapow 提供了记录自客户上次计费以来的小时数的功能。为此，使用特殊复选框将之前记录的小时数标记为“已计费”，并跟踪下一个日期的时间。

Kapow 安装包适用于 Windows、Linux 或 MacOS。

**[bara LGA](https://github.com/Baralga/baralga)**
bara LGA 是一款轻量级的时间追踪应用，允许使用计时器记录项目所花费的时间，或者手动记录时间——可以使用快捷方式快速输入数据。该工具提供基于任何特定时间段记录的数据的报告和图表，并允许将它们导出到 Excel、CSV 或 iCalendar。

该工具作为桌面应用程序或便携式版本提供，因此您可以从 USB 驱动器上使用它，而不是将其安装在机器上。
