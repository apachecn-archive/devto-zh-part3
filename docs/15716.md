# Firefox 多账户容器太棒了！

> 原文:[https://dev . to/lampewebdev/Firefox-multi-account-containers-are-awesome-357 c](https://dev.to/lampewebdev/firefox-multi-account-containers-are-awesome-357c)

我使用三种类型的账户:

*   私人的
*   全职工作
*   网站开发工作

如果你想知道我为什么用三种账户？请在下面留下评论。

我一直在不同网站的这些账户之间切换。

用我的工作帐户登录 Gmail，同时用我的私人帐户登录 Gmail。所以我可以看这两封邮件。

再比如 YouTube。我在每个账户上都订阅了不同类型的频道。

火狐多账户容器来拯救！

从我发现它们的那天起，我就在任何地方使用它们，因此 Firefox 是我的主要浏览器。

它们是如何工作的？

火狐有`profiles or containers`。如果你真的看幕后，它只是你硬盘上的一个文件夹，firefox 在那里保存你的配置或历史等等。

您可以通过使用-p 标志启动 firefox 来更改该配置文件，然后会提示您一个配置文件列表，您可以在其中管理它们。所以这不是什么新鲜事，但现在你可以在同一个窗口中使用这些配置文件，但在不同的标签！

好吧，让我们来设置一下

*   1)安装 Firefox(你好队长明显)
*   2)安装这个[延长件](https://addons.mozilla.org/en-US/firefox/addon/multi-account-containers/)
*   3)乐在其中！

现在，当您长按“新建标签”按钮时，您会看到以下内容:

[![firefox screenshot](../Images/8cacf9b96302ed2388915b8e26fd88ec.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--HV7CZH-v--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://addons.cdn.mozilla.net/user-media/previews/full/196/196940.png%3Fmodified%3D1543520769)

默认情况下，Firefox 为你创建了 5 个容器！

*   1)默认容器(无颜色)
*   2)个人容器(蓝色)
*   3)工作容器(橙色)
*   4)银行集装箱(绿色)
*   5)购物容器(粉色)

如果你想像那样使用它们，就这样做，这是一个非常好的设置。你可以通过点击右边有 4 个方块的符号来编辑它们

[![firefox screenshot 2](../Images/4d7d45b690bd123959e8efd2c57a29b9.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--R5ebxnYK--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://addons.cdn.mozilla.net/user-media/previews/full/196/196941.png)

太酷了，现在我们有了多账户容器，我们可以同时登录 3 个谷歌账户，只需要一个标签！
