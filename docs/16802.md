# 软件就像诗歌

> 原文:[https://dev.to/futoricky/software-is-like-poetry-1nmp](https://dev.to/futoricky/software-is-like-poetry-1nmp)

当我在学校的时候，我写了很多诗，我记得我试图用最好的方式来表达一种感觉或情绪，我想表达的正是我想要的。这种感觉或情感可以用许多不同的方式来表达相同的信息，但作为内容的创造者，一个人的自我会寻找与你的思想试图在诗中描绘和展示的内容相联系的正确词语。编码和这个没什么区别。

当你写代码的时候，你是在创建以完成某件事情为目标的软件。某些东西代表了你，作为创作者，想要通过为该软件编写代码来实现的表达或投影。某些东西将被编码/书写，就像你的大脑试图描绘软件的行为来完成“某些东西”一样，写诗也是同样的过程。

诗歌可以用许多不同的语言写成。
`Software can be written in many different languages.`

写诗是为了表达一个信息或激发一种情感。它是为了实现某个目标而写的。
`Software is written to complete a task or produce a result. It is written to achieve something.`

诗歌可以用无数不同的方式来写，表达同样的信息或激发同样的情感。
`Software can be written in an infinite of different ways and complete the same task or produce the same result.`

诗歌可能写得不好，也可能无法表达信息或激发情感。
`Software can be poorly written and can also fail in completing the task or producing the result.`

最终，无论是写诗还是编写软件，都需要创作者有能力表达他或她想表达的任何东西，以实现它在头脑中是如何制造的。这些艺术表现为对立，情感和逻辑，所以我觉得如果一个人练习其中一种，他们应该考虑练习另一种，以创造一种观点和/或思维过程的平衡，以及拥抱生活中所有执行的二元性。
