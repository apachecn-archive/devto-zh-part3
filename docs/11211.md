### [](#why-wordpress-needs-security-plugins)为什么 WordPress 需要安全插件？

WordPress 是最受欢迎和使用最广泛的博客平台。全世界数百万人正在使用它。全球 27%的网站使用 WordPress 。因此，黑客和垃圾邮件制造者也对破坏 WordPress 博客的安全性感兴趣。

WordPress 安全插件是提高 WordPress 网站安全性的最简单的方法之一。WordPress 本身是非常安全的，经常更新以修复任何发现的漏洞。但是拥有能够尽快检测到安全问题的插件也不为过。

安全插件可以解决一些重要的漏洞，或者至少强化你的 WordPress 站点容易被黑客操纵的方面。

### [](#important-wordpress-security-practices)重要的 WordPress 安全实践

在这篇文章中，我回顾并比较了一些最流行的免费和商业 WordPress 安全插件。

请记住，即使安装了所有最好的插件，也应该使用通用的安全措施来保证 WordPress 站点的安全。你的 WordPress 安装应该定期更新，以确保你没有使用有已知漏洞的版本。

#### [](#strong-password-enforcement)强密码强制

要求用户使用强密码登录 WordPress 账户。

#### [](#block-user-by-ip-or-by-country)按 IP 或国家阻止用户
