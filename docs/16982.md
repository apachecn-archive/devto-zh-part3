# 2019 年的目标

> 原文：<https://dev.to/mskog/goals-for-2019-jlj>

我已经看到许多人在他们的博客上发布他们的目标，所以我想我也会在 2019 年的软件开发和职业目标上做同样的事情。

## 在 [Udemy](https://www.udemy.com) 完成 6 门课程

我使用 Udemy 已经有一段时间了，我发现它既简单又有用。2019 年，我挑选了 6 门感兴趣的课程，我将确保完成:

#### [Vue JS 2 -完整指南](https://www.udemy.com/vuejs-2-the-complete-guide/learn/v4/overview)

我知道反应，但对 Vue 知之甚少。这似乎是一个很好的起点。新的 Vue CLI 3 看起来很酷，有一个内置的 UI，我发现它很有趣，Vue 可以在 DOM 中已经有的 HTML 上操作。也许 Vue 可以用在现有的应用程序中，而不是像在 React 中那样替换组件？

#### [高级 CSS 和 Sass](https://www.udemy.com/advanced-css-and-sass/)

设计不是我的强项，所以我认为这个课程是学习更多的好方法。还买了 [RefactoringUi](https://refactoringui.com/) 的书，2019 年要读。

#### [Go:完全开发者指南](https://www.udemy.com/go-the-complete-developers-guide/)

我只知道动态语言，一些 Java 和一点点 Crystal。我想学习一门语言，当我需要速度的时候我可以使用它。

#### [铁锈编程语言](https://www.udemy.com/rust-lang/)

人们真的很喜欢铁锈。也许我也会？

#### [完成 C# Unity 开发者 2D](https://www.udemy.com/unitycourse/learn/v4/overview)

在 Unity 引擎里一边做游戏一边学习 C#？3 个优秀的东西在一个大包裹里？优秀的

#### [理解打字稿](https://www.udemy.com/understanding-typescript/learn/v4/overview)

我不懂打字稿。在这之后我会吗？人们一直对它赞不绝口，所以那里一定有什么东西。

如果你知道 Udemy 上任何其他很棒的课程，那么请随时给我发电子邮件或在 Twitter 上@我

## 读 12 本非小说类的书

我一直想读更多的书，所以如果我下定决心，今年我可能会成功。我目前正在读[工作中不一定要疯狂](https://www.amazon.com/Doesnt-Have-Be-Crazy-Work/dp/0062874780)，这是目前为止最棒的。

## 读完 12 本小说

如上。多读书！我最近在读刘的《黑暗森林》。这是该系列的第二本书，如果你喜欢科幻小说，我可以强烈推荐它。

## 启动和/或发布 4 个副项目

我总是开始做我没有完成的事情，我有一堆没有完成的事情。今年我承诺至少完成 4 个。第一个将是一个涉及用 Elixir 编写的视频游戏的网站，第二个可能是一个更好的方法来处理有很多简讯要读的问题。我当然会尽可能地开源。

## 在这个博客里写 12 条(不包括这条)

和上面的项目一样，我也有几份草稿没有完成。今年我会的。例如，我发表了一篇关于 Mac 上 Docker 的性能问题的文章，一份关于 Ruby on Rails 的大量性能技巧的列表，以及一篇关于 Hetzner 上新的块存储的评论。敬请期待！
