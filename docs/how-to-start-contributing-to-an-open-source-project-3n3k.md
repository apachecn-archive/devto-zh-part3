# 如何开始为开源项目做贡献？

我对此想了很多，我得出的结论是，就像结婚或生孩子一样，没有“完美”的时间开始为开源项目做贡献。

事实是，我必须腾出时间。

# 我准备去买一些

所以，我愿意开始为此留出一些时间。但是我仍然有很多疑问:

*   我如何开始选择正确的项目？
*   如果我不能接受挑战呢？
*   如果我花太长时间解决问题或其他事情，会发生什么？
*   开始为开源项目做贡献的好路线图是什么？

我所说的路线图是指:

1.  我应该从我自己的项目开始吗？
2.  我应该从一个受欢迎的项目开始吗？
3.  我应该只参与 Github 社区讨论吗？

你是怎么开始做贡献的？我真的很想知道你是如何开始的；)
