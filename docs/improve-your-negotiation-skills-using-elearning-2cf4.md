专业人士说实践是学习之母。要精通任何领域，你需要全身心投入学习。持续的练习会帮助你开拓新的视角，在不同的谈判领域获得熟练的技巧，并成为一个更好的沟通者。

你有很多练习的机会，尤其是在网上。例如，最新的趋势提倡用[谈判训练角色扮演模拟器](https://www.negotiations.com/approach/negotiation-sim/)来练习你的谈判技巧。这种学习策略包括实际的谈判技巧训练，即使是最谦虚的谈判者也能提高他们的信心和能力。简而言之，你与另一个人(在线)配对，练习不同主题的不同类型的谈判。阅读完机密说明后，你有时间准备你的谈判方法，你将利用它从对方那里获得一些东西。

可以通过语音或文字聊天，与对方达成协议。然后，你将和你的同事一起回顾你的得分图表，给你提供和接受反馈的机会。信不信由你，这是成为更好的谈判者的最简单也是最有效的方法之一！

### 关注社交媒体上的专家

利用现有的流行社交媒体网络，它有可能帮助你在谈判领域获得更多的知识和专业技能。这包括在他们积极使用的社交网络上寻找和关注不同的谈判专家。大多数情况下，这些人会留下线索，你可以在谈判中找到并利用这些线索。
这里有一些找到谈判专家的建议:

使用 LinkedIn、脸书和 Twitter 的搜索功能——只需搜索“谈判”,你就可以看到大量可能让你感兴趣的个人、文章、视频和其他类型的资源。点击、阅读并决定跟随作者是否是明智的长期教育投资。

去 Quora 或者 Yahoo Answers，就谈判这个话题寻求有见地的答案。对作者进行同样的操作，需要时跟踪他们。

使用搜索引擎查找关于这个主题的视频、文章和播客。关注那些网站/作者。

### 看博客和论坛，看网络研讨会，听播客

另一个让你开始谈判艺术的免费方法与上面解释的非常相似。

不要关注个人和他们的社交媒体页面，开始关注发布与你的需求和问题相关的有价值内容的博客和网站。一个简单的谷歌搜索将让你立即开始。在 YouTube 上关注，或者通过在网上搜索特定的关键词，如“[电子商务销售](https://aeroleads.com/blog/kick-start-your-lead-generation/)”或“销售谈判”，来寻找谈判论坛。
