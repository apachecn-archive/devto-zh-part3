# 不要放弃。继续迭代

> 原文：<https://dev.to/vintharas/dont-give-up-keep-iterating-4031>

*本文最初发布于[barbarianmeetscoding.com](https://www.barbarianmeetscoding.com/blog/2019/01/03/dont-give-up-keep-iterating)。[joséAlejandro Cuffia 在 Unsplash](https://unsplash.com/photos/_4815u_ACqQ) 上拍摄的照片。*

不要放弃。不断迭代。

每当我从事任何创造性工作时，同样的过程都会展开:

1.  我想到了一些我可以建造、书写、绘画、做的事情，我变得异常兴奋，充满了热情。
2.  我把脑海中的想法变成了很棒的东西。令人愉快的美丽杰作。我想象着当我完成它的时候会有多酷。
3.  我创建了第一个版本，并且总是**很糟糕**。
4.  嘣！巨大的虎头蛇尾。我每天都会感到紧张、沮丧和/或有点沮丧。我质疑我的技能、价值和价值:*如果这很糟糕，那么我也必须很糟糕*。
5.  现在我有选择了。我可以留在这个绝望的深渊，我可以放弃，或者我可以振作起来，继续迭代。
6.  不变的是，一致的迭代会产生更好的东西。有时，甚至是一些不寻常的事情。

所以现在还不要放弃。迭代。你手上这块丑陋的无定形的石头可能会变成一颗漂亮闪亮的钻石。
