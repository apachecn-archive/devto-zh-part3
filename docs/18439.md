# Devart 为 PostgreSQL 开发的全新标准版 dbForge Studio

> 原文:[https://dev . to/devart software/brand-new-standard-edition-of-dbforge-studio-for-PostgreSQL-by-devart-1d 86](https://dev.to/devartsoftware/brand-new-standard-edition-of-dbforge-studio-for-postgresql-by-devart-1d86)

**标准版**

现在 dbForge Studio for PostgreSQL 有两个版本，一个是具有基本功能的免费速成版，另一个是具有高级功能的标准版。标准版可以免费试用 30 天。试用期结束后，用户可以购买许可证继续享受独家功能，否则将仅限于免费的快速版。

**标准版包括以下独有功能:**

*数据导入和数据导出

*主从浏览器

*数据报告

*数据透视表

*SQL 代码片段

*查询执行历史

*执行大型脚本

*SQL 文档

-选择查询的 FROM 列表中的上下文提示

*   SELECT 语句的完整代码完成支持

*连接性

-SSH 连接

-对 PostgreSQL 11.x 的连接支持

-对 PostgreSQL 10.x 的连接支持

-对 PostgreSQL 8.3、8.2、8.1 的连接支持

-对 elephantsql.com 的连接支持

*其他改进

-安装文件的数字签名

-FIPS 合规

**dbForge Studio for PostgreSQL**是一款用于数据库开发和管理的 GUI 工具。PostgreSQL 的 IDE 允许用户在一个方便且用户友好的界面中创建、开发和执行查询，编辑和调整代码以满足特定要求。
