# 昨晚我的副业让一个人高兴，另一个人生气了

> 原文:[https://dev . to/chipd/last-night-my side-project-make-one-person-happy and-someone-fucked-8pn](https://dev.to/chipd/last-night-my-side-project-made-one-person-happy-and-someone-else-pissed-off-8pn)

我一直在做一个副业项目，大约两年了(断断续续)。终于到了好到让人愿意掏钱买的地步，这是一种不可思议的感觉。

今天早上醒来时，我收到一个通知，说我一夜之间就卖出了一件东西。太神奇了！这绝对是一天的良好开端。

但是当我打开我的电子邮件时，我看到一个不满的用户发来消息，说我浪费了他们的时间，因为他们认为这是免费的...

当然，这是两个不同的人，也许随着越来越多的人发现你的应用程序，这是一件很自然的事情，我想也许作为开发者，我们只是习惯了互联网上免费的酷东西。

但是作为开发者，我也认为我们需要确保我们的工作得到重视，我们应该知道软件不是免费的。当然，有时我们会构建一些很酷的东西，并把它放在那里只是为了好玩——例如我不久前和一个朋友构建的用于管理其他 chrome 扩展的 [chrome 扩展](https://chrome.google.com/webstore/detail/custom-chrome-extension-m/balnpimdnhfiodmodckhkgneejophhhm?hl=en),但这并不意味着我们构建的所有东西都必须免费。

只是想分享一下——我非常支持让人们通过捐赠按钮或在你的兼职项目上贴上价格标签来支付你的工作——但很高兴听到其他想法。
