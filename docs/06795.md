## 快速启动

### 全球的

要在您的项目中使用，只需导入 vfc 并安装到 Vue 中。

```
import Vue from 'vue'
import App from './App.vue'
import VFC from 'vfc'
import 'vfc/dist/vfc.css'
Vue.use(VFC)

new Vue({
  render: h => h(App)
}).$mount('#app')
```
