我碰巧在尼日利亚的一个平台 Hng 进行远程实习，这个平台可以帮助我轻松预订酒店。

在实习期间，我们被要求调查 Timbu 并找出网站上可能存在的漏洞。在我的网站冒险中，我发现了一些与 UI/UX 相关的错误，同样他们的搜索算法也很奇怪。

在 UI/UX 部分， [Timbu](https://timbu.com/) 有一些 UI/UX 问题，其中之一是列表页面，显示一些酒店列表的重复图像，而在预订页面，UX 是一些多么可怜，尤其是预订部分，他们使用 HTML 元素“表格”，这不适合设计，同样是一个糟糕的 UX 体验。

关于搜索算法，我尝试从搜索框中搜索“伦敦”,这意味着列出伦敦的住宿，而不是以伦敦的名字显示住宿，一些没有伦敦相关的，而另一个伦敦的在列表中。通常情况下，搜索是指按位置搜索，但它并不像搜索文本中描述的那样工作。

这是我在网站上发现的几个 bug，我认为负责网站的开发人员还有很多工作要做，但是 Timbu 值得一试。
