# 我们能更鼓励有抱负的游戏开发者吗？

> 原文:[https://dev . to/whittington/can-we-be-more-increase-against-aspirated-game-developers-2fjp](https://dev.to/whittington/can-we-be-more-encouraging-towards-aspiring-game-developers-2fjp)

几周前，我刚从一个 web 开发训练营毕业，那里很酷，但是我不太喜欢 web 开发。我决定尝试游戏开发；这是我一段时间以来的热情所在，也是我继续编写代码的原因之一。当我告诉人们我想成为一名游戏开发者时，他们的反应让我很沮丧。

每次我告诉别人我想成为一名游戏开发者，结果通常都是可以预料的。我通常会得到两种典型结果中的一种。

## [](#dont-do-it-it-sucks)不做，太烂了

获得这种回应的人通常声称自己是某个游戏开发商的朋友或认识某个游戏开发商，并声称这个游戏很烂，他们不喜欢这个游戏有 X 个原因，不管是报酬、有害的工作环境还是其他第三个原因。

## 每个人和他们的祖母都想成为一名游戏开发者

获得这种回应的人通常试图通过声称每个人都想成为游戏开发者来阻止我成为游戏开发者，并对游戏开发市场的竞争性质做出一些评论。

我并不是说人们对游戏行业的意见和批评是不正确的，我只是说也许应该给我一些鼓励，而不是因为我追寻我的梦想而立即贬低我——不管有多少人和他们的祖母有着同样的梦想。

当我在学校学习 web 开发时，几乎每个人都鼓励我——不断地提供技巧和建议，或者推荐学习和分享的资源。有一些鼓励是很好的，但是对于游戏开发，我几乎不想告诉人们我想成为一名游戏开发人员，因为我担心他们只会以大多数人的方式回应。

我最近参加了一个设计会议，一些设计师告诉我放弃，做一些无聊的事情，把游戏作为一种爱好，因为我永远不会成为一名游戏开发者，如果我做了，我会讨厌它。那是我的最后一根稻草。让我们都冷静下来，也许想出一些积极的事情来说，或者也许提供一些可行的步骤来帮助我，就像人们在 web 开发中所做的那样，或者根本不帮助我。感谢您的聆听。
