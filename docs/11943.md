# 展示你的盖茨比网站

> 原文:[https://dev.to/nickytonline/showcase-your-gatsby-site-266](https://dev.to/nickytonline/showcase-your-gatsby-site-266)

你有没有创建一个让你自豪的盖茨比网站？它可能是一个客户或只是你的个人网站。不管怎样，你都可以把它提交给盖茨比展览。

这很容易做到。只需按照这里的说明。简而言之，您在 GitHub 上派生 Gatsby，并将您的站点详细信息添加到他们的 showcase 配置文件中。一旦 PR 被批准并合并，下次他们部署 Gatsby 站点时，您的站点将会出现在列表中！

[![](../Images/0089610f2932095d70406eb6d030bafa.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--k01DIjLI--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/1qyylin8phaa942uauik.png)

做这件事的额外收获是:

*   对你来说是免费宣传
*   这是对开源的贡献
*   成为 GitHub 上 Gatsby 组织的维护者团队的一员
*   盖茨比团队给你免费的公关赠品。谁不想要盖茨比袜子？！

[![](../Images/c47d1deaee7b369c4a3dab51e6891402.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--7OulL3yg--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/dywep9yqgqj2rkiluc9o.png)
