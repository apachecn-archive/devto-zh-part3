> 搬到了[https://pavi2410.me/blog/replisp-another-new-language](https://pavi2410.me/blog/replisp-another-new-language)

你好👋，我做了一个叫‘REPLisp’的编程语言。这是一种类似 Lisp 的解释语言，由我的小手指🖖.用 100%纯 JavaScript 编写我做这个只是为了更多地了解语言以及它们是如何工作的。

在遇到这个项目的小故障后，我开始了这项工作。在经历了那边的一切之后，我觉得**“做一门编程语言是如此容易！！!"**，最终，几乎在一天之内做出了第一个版本*！我逐渐学到了很多关于编译器和语言的基础知识。*
