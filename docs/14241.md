# 创业失败的真实成本

> 原文：<https://dev.to/barretblake/the-true-costs-of-a-startup-failing-2cn9>

本周，我在《财富》杂志的网站上读到了一篇文章，题为《创业公司破产时会发生什么》。这篇文章关注的是创业失败后拿不到钱的供应商。它举了几个例子，说明在湾区食品配送初创公司 Munchery 倒闭后，一些小企业欠了不同数额的钱。虽然这篇文章对供应商在支付方面的损失以及风险投资公司应该如何承担更多责任的看法很有趣，但它甚至没有提到创业公司失败时最受影响的人:创业公司的员工。
