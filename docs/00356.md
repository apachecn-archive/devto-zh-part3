### [](#why-choose-vuejs-framework)为什么选择 Vue.js 框架？

在 2018 年，Vue.js 框架变得非常受欢迎，因为它非常具有适应性和轻量级。

如果你是 Vue.js 的新手，你很快就会发现这个框架的小尺寸实际上是最重要的优势之一。你会立刻下载它，而且这个优点对速度没有影响。事实上，它击败了笨重的 Angular.js 和 React.js。

Vue.js 具有可理解的结构，因此，很容易将其添加到您的项目中。这保证了最终您将拥有一个易于维护的可伸缩的 web 应用程序。

Vue.js 的顶级性能也是这个框架的另一大优势。Vue.js 实现了虚拟 DOM，这意味着快速渲染和最小加载时间。虚拟 DOM 只更新内存中您更改的那部分 DOM。其他框架只是重新渲染一切。

即使这不是您的第一个应用程序，Vue.js 的大量文档也会使您的开发过程更加容易。
