### [](#cool-work-tech-tools)酷派工作技术工具

为您和您的团队节省时间并提高生产力的工具。

**身份验证:**我们自己的现代化、基于云的自助式身份管理。通过易于使用且可定制以满足您的合规性需求的安全单点登录，将您的用户放在第一位。[阅读文章](https://www.authinity.com/)

**机器学习的 AWS 认证:**亚马逊向所有人开放内部的机器学习培训。您的团队可以通过下面的链接获得完整信息。[阅读文章](https://aws.amazon.com/blogs/machine-learning/amazons-own-machine-learning-university-now-available-to-all-developers/)

**Tadabase:** 世界上最简单、无代码的数据库 web 应用构建器。无需编写一行代码，您就可以在几分钟内创建一个强大、专业且潜力十足的数据库 web 应用程序，这将改变您的业务。[阅读文章](https://tadabase.io/?ref=betalist)

**Cube.js:** 无服务器分析框架—适用于您的应用的完整分析平台:前端 SDK 和 API，由完全托管的分析基础架构提供支持。[阅读文章](https://statsbot.co/cubejs/?ref=betalist)

**Sqoosh:** 谷歌 Chrome 实验室发布开源、基于浏览器的图像优化工具。[阅读文章](http://rplg.co/467e0dd0)
