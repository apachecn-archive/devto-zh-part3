# 你“如何”学习？

> 原文:[https://dev.to/pablomarti/how-do-you-study-4lpe](https://dev.to/pablomarti/how-do-you-study-4lpe)

我们都必须学习。一切都在不断变化。我们有工作、责任、家庭，我们都必须寻求健康的生活，所以我们必须睡得好，所有这些都需要时间，当你决定学习时，你会有两个选择:
1 -微笑，有机会成长，不仅仅是作为专业人士，而是作为一个人
2 -生气并做出愚蠢的决定

所以，因为你想要第一个选项，你怎么学习？这个问题引出其他问题:
1 -你如何决定学什么？2 -你如何准备你的学习时间表？(因为你将不得不移动东西，取消其他)
3 -你如何准备你的环境来保护你不被分散注意力？你用什么工具来学习？
5 -你的学习方法是什么？
