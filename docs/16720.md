# Ruby 契约:两全其美

> 原文：<https://dev.to/burdettelamar/ruby-contracts--best-of-both-worlds-2o1h>

在开发人员社区的众多激烈争论中，有一个是关于类型的:静态与动态。

在 Ruby 中，通过 gem `contracts`，我们可以拥有两个世界的精华。

我本打算在这里写这个强大而灵活的 gem，但转念一想，还是链接到这个 gem 的[作者](https://github.com/egonSchiele)的[精彩页面](http://egonschiele.github.io/contracts.ruby/)吧。
