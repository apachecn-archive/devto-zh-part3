# 极简主义和习惯

> 原文:[https://dev.to/ashleemboyer/minimalism-and-habits-224b](https://dev.to/ashleemboyer/minimalism-and-habits-224b)

查尔斯·杜希格的[的*习惯的力量*的](https://www.amazon.com/Power-Habit-What-Life-Business/dp/B007EJSMC8/ref=tmm_aud_swatch_0?_encoding=UTF8&qid=&sr=)是一本非常棒的书，它深入研究了决策和养成习惯的科学。我最喜欢从这本书里学到的是，习惯是我们最终会自动去做的事情，这意味着我们甚至不需要花费太多的精力去考虑去做这些事情。

如果你像我一样，你想得太多了。

[![A little kid leaning on a table and thinking](../Images/0a9bfdbef3ef2b75eb90b7fd96286926.png)T2】](https://i.giphy.com/media/JSgDJLtBZnx9m/giphy.gif)

思考本质上并不是一件坏事。人们倾向于把过度思考和情绪化联系在一起，这没什么。做一个感性的人没有错。接受它；跟着它跑。:)

但是有很多我们倾向于过度思考的事情可以很容易地简化。例如，有些人每天都穿同样的牛仔裤和衬衫，这样他们就不用考虑穿什么了。有些人吃同样的早餐，开同样的通勤车，或者在同样的路上遛狗。我厌倦了超级死板的套路，所以需要练习才能找到有效的方法。

极简主义在创造习惯的过程中起了什么作用？好吧，想想第一个例子:每天穿同样的衣服。这需要最小化你的衣柜，扔掉所有和你的衣服不搭配的衣服。我喜欢将极简主义视为一种生活方式之外的工具。通常，它帮助我为更大的进步做好准备。

把它想成是为做最重要的事情清理精神空间。如果你需要一些帮助来开始，我建议清理你的电子邮件订阅。我已经需要这样做几个星期了，在完成之前，我可能会尽量避免查看我的收件箱。🙈
