# 是什么造就了一个平台，或者说我们如何再造老蓝

> 原文：<https://dev.to/oddevan/what-makes-a-platform-or-how-do-we-recreate-old-blue-166i>

仅仅做点东西是不够的。这是值得的。所以如果我们要这么做，我们就要做好。先说过去。

## 是什么让老蓝这么优秀？

老蓝(因为害怕大红，我不会说出它的名字)是瓶中的闪电。任何网站都不可能希望重现同样的成功。这是在正确的时间正确的部分，真正取代它的将是意想不到的东西。那么什么是正确的部分呢？

### 其实写博客最简单的方法

老蓝消除了很多写博客的摩擦。这些不仅仅是技术上的挑战，尽管它也解决了这些问题。没有要配置的服务器，没有要下载的软件。你选了一个用户名然后嘣！你有博客。

大不了，其他服务(如 WordPress 和 Blogger)就是这么简单。Old Blue 真正擅长的是将内容放到你的博客上。你被允许甚至鼓励发布你找到的内容，而不仅仅是你自己写的内容。“reblog”功能进一步强调了这一点，该功能允许您轻松地将他人博客中的内容转贴到您自己的博客上，为您自己的博客提供内容，同时将其归属于原始发布者。

开博客的问题很容易解决。老 Blue 解决了更难的问题，即如何轻松地将内容放到博客上。

### 仪表板告白

尽管有了“重新博客”按钮，仍然需要找到可以重新博客的博客。为此，老 Blue 借鉴了当时新兴的 Twitter，增加了“关注”其他博客的功能。他们的帖子会以标准格式显示在你的“仪表板”上

虽然这减少了很大一部分定制工作，但它使保持博客比以往任何时候都更容易。不用担心 RSS 阅读器或配置不良的谷歌分析；读者开始阅读，博客也有了持续的受众。

### 2000 年代中期 Geocities 风格的自我表现

纯粹主义者会抱怨大多数旧蓝色博客的单栏布局。他们会谴责这种缺乏响应的行为，同时抱怨店主既没有听说过智能手机，也没有 27 英寸的显示器。一条评论抱怨说，老 Blue 上的网页设计状态类似于 2000 年代中期的 Geocities。我完全同意，但我认为这是积极的。

自我表达一直是社交网络的一部分。它从 Geocities 网站开始，迁移到 MySpace profiles，最终定居在旧的蓝色博客上。所有这些都允许几乎不受限制的风格，让网站所有者从互联网上随机挑选 HTML、CSS 和 JavaScript 片段，并将它们混合在一起，形成一个明确无误的 miasma。老蓝更进一步，允许免费定制域名。如果你不想让老蓝的名字出现在你的公共博客上，你就不需要它。

它看起来丑吗？对某些人来说。它有时会坏吗？是的。但它让人们拥有了自己的博客，让他们觉得自己的空间是真正属于他们的。

### 万事皆有可能

每个人都“知道”老蓝充满了非法/NSFW 材料。老实说，这让很多人很难认真对待这项服务。在专业环境中，服务最不需要的就是与工作相关的东西出现在对工作不安全的东西旁边！对于广告来说更是如此，这个可悲的事实剥夺了这项服务急需的收入。

然而，这种异常宽松的内容政策有一个副作用。内容创建者可以自由发布内容，而不用担心他们的内容会因为模糊的“违反服务条款”而被删除。在 LiveJournal 和 FanFiction.net 等其他在线社区名义上“打击”成人内容之后，这一点尤其重要。这些镇压充其量只是选择性地强制执行，并严重依赖社区报告；最终结果是名义上被禁止但不知何故被更广泛的社区接受或不知道的非法材料能够在那些网站上生存。

作品非法(甚至在其他方面令人反感)的内容创作者可以在 Old Blue 上自由发帖，而不用担心他们的内容会突然消失。这促使更多的人使用这个平台，反过来，这个平台以“更安全”的内容对其他内容创作者更有吸引力。网络效应占据了上风，使老蓝成为一股不可忽视的力量。

### 超特异性超固定。或者不是。

老蓝让注册和开博客变得异常容易。这个博客可以根据你的需要具体或概括。当你需要一个不同的空间时，你可以再开一个博客。另一个。另一个。

内容创建者可以为不同的爱好者、不同的内容安全级别或者仅仅是不同的想法创建不同的博客。这就产生了创造性命名的特定博客，如著名的“effyeah”命名的博客，或特别特定的名称，如“picsthatmakeyougohmm”

## 我们需要什么？

那么，根据这些原则，旧 Blue 的潜在替代品需要什么特性呢？

*   低摩擦签约
*   易于查找和发布内容
*   容易制作多个博客
*   轻松关注有趣的博客
*   开放式主题
*   自定义域选项
*   明确定义的(如果不是许可的)内容政策

其中五个是技术问题。好的编程和好的设计可以让这些功能唱起来。这个问题是最后一个社会问题:内容政策。

唯一在宽松的内容政策下存活下来的大型网站是我们自己的档案馆。它由变革作品办公室运营，这是一个非营利组织，致力于为那些没有家的作品创造一个空间。因此，他们投入了大量资源来确保他们的政策能够经受住法律挑战，他们依靠真正的减税捐款来资助网站，而不是紧张的广告商。任何真正希望填补旧蓝鞋的平台可能都需要采取类似的方法。

另一个选择是 WordPress。精明的网民知道 WordPress 有两个方面:免费网站，任何人都可以注册博客；开源软件，任何人都可以在自己的网络服务器上免费安装。虽然下载和安装 WordPress 不一定适合胆小的人(它需要一些网络服务器的技术知识以及如何维护它们)，但 WordPress 被广泛认为是最容易安装和使用的网络软件之一。

这种部署的便利性使得免费网站 WordPress.com 有了更严格的内容政策，因为任何受到不利影响的人都可以通过一点努力将他们的内容放到自己的博客上。这不仅仅是提供一个博客“备份”；WordPress 有内置机制，可以将内容从一个 WordPress 支持的博客转移到另一个博客，几乎不做任何改动。一个托管在 WordPress.com 上的带有自定义域名的博客可以被改变成一个自托管的 WordPress 博客，访问者几乎看不到任何改变。

虽然 WordPress 的方法没有消除内容政策的社会问题，但它确实降低了风险。如果一群用户发现内容政策很麻烦，他们可以建立(并支付)自己的 WordPress 平台。
