# 技术领导力快速测试

> 原文：<https://dev.to/peysal/technical-leadership-quick-test-1oc>

## 简介

你是程序员。然后有一天，高层管理人员决定将你提升一级。

*   在你的头衔上加上前辈
*   将现有头衔替换为经理
*   将销售线索添加到现有标题

## 资格

所以你一直在想，是什么让程序员或开发人员升级到下一个级别？

*   与其他开发商相比，更多的故事点燃烧？
*   每行代码引入的错误更少？
*   经验年限
*   发生火灾时的焦点？

## 领导潜力快速测试

现在，你如何知道你的团队成员是否有潜力成为领导角色的一部分？只要问任何一个 ***开发者*** 他们是否愿意成为这个人新团队的一部分。
你会惊讶于这个测试有多么有效。如果没有人跟随你，你就成不了领导者。
