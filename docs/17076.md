# 帮忙破被子！

> 原文：<https://dev.to/richienb/help-break-quilt-299o>

我需要你的帮助来尝试打破我的编程库，[被子](https://github.com/Richienb/quilt)以便在它发布之前暴露 bug。

为此，只需执行库中的任何函数，并尝试让它执行一些意想不到的事情。

例如，如果`add(True, True)`导致`2`，那么这是意料之外的。

一旦你发现了什么，简单地[创建一个闪亮的新的错误报告](https://github.com/Richienb/quilt/issues/new?template=bug_report.md)在便利的问题跟踪器中找到[在这里](https://github.com/Richienb/quilt/issues)。
