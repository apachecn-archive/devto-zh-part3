# 耐心是一种美德

> 原文:[https://dev.to/saral/patience-is-a-virtue-a1g](https://dev.to/saral/patience-is-a-virtue-a1g)

在 2018 年之前的几年里，在试图学习如何编码的多次失败尝试之后，我终于在 2018 年取得了突破。我曾试图从各种来源学习，但在遇到棘手的概念和问题后，或者因为我被我的全职工作占据，我就停止了学习。

2018 年，在辞掉全职工作，开始了一次失败的创业后，我准备再次着手编程。这次我选了哈佛著名的 CS-50 课程。在快速浏览了第一周的讲座，并通过 scratch 成功地编写了一个程序之后，这门课进入了 c 语言。我的习题集问题就是从这里开始的。然而，我坚持我的任务，慢慢地但肯定地我取得了进展。

这让我想到了这篇文章的标题“耐心是一种美德”在解决一个特定问题集的过程中，我多次碰壁。我记得在很多习题集中，当我最初看这个问题时，我不知道从哪里开始。我迷路了，满脑子都是“我做不到”或“天哪！这是艰难的”。但是当我坚持解决问题时，有趣的事情发生了，慢慢地问题集开始解开，我的代码慢慢成形。是的，我曾多次参考谷歌的指南。不过最后，当我设法解决了一个问题集时，感觉真好。

这种兴奋只持续了一小会儿，因为另一个问题集随之而来，随之而来的是类似的想法再次浮现。我又一次缓慢而坚定地完成了习题集。

2018 年的课程让我意识到我需要坚持不懈地坚持下去，我将能够解决这个问题。我还从经验中了解到，虽然斗争是巨大的，但伸出援手是关键。现在回想起来，我记得自己在 PSETs 中挣扎了几个星期甚至几个月，却没有得到我应该得到的帮助。我还没有完成我的 CS50 研究，我打算在 1 月完成。

我期待着将这种势头带到 2019 年，并希望很快找到一份工作。

(这是我的第一个帖子。还有，我不知道它是否属于这里。)
