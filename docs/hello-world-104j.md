“你好，世界”是打开你第一篇博文的合适方式，不是吗？

我创建了这个网站来记录和记录我成为一名 web 开发人员的历程。这个博客将会有关于最新技术趋势的反思、经验教训和想法。我也希望有一天，它能激励其他自学成才的 web 开发人员不断成长和学习，无论道路多么艰难。

现在让我绞尽脑汁的一些事情:

*   我是否具备成为一名自学成才的 web 开发人员所需的训练？
*   我对设计感兴趣，有什么方法可以让我适应设计吗？
*   我用 Github 做什么？
*   外面怎么这么多资源！？
*   我需要参加训练营吗？
*   我会搬出我妈妈的房子吗？
*   AngularJS？！Node.js？！这都是什么？！

所以我们开始吧...谢谢你和我一起踏上这个旅程！

羽衣甘蓝。

杰西

以下是 2016 杰斯的回答:

1)我走了半年左右的自学路线。我觉得自己被困在了以前的工作中，想要加快我的职业转型，所以我转向了新兵训练营。

2)作为 UX/UI 开发人员的角色是存在的。作为前端开发人员，对设计有眼光，不负责那方面的工作，是非常有用的。

3) Github 是我的生命线，因为它是我为工作和个人项目推/拉代码的地方。

4)无论是通过视频、阅读等方式，每个人的学习方式都不一样。能够通过不同的媒介解释技术概念也是一项非常有用的技能。因此，对于这个问题，我会说，有需要和需求的五千万不同的教程。

5)你不需要*参加新兵训练营。我当时觉得这很适合我，因为我真的很喜欢在我学习不熟悉的东西时课堂式的氛围。*

6)我仍在努力，但与几年前相比，我现在更有能力做到这一点。这是为了让我的财务状况恢复正常。

7) AngularJS 是一个 JavaScript 框架，用于构建单页面应用。单页应用程序就像网站，有一些细微差别。Node.js 是一个运行时环境，您可以在其中从浏览器之外运行 JavaScript 代码。
