# AWS Lambda Rust 运行时发布

> 原文:[https://dev . to/jeikabu/AWS-lambda-rust-runtime-announced-39lc](https://dev.to/jeikabu/aws-lambda-rust-runtime-announced-39lc)

AWS Lambda 公告中有一个叫做“Lambda 运行时 API”的特性，可以使用不同的编程语言。

这伴随着一个 [Rust 运行时](https://aws.amazon.com/blogs/opensource/rust-runtime-for-aws-lambda/)的立即可用！

有很多“无服务器”、“Wasm”和其他网络技术的时髦词汇在飞来飞去(一如既往)，我的立场通常是*等等看*。但这非常诱人，可能是我一直在等待的推动——我**喜欢 AWS。**
