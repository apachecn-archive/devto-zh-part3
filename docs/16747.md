# 10 开发项目免费(但专业)工具

> [https://dev . to/saulo co/10-免费-但专业-用于发展项目-12i7](https://dev.to/sauloco/10-herramientas-gratuitas-pero-profesionales-para-proyectos-de-desarrollo-12i7)

> 本文标有# translated:t0]用英语阅读

欢迎收看本系列文章中的第一篇**【免费啤酒】**(好吧，西班牙语没有笑话)，我们将在其中分析一系列网上资源，您可以免费使用而不出售您的项目质量。

当然，由于许多服务是免费的，因此将受到某种程度的限制，但您会发现它们完全符合正在开发的项目的需要。

**1 [【谷歌文件】](https://documents.new)**
*主要功能*:创建团队共享文件来起草分析和预算。下载包含生成文档的 PDF，准备提交给客户。
*隐性成本*:5734 谷歌我们所有的秘密计划？

**2[trelo](https://trello.com)**
*主要功能*:以上述文件为基础，分为单子、卡片和*检查箱*，使设备易于操作；及
*隐性成本*:基本服务完全免费。每个仪表板不能添加多于一个集成(“T0”)，每个附件最多可达 10MB。

**[【Heroku】](https://heroku.com)**
*【主要功能】*:将服务器的逻辑和配置(在我的情况下总是使用 nodejs API)托管在 heroku 促销提供的`free dynos`。
*附加功能*:

*   自动部署从开发终端提交的更改。
*   安全且可使用地储存机密应用程式资讯(环境变数、记号、密码)。*隐性成本* : Los `free dynos`只有 512 MB RAM 和 1 个 web/1 个 worker。

**4[【mlab】](https://mlab.com)**
*主要功能*:托管数据库，获取指向我们服务器逻辑的固定 URL。
*隐性成本*:数据库不得超过 500 MB。无法创建自动备份。不能垂直或水平缩放。

**5[【netlify】](https://netlify.com)**
*主要功能*:托管 API 的 web 客户端。获取访问内容的 URL。
*附加功能*:

*   取得指向自己网域的自订 URL。
*   获取指向服务 DNS 的所有域(或子域)的 SSL 证书。*隐藏成本*:单用户设备、安全选项和权限更有限。

**6[【freenom】](https://freenom.com)**
*主要功能*:获得完全免费的上级域。
*【隐性成本】*:它们是声誉低下或难以记忆的领域(虽然实际上取决于创造力和语言)。如果不及时续订，就会产生费用(有时是高级费用)，有时会使域名在一段时间(几个月)内无法检索，在这段时间后，域名将再次免费提供。

**7[【key base】](https://keybase.io)**
*主要功能*:私人 git 存储库和设备聊天。
*附加功能*:

*   身份验证服务。
*   与子组和通道进行组聊天。
*   安全地交换公共文件(公共文件的 url)或专用文件。
*   250 GB 存储容量。
*   公共 git 存储库、组和单个。
*   加密平台上的所有活动。
*   主要平台(web 除外)的客户。*隐性成本*:仍处于概念验证阶段，移动或 PC 客户不断发展，不断更新。

**8[【github】](https://github.com)**
*主要功能*:(我们都知道，但我会解释到底为什么使用)利用与 Netlify 的集成，自动部署我的开发成果。
*隐性成本*:它只允许公共存储库作为免费存储库，因此我只用于客户逻辑，没有秘密。

**9 [句](https://sentry.io)**
*主要功能*:生产过程中的误差监测。
*隐性成本*:仅用于个人项目和初期阶段。

**10 [VS 代码](https://code.visualstudio.com/)**
*主要功能*:编辑代码。
*附加功能*:

*   与 git 的本机集成，
*   多个扩展
*   与其他团队成员实时协作到调试实例。

*隐性成本*:5734 对微软我们所有的秘密计划？
