# 你是如何持续学习的？

> 原文：<https://dev.to/joshhadik/how-do-you-continuously-learn-1969>

在这个世界上，用户不断改变他们与技术的交互方式，突破性技术每隔几个月就会出现，新的 JavaScript 框架几乎每天都会出现，作为开发人员，我们能做的最好的事情之一就是不断学习。

无论这意味着学习一种全新的语言，如何遵循一种不同的编码模式，如何使用最新的 DevOps 工具，还是其他完全不同的东西，重要的是我们要找到将持续学习融入我们生活的方法。
