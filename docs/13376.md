# 使用 LambdaTest 屏幕截图 Chrome 扩展轻松进行跨浏览器测试

> 原文:[https://dev . to/lambdatest/easy-cross-browser-testing-with-lambdatest-screens-chrome-extension-43o 6](https://dev.to/lambdatest/easy-cross-browser-testing-with-lambdatest-screenshots-chrome-extension-43o6)

谷歌 chrome 商店于 2010 年推出，拥有超过 50，000 个 chrome 扩展。chrome 扩展流行和使用增加的主要原因之一是扩展的易用性。只需简单的安装，您现在只需点击一下就可以使用复杂的功能。

为了让网络应用和网站的浏览器兼容性测试更加简单快捷，我们决定向 Chrome marketplace 推出 LambdaTest 跨浏览器测试截图插件。

有了 LambdaTest Chrome 扩展，你只需选择浏览器组合，就能轻松生成你已经打开的页面截图。这就是你可以称之为**的跨浏览器测试变得超级简单！**

您只需选择浏览器组合(最大。一次最多 25 个)，只需点击捕获，你就可以在各种浏览器上找到截图，甚至不用进入 LambdaTest 平台。

## LambdaTest 截图 Chrome 扩展如何工作？

要使用 LambdaTest 截图 Chrome 扩展，您需要拥有一个 LambdaTest 帐户。创建帐户后(如果你已经有了一个帐户，步骤已经很少了)，只需进入 [chrome 网上商店](https://chrome.google.com/webstore/)并搜索 LambdaTest 截图，或者你可以使用 [chrome 扩展的直接链接:LambdaTest 截图](https://chrome.google.com/webstore/detail/lambdatest-screenshots/fjcjehbiabkhkdbpkenkhaahhopildlh?hl=en)

[![](../Images/3e7cdf1481f3c06cd3385d589d827aea.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--Rp-9YExQ--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://www.lambdatest.com/blog/wp-content/uploads/2018/08/Capture-3.png)

就像你对任何扩展所做的那样“添加到 Chrome”。你已经准备好使用它了。你会在 chrome 浏览器的右上角找到它。

[![](../Images/bc9229a846ec00cfdaca0d815ec506ba.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--dIYFlgE2--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://www.lambdatest.com/blog/wp-content/uploads/2018/08/r.png)

只需转到您要截屏的页面，点击 LambdaTest 扩展图标。输入您创建 LambdaTest 帐户时使用的电子邮件地址，并输入您的访问密钥。

## [](#where-can-i-find-my-access-key)在哪里可以找到我的访问密钥？

在你的 LambdaTest 帐户的设置部分下的个人资料部分，你会在那里找到你的访问密钥。

## [](#alternative-to-avoid-entering-of-access-key)替代避免进入访问键。

另一种避免这个登录过程的方法是在安装了 LambdaTest chrome 扩展后进入 LambdaTest dashboard 并重新加载页面。这样，您的访问密钥将自动配置到分机。

完成后，你将被重定向到截图页面:

[![](../Images/9e8797379a8b0df3cb47be3fc43a1d3b.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--2X9y-W01--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://www.lambdatest.com/blog/wp-content/uploads/2018/08/xyz-1.png) 

你甚至不需要输入你想要截图的网址，它会自动获取你打开扩展的网址(我们不限制你，你也可以手动输入！).只需选择配置(你可以从广泛的 Chrome、Firefox、IE、Edge、Safari 浏览器中选择)并点击 Capture。

你在页面上有截图！下载他们作为个人或所有在一个压缩文件，找到跨浏览器的问题，并采取更多的截图。

简单导航，用户友好，易于使用的 LambdaTest Chrome 扩展是你必须拥有的，如果你关心你的网站的跨浏览器兼容性测试。

注意:如果你是一个免费增值计划，你可以使用 LambdaTest 截图 Chrome extension 进行 10 次测试，但如果你喜欢它并想使用更多，请切换到我们的任何付费计划。
