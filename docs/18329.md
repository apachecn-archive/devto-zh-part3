# 学会在一个小时内构建分散式应用程序:介绍“零到 Dapp”

> 原文:[https://dev . to/block stack/learn-to-build-decentralized-applications-in-hour-introducing-zero-to-dapp-2ec 3](https://dev.to/blockstack/learn-to-build-decentralized-applications-in-an-hour-introducing-zero-to-dapp-2ec3)

对于开发人员来说，学习构建一个去中心化的区块链应用程序并不一定很困难或者很耗时。使用他们已经知道的语言和框架，Blockstack 可以轻松构建保护隐私、本质上更安全、提供数据可移植性的去中心化应用程序。

为了让入门更加容易，我们推出了[‘零到 Dapp’教程。](https://docs.blockstack.org/develop/zero_to_dapp_1.html)本教程是一个循序渐进的序列，旨在帮助开发人员快速了解 Blockstack 平台、分散式架构的优势，并创建一个可工作的分散式应用程序，所有这一切只需大约一个小时。

通过零到 Dapp 教程，开发人员将创建一个名为“动物王国”的多页 web 应用程序。在构建应用程序的过程中，没有以前的区块链或分散应用程序经验的开发人员将:

*   了解分散式应用程序(Dapps)与传统的 web 或移动应用程序有何不同。
*   探索 Blockstack 生态系统及其强大的平台功能。
*   通过应用挖掘发现潜在的可用资源。
*   使用标准前端框架构建多页面 web dapp。
*   部署和发布应用程序

除了学习一种新的方法和平台，完成零到 Dapp 教程并提交给[App.co](https://app.co)的开发者可以获得一件免费的限量版 t 恤(如下图)。

当你开始构建时，我们欢迎你加入 Blockstack 社区，你可以通过 [Slack](http://chat.blockstack.org/) 和[论坛](https://forum.blockstack.org/)与成千上万的其他开发者和 Blockstack PBC 团队联系。
