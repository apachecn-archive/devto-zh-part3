# 一篇博文能有什么好处？

> 原文:[https://dev . to/Maxwell _ dev/what-good-can-a-blog-post-do-220m](https://dev.to/maxwell_dev/what-good-can-a-blog-post-do-220m)

*[这篇文章最初发布在我的个人网站](https://www.maxwellantonucci.com/2019/01/19/what-good-blog-post-do.html)上，但认为主题适合 Dev。好到我把它交叉贴在这里。*

这篇博文和我写过的许多其他博文一样，都是从一种不安全感开始的。

在这种情况下，是当我看到一个同事的推文。调查显示，他们是一个社区活动组织的领导者，该组织在交通和教育等重要领域促进代表性不足的群体之间的平等。很明显，我为他们感到高兴，无论是他们如何发挥作用，还是这项重要的工作已经完成。但我不能否认其中夹杂着一些嫉妒，并不是因为他们也更善于微笑和打扮。我试着告诉自己，我早就忘记了。

不，我的想法是我是一名程序员，我的工作几乎完全局限于数字领域。有个同事为这个城市做了更多实实在在的好事。看到别人为了别人的生活做了多少，总是让我质疑自己是怎么度过时间的。也许我应该更专注于直接帮助他人，但我却如此专注于培养自己的编程技能。

最重要的是，这让我想知道，如果我的生活主要是写作和编码，我能有什么不同。如果只有为别人而活的生命才值得活下去，那么像程序这样的东西到底值多少钱？一篇博文能给世界带来什么好处？在网络末日之前，我们不能依靠博客文章来建造经济适用房，到那时，谁知道我会在哪里，我会剩下多少条腿！

老实说，我不知道我是否有一个真正的答案来解决我的这个担忧。我可能永远也不会。但我发现了一些小技巧，当这个问题不可避免地再次出现时，可以让事情变得更简单。希望他们也能帮助其他有同样感受的人。

### [](#savor-the-ripples-effects)细细品味涟漪效果

我很容易觉得我需要修复一切。当我在我的城市看到无家可归的人讨钱或翻垃圾时，就会发生这种情况。很容易告诉自己，因为我没有做什么给这里的每个人一个家，我在浪费时间摆弄 JavaScript，然后因为 JavaScript 而踢东西。如果我不能解决所有问题，我又有什么价值呢？

这种急于判断自己的心理很容易忽略一些事实:

*   我个人并不对我周围所有可怕的事情负责，所以我不需要把解决这些事情的全部负担都放在自己身上。
*   我不应该因为花时间关注自己或事业而感到内疚。没有其他人会为我而活，所以有时候我需要自私一点才能前进。

也许是最重要的一点:

*   无论大小，积极的改变都是值得的。

我不需要为我的城市里所有的无家可归者问题解决问题。人们的组织多年来一直试图解决这个问题，但问题仍然存在。假设我会永远修复它是不现实的。但是我可以把这样的问题变得更好，哪怕只是一点点。这不仅仅是什么都没有，而且对它所帮助的人来说是很多的。

对我来说，有一种形式是有多少人阅读我写的东西。我不能指望每个人都阅读我关于可访问性或博客习惯的帖子，然后每个人都接受那个建议并奇迹般地解决那些问题。但是他们仍然可以在很大程度上帮助少数人。它对他们的影响会影响到他们认识的人，并由此产生连锁反应。即使是帮助一个我尊敬的程序员同事这样的小事:

没有一个人可以独自解决一个大问题，无论是网络无障碍还是无家可归。这使得小的改进变得更加重要。

### [](#be-pragmatically-empathetic)讲求实效，感同身受

像过去几年中的许多美国人一样，我通过退缩到愤世嫉俗的外壳中来应对无情的消极气候。用漠不关心的零食更容易逃避坏消息。我不会为此责怪任何人——这通常是出于自我保护，而不是想成为一个混蛋。不管怎样，人们有很多其他的理由成为混蛋。

但是我现在才意识到这种犬儒主义有一个关键的缺点:我犬儒主义的一部分是认为改变是不可能的，也是没有意义的。总而言之，它扼杀了我改变或改善事物的所有动力。正如你所料，“想做更多的好事”和“不想改变任何事情”并不完全吻合。

> 表现得好像你的所作所为有所不同，的确如此。
> *~威廉·詹姆士*

因此，如果我不是因为个人或道德原因而更感同身受，至少有一个务实的原因。同情和怜悯处境更差的人是痛苦的，但这种痛苦是我做一些事情的额外动力。这种推动源于试图感受那些比我更糟糕的人的痛苦，所以做一些事情有助于减轻我和他们的痛苦。

今天要有同情心并不容易，但这让这种感觉变得更加有价值和重要。仅仅是感觉到这一点，以及它可能带来的所有痛苦，就让我知道我正走在为他人而活的正确道路上。

### [](#channel-your-own-strengths)发挥自己的优势

还有一个很明显的原因，我没有创建一个帮助他人的当地非营利组织:我可能不太擅长这个。

我以前写过关于接受我永远不能成为我最崇拜的人，就像一个虚构的律师一样。我也可以接受有很多我不擅长的事情:玩有组织的运动，在 Splatoon 2 中使用充电器武器，或者在试图偷偷将东西倒入海洋时删除书面痕迹。我从所有这些失败中吸取的最大教训是:

*   储物车库是储存不需要的东西的好地方，只要它们是室温的，并且装满了足够的海绵和橡皮泥...安全原因。
*   我应该接受自己，利用自己的优势。我可能达不到我的理想，但我会想办法做点什么。

第二个教训在这里几乎完全适用。我可能不具备在非盈利领域做好自己力所能及的所有事情的技能，因为我的商业管理和人际关系充其量只能算是平庸。但是我可以找到方法使用我的(假定的)写作、编程和文本编辑器快捷键来帮助别人。这也是我更加关注网页可访问性的部分原因，因为这一切都是为了在我热爱的领域里帮助他人。

尽我所能很重要。尤其是因为总有人做不到这一点，可能比我想象的还要多。

### [](#be-a-positive-presence)成为积极的存在

当其他一切都失败的时候，有时候我觉得在别人身边做一个积极的存在就足够了。我们不能总是朝着某个更大的目标努力，或者有时根本没有任何目标。因此，找到任何让我们周围的人感到更快乐的方法可能就足够了。这可以采取多种形式，例如:

*   搞笑。几乎所有人都喜欢笑话！如果他们没有，拿起你的射钉枪，去更高的地方。
*   重视他人的工作和技能。
*   称赞你注意到的某人的一些事情。
*   将知识传递给那些需要的人，这是编码人员试图回馈的常见情况。
*   如果他们看起来很沮丧，问你是否能帮上忙。即使只是听听他们的想法。
*   黑进他们的电脑，看看谁在困扰他们，邮件说人们不断闪光炸弹。
*   写一篇让人们振奋或鼓舞的博文，哪怕只是一小会儿。

有时这是我所能做的一切，但这永远是我能为帮助他人所做的最起码的事情。无私生活的全部意义在于让他人感觉更好，在我们与他人的日常互动中，我们都有很小的机会这样做。

#### [](#so-what-can-a-blog-post-do)那么一篇博文能做什么呢？

所以我再一次问自己:作为一名编码员和作家，像博客这样的东西能为这个世界带来什么好处？

很明显，我需要接受我永远做不到像那些为社区奉献一生帮助无家可归者的人那样好。但这并不意味着我什么都不能做。我仍然可以将我的力量投入到其他帮助他人的方式中，这些方式虽然规模较小，但仍然是有益的。我需要更多地享受这些小小的胜利，以及它们所带来的连锁反应。

我可以让网络应用程序更容易访问。我可以写帖子来激励，安慰，甚至逗别人开心。当人们看到我在身边时，我可以让他们感到更快乐或更放松(仍在努力)。我可以释放第五个脉轮的力量，摧毁整个仓库，但出于保险起见，我选择不这么做。我想说的是，一个人总会做些事情来回报社会，不管是大事还是小事。只要他们有爱心并愿意尝试，我相信他们能找到方法。

也许我永远不会做得像这位同事一样好。我仍然可以做尽可能多的好事。因为一旦我相信我能有所作为，我就会开始去做。
