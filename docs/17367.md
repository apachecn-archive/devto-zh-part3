# 回顾过去，展望未来

> 原文:[https://dev . to/Angela Riggs/reflecting-back-and-looking-forward-107d](https://dev.to/angelariggs/reflecting-back-and-looking-forward-107d)

我通常不写年终文章。其实我觉得这是我第一次。我也不打算写这篇文章——主要是因为我觉得自己没有真正完成任何事情。但当我回顾我的 2018 年时，我意识到我实际上已经完成了不少令我自豪的事情。我 2019 年的目标之一是更有目的性——也就是说，为我的前进道路做更明确的规划，我知道 2018 年有一些事情可以用来告知和激励我 2019 年的方法和目标。所以我的第一步是写这篇文章，思考我做了什么或没做什么，以及我如何利用过去一年的经验在新的一年取得成功。

## [](#2018-accomplishments)2018 年成就

总的来说，2018 年对我工作之外的职业成长来说是非常好的一年，我很高兴也很自豪地回顾过去。我感觉我越来越习惯于承认我的技术专长，并开始在软件测试社区中寻找我的声音和更多的参与。

✏️我在 2018 年写了很多！我写了我作为质量保证工程师的角色；了解[变革管理](https://dev.to/angelariggs/how-static-code-analysis-taught-me-about-change-management-4icg)；思考我喜欢成为什么样的[领导者](https://dev.to/angelariggs/how-to-be-a-rebel-at-work-5ffk)。除了为我自己的博客写作，李佳薇还主动要求我在《实用开发》上交叉发表文章，这太令人兴奋了！8 月，当 TechBeacon 让我为他们写一篇[文章](https://techbeacon.com/3-hats-qa-engineers-how-wear-them-well)时，我成为了一名*付费作者*，这让我有些震惊。一个月后，我为 TestCraft 写了一篇关于在开发者和 QA 工程师之间建立更好关系的文章。

👯2018 年初，我参加了一个为 PDX 女性编码的[职业&技术小组](https://www.meetup.com/Women-Who-Code-Portland/events/245945636/)。这很有趣(我和一些我喜欢的女性和同事分享了聚光灯)，但这也是我第一次和软件和技术领域的人谈论我作为 QA 工程师的工作。我记得我对参加小组讨论和回答问题普遍感到紧张，但我对能够回答好这些问题充满信心——没有骗子综合症！

📢2018 年是我第一次做大会演讲的一年！我谈到了在 PNSQC 的自动化可视化回归测试，以及我如何在 TestBash USA 帮助 T2 创造质量文化。我还在会议上遇到了一些非常棒的人，这种感觉真的很好。

💰除了写作和口语，我在 2018 年找到了一份新工作——这是我作为 QA 工程师的第二份工作，并且加薪 40%。在某种程度上，这是一个很好的保证，即作为一名质量保证工程师的成功不是侥幸，而且记住我的技能以及我如何实现这些技能是有需求和有用的，这也是令人满意的。

## [](#2019-goals)2019 年目标

我了解到我真的很喜欢公开演讲——我喜欢与他人分享我的经验和知识，并在测试社区中拥有发言权。我还了解到，在上台前一个小时左右，我会有一次焦虑发作，但一旦我站起来开始说话，我就会感觉很好。2018 年做了两场，所以我 2019 年的目标是三场！我将在六月的 refact 上演讲，我也在申请明年的其他几个职位。只是为了把它放到宇宙中，我特别为[剧组](https://www.associationforsoftwaretesting.org/conference/cast2019/cast2019-call-for-proposals/)和敏捷测试日([欧洲](https://agiletestingdays.com/)或[美国](https://agiletestingdays.us/))祈祷，我很乐意再次在 PNSQC 和 TestBash 上发言！

我认识到我喜欢领导，即使我的正式角色不是领导角色。我也认识到，在我支持和提高的热情中，我可能会推得太多或走得太快。即使我知道我建议的改变或想法更好，我也需要等待其他人赶上来，自己决定。我想在如何领导他人方面更加深思熟虑，在花时间分享我的想法和倾听他们的想法方面更加耐心。

我了解到社交媒体中有一些很棒的社区，尤其是 Twitter 和 Slack。在 2019 年，我想成为那些社区中更积极的参与者——更少的被动阅读，对正在发生的对话做出更积极和正面的贡献。我受到网上认识的人的启发，我想更多地参与到学习和支持中去。

我明白了，我喜欢学习，当我感觉自己一遍又一遍地做着同样的工作或解决同样的挑战时，我会感到无聊和没有动力。我想在 2019 年拓宽我的测试技能——安全测试和性能测试是我特别感兴趣的两个领域。我还想更好地理解和创造有效的 CI/CD 实践。

我在 2018 年专注于我的职业生涯，显然我在 2019 年仍然有很多职业目标！但我也想思考一下新一年的个人目标。我想学习，探索，成长。2019 年想做的一些事:

*   每月烘焙一到两次新的面包
*   每月参加一门 DIY 或艺术课程(我已经开始关注织布机编织课程了！)
*   制定并保持锻炼计划
*   参与一个常规的志愿者机会
*   回到俄勒冈徒步旅行和探险
*   读更多的书，听更多的播客

在高层次上，所有这些都意味着我想在新的一年里为自己抽出更多的时间。我将永远是一个进步中的作品，我想弄清楚 2019 年我想成为谁，如何成为谁，并在成为最佳版本的我上取得进展。

*你 2019 年的目标是什么？2018 年你学到了哪些东西？你有没有发现激励自己每天锻炼的秘密，或者发现了一个惊人的面包食谱？请留下您的评论——我很乐意收到您的来信！*
