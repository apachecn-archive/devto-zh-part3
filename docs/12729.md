# 寻求建议 pet 项目需要 React 组件库

> 原文:[https://dev.to/ntsdk/look-for-advice-需求-反应-组件-宠物项目库-1dkk](https://dev.to/ntsdk/look-for-advice---need-react-component-library-for-pet-project-1dkk)

你好！我有一个喜欢的项目，我计划在 React 中重写它。
关于项目的几个词:

*   它在浏览器中工作

*   它假设了大量的文本编辑，视觉编辑，情节和模式，所以它是桌面优先。

现在我在 UI 上使用定制的 Bootstrap 3。我减少了边距和填充以获得更密集的环境。

对我来说有价值的是:

*   这是假设用户将在我的程序工作几个小时。留住用户的眼光很重要。所以 UI lib 应该有颜色主题。

*   组件尺寸较小的主题。

*   基于类的定制的可能性，因为我使用几个第三方可视化工具，它应该集成在一起。

你可以通过回答问题来帮助我:

*   哪些组件库比较合适？

*   哪些组件库不适合？

*   什么颜色或颜色主题适合长期使用？
