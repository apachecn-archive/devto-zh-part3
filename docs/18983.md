# 免费区块链应用思路

> 原文：<https://dev.to/blockstack/free-blockchain-application-ideas-39ep>

基于区块链的应用程序可以为用户提供安全、可靠、更强大(甚至可以抵御审查)的体验，但这还处于早期阶段，还没有足够多的优秀应用程序被开发出来。

基本上，您可以使用您自己控制的身份登录他们，在您的设备上本地运行应用程序，将您的数据保存到您自己的私有云，并与也在做同样事情的朋友联系……而无需依赖中间人或集中式数据孤岛。这允许完全的自由和以用户为中心，这意味着如果我不喜欢一个应用程序，我可以通过随身携带我的数据流畅地转移到另一个应用程序。这也意味着一些应用看起来不太像传统的“应用”,因为使用这种基础设施的人可以以他们想要的任何方式重组自己。

为了帮助激励应用程序创建者构建高质量应用程序的未来，[我们最近推出了应用程序挖掘](https://blockstack.org/blog/introducing-app-mining/)。这意味着每个月都是一个新的机会，可以在注册应用程序中赚取目前每月 10 万美元的一部分。如果你一直在思考这个问题，或者有一些额外的时间，并且想要为生态系统做出贡献，这里有一些我们认为有用并且可能成功的应用想法:

*   分散的 Localbitcoins.com
*   类似 Earn.com 的分散式电子邮件客户端
*   投币式文件浏览器
*   分散的 Patreon 替代方案(信息不是免费的！:-))
*   向播客付费的播客
*   向艺术家付费的 Spotify
*   特定社区的约会应用
*   为您的 1.5、5、15、50、150 位朋友提供“邓巴”应用
*   你保存朋友照片的社交媒体
*   抑郁症患者的社交媒体(把他们从不自然的不快乐带到自然的不快乐！；-P)
*   “Blockstack experience”应用程序(关于去中心化的教育应用程序，带您了解使用 Blockstack ID 登录的方式和原因，显示您在私有 gaia cloud 中的文件等。)
*   应用程序将社区与宣传活动联系起来，使 facebook 的算法公开化。登录以查看进度，在其中保存文件和通信。
*   真人发生器:KYC 在某种程度上解决了西比尔机器人的攻击，但保留隐私。
*   Blockstack 堆栈令牌钱包应用
*   面向以隐私为中心的区块链应用的 Mixpanel
*   一个能让你联系到一件高度敏感的事情的应用程序
*   可审计的、自愿的坑洞填充
*   个人治疗应用程序，你可以记录你的私人会议
*   滑板车！使用我的 Blockstack ID 登录滑板车，并在您的 Gaia 私有云中拥有您的骑行历史。
*   这可以让许多交通网络以新的方式为你服务。
*   没有集中式数据仓库的 Github
*   没有集中式数据孤岛的松弛
*   一款上传 facebook 数据并进行再利用的应用。我不想只是下载我的 facebook 数据并删除 facebook，我想将其回收到一个新的应用程序中！推特也一样。
