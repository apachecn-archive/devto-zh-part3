# 如何成功改造网站？

> 原文：<https://dev.to/innoraft12/how-to-revamp-website-successfully-3nlb>

他们说“第一印象是最后印象”并没有错。一个普通用户会在最初的 8 秒钟内决定是否要进一步浏览网站。

按照以下步骤成功改造网站 2019:

1.  总是从“为什么”开始
2.  内容永远是王道，但展示才是女王
3.  将线框放置到位。
4.  在上线前把每件事都做好？
5.  一直在不断改进 UI！
