> 原文：<https://dev.to/grandemayta/googlebot-now-supports-latest-javascript-features-43e0>

谷歌在 Google IO 2019 上宣布，Googlebot 现在支持最新的 Javascript 功能。对于前端开发者来说，这是一个重要的改进。
