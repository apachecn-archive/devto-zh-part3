# 为开放源码软件做贡献

> 原文：<https://dev.to/mgh87/contributing-to-oss-44an>

嘿外面的伙计们！如果你像我一样，你会花大部分时间编写一些操作系统代码。你有没有想过为那些不错的框架、服务和其他很酷的东西做贡献？嗯，我做到了，但我得告诉你，我真的没有取得很大成就。

我在 GitHub 上做了一些较小的拉取请求，或者试图帮助解决调查过程中的一个问题。但仅此而已。

### 你以什么方式为 OSS 做贡献

我想知道你们是如何解决这个问题的，或者你们是否解决了这个问题？你有什么诀窍来选择一个值得你花时间的好项目吗？

### 如何改进我的方式对 OSS 世界产生更大的影响

所以，在未来，我想每周牺牲一个小时在 OSS 上工作，并尝试在 dev.to 上报告我的进展，以便有额外的动力来拖延这个过程。这大约是我的时间的 0.5%，应该是可行的。
