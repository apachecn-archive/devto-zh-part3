# 第 0 天:计划阶段

> 原文：<https://dev.to/iamjoross/day-0-planning-phase-4jkn>

### *2019——新的一年，新的目标。*

今天，我全身心地投入到#301DaysofCode 中去，成为一个足够有能力、更棒的软件开发者。不要再紧张，不要再懈怠，绝对不要再每天睡超过 10 个小时。

满怀热情，今年将是完全不同的一年，因为我已经设定了我想在 2020 年到来之前实现的目标。作为一名 4 年多的开发人员，我感觉自己就像一个没精打采地等待事情发生的怪杰。但是没有！没人有时间做那个！这次我会更加积极主动，一定会让事情发生。

从进入这个行业开始，我的最终目标就是被硅谷的公司雇佣*(你知道我的意思)*。这将是一项艰巨的任务，但当你从 0 到 N-1 开始时，没有什么是困难的。也就是说，我将从事开源项目，并开始制作个人项目。

这是我的# 301 日代码的第 0 天。在我头脑风暴的过程中，我会记录下我走的每一步，我面临的每一个障碍和我解决的每一个问题。这真的很令人兴奋，我非常渴望开始。我希望你和我在一起，成为我美好新旅程的一部分。你的支持真的会给我很大帮助。
