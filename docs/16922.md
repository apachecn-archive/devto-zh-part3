# 问题日记快捷方式

> 原文：<https://dev.to/_bigblind/a-question-diary-shortcut-42p0>

本周早些时候，我写了一篇关于我今年主题的文章。我想在我的生活中加入更多的*结构*，以系统的形式，帮助我改善那些我过于健忘、冲动或其他从长远来看不会让我快乐或健康的行为方式。

我想建立的基本系统是我自己的一个反馈回路。每天，我都想问自己同样的问题，看看我的答案如何演变。我试图找到一个应用程序来做这件事，但我找不到一个适合我的。

# 快捷方式

我创建了一个 IOS 快捷方式。它在我的 dropbox 中寻找文件`/diary/questions.txt`，并期望每行都有一个问题。它依次问我每个问题，并让我口述一个答案。这些回复会被保存到 dropbox 的日记文件夹中，并以当前日期命名。

你可以在这里下载快捷方式。
