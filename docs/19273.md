# 今天是世界艾滋病日

> 原文:[https://dev.to/gastongarcia/today-is-world-aids-day-3d9i](https://dev.to/gastongarcia/today-is-world-aids-day-3d9i)

今天，12 月 1 日，是世界艾滋病日。今年，联合国一直鼓励每个人了解自己的艾滋病毒状况。了解你的艾滋病毒状况不仅对你自己非常重要，对你的性伴侣也非常重要。

抗逆转录病毒疗法现在很神奇，艾滋病毒阳性的人可以过健康的生活，甚至停止传播病毒。但是*了解你的状态*是至关重要的——这是非常重要的第一步。

所以把话传出去。鼓起勇气，接受考验。
