# WTF 是 Git 别名？

*   一个简化 Git 函数的简写命令。

# 怎么做？

### 如何制作 Git 别名以及如何使用

*   通过键入> git config - global alias 来配置您的全局 Git 设置。(你想要的速记命令)(实际命令)

示例:git config - global alias.ci 提交

或者在您的。gitconfig 文件

> [别名]
> ci =提交
