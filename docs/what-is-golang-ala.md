# GoLang 是什么？

> 原文：<https://dev.to/petercour/what-is-golang-ala>

Golang 是 Google 创造的一种编程语言。它使用并发性作为第一类结构。最重要的是 golang 很简单:Golang 很简单，你可以在一个周末学会它！

为什么要学 golang？

*   围棋很容易学
*   golang 有一个易于使用的界面
*   非常适合 web 开发，甚至前端(GopherJS)
*   用于解决实际问题的实际语言。
*   Golang(欧盟和美国)有大量工作要做
*   目前许多顶尖的应用程序都是用 Go 编写的
