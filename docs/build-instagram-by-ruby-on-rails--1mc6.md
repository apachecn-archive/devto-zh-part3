# 用 Ruby on Rails 构建 Instagram

# 完成这个项目后，你会学到什么？

*   如何启动一个新的 Rails 应用程序？
*   来自工艺的设计系统
*   理解 MVC(模型—视图—控制器)架构
*   模型:主动记录迁移、验证、回调、关联和查询接口
*   视图:布局、局部和表单帮助器
*   控制器:动作，强参数
*   轨道布线
*   上传文件的活动存储
*   在 Rails 应用程序中使用 Bootstrap、Devise、Kaminari gem
