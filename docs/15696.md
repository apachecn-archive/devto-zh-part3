# 如何为每个人设计更有用、更易访问的网站

> 原文：<https://dev.to/alkatherifaizah/how-to-design-more-usable-and-accessible-web-for-everyone-20f>

数字技术迅速渗透到我们生活的方方面面。到 2020 年，世界上将有多达 200 万亿台联网数字设备。因此，用户在与特定设计交互时应该拥有的体验质量变得非常重要，这被称为用户体验(UX)。

我们(UX 设计师)肩负着重大责任，要制定更全面、更恰当的职业道德标准:我们的设计应该走多远？我们应该建造什么，不应该建造什么？关键是要检查我们正在构建的目标，看看我们是否与用户的目标一致。

以人为本的设计带来包容性
用户是一个非常普通的标签，我们大多数人容易忘记的是，所有用户都是以人为本的。人类对于如何认识和处理周围的世界有一个隐藏的蓝图。作为设计师，我们可以使用心理学的一些关键原则作为指导，来设计人们实际上是怎样的，而不是强迫用户遵从产品或体验的设计。人类已经进化到在自然环境中成长。因此，每个设计师都应该学习心理学的基础知识——是的，我们可以建立更具体的、以人为中心的产品和体验。

通过“以人为本”的设计，我们寻求在设计中优化人类健康。如果数字应用不符合自然限制，我们就不能指望人类能够直观地与它互动。这是一个挑战:除非我们知道用户、环境和目标，否则不可能确定某样东西是否可访问。

通用设计就是创造一个适合每个人的通用设计。然而，我们正在处理数字设计，我们的设计考虑非常不同于非数字。我们有自由，我们可以创造一个设计系统，它应该适应、适应或延伸，以解决每个人提出的每个设计需求。

网络用户的不同心理层面
可访问性和包容性意味着设计时要包容尽可能多的用户，考虑用户多样性的各个方面。将易访问性标准和可用性过程与真实的人结合起来，可以确保一个设计在技术上和功能上都可以被不同心理状态的人使用。随着对这一主题的理解、同情和讨论的增加，包容性设计考虑了人类在能力、语言、文化、性别、年龄和其他形式的差异方面的全方位多样性。

即使设计者或作者认为动态是有益的，它也会分散那些注意力不集中的用户的注意力，让他们忘记他们来你的网站是为了做什么。大多数情况下，访问者浏览页面，浏览一些文本，然后点击第一个引起他们注意或看起来像他们正在寻找的链接。糟糕的设计降低了清晰度，增加了误差；有些互动需要很长时间，以至于对注意力不集中的人来说变得极其困难。运动分散了他们的注意力，即使神经典型的人，或者那些没有神经模式特征的人很好。

焦虑:没有人会不知道自己为什么会去你的网站。如果用户去你的网站解决问题，他们需要知道在哪里可以找到解决方案。这个问题可能是所有用户都有的，但是当有焦虑的用户找不到他们需要的答案时，当前进的方向不明确时，他们会更加挣扎。在网站难以触及的角落隐藏关键信息，如联系方式或帐户停用说明，会导致用户感到无助。

如果我们向用户询问任何个人信息，比如电话号码，隐私是一个很大的问题，尤其是对于那些患有社交焦虑、害怕接到意外电话的用户来说。如果表单中需要一条信息的原因不是很明显，可以添加一些帮助文本来解释它。但是如果你没有一个很好的理由或者不能清楚地解释你为什么需要它，那么就摆脱这个领域。与网站和应用程序的不可预测性密切相关的是它们可能引发的压倒性无力感。每个表单和动作都应该有一个清晰的标题，清楚地说明表单的作用。

当与新事物互动时，人类会利用他们已经从过去的经历中获得的知识。无论是一个网站还是一辆汽车，他们都形成了一个系统如何工作的模型，然后他们将这个心理模型应用到系统相似的新情况中。设计师花了很多时间考虑使用他们网站的人，但是有一点“人物角色”几乎总是有共同点的——他们都很开心。让我告诉你，如果你的网站或应用程序很难使用，许多抑郁的用户就不会使用它。

假设用户非常喜欢你的内容，以至于他们会阅读和浏览它的每一部分，这是天真的，会给沮丧的用户造成很大的障碍。大量无止境的内容，就像墙到墙的文件，迫使用户筛选它来找到他们正在寻找的东西。如果内容需要很大的努力才能找到，沮丧的用户会忽略它。与“文本墙”相比，使用适当的标题处理、行长和内容长度来解释和记忆格式良好的内容(分块)要容易得多。

总之，让你的网站具有可访问性就是确保没有障碍阻止人们访问内容。例如，在界面上做更少的决定，这样你的用户可以为它做更多的决定。易访问性实践者和研究人员可以结合可用性技术来提高“可用的易访问性”。将可访问性、可用性和包容性结合在一起，可以更有效地为每个人带来一个更易访问、更易使用和更具包容性的网站。
