# 博客文章:电脑时间消耗——兔子洞之旅

> 原文：<https://dev.to/jonasbn/blog-post-computer-time-consumption---a-tour-down-a-rabbit-hole-5bmg>

计算机作为一种爱好/工作是非常耗时的。

总有一些事情需要调整、尝试、评估和去做。因此，大多数计算机程序员都是生产力上瘾者——我们需要把事情做完或扔掉，因为我们需要/想要/必须做的事情太多了，我们总是在分心和拖延中挣扎。

有一天，当我的电脑出现问题时，我想到了这一点，我解决了这个问题，然后反思发生了什么。

> 问题:如何从命令行更改默认浏览器？

我*谷歌了*我咬了一口，从 [StackOverflow 帖子](https://stackoverflow.com/questions/17528688/set-default-web-browser-via-command-line)上掉了下来。这篇帖子提到了一个名为`default browser`的应用，它是在 GitHub 上可用的[。](https://github.com/kerma/defaultbrowser)

成功！

我安装了工具，只是为了我自己的反思和理智，我写了[直到](http://jonasbn.github.io/til/osx/change_default_browser_from_cli.html) ( *今天我学了*)。

我开始使用这个工具，过了很短的时间，我发现我厌倦了写出浏览器的名字，比如:`firefoxdeveloperedition`

> 问题:如何获得 Bash 的制表符补全？

幸运的是，我已经为我一直使用的工具实现了几个`bash`的完成，而这些工具并不存在。

再一次突破...

由于`defaultbrowser`在没有任何参数的情况下列出了所有可用的浏览器，所以给出了很好的解决方案。

成功！

现在`defaultbrowser`工具有一个小麻烦，那就是 GUI 中的提示变化，我不是唯一一个发现这个[问题](https://github.com/kerma/defaultbrowser/issues/3)有点烦人的人。

> 问题:如何无声的换浏览器？

*   我开始阅读提出的问题，以了解所有方面和讨论
*   然后我又看了一遍`defaultbrowser`的文档
*   然后我读了`defaultbrowser`的源代码
*   然后，我阅读了苹果官方文档中所使用的 API

失败！

这在 API 中不受支持，一些解决方法在问题中被描述为注释，但没有看起来*完美的*。

但是回到这个帖子，因为我在闲扯，而你有更好的事情要做。

每当我们想调整、尝试、评估或做某事时，我们都站在兔子洞的边缘，不知道它有多深。问题是，我们享受乘坐的刺激，问题的解决和成功的感觉。

我对我的观点很满意，但是正如我在上面的例子中所展示的，总是有新的发现、问题、解决方案和东西要学，而时间却很少。

管理时间的最好方法就是承认这一点

*   我们都不是一个人的军队
*   开源岩石
*   合作对我们大家都有好处
*   分享是关怀

感谢你花时间阅读这篇文章，现在回到你的兔子洞，让我知道它带你去了哪里，你学到了什么或你建立了什么，让我知道你是否能帮助我或我能帮助你，但不要浪费我的时间或你自己的时间-因为这是有限的
