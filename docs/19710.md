# 我是一名安全审计员和研究员，问我任何问题！

> 原文：<https://dev.to/terceranexus6/im-a-security-auditor-and-researcher-ask-me-anything-2i2b>

我的名字叫 Paula，我已经在这里呆了一段时间了，但是我还没有做过“问我任何事情”的事情，现在想做了。

我是一名 22 岁的西班牙安全审计员，我正在学习计算机工程。我还学习了一门调查性新闻课程，因为我正在研究世界各地的互联网审查制度，这是我的主要兴趣和研究方向。我在西班牙共同创立了一个名为 [Interferencias](https://twitter.com/Inter_ferencias) 的隐私和安全意识协会，我很高兴看到很多人参与其中。

到目前为止，我已经从事了许多与技术相关的工作，因为我也对系统和硬件感兴趣。我做过系统分析师，在瑞士当过机器人老师，在一所大学校园里为进入科技行业的女生做过 Arduino 监测员，还在几家公司做过安全审计员。我大部分时间都在研究、写作和学习，我在不同的城市发表安全演讲，主要是在我的国家西班牙，但出于同样的原因，我本周将去维也纳。

我非常喜欢开源，我只使用 Linux，这是我的主要工作环境。出于好奇，我使用 Debian Jessie 和 Tails。

我目前正试图为当地女性开发者组织一个 python 女孩俱乐部，已经有一些女孩对此感兴趣了。我们的 github 是[这里](http://pythongirlsgranada.github.io/)。

我还了解了**IPFS，因为分布式账本是我最喜欢的安全技术话题之一。**

 **我试着以 **Security Sprint** 的名义在 DEV 上保持每周一次的发布，尽管这两个星期对我来说很难坚持下去。

**我很乐意回答任何安全、审查或任何你有**的相关问题，以及会见对这些话题感兴趣的其他开发者用户。请随意问我任何事情。**
