## [](#install-iterm2)安装 iTerm2

从这里下载 iTerm2 终端 3.3 版。这个链接是每夜构建的，但是你可以安全地使用它。我用了很长时间了。日常使用足够稳定。

提取 zip 文件并将 app 文件移动到应用程序目录。

## [](#styling-terminal)造型端子

*   打开 iTerm。转到首选项。
*   选择外观选项卡，然后选择主题中的最小选项。

[![](../Images/872a68166687f87d7a3e8779aef4fad9.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--gT59BdcV--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/dfehtibb1yp8xn7l3kzg.png)

*   选择配置文件选项卡。要更改终端的背景颜色，请将背景颜色的 RGB 十六进制设置为 263239，如下图所示。

[![](../Images/dc0a6c5e38d5a7717ca9ebb4dee271cb.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--dWLCQrqK--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/s7vngk7fiwobv7luewgb.png)

## [](#install-ohmyzsh)安装 oh-my-zsh

您可以使用 curl 或 wget 安装 oh-my-zsh。

**Curl 命令:**sh-c " $(Curl-fsSL[https://raw . github . com/robbyrussell/oh-my-zsh/master/tools/install . sh)](https://raw.github.com/robbyrussell/oh-my-zsh/master/tools/install.sh))

**或**

**Wget 命令:**sh-c " $(Wget[https://raw . github . com/robbyrussell/oh-my-zsh/master/tools/install . sh](https://raw.github.com/robbyrussell/oh-my-zsh/master/tools/install.sh)-O-)"
