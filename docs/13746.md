# 新版本-geek hub:Jetbrain ide 的 GitHub 集成

> 原文:[https://dev.to/geekhubdeveloper/new-release-geek hub-github-integration-for-jetbrains-ides-188k](https://dev.to/geekhubdeveloper/new-release---geekhub-github-integration-for-jetbrains-ides-188k)

[GeekHub](https://geekhub.codes/) 是为 Jetbrain 的 IDEs，如 IntelliJ IDEA、RubyMine、PyCharm、PhpStorm、WebStorm、Goland 制作的插件，允许你在编辑器上使用 GitHub 作为工具窗口。您可以检查、审阅、评论、合并拉取请求等。所有这些都在您的图形用户界面编辑器中。

# [](#what-you-can-do-with-it)你能用它做什么

GeekHub 在 ide 的工具窗口中显示了所有请求的列表。通过单击某个拉式请求，您可以检查其内容，并执行某些操作，如合并、查看、检查差异等。与当前本地分支相对应的注释会出现在您的代码中。

# [](#key-features)关键特性

### [](#make-comments-via-line-markers)通过线条标记进行注释

[![line markers](../Images/1b96cda37837ec3bf093845b95cd456d.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--ZUj5LFZl--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/lt9odeqjsemjfc3cb1hn.png) 
打开文件后，只需点击代码左侧的铅笔标记，即可对文件进行注释。带有注释的行会自动突出显示，您可以通过将鼠标指针悬停在线或右边的黄色框上来查看注释。

### [](#checkout-branches)结帐分店

[![Checkout Branches](../Images/5ddbe1588c957e4743a3c78c05c816b1.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--ZnyfZtQ---/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/mjkqj3cwisq7icvsj6ja.png) 
选择一个拉式请求后，只需按右上方的结帐分行按钮即可结帐一个分行。请注意，右上角的 CurrentBranch 按钮已经更新。只需点击它即可跳转到拉取请求页面。

### [](#merge-pull-requests)合并拉取请求

[![Merge Pull Requests](../Images/3d51f1c929596db05d9533311a18a559.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--61cZKySj--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/jz14lvf3wvr18vk68zgr.png) 
你可以通过按绿色按钮来合并拉取请求，就像 GitHub 一样。

### [](#make-commentsreply)做出评论/回复

[![Make Comments/Reply](../Images/137fc902dbd8934de5e09ba932378d06.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--Eor-S3Ay--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/fzmaq6svxyntq0jer57u.png) 
像 GitHub 一样点击文本框进行评论和回复。

[![Make Comments](../Images/10ee3510bd0e5ed463f8a7ee5f461e37.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--EvrR8GAQ--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/7jnokpammebjjma5cinr.png) 
您也可以通过点击悬停在线上时出现的蓝色加号按钮来添加注释。

### [](#open-files)打开文件

[![Open Files](../Images/0b122d664fae13e9ea6117c9ed4e97bf.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--AmS19vhS--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/xyv4rd0ca33j05wt9ftu.png) 
按下文件名或编辑的名字，看看他们在你的 IDE 上写了什么。

### [](#show-diffs)显示差异

[![Show Diffs](../Images/e9dfa45775439738d46ed099846ed6e2.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--DjOy8dOv--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/pg7oeqiod8sby8f0hlkw.png) 
按下“显示差异”按钮，打开一个显示“差异”的新窗口，允许您比较和编辑版本之间的文件。
