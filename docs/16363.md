# 你发现自己在反复寻找什么？

> 原文:[https://dev . to/its asine/what-you-found-you-find-you-self-searching-again-55mm](https://dev.to/itsasine/what-do-you-find-yourself-searching-for-again-and-again-55mm)

大概是第六次，我谷歌了一下`netlify multiple github repos custom domain`。第六次，我遇到了[这个紫色链接](https://stackoverflow.com/questions/48604843/how-to-host-multiple-github-repos-using-the-same-custom-domain-on-netlify)关于在主回购中设置重定向引用。

我能学会吗？不太可能。大部分的战斗是知道如何找到答案。因为我已经找到这个答案 6 次了，所以我越来越擅长了。至少这次我登录了 StackOverflow 来启动这个问题，并投票支持它和它的答案。

看起来大多数开发人员经常[在互联网上询问如何退出 vim](https://stackoverflow.blog/2017/05/23/stack-overflow-helping-one-million-developers-exit-vim/) ，但是你发现自己一次又一次地在寻找什么呢？

(对我来说第二名是`git cherry-pick`的文档，因为我不相信自己不会搞砸语法)
