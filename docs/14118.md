# 工作的多种形式

> 原文：<https://dev.to/geekgalgroks/the-many-forms-of-work-37na>

刚开始时，许多人认为工作只是你做的有形的事情，例如代码行、模块装运、冲刺跑等。工作远不止这些。

我今天刚刚完成的工作示例:

*   与同事讨论问题和[橡胶回避](https://en.wikipedia.org/wiki/Rubber_duck_debugging)
*   查看和阅读旧的提取请求
*   在白板上绘图和书写
*   问题解决后，举行一个小型的胜利舞会

我没有物质输出。没有代码行。没有发货。但我帮助一位同事解决了她的问题，并建立了一个前瞻性计划。这是一个富有成效的早晨。
