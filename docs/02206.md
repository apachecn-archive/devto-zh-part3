# 在简单的脚本中使用 await/async

> 原文:[https://dev . to/Henry jw/using-await-async-in-a-simple-script-do0](https://dev.to/henryjw/using-await-async-in-a-simple-script-do0)

**问题:**

您希望在简单脚本的顶层使用`await/async`，如下所示:

```
async function asyncFunction() {
    return new Promise(resolve => {
        //  simulate wait
        setTimeout(resolve, 1000);
    })
}

try {
    // This won't work!
    await asyncFunction()
} catch(err) {
    console.error('Something bad')
} 
```

**解决方案:**

将您的代码包含在一个[立即调用的函数表达式(life)](https://en.wikipedia.org/wiki/Immediately_invoked_function_expression)中。

```
async function asyncFunction() {
    return new Promise(resolve => {
        //  Simulate wait
        setTimeout(resolve, 1000);
    })
}

// This works
(async () => {
    try {
        await asyncFunction()
    } catch(err) {
        console.error('Something bad')
    }
})() 
```
