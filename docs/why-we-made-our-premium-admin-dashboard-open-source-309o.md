# 为什么我们让我们的高级管理仪表板开源

> 原文：<https://dev.to/flatlogic/why-we-made-our-premium-admin-dashboard-open-source-309o>

早在 2013 年，我们的团队就采用了 Bootstrap、Angular、React、Vue 等关键技术。NET 和其他工具来提供世界一流的管理仪表板。我们研究、分析、设计并开发了用户友好的管理仪表板。之后，我们卖出了 7000 多份我们的[管理模板](https://flatlogic.com/admin-dashboards)的许可证。

[![open-source admin dashboard](img/a9dd3678fd7da6d4d5aaecd21589271b.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--Q9idC4cY--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://flatlogic.com/blog/wp-content/uploads/2018/08/article_openSource.png)

对于管理模板，有两种主要的定价策略:溢价和开源。这些年来，我们一直在使用一种优质的商业模式，而且效果很好。我们的产品在可以下载之前就已经有了预付价格。与开源模型的溢价相比，模板的初始下载是免费的，但功能有限。

所以我们决定试一试，开源我们的高级管理仪表板。我们认为，通过给用户一个不花钱尝试模板的机会，开源可能对消费者和开发者来说是双赢的。它可以显著增加愿意集成和定制我们产品的人数。

你可能会问，什么是管理面板，有限的功能意味着什么？它是 web 应用程序中的一个管理平台，允许在 web 应用程序的用户界面中管理和操作数据。管理面板可以帮助您对用户行为提供一些可操作的见解，对违反网站规则或条款和条件的一些配置文件采取行动，并跟踪交易。管理面板可以变成一个仪表板。它可以显示业务统计和分析，而无需用户编写任何代码。您可以导出每周数据并创建不同种类的报告。我们用几个最重要的组件来限制我们的免费版本，如响应布局、风格引导组件、仪表板样本、表格、悬停侧边栏等。

因此，当我们开放这个代码库时，我们肯定准备好了从开源社区获得一些反馈——你可以在这里找到它。结果棒极了。销售额的增长，越来越多的下载，几十封带有反馈和建议的信件，以进行定制集成。

我们已经得出结论，开源模式肯定更适合我们。它给了我们宝贵的反馈，多年来，我们已经知道我们必须真正听取每个人的意见。构建软件是好的，但是围绕你的软件构建一个社区更好。

随着用户尝试新的用例并构建定制、添加、插件和框架，它的灵活性得到了提升。因此，我们可以为大量项目进行定制，从自然语言处理到机器学习，公司 KPI 和绩效指标。

我们客户的信任对我们公司在世界上的地位有着深远的影响。在过去的一年里，我们为客户创造了如此多的定制项目，我们觉得我们已经准备好拓展我们的视野了。因此，我们向用户提供足够的价值，这样他们就有理由在新案例上信任我们。

我们明白我们的例子并不能解决所有人的问题。开源商业模式是否会成功取决于你的产品和很多因素。评估什么目前适合你的，或者决定你希望你的公司是什么，是通过一个适合你的受众和公司总体目标的模式创造收入的第一步。所以我们想祝你好运，我们会努力让你更新，分享我们在自己的旅程中学到的一切。
