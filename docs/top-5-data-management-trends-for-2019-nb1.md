# 2019 年五大数据管理趋势

> 原文：<https://dev.to/shameemreza/top-5-data-management-trends-for-2019-nb1>

由于我们消费数据和与数据交互的方式每天都在变化，我们也有大量的创新来提高公司的敏捷性和运营效率。考虑到 EMEA 市场，我强调了决策者在 2019 年必须知道的以下 5 个相关趋势:

## 1。它将增加多云环境的使用

麦肯锡公司的一份报告显示，流向亚洲的数据比 2005 年增加了 45 倍。北美和欧洲等主要地区的数据分别从 5，000 Gbps 增长到 20，000 Gbps 和 1，000 Gbps 增长到 5，000 Gbps。基于 100-500 Gbps 和 2005 年低于 50 Gbps 的原始数据。

公司在国外经营，比以往任何时候都更加依赖技术。几乎不可避免的是，扩大多云的使用 IDC 估计，2021 年客户将在云计算和相关服务上花费 5540 亿美元，是 2016 年的两倍多。

内部数据和应用程序不会过时，但数据部署模式将会增加，因为它们会再次混合内部选项(自有数据中心)、软件即服务(SaaS)、基础架构即服务(IaaS)、托管云和私有云。

## 2。闪存供应短缺和价格将在 2019 年改善

根据 Gartner 2018 年 10 月的一份报告，预计到 2019 年年中，闪存将出现轻微的供应短缺，价格将在很大程度上稳定下来，这要归功于中国内存产量的增加。

更大的供应量和更低的定价意味着闪存部署将更多地用于运营恢复领域，而最近 14 天的备份和复制数据通常都托管在该领域。

在我看来，这种增加的闪存容量将导致备份机器映像(或副本数据管理)的即时装载的使用增加。

提供拷贝数据管理(CDM)功能的系统可以提供超越可用性的价值，以及更好的商业结果。利用备份和复制数据的一些实用示例包括:DevOps、DevSecOps 和 DevTest、补丁测试、分析和报告(测试、分析和生成补丁报告)。

## 3。预测分析将变得普遍和标准

预测表明，预测分析市场将在 2022 年达到 124.1 亿美元，这意味着自 2017 年以来将增长 272%，年复合增长率为 22.1%。

具体来说，亚太市场(APAC)有望在本预测期内实现最高的复合年增长率。

基于遥测数据的预测分析，本质上是由机器学习(ML 或机器学习)驱动的建议和推荐，是肯定会变得普遍和规范的类别之一。

## 4。“多功能”(或通才)职能将成为大多数 IT 公司的新运营模式

虽然前两个趋势集中在技术方面，数字的未来仍然是模拟的:人。

如果我们将人才短缺与新的基础设施以及濒临崩溃的公共云+软件即服务(SaaS)结合起来，我们会发现需要在广泛学科领域拥有经验的专家，他们还必须更好地了解业务。

例如，新加坡信息技术(IT)劳动力市场继续保持高招聘水平。

**a .对传统备份方法和解决方案的失望**

该领域的三家主要供应商将在 2019 年继续失去市场份额。事实上，该领域最大的供应商十年来一直在失去市场份额。

企业正在抛弃传统供应商，而更喜欢更敏捷、更具颠覆性和更动态的选项，这些选项能为它们提供在数据驱动时代发展所需的功能。

一份认知报告指出，亚太地区(APAC)82%的公司领导认为工作的未来在于智能机器。

**b .三个 C 的临界点:成本、复杂性和容量**

这三个 C 仍然是数据中心工作人员对其他提供商的解决方案不满意的原因。

总的来说，成本过高，不必要的复杂，缺乏足够的容量，这会影响备份、恢复或即时装载到虚拟机映像的速度。

这三个关键标准将继续成为公司增加或完全替换其备份解决方案的主要原因。

## 5。首批 5G 网络的到来将为分销商和云服务提供商(CSP)收集、管理、存储和处理尽可能多的数据创造新的机会。

5G 将很快被用于机器对机器的通信和物联网(IoT)技术。移动消费网络已经达到了 4G 速度足以满足大多数消费者需求的地步。

2019 年将更加关注技术的标准化和通用化测试。

在为未来设计的设备上，能够保证它们与可用的技术兼容，并在欧洲转化为真正的“千兆位社会”。
