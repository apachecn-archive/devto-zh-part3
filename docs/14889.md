# 贡献与倾销

> 原文:[https://dev.to/goyo/contributing-vs-dumping-3d91](https://dev.to/goyo/contributing-vs-dumping-3d91)

DEV 吸引我的一个原因是它主要由愿意分享他们的经验、知识和讨论事情的个人组成。

这与其他一些网站形成了鲜明对比，这些网站成了那些愿意用低质量、非技术性内容为自己或自己的产品做广告的公司的垃圾场。

遗憾的是，我今天的开发经验表明在这方面还有一些改进的空间:

[![](../Images/b9d8336a7cfc84dad4df54bb0395a8f1.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--LMTB03XS--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/s9t2acji0wgnn4df0sr2.png)

这并不是对 Octoparse 或任何其他人的攻击。我只是希望所有成员至少都在努力为社区做贡献，而不仅仅是把它作为达到另一个目的的手段。

如果我太苛刻或不应该在一个孤立的情况下流汗，让我知道，我很抱歉！
