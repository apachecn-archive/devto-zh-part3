# 2018:一年回顾

> 原文:[https://dev.to/ruphaa/2018-year-in-review-584p](https://dev.to/ruphaa/2018-year-in-review-584p)

又一个激动人心的一年开始了。我想花点时间反思一下 2018 年发生的一切，因为对我来说，这是激动人心的一年，无论是职业生涯还是个人。

这是我的第一篇博文，就像我的第一篇博文一样。所以对我放松点。

# [](#career)事业

我开始对前端的细微差别产生浓厚的兴趣，并决定寻找可以培养和提高我的前端技能的机会。我辞掉了原来的工作，在一家出色的公司获得了一个前沿职位。每天我都看到自己成为一个比前一天更好的开发者。

# [](#talks-and-meetups)会谈和聚会

*   我参加了一个由脸书开发者圈在钦奈举办的关于前端框架的会议。
*   我和其他 5 个人一起给本科生做了一个关于“虚拟和增强现实”和“网络支柱- HTML，CSS 和 JS”的演讲。

# [](#achievements-unlocked)成就解锁

*   在黑客马拉松中获得第一名
*   完成了 hacktoberfest，开始了我的开源贡献。(迫不及待地想收到我的 hacktoberfest 商品)
*   参加了 code 11 月(第一次在这里计时)。我看到了很多 codepen 的灵感，我现在完全投入其中。
*   创作了一些 CSS 插图，大部分灵感来自 dribbble 镜头。这是我给好奇者的答案-[https://codepen.io/ruphaa/](https://codepen.io/ruphaa/)

# [](#health-and-fitness)健康与健身

*   我开始健身，现在每周去健身房 5 次。
*   参加了为期 30 天的瑜伽挑战，并成功完成。瑜伽神奇地给你一种内心平静的感觉，增强你的自信，提高你的注意力。

# [](#bummers)无赖

*   我接受了挑战，但不知怎么的，我被岔开了。
*   我全力以赴开始了 11 月代码挑战，但最终没有完成。
*   计划建立一个投资组合，但没有取得多大进展。

# [](#lessons-learnt)吸取的教训

挫折在所难免。但是我们从中学到的才是重要的。我设定了很多奢侈的目标，但是没有相应地计划和管理我的时间。所以我会试着专注于

*   设定现实的目标
*   一次专注于一件事

# 2019 年有什么

*   多写博文。
*   建立投资组合
*   每周听 3 次播客
*   每天阅读一篇博文
*   每周创建一个代码笔
*   完成至少一轮`#100DaysOfCode`并记录
*   读 10 本书

2018 年对我来说真的是很神奇的一年。今年开局不利，但回顾过去，我对一切都心存感激。还没有结束，我非常感谢有一位出色的导师 [@flexdinesh](https://dev.to/flexdinesh) ，感谢所有的职业指导和激励，让我走上了正确的道路。

祝你来年过得愉快！
