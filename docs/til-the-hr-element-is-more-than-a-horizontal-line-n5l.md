# TIL:HR 元素不仅仅是一条水平线

> 原文：<https://dev.to/stefanjudis/til-the-hr-element-is-more-than-a-horizontal-line-n5l>

当你在设计中需要一条水平线时，你可能已经使用了`hr`元素来设计样式。今天我看到了一条由[威尔·博伊德](https://mobile.twitter.com/lonekorean/status/1086727522144931840)写的推文，他分享了`hr`元素具有语义含义！😲

从[到规格](https://html.spec.whatwg.org/multipage/grouping-content.html#the-hr-element):

> hr 元素表示段落级别的主题中断，例如故事中的场景变化，或者参考书的某一部分中另一个主题的过渡。

很高兴知道！
