# Javascript 中字母和变量的区别

帖子链接:[Javascript 中 let 和 var 的区别](https://www.skptricks.com/2018/11/difference-between-let-and-var-in-javascript.html)

本教程解释了 javascript 中 Let 和 Var 的基本用法和区别。当任何开发人员使用 JavaScript 时，通常会使用 let、var 关键字。我们大多数人都是随机使用它，因为在 JavaScript 中使用 let 和 var 有很多混乱。

Let Scope :
用 var 定义的变量的作用域是函数作用域或者声明在任何函数之外，全局。

Var 作用域:
用 let 定义的变量的作用域是块作用域。
