# 做一个莫扎特是什么感觉？

> 原文:[https://dev.to/mattdark/what-is-it-to-be-a-mozillian-4ha2](https://dev.to/mattdark/what-is-it-to-be-a-mozillian-4ha2)

2017 年 4 月，我收到了一份邀请，在墨西哥图斯特拉古铁雷斯的一所当地大学**萨拉查大学**发表演讲。事件发生在 4 月 18 日星期二，我讲述了我的莫扎特的故事。

我决定写这篇文章来反思作为一个墨子是什么？”，我给我的演讲取的标题。首先，这是我第一次在墨西哥做这个演讲。2016 年，我在哥伦比亚和西班牙做了这个演讲，但形式不同，是一个 **[Pechakucha](https://www.pechakucha.org/)** 演讲。演讲的重点是我作为一名志愿者在 Mozilla做了些什么，以及我是如何参与到墨西哥的社区中的。

[![Firefox plushie](../Images/01f88c1e030d34be2996b135d2c951c9.png) ](https://res.cloudinary.com/practicaldev/image/fetch/s--U7eNgsNJ--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://cdn-images-1.medium.com/max/800/1%2A0fyy6ej5_LDUYN-Iye_Nog.jpeg) *火狐 plushie 在墨西哥图斯特拉古铁雷斯举行的 Rust 第一次会议上。*

十多年来，我一直是自由开源软件(FOSS)的用户。我第一次安装 GNU/Linux 发行版是在 2006 年 12 月。那是 **Ubuntu** (一个基于 Debian 的 GNU/Linux 操作系统)5.04 或 5.10，我不记得确切的版本了，可能都是。

在接下来的几年里，我开始用部分业余时间为一些自由/开源软件项目和社区做贡献，帮助其他人将他们日常使用的应用程序迁移到自由软件，包括操作系统的改变，并在当地的技术活动中推广自由软件的使用。这也是我为 Mozilla 做贡献的部分原因。

**[Mozilla](https://mozilla.org/)** 是一个促进互联网开放、创新和参与的非营利基金会。在校园聚会上认识了一些 Mozilla 成员后，我加入了墨西哥 T4 的 Mozilla 社区。我相信 Mozilla 的使命，并认同该基金会为让互联网变得更美好所做的工作。

你可能因为 **[Firefox](https://getfirefox.com/)** 或 **[雷鸟](https://getthunderbird.com/)** 而知道 Mozilla，但还有其他产品和项目，如 **[Webmaker](https://webmaker.org/)** ， **[MDN Web Docs](https://developer.mozilla.org/)** ，**[Rust](https://rust-lang.org/)**(Mozilla 正在向全世界发布的一种编程语言) **[A-Frame](https://aframe.io/)** 还有**顺便说一下，关于火狐的标志有一个有趣的事实。**

你知道火狐的吉祥物不是狐狸而是小熊猫吗？许多人认为火狐的吉祥物是一只狐狸，当他们听到真相时会感到惊讶。

[![Firefox mascot](../Images/4925c68599c30169599d2c79fbc1f59b.png) ](https://res.cloudinary.com/practicaldev/image/fetch/s--dTb3jUXV--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://cdn-images-1.medium.com/max/800/1%2AQ2s4RbgzAyk2OVuOLziBpA.jpeg) *一只小熊猫。【来源:维基百科】*

## [](#my-mozillian-story)我的墨子故事

我最喜欢 Mozilla 的一点是，不管你来自哪里，也不管你以什么为生。你为实现 Mozilla 的使命所做的一切都很重要，这让你觉得你是它的一部分。来自不同城市、不同文化、不同专业领域的人们聚在一起，为我们都受益的项目做出贡献。学习、发展新技能、结识有趣的人、游览其他城市，只是作为一名莫扎特的经历的一部分。

我回顾过去的几年，记得一个害羞的工程学生，然后我看到了下面的图片，作为一名演讲者超过十年，这让我想到作为一名 Mozillian 如何帮助我提高了公开演讲的技能，因为我一直在技术和创新活动上谈论 Mozilla，它的使命和项目。

[![Pechakucha in Madrid](../Images/47b67e06e1d430f675d0fc04243a8c15.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--qigWvFLk--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://cdn-images-1.medium.com/max/800/1%2AJFzxmdchmi76K9vJdOdUzw.jpeg)*Pechakucha # ICDemocracia。西班牙马德里*

在过去的 12 年里，Firefox 一直是我的默认浏览器。如果你在你的语言中使用 Firefox，这取决于本地化团队所做的工作。在 Mozilla 社区的最初几年，我与其他 Mozilla 人合作，帮助将 Firefox 本地化为墨西哥西班牙语。

[![Firefox](../Images/daa6a13e01c0f28cfa38ea778ada4184.png) ](https://res.cloudinary.com/practicaldev/image/fetch/s--ngmJnZgu--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://cdn-images-1.medium.com/max/800/1%2Aj8_nheBMw1bb_jEO-dE_Lg.jpeg) *火狐*

然后在 2013 年，Mozilla 给了世界一个新的移动操作系统，命名为 **Firefox OS** 。我是墨西哥火狐操作系统发布团队的一员。我的第一部智能手机搭载了火狐操作系统。作为该项目的一部分，我有机会参加一些技术活动，发表关于 Firefox OS 以及如何为新平台开发应用程序的演讲和研讨会。

[![Firefox OS Launch Camp](../Images/36b1ddac858104430db89781bfe59bf3.png) ](https://res.cloudinary.com/practicaldev/image/fetch/s--aoW5EGVs--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://cdn-images-1.medium.com/max/800/1%2AeEsCb2Xv73IKqCcDmk-a6Q.jpeg) *火狐 OS 在哥伦比亚卡塔赫纳的发布团队*

Rust 是一种专注于安全性、速度和并发性的系统编程语言。Rust 第一个稳定版于 2015 年 5 月 15 日发布。三年前，我开始学习 Rust，从那以后，我参加了一些关于它的活动。现在我是墨西哥[的 Rust 社区的一员。](https://twitter.com/rustlang_mx)

[![Rust Logo](../Images/77acf229991b4abec871c2b7ddce9704.png) ](https://res.cloudinary.com/practicaldev/image/fetch/s--KuBEVU8n--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://cdn-images-1.medium.com/max/800/1%2AHLxC71SIkE5-xD5OorIEGw.png) *铁锈*

2017 年期间，我在墨西哥、哥伦比亚和危地马拉参加了一些关于 Rust 的活动，用西班牙语录制了一些网络研讨会，9 月 1 日，Rust 的第一次会议在墨西哥图斯特拉古铁雷斯的 **[CodigoFacilito](https://codigofacilito.com/)** 总部举行。

[![Meetup of Rust](../Images/c7249b980249a3b7fed0e05bb333daf3.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--yTjRnDEN--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://cdn-images-1.medium.com/max/800/1%2An1kT8er6Dz0oEiZ4kCIDMQ.jpeg)*Rust 在墨西哥图斯特拉古铁雷斯的第一次聚会。2017 年 9 月*

## [](#experiences)经历

作为一名莫兹利人的部分经历是，我获得了一些通过其他方式无法获得的机会，比如结识有趣的人，去墨西哥和国外的其他城市旅行。

多亏了 Mozilla 和我参与的自由/开源软件项目，我认识了来自世界各地的有趣的人，他们做着令人惊叹的事情，认识了我的一些英雄和我崇拜的人，比如 Mozilla 的 Mitchell Baker，Django 的联合创始人 Jacob Kaplan-Moss，或者台湾的数字部长 Audrey Tang。

出国旅游需要什么？这个问题我有时会问一些我演讲的与会者。常见的回答都和钱有关。访问另一个国家需要护照或签证。但大多数时候，你只需要擅长某件事，比如编程，或者作为志愿者将你的部分空闲时间投入到一个项目中，做你热爱的事情。机遇随之而来。

有了 Mozilla 和我参与的项目，我收到了在墨西哥、哥伦比亚、加拿大、西班牙和危地马拉举办讲座和研讨会，或者参加技术和创新活动的邀请。去年，我去了阿根廷的罗萨里奥，参加了阿根廷公民创新实验室( **[#LABICAR](https://www.santafe.gob.ar/ms/labicar/)** )和“**互联网促进开放创新**”对话小组，这是开放议程的一部分。

## [](#final-thoughts)最后的想法

我试图解释什么是 Mozillian，可能还不够清楚，但这是一种你必须经历的经历(如果你有兴趣为 Mozilla 的使命做贡献)，这样你才能说清楚到底是什么。而且几乎可以肯定的是，这篇文章的文字在不久的将来会发生变化，以便更好地解释它。

在过去的七年半时间里，我一直是一名莫扎特迷，对此我感到无比自豪。下面有一张 Firefox 纪念碑的照片，它位于加利福尼亚州旧金山 Mozilla 办公室外面，那里写着过去和现在 Mozilla 人的名字，包括我的名字。

> 今晚睡个好觉，因为我们正在让互联网成为一个更好的地方
