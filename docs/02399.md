## 装置

将这一行添加到应用程序的 Gemfile 中:

```
gem 'rubocop-yayoi'
```

## 使用

### robocop 配置文件

把这个放进你的`.rubocop.yml`。

```
require: rubocop-yayoi 
```

现在你可以运行`rubocop`，它会自动加载 RuboCop RSpec cops 和标准的 cops。

### 命令行

```
rubocop --require rubocop-yayoi
```
