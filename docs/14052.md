我是一名开发人员，我熟悉的工具或技术组合包括使用 C#、ASP.NET、SQL、实体框架、SignalR、JavaScript(Angular，React)和 TypeScript。

我已经在 Xamarin 上做了几个项目(目前主要是表单，但是我也在考虑在 iOS 和 Android 上工作.....)我能够在 Visual Studio 中使用 C#创建移动应用程序，这实际上很酷，但是我对 ReactNative 特别感兴趣，因为似乎更多的开发人员更喜欢它而不是 Xamarin，所以我想知道您对 Xamarin 或 ReactNative 的体验，您会选择哪一个，为什么？
