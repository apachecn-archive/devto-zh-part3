# 使用 Symfony 组件的前 3 个项目

> 原文:[https://dev . to/vidushisingh/top-3-projects-using-symfony-components-4 i69](https://dev.to/vidushisingh/top-3-projects-using-symfony-components-4i69)

Symfony 为 PHP 框架开发敏捷 web 应用铺平了道路，被开发者们公认为**最快的 PHP 框架**。

Symfony 框架为开发者提供了简单、稳定、可扩展和灵活的方式，便于开发 web 应用程序。

Symfony 由来自 120 多个国家的 600，000 多名开发人员组成的强大团队提供支持，他们不断为支持所有 Symfony 用户构建优秀的 web 应用程序做出贡献。

其他框架也是有益的，但是 Symfony 巨大的社区支持、持续的改进和对改进的贡献使它脱颖而出。

项目开发需要一个框架，选择正确的框架是非常重要的，已经有许多好的项目，但是很少有人以正确的方式使用它，并在很短的时间内在市场上推广。

下面我们挑选了**使用 Symfony 组件**的前三个电子商务项目，从最初阶段开始，它们就不断地推动和稳定**电子商务世界**:

## [锡拉丘兹大学](#sylius)

**[Sylius](https://sylius.com/)** 是基于 Symfony 全栈的最佳电子商务框架，强大的代码质量、有效的测试文化、内置的敏捷(BDD)工作流、卓越的灵活性使其成为业务需求的最佳解决方案。

强大的 REST API 允许在任何设备上轻松集成和创建独特的客户体验。

西里乌斯的现场演示-[https://sylius.com/demo/](https://sylius.com/demo/)

使用全栈**行为驱动开发**构建，与 **phpspec** 和 **Behat** 合作，旨在创造产品并以专业开发、咨询和培训服务支持产品。

他们相信科技能促进商业发展，让人们更容易交易商品和服务，让商业世界变得更美好。同时通过分享我们的代码和想法来帮助开源社区。

Sylius Github 知识库-[https://github.com/Sylius](https://github.com/Sylius)

## [](#oro)奥罗

领头的 **[奥罗克](https://oroinc.com/)** 。由开源商业应用领域的行业大师于 2012 年成立，是 OroCommerce、OroCRM 和 OroPlatform 背后的软件公司。

OroCRM 的现场演示-[https://oroinc.com/orocrm/orocrm-demo](https://oroinc.com/orocrm/orocrm-demo)

**OroCRM** 为从零售到营销和客户服务组织的所有类型的企业提供强大的客户管理工具、详细的分析和广泛的第三方软件集成。

https://github.com/oroinc Github 库-[ORO](https://github.com/oroinc)

为企业 web 应用程序的快速应用程序开发而打造的平台，支持内置特性和组件，可降低整体开发成本。

## [](#uvdesk)UVdesk

[UVdesk](https://www.uvdesk.com/en/opensource/) 是独立的服务台解决方案提供商，可作为基于 SaaS 的经济解决方案和适合各行各业的免费开源企业系统，提供最佳的客户支持解决方案。

SaaS 解决方案收取较高的代理成本，并对定制设置了限制，并且不是每个开源服务台对客户支持都是安全的，因为它包含与业务和客户相关的重要数据。

UVdesk 开源 Github 资源库-[https://github.com/uvdesk/community-skeleton](https://github.com/uvdesk/community-skeleton)

每个企业都是独一无二的，都是为了发展而发展的，一个专用的系统不会适合所有人，它可能需要复杂的定制以及多代理创建功能，因此 UVdesk 开源服务台最适合使用。
