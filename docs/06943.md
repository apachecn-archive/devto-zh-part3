# 死忠视窗迷的视窗版 Ubuntu

> 原文：<https://dev.to/almenon/ubuntu-on-windows-for-die-hard-windows-fans-7h6>

在过去的几年里，我一直完全在微软工作。打字稿。C#。VB。MSSQL。IIS。蔚蓝色。这样的例子不胜枚举…

但现在我加入了一家新公司，他们根本不用 windows。这是一个彻底的范式转变。JS 而不是 TS。Python 代替 C#。Postgres 而不是 MSSQL。Nginx 和 AWS 而不是 IIS 和 Azure。

我可以和 windows 说再见了。有了铝热剂和木筏，我可以给我的电脑一个体面的海盗葬礼。然后我会用 macbook air，坐在咖啡店里，告别☕ENTERPRISE DEV☕，变成⚡ROCKSTAR HACKER⚡.

但是你知道吗？这些年来，我变得有点喜欢窗户了。它很笨重。很慢。但我是伴随着它长大的，该死。所以……

输入 WSL，这是用于 Linux 的 [Windows 子系统](https://docs.microsoft.com/en-us/windows/wsl/install-win10)，也以其更为笨拙的缩写 BOUOW 为人所知。有了这个坏男孩，你可以在和 windows 一样的环境中拥有一个成熟的 ubuntu 系统！万岁！问题解决了！

但没那么快。您仍然需要学习所有新的 linux 概念和命令。准备好迎接一辆装满虫子的小丑车…
