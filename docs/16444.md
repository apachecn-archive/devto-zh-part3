# 所有 Devart ODBC 驱动程序的新版本已经发布！

> 原文:[https://dev . to/devart software/new-versions-of-all-devart-odbc-drivers-are-always-released-1cl 9](https://dev.to/devartsoftware/new-versions-of-all-devart-odbc-drivers-are-already-released--1cl9)

用于数据库管理和开发的数据库连接解决方案和工具的捷克软件提供商 Devart 已经推出了用于数据库和云的所有 ODBC 驱动程序的升级版本，其中包括用于 macOS 和 Linux 的 ODBC 驱动程序的变更分发，以简化安装和配置过程。现在它们作为用于 Linux 的 DEB 和 RPM 包以及用于 macOS 的 PKG 包分发。我们的 ODBC 驱动程序为基于 ODBC 的应用程序提供了高性能和功能丰富的连接解决方案，以直接从 Windows、macOS、Linux(32 位和 64 位)访问最流行的数据库。

更多有用的功能和实现的改进:

*在所有 ODBC 驱动程序中，增加了强制 ODBC 2.x 行为的可能性。

*现在 BigCommerce 的 ODBC 驱动程序支持 OAuth 认证。

InterBase 驱动程序中增加了对系统加密密码(SEP)的支持。

*我们的 ODBC 驱动程序已经支持 Oracle 18c。

* PostgreSQL 的 ODBC 驱动程序支持 PostgreSQL–11 的最新版本。

*在 SQLite 的新版本 ODBC 驱动程序中，直接模式基于 SQLite 引擎版本 3.26.0。

*目前的驱动程序版本支持 Zoho CRM 版和 OAuth 认证。

在[https://www.devart.com/odbc/](https://www.devart.com/odbc/)了解更多关于数据库和云的更新 ODBC 驱动程序。
