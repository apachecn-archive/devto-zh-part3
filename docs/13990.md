# 帮助收集数据

> 原文:[https://dev.to/peiche/help-gathering-data-4c3b](https://dev.to/peiche/help-gathering-data-4c3b)

我甚至不知道从哪里开始，所以我想我就把这个扔出去，看看它会怎么样。

我女儿是残疾人。她从事体育和舞蹈，但我和妻子花了几年时间才知道她现在参加了哪些组织，而且主要是通过口口相传。其他家庭告诉了我们，因为他们有我们没有的多年经验。现在我们把信息传递给其他经验不足的家庭。这是我们交谈过的每个人都同意的问题:确实没有好的方法来发现什么组织在那里，以及他们能提供什么帮助。

有一些网站，像 https://www.challengedathletes.org/resources/，实际上只是网站列表，但没有更多的迹象表明这个团体有轮椅篮球，那个团体有适应性芭蕾，诸如此类的东西。所以我在想，如果我建一个提供索引的网站会怎么样。可搜索，分面，像 Algolia 或 AWS Cloudsearch。这部分我能做。但是我该如何着手收集信息呢？我能设法刮掉它吗？如果有，我该如何组织？我是否通过请愿/r/disability、我的家人所属的脸书支持团体以及互联网上的其他地方进行众包？

我可以设计数据模型。我可以创建 webapp。我可以让它变得又快又漂亮又容易使用。但是我如何获得数据呢？
