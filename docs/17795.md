# “难道我们就不能...?"

> 原文:[https://dev.to/nimmo/cant-we-just-56g](https://dev.to/nimmo/cant-we-just-56g)

想去月球吗？ *刚飞到那里*。
**想把所有的空闲时间都用来做自己喜欢的事情？** *刚辞职*。
**想进乐队？** *刚开始一个乐队*。
**想写博文？** *就写一个*。
**晚餐想吃牛排？** *说做就做。两分钟的工作，完成了。*

大多数人很容易明白为什么你不能*就*飞到月球，或者*就*辞职。

但是你可能认为你可以*组建一个乐队。*

只写一篇博文是合理的。

任何人都会做牛排，对吧？

人们不喜欢自己的成就被轻视。那么，为什么我们很快就把我们需要解决的问题变得微不足道了呢？

也许这是错误的乐观。但是“只是”这个词是贬低别人的工作量或者你要求他们完成的任务的一种快捷方式。

告诉某人“就做 x”很容易意外增加他们的压力水平。考虑这两个问题:

> “你能写一篇关于服务测试的博文吗？”

这告诉对方你不欣赏你要求他们做的事情。

> “你能提供一份服务测试指南吗？”

这还差不多。你永远不会要求爱因斯坦仅仅证明原子理论。
