如何在 OpenSuse 上成功安装 EB cli 您确实需要在之前安装一些开发构建库**，以便 make 成功构建。** 

```
# sudo zypper in gcc zlib-devel libffi-devel libopenssl-devel
```

这应该为 Suse 用户省去很多麻烦。
