# [](#crawl-index-rank)抓取，索引，排名

首先要明白的是，搜索引擎在互联网上搜索站点页面。一旦它发现新的东西，搜索引擎将索引页面内容，使其成为搜索的一部分。一旦内容被编入索引，它就可以与类似内容的其他搜索进行比较。

术语 SEO，或搜索引擎优化，是让内容在给定的搜索条件下排名更高。你专注于特定的关键字，有人会用它来找到你的网站。排名越高，网站的访问量就越大，这可能是营销、增长甚至收入的来源。

# [](#canonicalurl)规范 _ 网址

当创建内容时，您希望它被尽可能多的观众看到。这可能意味着在不同的网站上发布相同的内容。通过这样做，你可以触及多个受众。

但是这里有一个障碍。你希望所有的流量加在一起，使内容排名更高！这就是 canonical_url 的用武之地。(这篇文章实际上使用了我公司博客[十英里广场](https://tenmilesquare.com/a-primer-on-seo/)中的一个)

如果你看分析，你仍然可以看到不同的流量来源。但是出于 SEO 的目的，有一个“家”应该继续在搜索结果中排名靠前。如果您的 canonical_url 用于比您的主域级别更高的高排名域，这一点尤其重要。我将在下面深入探讨这个问题。

如果你正在寻找如何在你的网站上实现这一点，看看[具体的方法](https://support.google.com/webmasters/answer/139066?hl=en)。

# [](#backlinks)反向链接

你可能听说过反向链接，以及它们对提高你的内容排名有多重要。有多个东西引用你的帖子真是太棒了！越多，搜索引擎就越倾向于把这篇文章排在更高的位置。

这被认为是另一个网站的“信任投票”。如果反向链接出现在排名很高的网站上，它会特别有价值。因此，为你的页面建立一个“家”意味着所有的互联网善意都被传递到一个特定的位置，而不是分割你的内容位置并与你自己竞争。

为了更深入地了解反向链接，看看如何在你自己的网站上从中受益，[查看这个资源](https://backlinko.com/backlinks-guide)。

# [](#duplicate-content)重复内容

现在我们进入这篇文章最棘手的部分。重要的是要知道，一个域内的重复内容与不同域之间的重复内容有不同的规则。我将重点讨论后一个问题。

正如我们上面提到的，就像网页可以排名一样，域名也可以排名。

> 有很多关于如何提高域名排名的细节，包括加载时间，工作链接等。那可能是另外一个帖子。

特定域满足用户查询的频率越高，它的排名就越高。如果内容存在于两个不同的站点上，这一点非常重要。如果你没有这个规范的 url，搜索引擎可能会认为排名较高的域名是信息的原始来源，甚至不会对你的内容进行排名！

这似乎不公平，但确实有道理。搜索引擎试图将你的查询与独特的、有价值的内容匹配起来。这意味着要处理多次出现的内容。给你看三个内容相同的点击是不相关的。

多年来，这种处理方式已经变得相当复杂。有些域名充满了重复的内容。在这种情况下，搜索引擎可能会认为网站上的所有内容都可以在其他地方找到，对其进行索引是一种浪费。

当人们谈论“惩罚”重复内容时，这通常就是他们的意思。有人会说搜索引擎不会惩罚重复的内容，而是奖励独特的内容。这是一种语义学。但重要的是要明白，搜索引擎寻找的是有价值的，独特的信息。

要真正深入探究这个问题的复杂性，这是一篇很棒的[帖子](https://www.hobo-web.co.uk/duplicate-content-problems/#what-is-duplicate-content-)。

# [](#seo-can-change)SEO 可以改变

随着时间的推移，搜索引擎越来越擅长判断内容。这通常是积极的，但有时也会有不良的副作用。如果你想提高你的搜索引擎优化，或者更好地理解它，跟踪最新的规则。其实他们昨天才换的！
