## [](#the-what)那个啥

我机器学习之旅的第一个坎是，WTF 是机器学习？

“机器学习是人工智能(AI)的一种应用，它为系统提供了自动学习和根据经验改进的能力，而无需显式编程。”——来源[此处](https://www.expertsystem.com/machine-learning-definition/)

因此，基本上这是人工智能的一个应用，它使系统能够在没有“如果/否则”条件的情况下从经验中学习。

## [](#why)为什么？

我们为什么要学习机器学习？简单的答案是为什么不呢？拥有创造某种东西的能力，这种东西有可能使我们人类变得更好，可以在日常生活中帮助我们，这真是太棒了！此外，你可能会创建自己的贾维斯(像托尼斯塔克)，这是我的目标。

## [](#where-to)去哪里？

所以现在的问题是我如何开始学习 ML？

首先，你需要一种编程语言来使用。可以选择 Java、Python、C#等语言。我已经习惯了 python，已经有很多 ML 社区在使用 Python，所以在剩下的旅程中，我们将使用 Python。如果你没学过 Python，我推荐 [Codecademy](https://www.codecademy.com/learn/learn-python-3) 课程。

你可能在谷歌上搜索过如何学习 ML，你可能在某处读到过需要大量的数学知识才能开始。事实并非如此。是的，学习数学会很有帮助，但有时，即使是最简单的知识也会非常有用。

因此，即使有最简单的模型和不太多的数学知识，你已经可以做这么多了！这是个好消息，不是吗？

## [](#finding-the-map)寻找地图...

现在去找教程。我在网上搜索了很多 ML 教程。我查了 Sentdex，Udacity，Google，和 Coursera。但是我发现最适合我的教程是 Kaggle。

什么是 Kaggle？摘自 Zeeshan-ul-hassan Usmani 的笔记本:

> 1.  "ka ggle is the AirBnB for data scientists."
> 2.  "ka ggle enables data scientists and other developers to participate in running machine learning competitions, writing and sharing codes, and hosting data sets."

检查一下 [Kaggle](https://www.kaggle.com/) 。

Kaggle 托管了成千上万的数据集，供数据科学家和开发人员使用。它还举办开发人员和数据科学家可以参加的竞赛。你甚至可以在一些比赛中赢得奖金($$$)。

许多人不知道的是，Kaggle 也有一个“ [Learn](https://www.kaggle.com/learn/overview) ”部分，这样你也可以开始学习关于 ML、深度学习、数据可视化和许多其他东西！

在 Kaggle 学习非常容易，因为它有这些小“笔记本”，里面包含文本、代码和代码的结果。这些小笔记本很容易解释这些主题的概念以及它们的作用，而不会让你进入想象的世界。此外，用于代码的数据很容易在笔记本的“数据”选项卡中找到，您可以下载并亲自试验。万岁！💥

Kaggle 也有一个很棒的社区，里面有专家和像我这样的新手。所以你永远不会被遗漏。

这篇文章不是由 Kaggle 发起的，我也不隶属于它。我真的很喜欢在卡格尔❤️学习

我们都有不同的学习方式。我喜欢阅读简短的文章，这些文章用尽可能少的代码简单地解释了我的特定用途，Kaggle 很容易为我做到这一点。你绝对应该去看看 Sentdex、Udacity、Google 和 Coursera，因为它们提供了很好的、全面的内容，可能也符合你的学习风格😉
