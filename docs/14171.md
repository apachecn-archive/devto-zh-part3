# 用 LambdaTest 实验验证 HTML 日期属性的跨浏览器兼容性

> 原文：<https://dev.to/lambdatestteam/verify-cross-browser-compatibility-of-html-date-property-with-lambdatest-experiments-38ca>

有 5 种主要的浏览器，其中大多数都有超过 20 个版本，仍然被人们积极地用来访问互联网。然而，每个浏览器在运行 [HTML CSS 和 JS 代码](https://www.lambdatest.com/blog/complete-guide-on-creating-browser-compatible-html-and-css/)时都有一点不同。

因此，我们已经开始了 LambdaTest 实验，并探索这种浏览器的多样性。在这篇博客中，我们将探索一个简单的 HTML 属性 date。

如果你看名字本身，那么日期是最简单和最基本的输入类型之一，然而令人惊讶的是，它不被许多主流浏览器如 Internet Explorer 和 Safari 所支持。

现在你可以在 chrome 浏览器中打开[https://www.lambdatest.com/experiment/date-support.html](https://www.lambdatest.com/experiment/date-support.html)，你可以看到，在 Chrome 浏览器中，简单的日期字段有如此多的功能。你有一个日历下拉菜单，可以用来选择日期，你有一个滚动功能，可以用来上下滚动日期、月份和年份，甚至还有一个重置功能。

仅仅一个简单的标签，你就可以获得如此多的功能。

然而在其他浏览器中，情况完全不同。

例如在 Internet Explorer 中，它将完全不同。

如果你在 windows 上，并且想在 Mac 上测试，反之亦然，你可以使用 LambdaTest 在其他操作系统上测试

所以打开 LambdaTest。输入网址。选择另一个浏览器，如 IE 或 Safari(本地计算机中没有的浏览器)。

如果你认为我为什么要关心一个像 Internet Explorer 这样令人讨厌的浏览器，那么快速检查一下事实，它仍然被 2-5%的互联网用户积极使用，相当于大约 2000-3000 万用户。因此，如果你有一个面向客户的网站，你应该关心互联网浏览器。

比如说，一旦你点燃了。

你可以看到，你在 Chrome 中看到的所有功能，如下拉菜单和滚轮等，都消失了。这将只是一个简单的输入文本字段，就像任何标准的文本输入。类似地，这在 Firefox 中也有一点不同的体验。所以，切换浏览器吧，这次用最新的火狐，windows 8.1。发射。

火狐的问题是，因为它是最流行的浏览器之一，人们认为没有必要对它进行测试。这是受欢迎的，它应该完美地工作，因为它在铬工作。大多数情况下，确实如此。然而在某些情况下，它并不总是相同的。

同样，你会发现在这个例子中，日期标签的体验与 chrome 标签完全相同，尽管下拉式用户界面略有不同。也不会有日期辊功能。所以，如果你指望这里有同样的功能，那么你就错了，我的朋友。

Safari 上的 [LambdaTest](https://www.lambdatest.com/) 也是如此。

快速切换浏览器，选择 safari 并启动。

很多时候，检查旧版本变得很困难，因为你必须降级到旧版本，这是 Mac 机的巨大痛苦。

你会发现 Safari 9 也和 ie 浏览器一样，没有下拉菜单，没有滚轮。普通文本输入框。标签根本不兼容。

所以正如你现在可能已经看到的，一个简单的标签在不同的浏览器中有完全不同的体验。这就是为什么我们需要[跨浏览器测试](https://www.lambdatest.com/)。
