# 面试时你最大的“危险信号”是什么？

> 原文:[https://dev . to/David _ j _ eddy/面试时你最大的危险信号是什么-4ec8](https://dev.to/david_j_eddy/what-is-your-biggest-red-flag-when-interviewing-4ec8)

面试是一个双向的过程(不要让任何人说不是这样)。组织需要能够“做工作”的人。人们“需要得到报酬”。除此之外，上班是一种令人沮丧的经历，也是我们一天中做得最好的事情。

这些天就业市场很紧张。据我所知，失业率不到 2%。对熟练工人的需求如此之高，以至于一些组织开始提供额外津贴，这让互联网泡沫出现前的工程师们嫉妒不已。因此，我们在采访组织时需要更加细心和勤奋。

对我来说，这是‘你在压力下表现得好吗？’根据经验，这通常意味着“我们不知道如何安排工作”，“我们没有能力推迟业务需求”，或者“需求每天都在变化”。没有一种情况是我想参与的。

你呢？你的“危险信号”是什么？
