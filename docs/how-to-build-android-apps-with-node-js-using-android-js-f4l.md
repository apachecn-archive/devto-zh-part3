# 什么是 Android JS？

Android JS 是在 GitHub 上开发和维护的开源框架。Android JS 允许使用最初为 web 应用程序开发的前端和后端组件来开发 Android 应用程序:后端的 Node.js 运行时和前端的 Android Webview。Android JS 框架可以用于采用 JavaScript、HTML 和 CSS 等前端技术的 Android 应用程序。
