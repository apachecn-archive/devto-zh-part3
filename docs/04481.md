# 我的孩子想成为程序员怎么办？

> 原文:[https://dev . to/lucagrandicelli/what-if-my-child-want-to-be-a-make-a-what-if-my-child-want-be-a-make-a-make-a-programmer-368d](https://dev.to/lucagrandicelli/what-if-my-child-wants-to-become-a-programmer-368d)

我经常看着我儿子的眼睛，看着他们完全坠入爱河。不过这并不奇怪；那是我美丽的孩子，当然我希望他一切都好:美好的生活，好朋友，真爱，可能还有一份好工作。

但是，如果我的孩子想成为一名程序员呢？

[![sada](../Images/8be8ae07edd5dab0ac1d9dfb08b68932.png)T2】](https://i.giphy.com/media/l0CRCmMBYQbL7dCmI/giphy.gif)

也许现在回答这样的问题还为时过早，因为在近 20 年内，我们的 it 世界可能会发生翻天覆地的变化。也许网络会彻底改变，但有一点我可以肯定:会有程序员。很多程序员。

正因为如此，我认为我们现在有责任为我们明天的孩子建立一个更好的 IT 世界，甚至变得更负责任，并意识到我们今天作为程序员所做的事情将对明天的社会产生影响。

然而，问题仍然存在:你会向你的孩子推荐这份工作吗？你能阻止他吗？此外，你不认为我们应该开始根据软件、工具和编程语言对未来的潜在影响而不是眼前的需求来判断它们，承认我们所做的事情具有某种社会责任吗？

只是我认识开发人员:)
