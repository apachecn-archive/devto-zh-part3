# Slack 遇到了编码之爱

> 原文:[https://dev.to/georgi/slack-meets-the-coding-love-da8](https://dev.to/georgi/slack-meets-the-coding-love-da8)

外面有喜欢编码的粉丝吗？对于那些还不知道它的人来说，它基本上是一个包含各种情况下编码相关的 gif 的网站。他们有一个很大的社区，欢迎每个人提交帖子。

在我的团队中，我们使用 Slack 进行交流，我相信你们中的一些人也这样做，我们每天都会发送大量的 gif。唯一缺少的是一个 Slack 应用程序，它与 CodingLove 集成在一起，允许我们通过搜索他们的数据库来找到正确的 GIF。迄今...

作为一个兼职项目，我开始玩 Slack 并了解他们的 API，所以我创建了一个简单的应用程序，让你搜索一个短语，打乱结果，并在你所在的频道上发布最终的图像。该应用程序已经完全通过了 Slack 的审查和批准，我期待着你们的一些初步反馈，你所要做的就是将它安装在你的工作区，并尝试一下！

这是目录中的应用-[https://slack.com/apps/AFNEWBNFN](https://slack.com/apps/AFNEWBNFN)。让我知道你的想法！我们已经在工作中大量使用它了；)
