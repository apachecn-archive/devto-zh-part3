# 如何整理一堆你想同时做的事情？

> 原文:[https://dev . to/kenystev/how-to-declutter-the-bunch-of-stuff-you-want-to-do-the-things-at-time-33 f1](https://dev.to/kenystev/how-to-declutter-the-bunch-of-stuff-you-want-to-do-at-the-same-time-33f1)

我决定从这篇文章开始我的 dev.to 之旅。从几个月前开始，我一直在想很多我想做的事情，但是话说回来，我一直在想 T2，而不是 T4。

那只是因为我的时间开始被其他事情占用了。去年，2018 年，对我来说是令人敬畏的一年，但对我来说也是改变人生的一年。

# [](#2018-journey)2018 征程

#### [](#awesome-study-year)牛逼的学习年

1.  我遇到了我大学时最亲密的朋友，他们也是同一研究领域(计算机科学)的同事，所以好多了
2.  我认识了新朋友，他们也成了好朋友
3.  毕业前几个月开始全职为一家公司做网页开发
4.  大学毕业
5.  学习了很多关于 web 开发流程和 docker 容器的新东西
6.  特别是 React.js、ES6、HTML5、CSS3 的学习和经验
7.  可以从大学里持续的工作和学习中休息一下
8.  实现了我去另一个国家度假的梦想
9.  我参加了 2018 年的 hacktoberfest 并获得了 5-PR 贡献礼物，合作伙伴是 **DigitalOcean** 、 **Github** 、 **Docker** 和 **Twilio**
10.  又开始专注于学习日语了

#### [](#life-changing-year)人生变化年

1.  进入了付账单的成年生活 xD
2.  现在不是大学而是工作占据了我大部分时间
3.  每天起得太早(*我还是不习惯*
4.  到家时太累了，所以我不能专注于我的个人项目
5.  管理我的时间的问题，制定我的时间表并坚持执行
6.  感到压力是因为有太多的事情和项目我想去做，或者成为其中的一部分，或者我想每天坚持的习惯

正如你所看到的，2018 年对我来说是伟大的一年，但毕竟我有过压力，真的有压力，因为我以为大学结束后，我会有更多的时间花在我真正的梦想和激情上，但这根本不是真的。

每天我都试图:

```
- read at least one chapter of a book before work
- do my work as front-end developer in my job
- study at least 30mins of Japanese Vocabulary (I ended up studying more than an hour)
- practice my listening skill of English by watching at least a chapter of any series
- study/practice at least 30mins of Kanji
- study/practice 1 or 2 hours of Japanese grammar
- do at least 1 hour of swimming sport 
- go to the gym / work out
- go to the church (we've meetings everyday, its name is: The Light of the World)
- work on a personal project to improve my skills with Angular
- make a personal web portfolio
- make a personal project for improving my react native skills
- make a personal project for improving my compilers knowledge
- create a personal indie game and publish it
- learn unity
- contribute to opensource projects like Rosalila Engine
- learn Haskell better
- create and publish an npm module
- practice guitar
- read a book before sleep 
```

很明显，我总是沮丧地结束一天，因为我不能在一天内完成所有的事情，我只是比别人多做一些或者什么都不做。

但后来我意识到实际问题是我试图同时做所有的事情，因此我觉得我一天中没有时间完成我想做的所有事情。

在特雷罗，我还有一大堆事情要做，但现在我决定为每一件事制定目标，并开始一个接一个地完成，或者至少每个月完成几件。

关键是从优先考虑你爱的人、你相信的人开始，但是一个接一个，不要推卸责任。

我将开始以越来越多的方式为开源项目做出贡献，从微小的改变到有意义的改变，我也意识到我通过在论坛、聊天频道、视频教程等地方写文章和帮助人们来支持开源社区。

感谢您的阅读，祝我旅途顺利！；)
