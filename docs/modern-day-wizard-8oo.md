# 现代巫师

> 原文：<https://dev.to/aewens/modern-day-wizard-8oo>

追溯到中世纪，巫师是指那些能够运用力量完成被普通人认为是魔法的行为的人。
然而，我们知道他们不是真的使用魔法，而是精于欺骗的艺术。
正因如此，“巫师”一词不仅指拥有神奇力量的人，也可以指[一个技艺高超的人](https://www.merriam-webster.com/dictionary/wizard)。

也就是说，在现代，人们喜欢将那些技术娴熟的人视为类似于巫师的东西。这可能是因为克拉克三定律中的第三定律，该定律称“任何足够先进的技术都和魔法没什么区别”。
此外，从 *[计算机程序的结构和解释](https://mitpress.mit.edu/sites/default/files/sicp/full-text/book/book-Z-H-9.html#%_chap_1)* 中有一段精彩的描述，描述了编程因素如何与技术的魔力相类比:

> 实际上，我们用咒语召唤计算机的灵魂。
> 
> 计算过程确实很像巫师对灵魂的想法。
> 
> 它看不见也摸不着。
> 
> 它根本不是由物质构成的。
> 
> 但是，很真实。
> 
> 它可以从事智力工作。
> 
> 它可以回答问题。
> 
> 它可以通过在银行付款或控制工厂的机器人手臂来影响世界。
> 
> 我们用来变戏法的程序就像一个巫师的咒语。

考虑到这一点，没有任何其他背景的简单称呼自己为巫师，会倾向于中世纪类型的巫师，而不是精通技术的那种。因此，为了更好地区分这两者，我给它们起了个名字“现代巫师”。
