# 用 JavaScript 显示祝酒词的简单方法

> 原文:[https://dev . to/river sun/an-easy-way-to-display-a-toast-with JavaScript-17gp](https://dev.to/riversun/an-easy-way-to-display-a-toast-with-javascript-17gp)

```
<!DOCTYPE html>
<html>
<body>
<script src="https://riversun.github.io/jsframe/jsframe.js"></script>
<script>
    const jsFrame = new JSFrame();
    jsFrame.showToast({
        html: 'This is a simple toast', align: 'top', duration: 2000
    });
</script>
</body>
</html> 
```
