## [](#coursera)Coursera

我是 Coursera 的粉丝已经很多年了。我已经用这个平台修了几门不同学科的课程。Coursera 是由斯坦福大学的两位教授在 2012 年推出的，作为顶尖大学最好的教育工作者可以选修的一组课程。在我看来是最好最全面的。自成立以来，它已经有了很大的发展。

2018 年去年 11 月的一天，我细读了 Coursera 提供的计算机科学和编程课程。特别是网络开发部分。我浏览了所有可用的课程和专业。专业是围绕一个主题的一组课程。除了网上其他地方的一些博客帖子，这是我对 web 开发的介绍。

[面向所有人的网页设计(网页开发和编码基础)专业](https://www.coursera.org/specializations/web-design)由五门课程组成:

*   [html 5 简介](https://www.coursera.org/learn/html?specialization=web-design)
*   [CSS3 简介](https://www.coursera.org/learn/introcss?specialization=web-design)
*   [与 JavaScript 的交互性](https://www.coursera.org/learn/javascript?specialization=web-design)
*   [采用响应式设计的高级造型](https://www.coursera.org/learn/responsivedesign?specialization=web-design)
*   [人人共享的网页设计顶点](https://www.coursera.org/learn/web-design-project)。

除了顶点工程，我已经完成并收到了所有的认证证书。下个月我打算参加考试并获得一个认证证书。这意味着我将完成整个专业，所以我也得到了证书！

Coursera 的大部分课程都是免费审核的，但是需要付费或按月订阅才能获得认证证书。有需要的学生可获得每门课程的奖学金。这就是我如何能够参加课程并获得认证证书的方法。

面向所有人的网页设计(网页开发和编码基础)专业的所有课程都来自密歇根大学，由同一位教授教授。HTML5 入门和 CSS3 入门课程是对前端 web 开发的极好介绍。我发现它们信息量大，容易理解。对于具有响应式设计的高级造型也是如此。

然而，我建议避开与 JavaScript 的交互。这位教授试图在 JavaScript 中涵盖太多的主题，而没有深入任何细节。课程太短了。关于函数的讲座只有区区五分钟。对于 JavaScript 中的大多数主要主题，也可以说是同样苍白无力的介绍。学生应该得到比这更好的。教师开始使用 DOM 解释开始的概念，但没有充分解释 DOM 是什么。这门课让我非常沮丧。学习 JavaScript 有很多更好的资源，我将提到其中的一些。

Coursera 有更多的 web 开发课程和专业，涵盖了整个系列，尤其是 JavaScript 全系列。我还没有决定是否要参加 Coursera 的网络开发课程。

## [](#udemy)击球

学习 web 开发的资源广泛多样。我目前的主心骨是我的几门课程。注意！在你购买这些课程之前，确保它们是打折的。它们的价格可能高达 200 美元，但通常以 10-20 美元的价格出售。

我已经完成了 Colt Steel 的[Web 开发者训练营](https://www.udemy.com/the-web-developer-bootcamp/)的前端组件，正在学习 Angela Yu 的[完整的 2019 Web 开发训练营](https://www.udemy.com/the-complete-web-development-bootcamp/)的后端。当我学完目前正在学习的两门 JavaScript 课程后，我将继续学习 Angela 课程的前端部分。

一些 Udemy 课程是免费的，但是你经常会发现那些仅仅触及你想学的学科的皮毛。正如我前面提到的，等到你感兴趣的课程打折时再买。我敢说 80%的时间球场都在打折。

我强烈推荐 Udemy，它对 web 开发有很好的介绍。只要你对你选择的课程有选择性。不要随便挑。重要的事情包括评级、评论和注册学生的数量。一定要阅读大量有思想有意义的评论。牢记这些因素将确保你做出明智的选择。

我链接的这两个完整的入门训练营给了你一个很好的 web 开发入门。更重要的是，他们给你一个想法，你下一步应该去哪里，你可能需要更深入地学习什么。Udemy 有几门 JavaScript 课程值得一提:

*   [现代 JavaScript Bootcamp (2019)](https://www.udemy.com/modern-javascript/) 安德鲁·米德教授

*   Anthony Alicea 的 JavaScript:理解奇怪的部分

尽管 Andrew 在现代 JavaScript 开发人员训练营中的步伐太快了(在我看来)，但总的来说，他很好地解释了事情发生的原因。课程非常全面。也就是说，几周以来，我也因为他的一些讲座中我认为不直观的方法而受挫。我的挫折应负部分责任。学习是一个过程，有时我们需要调整我们的方法。

Anthony 的《JavaScript:理解怪异的部分》虽然是几年前的，但仍然是学习 JavaScript 的一个极其有价值的来源。托尼打开引擎盖，向您展示 JavaScript 是如何工作的。这是我真正欣赏的教学，也是我反应最好的教学。和安德鲁一起上这门课是互补的，非常有帮助。

很多人不太喜欢 Udemy，但对我来说，只要我注意到选课过程中涉及的因素，这就是一个巨大的胜利。

## [](#treehouse)树屋

另一个我还没怎么用过的资源是 [Treehouse](https://teamtreehouse.com/) 。Treehouse 是一个基于订阅的学习平台，从每月 25-90 美元的任何地方，你都可以学到许多编程相关的东西。

我目前正在修两门关于算法和数据结构的课程。我不确定我会从树屋学习多少课程，因为我是从 Udemy 获得基础知识的。但是，我想探索这个机会。

注意！向你当地的图书馆查询，看看他们是否提供数字服务。有了当前的借书证，这些几乎总是免费的。你的图书馆可能会帮助你免费访问 Treehouse 甚至 Lynda(LinkedIn 现在拥有的另一个学习平台)。又一次大胜！
