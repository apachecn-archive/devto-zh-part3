## [](#introdu%C3%A7%C3%A3o)导言

个人来说，作为 2019 年写更多的目标，我将从另一篇帖子启发的帖子开始。问题的岗位是‘t0’的东西我不知道怎么 2018 年 do [丹阿布拉莫夫](https://twitter.com/dan_abramov)。

对于不知道的人来说，丹阿布拉莫夫是 Redux，Create React App，React 的共同创建者，以及许多其他项目的维护者。那家伙在 Facebook 上工作，还是社区工作的重要参考点，也就是说，那家伙很好！

只是我们觉得像他这样的人什么都知道。但这不是真的，他在[邮报](https://overreacted.io/things-i-dont-know-as-of-2018/)上很好地表现了这一点。

当我看到帖子时，我提出了阅读和润色的问题，因为我知道很多人最终会因为不知道 x 或 y 而感到不适。我想这样的职位会有助于你们思考。然后在网上，我收到了这样一个请求:

## [](#coisas-que-eu-n%C3%A3o-sei)我不知道的事情

如果要写我不知道的一切，邮局会没完没了，所以我会写一些东西，也许人们认为我知道，或者他们认为知道对于像我这样的职业来说是必不可少的，实际上也不是绝对必要的。

### [](#gram%C3%A1tica-em-ingl%C3%AAs)英语语法

> 威利，你在一家美国公司工作每天都说英语！你不懂语法是什么意思？

因为这是一个监狱，我知道如何交流，今天，我会说我说得很好，但我的学习是很多实践和生活。如果你问我`Present Perfect`和`Present Continuous`的区别，我会告诉你我在那里见过，但我不记得=p

### [](#edi%C3%A7%C3%A3o-de-v%C3%ADdeo)视频编辑

尽管我在设计我的课程时做了很多事情，包括编辑。我只知道基本的修剪不好的部分，和别人一起，加入一些小过渡，仅此而已。我不知道从哪里开始。

### [T1】Comandos Unix e Bash](#comandos-unix-e-bash)

就像丹一样，我知道`ls`、`cd`、一些不太基本的东西，比如用`grep`作日志，但我的知识却在那里。不要指望我会有那些精心设计和做得很好的剧本。

### [](#express%C3%B5es-regulares-regexp)正则表达式(Regexp)

我就是‘t0’糟透了！实际上，我讨厌！我知道它有它的用处，它救了很多次，但我只是不能让它进入我的脑海。感谢上帝我们有 [Regex 101](https://regex101.com/) 进行测试，没有这个工具，我将是一个彻底的阅读。

### [](#php)PHP

这是我接触过的第一种网络语言但我的接触是由那个自称懂得编码的老板编写的超级冈比亚密码-我...。

如果我今天需要用这种语言做任何事情，我需要从头读起，因为我不知道一个好的做法，更不用说像 Laravel 这样的框架了。

### [](#linguagens-de-baixo-n%C3%ADvel)低级语言

我在大学里做了 C/C++并且也有一些程序集，但是这些东西我每天都不使用，所以我几乎什么都不记得了。

### [](#docker)码头工人

我知道+或-有什么用，我见过有人用它。我从来没有离开设施开始学习，但那时我甚至没有学习~~【xd】~~。

### devo PS/deployed 等。

如果这是一个简单的应用程序部署到 Netlify 的生活，我知道很容易。但像 Toptal 这样的东西，有各种流程和自动化，我什么都不知道。我只是感谢那些神奇的事情发生了让一切都变得可行。

### [](#react-nativeionic-ou-qualquer-outra-coisa-mobile)反应原生/离子或任何其他移动事物

我所做的唯一一件事就是辅导，我从来没有去过，尽管这些都是有趣的事情，而且我打算今年再多看看。

## [](#conclus%C3%A3o)结论

不管怎样，就像我说的，我可以花一整天写一些我不知道的东西，但也可能变得无聊。重要的是，我不知道很多事情，但我也知道其他的事情，这些事情足够我现在的职位，也让我把我所知道的一点点传递给别人。

不知道世界上所有 1298319028 JavaScript 框架或所有类型的算法也没问题。重要的是你知道你的工作所需要的，当然，你总是想提高和学习新的东西，但一切都在你的时间。

想一次学到一百万件事是没用的，最后你会因为没有得到你想要的一切而沮丧。我对你 2019 年美好快乐的建议是你把小范围的小事和优先事项定下来，当它结束时，你可以再加一些东西。

完成几件小事时的成就感远胜于做了一半多事的感觉。祝大家 2019 年快乐，密码太多，走吧！
