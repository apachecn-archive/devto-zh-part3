# 当你陷入困境时，你会怎么做？

> 原文：<https://dev.to/dowenb/what-do-you-do-when-you-are-stuck-1e4n>

我一直在和一个 bug 做斗争，我已经努力调查了超过 24 个小时了。在这一点上，我需要休息，所以我正在考虑去健身房，如果你了解我，你会知道这是极端的。

[![Gym Workout](img/62447ef95d7401275489516c74abb3f1.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--f30kjeGI--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/uf0vb64w22wnstcgzxgs.jpg)

这让我想到了一个半严肃的问题，当你陷入困境时，你会怎么做？

如果我在某件事上耽搁了一小段时间，我会泡一杯茶。也许看几分钟关于网飞的节目，让我的大脑思考。

但是现在我又多困了一会儿——怎么办？当然答案的一部分是寻求帮助，我也是。但是我更多地考虑其他人用来解决我可能忽略的问题的策略。
