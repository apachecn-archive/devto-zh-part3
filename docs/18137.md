# 学习编码？离开教程跑步机！

> 原文：<https://dev.to/aryanjnyc/learning-to-code-get-off-the-tutorial-treadmill-c6p>

你想学习编码。你的游戏计划已经制定好了。HTML、CSS 和 JavaScript 是你首先要学习的东西。为此，你可以使用[自由代码营](//freecodecamp.org)、[代码学院](//codecademy.com)、 [YouTube](https://www.youtube.com/channel/UCERIxMohPPYmwjtHF3DdlJQ) 等等。您将同时学习教程并完成所有练习。您可以在自己喜欢的文本编辑器中跟随教程的代码。你真的在学习编码。还是你？

想象一下你决定进入拳击界。你可以从一些轻微的运动开始。你会打重包，掌握速度包。你会和他们中最好的人一起打太极拳和跳绳。然而，你真的在学习拳击吗？没有陪练，没有犯错，没有从错误中吸取教训，你会有足够的信心进入拳击场与真正的对手比赛吗？

日复一日地阅读和观看代码教程，希望在软件领域获得一席之地，就像跳绳和击打速度袋，希望成为一名伟大的战士。迟早，你必须离开那个单调的教程，进入拳击场。你*必须*开始建造*你自己的*项目。你需要亲自动手，学会在编码时不断地搜索，学会阅读文档，学会发现并消除代码中令人讨厌的错误。你看过或读过多少教你如何:

1.  谷歌一个编码问题？
2.  阅读文档？
3.  调试？

几乎没有。这是你在自己的上构建*时学到的(后面的视频教程不算)。您需要跌倒并擦伤膝盖，才能确切地了解构建一个可工作的应用程序是什么。*

# 找到对手

你需要一个与你所拥有的知识相称的项目想法。例如，如果你只知道 HTML 和 CSS，也许创建下一个侵犯隐私的社交媒体网站是一个值得一战的对手。一个更相称的对手将是一个[致敬页面](https://learn.freecodecamp.org/responsive-web-design/responsive-web-design-projects/build-a-tribute-page)或一个[产品登陆页面](https://learn.freecodecamp.org/responsive-web-design/responsive-web-design-projects/build-a-product-landing-page)。

最理想的情况是，这个项目将是你内心最亲近的东西。你会兴奋地进入拳击场，一轮又一轮，去战斗。然而，如果你需要灵感，请查看 [FreeCodeCamp](https://learn.freecodecamp.org/responsive-web-design/responsive-web-design-projects) 获取项目创意。

# 跃入擂台

是时候开始这个项目了。这本身就够吓人的了。你需要找到一个文本编辑器(试试 [Visual Studio 代码](https://code.visualstudio.com/))，[初始化一个 git 库](https://git-scm.com/docs/git-init)，安装必要的库&包，等等。您需要创建一个开发环境，您可以轻松地使用它来创建您的项目。第一次是最困难的*但是*你会很高兴你学会了这至关重要的一步。这种知识只有通过经历环境设置的痛苦才能获得。

**无耻之徒**:我创建了一个 [YouTube 视频](https://www.youtube.com/watch?v=3QngsWA9IEE)，在视频中我使用 [create-react-app](https://facebook.github.io/create-react-app/) 建立了我的开发环境。

# 从不停留下来

当你在建造的时候，你会被击倒…很多次。你会遇到无情的错误、难以理解的错误和糟糕的文档。数到 10 之前你会想一直呆在垫子上，但是你永远也不会趴下。对你来说幸运的是，只要你需要赢，这场战斗就能继续。只要你每天做你的项目，你就会让你的对手筋疲力尽并完成那个项目。此外，一路上你都有帮助。 [Twitter](https://twitter.com/aryanjabbari) 、 [/r/webdev、](https://reddit.com/r/webdev) [/r/freecodecamp](https://www.reddit.com/r/freecodecamp) 和 [FreeCodeCamp 论坛](https://www.freecodecamp.org/forum/)都有非常棒的社区，当你把这个项目带回家时，他们会非常乐意帮助你。利用这些社区。

**不要脸的塞**:如果你有关于 web 开发的问题，欢迎随时发微博给我， [@AryanJabbari](https://twitter.com/aryanjabbari) 。我经常发微博，也很乐意帮忙。

# 爬上官阶

恭喜你！你赢了你的第一场比赛。您用 HTML 和 CSS 创建了一个响应式 web 应用程序。你从 [Bootstrap](https://getbootstrap.com/) 或者[布尔玛](https://bulma.io/)那里得到了帮助。但是是时候为你的下一场比赛训练了。是时候打速度袋，跑教程跑步机了。您需要为下一场战斗做好准备:JavaScript。你需要看 JavaScript 的磁带，做练习。当你准备好了，你需要再次爬上那个环。

# 永不停止的战斗

总是有项目要构建，有框架要学习。通过交替训练(观看教程)和战斗(创建项目)来保持你的技能。你的事业一定会腾飞。

祝你好运！
