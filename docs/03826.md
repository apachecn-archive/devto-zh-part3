# 我如何跟上前端世界

> 原文：<https://dev.to/yashints/how-do-i-keep-up-with-frontend-world-2e3h>

我从事 web 开发已经有很长时间了，直到四五年前，我才不得不跟上这辆高速行驶的列车。

# 背景

有很多次，当我从某个地方听到某个短语，不管是在一次会议上，与同事交谈，还是只是在网上冲浪，我意识到，哦，我甚至不知道这个词的存在。

*有角度*、*反应*、 *Vue* 、 *RxJs* 、 *NgRx* 、 *TensorflowJs* 、 *CSS Houdini* 、 *Webpack* 、*包裹*、*大口*、*咕哝*、 *npm* 、*😵。*

那些时刻可能会很可怕，尤其是如果突然每个人都在谈论它。最后，在一个星期天的下午，我坐在阳台上，心想我不能再这样下去了，不能总是落后于正在发生的事情。

我决定和一些我认识的人交谈(面对面，在 Twitter，LinkedIn，会议和聚会上),他们大多数时候都走在游戏的前面，看看他们为了达到现在的位置都做了些什么。

那天下午，我坐在阳台上喝着温热的咖啡，从那时起，我学会了一些技巧和窍门，让你觉得自己知道的足够多，能够跟上正在发生的事情。所以我决定与你们分享我的学习(并在评论中听到你们的)。

# 1。阅读

学习最重要的部分是阅读(至少对我来说)。我总是会留一些时间阅读(稍后我会分享一些资料👇🏽)关于正在发生的事情。我读书的时候一般不会太深入某个东西，只是为了了解题目所讲的背景和故事。

你可能想知道什么时候，我知道对大多数人来说这是最关键的部分。虽然我住在郊区，但我不开车去上班。我坐公共汽车，把所有的时间都花在阅读上。我也会在睡觉前或清晨醒来后花一些时间来了解野外发生的事情。

**注意:**我通常会在社交媒体上看到文章和帖子，并使用 [Instapaper](https://www.instapaper.com/) 来跟踪我的阅读列表。它有一个 iOS 应用程序，当你从任何地方分享你的链接时，它会给你一个很好的选择。

# 5。宠物项目

许多人，包括我自己，通过实际行动学到了最好的东西。我需要通过编码来学习一些东西，不管是一项新技术还是一个库，或者只是一个新的 API。

兼职项目(又名宠物项目)是一个开发人员成长的好方法，无论是个人还是专业。他们让你走出舒适区，学习新技能，锻炼你的创造力。但是，在应付日常工作、孩子、朋友、家庭和无数其他承诺的同时，很难完成任何事情。

过去，我曾试图将兼职项目塞进我日历中其他项目的缝隙中。这充其量是令人沮丧的；最坏的情况是完全无效。我一直觉得我在欺骗我最热爱的项目。那么我们该怎么办呢？

## 从小事开始，但要坚持做下去

开始一个项目肯定很难，过一段时间再回到这个项目，当你已经在做一些完全不同的事情时，再重新开始就更难了。

所以每天做一点可以让你保持在正轨上，而不会感到无聊或疲劳。如果你每天坚持这样做，你会很容易从你停下来的地方找到它，因为它还在你的脑海里。

## 没有期限或压力

不要给自己的副业设定期限或压力。这不是一个客户约定，你不想很快失去你的兴奋。保持每周专注于一两项任务，在轻松的氛围中完成。

慢慢来，喝杯果汁(或啤酒，或咖啡😂)或者新鲜水果得心应手，让小长假的感觉战胜日常的敏捷心态。关上脸书，停止滚动你的推特，专注于你承诺自己每天要做的事情。

一旦习惯形成，压力就消失了，你开始感受到做自己热爱的事情的快乐。

## 将项目时间移动到时间

不要在一件事情上工作太久。当我们看到每天都有新的语言和技术引入时，我们应该相应地改变我们最喜欢的项目来使用不同的语言/技术。

我特别喜欢 frontend，但是尝试跟踪 security、azure、IoT、Hololens 等。在其他领域表现得更好。

我的方法是完成一个我开始着手的项目，然后用我想熟悉的新技术创建一个新项目。

## 和某人谈论这件事

这一个与你的宠物项目没有任何关系，但是帮助你发现弱点和新的挑战，因为它接触到爱问问题和学习新事物的人。

即使它是一个内部的棕色袋子会议，或演示，或你正在谈论一个外部事件，如聚会或会议，该项目可以帮助你轻松地收集内容，并演示一些即使演示神也无法阻止的事情(因为你以前已经玩够了)。

# 6。写作

有几个原因可以解释为什么写一篇文章可以帮助你学习新的东西，并把这个主题放入你的长期记忆中。

简而言之，写作可以帮助你:

*   稍微慢下来，专注于一个话题。
*   当你想出如何解释的时候，用不同的方式思考一些事情。
*   从概念上考虑问题，而不是从更狭隘或更实用的角度。
*   更清楚地研究事物，用链接记录你的研究。
*   以后更容易记住事情。
*   以后想不起事情的时候有个好地方可以搜一下。

我总是对自己这样认为(这是我从我的一个同事那里学到的)，不要为别人写作。为你自己写，并确保你在以后的某个时间点详细写下你需要知道的一切。这也会帮助其他人。

# 有纪律

根据许多哲学家的说法，自我训练就像肌肉，你训练得越多，你就越享受结果。即使你遵循了我到目前为止提到的所有要点，但没有给自己养成一个良好的日常习惯，这对你没有任何帮助。

想想有多少次你开始做一件事却半途而废，或者从未完成。无论如何，坚持一个时间表并遵守它是非常重要的。

以下是我学到的一些技巧:

*   给自己定一个目标。设定达到目标的里程碑，并定期评估你的进展。
*   不要拿自己和别人比较。总会有人比你聪明。所以尽量讲道理，考虑自己的能力。同时，不要把自己的目标定得太低，努力让自己变得比昨天更好。
*   有自我激励，为你的巨大成就表扬自己，当你做某事失败时有一点小小的惩罚。

# 永不放弃，学会放松

有时你可能会对自己的进步感到沮丧。或者简单地认为你永远无法像你周围的其他人一样跟上。

让我告诉你，当那发生的时候，你需要放松。这是筋疲力尽的众多迹象之一。你可能给自己太多压力，花费超过了身体的承受能力。

这可能是休息、度假或改变方向的好时机。永远不要害怕告诉自己你需要休息。世界永远不会停止，所以你应该停止不停地追逐它。
