# [](#wear-your-conversation-starter)戴上你的对话启动器

当你去参加一个社交活动，比如聚会或会议时，穿一些会让别人和你搭讪的衣服！考虑到工程着装通常是非正式的，由许多帽衫和 t 恤组成，穿上一件对话开场白是非常容易的。

示例:

*   具有以下任何一种特征的 T 恤衫:
    *   最喜欢的运动队或运动项目
    *   最喜欢的软件
    *   你来自哪个城市
    *   你来自哪个州
    *   你的大学
    *   你最喜欢的电影，电视节目，电脑游戏，乐队等。
    *   你所在的组织
*   具有上述任何一种特征的帽子
*   具有上述任何一项的笔记本电脑贴纸或包贴

**关键是知名度。**你希望有人能够很容易地发现你的话题。通过展示你感兴趣的东西，那些有相同或相似兴趣的人，会经常走到你面前，为你开启对话！通过这样做，你将摆脱不得不主动发起大部分对话的困境。就我个人而言，我发现当有人向我走来时，与他们交谈要比我走向他们容易得多。

### [](#look-for-conversation-starters)寻找对话的开始

另一方面，当你参加聚会或会议时，要留意那些可能穿着谈话开场白的人。如果你看到有人穿着印有你的城市旗帜、母校或最喜欢的运动队的 t 恤，走上前去，利用这一共同点开始对话。嘿，我看到你穿着一件 XYZ 的衬衫，那是你上学的地方吗？你是从那里来的吗？等等。像这样的一点共同点会让谈话变得比以“哦，今天天气真好”这样的话开始要容易得多

戴上你的对话启动器，你甚至不用说一句话就能和一个陌生人找到共同点，这将会让你更容易建立关系网和与人交往。所以，拿起你最喜欢的 t 恤，走出去，结交一些新的书呆子朋友！🤓
