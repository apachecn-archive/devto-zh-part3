# 时间拳击

> 原文:[https://dev.to/justinthekiwi/timeboxing-118i](https://dev.to/justinthekiwi/timeboxing-118i)

开发软件是那些会让你一个话题接一个话题的任务之一。有时候这很好，有时候你真的想完成工作。

时间框就是设定一个特定的时间段来完成一项任务，或者至少在这项任务上取得一些进展。人为的截止日期会制造人为的紧迫感，帮助你集中注意力，只做需要做的事情。通过设定一个停下来回顾的时间，你可以防止自己陷入死胡同太久。你可以给自己一个喘息的机会，尝试不同的任务或者寻求帮助。这也使得处理一个庞大的任务变得更加容易，只需要承诺一个小突发事件，然后你就可以离开了。
