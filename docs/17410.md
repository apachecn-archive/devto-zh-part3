*   Web 开发
*   游戏开发
*   数据科学

你决定选择哪一个取决于你的目标和兴趣。如果你没有特别的兴趣，那么 web 开发是一个不错的起点。

## 为什么要开发 web？

web 开发可能是最好的编程领域，这有几个原因:

*   Web 开发通常被认为比游戏开发和数据科学等其他领域更容易。
*   Web 开发教会你编程，就像任何其他类型的开发一样。无论你最终从事哪个领域，这都将是你职业生涯的一个良好开端。
*   Web 开发是现在的趋势。应用程序开发越来越趋向于 web 技术。我们有渐进式网络应用程序(pwa ),它变得越来越突出，并取代了许多企业的原生应用程序。也有可能用诸如 electronic、React Native 等技术制作本地应用程序。这些是跨平台的，并且使用 web 技术。简而言之，web 开发现在是一个非常好的领域。
*   Web 开发是一项伟大的事业。你可能会很快得到你的第一份前端开发工作。有些工作只需要 HTML 和 CSS，学起来相当快。

## 什么是 web 开发？

听起来，网络开发与制作网站有关。但是如上所述，web 开发中使用的技术的用途远不止于此。它们可以用来制作渐进式网络应用程序、本地应用程序、数据可视化、游戏等等。

## web 开发的组件

Web 开发(以及一般的编程)可以大致分为三个主要部分:

*   前端开发
*   后端开发
*   开发运营

### 前端开发

前端开发基本上就是一个网站长什么样，有什么交互性。

它包括 HTML、CSS 和 JavaScript。

HTML 是一个网站的结构和内容。

CSS 是网站的风格，网站看起来像什么。

JavaScript 影响网站的交互性。例如，您可以使用 JavaScript 让网站在点击按钮时做一些特殊的事情。

### 后端开发

后端开发发生在服务器上。后端的通常功能如下:

*   处理表格和付款(例如，当您在网站上购物并提交付款细节时)。
*   返回搜索结果(例如，当您在亚马逊上搜索产品时)。
*   创建帐户(例如，当您注册一个新网站时)。
*   并且还向网络浏览器提供网页。

### 开发运营

开发运营是 web 开发和其他提高开发效率的事物的“外围设备”。例如虚拟主机、源代码管理和其他东西。

### 应该从哪个入手？

在我看来，除非你对后端开发有强烈的偏好，否则你应该从前端开发开始。

开发运营并不是真正的开发，它更像是开发的外围设备，所以可以说，你不能以此开始。

留下后端或前端。您可以从其中任何一个开始，但是从前端开发开始有几个好处。

*   后端开发有很好的第三方解决方案，比如 Google Firebase。您可以使用这些来制作完整的应用程序。你甚至可以创建一个静态网站或应用程序，除了一个基本的服务器外，它甚至不需要后端来发送页面。然而，除了模板或让别人替你做，你不能真正避免前端开发。
*   前端开发比后端开发容易。HTML 和 CSS 肯定比 JavaScript 或其他编程语言更容易掌握。
*   学了前端开发之后学后端开发就容易多了。JavaScript 是一种完整的编程语言。了解任何一种编程语言都会让你的第二种语言学习起来更容易、更快。另外，如果您选择 Node 作为您的后端语言(这是一个很好的选择)，那只是您已经学过的 JavaScript。

## 如何学习 web 开发

要学任何东西，你需要做的最重要的事情就是投入时间。根据我的经验，投入时间似乎比有一个好的学习计划更重要。

但是当然，有一个好的学习计划可以大大加快这个过程。

再往下一点，我列出了我的经验中学习 web 开发的最佳途径，以及最好的资源。

但首先，我们来谈谈什么是学习 web 开发的坏方法。

### 糟糕的计划

如果你在论坛上搜索学习 web 开发的最佳方法，你会得到很多答案，告诉你“只是构建东西”。这是一件很好的事情，但是对于一个完全的初学者来说，仅仅构建东西和边做边学是不够的。

首先，作为一个初学者，你不知道你不知道什么。直到偶然遇到一件事，你才会知道它的存在，知识会有缺口。

所以你也需要从全面的课程中学习，这些课程会让你接触到你需要知道的一切。

并以此来建造东西。

### 在迭代中学习

你不会在第一次遇到时就学会所有的东西

要成为一名优秀的网站开发者，你需要知道很多事情。刚开始时，对某件事了解得很深是不够的，你必须对很多事情都略知一二。你需要有一个完整的知识基础。

这就是为什么在迭代中学习是最好的。例如，当你第一次浏览 JavaScript 时，你会学到它的基础知识。虽然您可能对一些事情的幕后工作方式有疑问，但并不是所有的问题都应该马上得到解答。如果没有扎实的基础知识，学习晦涩难懂的特性只会浪费太多时间。

最有效的学习方式是在迭代中。学习一项技术的基础知识。用它练习一段时间，然后构建一些东西。然后过一段时间(几周或几个月)再来，多学一些。下次你会学到更深一层的东西。诸如此类。

不要担心，过一段时间这感觉会是一个自然的过程。基本上，你会学到越来越多的东西，但不是一次完全学会一个主题，而是随着时间的推移，你会学到更多。

## 网络开发学习计划

### web 开发入门

我建议从课程开始:[前端大师 Web 开发完整介绍](https://frontendmasters.com/courses/web-development-v2/)

这是一个 11 小时的课程(对于 web 开发课程标准来说不算长),简要介绍了前端 web 开发的所有重要主题。

学完这门课程后，你会成为一名网络开发专家吗？不会。但本课程的好处是，它会让你接触到你需要知道的所有技术，所以当你学习下一门课程时，事情会变得更容易。你已经遇到了它们，它们会变得有意义，你将能够比其他方式更深入地学习它们。

这就是上面提到的迭代学习原则。

不要试图 100%地学习这门课程，它是作为一个快速介绍和接触，让你学习一些基础知识。

### 学习 HTML 和 CSS

前端开发的第一步是学习 HTML。这是 3 种语言中最容易学的(HTML，CSS，JavaScript)。

HTML 基本上只是使用正确的元素来做你想做的事情。大概就是这样。例如，如果你正在写一个段落，使用段落标签。如果你正在写一个标题，使用标题标签，等等。

CSS 稍微难一点，但是不算太差。用 CSS 你基本上声明了 HTML 的风格，它是如何在视觉上出现的。例如“颜色:红色”的声明使文本颜色显示为红色。总的来说，学习它的基础知识也相当容易。

#### 你需要的三种资源

*   柯尔特·斯蒂尔的网络开发者训练营
*   [MDN](https://developer.mozilla.org/) 教程
*   [免费代码营](https://www.freecodecamp.org/)

此外，永远记住谷歌任何你卡住了。谷歌是程序员最好的朋友。

[网络开发者训练营](https://www.udemy.com/the-web-developer-bootcamp/)

可能是 web 开发初学者的最佳课程。

它教你 HTML、CSS、JavaScript 的基础知识，然后继续学习后端和更多的主题。总的来说，这是一门很棒的课程，我强烈推荐它。这是我开始学习的课程，它帮助我找到了第一份工作。它也有项目，老师和助手总是在那里回答你可能有的任何问题。

[MDN](https://developer.mozilla.org/)

大概是有史以来最好的学习 web 开发的网站，尤其是 HTML。它几乎包含了你需要知道的关于 HTML、CSS、JavaScript 和大量其他东西的一切。虽然 Web Developer Bootcamp 对于初次学习和让您快速上手非常有用，但 MDN 要完整得多，会教您难以置信的细节。

作为一个警告，注意 MDN 中的细节是极端的，你不应该试图一下子学会所有的东西。第一次，我建议浏览教程，查找你遇到的任何问题。在很大程度上忽略引用。当你第二次重复学习 HTML 和 CSS 时，或者当你遇到困难时，再回来看看。

[免费代码营](https://www.freecodecamp.org/)

在这里，你将基本上练习你所学的。这不是你所需要的全部练习，但已经很不错了。这些挑战很好地强化了你所学到的东西，而且这些项目也特别好。

#### 如何浏览资源

我推荐这三个一起做。具体来说，先完成 Web 开发者训练营的 HTML 部分，然后浏览 MDN 中的 HTML 教程，最后完成自由代码训练营的 HTML 部分。

我以这种方式推荐它的原因是因为 Web Developer Bootcamp 课程通过视频进行教学，因此它很好且易于学习。之后，MDN 教程非常全面，所以他们会非常详细地教你 HTML。最后，自由代码营更多的是实践而不是学习，所以做起来也不错。有了免费代码营，开始“响应式网页设计认证”，做“基础 HTML 和 HTML5”。

完成每个资源的 HTML 部分后，重复 CSS 部分的过程。免费代码营，完成“响应式网页设计认证”。

### JavaScript

我推荐的一门 JavaScript 课程是:[现代 JavaScript 训练营](https://www.udemy.com/modern-javascript/)，作者是[安德鲁·米德](https://www.udemy.com/user/andrewmead/)

安德鲁·米德是最棒的。我上过他的一些其他课程，我可以自信地说，这些课程是我一生中在任何方面上过的最好的课程。我给他留下了五颗星的好评。我也可以自信地说，他可能是我作为一名初级程序员找到一份好工作并取得优异成绩的最大原因。我知道我听起来像个超级粉丝，但这是真的。

最后，对这三个资源重复您对 HTML 和 CSS 所做的过程，这一次从它们中执行 JavaScript 部分。

*   继续完成 Web Developer Bootcamp 课程的剩余部分，直到完成所有 JavaScript 部分。到达“后端基础”部分时停止。我们以后会回到这个话题。

## 下一步，准备工作

完成这些课程后，你会有一个非常扎实的知识水平，你也会有一些很好的实践。

接下来的步骤是通过制作更多的项目进行更多的练习，并创建一个文件夹。

这个作品集对你找到第一份工作很有帮助。事实是，雇主并不了解你或你的个人技能。他们只有你的投资组合来判断你是否有能力做这项工作。所以投资组合是必不可少的。

### 为你的作品集建立一个网站

你需要一个网站来存放你的作品集。

建立网站有很多方法:

*   静态网站(使用静态站点生成器构建，如 [Jekyll](https://jekyllrb.com/) 、 [Hugo](https://gohugo.io/) ，或其他基于 JavaScript 的站点生成器)。
*   带有自定义主题的 CMS(比如带有自定义主题的 WordPress)。
*   带有高级主题的 CMS。
*   带有页面构建器和主题构建器的 CMS(比如带有[element 的 WordPress 或者 Pro](https://elementor.com/pricing/) )。
*   使用站点构建器，如 [Wix](https://www.wix.com/) 或 [Squarespace](https://www.squarespace.com/) 。

有更多的选择，其中大多数都很好。

我的推荐:带有页面生成器和主题生成器的 WordPress 网站

我个人的建议是用 [Elementor Pro](https://elementor.com/pricing/) 页面构建器和主题构建器以及 [Astra 主题](https://wpastra.com/)来构建一个 WordPress 网站。这样做的原因是，这可能是为自己制作作品集网站最简单快捷的方法。

建立一个完全定制的网站是好的，但它可能需要很长时间。定制开发只是需要很多时间。但是有了像 [Elementor Pro](https://elementor.com/pricing/) 这样的东西，你就有现成的模板和组件来帮你了。你几乎可以在几天内拥有一个网站。

当然，如果你准备好迎接挑战，请随意创建一个定制网站。这将需要更长的时间，但结果将是为您定制的，这将是一个很好的实践。

建立一个基本的静态网站是免费的，然后你可以在 [GitHub](https://pages.github.com/) pages 或者 [Netlify](https://www.netlify.com/) 上托管(我推荐 [Netlify](https://www.netlify.com/) )。为此，你肯定想使用静态站点生成器，比如 [Jekyll](https://jekyllrb.com/) 或 [Hugo](https://gohugo.io/) 。我推荐[杰基尔](https://jekyllrb.com/)。如果没有其中的一个，构建它是可能的，但是效率要低得多，除非你不必学习如何使用静态站点生成器。

你也可以使用像 WordPress 这样的 CMS 来构建你自己的定制主题。

至于其他网站建设者，如 Wix 和 T2，我对他们没有任何经验，所以我无法评论。

在主机和域名方面，我推荐由 [Dreamhost](https://www.dreamhost.com/r.cgi?2307408/hosting/shared/) 或 [Siteground](https://www.siteground.com/go/sargalias-wp) 托管，域名来自 [Hover](https://hover.com/bjF8ZW4o) 或 [Namecheap](https://www.namecheap.com/) 。我对所有这些服务都有很好的个人体验，而且价格也很合理。
