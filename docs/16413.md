# 分享你对 2019 年发展的看法

> 原文:[https://dev . to/stichbury/share-your-views-on-development-in-2019-loh](https://dev.to/stichbury/share-your-views-on-development-in-2019-loh)

你好。我刚刚加入这个社区，并期待着贡献一些关于图形数据库和人工智能编码的想法。但那是下周的事了:o)

今天，我只是想为我不时与之合作的公司 SlashData 做一个快速的大喊(我们最近出版了一本关于如何向开发者营销的书...线索:不要光顾或炒作)。本着这一信息的精神，我保证不会在 DEV 上发布太多此类帖子，但我确实想在调查结束前分享一下。

你可能以前遇到过 SlashData。他们是一家分析公司，通过定期的大规模开发人员调查来跟踪全球软件开发人员的趋势。

回答调查的人提供研究数据，帮助顶级技术公司了解谁是开发者，他们使用什么工具，他们下一步要去哪里。

SlashData 刚刚发布了第 16 期开发者经济调查。SlashData 会向每个参与调查的人捐赠慈善基金。这个月，他们打算捐赠 2000 美元来帮助更多的孩子从事软件开发。

如果你有兴趣分享你对软件开发的看法，或者认识可能有兴趣的人，请点击这里查看调查。
