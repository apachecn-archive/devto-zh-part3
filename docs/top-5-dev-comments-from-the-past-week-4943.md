通过将默认的终端编辑器设置为 vim 来学习 vim，并为您喜欢的 IDE 下载一个 vim 键映射。你甚至可以通过下载火狐或 Chrome 的 vimium 来进一步体验。它会让你在大多数系统中走得更远。

如果你不喜欢 vim，那就学习你最喜欢的 IDE 和操作系统的快捷方式。

如果你像我一样在过去的 20 年里有着古怪的坏习惯，现在是学习触摸打字的好时机。它使打字更容易，你会更快地学会你的快捷键。

这是一个习惯性的事情，第一周感觉笨拙，但令人惊讶的是，你的肌肉记忆建立得很快。

我要补充另一个谎言:

# 工作中被欺负是不行的

这可以是随意的，也可以是公然的挑衅:

*   你和你的工作正被其他团队成员“温和地”嘲笑
*   因为你的性别/性取向/种族，你被排除在讨论之外...
*   有人告诉你，你不是因为你的性别/性取向/种族而被排除在这些讨论之外，而是因为[在此插入糟糕的借口]。
*   你受到了(口头或书面)威胁
*   一直都是大棒加胡萝卜
*   你永远做得不够
*   ...

如果你在上班时感到恶心，或者如果你在家里休息(或者更糟，睡觉)时一直想着队友的话，不要告诉自己这是正常的。或者你是个胆小鬼。你没有，你可能被有毒的人包围了。在被拖入一场会让你死在里面的艰苦战斗之前离开。
