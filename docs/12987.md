# 我去度假了，你也应该去

> 原文：<https://dev.to/ben/i-took-a-vacation-and-you-should-too-3kb7>

我并没有因为一个假期而特别疲惫，或者极度渴望，但是我上周安排了一次短途旅行，回来后我感觉自己的工作很有成效，也很清晰。这种感觉很棒。

总的来说，我觉得自己在工作和生活之间保持了良好的平衡，但我真的没有花太多时间休息。这有几个原因。我喜欢我的工作，一般来说，我并不总是喜欢假期。就我想用我的时间做什么而言，计划和旅行的开销有时对我来说似乎是一种浪费。作为一家公司的创始人，我不能说我没有一点压力来推迟休假，因为不同的利益相关者都希望我有所作为。在我的记忆中，我的单亲家庭也从未度过假。所以奇怪的是，我不知道*如何真正享受全职假期。*

总之，我花了几天时间，看起来这对我的大脑很有好处。回来后，我对重要的长期决策有了一个非常清晰的想法，并对作为开发者和总监为 [dev.to](https://dev.to/) 做贡献感到兴奋。我不在的时候，什么都没发生，网站的流量也创下了历史新高。 <sup>*作为总体趋势的函数，我们在大多数周创下历史新高。*🙃</sup>

我希望你也能找到时间去度假。这将提高你的工作效率，通常对你的健康有益。不要拖得太久。
