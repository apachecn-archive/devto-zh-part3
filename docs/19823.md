> 原文:[https://dev . to/visualizer true/how-to-decision-the-next-to-learn-in-web web app-development-131m](https://dev.to/visualizertrue/how-to-decide-what-to-learn-next-in-webwebapp-development-131m)

我目前是一名学士后学生，即将在今年春天完成计算机科学学位。最近，我一直在努力决定或寻找编程/开发中我热衷于追求的东西，直到大约一个月前我进入了 web 开发。这是我第一次有灵感，我从为自己设计和创造哪怕只是简单的东西中获得了很多乐趣。

幸运的是，JS 和 CSS 对我来说非常直观，因为它只是应用了我在攻读学位时已经学到的编程原则。但是现在我正处于进一步拓展我的知识库的下一个阶段，有太多不同的框架和工具需要我开始学习，我甚至不知道从哪里开始。我想知道你们是否经历过这种情况，对于如何决定下一步的学习道路，你们是否有任何建议。干杯！
