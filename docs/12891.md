# 不要像我一样...松鼠...

> 原文:[https://dev.to/peterwitham/dont-be-like-mesquirrel-44i1](https://dev.to/peterwitham/dont-be-like-mesquirrel-44i1)

这是一个在学习新事物时保持正轨的警示故事！

于是，我开始了盖茨比式的学习和网站创建，大概花了三到四个星期。一切都很顺利，我有了一个基本的但并不令人兴奋的模板，并且我正在迅速地将五年来的 WordPress 帖子转换成新的格式。

然后我有了一个想法，一个可以让设计变得更好更简单的想法。

正如我们英国人所说，这就是一切变得不可收拾的地方。接下来我所知道的是，我的设计完全被破坏了，我已经把我的注意力从传输内容转移到修复 CSS 和学习新的更现代的做事方法。

现在我被困在那块石头和那个地方之间。

所以不要像我一样，坚持最初的简单想法，把它作为学习新东西的借口，并坚持到底。避开路边的强光灯和糖果摊。
