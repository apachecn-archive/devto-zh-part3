# [](#how-to-make)如何制作

## [](#material)材料

-纸杯
-电机
-AAA 电池 x3

首先在杯子底部打一个洞。

[![](../Images/64187ceb2d60ae802f555253dfc28324.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--cfM26Gtt--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://storage.obniz.io/images/2def9bbac0cefa0e1534300972261)

用双面胶带将电机固定在杯子上。

[![](../Images/d884005d431e0acbd6205cc24fcf61fc.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--9nkJyRyX--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://storage.obniz.io/images/d135b02c648cca831534300978409)

[![](../Images/bbcdea3afbcd9b8d04c7922396a38179.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--leowseji--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://storage.obniz.io/images/8b3f450f45e67a601534300975237)

将电机电缆连接到 obniz io0 和 io1。

将电池放入外壳，并将电缆焊接到 obniz J1 上。

注意短路！

[![](../Images/073833e2217d054281949e10e224b6f3.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--CR9dDJ59--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://storage.obniz.io/images/5fedb297590bc1761534300967277)

最后，用另一个杯子做一个动物。
