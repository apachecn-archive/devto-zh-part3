# 儿童项目。摇一摇！动物！

> 原文：<https://dev.to/obniz_io/kids-project-shake-animal-1bp6>

# 如何制作

## 材料

-纸杯
-电机
-AAA 电池 x3

首先在杯子底部打一个洞。

[![](img/64187ceb2d60ae802f555253dfc28324.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--cfM26Gtt--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://storage.obniz.img/2def9bbac0cefa0e1534300972261)

用双面胶带将电机固定在杯子上。

[![](img/d884005d431e0acbd6205cc24fcf61fc.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--9nkJyRyX--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://storage.obniz.img/d135b02c648cca831534300978409)

[![](img/bbcdea3afbcd9b8d04c7922396a38179.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--leowseji--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://storage.obniz.img/8b3f450f45e67a601534300975237)

将电机电缆连接到 obniz io0 和 io1。

将电池放入外壳，并将电缆焊接到 obniz J1 上。

注意短路！

[![](img/073833e2217d054281949e10e224b6f3.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--CR9dDJ59--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://storage.obniz.img/5fedb297590bc1761534300967277)

最后，用另一个杯子做一个动物。

## 如何移动

运行程序，摇一摇智能手机！
