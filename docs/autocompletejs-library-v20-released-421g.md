> 简单的自动完成纯香草 Javascript 库。 **v2.0**

autoComplete.js 是一个简单的纯香草 Javascript 库，为速度、高通用性和与各种项目&系统的无缝集成而逐步设计，是为用户和开发人员设计的。

自动完成搜索应该让用户(最终用户和开发人员)的体验更快、更容易、更高效、更流畅。实际上，它不应该有任何感觉，用户应该通过自动完成功能快速找到他们想要搜索的内容。这就是为什么 autoComplete.js 基于以下核心原则进行设计和构建。

*   最终用户:

    *   简单
    *   直觉
    *   可用性
    *   速度
*   开发者:

    *   轻量级选手
    *   可定制的
    *   零依赖性
    *   无缝集成
    *   可攀登的
    *   开放源码
