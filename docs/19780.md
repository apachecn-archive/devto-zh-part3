# 我们来谈谈健身吧

> 原文:[https://dev.to/sgolovine/lets-talk-about-fitness-2j8c](https://dev.to/sgolovine/lets-talk-about-fitness-2j8c)

过去的两年是我人生中的过山车。大学毕业，搬到自己的地方，找了一份工作，又找了一份工作，开始了几个副业，生活很疯狂。但是在过去的两年里，我生活中的一部分遭受了最大的痛苦，那就是我的幸福。我说健康是因为这不仅仅是因为我的体能在下降。

整个大学期间，我都非常活跃，吃得很好，并确保每天至少去一次健身房或进行一些体育活动。当我离开大学时，一切都结束了。首先是健身房，在工作和我众多的兼职项目之间，我根本没有时间(或精力)去，所以我停止了。大约在同一时间，我的饮食也开始下降，我吃的健康食品越来越少，开始更多地依赖餐馆和快餐。最后，在节食和锻炼的同时，我开始出现心理健康问题。虽然无论如何都不严重，但对于一个从未处理过抑郁症的人来说，突然不得不处理它，我知道有些事情必须改变。

改变的问题是我缺乏纪律性。我知道我有走捷径的倾向，以简单的方式去做，而这一切都不会很容易，也没有捷径可走。我知道，如果我只是试图坚持下去，我会很快戒掉，回到我的旧习惯，所以我利用自己的弱点成为我的新优势。以下是我做的一些改变:

1.  找到了一份新工作:大学毕业后，我开始为一家离我家大约 20 英里的初创公司工作。在那里，我长时间工作，每天有 2 个多小时坐在车流中。这份新工作虽然公司规模更大，开销也更大，但它不仅减少了我的工作量，还将我的通勤时间从 1 小时减少到了 10-15 分钟。

2.  删除了我手机上的所有送餐应用:Uber Eats、Postmates 和 Dominos 都被解雇了。如果我要沉迷于垃圾食品，我会让它变得越难越好。

3.  用健康食品取代所有垃圾食品:快餐现在很难买到，但我仍然有一个装满不健康食品的储藏室，我会被吸引去当我的宵夜。我知道我不会改掉吃零食的习惯，所以我选择换掉我吃的零食。扔掉馅饼和薯条，加入健康的营养棒和水果。事实证明，在你的脸上堆上苹果或橘子在那一刻有着同样的效果，而且当你醒来时效果会更好。

4.  在我的兼职项目上休息一下:兼职项目是一个很好的学习方式，但也可能会耗费时间。我决定暂停所有新的副业项目，把全部精力放在为自己创造更好的生活方式上。

5.  去了健身房。每一个。单身。戴:这是最难的改变，但也是最值得的。我知道我不能只是说“我一周去 3 次”，因为我会找借口。每天都去在我的脑海中，每天去健身房就像起床后刷牙或上班前淋浴一样。你不质疑做这些事情，你只是去做。

自从我开始这个团已经有 6 个月了，结果已经出来了，我做得更好了。健康状况有所改善，我的饮食有所改善，我的抑郁症发作也消失了。更好的是，我现在的工作效率更高了，并且恢复了我的几个兼职项目。作为开发人员，我们经常认为我们和我们的代码一样不可战胜。代码是一回事，但身体健康是完全不同的野兽。在编写更多代码时，不要吝啬对自己身体的照顾。它会回来咬你的屁股。
