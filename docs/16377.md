**注意**:这开始了我的博客到 dev.to 的交叉发布！随着我发现 RSS 的魔力，更新的条目将会出现！

**2019 年 1 月 11 日**

你好！我是 Michael C，来自波士顿的前端开发人员，这是我博客的第一篇词条！当我开始学习 web 开发时(以及在学习过程中的很多时候)，我犯了所有可能的错误。学习软件开发可能是一项艰巨的任务，尤其是在快速发展的 web 开发领域。对我个人来说，我已经到了“未知的未知”已经消失的地步，我想尽我所能让其他人更容易学习 web 开发。

这个博客的目的是作为一个教练资源来补充大学课程，Udemy，Treehouse，无论人们最初是用什么来学习材料的。会有一些小的教程和解释，但我觉得对学习目标和实现目标的顺序有一个清晰的理解比准确地知道某人使用什么资源来学习 React 更重要。

我偶尔会写艺术和其他实践如何影响我如何接近编程，或者详细描述我正在学习的东西，但是主要的博客条目将每隔一个星期五出现。最后，我想听听你的意见！你在纠结什么？你想学什么？请在推特上或通过电子邮件联系我！

两周之后:我将分解大学 vs 编码训练营 vs 自学的利弊，以及我为什么走自学路线！
