# Ryan Dahl 对 Javascript 的贡献

> 原文:[https://dev . to/real Abbas/Ryan-Dahl-contribution-to-JavaScript-4 BCE](https://dev.to/realabbas/ryan-dahl-contribution-to-javascript-4bce)

1995 年，Javascript 被用于构建动态交互式网页。客户端的动画和交互式组件是使用 Javascript 构建的。在接下来的十年里，javascript 的使用一直保持不变，直到 2009 年，一位名叫 Ryan Dahl 的软件工程师发明了一种在服务器端运行 javascript 的新方法。他将 Node.js 引入了开发领域。

起初，人们没有意识到在服务器端运行 javascript 的魔力，他们开始忽略它，但在 2010 年晚些时候，Isaac Z. Schlueter 发明了 npm(节点包管理器),这是纯 Javascript 编写的。人们开始转向 Javascript。

Google 推出了 AngularJS，JAVASCRIPT 成了热门话题。每个人都开始使用它，许多其他 javascript 框架也像 Node.js 一样建立起来。在那之后，javascript 社区急剧增长，学习框架成为一种趋势。随着 web 开发中 Typescript、CoffeeScript 的引入，javascript 的领域从 web 浏览器扩展到了本地移动和桌面应用程序。
