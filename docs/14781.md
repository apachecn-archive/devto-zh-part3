# 开源墓地💀

> 原文:[https://dev.to/areknawo/open-source-graveyard-548 a](https://dev.to/areknawo/open-source-graveyard--548a)

自从我学会了编码，知道 GitHub 是一个让世界各地的开发者免费分享他们的工作和成果的神奇平台，我就有了一个梦想。最终做出对他人有用的东西并获得甜蜜的 1000 颗星星。这是我一段时间以来的目标。

也就是说，我已经做了 2 个开源项目，或者说只是一个。第一个是彻底的灾难，方向完全错误。第二，在 GitHub 上有大约 250 颗星，稍微好一点，但是[有一些问题](https://areknawo.com/coding-controversy)。所以，第一个肯定是在我的开源墓地结束的，第二个是在一个有争议的位置，是时候让我做第三次可能是成功的尝试了。

但是，我很想听听你的经历！你的项目有没有达到过 1000 星？你有多少？和...你的开源墓地里有什么...如果它存在的话？😁
