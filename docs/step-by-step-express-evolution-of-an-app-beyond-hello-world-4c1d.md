# 一步一步快车:超越“你好，世界”的应用程序的演变

> 原文：<https://dev.to/azure/step-by-step-express-evolution-of-an-app-beyond-hello-world-4c1d>

过去几周，我一直在做一个项目，向普林斯顿的学生展示如何构建和部署 Node.js 应用程序。

我的意图是专注于它的*构建*部分——对于学生和单独开发人员来说，部署应该尽可能的不可见和无痛。因此，我一步一步地构建了一个小小的网络应用程序，从你已经看过一千遍的“你好，世界”开始，并将其构建成更具功能性的东西。

基本上，这是我希望在开始学习 Node.js 时就拥有的关于*构建一个真正的应用*(不仅仅是“hello，world”)的指南。
