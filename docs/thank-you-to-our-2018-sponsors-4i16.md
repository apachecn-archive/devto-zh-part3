# 感谢我们 2018 年的赞助商

> 原文：<https://dev.to/devteam/thank-you-to-our-2018-sponsors-4i16>

在这个感恩节，我们想感谢 2018 年以来 DEV 社区的赞助商。

在过去的十个月里，我们与一批非常棒的公司合作。没有他们的财政支持，我们根本不可能走到这一步。如果你想购买他们的产品或服务，一定要去看看！

以下是社区赞助商的完整列表，包括他们的标语，按最初出现的顺序排列:

*   **[Nylas](https://www.nylas.com/)** — Nylas 是工程师为工程师打造的 API
*   **[三字节](https://triplebyte.com/)** —做一次编码测试，获得多份工作机会
*   **[阿尔戈利亚](https://www.algolia.com/)** — Fast &相关搜索 API，现可免费读取操作
*   **[流形](https://www.manifold.co/)** —流形是独立开发者服务的市场
*   **[数字海洋](https://www.digitalocean.com/)** —一体化云平台开发者&他们团队的最爱
*   **[熨斗学校](https://flatironschool.com/)** —学习。爱情。代码。今天就开始在线学习
*   **[Bugsnag](https://www.bugsnag.com/)** —针对 web、移动和服务器应用的强大错误监控
*   **[推送器](https://pusher.com/)** —面向 app 开发者的沟通协作 API
*   **[木材](https://timber.io/)**—Node、Ruby 和 Elixir 应用程序的最佳日志记录
*   **[Mapbox](https://www.mapbox.com/)** —开发者的位置平台。
*   ——将工程师与成长中的纽约&旧金山科技公司的工作联系起来
*   **[流](https://getstream.io/)** —用于构建可扩展的新闻提要和活动提要的 API
*   —与地球上任何人沟通的 API
*   **[Qlik 分支](https://developer.qlik.com/home)**——用 Qlik APIs 帮助开发者把数据变成他们的盟友
*   —开发人员的错误、崩溃和性能监控软件

* * *

特别感谢 DigitalOcean，它已经坚持了 8 个月，而且还在继续！这种程度的一致性是非常有影响力和重要的。

感谢 Pusher，他是我们的第一个白金赞助商，也是他让[第一次开发竞赛](https://dev.to/devteam/first-ever-dev-contest-build-a-realtime-app-with-pusher-4nhp)成为可能。

最后向尼拉斯、阿尔戈利亚和三胞胎欢呼。在我们考虑“媒体工具包”、赞助合同或发票系统之前，这些组织就已经站了出来。他们全力以赴，纯粹是出于支持这个新兴社区的愿望。
