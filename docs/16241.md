# [](#intended-audience)预期受众

虽然所有经验水平的程序员都可能会发现这个系列很有用，但我特别针对 Python 新手。但是，我假设您对编程有非常基本的了解。特别是编码主题将更加关注 *Python* 的做事方式，而不是底层的通用概念。

也就是说，如果您是一名中级 Python 开发人员，您可能仍然会发现阅读本系列很有帮助。虽然我从事 Python 工作已经将近八年了，但是直到最近几年，我才真正对其中的一些主题感兴趣。这些是我希望我曾经有过的解释！

# [](#what-you-wont-find-here)你在这里找不到什么

我在这里讨论的所有话题都要深入得多。然而，我不想把水搅浑，所以我将省略相当多的细节。一旦你对一个主题感到舒服，并且自己已经做过几次，我建议回去通读关于这个主题的[官方 Python 文档](https://docs.python.org/3/)。

# [](#a-note-on-python-versions)关于 Python 版本的一个说明

Python 2 的正式寿命即将结束，所以您应该尽快学习并开始使用 Python 3！除非另有说明，否则整个系列都面向 Python 3，偏向于 3.6 和 3.7。
