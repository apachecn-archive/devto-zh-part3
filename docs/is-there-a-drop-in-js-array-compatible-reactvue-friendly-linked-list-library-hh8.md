# JS 数组兼容 React/Vue 友好的链表库有下降吗？

朋友们好！

我有以下收藏:

```
messages = [{ ... }, { ... } ] 
```

我有时会把它画在上面，解下来，推到上面。它击中了我...两个月前...那*真的*它需要是一个链表，而不是数组；因为有时我需要根据下一条或上一条消息中的信息来呈现不同的东西。

我推迟了交换集合的底层数据结构，因为大多数错误与 messages 集合的数组无关。

然而，我终于(再次)碰到了一些错误，在这些错误中，将数组换成链表似乎是最合理的做法。

你们中有谁知道一种插入式 JavaScript 数组替换，可以很好地配合 Vue 2 的反应性？

我发现大多数插件替换没有考虑到 Vue 要求你在给一个对象添加新属性时使用`Vue.set`；大多数 LinkedList 实现都使用对象作为底层数据结构。

任何帮助都将不胜感激！
