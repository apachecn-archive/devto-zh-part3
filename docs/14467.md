# 极简主义者学习资源

> 原文：<https://dev.to/zoharadora/the-minimalists-learning-resources-113l>

你好，世界，我知道有很多文章引用了大量的免费学习资源，光是看着这些，我就感到筋疲力尽了。

我只是想分享到目前为止帮助我学习各种开发技术的*基本资源*。

在我继续之前，我有一点背景知识，我实际上有 Java 和 C++的编程经验，这就是为什么一些迎合初学者的资源对我不起作用。

在您进入这些资源之前，您可能想看一下 [Kamran Ahmed 的前端 web、后端 web 和 DevOps 开发路线图](https://github.com/kamranahmedse/developer-roadmap),我发现这对于规划学习内容非常有用(并且意识到有太多的东西需要掌握！)

**互动学习** :
[自由代码营](https://www.freecodecamp.org)
[隐型僵尸](https://cryptozombies.io)

**阅读** :
[辛辛苦苦学习 Ruby](https://learnrubythehardway.org/book/)
Flavio Copes 的文章关于: [JavaScript](https://medium.freecodecamp.org/the-complete-javascript-handbook-f26b2c71719c) ， [Vue.js](https://medium.freecodecamp.org/the-vue-handbook-a-thorough-introduction-to-vue-js-1e86835d8446) ， [Node.js](https://medium.freecodecamp.org/the-definitive-node-js-handbook-6912378afc6e) ， [Express.js](https://hackernoon.com/the-definitive-guide-to-express-the-node-js-web-application-framework-649352e2ae87) ， [React](https://medium.freecodecamp.org/the-react-handbook-b71c27b0a795)

**助力语游**:
[Y 分钟学会 X](https://learnxinyminutes.com/)

**奖金**(以及详尽的免费资源):
[电子书基金会的免费编程书籍](https://github.com/EbookFoundation/free-programming-books)

随着我学习和探索其他技术，我将不断更新这个列表。请在评论中分享对你个人有用的资源！
