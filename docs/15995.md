今天我宣布一个正在进行的新系列。本系列将教授 C 语言的套接字编程，并希望提供清晰易懂的材料，尤其是对于那些刚接触套接字并想学习的人。另外，如果你坚持到最后，我想你们都会喜欢的最终项目的奖励。

## [](#why-learn-socket-programming)为什么要学习 Socket 编程

在下一篇文章中，我将更深入地探讨什么是套接字，它提供了两个进程之间的通信方式，如果它们都在同一个网络上，那么这两个进程就可以跨越世界。您可能想学习套接字来做到这一点。还有一个原因就是单纯为了学习。增加你对周围世界的了解并在这个过程中变得更加见多识广并没有错。

## [](#prerequisites)先决条件

本系列假设其读者对套接字没有什么经验。也就是说，你仍然需要如何用 C 甚至 C++编程。
