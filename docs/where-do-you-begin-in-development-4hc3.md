### 从哪里开始？

这么多的语言要学，这么多的路要走，那么一个全新的人该从哪里开始呢？我是 Aimee，一名前端开发人员，我在 2016 年开始我的 web 开发人员工作，是一名初级 web 开发人员。我只知道 HTML，CSS 和一点点 jQuery。作为一名大三学生，有太多的东西要学，这可能有点让人不知所措，但不要放弃，保持动力，继续前进。我将向你展示一些你可以走的路。

### 前端开发者

您可以成为前端开发人员、后端开发人员或全栈开发人员，这只是三种角色。

如果你想成为一名前端开发人员，像我一样，我会从学习 HTML 和 CSS 开始，它们对网站建设都非常重要。如果你还不知道 HTML 代表超文本标记语言，CSS 代表级联样式表。有了这两者，你就可以建立静态网站。你可以把 HTML 想象成一个蛋糕，一个上面什么也没有的普通蛋糕，然后你可以添加一些 CSS，就像装饰一样，这样你的蛋糕就变得漂亮好看了。

然后是 Javascript，让它变得更加花哨，放入一些蜡烛和糖屑。Javascript 是在网页上进行交互，就像菜单栏切换或手风琴打开/关闭。在你学会了 **HTML** 和 **CSS** 之后，我会开始学习这个。也有一些你可以开始学习的库和框架，比如 **jQuery、React、Angular** 或 **Vue** 。但是我会坚持使用普通的 Javascript，在深入学习之前先学习它的基础知识。

也给自己设置一个 [**Github**](https://www.github.com/) 账户，学习 git 的基础知识，我会在另一个博客中讨论，但是学起来很简单。这只是对你的代码的版本控制，所以如果你犯了一个错误，你可以回到一个旧版本。

另一个不错的网站是 [**Netlify**](https://www.netlify.com/) ，如果是静态网站，你可以在上面免费托管网站。

给自己下载一个好的文本编辑器，我以前用的是 [**崇高文本**](https://www.sublimetext.com/) 但是我现在已经转移到 [**VSCode**](https://code.visualstudio.com/) 它们都有一些很棒的插件和主题。

### 后端开发者

所以后端开发人员是服务器端的，为此你需要了解 PHP、Java、Python、Ruby 等语言。Net，C#，C，MySQL，NodeJS 。节点允许您在服务器端使用 Javascript。因此，你可以有一个有前端和后端的应用程序，例如，你在网页上填写一个表单，它通过脚本将数据从前端发送到后端，服务器端将表单数据存储在数据库中。

### 全栈开发者

这让我想到了全栈开发人员，这基本上是一个前端开发人员和一个后端开发人员，同时了解客户端和服务器端。所以这完全取决于你的偏好和你对什么感兴趣，至于我，我喜欢前端，我喜欢让设计变得生动。
