Python 是一种语言，它以一种简单的方式允许从初级项目扩展到复杂的数据处理和服务动态网页的应用程序。但是当您增加应用程序的复杂性时，很容易引入潜在的问题和漏洞。在这次演讲中，José Manuel Ortega 将讨论 Python 代码中常见的安全问题，比如远程命令执行和 SQL 注入。当然，他会采取措施来防范这些和其他类型的漏洞。
