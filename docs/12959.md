# 在[通讯工具]上打扰别人

> 原文:[https://dev . to/nimmo/threeting-people-over-communication-tool-234 l](https://dev.to/nimmo/bothering-people-over-communication-tool-234l)

我已经远程工作了几年，我仍然发现自己在学习关于实践的东西。

我最近向一位同事暗示，我不喜欢在 Slack 上问问题来“打扰”别人。

我描述了在办公室工作是多么容易；我可以观察人们是否很忙，如果他们不忙，我可以问他们任何我需要的问题。

知道我的消息会导致通知，我就不再发送它了。我不知道那个人在做什么。我想避免打扰他们。这让我把想问别人的问题“储存”起来。

我的同事告诉我，其他人对他们自己的通知的管理不是我的问题。她是对的。

我一直在浪费自己的时间，对别人的做决定。甚至我关于在办公室工作更容易的建议也不是真的；问“有时间吗？”当面是一个更大的干扰。

也许其他人都已经自己想明白了，但是如果有人还没想明白:你的同事都是成年人了。他们可以管理自己的时间和提醒。

提出问题，继续你的一天。
