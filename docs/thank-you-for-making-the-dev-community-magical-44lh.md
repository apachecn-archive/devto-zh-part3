# 谢谢你让 DEV 社区变得如此神奇

> 原文：<https://dev.to/devteam/thank-you-for-making-the-dev-community-magical-44lh>

当我们每天在这个社区工作时，很难不充满感激之情。我们开始日复一日地让软件社区感觉更像一个社区。这项工作是教师、分享者和信徒群体赋予我们的特权。

这个项目的任何增长都不是必然的。每周都是新的先有鸡还是先有蛋的问题，还有很多没有答案的问题。我们非常依赖社区的共同信念，即我们走在正确的道路上。它帮助我们克服了许多错误、缺失的功能和不一致的体验。我们总是得到许多仙女教母的支持，这些仙女教母构成了这个奇妙的空间。

我们不是第一家尝试这类事情的公司，也不会是最后一家。但是，如果我们要继续发展，继续做新的事情，这将是我们迄今为止的相同原因

作为这个地方的建设者和管理者，总有一场小火需要被扑灭。这个社区的生命线是人们非常愿意分享、教学和变得很棒，这种事情一直在发生。就像某种魔法。
