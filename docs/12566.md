> 原文:[https://dev.to/jude_johnbosco/my-journey-in-tech-4hg6](https://dev.to/jude_johnbosco/my-journey-in-tech-4hg6)

从 2015 年末开始，我作为一名使用 Wordpress 的开发者开始了这段非常科技的旅程，有趣的是它看起来如此有趣，我觉得我已经到达了世界之巅。但事实并非如此，因为随着时间的推移，我看到人们在技术领域做了很多事情，无论是作为开发人员、技术顾问、技术博客/作家、图形和许多其他东西。但是我尽可能多地尝试用我已经知道的东西去接触人们(到目前为止，我已经很好地使用了我的 Html/Css 和 Bootstrap 和 javasript)。我创建了一个博客，让科技教育离我的所在地更近，我做到了。

我注意到我所拥有的还不够，所以我试着学习更多来发展自己和教育别人。今天，我是一名初级前端开发人员，用我赚的一点钱来降温。反应堆的最新进展
