## Django 是什么？

用 Python 写的 Django 自称是为有期限的完美主义者设计的网络框架——我不得不同意。Django 提供了如此强大的开箱即用功能，并且是基于 Python 构建的 Python 有自己的库，PyPI——你可以依赖它。很容易理解为什么 Django 是当今顶级的 Python web 框架，并且是所有编程框架的前六名之一。

## Django 框架

Django 是一个“包含电池”的框架。有一个内置的工具可以帮助你完成任何你想完成的事情。需要用户认证？姜戈掩护你。需要验证用户输入的数据吗？你猜对了。清理后需要将数据存储在数据库中？是的，可以。它甚至有内置功能，可以防止您无意中让您的项目暴露于安全漏洞——例如每个请求中都内置了跨站点伪造保护。一切都准备好了，等待你去处理。

尽管如此强大和受欢迎——或者可能正因为如此——Django 有很高的学习曲线。幸运的是，关于 Django 功能的文档非常优秀，本教程将带您越过这条曲线，进入 Django。
