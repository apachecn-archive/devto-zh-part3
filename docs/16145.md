# [](#the-idea)想法

在这个多部分系列中，我希望建立一个完整的、功能齐全的、快速的、开源的类似 Instagram 的应用程序。

基本想法是有一个 Flutter 应用程序，由 Firebase 后端提供支持，支持添加/喜欢/删除图像，类似故事的东西，性感的 UI 和酷的过渡，完全的 CI/CD 支持，测试等。

本练习的目的是学习构建具有高可伸缩性、高效率和可升级性的大型复杂应用程序。，同时也回馈社会。

我们来初始化一下 flutter app

```
 flutter create instagram
    cd instagram
    flutter run 
```

你应该在手机上运行一个 flutter 应用程序，运行 flutter 演示应用程序。
