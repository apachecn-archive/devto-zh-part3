# 2018:一年回顾

> 原文：<https://dev.to/flexdinesh/2018-year-in-review-4agf>

我们还有 12 天就要结束又一个非凡的一年了。我将在一周内写一篇**年回顾**的帖子，并贴上`#yearinreview`标签。我鼓励其他人也这样做。这会给你一个机会回顾今年给你带来了什么，并为明年设定一些现实的目标。

我会把我的帖子限制在这个范围内，但它可以是你想写的任何东西。

*   职业进步
*   设定目标
*   粉碎的目标
*   目标过期
*   学到的新东西
*   解锁新成就
*   旧东西都处理好了
*   任何让你开心的事

一切都是为了共同成长。
