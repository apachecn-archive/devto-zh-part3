# IMHO 认为有害

> 原文:[https://dev . to/arturmartsinkovskyi/imho-considered-habital-l74](https://dev.to/arturmartsinkovskyi/imho-considered-harmful-l74)

人们喜欢分享他们的观点。他们在评论、文章、演讲中这样做，甚至你现在正在阅读的文本也是写这篇文章的人的观点。可以有自己的观点，思考现实，绘制周围世界的地图，然后与他人分享你的地图。向他人表达你的想法，让他们关注某个话题，这是社会发展的一个重要因素，对我们所有人都有帮助。

# [](#onion-of-opinon-crying-up-for-help)洋葱般的指点江山，大声呼救

虽然，保护你自己和你的观点免受其他人可能的批评和严肃的考虑是不对的。我经常看到人们以 IMHO 免责声明开始他们的帖子或评论，然后是他们想与世界分享的实际材料。这在很多层面上都是错误的。

# [](#humility-considered-harmful)谦逊被认为有害

IMHO 作为思想的训练轮，对世界说-不要评判我，它只是一种观点，可能是错误的。它阻碍人们参与讨论并认真对待意见。如果有人指出你理论中的一些缺点，而你不确定你能保护和/或调整它，为什么还要开始表达这个想法呢？

在你的世界观中，对一些事情不确定是正常的，只要你明确地指出并解释你对它们的不确定性。这种解释可能是富有成效的对话的基础，并促使人们对一个问题有自己的想法。IMHO 则不同，它是一种不提出任何问题的讨论停止器，只是一个停止标志。它不能解释任何事情，它只是表明这个人很安心，避免了对他们的想法的攻击，并建立了一个(相当无助的)谦逊的避难所。

是一种懒惰的思维方式导致了[主观主义谬误](https://en.wikipedia.org/wiki/Relativist_fallacy)的滑冰。如果这仅仅是*你的观点*而不是你想证明是真实的想法，你可以跳过使其可信所需的步骤，你可以声称在这种情况下真理是相对的，因为我们谈论的是*的观点*而不是*的事实*，并且用**相对主义**的闪亮盾牌来转移任何批评。这是一根无知的魔杖，无论如何都不应该使用。一些主观比其他人更好，有一个衡量它的标准，被这个四个字母的缩写解散。一些想法比其他的更好，给每个想法一个参与奖，我们不会更接近真相和实用性。我们伤害他们。

# [](#baby-dont-do-no-harm)宝贝，不要无伤大雅

坚持你的想法，勇于犯错并修正你的假设，直到你达到一定程度的确信，让一个想法变好。但是请不要谦虚，因为这不会帮助你更接近真相。

P.S .这是我的看法，但我并不认为这是一种谦逊和准备讨论的态度。
