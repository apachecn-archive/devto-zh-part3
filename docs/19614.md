Vue CLI 已经发展成为一个用于快速 VueJS 开发的完整系统。Vue CLI 3.0 配备了非凡的工具，通过提供快速直观的项目搭建、webpack 可感知的默认值、零配置、可扩展性等等，增强了开发人员的体验。

但可能，它最用户友好的产品是通过一个漂亮的基于浏览器的图形用户界面 Vue 项目管理器来创建和管理 Vue 项目的能力。在这篇博文中，您将了解如何使用 Vue CLI 3.0 图形用户界面轻松创建、导入、管理和定制 Vue 项目。
