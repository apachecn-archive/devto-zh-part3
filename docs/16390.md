# Java 结构

> 原文：<https://dev.to/vrnsky/java-structure-4ja0>

Java 是一种静态类型化的编程语言。这意味着程序员需要声明变量的类型和名称。它制成后，我们可以使用它。

T2`int wheels = 4`

通过这个字符串，我们告诉编译器我们需要一个带名字轮的整型变量。让我们来谈谈 java 中的数据类型。我们知道存在下一种类型:

## [字节](#byte)

这种类型是 8 字节有符号整数。它可以通过使用大数组来优化速度。最小值为-128，最大值为 127

## 短

这种类型是 16 字节有符号整数。也用于优化速度和使用大数组。最小值为-256，最大值为 255。

## int

根据标准，这是 32 字节的整数。最小值 2 <sup>-32</sup> ，最大值 2 <sup>32</sup> - 1。默认情况下，用该类型声明的所有变量都设置默认值 0。

## 龙

这是 64 字节整数。在 Java 8 中支持无符号值，通过这种方式我们可能得到最大值 2( <sup>64)</sup> - 1

## 浮动

这个原语需要用浮点(单精度)表示值。它是 32 字节的值。不要将此类型用于货币值操作。

## 双精度

双精度，它是 64 字节的值。不要用于货币值运算。

## 字符

我们需要这个原语来表示一个 unicode 符号。最小值为\u0000，最大值为\ uffff。

## 布尔

这个原语只需要表示两个值——真或假。我们需要标记一些表达式。

## 字符串

除此之外，在 java 中还存在一个表示字符串的类。如果你把某个字符串放在 double quoute，并把它赋给某个字符串变量，我们得到的是带有给定字符串的字符串变量。

你写的是
`String author = "vrnsky";`

但是它里面叫
`String author = new String("vrnsky");`

还要注意标记为 final 类的类。它告诉我们这个类的每个实例都是不可变的。这很好。
