# 你应该为你的下一个项目选择 Asp.Net MVC 的五大理由

> 原文：<https://dev.to/mindtechcon/top-5-reasons-why-you-should-choose-aspnet-mvc-for-your-next-project-54ja>

绝大多数软件开发人员经常经历与他们偏好的开发框架相关的复杂性。

了解为什么 ASP.Net MVC 平台比 Web 表单更受欢迎:

**关注点分离(SoC)**

MVC 产生了一种粒度形式的编码，非常有组织和干净。从技术角度来看，当事情变得简单时，Web 应用程序发现测量它们的功能要容易得多。

**简化与客户端丰富应用的集成**

随着技术的发展，网络应用变得越来越丰富，与你的桌面应用不相上下。

**对搜索引擎友好**

搜索引擎发现抓取网址更容易。由于是无状态的，MVC 使你能够使用一个窗口同时生成几个 web 浏览器。

**证明值得要求更多控制的开发者**

每当开发一个页面时，您一定在管理原始 HTML 时遇到过问题。这种原始 HTML 是由你在微软 ASP.Net 官方网站上看到的 web 表单中的控件自动生成的。

**测试驱动开发(TDD)**

MVC 有助于频繁测试落在网站上的东西。当应用程序经历这一额外的测试层时，它们获得了一个额外的防御屏障来抵御不可预见的移动。

**结论**

一旦应用程序被分离成视图、模型和控制器，它有助于管理复杂性。ASP.Net MVC 平台不涉及基于服务器的视图状态或表单，这使得开发人员能够进一步控制应用程序的行为模式。
