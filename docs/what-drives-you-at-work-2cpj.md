# 是什么驱使你工作？

> 原文：<https://dev.to/tapresle/what-drives-you-at-work-2cpj>

### 我想全身心地投入到工作中

我这里所说的投资，实际上是指我想知道我的用户是谁，我的工作能让他们做什么？它如何真正帮助他们？在我的整个职业生涯中，我一直在开发我从来没有用过的应用程序，因为我的工作针对的用户群与我的职业道路完全不同。这并不意味着我不能为他们投入工作。知道我和我的团队直接构建的应用程序使我们的用户的工作成为可能或更有效是我想要的投资，并且有直接的用户反馈是这种投资的巨大回报。没有这个反馈环，我就没有动力去真正完成事情。

### 我更喜欢一个有趣的问题

这并不意味着我不能从实现简单的事情中找到乐趣。有时候，看起来最简单的设计和功能却隐藏着一个怪癖，需要比最初想象更多的思考和设计。我有动力让事情变得更好，无论是代码、过程还是人，这是我发现最有趣的工作。当一个团队开始陷入他们的舒适区并拒绝突破界限时，我知道是时候继续前进了。

### 我需要透明

这个其实是双重的。其中一个方面是团队内部的透明度，我们的目标是知道我们要去哪里，以及团队如何看待事情的进展。在觉得自己可以敞开心扉的团队和保守自己一举一动的团队中工作过，这两者之间有着天壤之别，即使在人们有时无法相处或意见相左的团队中，更透明的团队总是压力更小、生产率更高。第二个方面是管理结构的透明度。作为一家公司，我们将走向何方？当这方面不透明时，会降低士气，因为在团队的角色以及我们的工作如何让我们更接近公司的目标方面存在一定程度的不确定性。
