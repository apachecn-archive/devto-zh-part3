# 为什么我拒绝了我的第一份全职开发工作

> 原文：<https://dev.to/bdlb77/why-i-turned-down-my-first-ever-full-time-development-job-4hog>

我们中的大多数人经历了至少几个月的痛苦和低谷，找到了第一份开发工作，然后前往 LinkedIn，欣喜若狂地改变我们的工作状态，以炫耀我们应得的灌输和最终成为行业标准开发人员的成就。

我花了九个多月的时间在网站的求职页面上搜寻信息，撒下了一张非常广泛而可靠的求职网，希望能钓到一条鱼，不管是什么类型的鱼。我申请了许多公司，但最终都被我遗忘了，两个月后才收到一封可爱的拒绝信，信中详细描述了我的简历给人留下了深刻印象，但还不够深刻，或者我不适合他们的团队。那么，为什么在发送了 100 多份申请和几乎同样多的拒绝之后，我拒绝了第一个圣杯，第一个加入享有盛誉的独家俱乐部成为真正的专业开发人员的邀请？

在回答这个问题之前，我先简单介绍一下我作为一名 web 开发人员的经历。我于 2018 年 4 月从 Le Wagon coding bootcamp 开始，在那里我碰巧遇到了我的女朋友，她会说服我(尽管相当容易)在 2018 年 7 月收拾行李搬到柏林。然后，我开始在柏林的 Le Wagon 担任教学助理，同时像连珠炮一样，向我能找到的任何一家公司发出求职申请，这些公司发布的招聘信息都与我想要的相差甚远。在 11 月份，我碰巧从事承包和自由职业，成为一名网络开发人员，幸运的是，这让我能够支付足够的账单来维持生活。

现在回到手头的问题，为什么我会考虑拒绝我作为开发人员收到的第一份全职工作邀请呢？

就薪水和通勤而言，这份工作并不体面，但也不糟糕，**尤其是在这个时候，任何工作都像是找到了金矿**。当我就薪水进行谈判时，我要求的薪水是计算机科学毕业生的行业标准，但我遭到了反驳，因为我只是一名新兵训练营的毕业生。这对我来说很好，因为我知道不要在乎薪水。通勤时间会相当长，但这也是我愿意做出的一个牺牲，为我进入专业开发人员的领域迈出第一步。

所以，我决定在这家公司试用一天，然后去上班，看看它会是什么样子。我们一致同意，在负责合同的经理回来之前，我将首先作为一名承包商，并可以处理一个全职职位来加入团队。这一天过得相当不错，我比我预期的更开心。我还获得了 React 本地开发人员的理想职位。我最近把 React 作为一个焦点，所以在这方面这再好不过了。由于公司是一个代理机构，我也知道我会有很多机会接触几个不同的项目，我不会在我的整个任期内都停留在一个 API 上(希望如此)。我甚至有很大的自主权，和其他开发人员一样直接工作，只是完成项目经理交给我的任务。

一切看起来都像是一份漂亮的工作，放在银盘子里端给了我(不完全是黄金，因为通勤时间比我期望的要长一点)。尽管通勤和金钱，我准备接受这份工作，如果提供的是值得的。然而事实并非如此。是的……是的……你没看错，我拒绝了我的第一份开发工作，别担心，我仍然认为我做这份工作很疯狂。但是，为什么呢？因为:

### 1。师徒制

作为第一份工作，只有一年左右的经验，我最想寻求的是导师。在这家公司工作了一天后，我注意到我和所有经验丰富的开发人员受到了同样的待遇，似乎没有一个我可以纠缠的全职导师。**作为开发人员，你的第一份工作应该专注于学习和成长**。没错，我本来可以接受这份工作，自己学到很多东西，但有一个你可以向其提问的学长是很有价值的。你可以直接找到他们，让他们在几分钟内帮你解决问题，而不是花四五个小时在一个特性或一段代码上。一个有指导方针的公司表明他们关心你的成长，而不仅仅是你的产出。

### 2。自我价值

经历了第一天，我知道自己被低估了。我和其他开发人员做着同样的工作，而我挣的钱却相对较少。作为第一份工作，我早些时候确实说过，只要你能支付账单，你的工资就不重要，然而，没有导师或教育资本的较低工资向我表明，他们认为我是廉价雇员。他们似乎并不关心对我的成长或教育的任何类型的投资的细节，并期望我做和其他开发人员一样的产出。如果你的收入远低于行业标准，**我认为你应该期望得到指导和工具的补偿，以帮助你成长为一名开发人员，达到下一个水平**。对我来说，这份工作似乎并不关心这些事情。

### 3。自我成长

对于这份工作，我来回要花一个半小时。虽然这有足够的时间来听播客(我肯定是这样做的)，但这样损失的时间是值得的。作为初级开发人员，我们需要抓住每一个可以获得的机会继续学习和建设。因此，我们必须问自己的问题是:这份工作是否会花一些时间在公司内部开发个人项目或进行时间学习？如果答案是否定的，那么我们应该能够利用自己的时间来学习和成长，或者找到一家提供这种服务的公司可能会更好，或者愿意灵活地与你一起找到一个有利于你个人成长的共同解决方案。

### 结论

作为初级开发人员，我们必须将这些作为我们第一份工作的要求。我拒绝了这份工作，因为最终我被认为是一个工具，他们不打算让我变得更好。我认为一家公司雇佣初级员工有两个主要原因，1。我们是廉价的雇员和 2。他们认为你有潜力，有可能把你塑造成他们需要的样子。我更愿意为一家花时间促进我个人成长和编码知识的公司工作。一家公司想雇用我，因为我更便宜，他们告诉我，他们不是一家根据个人潜力来评估个人价值的公司，而是根据他们可以获得的利润来评估的公司。作为大三学生，我们必须了解我们的自我价值，并且知道我们拿较低的薪水是因为我们应该用学习和成长的选择来补偿，当然还有一个导师，我们可以日复一日地用他们自己曾经有过的所有天真的问题来纠缠他。开发者社区最伟大的事情之一是开放和在需要时伸出援助之手的意愿。我对这份工作的期望丝毫不减，因为这是我想要的工作氛围，也是我希望加入的公司。

我很想听听你们在寻找第一份开发人员工作时的经历。迈出第一步是非常重要的，但是你会牺牲多少，什么是牺牲太多呢？

我希望你们都喜欢！
