# [](#streaming-building-cyberpunk-future)流媒体《打造赛博朋克未来》

很长一段时间，我的一些朋友和会议发言人同事称我为“疯狂科学家”类型的人，因为我倾向于用许多技术做疯狂的事情。我没有那么疯狂，我只是热爱我们生活的这个时代。我不仅对技术着迷，而且我努力把我的手放在我感兴趣的每一个领域。我想我就是这样进入做 Web/Mobile/AR/VR 和 IoT 的。我最近也在研究人工智能和区块链，并在云计算、无服务器和 GraphQL 生态系统方面做了很多工作。

通常，当我参加会议时，我喜欢沉浸在连接这些话题的对话中，或者谈论我已经创建或计划创建的很酷的演示。

我感到与世界分享所有这些知识的冲动。我热爱教学，热爱实验。我最近开始参加训练营，比如 1 月份的 [ReasonML 训练营](https://medium.com/r/?url=https%3A%2F%2Fwww.youtube.com%2Fwatch%3Fv%3DF2rfxtoZpB0%26list%3DPLqrUy7kON1mdq7DLMqQLeTOYRR3hlHaGx)，2 月 11 日欢迎你参加我的免费 [GraphQL 训练营](https://tylermcginnis.com/free-graphql-bootcamp/)。我还开通了 3factorRadio 播客，这样我就有借口和牛逼的人聊天了😉关于 GraphQL 和无服务器生态系统以及一般的现代软件架构模式。

但我想分享的不仅仅是我所做的。我想了想我在 Twitch 上的经历，我决定简单地
