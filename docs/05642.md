> 目前我们使用 Android Studio 开发的应用程序，如 IDE 和 Java 或 Kotlin 编程语言。我们使用支持库在 Android 设备中添加各种类型的小部件和兼容性。目前有很多支持图书馆，如:**com . Android . support:support-v4，com.android.support:support-v1，**等。
