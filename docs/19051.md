# 如何从创业失败中幸存下来

> 原文:[https://dev.to/damnjan/how-to-survive-startup-crash-29l4](https://dev.to/damnjan/how-to-survive-startup-crash-29l4)

这是一个非常个性化的故事，它基于我的第一次创业失败。即使它非常具体，一些或大多数情况不适用于你或你的创业公司，我相信在读完这篇文章后，你会比我更聪明，因为至少，我希望你不会重复我的错误。

### [](#introduction-to-my-startup)我的创业介绍

我是能源交易初创公司 http://balanciq.com 的联合创始人。我们试图为大型能源贸易公司开发一个软件，通过自动化他们的手动数据输入来节省他们的时间并减少错误。但当你继续阅读时，你会发现它失败了，这里是导致崩溃的事件和环境链中的主要环节。

### 你的环境缩小了你的范围

当我在柏林、都柏林或纽约等一些著名的创业城市长大时，我没有足够的运气获得创业想法。相反，我在东欧的巴尔干地区长大。如果这个地理名称对你没有任何意义，不要担心，我会试着简单地解释我在那里遇到了哪些障碍。在巴尔干半岛，大多数人不习惯为数字产品付费的概念，这就是为什么你不能提出像 Spotify、网飞等这样的想法。简单地说，这种想法永远不会出现在你的脑海中，因为你没有接触过这个概念。这正是我遇到的问题，我开始开发我的平台，用我在周围发现的特性和规范。我对欧洲的广泛法规有所了解，但对北美、亚洲和其他可能的市场却一无所知。这就是为什么走出去看看你的世界之外的世界是如何运转的非常重要。这是你能为一些遥远地方的人们遇到的问题想出解决办法的唯一方法。

### [](#sit-and-read-first)先坐着看书

我不喜欢“照本宣科”地做每一件事，通常我认为我们应该尝试。但是发生在我身上的事情是完全可以避免的，因为我不是第一个。有些已经经历过失败的人会在他们的出版物中解释说，比如《精益创业》中的埃里克·赖斯
[![The Lean Startup](../Images/d0e447388808f2216412d19cac32dbd9.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--DdVWGWvp--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/yuyl81unw69s3nvro4pt.jpg) 
我做错了什么？首先，我构建了一个产品，然后问客户他们需要什么。这是如此可怕的错误，它花费了我这么多时间去开发一些没人会用的东西。当然，你不可能在一张纸上画了几年的草图后就开始创业，一旦你有了足够多的客户，你就可以开始创业，但是你必须和你的客户一起打造你的产品。

### 尽快找到你的第一个客户

无论花费多少，在开发过程中尝试立即吸引一些客户，并尽可能经常地向他们寻求反馈。还有，给他们免费提供一个产品。我们在获取客户方面开局不错。我们有几家公司站在我们这边，我们经常从他们那里得到反馈，但后来我们开始到处旅行，推销我们的商业理念，这使我们远离了我们真正的客户。那就悲哀了:(

### [](#quit-your-job-if-you-can)能辞职就辞职

如果你没有 100%投入，就不会有成功。如果对面有人告诉你，不要相信。如果你想快速测试你的创业，你需要把你所有的努力和力量放在你的创业上。否则，它可能需要更长的时间，如果失败，你会失去更多的时间和精力，如果你有你的常规工作并行。我确实犯了那个错误，我兼职做了一些自由职业项目，这只是推迟了我的进步。

### 没人会偷走你的想法

这是大多数创业公司都存在的恐惧。但是不要担心，不管你的想法有多好，如果你已经在努力了，请随时与他人分享。偏执狂没有帮助，但分享可能会有帮助，因为你的朋友、业务联系人或家人会给你他们的意见或想法，这可能会影响你将如何开发你的产品。我个人秘密地保留了我的想法一段时间，但只有当我向我的家人和朋友透露时，我才开始得到一些很酷的建议，并获得了一些在当时对我有帮助的伟大联系人。

### 失败的创业是重新开始的好机会

我真的不需要解释这个。如果你已经尝试过创业想法，很有可能你会很快尝试另一个，因为你只是喜欢它。
