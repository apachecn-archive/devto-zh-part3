# 新的一年和新的开始:我将加入开发团队！

> 原文：<https://dev.to/aspittel/new-years-and-new-beginnings-im-joining-the-dev-team-3dnb>

我非常非常兴奋地宣布，我将从 2019 年开始加入开发团队！🎉

成为 DEV 社区的一员对我的职业生涯和个人来说都意义重大。与社区互动并向其学习真是太棒了，我迫不及待地想成为其中更大的一部分！

教育和改进我们作为一个行业教授代码的方式是我的热情所在，我发现我已经能够在 DEV 上接触到比我亲自或在其他平台上更多的人。我真的希望能有所贡献，让它成为一个平台，让更多的人可以学习和教学。

我的头衔是软件工程师，我将主要从事前端工作，包括编写代码和管理开源贡献。由于我在加入团队之前是一名用户和贡献者，我将能够对 DEV 给出不同的观点！

作为我的角色的一部分，我还将担任开发人员倡导者，所以我将在会议上发言并鼓励社区参与！

当然，我仍然会在 DEV 上写作和创作内容！

迫不及待地想更多地参与社区活动！
