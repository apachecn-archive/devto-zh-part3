本周主题:**投资组合**

### 描述

作为一名开发人员，强烈建议你准备一个文件夹，里面有一些你正在做的项目。我说一些，因为也许不是你所有的项目都值得放入投资组合。为了给你未来的雇主或客户留下深刻印象，你应该把你最好的作品放在你的文件夹里。
