# 加入了日常编码问题？用网格整理自述文件！

> 原文：<https://dev.to/caseywebb/joined-the-daily-coding-problem-snazz-up-that-readme-with-a-grid-4a0i>

我最近加入了[日常编码问题](https://dailycodingproblem.com)在看了[这个帖子](https://dev.to/cwetanow/daily-coding-problem-1-23e0)之后。

像这样的小编码问题正合我的胃口。你知道我还喜欢什么吗？徽章。

所以，我提出， [dcp-badger](https://github.com/caseyWebb/dcp-badger) 。

[![](img/d1c2abaf29011b17dcc6163eb5c97720.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--r0ARps8A--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://apps.caseyWebb.xyz/dcp-badger.svg%3Frepo%3DcaseyWebb/dcp)

这是一个非常简单的小应用程序，它使用文件命名模式来构建上面的网格，以跟踪您的完成情况和活动，这是一个有趣的机会来尝试一些我已经好奇了一段时间的事情，比如:

*   [RegExp 命名的捕获组](https://github.com/tc39/proposal-regexp-named-groups)
*   服务器端 JSX
*   动态 SVG(受 shields.io 启发)
*   [zeit/micro](https://npmjs.org/package/micro)

所以是的。每个人的网格！+1 虚荣心。
