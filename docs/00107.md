# 一个简短的恐怖故事

> 原文:[https://dev.to/thepaulmcbride/a-short-horror-story-3af8](https://dev.to/thepaulmcbride/a-short-horror-story-3af8)

每个在科技行业工作的人都有一两个恐怖故事。犯错误和做蠢事都是人类的一部分。记住这一点，让我告诉你一些我最近做的蠢事吧！

周一，当我开始工作时，我从背包中拿出笔记本电脑，放在桌子上，同时打开盖子，插上外部显示器。这就是事情开始出错的地方。

当我将显示器插入笔记本电脑时，两个屏幕都空白了几秒钟。这是完全正常的，因为笔记本电脑试图找出最佳分辨率来显示所有内容。

现在，在周一，我计划了一个忙碌的早晨，所以为了节省一点时间，当屏幕还是空白的时候，我输入了我的密码，然后按了回车键。不幸的是，当屏幕空白时，我的笔记本电脑检测到我戴着 Apple Watch，自动解锁并重新打开了 Slack。

我刚刚在 Slack 中输入了我的密码，并发送给一个拥有 2000 名会员的频道。

呀！
