# [饭桶屎](https://twitter.com/git_shit)——屎开发者说

我想给你介绍[饭桶屎](https://twitter.com/git_shit)！这是一个 Twitter 机器人，从 GittHub 获取愤怒的提交消息。这是一个非常简单的使用 Octokit 和 Twitter API 的节点脚本。它剔除了电子邮件、回购、&网址，以保持一切匿名。它一天只运行几次，所以不会堵塞你的饲料。
