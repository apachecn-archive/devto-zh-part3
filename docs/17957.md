# 彻底的坦诚:以正确的方式反馈

彻底坦诚是一个强有力的概念，它告诉我们，要给予适当的反馈，我们需要亲自关心我们给予反馈的人，并直接挑战他们。

在这篇文章中，我想分享一个在我的工作场所产生了巨大积极影响的故事。

接下来，我将写一篇后续文章，深入探讨如何在工作和个人生活中成功运用它。

## [](#radical-candor-at-work)**工作中的激进坦诚**

我团队中的一个开发人员，让我们称他为 John，曾经在其他开发人员中有相当大的负面影响。他是团队的领导者，但具有对抗性，有争议的观点，并会吸引利益相关者加入他的“一边”，鼓励“我们对他们”的心态。

当时，我并没有作为 Scrum Master/团队教练在他的团队中工作，但是我确实想要……不，我需要做点什么，因为这影响到了所有的团队。作为一个新的 Scrum 大师，我不太确定如何管理他，尽管我和他团队中的其他开发人员关系都很好。我曾几次试图帮助他的团队，但都没有成功，所以现在我能做些什么不同呢？

我必须承认，因为他的毒性，我的一部分认为他离开公司会更好。这会让办公室里每个人的生活变得更好，当然也会让我的工作更轻松。但是等等，我是球队教练，该死的。我来指导他怎么样？我的职责是创造一个支持性的、鼓舞人心的工作环境，在这个环境中，成功只不过是存在的结果。所以，我只是坐在那里，对情况没有任何帮助。

正如我在之前提到的，这是我第一次作为全职 Scrum 大师工作，因此我一开始就犯了很多错误。在那个特别的时刻，我意识到我从来没有花时间去了解他，即使在加入团队一年多之后。我这边是个巨大的失败。

所以安排了一次与他的谈话，我们开始了第一次一对一的会谈。我们讨论了他的团队遇到的一些问题。他刚刚担任领导职务，出现了让他措手不及的情况，说了一些不该说的话，现在我们正在采取措施解决冲突。在这次谈话中，我对他谦逊的态度感到惊讶:我在团队会议上感受到的他的毒性在这次一对一的谈话中没有出现。

我继续努力。

本着彻底坦诚的态度，我使用我们的内部反馈工具给约翰留下了真诚而有力的反馈。彻底坦诚是我的公司经常谈论的一种技巧，我也开始在生活的其他领域使用。这个想法是，如果你真的关心某人和他们的成功，你应该让他们知道他们可以在哪里成长，即使这让人感觉不舒服。如果你不这样做，你就是在伤害那个人。你阻止了他们成长的机会。和他有了第一次见面，对他有了更多的了解，我所希望的就是他能成功；他的成功意味着帮助他的团队成功；他的团队的成功意味着帮助我们整个技术团队取得成功。我想他明白这一点，对反馈留下了一个大大的“谢谢”。

## [](#radical-candor-followup)**激进坦白追问**

几周后，我们共进了一次社交午餐，我们谈论了旅行计划、职业生涯、在其他国家的生活以及我们目前的工作。我们终于在更深的层次上相互了解了。

我了解了很多关于他的事情。我知道了为什么我之前认为他是一个有毒的人。当时，这个部门有一个领导问题，所以人们有点太舒服了。约翰试图让人们走出他们的舒适区。他是对抗性的，因为他试图让其他人成为更好的开发人员。我不同意他的策略，但他和团队中的其他开发人员有着同样的想法:他想从事很酷的项目，他想让人们进步，他想创建一个很棒的团队。

一些古老的教导浮现在脑海，提醒我每个人都有一个故事。每个人都有理由以自己的方式行事，除非我们了解他们，了解这些理由，否则我们不应该认为自己知道到底发生了什么。

快进到今天，约翰和我在工作中关系很好。我不认为他的态度是有害的，他明白我的目标是看到他和他的团队取得成功。

我不是说他行为的改变是我造成的。我要说的是，激进的坦诚帮助我了解了他，这让我对他的看法完全改变了，帮助我理解了为什么他在那段时间如此具有对抗性，以及他实际上想做什么。现在，我们都更好地准备支持我们团队的成长。
