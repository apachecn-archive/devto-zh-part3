# 你们都是超人吗？

> 原文：<https://dev.to/shahriarsiraj/are-you-all-superhuman-4l2n>

你好，

最近我参加了几个关于 Web 开发人员角色的面试。虽然我有两年多的 PHP 经验，但不幸的是，我不能回答一些简单的问题。我知道，如果我能进入我的电脑并编写代码，我就能正确回答这些问题。

在我目前或以前的工作中，每当我需要查找一些语法时，我只需在谷歌上搜索，就可以完成语法或适当的功能。但是我从来没有试图把它们都背下来。

根据我以前面试的经验，现在我似乎必须记住所有关于 PHP 或任何我想使用的语言的知识。

是我做错了什么吗？或者你是怎么把所有东西都背下来的？
