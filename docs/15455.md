# 为什么我还在用 cakephp

> 原文：<https://dev.to/vjnvisakh/why-do-i-still-use-cakephp-24n5>

我已经使用 cakephp 一年多了，我喜欢使用它。以下是我发现的一些最好的东西。

1.  **惯例**:虽然不是强制使用，但是我找不到不使用的理由。一年前看起来如此愚蠢的事情，是我最近接触的每个框架中的必备内容。它让你的生活变得如此简单。

2.  简单设置:我已经用这个宝贝在一个晚上设置了带有管理面板的动态网站。这一切都归功于它的 crud 脚手架和模板。

3.  **路由**:尽管这被很多人使用，但感觉很好，每次都能得到友好的网址。

4.  **PHP 的强大** : PHP 是一门非常强大的语言。当然，市场上有很多新的搜索引擎，但是有这么大的社区和快速的搜索结果是上帝的恩赐。P.S .我不反对 JS 框架。只是觉得还没必要用。

5.  **调试**:哦，日志文件总是会保存你的。

6.  ORM :另一种帮助你同时运行多个查询的好方法。

7.  **插件**:很多精彩的片段。甚至一些最大的公司也提供他们产品的 cakephp sdks。

我还没有感受到 PHP 7 的威力，它号称比旧版本至少快 2 倍。手指交叉。
