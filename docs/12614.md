这里用一点 SQL，那里用 Groovy 做一点负载测试...一些有趣的 JavaScript 应用程序，但从来没有真正深入到软件工程或代码职业中。现在一切都变了。在我的新角色中，我将使用 [Cypress](https://www.cypress.io/) 自动化我们的测试用例，我必须说，我很兴奋。

因此，一旦我从一个业余爱好者的理解转向一些更实用的东西，我的许多未来计划都将成为可能。尽管我起步晚，但我可能永远不会成为一名伟大的程序员，但这不是我的目标。我的目标是扩大我作为团队测试员的作用，并打开通往更大的实验和学习的大门。
