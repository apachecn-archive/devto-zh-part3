# 七年后的今天，我离职了

> 原文:[https://dev.to/turnerj/i-left-my-job-today-f1a](https://dev.to/turnerj/i-left-my-job-today-f1a)

今天是我过去 7 年工作的最后一天。这一天已经过了很长时间，但不是因为这份工作不好，而是因为我有一个很长时间的宏伟计划。

这篇文章的部分灵感来自 DeChamp 的文章 [25 年的编码，而我才刚刚开始](https://dev.to/dechamp/25-years-of-coding-and-im-just-beginning-442n)。

## [](#in-the-beginning)开始

我从小就对电脑着迷。我记得在幼儿园时，有一台 Windows 95 机器在运行一些游戏，我就被它吸引住了——甚至可能从其他孩子那里抢走了一点。

在小学早期，我正在炫耀我惊人的技能...更改桌面背景。你应该看到过其他同学眼中的惊愕！

随着我的技能变得更好，我更清楚地定义了我长大后想做什么。我对我为学校做的作业没有多少感情价值，但有一个“你长大后想做什么”类型的作业，我很高兴我妈妈保留了下来。

请记住，我写这篇文章的时候才 11 岁，我不会拼写来拯救我的生命。

[![The letter of what I wanted to be when I grew up](../Images/591e49bfa854b01bb79f7c5c4d7e46a9.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--Tl-TD52v--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/8reemupsv4zsauleyj1p.jpg)

现在，我还没有那封信中提到的一半的学位，但是我想表达的核心意思并没有改变。我确实想成为一名计算机硬件开发人员、计算机软件开发人员和公司创始人/所有者。

到我上高中的时候，我懂一点 JavaScript，并且正在向我的一个朋友学习 PHP 和 C#。没有这位朋友，我就不会成为今天的程序员，为此我真诚地感谢他们。

由于对高中的进步速度不满意，我离开了学校，开始了一门特殊的课程，这门课程使我比其他方式更早地进入了大学。

拿到学位，做 Apex/PHP/。NET/SysOps/anywhere-the-company-need me-to-doing for the next 7 年，今天是我工作的最后一天。

## [](#starting-something-new)开始新的事物

为自己工作既有一定的自由感，也有一定程度的未来不确定性。最大的不确定性之一可能是资金的流入，这取决于你努力为自己挣多少钱，你可能会有几个月的低收入或无收入。

我个人正在拓展业务，制造一款 SaaS 产品，尽管我所有的开发时间都是“像啤酒一样免费”，但当你需要担心法律成本或第三方服务成本时，事情会变得更加复杂。你可能还没挣到一分钱就已经损失了几千美元。

多年来，我可能一直在兼职经营一家小企业，但摆在我面前的这一切仍充满未知。我还不知道我的目标受众是否会接受我的产品(尽管迄今为止我交谈过的人似乎都喜欢它)。我不知道我是否在代码库中犯了一些可怕的错误(错误会发生，但你不希望出现第一天失败的情况)。

我可能不是实现我的愿景的最佳人选，但我认为如果我尝试了，那就是最好的。处在一个你可以做你想做的事情的位置上，却让机会溜走，这可能会伴随你一生。

虽然我有很大的希望在努力中获得成功，但有时你需要冒险去得到饼干...
