# 直运到底是什么，怎么运作的？

> 原文:[https://dev . to/moderndaytech/到底什么是 drop shipping-and-how-it-work-3p 71](https://dev.to/moderndaytech/what-exactly-is-dropshipping-and-how-does-it-work-3p71)

直运是一种零售履行方式，商店不会库存销售的产品。相反，当商店销售产品时，它从第三方购买商品，并直接运送给顾客。结果，商家永远看不到或摸不到产品。

直运和标准零售模式的最大区别在于，销售商家没有库存或拥有存货。相反，商家根据需要从第三方(通常是批发商或制造商)购买库存来完成订单。
