# 创造性计算:为什么孩子们应该学习逻辑编程

> 原文:[https://dev . to/lechenco/creative-computing-why-kids-should-learn-logical-programming-4a 71](https://dev.to/lechenco/creative-computing-why-kids-should-learn-logical-programming-4a71)

在过去的几年里，我们已经进入了一个人工智能和自动化的时代。这种未来/现在的形象一直是讨论和恐惧的目标，因为大量服务的自动化将使许多工人阶级不复存在。你可以在[机器人会抢走我的工作吗？](https://willrobotstakemyjob.com/)网站。

然而，如果工作会消失，我们只需要创造更多的工作，对不对？计算机系统会处理最简单和重复的领域，所以我们只需要转移到更复杂和创造性的领域。因为，对于大多数人工智能进化来说，它永远不会取代我们的人性。

但是，为了实现这一点，我们必须让我们的孩子为这样一个未来做好准备，在这个未来，他们的创造力和社交技能将比他们从事重复性和大规模任务的技能更需要(那是计算机的领域)。一种方法是用像 [Scratch](https://scratch.mit.edu/) 这样的工具来教我们的孩子逻辑编程。这样，孩子们能够用多种方式解决问题，在一个有趣的练习中运用他们的逻辑技能。
