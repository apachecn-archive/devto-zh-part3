# 蒙得维的亚的第一个 JS 工作坊！

> 原文:[https://dev . to/juanigalan 91/first-workshop-js-in-montevidea-4a 58](https://dev.to/juanigalan91/first-workshop-js-in-montevideo-4a58)

近年来，在自由市场，我们已经开始在所有前端使用新技术。这是一个漫长的过程，但现在我们有了一个前端技术堆栈，它包含以下组件:

*   节点 JS 作为我们前端应用程序的服务器
*   React JS 用于我们的客户端代码
*   网络包

随着这一新的采用，我们的工作和发展方式真的发生了变化。我们大幅降低了开发、版本生成和部署时间，以及安装和测试时间。对于我们所有在自由市场开发前端的人来说，这是一个巨大的成功！

由于这个原因，我们认为这是一个在乌拉圭社区推广使用这些前端技术的好机会，这也是我们在蒙得维的亚组织首批 JS 研讨会之一的原因。

## [](#from-scracht)来自 scracht！

我们希望开始，这样任何对前端真正感兴趣的人都可以加入这个活动，并了解一些这些技术。考虑到这一点，我们决定举办 React JS 研讨会。

React 是脸书开发的一个库，面向组件的开发。它寻求独立地创建视觉单元，以便以后这些组件可以一起工作来创建更复杂的应用程序。这种方法对我们来说非常重要，因为我们在开发前端时的目标之一是 Mercado Libre 在整个用户体验中保持视觉一致性，而使用 React JS，我们可以非常简单快速地共享不同的组件来保持这种一致性。

有了这些，我们决定与 Martin Varela 一起组织一次活动，让社区能够更接近并了解一些这种技术，拥有基本的概念，并能够在他们的公司或个人项目中推广使用。

## [](#the-event)事件

这个工作坊于 5 月 26 日举行，名额很快售罄。大约有 40 人来到了 WTC 的自由市场发展中心。

在研讨会期间，我们开展了一项将理论与实践相结合的活动。我们的想法是举办一个研讨会，在这里你可以快速应用所看到的概念，以增加对它们的理解。采用这种方法，我们设置了一个实践练习，每个人都必须制作一个自由市场产品的列表和搜索引擎，我们在每个阶段都使用了实践部分的概念。

与此同时，我们制作了一份[实用/理论材料](https://workshops-js-montevideo.gitbook.io/react/)，作为执行这些练习的指南，并允许人们以更详细的方式通过代码练习参与研讨会。这些材料让他们可以看到例子，并有一个可以返回的知识库。我们使用了 GitBook 这个奇妙的工具来制作这个素材。
