### [](#today-marks-the-first-day-of-my-take-on-100daysofcode)🎉今天是我参加#100DaysOfCode 的第一天！！！🎉

如果你没听说过，#100DaysOfCode 挑战程序员每天至少花 1 小时编码 **100 天**！规则包括记录你的进展，你可以找到一个大型的在线社区，有新的和有经验的程序员参加这个挑战。

参与是帮助保持日常编程习惯的一种引人入胜的方式。如果你陷入困境，有一个庞大的支持社区将帮助你找到力量，即使在你不想尝试在 vim 中导航的日子里。你可以在这里找到更多关于挑战[的信息。](https://www.100daysofcode.com/about/)

# [](#why-am-i-doing-this)我为什么要做 THIS❓

当我第一次听到这个挑战的时候(差不多 5 个小时前)，我立刻知道这是我想做的事情。像你们中的许多人一样，我很快原谅了自己过于频繁的懈怠，所以我想是时候振作起来了。我已经准备好开始挑战自己的极限，我很好奇我是否有能力为 100 美元做出承诺。连续的。几天。(稍等...那么为什么会写# ***50*** DaysOfCode 呢？🤔继续读！)那么，我打算通过这样做来实现什么呢？

# [](#my-goals)我的目标☑️

1.  完成这项挑战。每天致力于某事本身就是一项壮举！💪💪
2.  *造东西！*49 天后，当我回顾这次挑战时，我希望有成就感，并有所表现。我希望一点一点地建立我的势头，并有一个项目启动和运行(不一定要完成)当我完成#50DaysOfCode。我已经有一些想法了！
3.  找到我的禅。我发现自己很难集中精力去完成工作。这就是为什么我的目标之一是学习如何平静我的心灵，并找到适合我的方法。也许这意味着创建完美的 Spotify 播放列表🎶或者在自习室找个最舒服的地方，我们就知道了。
4.  *👏格林（姓氏）；绿色的👏向上👏那👏开源代码库👏轮廓👏。*这主要是我在这里向朋友们炫耀的。😜

# [](#my-rules)我的规则📜

这个挑战的伟大之处在于，它是为了你自己的成长，也是你选择开始的你自己的冒险。也就是说，我将根据我的需求/愿望来设计这个挑战，并给出我自己的定义。

1.  *#**50**DaysOfCode。我想尽快开始这项挑战，但问题是它会落在我的下一学期。除了选修 6 门课程和找一份暑期工作，我还希望参加其他课外活动(包括萨尔萨舞💃)!由于我不确定自己的承诺水平，我决定承诺 50 天，而不是💯。*

2.  我个人的故事。挑战赛建议在 twitter 上与参加挑战赛的其他人联系。我没有推特，所以我来了。严肃地说，我是为自己参加这个挑战的，所以我将为他们记录这个挑战。这是一个让我做笔记和反思自己进步的空间。

    如果你想继续下去，你可以在这里找到我参与的证据。注意:我不是一个强有力的作家，所以有些日子可能只是要点或混乱的想法。

3.  *骗天。*我允许作弊日(因为我没有完全的信心在没有安全网的情况下完成这件事)。更重要的是，这给了我必要的休息时间，同时也让我保持在正轨上。#自我关怀#自我关爱🤟

    然而，我不能随心所欲地拿走它们。我给自己分配了越来越不愉快的“惩罚”。我第一次错过一天，我需要在健身房多呆一个小时。接下来的一个在那之后，我必须一个小时不使用手机(相信我，这对于我这样的千禧一代来说几乎是不可能的😩).下一次需要帮助我的妈妈花园一个小时(严重恐怖)。如果任何人有其他惩罚的想法，请在下面评论！

4.  *每日目标。*我的生活需要组织！！每个人都知道和朋友聚会最难的部分是决定去哪里吃饭。所以每天早上，我都会完成那个进步的第一步，给自己写下一个每天的目标。

5.  *#？HoursOfCode。*最后但同样重要的是，我的最后一条规则是，我需要记录我在编码上花了多少时间。这只是为了让我回顾过去，衡量我取得了多大的成就。

    这包括工作或课堂作业吗？这由我决定！你编码得越多，你成长得越多——任何一点学习都是有价值的！如果我觉得我做的工作有意义，我会把它包括在这个挑战中。但是如果我觉得我每天工作 8 小时编码是一种欺骗，我会把它去掉。😛

# 我准备好了吗😶

起初，我怀疑自己是否能遵守#50DaysOfCode，但现在我已经付出了这么多努力来写下这些，我觉得我应该完成这个挑战。我已经为接下来的 49 天做好了准备，T2 兴奋不已，所以让我们一起来吃面包吧！！🍞

25 天后，我会有一个中点检查，看看我的进展如何，然后我们可以检查我的感觉。😬我是否承诺每天至少花一个小时写代码(记住我还没有开始上学，所以我没有任何借口)？我是在朝着我的目标前进还是需要重新评估它们？我的 Github 贡献看起来怎么样？
