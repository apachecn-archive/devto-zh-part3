# 主要开发商

> 原文:[https://dev.to/eduardsi/the-principal-developer-26ab](https://dev.to/eduardsi/the-principal-developer-26ab)

今天我和我的前同事亚历克斯通了电话。Alex 在一家快速发展的国际组织中担任软件开发人员。

我:“事情怎么样了？”

亚历克斯:“我的公司正在寻找一个主要的开发商。”

我:“你申请那个职位了吗？”

亚历克斯:“我申请了，但是我的申请被拒绝了。CTO 认为我不能胜任那个角色。他正在寻找更好的候选人。”

我:“你觉得你够好吗？”

亚历克斯:“当然！七年多来，我一直在写代码，解决错综复杂的技术问题。我编写干净的代码，主动修复技术债务，像忍者一样完成 JIRA 的任务。我比其他队友更快更好”。

我:“我猜这就是你不合格的确切原因。”
