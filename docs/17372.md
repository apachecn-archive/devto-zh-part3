## [](#-and-im-only-on-day-003100)...而我只是在第[003/100]天！

第一次听说 *#100DaysOfCoding* 挑战赛是在 2014 年。我读过一篇关于 Jennifer Dewald 的文章，讲述了她如何通过连续 180 天每天制作一个[网站来自学编程。她的成就给我留下了难以置信的印象，从那以后，做这样的挑战就成了我的待办事项。](https://jenniferdewalt.com/)

但是我还没准备好...直到现在。在我决定是时候接受挑战之前，我需要阅读由[瑞秋](https://dev.to/rachelzzhu)写的另一篇鼓舞人心的[博客文章](https://dev.to/rachelzzhu/my-50daysofcode-challenge-2e13)。

# [](#why-am-i-doing-it)我为什么要这么做？

我一直很钦佩那些在办公室呆了一天后回到家，坐下来从事他们的业余项目的软件开发人员。很明显他们热爱编码，对自己的能力很有信心，他们把编码当成一种消遣。这些品质也是他们在工作中表现出色的原因。

那从来都不是我。我很有动力，很兴奋，我喜欢编程，但是当我应该坐下来的时候，我怀疑自己，最终看了《网飞》。原因是我想避免处理和克服自我毁灭的想法:

*   *恐惧和自我怀疑*:“我做不了这个项目。我会失败，一旦失败，我将一事无成。最好根本不要开始，以保持我的选择余地！”
*   *懒惰*:“这很辛苦，不是吗？我累了，应该放松一下。”
*   完美主义:“如果我做这个挑战，它必须是完美的。如果我每天没有取得令人印象深刻的进步，那就不算数。你知道你会失败，所以不要尝试。”

我知道这是我自己筑起的一堵墙，我可以推倒它，让天空成为我的极限！

(旁注:我注意到这些想法和那些让我减肥失败的想法是多么相似。恐惧甚至会阻止我开始，懒惰会让我呆在沙发上，远离健身房，完美主义会导致非黑即白的思维，这反过来使我的体重像溜溜球一样反弹，直到我筋疲力尽地放弃。剧透:我在 2018 年克服了这种行为，成功减掉了 16 磅，并轻松保持了体重💪)

# [](#my-goals)我的目标

*   提高我的 Python 和 C++的流畅性。
*   练习数据结构和算法。
*   打破我头脑中的障碍，增长我的自信。
*   让坐下来编码变得容易，并养成比起网飞更喜欢编码的习惯。
*   实施项目，贡献开源。
*   在 [Codesignal](https://codesignal.com/) 中获得这些徽章，并更新我的 Github 贡献日历！💪

# [](#my-rules)我的规则

*   每天至少编码一小时。
*   在一周中放轻松:在 [Codesignal](https://codesignal.com/) 中处理小的编码挑战。
*   周末做更大的项目和开源贡献。
*   把我学到的东西总结在一篇博文里，每周一篇，周末写。
*   我可以通过编码额外的时间来赚取“小丑”。当我想去周末旅行或野营时，这些会派上用场，我知道在那些日子里我不会写代码。
*   慢慢来。每天一小步比尝试太多、太快、潜在的精疲力竭和放弃会让我走得更远。我会自然地变得更好更快，在我知道之前，我会在每天的基础上大步前进。

# 到目前为止我学到了什么

挑战才进行了三天，但我已经学到了很多！

每当我在 Codesignal 中完成一个新的编码任务时，我都会查看其他人的解决方案，并将他们的代码与我的进行比较。通过这种方式，我学会了不同的思考方式，而且往往能更好地实施解决方案。

到目前为止，我已经了解了:

*   Python 2 和 Python 3 之间的差异，例如:
    *   虽然`/`在 Python 2 中是整数点除法，但在 Python 3 中是浮点除法。为了在 Python 3 中得到整数除法，你必须使用`//`。
*   您可以使用[扩展切片](https://docs.python.org/release/2.3.5/whatsnew/section-slices.html)反转列表，例如`myList[::-1]`
*   [Lambdas](http://book.pythontips.com/en/latest/lambdas.html) 是匿名函数。使用它们的语法是`lambda argument: manipulate(argument)`，例如`add = lambda x, y: x+y`
*   [列表理解](https://docs.python.org/3/tutorial/datastructures.html#list-comprehensions)和[字典理解](https://docs.python.org/3/tutorial/datastructures.html#dictionaries)
*   [`map`](https://docs.python.org/3/library/functions.html#map) 功能

此外，我确实对自己的能力更有信心了，自我怀疑之墙正在迅速倒塌。我需要说服自己坐下来编码的时间越来越少，一旦我开始了，我就不想停下来！
