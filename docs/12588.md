# 如何开始你的第一个数据科学项目

> 原文：<https://dev.to/stevenbruno/how-to-start-your-first-data-science-project-4i8n>

近年来，数据科学吸引了大量研究人员、有抱负的专业人士和爱好者。虽然许多工具和技术已经存在多年，如果不是几十年的话，但该行业在过去的 5 年里才真正起飞。良好的公众形象、不断增长的技术能力和高收入的前景结合在一起，形成了这个新兴领域的完美配方。

太空是如此的诱人，以至于连我都决定去试水。在接下来的几个月里，我将使用 python 创建我的第一个数据科学项目，作为我参与[芝加哥 Python 用户组](https://www.chipy.org/)导师计划的一部分。

如果你处在我的位置，你可能想知道如何开始。在比我聪明得多的人的帮助下，我精心制作了一个 5 步序列来开始。事不宜迟:

### **第 1 步** |重新考虑你的意图

人们很容易被“数据科学家”这个角色的浮华和魅力所吸引。然而，对于 100 个不同的雇主来说，这个头衔本身可能意味着 100 种不同的东西。如果你是一名有抱负的专业数据科学家，要知道这个头衔正遭遇一点身份危机。假设你已经做了调查，不仅仅是为了诱人的报酬，而且仍然感兴趣，那么继续第二步。

### **第二步** |基本熟悉工具和技术

数据科学本质上是编程和统计。目前实践中最流行的两种语言是 Python 和 r。在开始之前，参加一个在线数据科学课程并了解一些基本的统计数据并不是一个坏主意。这也将帮助你确定这种工作是否是你感兴趣的。在过去的一年里，我浏览了无数来自在线社区的推荐，这些课程似乎收到了很多好的反馈。

*   [用于数据科学和机器学习的 Python 训练营](https://www.udemy.com/python-for-data-science-and-machine-learning-bootcamp/)
*   [斯坦福统计学习](https://lagunita.stanford.edu/courses/HumanitiesSciences/StatLearning/Winter2016/about)
*   [吴恩达的机器学习](https://www.coursera.org/learn/machine-learning)

为了准备我的项目，我已经完成了何塞·波尔蒂利亚·乌德米课程的一个相当不错的部分，我发现这个指导非常出色。提醒一句:不要陷入无限的教程循环。只要把基本的东西记下来，然后尽可能快地着手你自己的项目。虽然上面列出的课程是很好的资源，但是通过解决自己项目中出现的问题，你会学到很多。你也会有一些很酷的东西来展示。

### **第三步** |选择一个你感兴趣的话题

这一个是相当自明的。如果你正在努力思考一个项目的想法，只需要考虑你自己的兴趣。从[苹果](https://www.kaggle.com/moltean/fruits)到[动物园](https://www.kaggle.com/uciml/zoo-animal-classification)，如果你能想到一个主题，那么很有可能存在关于它的数据。最好的项目想法是你实际上坚持的想法，所以选择让你兴奋的东西是对你最有利的。

### **第四步** |在数据上磨练

明白你的主题了吗？很好，现在是选择数据集的时候了。Kaggle 是一个惊人的资源，就像[谷歌的数据集搜索](https://toolbox.google.com/datasetsearch)一样。对我来说，我喜欢天文学，所以我正在看[美国宇航局的系外行星档案](https://exoplanetarchive.ipac.caltech.edu/)，并开始设想各种关系和模型可能会被提取出来。*旁注* -网络开发者们，请通过[允许](https://productforums.google.com/forum/#!topic/webmasters/nPq4BW6iPIA)谷歌用他们的搜索工具找到并公布你们的数据，让数据科学家的生活变得更轻松。(除非你反对谷歌，这也没关系)

### **第五步** |潜入你的数据！

在这一点上，您已经有了开始的工具、技术和数据。卡住了？有很多在线社区愿意帮忙。祝你好运！
