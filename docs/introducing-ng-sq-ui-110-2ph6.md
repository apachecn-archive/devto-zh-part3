# 简介:ng-sq-ui 1.1.0

> 原文：<https://dev.to/ardentia/introducing-ng-sq-ui-110-2ph6>

我们非常激动地宣布 ng-sq-ui 1.1.0 的发布！

以下是最新消息:

将所有独立模块分离为包。您现在可以只安装您需要的东西了- @sq-ui/ng-sq-common、@sq-ui/ng-datatable、@sq-ui/ng-datetime-picker、@sq-ui/ng-modal。

新特性@sq-ui/ng-datatable！
您需要的数据表，作为一个独立的包随时可用，不需要其他东西！

文档更新！从这个版本开始，我们有了多个包，现在文档的结构更好地代表了这些包。

更新示例！
所有 gif 图像都被删除。相反，为每个特性和模块提供了 Stackblitz 示例。

所有 Stackblitz 示例列表:
@ sq-ui/ng-datatable:[https://ng-sq-datatable . stack blitz . io](https://ng-sq-datatable.stackblitz.io)
@ sq-ui/ng-sq-common:[https://ng-sq-ui-progress-bar . stack blitz . io](https://ng-sq-ui-progress-bar.stackblitz.io)
@ sq-ui/ng-datetime-picker:[https://ng-sq-ui-datetime-picker . stack blitz . io](https://ng-sq-ui-datetime-picker.stackblitz.io) 

部署更新！我们已经改进了自动部署，最新版本的文档将在[https://sq-ui.github.io/ng-sq-ui/#/](https://sq-ui.github.io/ng-sq-ui/#/)发布。我们还改进了实例应用程序，您现在可以在[https://ng-sq-ui.xyz/](https://ng-sq-ui.xyz/)上找到！

通讯更新！
增加了[https://spectrum.chat/sq-ui](https://spectrum.chat/sq-ui)社区用于公开发布。
为普通 SQ-UI 讨论增加了新的松弛通道【http://bit.ly/ng-sq-slack。(剧透:在今年的某个时候，我们也会尝试为 Vue 发布 SQ-UI！我们希望听到您的建议和/或意见！)

旧的 NG-SQ-UI 松弛通道([http://bit.ly/ng-sq-ui-slack](http://bit.ly/ng-sq-ui-slack))将被留下用于开发需求和角度特定的开发。
新联系人邮箱: [contact@sq-ui.xyz](mailto:contact@sq-ui.xyz)
