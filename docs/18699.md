# 生活更新:我正式成为开发者了

> 原文:[https://dev . to/beznet/life-update-im-official-a-developer-50bj](https://dev.to/beznet/life-update-im-officially-a-developer-50bj)

有一段时间了。我保证我的缺席是有原因的，而不是因为我放弃了编码。事实上，恰恰相反。如题，我得到了一份网络应用程序开发员的工作！我是 9 月份被雇回来的，到现在已经有 3 个月了。从那以后，编码比以往任何时候都更成为我日常生活的一部分，所以我的博客更新已经退居二线了。

我为长时间的中断道歉。从面试过程(对我这样的新手来说相当残酷)到工作本身，生活已经变得超级累人。不要误解我，我每天都在学习很多东西，但我现在知道这是真正旅程的开始。得到这份工作只是开始。

我肯定会对面试、个人奋斗、我如何克服某些障碍并最终得到这份工作做一个完整的回顾，但同时我想让每个人都知道我还活着，做得很好。下次见，编码快乐！
