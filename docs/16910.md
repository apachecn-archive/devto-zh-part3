# 在现实世界中过数字生活

> 原文:[https://dev . to/jean Carl/living-a-digital-life-in-a-a-physical-world-21ia](https://dev.to/jeancarl/living-a-digital-life-in-a-physical-world-21ia)

智能手机成为许多人日常生活方式的一部分已经有十年了。随着技术以似乎是指数级的速度发展，智能手机现在包括多个传感器质量不断提高的摄像头、增强现实应用和游戏，以及多种相互通信的方法。许多可用的应用程序导致了照片的无尽定制，社交网络在数百毫秒内将这种多媒体传播到全球各地，传播给可能认识也可能不认识创作者的人。

智能手机并不是第一个实现这种分享的手机。数码相机提供了一种较慢的分享方式。一个人会花一整天的时间拍照，回家后把照片下载到台式电脑上，查看，然后上传到社交媒体网站，比如 MySpace 或你的个人地理城市网站(啊，美好的回忆)。它并不像今天这样立竿见影或触手可及。

本周的新年前夜，走在旧金山的街道上，随着时钟越来越接近午夜，我反思了自己的一年，并开始思考技术是如何在其中发挥作用的。当每个人都在等待恩巴卡德罗渡轮大楼上的烟花表演时，我观察到技术如何渗透到我们的物质生活中，似乎将我们带入了一个数字世界，这个世界在某种程度上以一种奇怪的悖论相互平行。

因此，我列出了这项技术已经实现的五件事情，我站在街道中间观察，想知道 2019 年及以后的未来会怎样。

# [](#texting)发短信

无论如何，发短信并不新鲜。然而，感觉越来越多的人走在街上发短信，对周围的环境一无所知。短信的即时性会导致一种紧迫感，所以毫不奇怪，人们会觉得需要立即回复，而不是在到达目的地后才有时间回复。

啊，别忘了那些表情符号！表情符号是传达回应的快捷方式，无需费力地打出信息。；-)现在有超过 2800 种表情符号可用，你不再局限于二十世纪的黄色笑脸。

# [](#streaming)串流

随着午夜的临近，人们通过网络向不在身边的人直播他们所看到的视频。实时流媒体将朋友和家人带到您身边，与您分享他们原本无法分享的时刻。

你观察过有人直播吗？经常看到流光拿着智能手机漫无目的地指指点点，聊着天，不再出现在现实世界中。如果另一端的观众患有晕动病，那他们就惨了。

还记得以前人们使用蓝牙耳机，你不知道他们是否在和你说话吗？

# [](#photography)摄影

当午夜钟声敲响时，烟花迎来了新年。当然，人们拿出智能手机拍摄照片和视频，并在社交媒体上分享烟花。

毫不奇怪，许多人盯着智能手机明亮的屏幕，而不是亲自享受这一时刻。顺便说一句，这些烟火真的很棒！他们怎么处理他们的车？在走廊中间拦住了他们。显然，当时间进入新的一年时，交通法规就暂停了？

对于那些彬彬有礼的路人来说，不走在十英尺外合影的人的正前方是一个挑战，因为这一过程中会暂时完全阻塞人行道。还是那句话，在这个宇宙中，礼貌似乎是一个片面的东西。

# [](#videos)视频

在回家的火车上，一名乘客正在用智能手机观看他们从烟花表演中拍摄的视频，准备发布到社交媒体上。如今，听到有人不戴耳机播放视频或音乐并不罕见，他们并没有意识到这是对他人的干扰。

没关系，我戴上耳机就不打扰你了！很抱歉。

# [](#driving-and-navigating)驾驶和导航

GPS 和地图在智能手机中已经变得如此普遍，尤其是在拼车应用程序中，以至于我们几乎都不知道自己在哪里了。司机们被屏幕上的东西分散了注意力，他们没有注意到车外的东西。

作为一名行人，一辆汽车在人行横道上不合法让行的情况每天至少会发生几次，如果不是更频繁的话。你猜对了，当司机经过我身边时，通常会有一个明亮的屏幕照亮他的脸。

过去，加州的停车标志只不过是建议司机应该考虑一些横过他们道路的东西。

我可以列出更多的事情，但现在是时候停止使用技术，去读一本书了，就像过去的美好时光一样。我感兴趣的是你所看到的这个新宇宙，以及它是如何在现实世界中的数字化身中生活的。
