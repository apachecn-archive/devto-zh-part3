# 你的播客现在在播什么？

> 原文:[https://dev . to/Nicky tonline/what-s-in-your-podcast-rotation-right-now-47 L2](https://dev.to/nickytonline/what-s-in-your-podcast-rotation-right-now-47l2)

很好奇现在每个人都在听什么播客，不管是不是技术性的。最近我一直在听乔·罗根的歌。他们很长，但他有来自各行各业的有趣的客人。也许不是每个人都喜欢，但我一直很喜欢他的播客。 **更新 2020:** 我只听过他的几集播客，但没有意识到我没有听过的其他集，它们对人们造成了伤害和危险。所以绝对不在我的播客列表里。

我最近还发现了托德·加德纳和大卫·沃尔什的《脚本与风格》播客。不知道为什么这个还没有出现在我的雷达上。
