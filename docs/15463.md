# 微软将在 1 年内终止对 Windows 7 的支持

> 原文:[https://dev . to/lefebvre/Microsoft-ending-windows-7-support-in-1 year-3 ngj](https://dev.to/lefebvre/microsoft-ending-windows-7-support-in-1-year-3ngj)

这是你的第一个提醒:明年，2020 年 1 月 14 日[，微软将终止对 Windows 7](https://www.microsoft.com/en-us/windowsforbusiness/end-of-windows-7-support) 的支持。不久前，当 Windows XP 寿终正寝时，我们经历了这一点(没有人真正关心 Windows Vista 何时寿终正寝)。Windows 7 非常受欢迎，因为它比 Vista 好得多。Windows 8 的许多用户界面变化根本不受欢迎，这也于事无补。

但是是时候开始准备更新了。你肯定会想更新到 Windows 10，而不是 Windows 8。Windows 10 其实真的很好，定期更新。我想你会发现从 Windows 7 过渡到 Windows 10 相对来说并不痛苦。事实上，去年早些时候，Windows 10 的使用量已经超过了 Windows 7。遗憾的是，你不能再获得 Windows 10 的免费更新，所以你必须购买，但既然微软已经表示 [Windows 10 是 Windows](https://www.theverge.com/2015/5/7/8568473/windows-10-last-version-of-windows) 的最后版本，你应该不需要再次购买。

2020 年 1 月 14 日以后不更新想坚持 Windows 7 会怎么样？根据微软的说法:

> 你可以继续使用 Windows 7，但一旦支持终止，你的电脑将变得更容易受到安全风险的影响。Windows 将运行，但您将停止接收安全和功能更新。

目前，您可以在计划更新时继续使用 Windows 7。当然，Xojo 本身和用它开发的应用程序也可以继续在 Windows 7 上运行，尽管你需要确保你已经安装了服务包 1。
