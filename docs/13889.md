# 反编译和反汇编从 Kotlin 源代码编译的 JVM 字节码

> 原文:[https://dev . to/yuya bu/decompile-and-disassemble-JVM-bytecode-compiled-from-kot Lin-source-code-2cd 9](https://dev.to/yuyabu/decompile-and-dissasemble-jvm-bytecode-compiled-from-kotlin-source-code-2cd9)

这是一个尝试使用 javap(反汇编程序)和 jad(反编译程序)的 Kotlin 代码的实验。

```
fun main (args: Array <String>) {
    println ("Hello, World!")
} 
```

代码来源
[https://kotlinlang.org/docs/tutorials/command-line.html](https://kotlinlang.org/docs/tutorials/command-line.html)

像其他 JVM 语言一样，Kotlin 被编译成一个类文件，可作为 Java 执行。
我们可以验证是因为 javap 可以用 jad 显示 JVM 的助记符并反编译为 java 源代码。

编译这个。

```
kotlinc hello.kt -include-runtime -d hello.jar 
```

提取 jar 并对 hello.class 使用 javap，jad

```
jar xf hello.jar 
```

# [](#javap-disassembler)javap(反汇编器)

## [](#preparation)准备

这是不必要的，因为它从一开始就在 JDK

## [](#execution)执行

```
javap - c HelloKt.class 
```

# [](#jad-decompiler)jad(反编译器)

## [](#preparation)准备

如何介绍 jad

```
brew tap caskroom / cask
brew install caskroom / cask / jad 
```

## [](#execution)执行

您可以使用-a 选项将助记符显示为注释。

命令

```
$ jad -a HelloKt.class
$ cat HelloKt.jad 
```

```
import java.io.PrintStream;
import kotlin.jvm.internal.Intrinsics;

public final class HelloKt
{
```
