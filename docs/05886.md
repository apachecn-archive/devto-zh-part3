# 如何回答—“应该先学什么编程语言？”

> 原文:[https://dev . to/marianyp/how-to-answer-what-programming-language-should-I-learn-first-24i 4](https://dev.to/marianyp/how-to-answer-what-programming-language-should-i-learn-first-24i4)

## [](#introduction)简介

软件开发领域的初学者经常会问这个问题。然而，它似乎没有一个简单的答案。这篇文章将解释为什么在选择你的第一语言时要小心，以及你如何自己回答这个问题。所以让我们开始吧！

## [](#misconceptions)误解

通常，当这个问题得到回答时，你可能会得到两个可能的答案之一。有人可能会说:“学 X 语言吧！这是最好的，也是未来”。从某种意义上说，这可能是真的。或者他们可能会说“没关系，随便挑一个”。我对后一个有问题，因为虽然它有些真实，但也有误导性。

为了说明这个问题，我想让你想象你正在买车。你去镇上附近的汽车经销店问经销商:“我应该买什么车？”作为回应，经销商说，“这有关系吗？随便挑一个就走！”。于是你接受了他的建议，随便挑了一辆车，付了钱，然后离开。会发生什么？很有可能，你只剩下一辆你不喜欢的车。这辆车太旧了，注定很快就会抛锚。不仅如此，它的外观也不好看。你最终把钱花在了一辆自己都不喜欢的车上，现在你又回到了起点。再选一辆车，再花更多的钱。

在随机选择第一种编程语言时，可能会出现类似的问题。选择一种随机的编程语言可能会让你进入一个你不感兴趣的领域。导致你浪费时间。让你精疲力尽或气馁。

这就是为什么你需要自己回答这个问题。只有你自己知道你的优点和兴趣。

## [](#what-fields-interest-you)你对哪些领域感兴趣？

有了这些信息，我们实际上需要改变问题。而不是问“我应该先学什么编程语言？”，你需要问“我对什么领域感兴趣？”。然而，要回答这个问题，你需要知道软件开发中有哪些领域。这需要你做一些研究，但我会帮你快速启动。以下是 2019 年的一些热门领域。

1.  前端 Web 开发
2.  后端 Web 开发
3.  移动开发
4.  游戏开发
5.  数据科学
6.  网络安全

在研究其中一些时，你可能会发现一两个让你感兴趣的。然后，你可能会决定调查该领域使用什么语言，并从那里开始，对吗？

现在你可能会想，现在怎么办？很重要的一点是，仅仅因为一个领域/行业让你感兴趣，并不意味着它就适合你。记得我在同一个岗位的时候，对游戏开发领域很有好感。我决定花几个月的时间尝试学习多种语言和游戏引擎(这本身就是一个错误)。然后我意识到游戏开发在一个持续的基础上使用数学。由于我是一个数学很差并且讨厌数学的人，我决定游戏开发不适合我。这不仅仅是因为我不擅长数学，因为网上有很多资源可以解决这个问题。但事实是我不愿意学习这些东西。我对学习游戏开发很感兴趣，也愿意学习。但是我对学习游戏开发的先决条件不感兴趣，也不愿意。

## [](#where-should-you-go-from-here)你该何去何从？

所以你已经对你的兴趣领域做了研究，并且对进入这个领域的前景有了一个相当不错的想法。你现在应该进入下一步选择你的语言，这意味着你的问题几乎已经得到了解答！但是既然你已经走到了这一步，你可能已经注意到有很多工具和技术。不要陷入其中，花太多时间决定使用哪一个。现在你知道你想要什么领域，进入这个领域是你的首要任务。所以你想选择一些东西，不一定是随意地，而是迅速地。

每种编程语言都有它的优点和缺点，你最终会学到，但在大多数情况下，它们都做同样的事情。简化你找到的列表，然后挑选最有用和初学者友好的，通常是提示问题的答案。那么我们如何做到这一点呢？

这需要你做更多的研究。首先，问问你自己，你是否打算利用这个领域来获得收入。如果是这样的话，试着在你选择的领域查找你所在地区的工作。上像 Indeed、Glassdoor、LinkedIn 等网站。，并了解哪些技术使用得最多。接下来，您希望了解哪些技术拥有最多的在线资源。语言 A 可能没有很好的文档，只有很少的教程来帮助你入门。但是语言 B 可能有很好的文档和非常好的课程供你学习。

这是否意味着语言 B 比语言 A 更好？一点也不！但是对于初学者来说，B 语言会更友好，并且会帮助你学习如何像程序员一样思考。

这也是需要记住的一个关键因素。因为这是你的第一门编程语言，你还不具备程序员的思维模式。在学习你选择的语言时，你会想要学习如何实际使用它的特性。

## [](#conclusion)结论

虽然这看起来很多，但我可以向你保证这不是。仔细选择第一语言的整个过程可能需要一天或一个月的时间。但是这将比没有计划的盲目投入节省你更多的时间，并且从长远来看都是值得的。
