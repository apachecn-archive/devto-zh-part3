# [](#csv-based-pricing-in-magento-2-based-on-length-amp-width-of-products)基于 CSV 在 Magento 2 中定价基于长度&产品宽度

基于 CSV 的定价扩展帮助店主定价和销售按尺寸(长度和宽度)销售的产品。您可以在产品页面上显示长度和宽度输入字段，以便用户可以输入他们想要的值。当用户输入值时，价格会自动计算。商人可以定价和出售绘画，织物，木材，地板，瓷砖等。以及其他基于高度和宽度出售的产品。

为此，商家必须创建一个包含基于长度和宽度的产品价格的 CSV 文件。例如，您可以在 CSV 文件中以下列格式添加长度和宽度的价格。

看看这个 FME CSV 定价 Magento 2 扩展，使您能够将基于 CSV 的定价解决方案添加到您的 Magento 2 商店。

**主要特点**:

*   通过 CSV 矩阵表进行产品定价
*   基于长度、宽度或高度的二维定价
*   最小值和最大值的自定义消息
*   为“找不到大小”消息自定义消息
*   产品的特定产品和全局 CSV 文件设置
*   五种测量单位:米、毫米、厘米、英尺、英寸
*   用分数计算
