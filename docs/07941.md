# 声音的后果

> 原文:[https://dev.to/halented/the-consequence-of-sound-46ak](https://dev.to/halented/the-consequence-of-sound-46ak)

声音是强大的。没有事先训练，声音可以让你高兴，生气，或平静。某些声音会让你发疯，或者让 T2 产生幻觉。我们经常不假思索地经历声音，或者让生活中的其他人控制我们听到什么和什么时候听到。不一定非要这样！

有很多很多方法可以让你抓住声音的力量，让你的生活更愉快，让你的一天更有效率。我经常想到的两个现象(也是我觉得非常有用的)涉及到使用具有状态依赖记忆和经典(或巴甫洛夫)条件作用的声音。

状态依赖记忆于 1937 年在伊利诺伊州首次进行了测试，爱德华·吉尔登和埃尔默·卡勒发现，在学习某个特定技巧时处于相同“意识状态”的狗，如果意识状态发生变化，则很难记住这个技巧。在这种情况下，狗是高的。

改变“意识状态”可以有很多不同的含义。它可能意味着从睡眠到觉醒，从清醒到醉酒，从放松到警觉，或者，在我们的情况下，只是听一首真正优秀的歌曲。音乐和声音有能力像其他药物一样迅速影响你的意识状态。

在应用程序中，这意味着如果你在学习编写 javascript 代码时打开同样的海洋噪音，你将更有可能记住 javascript。这是一种有趣而有效的方法，可以将你的技能分类，并在正确的时间将你的大脑“侵入”到正确的位置。

依赖于状态的记忆和经典条件作用是密切相关的。如果说状态依赖学习是一种精神黑客，那么经典条件反射就是它的物理副本。众所周知，经典条件反射是由一个人推广开来的，也是在狗身上。当巴甫洛夫注意到狗不能根据命令有意识地流口水时，他的工作真正开始了，然而，它们可以被训练这样做。同理，你也不可能总是一觉醒来就干劲十足，准备工作。但是，如果你意识到自己什么时候最有动力，并且总是播放特定的噪音，你最终可以训练你的身体作为对噪音的反应而提高生产力。

需要注意的是，在实践中，很容易意外地将这种训练周期颠倒过来。如果你持续转动，打开一个噪音，告诉自己“要有效率，该死的！”那么你实际上是在训练自己害怕或不喜欢那种噪音。所以尽量避免那个。没有亲身经历什么的。🙄
