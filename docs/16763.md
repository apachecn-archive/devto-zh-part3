# 一年家庭办公室，我学到的东西

> 原文：<https://dev.to/mfahlandt/one-year-homeoffice-things-i-have-learned-3518>

老实说，只有一年多一点。实际上是一年零三个月。

首先也是最重要的规则是:

### 每个工作日都要穿裤子，一条你会在办公室穿的好裤子。

我很快就学会了。当你起床，穿好一天的衣服，你就可以在工作和业余时间之间划出一条清晰的界限。所以你一天有一个干净的开始，晚上你可以期待在晚上得到“工作”的衣服。我的妻子也抱怨我看起来越来越没有吸引力，所以这是必须的，你不想看起来像桥下的巨魔。
积极的一面是，你总是为即将到来的令人惊讶的视频聊天做好准备。

### 设定基本工作时间，但不要起身坐在办公桌前

对我来说，创造某种工作日是很重要的。所以我给自己设定了开始工作的固定时间。我在早上 8:30 到 9:30 之间开始工作。这也很容易让所有想联系我的人知道我什么时候有空。重要的是还有一个但是。我从不起身直接坐在办公桌前，基本上我创造了一种工作方式。所以起床后的第一件事就是准备咖啡和遛狗，之后我吃早餐，然后开始工作。这也有助于你创造一些常规，不要陷入睡过头的陷阱。

### 有刹车，不要伏案吃饭！

这可能是最大的问题之一，而不是安排休息或在办公桌上吃饭。这也是我长期以来犯的一个错误，有时还会犯。在你的桌子上吃东西或者不做适当的刹车。我试着至少每两个小时从办公桌前站起来几分钟。伸个懒腰，弄点喝的什么的。如果你有便携设备，换个地方。如果你想在沙发上坐一会儿，那就去吧！但是请休息一下，停止工作，让你的头脑清醒一下。因为狗，我在工作时间至少要出去一次。如果你没有类似的事情，找一个活动让你远离工作。它有助于获得新的能量。

### 投资您的设备

我们整天坐着，所以投资一个适合你的好的装置。买一把合适的椅子，你的背会感谢你的。我低估了这一点很长一段时间，现在我不得不每隔一段时间就去看看后面的医生。不要犯同样的错误。向你的雇主寻求支持。公司不必为你保留一个办公室位置，这可以节省很多钱，所以如果为一把好椅子或者一张桌子付钱，那就太公平了。还需要一个适合你的键盘，一个合适的网络摄像头和麦克风/耳机，你的客户和同事会感谢你的。

### 感到如此孤独

一个巨大的挑战是人类互动的分离和缺失，这对于所有人来说都不容易或无法实现。你不会看到很多人。对我来说，这正是我所需要的，比起在任何办公室工作，我更喜欢独自在家工作。但是你和人的接触会少一些。我在工作之余做运动和见朋友，这对我来说就足够了。对许多人来说，这不是。老实说，我帮不了你。我的一些同事整天都在 skype 上，想找个人聊天。

### 哦香香分心

另一个难点，注意力分散。厨房可以打扫，花园需要修理，在网上拖延；在家比在办公室更容易分散注意力。请记住，你的家人也可以分心。当我妻子在家的时候，我觉得我比她不在家的时候做得少。这个怎么处理。分散上网注意力的最简单方法给自己弄一个有定时窗口的网站拦截器，这样你就不会上 facebook 或 youtube 了。对于其他事情，回到我们说过的定义工作时间的那一点，让你的家人清楚，这个时间是工作时间。

### 停止工作！

另一方面，你可能最终会无法停止工作。所以，每天给自己设定一个停止工作的固定时间。我认为这是一个比分心更大的问题，至少对我来说是这样。大多数情况下，我在下午 6 点到 7 点之间停止工作，从那以后我不再阅读任何邮件，我也不再在周末工作(除了高时间项目，但那是另一个故事)。如果你每天都加班，对你自己没有任何好处。

### 保持健康

也许你正在错过下班后的散步或健身工作室。不要！给自己找一项体育活动来保持健康，因为你甚至会比在办公室工作时更少出门。好事情是你可以在下午去健身室或攀岩屋。你自己定义你的工作时间。这是一个巨大的优势，利用它！我停止了它，它对我不好，现在我每周至少在一个完全空的大厅里爬两次。

### 最后一句话

这是我的观点，所以不一定适用于你。你在家工作吗，还有其他建议吗？让我知道！你认为在家开始工作会引起评论吗
