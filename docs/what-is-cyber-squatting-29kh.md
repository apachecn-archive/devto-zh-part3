# 什么是域名抢注？

> 原文：<https://dev.to/vjnvisakh/what-is-cyber-squatting-29kh>

域名抢注是恶意购买域名的过程。假设你正在组建一家名为 **xyz** 的新公司。你去域名注册机构输入 xyz.com，你发现该域名不可用，你不得不以更高的价格购买。

然而，这并不总是意味着它有一些错误的做法。当前的域名所有者可能真的想要那个域名供将来使用。

如果发现域名所有者与该域名没有关系，也没有基于该域名开发任何产品的计划，或者只是利用该域名销售广告，那么将受到惩罚。

要了解谁购买了域名，请前往[https://www.whois.net/](https://www.whois.net/)
