# 什么是智能合约？

> 原文:[https://dev.to/dapp_com/what-are-smart-contracts-2bfb](https://dev.to/dapp_com/what-are-smart-contracts-2bfb)

智能合约是令人难以置信的代码行，使 dapps 成为可能；没有它们，就不会有加密货币或有趣的应用程序运行。智能合约确实已经从其最初的功能走了很长的路，表现为交易的代理，并将区块链技术的能力推得更高。在我们对智能合约的研究中，我们遇到了最初的发明者尼克·萨伯，让事情变得更有趣的是，他被认为是“中本聪”(比特币创始人使用的化名)背后的人或其团队的一部分。这位美国密码学家早在 1998 年就用自己的名为“比特黄金”的加密货币开发了这一概念，并将他在 1994 年首次提出的智能合约付诸实践，作为消除交易中间人的解决方案，从而为货币的使用创建了一个安全的网络。他的论文似乎不仅预测了我们十年后随着比特币的发布而付诸实践的一切，而且在论文中，我们还看到了他的一些想法被彻底实施。

然而我们跑题了；我们想做的是解释智能合约是如何工作的，并看看这种巧妙的编码方法提供的其他功能。尼克·萨博是否真的是中本聪，这要由你来决定。然而，智能合约无疑是萨伯的创意，我们应该感谢他让我们今天使用这项令人兴奋的技术。

智能合同如何工作

简单来说，智能合同是买卖双方之间简单交易的自动执行合同。智能合约可以具有的最简单功能的细节如下:它只能确认卖方正在以一定的金额出售物品，并且买方愿意并且能够在代码内支付该金额，该代码需要由区块链网络上的分散式分类帐进行验证。一旦提出购买请求，智能合同就会确认所有条件都得到满足。在满足标准的情况下，交易可以顺利进行。如果有问题，交易中有问题的贵重物品将被归还给它们原来的主人。在这两种情况下，分散的分类帐将更新买方或卖方拥有的当前值。每一笔交易都被记录在更新的分类账中，以确认每一笔交易都发生在特定的日期、特定的金额以及两个特定的账户之间，从而创建了一个“透明”和“可信”的区块链

智能合约需要首先进行完美的编码，然后插入到区块链中，以使智能合约的功能有效。一旦代码被编写、验证并存储在分散的分类帐中，网络上的每个人都可以直接调用该函数来帮助进行交易。每个人都可以使用该功能，但没有人能够改变它，这使得区块链技术成为各种操作的值得信赖的解决方案，并确立了其至关重要的“廉洁”本质。

以太坊的智能合约

我们刚刚描述的是比特币的智能合约，它旨在详细记录谁拥有什么或被用作一种货币，从而在转移大量资金时无需第三方(如银行和政府)。

另一方面，以太坊被明确设计为以不同的方式使用智能合约。以太坊的创造者 Vitalik Buterin 对区块链进行了改造，允许开发者在智能合约中添加功能，以支持他们(开发者)的各种需求。编写智能合同非常困难，因为以太网以字节码或者简单地说，以 1 和 0 的形式读取它们。为了让开发者更容易创建更加复杂的智能合约，Buterin 创造了编程语言“Solidity”。使用 Solidity(类似于 JavaScript)的开发人员现在有办法以他们想要的方式设计他们的智能合同。

凭借这些功能，以太坊成为了一个虚拟机，它可以提供在分散的以太坊网络上运行软件、网站和应用的功能。最终，以太坊的目标是达到“图灵完整性”，在这种情况下，这意味着能够在分散的以太坊网络上模拟计算机的能力。

运用它的能力

如果你想尝试以太坊的功能，请访问我们网站上的 dapps。最有趣的是智能合同可能被应用于与官僚中心机构特别相关的领域的方式。另一个应用是关于你的个人官僚文件——从你的出生证明到你的驾驶执照。想象一下，如果每一份与你有关的法律文件都存储在一个分散的网络中，如果你需要其中任何一份文件的副本，你只需在确认你的身份后发送一个请求，你就可以立即获得所请求的文件。这种系统将加快原本非常缓慢和繁琐的过程。

更进一步，可以编写智能合同，在买房、买保险或缴税时自动更新所有法律文件。由于从交易到所有权的一切都是透明和廉洁的，这将减少逃税，并顺利创造一种更轻松的生活和治理方式。

例如，甚至亚马逊也有可能通过自动化一切和使用智能合同来参与进来。智能合同将确保交易完成，商品从仓库发出，并保证包裹到达目的地。这听起来像科幻小说，但亚马逊已经在使用无人机维护仓库库存，甚至送货。

有趣的未来在前方

智能合同可以成为改变我们社会的不可思议的力量。他们已经展示了他们巨大的能力和优势。以太坊有巨大的潜力创造一个更加惊人的未来，特别是如果它成功实现图灵完全性，一切都是分散的，可以在分散的网络上运行。

然而，实施其他支持应用程序来改变大多数机构的工作和处理文档的方式将是一个挑战。很少看到成熟的机构(法律或政府)采用创新和前卫的技术。因此，私营公司很可能会率先试水，实施区块链技术，创建分散网络，并为其特定需求开发智能合同。

最终，智能合同很有可能会被应用来让我们的未来变得更美好。是以太坊还是其他一些区块链还有待观察。
