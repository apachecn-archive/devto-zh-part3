# Ruby 程序员工资

在美国，一个 Ruby 程序员可以赚到 16 万美元(高级)。一个大三学生通常能挣 9 万美元

在欧洲，工资要低得多。最低的是葡萄牙，高年级学生最高收入为 26.000 美元，低年级学生为 16.000 美元。

一般来说，在欧洲，大三学生的收入大约是 5 万美元，大四学生是 8 万美元。薪水取决于国家(以及生活成本和税收)。
