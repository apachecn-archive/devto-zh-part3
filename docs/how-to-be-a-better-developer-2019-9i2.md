# 如何成为更好的开发者 2019

> 原文：<https://dev.to/kaperskyguru/how-to-be-a-better-developer-2019-9i2>

我最近在心里想到了这篇文章，我需要向我的学员解释他们如何在编程 2019 中变得更好，所以这是我想到的。从作为专业软件开发人员的经验来看，编码或编程是一项技能，每项技能都需要时间来掌握。

选择一条道路:编程中另一个令人困惑的部分是，有很多职业或语言或工具，事实上有很多新的东西要学，几乎每天都有新的版本出现。你必须仔细选择一条特定的道路，无论是前端、后端还是整个堆栈，根据经验，如果有人专注于某个特定领域更长的时间，他可能会变得更好。

所以我会建议你选择一条路，并坚持走更长的一段时间。

**耐心:**你必须对代码有耐心，学习编程需要很大的耐心(我必须承认)，从编码到调试，你会逐字逐句地阅读你的代码行以了解流程，因为这是一门手艺，所以你必须花时间来精心制作。

**网络:**当我开始时，我从不相信这一点，因为我从一开始就不是网络的狂热爱好者，但在朋友的帮助和建议下，我学会了如何与人交往，我无法告诉你这对我的编程生涯有多大的帮助，你也应该试试。
