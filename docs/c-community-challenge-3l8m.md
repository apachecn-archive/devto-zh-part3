# C++社区挑战

> 原文：<https://dev.to/codecademy/c-community-challenge-3l8m>

你好，dev.to，我叫 Sonny，是 Codecademy 的课程开发人员。我们最近提议、设计并构建了我们的 [Learn C++](https://www.codecademy.com/learn/learn-c-plus-plus) 课程，该课程于 12 月份推出。在过去的三个月里，[有 184，038](https://thepracticaldev.s3.amazonaws.com/i/yic3n8iuwhftrq6jqz0n.png) 名学员开始学习这门课程，目前它是我们网站上第五大最受欢迎的课程。😱

所以为了庆祝，我和我的队友玛丽埃尔向我们的专业学习者发起了一个初学者友好的 C++社区挑战，这个挑战将持续到本月底。我们想回馈社区，也希望激发学习者更多的合作&团队精神。还有奖品:Codecademy 帽衫和 Arduino Unos！

### 如何进入:

1.  在我们的终身学习者中找到一个代码伙伴。👯
2.  一起决定和建立一个项目。🌱
3.  在 3 月 31 日之前提交一个到 GitHub 库的链接。瞧啊。🙌

[![High Five](img/464748a00de4a8b74f2fdf78ff8f4a61.png)](https://res.cloudinary.com/practicaldev/image/fetch/s---ZiVq3l7--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_66%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/ifbxcof0g1qng5inkf83.gif)

### 挑战规则:

提交的奖品截止日期为**2019 年 3 月 31 日星期日美国东部时间晚上 11:59**。你的项目可以是你的团队决定的任何东西。将根据以下内容进行判断:

*   在你的代码中使用[向量](https://www.codecademy.com/learn/learn-c-plus-plus/modules/learn-cpp-vectors)和[函数](https://www.codecademy.com/learn/learn-c-plus-plus/modules/learn-cpp-functions)。
*   协作(一起工作时，编码更有趣)。
*   代码的原创性和可读性(让它成为你自己的代码，并使用风格约定和注释来保持可读性)。

获胜者将在四月的第一周宣布。
