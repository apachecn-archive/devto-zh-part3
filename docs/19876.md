# 什么是隔间佛+我们如何变得快乐

> 原文：<https://dev.to/cubiclebuddha/what-is-cubicle-buddha-how-we-re-gonna-get-happy-1ml9>

幸福。这是我们基本上想要的。我们想要最终康复后的平静。

当然，但是你(理所当然地)问:“那我呢？我可以在瑜伽课上很好地冥想……但我工作在一个快节奏的行业，这个行业一直在推动我取得更多成果。”和平也为你而存在。我们甚至可以将冷静带入工作场所。

那么，在一个不安定的世界里，我们如何平衡对宁静的追求呢？

我们将回答这个问题以及更多问题。请尽快回来阅读一些实用的文章，比如如何在不知道自己正在冥想的情况下进行冥想的技巧。我会分享我在工作中的失败(*其中有很多*)以及在这个过程中我对自己的想法的了解。我不会孤军奋战——每个发现过程都与社区合作得最好。我喜欢一行禅师的这句话，他将社区(或“僧伽”)描述为:

> “冥想的作用，社区建设的作用，我们日常生活中的治疗和改造的作用对环境至关重要。你不能光有项目和倡议，却不关心内心的痛苦。”

这就是为什么我认为一行禅师是我社区的一部分(尽管我从未见过他)。令人难以置信的是，他谈到了项目/工作和内部成长之间的平衡。我很清楚，成为一名更好的员工意味着成为一个更好的人。正如我所怀疑的，快乐的人更有效率。

我刚刚开始进入一个更平静、更快乐的工作场所，但我从朋友和同事那里了解到的关于他们正念的经历让我深受鼓舞。因此，这就是我将依靠你的问题和投入。在[推特](https://www.twitter.com/cubiclebuddha)、[脸书](https://www.facebook.com/cubiclebuddha)上与我们分享，或者在下面的评论中告诉我们正念是如何影响你的工作的。我真不敢相信，仅仅读过[的第一篇文章](https://dev.to/cubiclebuddha/samsara-5-agile-techniques-to-end-suffering-and-increase-learning-27cc-temp-slug-32809)，朋友和同事们就已经提出了问题和话题，这有多棒啊！这真是鼓舞人心。让我们互相帮助，学习如何找到平静。
