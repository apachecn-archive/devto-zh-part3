# 我多么讨厌道德黑客这个词！

> 原文:[https://dev.to/yafkari/how-i-hate-the-term-伦理-黑客-al8](https://dev.to/yafkari/how-i-hate-the-term--ethical-hacker-al8)

请对我好一点，这是我的第一篇帖子，我没有特别的英语技能，所以请随时纠正我！😁

如今，**安全已经成为世界上一个至关重要的问题。无论是现实世界还是虚拟世界。**

今天，我们只谈论网络犯罪，网络战争，数据窃取，...黑客攻击已经成为一个可悲的现实，任何人都无法逃避。
无论是个人，还是像脸书或谷歌这样的小型或大型公司。
我们都容易受到恶意黑客的攻击，无论是为了获取金钱、数据还是娱乐等...

对 IT 安全分析师、道德黑客……的需求激增。这是一件好事，因为安全和隐私比任何事情都重要。(当我最终离开学校时，这对我也是一件好事😆)

但如果有一件事让我困扰的话，那就是每次都使用的“道德黑客”这个词，因为它暗示黑客天生就是不道德的。我们将不得不再次回到黑客的真正定义(至少是我的定义)。

对我来说，黑客是一个充满好奇心的人，一个不仅会使用他的烤面包机，还会打开它，了解它的工作原理并最终定制它(例如，添加一点祝我们美好一天的声音)的人。
*一个伟大的想法，在我看来非常有用😂。*

因此，电脑黑客也会遵循同样的原则。

在这种情况下，有不同的类型。

有:

*   **黑帽子**受金钱和未经授权进入他人系统的快感所驱使的人。← ***真正不道德的人***

*   或多或少以“道德”方式行事的灰色帽子。这些人通常进入一个系统，没有再次授权，但没有破坏它，为了这样做的乐趣或后来通知公司他们刚刚入侵，以便他们可以修复漏洞。

*   **白帽子**谁是故事中的“好人”。
    我觉得[维基百科](https://fr.wikipedia.org/wiki/White_hat)的定义很好，我就这么说吧:
    一顶白帽子就是*执行入侵测试和其他测试方法以确保组织信息系统安全的计算机安全专家。*

所以我更喜欢“IT 安全专家”、“pentester”等称呼...

今天就到这里，我希望你喜欢这篇小文章，你会反对*这个不应该被命名为*的东西的扩张😂。
