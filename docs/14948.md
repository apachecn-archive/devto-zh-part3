# 你最喜欢的社区是什么？

> 原文:[https://dev . to/maniframes/what-is-your favorite-communities-5d7f](https://dev.to/maniflames/what-are-your-favourite-communities-5d7f)

在过去的一个月里，通过在 DEV 上发帖和与他人互动，我学到了很多。不仅仅是关于代码或生产力，而是关于社区有多棒。DEV 上的每个人都帮助别人，同时保持礼貌，这真的很酷。几乎在每个社区，我只是一个偶尔路过的潜伏者。这是第一个让我觉得我可以自己张贴东西并成为对话的一部分的社区。

我意识到很多开发者都有 twitter 账户。在我内心深处，我知道 twitter 上有一些开发者，并关注了一些。在从 devs 开始探索之后，我掉进了一个兔子洞，里面有一大堆使用 twitter、电子邮件列表、个人博客和 dev 这样的网站来分享知识的社区和人。我觉得我只是触及了有趣的人和社区的表面。

你最喜欢的社区和/或开发者是什么，为什么？我想发现更多的精彩😁
