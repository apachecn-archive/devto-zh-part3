# 所以你想在家工作…

> 原文：<https://dev.to/dealeron/so-you-want-to-work-from-home-322k>

任何在办公室工作了很长时间的人都曾有过同样的想法，“每天在家工作该有多好？“只要你怀着正确的期望去做，并避免一些典型的陷阱，你会获得难以置信的回报。

你需要考虑的第一件事是你的公司是否准备好处理远程员工。查看我们之前的文章,看看这家公司能做些什么来让每个人的事情变得更容易。另一个主要障碍是，如果你的工作绝对需要你在现场。我想，当一名在家工作的建筑工人是很难的。DealerOn 的开发团队实际上只要求我们有一台计算机和一个互联网连接，所以让我们开始吧！

### **这是我做过的最好的决定！**

让我们首先列出远程工作的一些主要好处。胡萝卜在大棒之前。

通勤——现在美国的平均通勤时间是[单程 26.4](https://www.washingtonpost.com/news/wonk/wp/2017/02/22/the-american-commute-is-worse-today-than-its-ever-been/?noredirect=on&utm_term=.54fea048245f) 分钟。在较大/密集的城市地区，这可能高达 60-90 分钟。长途通勤可能会带来压力，如果是因为令人沮丧和不可预测的交通，压力会加倍。对于这个国家的大部分地区来说，由于缺乏公共交通选择，开车是浪费时间。我相信你会同意，你可以在一天中多花一个或三个小时找到更好的事情去做！

舒适——对你来说，即使是最好的办公室也可能不如你自己的地方舒适。你可以单独控制恒温器，你可以随心所欲地大声放音乐，如果你整天不穿鞋也没人会做鬼脸。你也有选择家具的最终决定权。不过，一定要确保你还是穿着上班的衣服。没人想在镜头前看到你穿着睡袍。

宁静——有些办公室可能会很嘈杂和混乱。背景对话、电话和打印机都会像乌云一样笼罩着你，分散你的注意力。一个解决办法是戴上耳机来淹没其他一切，但没有什么能比得上你自己私人空间的宁静。

多任务处理/灵活性—如果你整天都呆在家里，做好某些家务会变得容易得多。洗衣和洗碗是两项工作，可以在工作前开始，然后在午休时完成。如果一个重要的包裹需要签名或者有被偷的危险，你已经在那里了。如果你有宠物，它们会喜欢有你在身边和更频繁的上厕所。如果你有一个生病在家的孩子，你也可以在那里，但仍然可以工作，这取决于他们的年龄。

### **会出什么差错？**

远程办公听起来很神奇，为什么我们现在不都这样做呢？！？！除了管理层的抵制之外，还有一些原因说明它并不适用于所有人。

分散注意力——在多任务处理部分，我确保只提到工作前后和午休时间，因为这也很容易被你家里所有要做的事情分散注意力。我有一个惊人的视频游戏、电影和电视节目库。我晚上做的家务比我想做的还要多。不要因为现在太方便了，就陷入为了家庭项目和娱乐而忽视工作的陷阱。

孩子是另一个会分散你注意力的大问题。人们很容易认为在家工作是节省保姆费用的一个好方法，但是低于一定年龄的孩子需要不断的监督和精神刺激。他们不明白你在“工作”，你没有时间听 PJ 面具最新一集的完整回顾。如果他们通常在学校或托儿所，但不得不生病呆在家里，这可能不是一个问题，因为他们一天的大部分时间都在床上。对于孩子呆在家里的最小年龄没有硬性规定，但是要明白你有工作要做，并且可以自娱自乐。

**沟通**—如果你的同事都在同一个办公室，而你远程工作，你很难和你的队友保持同样的沟通质量。我们在上一篇文章中讨论了一些更大的障碍，比如使用 Slack 和视频会议，但是仅仅有工具是不够的。因为你的经理/队友在任何时候都不能看到你，他们更容易忘记你或者不知道你在做什么。通过近乎过度的交流来克服这一点。在聊天室里要特别健谈，直接给有问题的队友发信息，尽可能在开会的时候开着相机。你是团队的一员，所以要确保你随时都可以被联系到，并参与对话(在工作时间)。

另一个可能影响与团队沟通的问题是时区差异。也许你正在这个国家的另一个地方或者地球的另一边远程工作。你的两个选择是适应家庭办公时间或当地工作时间。如果你在自己的时间工作，确保在重叠时间从你的队友那里得到你需要的东西，把你可以在自己独处时做的事情留到其他时间。最令人沮丧的事情莫过于需要别人回答问题，但却知道他们半天都没空。

孤独对人们的影响不同，但值得注意，因为许多新的远程工作者在做出改变之前没有考虑到这一点。当你在办公室工作时，几乎总会有人和你谈论工作或者新闻或流行文化中发生的事情。太多的这种情况可能是你首先想要远程办公的原因。问题是太多的孤立会对你产生负面影响。如果你的公司不使用聊天/视频会议，你可能一整天都不会大声说话。一定要打破这种局面，与人共进午餐或在下班后做一些社交活动。除非你经历过，否则很难知道这对你来说是不是一个问题。不要忽视这一点，单独监禁被视为监狱中的一种惩罚是有原因的。

**模糊的线条** —当你的家和办公室在同一个地方时，你很容易永远不会“离开”工作。拥有一个专用房间作为*办公室*会有所帮助(而且可能是一笔不错的税务注销**)，但是即使你没有这样的空间，保持正常的工作时间也很重要。每一两个小时休息一下，离开你的座位伸伸腿，离开你的办公桌去吃午饭，并且知道你什么时候会停下来过夜。避免精神枯竭比额外工作的短期收益更有利于你的健康。

如果你的公司对在家工作持开放态度，并且你知道你将面临什么，那么在家工作会是一份相当不错的工作。只要你为自己设定一些基本规则，我列出的所有复杂情况都是可控的。如果你仍然不确定，试着每周在家工作一到两天。一旦你适应了，你将面临的最大障碍是，每当有人抱怨他们的通勤时，不要用纳尔逊·蒙茨的形象来回应。
