# 学习 Golang 需要什么

> 原文：<https://dev.to/mairelin/what-do-you-need-to-learn-golang-15ai>

我从事 Java 开发已经很多年了，但是在过去的 2 年里，我对 Golang 编程语言产生了浓厚的兴趣，大多数时候人们在听我讲述时会问我，我需要学习什么？

老实说，我一直认为，对于学习生活中的任何事情，你只需要真的想做，并付出任何必要的努力来实现它。顺便说一句，根据你的经验，有一些有用的方法可以指导你如何开始学习。我特别把它分成不同的部分:

仅基于面向对象程序设计的经验:
Golang 是一种多参数语言，你可以自由地构建一个完全功能化的架构，但你也可以选择使用面向对象程序设计实践和模式，因为该语言的部分定义是基于面向对象程序设计的概念。正因如此，你需要花时间去理解函数式编程，才能知道你在用 Go 做什么，为什么。

**基于过程化编程的经验 base:**
如果你来自 C 之类的语言，可以开始上 base Golang 教程，重新开始，因为 Go 就像一个打了类固醇的 C。在基于架构模式的情况下，您需要寻找好的实践和模式。

**基于脚本的经验:**
如果你来自脚本，因为你是一个 sysadmin 或者只知道 shell 的人，你可以在学习 Golang 的同时阅读关于编程基础的书籍和帖子。

**以前没有编程经验:**
对于每一个刚入门的人，我建议先学习一下编程的基础知识，然后可以去投一个 Golang 教程。
