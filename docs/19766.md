## [](#how-to-find-vscode-outdated-extensions)🤔如何找到 VSCode 过时的扩展！

1.  只需从侧边栏进入你的扩展视图
2.  您也可以按下 COMMAND (⌘) + SHIFT (⇧) + X)
3.  字面上，键入`@outdated`
4.  VSCode 将列出所有过时的扩展
