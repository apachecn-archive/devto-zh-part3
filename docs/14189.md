# 如何做一个网站

> 原文:[https://dev.to/reliefs/how-to-make-a-website-fkm](https://dev.to/reliefs/how-to-make-a-website-fkm)

首先，你需要一个文本编辑器，如果你使用 windows，你可以使用记事本，或者如果你想去专业搜索崇高的文本，在 Linux 去与 vim 或其他编辑器，和 mac...你自己想办法

现在你已经打开了文本编辑器，你应该开始输入。

```
<html>
<title>
My websites title
</title>
<body>
<h1>This is my website headline</h1>
Welcome to my website
<img src="https://markmanson.net/wp-content/uploads/2010/09/stones.jpg">
</body>
</html> 
```

例子:[https://jsfiddle.net/58ngefro/1/](https://jsfiddle.net/58ngefro/1/)

现在点击保存并将文件保存为 mywebsite.html，现在用鼠标在目录中打开 mywebsite.html 文件。

总是当你创建语句时，你应该记得使用<>并再次用>关闭它，这是一种让代码知道何时打开和关闭新语句的区别的方法。

现在我们有了一个基本的网站。如果你学会了这三个命令，

and . You will be able to understand every HTML line in the matter of a few seconds.

现在我们有了基本的，我们应该继续向你的新网站添加不同的东西，如图像和不同的文本。

```
<html>
<title>
My websites title
</title>
<body>
<h1>This is my website headline</h1>
Welcome to my website
<img src="https://markmanson.net/wp-content/uploads/2010/09/stones.jpg">
</body>
</html> 
```

例子:[https://jsfiddle.net/xo981m4s/](https://jsfiddle.net/xo981m4s/)

保存文件并打开它。

现在你已经创建了你的第一个网站，包括标题、标题、文字和图片。在 HTML 中你可以使用成千上万的命令。但是在这个教程中，我们只是介绍了基本的。
