# 我希望年轻时就知道的 7 件事

> 原文:[https://dev . to/Hussein _ cheayto/7-things-the-I-wish-I-know-when I-younger-135g](https://dev.to/hussein_cheayto/7-things-that-i-wish-i-knew-when-i-was-younger-135g)

从别人的错误中学习是最好的方法。我有了创建一个博客账户的想法，因为我想向全世界传播我的知识。还有，我希望你回避并意识到我之前不知道的事情。

**1-编程**

当我大学毕业时，我不知道我想获得什么样的学位。我的生活没有目标。在选择电气工程之前，我浪费了一年时间，这也不是我的选择。

在我的第二个学期，我学了我讨厌的 C++编程，我勉强及格。尽管讨厌编程，但在大学三年级时，我爱上了编程，因此我希望自己是一名计算机科学专业的学生，而不是电气工程师。我会在 3 年内毕业，并在 5 年内获得 2 年的经验，这是成为一名工程师所花费的时间。

我并不后悔成为一名工程师，但如果当时有人建议我，告诉我有一种叫做计算机科学的东西，我的生活会容易得多。

**2-时间值**

我过去常常把时间浪费在玩电子游戏上，花了无数个小时，而我本应该阅读、浏览 Youtube 或规划我的未来。

我最近意识到时间比金钱更重要，因为时间一去不复返。有效地管理你的时间会给你带来很多很多的钱，而在相反的情况下却不是这样。

我总是用这句激励我的话来提醒自己:

“在他们睡觉的时候工作，

在他们聚会的时候学习，

那就在他们花钱的时候存钱吧

像他们梦想的那样生活”

**3-励志视频**

励志励志视频让我的生活天翻地覆。在听完丹洛克、韦斯布朗、史蒂夫乔布斯、沃伦巴菲特等人的演讲后，我选择了我的人生目标，那就是:当我死的时候还活着，就像史蒂夫乔布斯、爱因斯坦和特斯拉一样。我只是设定我的目标，随着时间的推移，我将学会如何通过克服所有的障碍来实现它。

**4-失败很重要**

我以前失败时会很沮丧。一次失败导致我输掉了许多其他的“战斗”。事实上，当事情不尽如人意时，我会很早就放弃，变得沮丧和愤怒。我的建议是:永远不要放弃，生活会尽可能地打击你，但你应该永远站起来，重新战斗。让失败成为成功的孪生兄弟的是，当你再次站起来时，你会成为一个更好、更强、更有决心的你。

这是编程教给我的。我对编码充满热情，我认为如果我想成为最好的程序员，我应该克服这个挑战，并以这样或那样的方式解决问题。

**5-成为房间里最聪明的人**

这个建议真的很重要，对工作很有帮助。在你的头几周，试着察觉到没有他/她，公司将会失败。这个人应该成为你暂时的榜样。你要拿自己和他/她比，看看他/她有哪些比你更好，更有技巧的地方。根据你的分析，你应该列出一个你应该获得和提高的技能的清单，以便成为他/她的技能竞争者。下一步是当你战胜了那个人，寻找另一个“目标”。当你找不到工作时，考虑寻找更高层次的公司或工作，或者如果你有足够的经验和钱，如果你创建自己的公司将是一个很好的主意。

6-拥有这个地方

当你第一次进入一个地方时，表现得好像你是主人，其他人是你的客人。这将极大地提高你的自信和个性。

**7-将恐惧转化为能量**

韦斯·布朗总是谈论阻碍他人实现目标的恐惧。直到最近我才明白恐惧。事实上，他的意思是，当你对某项技能充满信心时，你会感到恐惧，因为你害怕失败。

在某种程度上，对一项技能感到舒适会让你感到安全。你应该承认在学习新技能时你会很糟糕，但随着时间的推移，你会克服恐惧和障碍，最终掌握这项技能。

当我决定学习 web 开发时，我面临着这种恐惧，而我对 web 开发一点概念都没有。起初，这很有诱惑力，但也令人沮丧，尤其是当我看到讲师(来自一个在线课程)如此顺从地编写大量代码时。经过一个月的努力，老师看起来太难的东西现在对我来说很容易，我可以像他一样不用看文档就能编码。
