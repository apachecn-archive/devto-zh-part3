# 我征服了它！

“征服什么？”你会问。我尝试开微博已经有两年多了。出于某种原因，我仍然没有能够在无数的草稿中发表任何一篇！

我很难决定第一篇文章应该写什么。它必须是博客的解释者，还是直接进入博客；读起来不好怎么办，听起来不连贯怎么办？所有有效的问题。

当你读这个的时候，不管它读起来好不好，不管它是否有凝聚力。没关系，因为，我终于可以出版了！(或在这种情况下提交)
