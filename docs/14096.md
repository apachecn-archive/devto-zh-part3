# 提高网站页面速度

> 原文:[https://dev.to/reliefs/increase-website-page-speed-2d4k](https://dev.to/reliefs/increase-website-page-speed-2d4k)

许多网站需要大约 2-5 分钟来加载，在本教程中，我们将看看如何在你自己的代码和一些其他框架内提高你的网站页面速度。

当你想提高页面速度时，有两件事你应该考虑，如何避免内存泄漏和如何避免相同的请求反复出现。

当看框架的时候，一个重要的东西是 Wordpress。框架是一个避免缓慢加载时间的轮子，原因是你安装更多的代码(插件)来提高页面速度，这有时会工作，但不会提高 95%的速度。

每个网站的页面速度都有可能提高 95%。如果你自己编写网站代码，停止使用 JQuery，如果你只使用它两个功能。停止为一个命令使用一个包，并停止在每个站点或每个请求上加载这个包。
