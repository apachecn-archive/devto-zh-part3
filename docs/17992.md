# 这就是飞梭赛车。

> 原文：<https://dev.to/jsrn/now-this-is-podracing-3foe>

前几天我在听最近一集的 Syntax ,他们在讨论曾经风靡一时但现在不再受欢迎的工具。

对话从[大口](https://gulpjs.com/)和[咕噜](https://gruntjs.com/)开始，我发现自己变得非常怀旧。在昏暗的路灯下从办公室走回家，我几乎发现自己变得彻头彻尾的*情绪化*。

当我开始认真研究 web 开发时，我并不真正了解这个行业中的任何人，所以我主要是通过教程网站和博客来了解最新情况。我只是偶然发现了一些好的工具和实践。最终，我读到了一篇关于 Grunt 的文章，这是自动化构建过程的时髦新方法。对我来说，这就像一瞬间从 0 到 100。

[![the quickening!](img/00a1ab44faddea6346fcd2d0d5031e0e.png)](https://i.giphy.com/media/Jx8BU2Ofy1W1y/200.gif)

无论如何，听到 Grunt 让我想起了我职业生涯的早期，我第一次拿起一些东西，感觉它让我的能力增加了十倍。

我想知道这里是否有人在他们的发展道路上有过类似的时刻？你第一次发现自己在想什么时候“现在我们用煤气做饭了！？
