# 简历审核

> 原文:[https://dev.to/kaydacode/resume-review-1oei](https://dev.to/kaydacode/resume-review-1oei)

让我们帮助获得一些行业的新鲜人才。通过检查拼写、语法来帮助他人，确保他们的简历易于阅读，以及你在旅途中遇到的任何其他你认为对他人有益的提示。

规则很简单:

## [](#juniors)后辈:

在下面贴一个你简历的链接，让其他人查看，或者得到额外的关注。

## [](#seniors)学长:

请给出反馈和回复，这样我们就不会堆积在同一个问题上(但你绝对可以提供额外的反馈)。

*如果您不想让您的个人信息出现在那里，请用占位符替换，以免布局混乱。*

这个帖子是在 Twitter 上开始的。随意跟随任何一个或两个地方！
