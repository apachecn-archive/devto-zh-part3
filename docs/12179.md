# [](#tips-for-gaining-followers)获得追随者的秘诀

既然我们已经探讨了追随者的细分，让我们来看看我为了获得追随者所做的一些事情。

我想指出的是，关注者数量绝不应该对你的自我价值产生影响。虽然我成长迅速，但人们往往需要数年时间才能获得追随者。所以请对这些建议有所保留。

### [](#post-consistently)岗位一致

我从我的同事那里得到的最大的建议是坚持发帖，即使没有人在听。总的来说，人不是一夜成名的；这需要时间。你发布的越多，你积压的内容就越多。人们喜欢关注有大量内容的人。

### [](#interact-with-the-community)与社区互动

我尽最大努力回复我收到的每一条信息，并与人们互动。这被证明是无价的。人们想知道你在听他们说话。当我与社区中的人进行积极互动时，我倾向于更加重视他们的推文。

### [](#post-relatable-content)帖子相关内容

我试着在推特上发布与所有人相关的事情，我不会总是粉饰一切。我不知道为什么越来越多的开发人员不会对求职面试失败或与冒名顶替综合症作斗争直言不讳。这些问题影响了几乎所有的开发者，公开谈论它对我的影响极大地影响了我的追随者。

### [](#be-myself)做自己

你永远不要试图成为另一个人(无论是在现实中还是在网上)。如果你因为发布不能反映你真实身份的东西而变得受欢迎，那你是因为错误的原因而变得受欢迎。

### [](#create-content-that-interests-me)创作我感兴趣的内容

我坚信你应该创造你感兴趣的内容。无论是通过博客还是制作视频教程，找到你感兴趣的平台和话题。如果你不喜欢写或谈论你创造的话题，没有人会喜欢读它。仅仅因为一个主题已经被博客讨论过，并不意味着你不应该在这个主题领域创造内容。人们以各种不同的方式学习；也许你的内容将会到达一个以前在那个领域挣扎的开发者那里。

### [](#never-set-goals-to-gain-x-amount-of-followers)永远不要设定目标来获得 X 数量的追随者

我看到过几条推文设定了达到一定数量粉丝的目标，但我敦促人们不要关注数量，而是关注你收到的粉丝的质量。如果你总是追逐数字，你永远不会快乐(无论是减肥还是关注人数)。关注你通过社交媒体平台进行的有意义的互动。

# [](#negative-effects-of-social-media-growth)社交媒体增长的负面影响

虽然我完全敬畏我在 Twitter 上取得的增长，但它也不是没有缺点。如果你的目标是扩大你的追随者基础，请注意这不全是彩虹和蝴蝶。以下是我每天都在纠结的一些问题。

### [](#time-consumption)消耗时间

我花大量时间与社区互动(每天超过 4 小时)。这感觉就像是一份全职工作。这影响了我的精神健康。这也意味着我花更少的时间做其他事情，如锻炼、写博客和阅读。

### [](#negativity)消极性

获得一个追随者会增加你收到的消极和粗鲁评论的数量。这是我很纠结的一件事。我是一个讨厌冲突的讨人喜欢的人。我还在学习如何处理消极情绪。

### [](#pressure-to-create)压力创造

我喜欢创作内容，但最近我感到压力很大，要尽可能多地推出内容。然而结果是，质量会受到影响，我的心理健康也会受到影响。学会分清轻重缓急是我每天都在努力解决的事情。

# [](#conclusion)结论

获得追随者是一种奇妙的感觉。知道有 42，000 人在阅读我的内容，我既害怕又激动。如果你的目标是获得追随者，我希望这些建议有所帮助。但请记住，无论你有 20 个追随者还是 10000 个，你都不应该牺牲自己的心理健康或价值观。
