# FindCollabs 黑客马拉松的可发现性

> 原文:[https://dev . to/vynce Montgomery/discovery-at-the-findcollabs-hackathon-4f2n](https://dev.to/vyncemontgomery/discoverability-at-the-findcollabs-hackathon-4f2n)

在某种程度上，FindCollabs [黑客马拉松](https://findcollabs.com/hackathon)旧金山分部的参与者代表了湾区科技世界的更大文化。我们来自四面八方。我们从编码训练营的学生到有几十年经验的程序员。我们是尖端创业公司的爱好者和主管，也是行业中坚分子的个人贡献者。我们带着模糊的想法、具体的计划和未完成的技能，准备好面对 Github repo 并被告知要做的事情。

这种多样性让 Krzysztof 感到惊讶和耳目一新，他希望每个人都带着自己的项目出现，而不是准备好其他的想法。他说这是他以前的经历。但是，FindCollabs 活动对开放性和创造性的关注，加上它并不局限于在非常有限的时间内亲自贡献代码，似乎让人们愿意帮助他人。见到来自底特律的玛丽安娜，我也很惊讶；我半辈子前从中西部搬到了海湾地区，但当我遇到其他人也这样做时，我仍然感到非常高兴。

即使不算在芝加哥遇到的拦路抢劫，花的时间也比 36 个小时稍长。

不过，有一位程序员特别体现了这种快乐，并带来了希望。她向杰夫做了自我介绍，并解释说，当她搬到美国并决定学习编程时，她的养生之道包括软件工程每日播客。她还不懂英语，所以她是从他的声音中学到的。她的项目是[时代精神](https://findcollabs.com/project/XtTki3y2QKVQcjVeqYAT)；一个浏览器扩展，当打开一个新的浏览器标签时，它会显示简单的投票问题。诚然，这不会很科学，但它会提供一点火花和对电脑屏幕之外的世界的认识。

这些意外发现和有机发现的时刻对诺布尔来说是必不可少的，对他来说，在我们周围的现实世界中存在似乎是生活中近乎神圣的一面，他担心我们正在数字化的竞赛中失去这一面。当我们讨论活动结束后去哪里吃饭时，每个人都伸手拿手机搜索；但是 Noble 提倡上街走走，看看什么好看。不是说他是勒德分子。他对良好的数据呈现(如可视化)感到兴奋，并希望找到使用我们的感官及其自然数据处理能力的最佳效果。以一种能让我们置身其中的方式呈现数据不是很好吗？

当然，可发现性是现代计算体验中的一个永恒话题。FindCollabs 的主项目列表是按照更新的时间顺序排列的，这意味着每次你去的时候，你都可以看到最新的项目。当然，还有根据项目类别和开放角色的过滤器，以及项目搜索工具。不出所料，这也是这些项目的共同话题。

[![](../Images/27a8537289905eb01b5640bf0a8c535d.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--c2satfkG--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://cdn-images-1.medium.com/max/1024/1%2Ae_Fe2ai1Z0LwKeHJSYxBvg.png) 

<figcaption>这似乎真的是街景之后的下一个逻辑步骤。</figcaption>

Ombi 希望通过视频直播远程查看“氛围”,从而改善发现你喜欢的地方的过程。你可以使用风格搜索来找到一家具有工业美学或者友好的公共感觉的咖啡店。其他项目包括音乐日历和临时聚会组织，以及专门针对大学生的[社交网络](https://findcollabs.com/category/social-networking)，学术合作，当然，还有硬件黑客等极客亚文化追求。探索和发现。这个世界如此之大，充满了有趣的事情，以至于我们无法一饱眼福。有时候秘诀就是睁开你的眼睛，接受你所看到的。有时候，让这些神奇的盒子帮助我们找到我们自己都不知道在找的东西会很有帮助。

如果你想把你的想法、计划或未开发的技能带到各种各样的创造性努力中，FindCollabs 黑客马拉松将持续到太平洋时间 4 月 14 日午夜。它不仅限于软件；任何有创意的合作项目都欢迎参加。详情请见[https://findcollabs.com/hackathon](https://findcollabs.com/hackathon)。
