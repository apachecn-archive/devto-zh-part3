# 大声学习意味着什么

> 原文:[https://dev . to/ashleemboyer/what-it-mean-to-learn-out-loud-BCE](https://dev.to/ashleemboyer/what-it-means-to-learn-out-loud-bce)

我第一次听到“大声学习”这个短语是在[办公时间播客](https://discoverpraxis.com/office-hours/)中。在[的这篇文章](https://discoverpraxis.com/3-reasons-learn-loud/)中，播客主持人 T.K. Coleman 给了我们这样做的三个理由。我想把重点放在他给出的第一个理由上，并告诉你如何把它应用到求职中。

假设你正在面试，他们问了你一个你不确定答案的难题。随着你和面试官之间尴尬的沉默越来越多，要求你立即做出回应并显得聪明的压力也越来越大。你越来越不舒服，大脑一片空白。我想不出答案。

事情不一定会变成那样。你可以，而且我相信大多数面试官会欣赏它，大声思考和学习。T.K .提出了与此相关的两个要点。

> 知道一个人知道什么和知道当他们不知道时和那个人一起工作是什么感觉是非常不同的。后者对那些必须和你一起工作的人来说要重要得多。因为每个人都有他们需要学习的东西，所以当你能够表现出你有耐心和坚持做这件事的能力时，这是非常有价值的。
> 
> ...
> 
> 当人们能看到你学习的时候，他们仍然会看到你在知情的情况下能有多有效。但更重要的是，他们会看到当你毫无头绪的时候，你是多么的稳定和沉着。

总有一天，你会访问谷歌，栈溢出几十次。其他的，你可能只需要去一两次。两个都好。任何明智的人都不会指望你记住工作中的每一件事。外面有太多的信息，所以我们必须保持头脑清醒，只看最重要的部分。

既然学习是任何工作的期望，那么对于面试官来说，重要的是某人如何学习，而不是他们是否记住了每个大 O 符号。展示你的学习技能的最好方法就是大声说出来。当您在白板伪代码测试中遇到难题时，请逐步完成您的思考过程。如果你卡住了，就说出来！然后继续解释你下一步要做什么。

你的搜索词是什么？你以前见过这种问题吗？你会问队友什么问题？通过*做*很好地回答这些问题，你给面试官留下深刻印象的机会是巨大的。
