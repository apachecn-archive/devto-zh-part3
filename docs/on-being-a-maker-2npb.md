# 作为一个创造者

> 原文：<https://dev.to/latazzajones/on-being-a-maker-2npb>

不先杀死一些植物，你永远不会成为一个成功的园丁。

种植和制造东西是一种技能而不是天赋。当我还是个孩子的时候，我记得人们一直在谈论我有什么天赋或者没有什么天赋。我是一个“有才华的艺术家”和“有才华的演员”，但我从来都不是一个有才华的学生或有才华的工程师。这导致了我上大学时的许多挫败感和不安全感。在我看来，我的成功或失败都是因为天赋。

直到我遇见了我的丈夫，我才明白，天赋的成分很少，很多来自于他每天表现出来的品质；努力工作，自我照顾和耐心。他是一个非常独特的人。

这些天来，我们期望立刻得到我们想要的东西。娱乐(网飞)，食物(UberEats)，交通(优步或 Lyft)，东西(亚马逊)都在我们的指尖，在一个小时内就在我们的家门口。生活在这种即时满足的新经济中，意味着人们很容易忘记有时需要时间才能得到自己想要的东西。

如果你想做一些漂亮、美味或有益的东西，你必须准备好接受失败，并根据你所学到的东西不断重复。这个过程需要时间，经历这个过程中的起起落落和挫折是一项技能，你必须通过实践来培养。开始制造东西，你会变得更善良、更有耐心，并且更擅长制造东西。
