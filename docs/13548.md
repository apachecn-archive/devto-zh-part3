这是四部分系列的第一部分:你的网站是如何组成的。你可能已经很了解你网站的这一部分了，但是确保你真的知道也无妨。每个在线网站都是由代码组成的。这是没有办法的，也没有任何替代品。这就是为什么在 web 开发中如此强调编码技能的原因。

这也是为什么编码技能之间有如此大的差异。前端开发人员负责所有用户界面/体验方面的工作，后端开发人员处理数据，全栈开发人员两者都可以做。大多数人都熟悉前端和后端这两个术语，但是这里有一些关于这两个术语如何与你的网站相关的信息。

## [](#frontend)前端

当一个人访问你的网站时，这是他们所看到的。前端。你对前端代码所做的一切都应该关注可用性和响应性。拥有一个好的用户界面会让你的网站更容易导航，确保你的网站能在不同的设备上响应会有助于良好的界面。

前端是使用 HTML、CSS 和 JavaScript 向用户显示他们需要什么的地方。HTML 是你首先能够在屏幕上显示任何东西的方式。它是所有网站的基础，也可以说是 web 开发中最容易学习的部分。CSS 只是用来设计 HTML 的样式。使用正确的样式表，你可以使用户界面 1000%的更好。

至于 JavaScript，则是前端开发的重中之重。这就是如何向 HTML 和 CSS 添加逻辑的方法。这也是你如何改善用户体验。您可以通过表单限制他们的流程，或者向他们显示消息，以帮助他们以您想要的格式向您提供您需要的数据。JavaScript 是你能学到的最有价值的前端开发语言，因为它让你接触到更多的工具，比如框架和库。

您可以使用 JavaScript 库来处理数据和布局。有很多 JavaScript 框架会让你写 HTML 和 CSS 变得更简单。你的大部分前端工作都将使用 JavaScript。如果您喜欢，甚至可以使用 JavaScript 编写 HTML 和 CSS😉。

不管你决定用 JavaScript 做什么，确保你的前端关注两件事:用户界面(UI)和用户体验(UX)。对于前端，您可以而且将会担心更多的事情，但这些都是大问题。你的用户界面保持页面有条理，不管它在多大的屏幕上被浏览。您的 UX 可以帮助用户浏览您的网站，并完成您希望他们执行的任何操作。

## [](#backend)后端

后端开发比前端开发更难，因为它承担了更多的责任🤷‍♀️.后端是处理安全问题的地方。从用户那里获取信息并将其直接放入数据库是可能的，但请不要这样做。您绝不会希望像这样处理用户输入。

这就是为什么我们有后端开发。在这里，您可以为用户提供不同的权限级别来访问站点的受限部分。这是对用户数据进行彻底验证的地方。这是您处理网站和数据库之间所有通信的地方。这就是后端更复杂的原因。

您既要与前端通信，又要与数据库通信，您必须将这些职责分开(链接)。有如此多的最佳实践和框架，你可以选择你想学的语言。有 C#、PHP、Java、Python 和 Ruby 等等。每一个都有各自的框架和生态系统。

另一种对后端有帮助的语言是 SQL。它是关系数据库的语言，你很可能会用到它。能够作为后端开发人员编写 SQL 将允许您查询数据库，以便您可以创建、读取、更新和删除数据，这将需要您做很多工作。一旦你学会了 SQL，并且对它了如指掌，你就可以开始做一些真正的后端魔术了。

没有人能从前端看到你的后端代码。他们能得到的最多是发送到后端的请求和一些信息，其中有一个很强的可能。后端代码是为了使网站尽可能防黑客攻击。

希望这解决了前端和后端之间的一些大的差异，以及它们在你的网站的大计划中的位置。你需要记住的主要事情是代码不是整个网站。至少网上有就不会。
