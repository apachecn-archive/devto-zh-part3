# 硬件理念

> 原文:[https://dev.to/maciekchmura/hardware-idea-541b](https://dev.to/maciekchmura/hardware-idea-541b)

对我来说，硬件这个词可能太大了。一个小玩意？帮手？

## [](#problem)问题

我正在学习触摸打字。现在我几乎不用看键盘就能输入所有的单词。
当我不得不写代码时，问题就开始了。除了输入变量、函数和方法的名字，我还必须输入特定语言的语法:
`: ; " ' { [ ] } - + = ( )`...至少在 JavaScript 世界里是这样。
当我命令我的“小手指”按下减号键帽时，屏幕上显示减号的过程是这样的:
`[]`退格键`0`退格键`-`。有时候我需要更多的步骤才能最终得到我想要的。或者更糟，最终我不得不看着键盘...

## [](#solution)解

在标准键盘的 J 和 F 键帽上有一个凸起的点或条。这是食指不用看就能找到主行的指示器。非常有帮助的好主意。
如果“小手指”也有类似的指标会怎样？😄在那些难以瞄准和够不着的键帽上。

[![layout](../Images/57435772ff483f5db4219b284a5a55fe.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--Ts0RVcPE--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/tpgvrxl306t0rubbihro.png)

当接触到键帽时，你可以立即知道你是否按下了正确的键。
可能只有`0`和`1`上有指示器。或者在`[`甚至`\`上会有一些额外的。
当然对形状没有限制。只有一个条件:形状需要易于区分。

怎么修改那样的键盘？
贴纸！！！
黑色乙烯贴纸。透明贴纸。任何会有这种“凸起”的东西。
