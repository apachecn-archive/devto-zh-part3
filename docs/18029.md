# node . js 运行时环境到底是什么？

我们已经告诉过你 Node.js 是一个基于 Javascript 的运行时环境。但是，你一定想知道它是如何工作的，Node.js 开发是如何发生的？让我们带您简单浏览一下。

节点运行时环境包含了运行一个成功编写的 Javascript 程序所需的一切。Node.js 构建于 Chrome V8 Javascript 引擎之上，使用非阻塞 I/O 阻塞模型。I/O 模型代表输入/输出模型，它被认为是既可扩展又高效的。非阻塞 I/O 认为每个 I/O 调用都必须被应答。更简单地说，非阻塞 I/O 消除了对多线程的需求，因为服务器能够一次处理多个请求。

*现在让我们浏览一下顶级移动应用开发者使用的一些框架。*

**离子:**

Node.js 框架做的移动 app 开发中覆盖面最广的名字。这个框架更好的一点是它是免费的。它在客户端工作，帮助构建基于 Javascript、HTML 和 CSS3 组合的应用程序。它让 web 开发人员能够非常快速地自由构建、测试和部署跨平台的移动应用。此外，HTML5 元素有助于移动应用程序开发人员制作混合应用程序。Ionic 框架提供了灵活性，因为您的应用程序可以轻松地从一个操作系统转换到另一个操作系统。此外，它与 Angular 这样的软件开发工具包配合得非常好，并且使用 Phonegap 和 Cordova 插件

**莫吉托:**

Mojito 基于雅虎的模型视图控制器框架。由 Cocktail 支持，这是一个雅虎内置的基于 JavaScript 的移动开发平台。它包含了一组技术，使得构建和托管在客户端和服务器端环境下都能完美工作的应用程序变得容易。基于 JavaScript web 应用框架的 Mojito 可以同时在浏览器和服务器上运行。这为开发人员提供了灵活性，使他们可以为后端和前端的客户端使用单一的 JS 代码库。JavaScript、Node.js、HTML5、YUI3、CSS3 和 YQL 是 Mojito 应用程序的构建支柱。

**Express.js:**

Express.js 可以算作最好的 Node.js 框架之一。该框架包含超过 26000 个网络和移动应用。这种流行的原因可能是由于大量的好处，如高性能、高测试覆盖率、多路由潜力、HTTP 助手、内容协商、数据库集成和模板引擎。非常适合移动应用程序开发，它支持大量插件，并拥有出色的路由 API。有了圆滑的轮廓，它给了开发者对他们项目的最大控制。要使用 Express，有必要了解客户机-服务器如何工作，当然还有 HTML 和 Javascript。该框架使用了 MEAN stack 的后端以及 MongoDB 数据库和 Angular JS。

**Sails.js:**

Sails 框架的构建理念是缩小实现和开发之间的差距。所以，如果你问我们，这个框架让你可以灵活地创建从小型聊天应用到跨国公司应用的任何东西。你只需要几天时间就能让它运转起来。对于高级应用，可以使用框架的 MVC 模式。Sails 的一个很棒的特性是你可以将你的代码连接到你选择的数据库。甚至与许多其他竞争对手不同，您可以在 SQL 和 NoSQL 数据库之间轻松转换。此外，Android、IOS、React.js、Angular.js 等兼容性特性以及 REST APIS 和 WebSockets 等不同包的可用性使它成为一个高选择。该框架在需要实时数据更新的地方非常有用。所以，很多大公司如威瑞森、微软等。正在使用风帆。

因此，如果你正在寻找建立一个独特而强大的移动应用程序，Node.js 是最好的选择。
