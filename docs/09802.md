# [](#youtube-downloader-buttoniframe-api)Youtube 下载器按钮/Iframe API

Youtube 到 mp3 转换 API

#### 将 HTML iframe 的“src”属性设置为:

```
https://api.download-lagu-mp3.com/@api/button/{format}/{YouTube-Video-ID}
```

#### 其中`**{format}**`是下列之一:

*   " **mp3** ":对于 mp3 结果
*   "**视频**":用于带音频结果的视频
*   "**视频流**":仅用于视频结果
*   "**音频流**":针对纯音频结果
