代码编辑器是基本的编程工具。代码编辑器是我们编写脚本的地方，程序员大部分时间都在代码编辑器中度过

它们是两种类型的代码编辑器

*   集成驱动电子设备
*   轻量级编辑器

## [这里](#ide)

IDE(集成开发环境)是一个功能强大的编辑器，具有许多功能，如自动完成、调试器、构建自动化、版本控制系统(git)等。它在*项目级别*上运行。

通常 IDE 是重型软件，因为它在开始时加载更多的数据，分析项目结构。因此不建议资源有限的系统使用。

Webstorm 是著名的 web 开发平台之一，它是一个付费平台。

> 对于初学者来说，不建议使用 IDE 作为代码编辑器。

## 轻量级编辑器

轻量级编辑器简单但快速。

它们主要用于即时打开和编辑文件。

对于像自动完成、调试这样的额外功能，您可能需要安装额外的插件。

**轻量级编辑器**

*   [VSCode](https://code.visualstudio.com/) (微软，开源)
*   [Atom](https://atom.io/) (Github，开源)
*   [括号](http://brackets.io/) (Adobe，开源)
*   [崇高的文字](https://www.sublimetext.com/)

编辑的选择是个人的，选你最喜欢的。

> 我最喜欢的是 VSCode，我通常用它来做项目和简单的文件编辑。
