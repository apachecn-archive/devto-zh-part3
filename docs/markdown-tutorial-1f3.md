# 降价教程

> 原文：<https://dev.to/manish169/markdown-tutorial-1f3>

本教程教你 Markdown 语言的基础。Markdown 是一种轻量级标记语言，用于为纯文本添加格式。让我们开始吧。

# 标题

您可以在 Markdown 中添加一个标题，方法是在您想要放置的标题前添加#。例如:

# 标题 1

## 标题 2

### 标题 3

#### 标题 4

##### 标题 5

###### 标题 6

在一次减价中，你最多可以列出 6 个标题。

# 段落

对于段落，你需要使用一个空行来创建一个新行，并在那里添加文本。例如:

这是第一段。

这是第二段。

这是第三段。

您可以在 Markdown 中添加任意多的段落。这和我们在纯文本中做的是一样的。

# 换行

在 Markdown 中，要添加换行符，您需要键入两到三个空格，然后按 return 键。下面是一个例子:

这是第一行。

现在你有了第二行。

您可以在 Markdown 中添加任意多的行。

# 强调

## 加粗

要加粗一个文本，你需要把文本用双星号**加粗**

## 斜体

要倾斜一个文本，你需要把文本放在一个星号中，即*倾斜*

## 粗体&斜体

Markdown 提供了一个可以同时加粗和倾斜文本的功能。请参见下面的示例以获取更多参考信息。

***粗体&斜体*** 。

# 屏蔽引号

整体引语(也称为长引语或摘录)是书面文件中的引语，作为段落与正文分开。

因此，为了使用文本作为块引用，您需要在想要设置为块引用的文本前面使用尖括号' > '。例如:

> 这是块引号中的第一行
> 
> 这是块引号中的第二行。

# 嵌套块引号

您可以在块引号内使用块引号。例如:

> 这是一个单引号。
> 
> ```
>  This is a nested block quote. 
> ```

其语法是:

```
>This is a single block quote.
>
>      This is a nested block quote. 
```
