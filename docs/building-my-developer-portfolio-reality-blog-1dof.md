# 构建我的开发者组合现实博客。

> 原文：<https://dev.to/rapulu/building-my-developer-portfolio-reality-blog-1dof>

今年，我下定决心要建立我的在线作品集，我希望在年底之前，我能写下我是如何得到我的第一份梦想中的远程工作的文章。

我已经意识到，不管你有多内向，如果你不把自己放在那里，你可能最终不会取得很大成就或任何事情，不幸的是，我属于这一类。我是一个技术爱好者，一个很长时间的开发者，但是我不能指出一个我参与的成功项目。我有机会建造的所有项目不是停工就是不成功，那么你如何把它加入到你的投资组合中呢？

[![Learning how to work](img/9a6c72f95bd318e963eed89170d9dfea.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--4ipr9pMX--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/zmze9iaca9esn256wxjv.jpg) 
今年我像一个婴儿一样重新开始学习如何走路，我是一个初学者，学习编写代码的最佳方法，如何在团队中工作，如何使用最新的开发技术，建立我的操作系统档案，技术博客，基本上我会随着技术的变化而变化，并希望在此过程中获得我的远程工作。

我已经决定在博客上记录我的活动和经历，目前我正在参加两个无偿的远程实习项目，尽管其中一个实习项目声称如果你达到一定的水平，他们会发给你现金。好吧，我不在乎我是来体验的。

敬请期待，我会在博客中记录实习的每一步！
