# 设置我的新 Mac(第 1 部分—我使用的应用程序)

> 原文：<https://dev.to/zellwk/setting-up-my-new-mac-part-1the-apps-i-use-2n98>

我 2019 年的第一个任务是弄台新电脑。我不想换电脑，但是我的旧电脑坏了，我别无选择😭。

因为我已经在换电脑了，我觉得分享我日常使用的应用程序会很有趣。

我希望你会对其中一些感兴趣！

## 用于编写应用程序

我做了很多写作工作(为博客，也为我做的课程)。以下是我使用的应用程序:

1.  [尤利西斯](https://ulysses.app)。这是我做创意策划、概述、起草、写作和编辑工作的地方。
2.  火箭:火箭帮我写表情符号，就像我在 Slack 上写一样。例如，要获得😀，我打`:grinning`。它有自动完成功能，所以输入表情符号变得轻而易举😁。
3.  [海明威 app](http://www.hemingwayapp.com) :我用海明威 app 来确保我的句子易读易懂。
4.  语法上:我用语法来帮助我改正任何明显的语法错误。
5.  字数统计应用程序:这是一个菜单栏应用程序，它会告诉我今天在电脑上写了多少字。我的最高记录是 12000 字！
6.  标记 2 :我使用标记 2 将 Markdown(来自 Ulysses 或我的文本编辑器，VS 代码)转换成 HTML 或富文本。然后，我将富文本粘贴到 [Convertkit](http://mbsy.co/clb2r) 中的 WISYWIG 编辑器中(用于发送电子邮件)。
7.  [Skitch](https://evernote.com/products/skitch) : Skitch 帮我为文章快速注释图片。
8.  Kap 帮助我制作看起来不那么滑稽的 gif 和视频(但是 gif 很大...).

## 申请编码

当然，编码是我工作的一大部分。否则我怎么称自己为开发人员呢？🤪

1.  [Visual Studio 代码](https://code.visualstudio.com):我首选的文本编辑器。我制作了一个[视频系列](https://www.youtube.com/playlist?list=PLht38HefjmzGWN0CUHGqjliTSuhXFTDG5)来帮助你设置 VS 代码，如果你想使用它的话。
2.  [iTerm2:](https://iterm2.com) 比起内置的终端 app，我更喜欢 iTerm2。这更容易配置，我可以有分屏(这使得多个终端窗口更容易管理)>
3.  [塔](https://www.git-tower.com):比起其他 Git 客户端，我更喜欢塔。如果您想尝试 Tower，可以使用`zell20`获得第一年的八折优惠！(我分享这个不拿钱)。如果你想学习 Git，我也有一个[视频系列](https://www.youtube.com/playlist?list=PLht38HefjmzGpNHWKlTLQAbPlwFRFd-2z)给你。
4.  [Postman](https://www.getpostman.com) : Postman 好调试 API。
5.  MongoDB Compass :这有利于检查我的 MongoDB 数据库中存储的数据。

## 我使用的浏览器

像其他 web 开发人员一样，我在自己的电脑上安装了许多浏览器(主要用于测试)。以下是我使用的一些:

1.  Safari:我浏览网页的主要浏览器
2.  [Chrome 金丝雀](https://www.google.com/intl/en/chrome/canary/):我调试 JavaScript 的主浏览器。
3.  [Firefox Nightly](https://www.mozilla.org/en-US/firefox/channel/desktop/) :我调试 CSS 的主浏览器。
4.  [Chrome](https://www.google.com/chrome/) :调试用。
5.  [Firefox](https://www.mozilla.org/en-US/firefox/new/) :用于调试。

(如今，我很少用 IE 或 Edge 进行测试，因为通常不再需要这样做了)。

## 设计应用

我可能长得不像，但我也对设计感兴趣。我设计我建立的每个网站。以下是我使用的工具:

1.  Sip 是一个简单的颜色滴管工具。
2.  [草图](https://www.sketchapp.com):用于设计
3.  [字体](https://typefaceapp.com):用于管理我 Mac 上的字体。比字体书好多了！
4.  [Adobe 字体](https://www.adobe.com/sea/creativecloud/desktop-app.html):用于将 Typekit(现在的 Adobe 字体)中的字体同步到我的电脑上。

## 制作视频的应用:

我已经创作视频大约 1 年了。以下是我一直在使用的应用程序:

1.  [Audacity](https://www.audacityteam.org) :用于录制音频
2.  [Screenflow](https://www.telestream.net/screenflow/) :用于记录我的屏幕和面部
3.  [OBS](https://obsproject.com) :用于在视频上定位我的脸。我希望在未来的某个时候可以用它来进行直播。
4.  [网络摄像头设置](https://itunes.apple.com/sg/app/webcam-settings/id533696630?mt=12):我所在地区的灯光设置为 50hz(美国为 60hz)，因此我的网络摄像头在录制时会闪烁。这是正确的。
