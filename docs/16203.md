# 使用 Python 进行令人惊叹的数据科学

> 使用 Python 实践数据科学的优秀资源的精选列表，不仅包括库，还包括教程、代码片段、博客文章和演讲的链接。

#### 核心

[熊猫](https://pandas.pydata.org/)——构建在 [numpy](https://www.numpy.org/) 之上的数据结构。
[scikit-learn](https://scikit-learn.org/stable/) - Core ML 库。
[matplotlib](https://matplotlib.org/) -绘图库。
[seaborn](https://seaborn.pydata.org/)——基于 matplotlib 的数据可视化库。
[熊猫 _ 汇总](https://github.com/mouradmourafiq/pandas-summary) -基本统计使用`DataFrameSummary(df).summary()`。
[熊猫 _ 简介](https://github.com/pandas-profiling/pandas-profiling) -描述性统计使用`ProfileReport`。
[sklearn_pandas](https://github.com/scikit-learn-contrib/sklearn-pandas) -乐于助人的`DataFrameMapper`类。
[缺失号](https://github.com/ResidentMario/missingno) -缺失数据可视化。
[彩虹-csv](https://marketplace.visualstudio.com/items?itemName=mechatroner.rainbow-csv) -要显示的插件。颜色漂亮的 csv 文件。
