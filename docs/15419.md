# JavaScript 从哪里开始？！

> 原文：<https://dev.to/pantsme/where-to-begin-with-javascript-fm5>

所以我想我应该开始用 JavaScript 记录我的旅程，这样我不仅可以和别人分享我的旅程，还可以写下我走了多远的书面记录，这样我就可以回头看了。

### 背景

我已经在 IT 界的网络和系统方面工作了 8 年多。我最近被一家在线公司聘为基础设施工程师/开发工程师，这家公司非常适合我。我厌倦了现场 IT，我可以肯定地看到，基于云的基础架构是大多数 IT 专业人员想要进入的下一个趋势领域。

当我在这里的时候，我学到了很多新的东西，并真正开始与开发人员携手进行部署，对我们应用程序的不同部分进行故障排除，并使用一些较新的工具，如 Octopus、Jenkins 和 Docker。这真的让我很兴奋，因为我知道当我从里到外了解了一个系统，剩下的就是日常维护时，我会感到厌倦。

现在，促使我开始学习 JavaScript 的原因是看到所有的开发人员能够在他们的头脑中创造一个想法。我脑子里有太多很酷(至少对我来说)的想法，但我无法实现，因为我不知道如何编码。这是我过去 15 年来一直想做的旅行，我想现在是时候了。

### 旅程开始

几周前，我开始决定我想学哪种语言，这是 Python 和 Javascript 之间的一场势均力敌的比赛。我选择 JavaScript 仅仅是因为它可以被广泛使用，以及在 node.js 方面，你可以在服务器端使用它。我觉得这不仅对我现在的工作很方便，对我的个人项目和职业发展也很方便。不要误解我，我认为 Python 非常强大和有用，一旦我很好地掌握了 JavaScript，我可能会涉足它。

所以首先，我需要找到一个可以学习的资源。我做了一些谷歌搜索，在网上找到了一个又一个资源。这是令人生畏的，我可以说我花了很多时间只是寻找新的资源，而不是学习。我知道我的学习方式不是仅仅读一本书，然后让它在我的大脑中根深蒂固。我需要锻炼，大量的锻炼。我正在浏览的两个网站是 [JavaScript.info](https://javascript.info) 和 [FreeCodeCamp](https://www.freecodecamp.org/) 。也特别感谢 Android 上的 Grasshopper 应用程序让我无论在哪里都能学习。我喜欢 freecodecamp 上的内置练习，我也喜欢 javascript.info 在大多数课程结束后给你布置任务。我发现这非常有帮助，因为我必须在自己的机器上完成任务，保存文件，并创建一个我学习的知识库，如果有什么事情变得有点模糊，我可以回顾和重温。我的评论技巧已经无与伦比了！哦，还有一个提示，对于任何编程新手来说，在大多数编辑器中注释一行代码，点击`Ctrl+/`或`Ctrl+Shift+/`来注释一段代码！

### 目前为止我在哪里

到目前为止，我有变量，打印，警告，和一些与循环的工作。我还学习了数组和字符串以及如何操作它们。这只是目前的基础，但我正在取得进展，我为此非常高兴。当我现在看到一些基本代码时，我至少可以破译它试图做什么。进度很慢，因为时间过得很慢，而且只有这么多，但是我越来越喜欢学习，而不是缺乏动力。这对我有好处。很少有事情能保持我的注意力，我有一种感觉，编程将是其中之一。

随着我的进步，我将在这个系列中发布更多的帖子。如果你对学习 JavaScript 的人有任何建议，请告诉我们。听到已经了解他们的东西并知道他们的建议的人的意见总是好的。感谢您花时间阅读我的故事！
