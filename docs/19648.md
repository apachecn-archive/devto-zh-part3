# 令人困惑的期望带来的挫败感

在被告知微波炉不含金属多年后，按照我的新微波炉上的说明烧水时使用勺子还是感觉怪怪的。有一张贴纸告诉我，当用微波炉加热水时，我必须加一个勺子，但我没有看到自己这样做，因为“无金属”是如此根深蒂固，我已经看到了很多这样做的人的闪电视频。

<figure>[![A lift control panel with a numeric keypad and the letter C in the display](../Images/e649e054f783fc5f3f812cfd1c583e13.png)](https://craignicol.files.wordpress.com/2018/11/dsc_0045.jpg) 

<figcaption>电梯设计中的抽象</figcaption>

</figure>

这张照片来自当地医院的电梯。你按下一个楼层号码(而不是上/下),显示屏会显示一个字母，告诉你该乘哪部电梯。这个额外的抽象让许多用户感到困惑，因为我在喝咖啡休息时帮助人们使用它们。

我在很多网站上都看到了同样的事情。那些“很酷”的网站，当我向下滚动的时候，它们会通过横向移动或者放大来打断滚动。打破后退按钮的 SPA 网站。在移动浏览器上使用 outlook web access 很痛苦。我不断收到“您确定要重新发送吗？”或者当我试图从电子邮件返回我的邮件列表时的登录提示。

我从来都不是一个伟大的视觉网页设计师，[我绝对是一个功能超过形式的类型](https://craignicol.wordpress.com/2016/04/15/smart-is-subtle/)，但是我总是从被称为渐进增强的原则开始。让它以最简单的方式工作，并在一个强大的、可访问的基础上添加铃铛和口哨。这个基础已经从 HTML，到 HTML+CSS+JS，到我最新项目中的 TSX，从桌面到移动，但它始终基于文本和形式的基本原则:易于阅读，易于遵循，并以你期望的方式工作。

如果你试图通过打破期望来脱颖而出，你想隐藏什么？
