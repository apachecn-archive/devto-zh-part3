# 然而，凯西·洛特曼编码了(以及更多)

我是一名 web 开发人员，但首先，我是一名社区成员。我的社区包括 Twitter 上的女性技术人员、在红色州争取变革的进步人士、 [r/weddingplanning](https://www.reddit.com/r/weddingplanning) 、[租户权利和经济适用房倡导者](//facebook.com/RentersTogetherLNK/)、我的读书俱乐部、在 [Glitch](https://www.glitch.com) 上制作酷东西的人，等等。我的社区是驱动我的动力——编码是我的工作，但不是我的生活，即使那些社区成员中的一些只是因为我编码才有意义。

## 2019 年我继续编码是因为...

我需要那美元来支付我的抵押贷款。

[![Woman getting rained on by dollar bills](../Images/aeded937ecc8a34a7b174f6215ead6de.png)T2】](https://i.giphy.com/media/5fBH6z8aMV1RbA4FaSc/200w_d.gif)

还因为我对自己的新角色 [@glitch](https://www.glitch.com) 社区健康工程师感到非常兴奋。我的工作是建立工具，以确保随着故障的增长，我们不会成为其他大型社交网络变成的那种少数群体的地狱。这份工作结合了我在 Javascript 方面的技能和我在个人和网络上建立包容性社区空间的热情。

*   我在上一份工作中于 2018 年交付的一些大项目，特别是围绕我们的 PHPUnit 测试套件的主要速度改进。
*   我今年用 Javascript 做的所有愚蠢的兼职项目。我已经很久没有以编程为乐了，被构建在 Glitch 上的人们惊人的创造力所包围，激励着我加入进来。
