嗨，我是 Alexandru，一个年轻的 PHP 开发人员。我在 web 技术方面有超过 6 年的经验，在现在不存在的 Flash 方面也有几年的经验。

我已经阅读了这个社区的文章有一段时间了，并感到有灵感写一点关于我自己的东西。

关于 http://codingrealm.com/早期的更多信息

我对 web 开发充满热情，尤其是设计系统架构和后端代码。我确实涉猎了 Vue.js 和 js...但是我不太关心 CSS(我的大脑从来没有正确地思考过它:P)。然而，我确实喜欢 UX 的任何东西。

经过多年的 Flash 动画和迷你应用，我从零开始开发 PHP 和 web。这很艰难，至少一开始是这样。我不得不一步步来:

*   旧代码库调试
*   使用 Mysql 集群、内存表、数十个工人等的分析引擎
*   Wordpress 开发
*   客户购买的随机“框架”,他们后来希望一切都改变
*   新的应用程序开发，从 Laravel，v3 到 5
*   领导创建和维护网站的团队，现在为我们的客户推动整个业务

我的经历、漫漫长夜、随机错误(看看你的表情符号和 strlen)、打破 IE 的 console.log 以及许多许多其他东西帮助塑造了我，现在作为回报，我正在努力塑造我工作场所的年轻开发人员。

我想我会试着多写一些，我已经开始喜欢上它了。
