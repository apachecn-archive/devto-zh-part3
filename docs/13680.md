# 阅读对作家的重要性

> 原文：<https://dev.to/georgefaulkner/the-importance-of-reading-for-writers-45dm>

我们经常听说阅读对提高写作技巧至关重要。如果你认为你能写作，并且你有能力帮助你以写作为职业，最重要的因素是你大量阅读。成为一个热心的读者可以在很多方面帮助你提高写作技巧。同样重要的是要知道，如果你不喜欢阅读各种类型的书籍，你可能不会在你的写作生涯中出类拔萃。阅读是高质量写作的基础，如果你想成为一名故事作家、散文作家、编剧或小说家，你需要明白阅读将是你成为作家的基础。在这里，我们将讨论阅读将如何影响你的写作模式，并有助于你提高写作技巧。

**增加词汇量**

阅读的最大好处是它极大地增加了你的词汇量。每个单词都有多种意思和用法，只有通过阅读，你才能学会以多种方式使用一个单词。此外，当你成为一个狂热的读者时，你会开始理解一个单词在一个句子中的各种用法。你开始理解和学习新的短语，这将有助于你通过单词和各种句子结构来表达你的想法。

**增强想象力/**

阅读能提高你的想象力。当你阅读的时候，你想象发生的场景，甚至根据书面解释描绘人物。它帮助你认知成熟，批判性地分析各种情况，理解并证明给定的解决方案。所有这些练习都有助于训练你的大脑吸收各种情况并做出相应的反应。阅读不会提高你的写作技巧，但会使你提高解决问题的技巧，这在实际生活中是必不可少的。

**介绍各种写作风格**

当你阅读各种体裁、不同经典和现代作家的作品以及各种出版物时，你会逐渐开始认识和理解不同的写作风格。对写作风格多样性的理解将使你能够发展你自己特定的写作模式。对于一个作家来说，有一种独特的写作形式是很重要的，这将使他在联盟中有别于其他人。

**了解最新趋势**

阅读名著和著名作家的文学作品能使你增加见识，纠正错误，增加对现实生活的了解，但阅读你那个时代的作家和现代畅销书会增加你对最新趋势、读者想读的话题和普通读者关注的现状的了解。此外，阅读将提供一个关于现代写作方法的综合概念，包括短语和行话的用法。

在生活中灌输纪律

当你明白你必须通过阅读来提高写作水平时，你需要制定一个时间表来反复灌输定期阅读。你会自动感受到生活中的纪律和规定。在你的日常生活中定期阅读将训练你的大脑每天吸收词汇、观点和趋势，导致稳定的成长。此外，为了获得最好的结果，定义一个你必须完成阅读材料的时间表。指定的时间框架将增加纪律性、调节性和自制力，当你将写作作为你的职业，并且必须每天在截止日期前提交书面作品时，这将对你有所帮助。

**难以想象的学习成果**

尽管我们已经讨论了大量常规阅读的结果，但阅读提供的接触和理解是不可想象的。大脑的认知增长从阅读中加速，这导致了整体人格的发展和提高，阅读提供的理解能力是不可想象的。大脑的认知增长因阅读而加速，这导致整体个性和个性的发展和改善。
