# 开源 Pocket CLI

> 原文：<https://dev.to/ildella_18/open-source-pocket-cli-42ec>

大家好。

在我关于 dev.to 的第一篇文章中，我想让你注意我最后的努力:一个新的 [Pocket CLI](https://github.com/ildella/pocket-cli)

是一款面向 [Pocket](https://getpocket.com) 的开源交互式终端应用，这是一款著名的“稍后阅读”应用，于 2017 年被 Mozilla 收购，现在默认安装在 Firefox 中，带有 Chrome 插件和移动设备及电子阅读器应用。

虽然在浏览器或移动设备上添加文章和阅读要舒服得多，但有些任务在 CLI 上更容易完成，比如某种搜索、组织和清理。

该应用程序现在支持标准功能，如搜索、在浏览器中打开文章、存档/删除/收藏文章。所有这些都在一个彩色的交互式 CLI 中，*应该*在所有操作系统上工作。但是刚刚推出，所以请给我反馈错误和改进。

它是用 Javascript 编写的，当然是在 Node.js 中运行。它使用最新的 Javascript 特性，因此可以在 Node 11+上工作。

是在 [GitHub](https://github.com/ildella/pocket-cli) 上开源的。检查它，克隆它，启动它，在 Twitter 上给我一个反馈(【https://twitter.com/ildella】)。

享受吧。
