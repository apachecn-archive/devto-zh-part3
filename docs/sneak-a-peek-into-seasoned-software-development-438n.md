# 窥探经验丰富的软件开发

> 原文：<https://dev.to/recursivefaults/sneak-a-peek-into-seasoned-software-development-438n>

有没有想过其他人是如何工作的？配对或 TDD 看起来如何？希望能够向从事这项工作多年的人提问？想看退伍军人努力工作吗？

我和我的朋友 [@schmonz](https://twitter.com/schmonz) 时常会聚在一起，在某件事情上结对编程。

我们想通过在 Twitch 上播放打开我们合作的大门。

告诉我们您是否感兴趣，以及您想看什么。请告诉我们您希望从我们的流媒体会议中获得什么。我们会乐在其中，我们也想让你们乐在其中！

关于我:我写软件和教练团队已经十年了。我的使命是帮助创建改变世界的团队。我在软件开发的职业生涯中指导过一些人。人们说我有一种独特的天赋，可以解决让别人望而却步的问题。

关于 schmonz:他没有给我任何东西，所以我告诉你我的想法。首先，他是我见过的最有洞察力、最聪明的软件开发人员之一。他深思熟虑，熟练地解决问题，当我回头看他的作品时，毫无疑问比他发现的要好。全世界的人都称赞他们从他身上学到了很多东西。我期待着我们的每一次交谈和一起工作。

无论如何，让我们知道你的想法！
