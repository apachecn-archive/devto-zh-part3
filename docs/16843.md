# 马克！马克

[![](../Images/fa79ea73c5617fea4b17d85cc9b11723.png)T2】](https://raw.githubusercontent.com/gsantner/markor/master//app/src/main/ic_launcher-web.png)

**Text editor - Notes & ToDo (for Android)**
Simple and lightweight, supporting Markdown and todo.txt

[![Get it on F-Droid](../Images/c542d98a93a10bd64b8e6b6a0126cff4.png) ](https://f-droid.org/repository/browse/?fdid=net.gsantner.markor) [![Get it on Google Play](../Images/b689418020732e1cc707e72e0cde59a1.png)](https://play.google.com/store/apps/details?id=net.gsantner.markor) 

Markor 是 Android 的文本编辑器。这个项目旨在制作一个通用的、灵活的、轻量级的编辑器。Markor 利用 Markdown 和 todo.txt 等简单的标记格式进行笔记和列表管理。马克尔在处理文本方面是多才多艺的；它还可以用来保存书签，复制到剪贴板，快速打开文本链接等等。创建的文件可以与任何平台上的任何其他明文软件互操作。Markor 使用开放格式，是自由软件，公开开发并接受社区贡献。

之前我在[我用来写博客的工具](https://dev.to/bauripalash/tools-i-use-for-blogging-1h92)中提到，我仍然主要使用 Markor 来写✏的文章和做笔记。

事实上，我喜欢马克尔的很多地方(奇怪的是😉) :

*   这是 FAAAAAAAST....（😉我甚至用它在我的镇上漫游😞

*   每个文档都存储在通用文件系统文件夹中，因此我们可以使用其他应用程序进行编辑(😉**以前我用相机做笔记**😞)

*   它完全离线，没有一个 http 请求😉**不用担心，保管我的超级骗子 Huper 的秘密核密码**😞)

*   高度可定制(😉我甚至可以把它变成独角兽😞)

*   以 HTML 和 PDF 格式转换、预览和共享文档(😉**我已经扔出 MS/Libre Office** 😞)

*   它甚至有一个黑暗模式❤(😉**没有臭味**😞)

*   很明显，它是免费开源的❤❤

玩笑归玩笑，我真的认为 Markor 是一个非常好的写作应用，我每天都用它，你有时也应该试试..🙂
