# [](#textua11y-whats-coming)🖍️ textua11y:什么来了

不幸的是，几个因素让我别无选择，只能让 textua11y 成为一个私有存储库。在我再次打开它之前，我需要制定一个许可证和投稿指南。如果您有任何问题或建议；但是，请在 [Twitter](https://twitter.com/ashleemboyer) 上联系或在下面发表评论。

至于更令人兴奋的事情——特性和功能，下面是我的待办事项清单(排名不分先后):

### [](#1-fix-the-layout-and-make-the-sidebarcontent-scroll-separately)1。修复布局并使侧栏/内容单独滚动。

这将是一个快速的。目前的情况在大多数屏幕高度下都很好，但是如果你碰巧想让你的窗口变小，它就不再漂亮了。

### [](#2-make-the-sidebar-collapsibleexpandable-on-mobile)2。使侧边栏在手机上可折叠/可展开。

这个工具并不打算成为一个移动工具。我假设大多数人花时间在电脑上设计 web 应用程序和网站。也就是说，它应该最低限度地适用于移动设备。该链接的大部分分享是通过 DEV 社区网站或 Twitter 进行的，这意味着大多数人都在通过手机预览该应用程序。我想留下一个好的第一印象。

### [](#3-improve-the-color-picker-component)3。改进颜色选择器组件。

颜色选择器工作得很好，但我认为它们需要更容易使用。选择颜色后，除了刷新页面，没有其他方法可以“重新开始”。工作量太大了。应该有一个简单的解决方法:在最后一步选择颜色之后处理一次额外的点击。此外，我想添加更多的文本来给用户提供更多关于他们的选择或他们的鼠标悬停在什么上面的信息。
