### [ [特色](https://github.com/jarun/nnn#features)[[快速入门](https://github.com/jarun/nnn#quickstart) ] [ [插件](https://github.com/jarun/nnn/tree/master/plugins#nnn-plugins) ] [ [Wiki](https://github.com/jarun/nnn/wiki) ]

`nnn` ( *n* )是一个功能齐全的终端文件管理器。它很小，接近 0 配置，而且[快得令人难以置信](https://github.com/jarun/nnn/wiki/Performance)。

它被设计成不引人注目的智能工作流，以匹配思路。

`nnn`可以分析磁盘使用情况，批量重命名，启动应用程序和挑选文件。插件库拥有大量插件来进一步扩展功能，例如[实时预览](https://github.com/jarun/nnn/wiki/Live-previews)、(卸载)挂载磁盘、查找&列表、文件/目录差异、上传文件。一个[补丁框架](https://github.com/jarun/nnn/tree/master/patches)托管相当大的用户提交的补丁，这些补丁本质上是主观的。

独立(neo)vim 插件- [nnn.vim](https://github.com/mcchrish/nnn.vim) 、[vim-float term nnn 包装器](https://github.com/voldikss/vim-floaterm#nnn)和 [nnn.nvim](https://github.com/luukvbaal/nnn.nvim) (neovim 独占)。

运行于 Pi， [Termux](https://www.youtube.com/embed/AbaauM7gUJw) (Android)，Linux，macOS，BSD，Haiku，Cygwin，WSL，跨 DEs 或一个严格的 CLI env。

[*(还有更多)*](https://github.com/jarun/nnn/wiki/Basic-use-cases#the_nnn-magic)
