# 什么是 IssueHunt Fest？

IssueHunt Fest 是为期一个月的 OSS 赞助活动，在每年的五月和十二月举行。

在活动中，IssueHunt 向赞助商募集资金，并将资金分配给 IssueHunt 上列出的 GitHub 项目。

在发行过程中有大量的资金问题。我们做赏金猎人吧！⛴💰

事件的日期和时间:

*   开始时间:2019 年 5 月 1 日
*   结束时间:2019 年 5 月 31 日

# 我们开始这个活动的原因是

最初，我们将 IssueHunt 作为维护操作系统的工具，名为 [Boostnote](https://github.com/BoostIO/Boostnote) 。

尽管 Boostnote 的大部分开发都是基于社区的，但我们没有办法奖励我们的开发人员。在与我们的团队讨论了这个问题后，作为一种方式向维护 Boostenote 的开发者表示感谢。

在启动 Boostnote bounty 计划([boost note bounty 的一篇文章](https://medium.com/boostnote/boostnote-bounty-program-f3d14c0fc87d))后的短短一周内，我们就收到了数百个拉取请求。

我们认为我们做的这个程序可能有助于建立一个可持续的开源生态系统，所以我们决定把它剥离出来，成为 IssueHunt，一个开源软件的慷慨平台。

著名的开源项目如 Jekyll、Material-UI、AntDesign、NW.js 等已经加入 IssueHunt。
