# [](#still-havent-found-what-im-looking-for)还没找到，我要找什么

我正在寻找一个完美的邮件客户端，这似乎是一个永无止境的旅程。我经历过像航空邮件和 Spark 这样的东西，但它们都缺少某种东西，因为它们都是专有的，*我对此无能为力。*

[除此之外](https://github.com/dploeger/shimmer)

我刚刚上传了一份文档草稿到 github repo 中，这是一个现代的、独立于平台的、开源的、闪亮的新邮件客户端，我暂时称之为“Shimmer ”,我希望像我一样热爱电子邮件的所有人都能参与它的设计和开发过程。

*   请参加讨论。公开存储库中的问题，讨论文档的某些方面。给我你的两分钱
*   如果你想参与开发过程，请举手。我不想一个人去做。为此我需要聪明的头脑。请说明你对建筑的哪一部分感兴趣
*   如果你有其他可能感兴趣的同行，请分享这篇文章和回购协议

让我们共同努力。

谢谢您们。
