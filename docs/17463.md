# 让我们再试一次:新博客

> 原文:[https://dev.to/sezginege/let-s-try-again-new-blog-58gg](https://dev.to/sezginege/let-s-try-again-new-blog-58gg)

我真的很想停止在我的博客上发布这样的状态更新。我相信 Twitter 是一个分享这些更新的好平台，但是让我们破个例，我希望，这将是最后一次。

## [](#what-happened-to-the-old-blogs)[老博客怎么了？](#what-happened-old-blogs)

我有一些在不同平台上写博客的经历(Blogger/Wordpress)。我正在写关于技术的东西，我对我的流量很满意。然而，在某些时候，我失去了动力，我想，从技术角度来看，我分享的博客文章不够好。所以，我不再写博客了。

## [](#what-has-changed)[发生了什么变化？](#what-has-changed)

我相信我现在有更多有趣的东西可以分享。距离我上一次写博客已经过去几年了。在那些年里，我找到了一个机会与不同的团队在不同的有趣项目上合作，并且不得不解决太多的问题。我开始在远离祖国的国外生活。我去过很多地方，遇到了很多有趣的人。所以，我变了😋

## [](#the-motivation-behind-this-blog)[这篇博客背后的动机](#the-motivation-behind-this-blog)

我们在日常生活中都面临不同的问题，我们做的第一件事就是查看别人的博客帖子/维基页面/新闻等。很高兴看到外面有太多的人准备帮助别人，但我相信这个过程应该是相互的。

## [](#what-kind-of-blog-posts-you-should-expect)[你应该期待什么样的博文？](#what-kind-of-blog-posts-you-should-expect)

这个博客将成为我分享各种话题的经验和想法的地方。但很可能，主要内容将是关于技术的东西。

## [](#conclusion)[结论](#conclusion)

我真的很高兴能再次写博客，我希望，我能让这个博客活很多年！
