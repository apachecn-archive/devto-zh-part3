# 去哪里找我

> 原文：<https://dev.to/rrees/where-to-find-me-4olc>

这是一个很棒的服务，它提供了一个简单的索引页面，可以让你链接到不同的网络身份。我看到人们在 GPlus 上使用它，为那些还没有决定 GPlus 关闭后自己确切位置的人提供了另一种保持联系的方式。

作为一名开发人员，当我看到它时，我有点自责，因为它正是我喜欢的那种简单实用的 web 服务，我完全有可能创建它并允许其他人共享它。

相反,[完全归功于](https://wheretofind.me/about/)[基特·拉·图什](https://wheretofind.me/@kit),他不仅仅是想出来的，而且建造了它并分享了它。
