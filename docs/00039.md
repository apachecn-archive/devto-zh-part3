## [](#databases)数据库

AWS 有两种类型的数据库可用，SQL 或关系数据库(RDS)和 NoSql 非关系数据库(DynamoDB)。

## [](#rds)RDS

RDS 是一个 SQL 数据库，为不同的引擎提供了大量的选项。在这里你会看到亚马逊 Aurora，Mysql，Postgres，Oracle 和微软 SQL server。

## [](#dynamodb)DynamoDB

DynamoDB 是一个 NoSQL 数据库服务。如今 AWS 的目录中没有另一个 NoSql 选项。实现这一点的一种方法是创建一个 EC2 实例，并安装它 MongoDB 或 Oracle NoSql。
