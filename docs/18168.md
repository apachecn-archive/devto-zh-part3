#### Q1:我如何开始学习编码，从哪里可以获得资源？

学习的艺术取决于个人。有些人喜欢有具体时间表的学校系统。有些人喜欢一对一的辅导，有些人喜欢小组或同伴学习，而有些人可以灵活地适应技能。

无论哪种方式适合你，重要的是你首先要了解自己。这样你就不会浪费时间和金钱。

稍加研究，你就会找到一个能满足你需求的家教或服务。以下是一些在我或我的一些同事的编码之旅中对他们有所帮助的网站列表。

*   [FreeCodeCamp.org](http://freecodecamp.org)
*   [Codecademy.com](http://codecademy.com)
*   [Udemy.com](https://udemy.com)
*   [Udacity.com](https://udacity.com)
*   [Flatironschool.com](https://flatironschool.com/)
*   [pluralsight.com](http://pluralsight.com)

上面列出的一些网站收取合理的费用，这是值得你学习的技能。从免费课程开始是有益的，它让我回到浪费时间和金钱的状态。

免费课程可以帮助你调整学习内容和学习方式。值得一提的是，这些免费课程有时足以让你领先一步。

有很多免费的课程，有些是由脸书、谷歌和微软这样的大科技公司赞助的。

YouTube 上有些人给初学者提供了很棒的内容。以下是一些受欢迎的 YouTubers 视频列表:

*   共同课程
*   网络忍者
*   设计课程
*   旅行媒体
*   卢珀图斯

#### Q2:我需要计算机科学学位才能成为软件开发人员吗？

计算机科学学位不是软件开发人员的先决条件。如果你有机会得到一个去吧。

建立在任何让你脱颖而出的东西上是一个很好的练习。当你开始申请工作时，大部分的空缺职位都有计算机科学学位。

#### Q3:你会推荐我从哪种编程语言开始？

推荐一种编程语言取决于很多标准。

我想指出的是，每一种编程语言都是强大的，可以开始使用，并且可以解决任何问题。下面的答案应该作为一个指南。

下面的一些建议可能会因您所在的位置而异。我建议你用谷歌搜索你所在的地区，但一般来说，这是可以的。

#### 那么我们开始吧😊

如果你的兴趣是学习编码或者开始理解编程的概念。

**推荐语言** : Python 或者 PHP。

**理由:**简单易学，庞大的用户社区，生产中使用最多的语言，业内最受欢迎的编程技能和大量可供学习的资源。

如果您的兴趣是为特定目标构建，如移动或桌面。这同样取决于您的目标平台:

*   ***对于安卓* :
    推荐语言:** Java 或者 Kotlin，但是我推荐 Java。**原因**:庞大的用户群体、生产中使用最多的语言、业内最受欢迎的编程技能以及大量可用的学习资源。撰写本文时，Android 社区正在用 Kotlin 取代 Java。Kotlin 和 Java 有一些相似之处，包括语法。但是你应该想跟上技术吗？我建议您学习 Kotlin，即使社区和资源可能没有 Java 大，只要稍加研究，您就可以开始学习了。
*   ***对于 iOS* :
    推荐语言:** Objective-C 或者 Swift 但是我推荐 Swift。**理由:** Swift 是苹果团队在 2014 年推出的，目标是简化对 iOS 基础编程的理解，简单易学，易于理解，是 iOS 应用程序的现代语言。同样需要注意的是，Objective-C 一直是 iOS 和整个苹果家族的核心支柱，选择学习 Objective-C 可能会让你在 iOS 上有更强的基础。
*   ***对于 Windows* :
    推荐语言:** C#或者 C++但是我推荐 C#。理由: C#是由微软的一个团队开发的，以桌面和 web 为主要目标，简单易学，易于理解，但对于软件工程的深度知识来说，C++是一个更好的选择，因为它让你接触到更复杂的问题解决技术。
*   ***游戏开发* :
    推荐语言:** Python，Java，C#或者 C++但是我推荐 C++。**原因:**一些最流行的游戏引擎都是建立在 C++之上的(虚幻引擎和 Unity 引擎)。
*   ***用于 Web 开发* :
    推荐语言:** HTML、CSS、JavaScript。这三个是构成网络界面的主要工具，学习这三个是很重要的。它们简单易学，易于理解，社区庞大，资源丰富。可以从 HTML 开始，以 JavaScript 结束。

注意:其他值得一提的编程语言包括:Ruby、Rust、Haskel、Go 等等。

#### Q4:作为软件开发人员，我可以使用哪些工具？

> 我知道你知道这个，但是首先，你需要一台电脑😂。
> 
> 既然白象已经出了房间。😉

这与您的计算机类型或系统配置无关。从任何开始，现在甚至手机都可以编码，但如果可能的话，让我们坚持使用 PC。

有一些应用程序可以帮助您完成编码之旅。虽然大多数电脑都预装了这些应用程序。一些个人、组织和社区已经构建了类似的应用程序，只添加了很少的功能，为您(开发人员)的项目提供了灵活性和权力。

您可能需要的应用程序或工具包括:

*   文本编辑器:一个用于编写纯文本文件的软件，你选择学习的任何编程语言都可以使用一个简单的文本编辑器。常见的**文本编辑器**有 Windows PC 的记事本和 Mac PC 的 TextEdit。文本编辑器的一些替代品: [VSCode](https://code.visualstudio.com/download) ，[括号](http://brackets.io/)，[原子](http://atom.io)，[崇高文本](https://www.sublimetext.com/download)等等。
*   大多数语言使用支持工具，如编译器、终端等。一些软件将这些支持功能捆绑到一个应用程序中。这些软件应用程序被称为 IDE ( **集成开发环境**)。最流行的 ide 是 Android Studio——面向 Android 开发人员，Visual Studio——面向 Windows 开发人员，X-Code——面向 Mac 开发人员，等等。
*   对于网络开发来说，你只需要上面提到的任何文本编辑器和一个**网络浏览器**，常见的网络浏览器是 Internet Explorer/Edge——对于 Windows PC 和 Safari——对于 Mac PC，这些浏览器的一些替代品是 Firefox、Chrome 和 Opera。我推荐 Chrome，因为它提供了一个非常友好的环境，让你的代码调试变得容易。
*   **GIT:🤔是的，不要对我发誓，但你需要钱。GIT 是一个帮助你跟踪你的代码过程的软件，它使其他人在你的项目上的合作变得简单。GIT 很容易学，但是我建议你给它一个完整的教程时间表。**

#### Q5:什么时候可以开始求职？

读完这篇文章后😉。是的，你有兴趣成为一名软件开发人员的事实是一个开始。一些公司已经准备好与没有经验的人合作。

求职面试本身就是一个需要实践和训练的过程。最佳实践是始终进行在职培训。参加工作面试可能会产生以下两种结果之一:你得到了这份工作，或者你知道了你需要在哪些方面提高自己的技能。

#### 结论:

学习软件开发是令人兴奋的，但是仅仅知道编码是不够的:你需要学习如何学习和忘记🙃。学会理解而不是记忆。

做好在工作中学习的准备，永远知道自己的价值和极限，就像《人与国王》的故事一样。
