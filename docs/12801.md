# 你如何创造准确的报价，并坚持下去？

> 原文：<https://dev.to/twentyonehundred/how-do-you-create-accurate-quotes-and-stick-to-them-11bi>

我和我的同事们有时会纠结于创造准确的报价/评估/时机，不管你怎么称呼它们。这些报价*通常*以小时或天给出，转换成实际的$$$成本。困难的部分是使它们尽可能准确，这样我们就不会让**范围蔓延**，突破**我们的预算**，并最终与客户进行非常**尴尬的对话**。

似乎它们是攻击这一点的两种主要方式；要么你尽你所能事先尽可能多地了解任务或项目**、*或*你**故意在项目和预算中留下一些模糊性和空间**以留出一些回旋余地。**

 **这两种选择似乎都有积极和消极的一面，就像前的*一样，很难锁定一些客户可能出现的每一种可能性。我有时会花一个小时、一天或一周的时间在一个规范上，而且(除了已经花了一周的时间之外)我仍然会被我或客户忽略的东西或我们认为会不同的东西弄得措手不及。*

然而，*后面的*选项，也就是我们让项目变得有些开放的地方，在最初是一个更困难的销售，并且*似乎真的没有*解决这个问题并且帮助我变得更准确。

我是从机构的角度来解决这个问题的。因此，项目、客户甚至技术都会有很大不同。我们尽我们所能销售我们最了解的东西，但这总是在发展和变化。

我真的很想听听其他人在这种情况下或类似情况下会怎么做！**
