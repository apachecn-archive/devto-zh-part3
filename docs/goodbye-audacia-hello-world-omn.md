受到 James Turner 最近关于开始发展旅程新篇章的帖子的启发，我想我应该写一篇公开告别 T4 奥达西亚咨询公司的文章，以及我在那里遇到的所有优秀的人。和我一起成长，看着我成长的人

*我甚至参与了其中一些的设计*

### 第一:开始

我记得当我第一次遇到可编程计算机时，我大约 6 岁:Amstrad CPC 464(这是一台可怕的英国计算机)

*嘶:我在[这里](https://blog.gaprogman.com/2014/02/my-first-computer-the-i-ever-used-or-programmed/)* 之前已经写过那台机器了

从一开始，我就喜欢我能告诉这台电脑做事情。这是一个很常见的开发历程的背景故事，有一个关于它的播客由[肖恩·怀尔德穆斯](https://twitter.com/shawnwildermuth)制作

他甚至正在制作一部关于此事的纪录片

我们最终得到了一台 Windows 3.1 机器，我喜欢摆弄 autoexec.bat 文件，不得不创建启动软盘来启动一些游戏

*X 翼 vs Tie 战斗机，我看着你*

但是我们主要用它来帮助我们做作业——我妈妈用它来通过她所有的大学课程。

大约在 2000 年，我有了自己的机器。它有一个英特尔奔腾 CPU，大约 64 兆内存，和(可能)500 兆硬盘。

这些可能不是正确的规格。已经快 20 年了

我喜欢那个东西，尽管我在拥有它的第一天就毁掉了 Windows，而且它必须重新安装。

我在大学里上编程课是很自然的事情

我一直在提到大学，但是因为我来自英国，所以这是 16-18 岁年龄段的人

然后继续攻读计算机科学学位，希望加入游戏开发行业。这个梦想没有实现，因为我发现了系统和 web 开发，并爱上了让服务器做疯狂的事情。

从那以后，我创建了与实验硬件对话的 R&D 应用程序(使用 WinForms)；无数的 web 应用(既使用 ASP NET 框架又使用 Core)；移动应用程序(使用 JS 技术和少量 Xamarin 的组合)；一些一次性的应用程序帮助我在播客领域保持高效；为一家公司管理 DevOps 部门。

到目前为止，这是一段艰难的旅程，而这仅仅是个开始。
