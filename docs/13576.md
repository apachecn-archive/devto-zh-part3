# [](#how-to-make)如何制作

## [](#material)材料

-电机和齿轮箱 x2
-底座为木头或塑料
-移动电池

首先，建立齿轮箱和电机。你可以在玩具店买到它。

并像木头或塑料一样固定在底座上。

[![](../Images/d541834ba88dd2b6157ddeb22ac24447.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--PfHBsE-a--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://storage.obniz.io/images/168b417bc553e6391534332825922)

最后，将这些连接到 obniz，并将 obniz 和电池放在底座上。

## [](#how-to-move)如何移动

运行程序。你会看到按钮。
按住一个按钮向前移动并转弯！
