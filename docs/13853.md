# 生活可能会很艰难，但我们非常擅长我们的工作！

> 原文:[https://dev . to/kaelscion/life-can-tough-but-we-dad-good-in-what-we-do-2g9o](https://dev.to/kaelscion/life-can-be-tough-but-we-are-damn-good-at-what-we-do-2g9o)

德弗士，借我你的耳朵...眼睛...耳朵？不管怎样，听着。过去的六个月对我个人来说很艰难。健康问题总是很糟糕，但当它们几乎要了你的命时，就真的很难受了。类似这样的事情会改变你看待世界的方式。也就是说，当你突然被迫承认你确实很重要，即使是以一种很小的方式，并且在事情发生之前，你只能忽视你的身体和思想的信号这么长时间，这可能是一种超现实的体验。我周一刚从重症监护室出来，我需要和那些和我一样需要帮助的人分享这些好处。因为如果没有支持、鼓励和无条件的爱，一个社区还能有什么意义呢(你们中的一些人有时会喜欢无条件地接受爱。你知道你是谁，我还是喜欢你😋)?

所以，这就是我们要做的:任何阅读这篇文章的人固有地认为他们所做的事情，无论是个人的还是职业的，都太微不足道了。对于我们所有人来说，认为“初级开发人员”就是“没有天赋或技能的咖啡爱好者”。对于那些越来越难反驳其他人的断言的人来说，开发运维/质量保证/前端/数据库管理员工作只是用来骚扰“真正的天才”的垃圾，以便“公司可以对让非关键员工参与重要工作感到满意”。任何从事科技行业的人，无论是女性、棕色皮肤还是黑色皮肤，戴着头巾、头巾或浴衣工作，或者只是因为外界的废话而被人看不起，或者有胆量将自己的文化或宗教传统置于他人的极端观点或无知意见之上，这是给你的。无论你是患有人们认为你在装病的看不见的疾病，还是人们认为你应该改变的看得见的外表差异，我们都要展示我们所有人的“老板”。

这是我的贡献:现在我正在为我的一个客户建立一个系统，这个系统有一个最低级和最精确的植物转基因生物测试套件。他们让我给他们一份全球玉米乙醇生物燃料市场参与者的“市场快照”,以便他们更好地知道如何根据他们的目标客户作为一个行业的发展方向对他们的产品和服务做出决策。为了做到这一点，我使用我的数据爱好者技能集，找到美国所有参与者的公司信息(位置、员工人数、总收入、股票价格等)。下面是两个脚本运行时的截图，这两个脚本从美国大陆的 204 个玩家的网站中提取特定的数据。top runtime 是我为此设计和创建的一个解决方案。第二个是我在网上找到的一个“复制粘贴”解决方案，稍微修改一下就可以执行与我的定制解决方案相同的任务。

“嘿，等等，杰克。你在这里哗众取宠？”你怎么敢问这个完全真实、准确、恰当的问题！但是说真的，这就是这个帖子的内容。我是一个独行侠，一个后端的人，一个建造那些从未见过也从未被认可的东西的人。多年来我一直如此。我通常不喜欢自吹自擂。但是在这里，在这个帖子里，吹牛吧。因为时不时地，我们需要向那些愿意听我们说话的人大喊，我们有多聪明，有多有创造力，有多聪明。没有人会为我们做这件事，所以去吧！发布截图或讲述你做过的事情的故事，这些事情你认为非常酷、有创意、聪明或优雅。跟我说说，社区里关于你最喜欢的项目或你过于紧张的绝妙解决方案不够“酷”，不足以得到你同事的积极关注。他们不仅足够酷，而且对那些家伙来说也太酷了。所以告诉我们，不要谦虚😁 😁 😁！我非常期待你的意见！
