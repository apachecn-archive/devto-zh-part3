# 工具

## Web 框架加固

*   no surf-Go 的 CSRF 保护中间件。
*   [gorilla/csrf](https://github.com/gorilla/csrf) -为 Go web 应用&服务提供跨站点请求伪造(csrf)预防中间件。
*   [gorilla/secure cookie](https://github.com/gorilla/securecookie)——为 Go web 应用程序编码和解码经过验证和可选加密的 cookie 值。
*   secure 是一个用于 Go 的 HTTP 中间件，可以满足你对 web 应用程序的大部分安全需求。
*   [未索引的](https://github.com/jordan-wright/unindexed)——替代禁用目录索引的`http.Dir`。
*   [beego-security-headers](https://github.com/gosecguy/beego-security-headers)-beego 框架过滤器，方便安全头管理。

## 图书馆

*   paseto-GO(Golang)中平台无关的安全令牌实现。
*   [hsts](https://github.com/StalkR/hsts) - Go HTTP 严格传输安全库。
*   [jwt-go](https://github.com/dgrijalva/jwt-go) - Golang 实现 JSON Web Tokens (JWT)。
*   [HTTP probe](https://github.com/tomnomnom/httprobe)-获取一个域列表，并探测工作中的 HTTP 和 HTTPS 服务器。
