# 技术面试

> 原文:[https://dev.to/thepa/technical-interviews-7f8](https://dev.to/thepa/technical-interviews-7f8)

太多的人问我参加技术面试是什么感觉。尤其是刚从大学出来的。当你开始申请湾区或其他地方的初创公司甚至成熟的技术公司时，你可以期待什么。

你的面试过程将分为三步。第一个是电话面试，这通常是与某人谁将审查你的适合，在这一阶段，你将有机会被淘汰的运行最快。第二阶段是与非技术人员进行面对面的面试，这个人将会面试你是否适合当前的管理风格。通常，在这次面试中，你会进入第三种类型的面试，但要确保你搞定这一次，因为他们会投票决定你是否适合这个职位。第三个是技术面试，这是你可能在白板上做一些结对编程或潜在的某种类型的编码示例的地方。你也可能会有一系列口头面试问题，这些问题可以衡量你对你所知道的编程语言的总体了解程度。这可能是一次非常艰难的面试。有时非常密集，因为这些会议可以持续几个小时。

一般来说，当你试图申请一份计算机科学相关的工作并被录用时，这是你将会经历的三种面试。由于它与非常具体的工作职能和你可能会看到的面试问题类型有关，我整理了一系列面试问题模板，可以帮助你为即将到来的与招聘经理或技术经理的接触做好准备。
