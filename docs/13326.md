### [](#new-house-new-lifestyle)🏠新房子=新生活方式

我的 2018 年的开始并不理想。这是相当累人的一次。我们最近(2017 年 11 月)搬到了新房子，所以这是一个适应新邻居、结交新朋友、记忆新旅行计划和路线等的艰难时期。除此之外，我们还有大量的贷款和债务要偿还给所有在我们困难时期帮助我们购买梦想中的房子的人。(我们爱你们所有的❤️人，并将永远感激你们 _ 😊️ ***)。***

我们逐月偿还债务，牺牲了几乎所有东西，比如披萨、电影、购物等等。告诉过你这是令人精疲力尽的时间。然后一月过去了，二月一日到来了。这种令人疲惫的开始变成了**“灾难开始”**。

> 我突然被新加入的工作解雇了。

为什么？因为我的雇主疯了。还是因为当初我傻到加入那家公司。还是因为我厌倦了在之前的工作中对抗负能量和肮脏的政治。或者是因为我想更快地换工作。雇主无法完成他们投标的项目，我被提前录用了。太早了。现在我成了他们工资单上的无用资源。所以，他们友好地请我回家，并付清了我所有的欠款。那么，没有任何事先通知就礼貌地要求回家算不算礼貌？我不这么认为。

> 对雇主说:不太好。它甚至不接近尼斯。你在我已经很艰难的时候破坏了我的生活。我不会忘记那可怕的一天和你带来的耻辱。至少你可以要求道歉。也许这能让我松一口气。

我在那个有毒的工作场所呆了两个月，这已经让我很沮丧了。我活得像一个行尸走肉。我已经开始找新工作了，但是在找到新工作之前，我不准备离开，以保持稳定的收入。

> 为了欣赏生活和美好的事物，你必须先经历不好的甚至更坏的事情。

在那家公司唯一好的事情是一个非常好的朋友，他给了我希望，积极的能量，让我的一天充满欢笑，度过了美好的时光，并给了我最美好的回忆。

> 对你——犯罪中的伙伴:虽然我们在一起工作的时间很短，但这是一个非常危险的地方。我希望我们能在更好的条件和更好的工作场所相遇，在那里我们可以互相教和学一些东西，并为 Android 和 iOS 开发出优秀的应用程序。但是让我说:一个非常特别和重要的**谢谢你所做的一切**，包括那些长时间的午休和我们一起喝的无限量的茶。谢谢你

### [](#freelancing-nightmare)👻自由职业=噩梦

这是一个大胆的声明。我知道。但是，至少这些年来对我来说是真的。我认识很多优秀的人，他们从事全职或兼职的自由职业&提供服务，他们在自己灵活的时间里在家工作，收入颇丰。我已经在所有平台上尝试过太多次了，包括本地和国际客户。但是，这不是我的事。句号。我不能做自由职业者。我也不会做自由职业者。我这边已经证实了。

我已经开始找新工作了。但是在我找到新工作之前，我有一个正在进行的自由职业项目。我想完成它，并得到一些钱来保持收入稳定。我对借钱给我的人许下了许多按时还债的承诺。但是现在游戏完全不同了。我不得不为家庭开支、水电费等安排资金。更别说偿还债务了。所以，这个自由职业项目成了我生存的手段。

但当你处于艰难时期时，通常一切都会变得颠倒。这发生在我身上。这个自由职业项目成了我的噩梦。它让我彻夜难眠。这分散了我对面试的注意力。这基本上毁了我的生活。当我拿到报酬时，已经太晚了。我告诉我的朋友和家人为什么我不能偿还债务。他们中的大多数人都非常支持和理解我，帮助我度过了这一切，不再有任何问题。但我做的更多的是从新人那里借更多的债，偿还给老人们，以延长我的时间线。

事实上变得很压抑。我一直很紧张，任何人都能从我脸上看出来。最重要的是，我有一个可怕的客户，他无限的电话，信息，不停的压力，不切实际的最后期限，以及没完没了的变化和工作。是的，你可以争辩说我没有更好地管理它或类似的事情。我告诉过你。自由职业不适合我。句号。

* * *

### [](#looking-for-job-depression)🔍找工作=抑郁

在那份可怕的工作之前，我已经马不停蹄地工作了 4 年，甚至没有更新过我的简历。所以，我基本上忘记了如何找工作，如何通过这些奇怪的面试和浪费时间的招聘过程。这些年来，我学到了很多东西，并且在 Android 开发方面做得更好，我的投资组合中有大约 100 多个应用程序。已经到了这样一个地步，当我得到任何在 android 上开发的需求时，很可能我已经在做类似的事情了。所以，我的 Bitbucket 账户变成了一个更好更个性化的 StackOverflow，Github，Gists，或者任何你想寻找答案的资源。

然后我有了开源、android 库和良好的 Github 简介的完整体验。你知道吗，我的一个开源库， [**EasyFlipView**](https://github.com/wajahatkarim3/EasyFlipView) ，已经被全球的 Android 开发者下载了超过 4 万次。

<figcaption>我一生中最骄傲最好的成就——[我的书](https://www.amazon.com/Wajahat-Karim/e/B00IQMGFU8/)</figcaption>

所以，我的印象是我已经得到了所有这些好东西，好的个人资料，Github，书籍，作品集等等。找份工作会有多难？不是我去找工作，而是有很多招聘人员在找我。我真傻，对吧？但事实证明，没人关心这些事情。他们(招聘人员)关心的只是继承的定义是什么，抽象类和接口有什么区别。或者我能不能在白板上写 Dijkstra 算法？在两个月的时间里，我接受了 40 多次采访。它变得如此令人沮丧和沮丧，以至于我不再相信自己。我的自信心空前低落。

> 我想:“我真的是一个开发者吗？这些年我做了什么？我做的这些库是我自己的还是我只是个假小丑？我真的有资格成为 android 开发者或开发者吗？”

我在另一篇文章中写了这件事，以及我是如何恢复自信并找到一份令人惊叹的工作的。

### [](#new-remote-job-happiness)💼新的(远程)工作=快乐

所以，经过大约 40+次面试，无数的笔试，在线测试，无限的问题，愚蠢的项目和任务，不眠之夜，焦虑的脸，疲惫的头脑，妻子，家人和朋友的大力支持，以及 2 个月的奋斗，我得到了一份很棒的工作，工资和福利很高，时间灵活，在家工作，几乎没有政治文化。

> 感谢 [**Contour Software**](http://contour-software.com/) 和 [**Trapeze Group**](https://www.trapezegroup.com/) (加拿大)给我这个机会从事如此大规模的项目。

[![](../Images/870ec4bfb94427174d8cee050337c512.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--gIiFvVYB--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://cdn-images-1.medium.com/max/1024/0%2ArFJL_RCjGyCYdvw2.jpg) 

<figcaption>我在 Contour 软件的工作站上安装了入职培训套件</figcaption>

在这份工作之前，我的所有工作主要是关于服务提供和基于消费者的应用程序。但在 Contour Software，我很幸运地为 Android 开发了一个企业级应用程序，并成为了 real team 的一部分。我们遵循严格的敏捷方法，对敏捷和 scrum 完全陌生，这是一次真正疯狂和具有挑战性的经历，我学到了很多新东西。

我不知道为什么我以前没有尝试过敏捷，但是现在我已经掌握了这个概念并且理解了它是如何工作的。现在，我也在我的个人项目中遵循这个原则(稍后会有更多的介绍)。就像我在做一个人的 scrum 系统。我有一篇文章计划写关于它和我到目前为止所取得的成就。

由于在 Contour Software 的工作提供了灵活的时间安排和在家工作的能力，我开始更多地探索这一点。我开始享受这种自由，这让我的生活变得轻松多了，我可以更加专注和自信地关注个人、家庭和职业。此外，远程工作经常会混淆你的工作和家庭，模糊两者之间的界限，所以这也成为一个挑战。(今年，我还计划写一篇关于如何管理远程工作和个人生活的文章)。

### [](#open-source-motivation)👍开源=动机

我的整个 2018 年都是关于开源和为它做贡献的。我称自己为**“偶然的开源贡献者”**因为老实说，我是偶然进入开源的。有时候，当你害怕某样东西，你正在逃避它，那么在紧张和恐惧中，你并不真的在乎该走哪条路，你只是跑着，跑向你首先看到的任何一条路。这就是我进入开源的故事。我逃离了自由职业，最终成为了一名开源贡献者。

[![](../Images/aa4402b6d13fd94d5ea872eda94a5d66.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--7KmCYFVv--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://cdn-images-1.medium.com/max/784/1%2AkFkAIxqTdCeXsXLsmA9MkQ.png) 

<figcaption>我的[在 Github 上开源投稿](https://github.com/wajahatkarim3?tab=overview&from=2018-12-01&to=2018-12-31)2018</figcaption>

我开始大量创建开源项目和 android 库。除了作业(远程)，我只做这个。

不自由职业。

没有创业。

没有副业。

我每天花 3-4 个小时创建新的库，做新的实验，修复错误，回答问题，并为其他项目做贡献，如 [Mozilla Mobile](https://github.com/mozilla-mobile) 等。为了更严肃地对待开源，我在 Twitter 上参加了 [**#100DaysOfCode** 挑战赛](https://twitter.com/hashtag/100DaysOfCode?src=hash)。每一天都工作并有所贡献是艰难的，但这是值得的，令人愉快的，也是一种人生经历。在这篇文章中，我分享了我的整个经历和我从中得到的东西。

[开源 100 天代码](https://android.jlelse.eu/100-days-of-code-in-open-source-3ceb38474a89)

## <g-emoji class="g-emoji" alias="computer" fallback-src="https://github.githubassets.com/images/icons/emoji/unicode/1f4bb.png">💻</g-emoji>安装

将此添加到应用程序的 build.gradle 文件中:

```
dependencies {
  implementation 'com.wajahatkarim3.clapfab:clapfab:1.0.6'
}
```

或者在 pom.xml 中添加 ClapFab 作为新的依赖项

```
<dependency&gt
  <groupId>com.wajahatkarim3.clapfab</groupId&gt
  <artifactId>clapfab</artifactId&gt
  <version>1.0.6</version>
  <type>pom</type>
</dependency>
```

### [](#public-speaking-community)🎤公共演讲=社区

[![](../Images/7eca588ccd5f080bd287f6cf68639d29.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--wRkcCJ0J--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://cdn-images-1.medium.com/max/1024/0%2AUjRaP8BEyzvRHINj.jpg) 

<figcaption>所有盾牌、卡片、贴纸等的集合。2018 年——我喜欢谷歌 IO 盾❤</figcaption>

2018 年 6 月，我开始涉足公共演讲和当地社区贡献。我上一次做公开演讲是在 2012 年，所以获得一个机会和平台有点困难。因为本地没多少人知道我。他们可能看过我的文章或 Github 项目，但他们并不了解我个人。因此，我开始给当地的活动和社区活动组织者或技术会议等打电话，请求他们让我在他们的活动中谈谈我的经历和学习。

我在开源和 Github profile 方面的工作帮助我验证了我的经验，我得到的第一个机会是在 [*NED Karachi*](https://www.neduet.edu.pk/) 的一个项目展览中担任评委，这是巴基斯坦工程和计算机科学领域的顶级大学之一。感谢 [**阿纳斯阿尤比**](https://medium.com/u/3e65873c331d) 为我牵线**萨姆拉赛义德**和**赛义德穆罕默德哈斯奈恩**邀请我成为 NED 年度活动 [ITEC 2018](http://www.neditec.org.pk/) 的评委。

这是我第一次担任展览评委，我见证了许多有趣的项目，从人形说话机器人到生物和农业技术解决方案，到移动应用程序和游戏，再到数据科学实验。我特别喜欢的一个项目来自一个 14 岁的年轻程序员，他使用[光子引擎](https://www.photonengine.com/en/pun)创建了一个具有实时多人游戏功能的 3D 第一人称射击(FPS)游戏。我很惊讶一个 14 岁的孩子怎么能使用如此先进的工具来制作一个完美的游戏。

> 我要向 [**穆罕默德·哈立德**](https://medium.com/u/49f4e27ed6bc) 大声欢呼，感谢他给了我一整天美好的陪伴，在大学的道路和公园里漫步，在自助餐厅吃了一顿美好的午餐，随后喝了许多杯茶，还有无数的笑话和谈话。谢谢你，伙计，让那一天变得难忘。

在这次展览之后，我得到了一个关于一个我想都没想过的事件的演讲机会。那是由 [**Google 开发者集团(GDG) Kolachi**](https://www.facebook.com/GDGKolachi/) **举办的 *Google IO Extended 2018* 。**再一次，非常感谢 [Anas Ayubi](https://medium.com/u/3e65873c331d) 为我接通了 [Eric Bhatti，](https://medium.com/u/4142dad0ad21)他邀请我谈论“Android 有什么新功能？”在 Google IO Extended 2018。感谢埃里克的谈话。

作为一名 android 开发者，这对我来说意义重大。我非常紧张，因为这是我 5 年来第一次公开演讲，而且是在这么大的平台上。埃里克让我感到舒适，并相信我。令人惊讶的是，那天的情况变得非常奇怪。那天晚上我几乎没有睡上 3 个小时。我的演讲是在上午 10:30。由于晚上没有睡觉，我不得不花 4 个小时直接去会场演讲。这是一个冒险的计划。我甚至不确定我是否能按时参加活动。埃里克又一次非常支持和配合。我参加了活动并做了演讲。主题是[“Android 有什么新功能？”](https://speakerdeck.com/wajahatkarim3/whats-new-in-android)。你可以在我的 [SpeakerDeck 个人资料](https://speakerdeck.com/wajahatkarim3) [这里](https://speakerdeck.com/wajahatkarim3/whats-new-in-android)查看幻灯片。

会谈进行得很顺利。大厅里挤满了人。人比椅子多，所以很多人站着，一些人坐在空地的地板上。这是 2018 年最美好的经历之一，它恢复了我对公开演讲的信心。那天我变得如此积极和兴奋。后来，我从观众那里得到了很多好的反馈，很多人通过社交媒体、电子邮件等与我联系。这次演讲后，我交了许多新朋友和新关系。
