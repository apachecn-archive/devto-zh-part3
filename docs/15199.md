# 走这条路 Swift 软件开发学习指南

> 原文:[https://dev . to/ronanrodrigo/follow-this-path-a-study-guide-for-software-development-with swift-5419](https://dev.to/ronanrodrigo/follow-this-path-a-study-guide-for-software-development-withswift-5419)

我在学习中总会遇到的一个困难是对我应该学的东西进行优先排序。最大的问题总是“这重要吗？”“这难吗？”“这门课会花我很长时间吗？”这些问题可能会助长拖延。

为了解决这个问题，我和一些朋友创建并组织了一个学习指南。它包括三个层次，每个层次都有许多任务要完成。所有任务都有优先级，从最基本的主题到最高级的，从最重要的到最不重要的。我们的目标是让本指南成为初学者和已经从事软件开发的专业人员的参考。

## [](#how-to%C2%A0study)如何学习

鉴于看板的普及，学习指南是在 Trello 中创建和组织的，这是一种虚拟看板。对于那些不知道的人来说，看板是一种组织一些任务，然后执行的方案/框架。要使用此板，您必须遵循一些规则:

*   永远不要跳过一个任务，永远从上到下拿第一；
*   避免从一级(栏)跳下，如果你完成了上一级，只进入下一级；
*   当你开始研究一个题目(卡片)时，把它移到学习栏；
*   学习栏只有一个科目；
*   使用卡片描述字段写下您对该主题的了解；
*   使用附在卡片上的链接作为内容参考，并在卡片上添加您自己的参考；
*   只有当你对谈论这个话题或者应用你所学的东西感到舒服的时候，才进入 Done
*   按照上面的建议，创建你自己的“完成”定义。

## [](#how-to-use-the%C2%A0board)如何使用棋盘

[![Trello Board](../Images/a4afe926e664f54f4e772839bf9e4f24.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--FY9LztYi--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/nf7xaw4nlen5u1was0l4.png) 
棋盘上有 5 列，前 3 列代表每一关。要获取学习指南，您必须遵循以下步骤:

*   创建一个 Trello 帐户；
*   访问[https://trello . com/b/hLGyiEEE/swift-developer-roadmap-template](https://trello.com/b/hLGyiEEE/swift-developer-roadmap-template)；
*   单击显示菜单；
*   点击更多；
*   点击复制板；
*   为您的学习指南定义一个标题；
*   点击创建，你就完成了。

如果你错过了什么，用复制到你的特雷罗板，你可以添加尽可能多的主题。但是由你来决定新项目的优先级。
