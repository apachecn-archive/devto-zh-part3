# 行为变化的 4 大法则

这本书围绕着你应该遵循的习惯养成过程的四个阶段。作者称之为行为变化的四大法则:

1.  让它显而易见
2.  让它有吸引力
3.  让它变得简单
4.  让它令人满意

# 倒置

说到好习惯，你可能会对你的生活感到满意。也许，你只是有一些你想改掉的坏习惯，比如抽烟，或者拼命查看社交媒体，把自己和别人比较。四大定律的反转拯救了我们。

1.  让它隐形
2.  让它没有吸引力
3.  使它变得困难
4.  使它不令人满意。

我真的很喜欢利用人性的想法，而不是专注于投入大量的精力成为一个新的下一级超人，摆脱我们的基因，让它为我们服务。人性中有很多东西会阻止我们改掉坏习惯。我们的大脑想要即时的满足，不接受延迟的回报。这本书列举了许多将这些障碍转化为我们优势的好方法。

# 身份

根据这本书，第一步是想出我们想成为什么样的人的想法。当我们意识到这个人的身份时，就更容易收集支持我们成为那个特定的人的习惯。否则，我们可能只是在闲逛。把它们写在纸上或其他地方。更容易让你觉得自己被委以重任。

# 实现意图

根据几项研究，如果我们将习惯与特定的时间和地点联系起来，我们更有可能坚持下去。这就是所谓的实现意图。当然，成功地坚持一个习惯还有很多其他的因素，它可能并不适合你。我从经验中知道，如果不明确，就很难按计划进行。使用公式并写下来。

公式如下:

```
I will [BEHAVIOR] at [TIME] in [LOCATION] 
```

# 习惯堆叠

另一个养成新习惯的好方法是把它和已经存在的习惯联系起来。如果实现意图不是你的事情，这可能对你有用。当然，你也可以将两者结合使用。既然我们已经有了一个习惯清单，我们可以很容易地找到一个新习惯的好地方。显然，最好将新习惯与好习惯联系起来，因为你可能想改掉坏习惯。只是时间问题。

公式:

```
After [CURRENT HABIT], I will [NEW HABIT]
```

# 环境

我们生活或工作的地方经常会妨碍我们养成习惯。这与地点无关，而是我们如何组织周围的事物。让你的房间成为你的朋友而不是敌人。如果你想每天都读书，那就每天早上在枕头上放一本书。如果你想弹吉他，把它放在你客厅的中间，等等。

我想我对这部分很满意，但总有改进的空间。例如，我努力提高我的睡眠质量，我的卧室里没有电视，但我睡觉前会用手机看书。我使用暮光之城应用程序过滤蓝光，但不仅仅是这样。我倾向于在章节之间查看社交媒体。我有一个 Kindle，但我的床边没有台灯，所以我想是时候买一个了。:)
