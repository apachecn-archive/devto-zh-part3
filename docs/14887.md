CS-Cart 简介

它是一个典型的服务器端软件程序，需要你开发一个完美的 web 服务器环境，包括 MySQL 数据库、PHP 解释器和 HTTP 服务器。遵循下面提到的四个步骤:

步骤 1:在本地计算机上运行软件

第一步，我们建议您在本地系统上安装 CS-Cart。您将安全地访问项目，试验设计和设置，然后将您的完美成果投入生产。最初，您必须在计算机上安装虚拟服务器。您需要下载免费的 web 服务器包来获得 web 服务器环境。CS-Cart 开发人员建议您使用 XAMPP。

它是一个通用平台，可以与 Linux、Windows 或 OS x 操作系统一起工作。根据需要你也可以用 MAMP (OS X)，LAMP (Linux)，Denver (Windows)来代替 XAMPP。如果你对潜在用户和高级用户进行复杂的设置没有任何问题，那么你也可以使用虚拟服务，比如 IIS 和 Apache。每当您完成虚拟服务器安装后，就需要下载最新版本的 CS-Cart 软件。将 web 服务器上的所有文件提取到工作区。

在开始安装过程之前，您必须手动设置文件权限。CS-Cart 使用 MySQL(PHP 使用的最流行的数据库管理系统)在 web 服务器上保存在线商店的数据。CS-Cart 安装向导将自动开发一个 MySQL 数据库。您需要确认您的用户帐户、数据库名称和密码。

第二步:商店布置

CS-Cart 附带了一个易于使用的安装后设置向导，有助于轻松创建基本商店。您需要运行设置向导来设置您的商店。默认情况下，商店的所有必需设置都会完成。仔细定制您的商店:记住它创建产品页面和类别:设置送货方式。设置支付网关。安装现成的功能，如畅销书、评论和评论以及搜索引擎优化。定制外观和感觉。

步骤 3:选择主机服务器和域名

当你的商店准备好了，那么现在你需要选择一个虚拟主机提供商。所选的 web 服务器必须支持 PHP 版本 5.3 到 5.4，以及 MySQL 版本 4.1 或更高版本。托管提供商必须允许您访问 cURL 支持；已安装的 GD 库；一个. htaccess 文件；Apache mod _ headers 支持；Phar 扩展；file_uploads(应允许以下指令:RewriteRule、RewriteBase、RewriteCond、AddHandler、RewriteEngine、allow、Options、Order、DirectoryIndex 和 Deny)。

步骤 4:迁移到 web 服务器

您需要备份商店的数据库，使用流程(管理面板–>管理–>数据库–>备份)并将所有文件打包到一个 zip 文件中。您必须清除缓存文件夹。利用托管文件管理器，您可以轻松提取 zip 文件并设置适当的文件权限。
