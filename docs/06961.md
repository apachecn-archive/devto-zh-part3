# 在互联网上做一个女人...“乐趣”:一项科学研究

> 原文:[https://dev . to/emmabostian/be-a-woman-on-the-internet-is-fun-a-scientific-study-a1d](https://dev.to/emmabostian/being-a-woman-on-the-internet-is-fun-a-scientific-study-a1d)

*注意:我试图让这篇文章尽可能的容易阅读，但是因为我想屏蔽名字&推特账号，我不得不使用一些图片。我很抱歉，并将更新 alt 标签以显示推文的内容。*

我在互联网上有过不少怪异、令人毛骨悚然、彻头彻尾的白痴互动。大多数时候我试着忽略它们，但我认为我会以良好的精神状态分享我在万维网上经历的“乐趣”。(*是的，这是讽刺。*)

我并不是说这种类型的互动只发生在女性身上，但我确实相信，如果一个男人发布了同样的推文，他们就不会收到同样数量的愚蠢信息。

无论我在网上发布什么，似乎人们都会立刻蜂拥而至，在我的话语中寻找错误，并立即就此批评我。

下面就来深入探讨一下我的超级官方(lol)关于“在网络上做女人”的科学案例研究。我会用科学的方法。

*另一个注意:我将在整篇文章中使用男人这个词，因为我在互联网上的所有时间里，只有**一个**女人发了一个愚蠢的回复。这是以偏概全吗？是啊，也许吧。但在我看来是个异类。我绝不是说只有男人才有这种行为。我只是说这种事经常发生...从...男人。*

如果我得到男人的评论说我在做归纳总结，那将是多么讽刺啊，因为他们显然不会读到这一段，讽刺会渗透到电脑屏幕上。再说一遍，整篇文章只是一个笑话。好的，谢谢，就这样。

# [](#step-1-make-an-observation)第一步:进行观察

互联网上的女性通常会收到更多愚蠢的评论来回应她们的内容。

不管我发布什么，我总是收到过多的评论，要么试图证明我是错的，坚持说我不可能发布，因为我是一个女人，不懂技术，要么斥责我说我太保守。

# [](#step-2-ask-a-question)第二步:提问

网上的男人总是在博客或推文中挑毛病，以便插入他们的个人观点吗？

# [](#step-3-form-a-hypothesis)第三步:形成假设

网上的男人，不管发布了什么内容，都会寻找他们能戳到的极小的漏洞，直到它们变成能反驳女人观点的大漏洞。

# [](#conclusion)结论

我发这个博客是为了开玩笑，因为在这种情况下除了开怀大笑还能做什么呢？心烦意乱是没有意义的；不值得我浪费时间。

但可悲的现实是，许多女性的情况比我糟糕得多，这种情况必须停止。如果你看到这种事情发生，站出来反对它。你可以尊重地这样做，但我们必须停止鼓励这种不断需要证明女性是错的。

我不是说人们不能有不同的观点，但是故意试图证明一个女人的话是胡说八道是不酷的，需要停止。
