# 意大利鼓舞人心的五十年

> 原文:[https://dev.to/rhymes/italys-inspiring-fifty-1im4](https://dev.to/rhymes/italys-inspiring-fifty-1im4)

激励 50 岁是一个项目，每年提名 50 名女性在技术领域，以提高知名度和庆祝鲜为人知的榜样。这主要是欧洲的首创(始于荷兰)，少数国家有自己的国家版本。意大利就是其中之一。

每个人都可以提名其他人，评审团会选出最后的 50 人。

今年的榜单突出了研究人员、高中 STEM 教师、软件工程师、大学教授、经理、创始人、首席技术官、技术主管、IT 主管等。

获奖者示例(链接指向他们的 Twitter 或在线状态):

*   Viviana Acquaviva :天体物理学家和数据科学家。纽约城市大学教授、研究员和机器学习专家

*   Valeria Cagnina : 17 岁的机器人专家、企业家和教师。她 11 岁时在米兰的 CoderDojo 学习编程。她在 TedX 演讲，在意大利参议院演讲，她在麻省理工学院参加了一个夏令营，她现在在全国各地的学校和公司教授机器人研讨会，对孩子和成人都一样

*   [Lorella Carimali](http://lorellacarimali.it/) :她对向学生教授 STEM 非常有热情，她被评为 2017 年全国十大最佳教师之一。她还写了《生活的平方根》这本书

*   西蒙内塔·迪·皮波:她是联合国外层空间事务主任。她一生都在太空机构工作。她也是一名天体物理学家，有一颗奇怪的小行星以她的:D 命名

*   [Daria Loi](https://www.linkedin.com/in/darialoi) :英特尔首席工程师，UX 研究员，人工智能专家

*   玛丽罗萨莉亚·轩辕洛尹:牛津大学的哲学家、伦理学家和研究员。她的专业领域是技术和人工智能中的伦理

*   Silvia Wang:Rocket Internet 的前经理，她在 2015 年创立了自己的公司， [ProntoPro](https://www.prontopro.it/) ，这是一个帮助用户寻找专业人士的服务市场。该公司现在有一百万用户和 100 名员工(平均年龄为 29 岁)

这么多惊人的例子:-)

你可以在这里找到 50 位女性的完整名单:[https://italy.inspiringfifty.org/italy-2018](https://italy.inspiringfifty.org/italy-2018)

我能对这份名单做出的唯一评论是，如果我没弄错的话，除了 Silvia Wang(她出生在意大利，父母是中国移民)之外，其他人都是白人女性。在某种程度上，意大利是一个少数民族不那么容易被看到的国家，除非我们谈论的是“坏移民”的回归，不幸的是，这似乎现在在整个欧洲甚至更远的地方都很流行。我可以写一本关于意大利隐形少数民族的书。我不想贬低任何人的价值(如果我觉得不好，我不会公布这个名单)，我只是想指出这一点。

无论如何，希望你会受到 STEM 中令人惊叹的女性的启发:)
