# 我 2019 年的目标

> 原文：<https://dev.to/airercode500/my-goals-for-2019-425c>

我在 SAP Concur 开始工作后不久，我和我的经理坐下来开了一个季度目标设定会议。我突出了我想成长的地方，给我的技能打分，我们提出了一个反映这些的目标清单。

几个月后，当我回顾这些目标并设定新目标的时候，我震惊地意识到我之前设定的目标已经变成了第二天性。

这就是我在这篇文章中应用的哲学。

# 1。与更多人见面

我有一群核心朋友，我每周或每月都要见他们几次，但我真的很想扩大范围，在西雅图地区结识新朋友。我经常去我公寓附近的狗公园，交了几个朋友；了解去那里的人(更重要的是他们的狗)。

去年，我开始参加聚会，我想走出我的舒适区多走几步，实际上尝试与那里的人交谈，并更好地了解他们。

理想情况下，能找到我可以偶尔一起喝咖啡或吃晚餐的人会很酷。另一个子目标是在该地区尝试新的咖啡馆和餐馆，这将是一个很好的方法。

# 2。成为导师(续。)

在整个大学期间，我尽我所能在这个项目中帮助指导年轻的学生。我坚信，通过教导和指导他人，我会不遗余力地学习新的东西或找到更好的教学方法。

我想在办公室内外继续这样做。我已经推荐了几个在聚会上与我接触的人，我想与他们保持联系，看看是否有什么我可以帮助他们推进职业发展的方法。

# 3。在会议上发言

我已经为西雅图地区 reacts meetup 做了几次演讲，但是我想更进一步，这次真的做一次会议演讲。在大学里，我做过很多小型的演讲，这是我非常喜欢做的事情。

我强迫自己学习新事物并深入学习它们的最好方法是报名参加一个关于它们的讲座。我想很快我会试着做一个“如何演讲”的演讲，但是大部分时间我想努力发展我更技术性的演讲。

# 4。更好的音乐发现(更多音乐会)

我喜欢音乐，但我不经常去寻找我可能喜欢的新东西。我倾向于一年建立 2-3 次播放列表，然后一直听下去，直到筋疲力尽，再做一个播放列表。

我在考虑尝试更多地使用潘多拉，让我的电台对不太受欢迎的艺术家更友好，这样我就可以更好地混合歌曲和情绪。我还想在西雅图地区找一些更小的乐队，这样我就能更经常地看到他们了(稍微便宜一点)。

* * *

有几件事情没有列在清单上(吃得更好，锻炼身体),但总体来说，它们与我想要的生活方向更一致，但这四个领域是我今年将投入最多精力的地方。

我刚刚想到的其他几个例子是:

*   为自己做更多的饭
*   探索新的远足和小径
*   继续尝试非正式约会

如果你对这些产生了共鸣，请随意联系，我很想听听你是如何开始新的一年的。
