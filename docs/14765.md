# 是 20-20-20👁️还是番茄🍅不为你工作？试试这个。

> 原文:[https://dev.to/3sanket3/is-20-20-20 或番茄不为你工作-尝试-4icg](https://dev.to/3sanket3/is-20-20-20--or-pomodoro--not-working-for-you-try-this-4icg)

不知道为什么，但这两种技术对我都不起作用，我需要一些能迫使我休息一下的东西。

所以我发明了😉下面的技巧对我很有效。这听起来可能很滑稽，但是很有效。

1.  喝 2-3 杯水🥛🥛🥛(或者尽可能多)
2.  开始编码💻
3.  你需要在 20-25 分钟内小便，去解决它🚽
4.  重复...

好处:

*   更多的水更健康
*   你需要休息一下
*   你需要离开你的座位，步行
*   一旦这样做了，你会感到放松，你的心会很平静。这将有助于你从一个全新的角度看待你的工作。
