圣诞快乐，所有可爱的人们。

5 天前，我和我的队友坐在一家咖啡馆里，我们讨论 Dev.to 有多酷，当时我正在写一篇关于 menubar 应用程序的文章。然后我突然问他，“如果有一个 Dev.to 的 menubar 应用会怎么样？”

他说“伙计！那就超级牛逼了！”

所以我问了其他几个开发者朋友，他们都同意这是个好主意。这就是为什么我决定尽我所能在 5 天内发布这个应用程序。作为给这个社区美丽人们的圣诞礼物。🤗

所以我现在和 Dev10 `v1.0.0`在一起。让我快速向你们介绍一下主要特性。

## [](#stack)堆栈

对于这个项目，我给自己的挑战是只使用普通的 Javascript。这并不是说我讨厌框架什么的，而是这个应用的范围还没有大到需要一个框架。除此之外，我还用过:

*   [车把](http://handlebarsjs.com)
*   [电子](http://electronjs.org)

## 安装

它是由电子构成的，所以启动过程非常简单

1.  `npm install`
2.  `npm install electron -g`
3.  `npm start`

就是这样，你会看到它在你的菜单栏中运行。
