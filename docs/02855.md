# 我最喜欢的 5 个前端备忘单和书签

> 原文:[https://dev . to/Vladimir _ dev/my-5-favorite-front-end-cheat-sheets-and-bookmarks-4nj4](https://dev.to/vladimir_dev/my-5-favorite-front-end-cheat-sheets-and-bookmarks-4nj4)

首先，让我先说一句，我认为不时地回归基础从来都不是一个坏主意。

我坚信，不管你有多有经验或多先进，你都应该偶尔花些时间重温一下你所做的事情的基本原则。不要把它想成是退一步，更多的想成是加固地基，让建在上面的房子更坚固。每次这样做，你都会发现新的见解。⠀

因此，即使这些帖子主要是针对初学者的(因为我也认为自己是初学者)，也许一些更高级的前端人员也会发现其中一些有用。

那我们开始吧。以下是我最喜欢的小抄和书签，我在工作的时候总是把它们放在手边 :

# [](#1-css-grid-cheatsheet)1。CSS 网格备忘单

*由马尔文公司*
[grid.malven.co](http://grid.malven.co/)

[![CSS Grid cheat-sheet](../Images/8f398d943f90b2645d1e8a8aa9478f2f.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--h-IG0Ts2--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/93z017jqxjicgd4gettb.png)

有许多关于网格的文章和参考资料，但是这一篇绝对是我目前最喜欢的。这是一个很棒的可视化备忘单，所有的设置都显示在一个页面上，这样你只需看一眼就可以访问(并理解)它们，这就是备忘单应该有的样子！

# [](#2-css-flex-cheatsheet)2。CSS Flex 备忘单

*由马尔文公司*
[flexbox.malven.co](http://flexbox.malven.co/)

[![CSS Flex cheat-sheet](../Images/3a7e29dab19f8785df206fea7780730f.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--CnwifSDD--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/gdwj085o01ldln9ygx7m.png)

与上一个相同，但带有 Flex。是的，我太喜欢这些了，所以在我的清单上给了它们两个位置！:)

# [](#3-easing-functions)3。缓解功能

*由安德烈[西特尼](https://dev.to/iskin)和伊万*索洛夫
T5】easings.net

[![Easing functions](../Images/3344fdba3913e728f0efd42fdaafd7e6.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--KUVN_Kur--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/m1vflqsg9ibxa8vfrxtg.png)

线性过渡太逊了，阿米利特？这一个有相当多的缓和函数的例子，连同尺寸、位置、透明度和渐变的预览。当然，包括贝塞尔曲线。

# [](#4-css-clippath-maker)4。CSS 剪辑路径生成器

*由班尼特·菲利*
[bennettfeely.com/clippy](https://bennettfeely.com/clippy/)

[![CSS clip-path maker](../Images/a76f4431c53ecf2f31cd5ecb7a23cda1.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--_LMBFXSo--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/w4bimhw3djkt5shhohg2.png)

这个可以让你只需点击几下就可以创建和预览剪辑路径！太简单直观了。

# [](#5-csstrickscom)5。CSS-TRICKS.com

*由一大群酷人组成的*
[css-tricks.com](https://css-tricks.com/)

[![CSS-TRICKS.com](../Images/62b7d7bbc6c6449cefad8665ba139cde.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--gNTUVOa8--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/4v19aa7018izfn2uauo9.png)
