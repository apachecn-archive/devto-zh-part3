# 崇高的文本 Vs 代码

> 原文:[https://dev.to/programmerrush/sublime-text-vs-vs-code-36jo](https://dev.to/programmerrush/sublime-text-vs-vs-code-36jo)

我们所有开发人员都熟悉许多编辑器或 IDE。但是在不同的项目或时间线中，我们需要在许多编辑器中迁移，最常见的免费编辑器是 Sublime Text & VS Code。

有时，我们在为即将到来的项目决定一个编辑器时感到困惑。以下是需要注意的要点。

**崇高文字:**

1.  每个人都喜欢多行选择编辑
2.  插件支持
3.  定义自己的语法
4.  可由许多插件扩展

**VS 代码:**

1.  大多数用户依赖集成终端
2.  几乎每种语言的插件
3.  监视代码和错误
4.  深受 typescript 或 Angular 开发人员的喜爱
