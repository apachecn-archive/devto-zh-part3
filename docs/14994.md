# 使用 stackoverflow，并且用好它

> 原文:[https://dev . to/mooreds/use-stack overflow-and-use-it-well-1fj 7](https://dev.to/mooreds/use-stackoverflow-and-use-it-well-1fj7)

亲爱的新开发人员:

Stackoverflow (SO)非常适合三种不同的开发人员(随着时间的推移，有些人可能是这三种人):

*   那些通常通过谷歌寻找答案的人(搜索者)
*   那些希望展示知识的人，通常通过回答问题(回答者)，以及
*   有具体问题要问的人(提问者)

每个开发者都可以是搜索者(你需要的只是谷歌)。但是你可以寻找不好或者好的答案。通过阅读所有的答案，理解搜索结果中出现的问题，并投票支持问题和答案来做好这件事。特别是投票是一种为整个开发者生态系统增加价值的低成本方式(因为你为其他搜索提供了价值)。我知道接受第一个答案并继续你正在努力完成的任务会更容易，但是试着多花一点时间去帮助别人。

但你应该努力成为一个回答者或提问者。看看你是否能回答问题或添加评论来澄清它们或指向其他答案。如果你有一个在 SO 上没有答案的问题，花时间建立一个可靠的问题。如果你最终找到了问题的答案，自己回答它来帮助其他人。

诚然，SO 不是最受欢迎的社区，但它是最优秀的问答网站，因此值得存在。别忘了，你提供的哪怕是最微小的帮助也可能会影响多年。我只有大约 3k 的声誉，但已经接近 95 万人。除了帮助别人，这也是一个展示能力的好方法，而且不会过多地脱离你的正常工作流程。
