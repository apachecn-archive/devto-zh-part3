我找不到一个笔记应用程序来勾选我感兴趣的所有选项:笔记是用 GitHub 风格的 Markdown 编写和呈现的，没有所见即所得，没有专有格式，我可以在所有笔记中运行搜索和替换，笔记支持附件，应用程序不臃肿，应用程序有一个漂亮的界面，标签可以无限嵌套，可以导入 Evernote 笔记(因为这是我以前使用的)。

所以我自己造了一个。
