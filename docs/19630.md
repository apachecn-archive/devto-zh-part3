# 我学到了开发人员应该做的 3 件无关的事情

> 原文:[https://dev . to/flipped coding/3-independent-things-I-learned-developers-should-do-3482](https://dev.to/flippedcoding/3-unrelated-things-i-learned-developers-should-do-3482)

生活中有比写代码或成为技术奇才更多的东西。开发人员应该知道如何做更多的事情，否则他们不会长久地做好开发人员。你可以努力拥有伟大的事业，同时仍然是一个完整的人。

这里有三件完全不相关的事情，每个开发人员都应该能够做到:

## [](#make-new-hobbies)培养新的爱好

你必须在生活中有一些其他的兴趣。编码很酷，但它不是你应该全天候做的事情。那是不健康的。尽可能多地学习 web 开发很容易，偶尔抵制这种冲动也是可以的。

记住你的其他爱好，比如削木头或者弹四弦琴。你不是一个写代码的机器人，所以享受你大脑的其他部分吧。如果有一家你想尝试的新餐馆，去看看吧。离开编辑一会儿，放松一下。

## [](#exercise)运动

没有必要像运动员一样锻炼，但是你需要站起来运动。当你整天写代码时，很容易坐在屏幕前几个小时不动。这对你的健康和编码能力来说绝对是最糟糕的事情之一。

即使你只是在街区周围散散步或者走几分钟，站起来动起来。如果你已经坐了一个多小时，现在就做。你的健康是你所拥有的最有价值的东西，所以不要忘记它。

## [](#read-people)阅人

人们阅读是一门艺术。你习惯于关注代码中最小的细节，那么为什么不去观察与你一起工作的人的细节呢？了解是什么让你周围的人兴奋，这将有助于提高你在工作和家庭中完成任务的能力。

这是一个比你能学到的任何技术技能都更能帮助你的技能。如果你决定做自由职业者或者你需要找一份新工作，知道如何读懂别人将会给你巨大的帮助。学会读懂别人需要一些时间，但是一旦你知道如何去做，你将会是这里最强大的开发者。

你可以做很多其他的事情，这些都没有列在这个清单上。对你来说，最重要的是做其他的事情。提高你的编程技能很好，但提高你的踢踏舞技能也很好。在编辑之外生活。求你了。
