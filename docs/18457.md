# 开始学习 HTML 的三个好地方

> 原文:[https://dev . to/adamlombard/three-great-places-to-start-learning html-2ae 0](https://dev.to/adamlombard/three-great-places-to-start-learninghtml-2ae0)

我总是在寻找很好的学习资源——为我自己，也为任何问我如何开始开发的人。

网上有很多学习 HTML 的地方，但是如果你没有任何经验，很难找到一个有用的地方。很容易认为你是从第一个地方开始，但却发现它实际上不是第一个地方，现在你完全迷失在森林中，黑暗而可怕，树上有发光的眼睛，JS 框架弹出来说“Boo！”每隔 30 秒，前面的小道上就有人在尖叫 web assembly，你可以听到远处气泡分拣和白板的嚎叫，你不知道这意味着什么，你只想回家！

我不能说我责怪你。

为了避免所有这些困惑，我向那些希望在无限的 web 开发海洋中摸爬滚打而毫发无损的人强烈推荐以下三个绝对初学者的 HTML 资源。这些都非常适合初学者，而且完全免费(就像，你甚至不需要花一个电子邮件地址)。

你的不可怕的 HTML 之旅从这里开始:

*   杰克·赖特在 12 分钟内学会 HTML(视频)
*   [在](https://sabe.io/classes/html/) [Sabe.io](https://sabe.io/) 开始使用 HTML
*   [Marksheet.io](http://marksheet.io/) 的[【Web】](http://marksheet.io/introduction.html)和[【HTML 5】](http://marksheet.io/html-basics.html)部分

我建议按照这个顺序看完这三个。他们将涵盖相似的内容，但从稍微不同的角度来看。你大概可以在一个下午完成所有的事情。在那之后，你将有一个很好的基础来学习更长、更深入的 HTML 或 CSS 课程。

祝你好运！让我知道你最喜欢的，初学者友好的，免费的资源。
