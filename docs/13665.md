# 自由职业者

> 原文:[https://dev.to/damcosset/freelancers-on-devto-5hnl](https://dev.to/damcosset/freelancers-on-devto-5hnl)

我现在正在推特上做一个小调查，问自由职业者一个简单的问题。你估计你有足够的工作，与你理想中想要的相比是太少还是太多？

我认为询问自由职业者(或前自由职业者)关于自由职业的经历会更有趣。

我特别想知道:

*   你是/曾经是什么类型的自由职业者？兼职？全职？只是为了晚上的非营利活动？...
*   作为一名自由职业者，你对自己的收入满意吗？
*   是什么让你改变了职业现状？从传统工作到自由职业者，反之亦然。

我真的很好奇来自世界各地的人们分享他们关于自由职业的经历。
