# 从 Laravel 转移到另一个框架(django 或 express)？

> 原文：<https://dev.to/alejandro/moving-from-laravel-to-another-framework-django-or-express-20e0>

*TL；博士:如果你在拉勒维尔很好，你会在 Django 和 Express 之间选择什么？*

嗨，我想从 Laravel 转到另一种技术，可能是 Django 或 Express，但我不确定该怎么做。

我主要开发 web 应用程序，所以我在 Laravel 方面做得很好，但是由于我已经使用它很长时间了，我想学习一些新的东西，然后将它用作主要武器。

在我看来，最直接的方法是做 Django，但我想知道为什么人们更喜欢 Express 而不是 Django，也许我没有看到什么。

我对 POO 语言感觉很舒服，我认为我有一个高级的 javascript，所以做 Django 会有点困难，我想知道这是否值得，或者也许只是坚持用 Laravel 作为主要武器

如果你认为有另一个值得一提的框架，就当我在努力寻找成熟的、可以在生产环境中使用的框架，我很少为每月超过 1 万的用户做 web 应用。
