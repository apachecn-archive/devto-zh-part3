# Android Q 最激动人心的功能

> 原文:[https://dev . to/trevornemanic/the-most-excited-features-coming-to-Android-q-3 kmc](https://dev.to/trevornemanic/the-most-exciting-features-coming-to-android-q-3kmc)

谷歌已经公开发布了 Android 的最新版本 Android Q Beta。随之而来的是许多对用户有益的改进。谷歌主要关注隐私、安全和性能，同时为可折叠等最新趋势做准备。事不宜迟，让我们深入了解 Android Q 中的最佳功能。

## [](#foldables)可折叠

可折叠手机在 2019 年首次亮相。无论你喜欢还是讨厌它们，都可以看到很多关于公司、开发者和用户尝试这种新外形的新闻。开发者还不能在现有的 Android 模拟器上测试可折叠文件夹(谷歌正在努力)。

## [](#granular-control-for-location-permissions)对位置权限进行粒度控制

谷歌回应了用户保护隐私的强烈要求。永远不要害怕暴露你的位置。现在，位置权限选项将有一个额外的选项，只在应用程序使用时给出位置。
[![Location permission option](../Images/4ac795a64d96acfb0b88e5801b8d6cea.png)T3】](https://res.cloudinary.com/practicaldev/image/fetch/s--t2hmkBF5--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/la1grucazdkpad5rk2ym.png)

## [](#wifi-performance-modes)WiFI 性能模式

移动游戏已经成为应用程序领域的一股强大力量。然而，由于许多游戏玩家依赖 WiFi，延迟仍然是一个问题。Android Q 将提供应用程序告诉设备使用低延迟或高性能模式的能力。这将大大改善各地手机游戏玩家的体验。

## [](#dynamic-depth-format)动态深度格式

就相机质量而言，Android 手机已经取得了非常非常大的进步。Android Q 将迎来另一个重大进步:动态深度格式。这种新格式将允许应用程序提供对模糊和类似功能的改进。动态深度格式将为应用程序开发人员提供实际查看深度的工具，他们已经模拟了不同程度的成功。期待看到更好的增强现实和 3D 图像。
