# [](#what-is-open-banking)什么是开放式银行？

要理解什么是开放银行业，我们需要理解它为什么会出现。

在[开放银行业](https://www.openbanking.org.uk/)之前，人们发现银行业被老牌银行所主导，而你作为银行业的新玩家，想要在这个行业中取得突破特别困难，更不用说在这个行业中茁壮成长了。这意味着客户对他们的资金或金融服务提供商没有太多的选择或控制权。

由此，开放银行业诞生了。
2015 年，欧洲议会发布了修订后的支付服务指令(简称“PSD2”)，旨在促进在线和移动支付创新技术的开发和使用。此外，2016 年，英国竞争和市场管理局强制要求 9 家最大的英国银行允许获得许可的初创公司访问其数据，直到跨国级别。2018 年才生效的裁决。

## [](#so-what-does-this-all-mean)那么，这一切意味着什么呢？

这意味着，通过客户的同意和适当的安全渠道，您可以获取客户的财务信息并提供您的金融服务。这对创业公司来说是一个巨大的优势，因为他们不必与大银行争夺客户，而是共享客户基础。这也使银行能够成为生态系统的驱动者，成为其他业务得以蓬勃发展的平台。

# [](#what-does-this-mean-for-the-african-startups)这对非洲的创业公司意味着什么？

1.  更好的可扩展性和客户体验

    从我自己(在莫桑比克)的经历来看，初创公司最大的痛点之一是支付整合。直到最近，我们才有了本地支付网关 [(M-Pesa API)](https://developer.mpesa.vm.co.mz/) 。在此之前，大多数初创公司不得不接受应用程序之外的支付(现金、转账等)，对现金流几乎没有控制，限制了你的可扩展性。
    当然，如果你在南非，有像 [Paygate](https://www.paygate.co.za/) 这样的服务可以帮助你，但如果你不在南非，那么你将面临更大的挑战。

2.  更便宜的商品和服务

    公开开放银行 API 也意味着创业公司提供的商品和服务会更便宜，因此更容易获得。通过与当地银行的整合，初创公司不再需要支付高额的国际交易费用(例如，如果整合到 paypal，就需要支付这笔费用)，因为交易现在是在当地进行的(在同一个市场内)。

# [](#but-what-are-the-challenges)但是，挑战是什么？

主要是:

1.  智能手机/技术渗透率低

    众所周知，非洲人口贫穷。因此，很少有人能使用智能手机，更不用说互联网了。根据互联网世界统计数据，非洲的互联网普及率约为 35%，远低于 54%的世界平均水平。因此，如果你的初创公司是智能手机或网络应用(让我们面对现实吧，大部分都是)，你的客户就会大大减少。
    但是一切都没有失去，因为 [USSD 的应用程序已经是革命性的](https://qz.com/africa/1296120/how-a-20-year-old-mobile-technology-protocol-is-revolutionizing-africa/)。

2.  大多数人没有银行账户。

    除了很少一部分人可以上网，更少的人有银行账户。根据麦肯锡的这项研究

    > “非洲的零售银行渗透率仅占 GDP 的 38%，是全球新兴市场平均水平的一半”

    这意味着即使你可以接受付款，但只有少数人真正有能力支付。
    随着银行业务的开放，金融普惠不再仅仅由银行驱动，越来越多的金融科技初创公司能够更积极地参与这一努力。

3.  支持立法机构不力

    非洲国家会采用 PSD2 或类似的方法吗？数据保护法呢？许多非洲国家没有管理互联网交易和信息的立法机构，但一小部分国家开始认识到互联网领域监管的重要性。对数据和电子处理交易的监管开始出现在议会中，并开始被认为是一个新的革命性行业发展的重要保障。
