> 原文:[https://dev . to/wuz/introducing-dank-neon-a-new-open-source-code-theme-2mh](https://dev.to/wuz/introducing-dank-neon-a-new-open-source-code-theme-2mh)

朋友们好！我有一个令人兴奋的新项目要和你分享！在过去的几个月里，[我和一个朋友](https://twitter.com/__jacobi)一直在调整和完善 [Dank Neon](https://dankneon.com) ，这是一个为霓虹灯爱好者设计的新编辑主题！

我们正在努力建设社区，欢迎您的任何意见和贡献！整个项目是开源的，你可以在 Github 上找到它。我将很快花一些时间来设置一些好的第一个问题，并制定一个后续步骤的列表！以下是 Dank Neon 的一些截图:
