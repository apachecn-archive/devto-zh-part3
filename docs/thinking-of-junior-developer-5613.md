# 初级开发人员的思考

> 原文：<https://dev.to/harken24/thinking-of-junior-developer-5613>

要开始这个故事，你也许应该先了解一下我。我是一名 20 岁的开发人员，没有大学学位，我刚刚度过了作为初级前端开发人员的第一周。

现在我坐在我的桌子旁，我不敢相信我是一名网络开发人员。事实上，我上了两年大学，运气不好，因为我不能强迫自己学习，我几乎连续两年每门课都不及格，我知道我的生活必须有所作为。因此，很自然地，我开始在互联网上寻找，并对编程产生了兴趣，因为我实际上涉猎了一些 QBasic 和 C 语言，但都是初级的，都被遗忘了。我决定尝试一下，从 web 开发开始，更准确地说是 HTML、CSS 和 JavaScript。

跳过 10 个月，我得到了第一份工作，是一名初级前端开发人员，在此之前我已经有了几个项目。那么，我是如何在学习编程的 10 个月内获得 3 个项目和一份工作的呢...和其他人一样容易做到。相信自己，我再怎么强调积极的心态对你思考和做事的方式有多大影响都不为过。如果你一直想着‘我做不到’，‘这太难了’，‘我永远也学不会’，那么你可能学不会，而且会很难，因为如果你不相信自己，你怎么能学会或取得成就呢？

基本上，我今天想分享的是**积极的心态**是你在某件事情上取得成功所需要的，没有人生来就知道所有的事情，每个人都在学习，所以你为什么要为自己不知道某件事情而感到羞愧呢？为知道自己并非无所不知而自豪，并让它推动你永远不要停止学习，永远不要停止为自己工作，因为在一天结束时，这才是最重要的。
