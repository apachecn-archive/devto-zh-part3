# OpenMoji

面向设计师、开发者和其他所有人的开源表情符号！OpenMoji 是由 Benedikt Gro、Daniel Utz、70 多名学生和外部贡献者开发的 HfG Schw bisch GMünd 开源项目。

这个 GitHub 存储库包含 OpenMoji 项目的所有源文件和导出的 png/svg 文件。
