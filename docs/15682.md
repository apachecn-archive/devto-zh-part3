# 为什么你应该学习打字稿

> 原文：<https://dev.to/progresstelerik/why-you-should-learn-typescript-45im>

这一次在吃饭睡觉代码播客中，我们了解了为什么开发人员学习 TypeScript 很重要。

在这一集里，David Pine(微软 MVP)分享了他使用 TypeScript 的经验。David 谈到了为什么 TypeScript 是一个有价值的工具，可以改善您对通常使用 JavaScript 编写的应用程序的体验。我们讨论如何编译 TypeScript，它的优点是什么，以及如何入门。

你可以在 SoundCloud 上听完整部《T0》和《T2》的前几集《T3》。
