# 深层黑暗网络

> 原文：<https://dev.to/flippedcoding/the-deep-dark-web-134n>

深度网络和黑暗网络并不是什么新东西。自从巨大的丝绸之路崩溃以来，黑暗网络得到了越来越多的关注。问题是，暗网只是深网的一小部分。除了毒品和其他见不得人的东西，深层网络还有更多的东西，但这些东西仍然存在。

## 它的用途

deep web 由互联网上的未索引网页组成。这些都是搜索引擎，比如谷歌，找不到的页面。deep web 中不需要包含非法内容的非索引页面。这些未编入索引的页面大多是你的个人社交媒体账户或网上银行账户。他们没有任何问题，我们只是不希望除了我们自己以外的任何人都可以访问这些页面。

所以当你登录到任何有你数据的网站时，你实际上每天都在使用深层网络。深层网络是存储您所有个人信息的地方。这就是为什么如果你用谷歌搜索你的名字，你的电子邮件账户不会显示出来。尽管深度网络因为组成黑暗网络的那一小部分而声名狼藉，但它并不坏。其实挺无聊的。

我们需要深度网络来保证我们在线数据的安全。我们所有的信息都在深层网络中，我们永远不想把它带到表层。这也是暗网存在的原因之一。人们不希望每个人都可以访问他们正在做的事情，或者知道他们正在做什么或者他们在哪里。

在像中国这样的限制性国家，一些人使用暗网作为分享信息的一种方式。这也是一个举报人可以举报他们发现的事情而不用担心被发现的地方。当然，还有卖非法东西的人。

## 如何使用

如果你想知道如何在 deep web 上发布东西，这一点也不复杂。记住，deep web 是由未索引的网页组成的。因此，如果你决定取消你在 YouTube 上的一个视频，或者你选择保持你的 Tumblr 博客不被索引，你只是向深度网络添加了更多的内容。深度网络真的可以如此单纯。

进入深层网络的网站并不困难。如果您有通过搜索无法找到的页面的 URL，则您有一个连接到 deep web 的 URL。任何没有被编入索引的网站都是深层网络的一部分，并不难找到。

另一方面，黑暗网络需要更多的工作来访问。首先，你可能应该使用一个 [VPN](https://flippedcoding.com/2019/03/14/using-a-vpn/) 。然后你需要下载 [Tor 浏览器](https://www.torproject.org/)。之后，你需要找到一些以. onion 结尾的 URL。你可能在 Reddit 上有些运气，但你肯定需要一些运气。

为了最大程度的安全，在打开 Tor 浏览器之前，一定要启动 VPN。黑暗网络上的东西可能更难找到，因为它被故意隐藏起来了。的。洋葱网站大约一半是超级非法的东西，如职业杀手、枪支、毒品和其他东西，如难以置信的暴虐国家的人们试图获取和分享信息。

深网和暗网并不是完全一样的东西。暗网只是深网的一小部分，深网大部分是私人信息。它们都不像听起来那么神秘，因为归根结底，它们都还是网站。

深网和暗网的重要之处在于，你可以在被追踪的可能性最低的情况下分享信息。你会惊讶地发现有多少政府机密通过深度网络*和黑暗网络*被泄露。但大多数时候，研究人员只是希望能够访问 deep web 保存的未索引信息。
