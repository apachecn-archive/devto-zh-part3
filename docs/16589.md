# Changelog:折叠评论(通过“详细信息”HTML 标签)

> 原文：<https://dev.to/ben/changelog-collapse-comments-via-the-details-html-tag-4nim>

现在，您可以通过单击出现在注释旁边的▼符号来折叠注释。

它不是 100%推出的，因为我们会让一些缓存自然清空以显示新功能。我们可能还需要对点击区域半径等做一些小的调整。

这个特性是使用漂亮的`<details>`标签构建的，所以这个机制不是由 JavaScript 触发的。
