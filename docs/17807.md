# 网络最让你恼火的是什么？

> 原文：<https://dev.to/owenfar/what-annoys-you-the-most-about-the-web-lmk>

我很想听听你对网络平台上最让你烦恼的事情的看法。

就我个人而言，这是少数几件让我心烦的事情:

1) **框架**

事实上，大多数工作都需要经验和“强有力”的技能。让我恼火的是，公司似乎不理解一些开发人员只是简单地使用普通方法。

2) **质量**

一个网站的质量和另一个网站的质量之间可能会有很大的误解。让我恼火的是，没有简单的方法来了解我正在使用的网站的质量、安全性和用户隐私政策。对我来说，HTTPS 就是不行。

3) **标题**

在我看来，最近每个人都在成为首席执行官或创始人。没有办法区分人们和他们工作的合法性。

请与我们分享，你在从事 web 开发时面临的一些主要痛点是什么？
