# 回顾 2018 年

> 原文:[https://dev.to/funkysi1701/looking-back-at-2018-3j3n](https://dev.to/funkysi1701/looking-back-at-2018-3j3n)

随着 2018 年开始接近尾声，让我们来看看过去一年的一些亮点。

1.  新家——我从 2018 年开始，目标是买一套自己的房子，我管理着它。10 月底，我真的搬进去了，但这是几个月来看房子、与房地产经纪人交谈、存钱、打包和拆包的过程。2019 年，我想为我的家做更多的事情，所以这只是一个开始。
2.  新工作——另一个目标是让我的职业生涯更上一层楼，这发生在六月。作为一个大团队的一部分工作很棒，我学到了很多东西，还有更多我想学习和贡献的东西，所以 2019 年在这方面应该很棒。
3.  新车-这不是我实现的目标，因为不幸的是在十月份我遭遇了一场车祸，没有人严重受伤，但这是一次可怕的经历。我的车报废了，这给了我一个买新车的借口。
4.  更多地和家人在一起是我的一个目标，我不确定我是否真的做到了。然而，在我们搬家期间，家人一直在我们身边，我们住得更近了，所以希望我能在这方面做得更多。
5.  庆祝结婚 5 周年。对我和妻子来说，这一年并不轻松，因为搬家和所有的压力，加上两个孩子，她被诊断患有产后抑郁症，作为一个家庭，我们仍然在处理这个问题。所有这些都表明我们度过了一个美妙的周末，我们需要尽快再次做类似的事情。
6.  家庭。2018 年包括了我和两个男孩的许多美好时光。看着他们成长真是令人惊奇，九月我们为他们举办了感恩节服务，七月我们有了一年一度的假期。
7.  闪电对话。这是我提到的唯一一个没有发生的目标，它在我的 2019 年名单上，所以希望我能把自己踢到行动上，但随着其他一切的完成，我不得不错过一些东西。
