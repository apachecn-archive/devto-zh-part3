# 你在等待构建的时候会做什么？

> 原文:[https://dev . to/dowenb/what-do-you-do-while-you-waiting-for-builds-431 b](https://dev.to/dowenb/what-do-you-do-while-your-waiting-for-builds-431b)

构建和部署我目前正在进行的项目可能需要 10-15 分钟。这就是 Docker 和 GitLab 管道等所有很酷的现代事物。

在我等待的时候，感觉还是很漫长。

有时候我会做一些非常有用的事情，比如更新吉拉的机票，计划测试，听播客或者做一些学习。

我在寻找灵感，人们还做什么？
