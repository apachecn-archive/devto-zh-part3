# 带有 React Native Navigation+Redux+Eslint 的 TypeScript + React Native 初学者工具包

> 原文:[https://dev.to/amitm30/a-typescript-react-native-starter-kit-with-react-native-navigation-redux-eslint-29hp](https://dev.to/amitm30/a-typescript--react-native-starter-kit-with-react-native-navigation--redux--eslint-29hp)

在 we 上寻找项目以开始使用 RN + TypeScript，令人惊讶的是没有显示出多少结果。所以，想到写一个我自己的。

## [](#this-is-an-opinionated-react-native-starter-kit-with-react-native-navigation-redux-eslint-to-build-ios-android-apps-using-typescript)这是一款固执己见的 [React Native](https://facebook.github.io/react-native/docs/getting-started) 入门套件，带有[React Native Navigation](https://github.com/wix/react-native-navigation)+[Redux](https://github.com/reactjs/redux)+[Eslint](https://github.com/airbnb/javascript)使用 [TypeScript](https://github.com/Microsoft/TypeScript-React-Native-Starter) 构建 iOS / Android 应用

该项目已经根据 [RN 入门](https://facebook.github.io/react-native/docs/getting-started)和来自[微软的 Github TypeScript React Native Starter](https://github.com/Microsoft/TypeScript-React-Native-Starter)repo 的指令进行了设置。

这个 repo 支持 RNN 支持的 React Native 的最新版本(v0.57.8+)。

## [](#who-is-this-for)这是给谁的？

希望使用 RN 快速开发应用程序并热爱 TS 的人。基本设置已经处理好了，只需`npm install`并开始运行。

*免责声明*:这是一种**固执己见的**用 RN 构建应用的方法。项目结构的灵感来自贡献者构建的多个生产应用程序。

该项目使用并鼓励使用行业最佳实践/工具/库，如 RNN、redux、eslint、关注点分离和结构来构建可维护的应用程序。
