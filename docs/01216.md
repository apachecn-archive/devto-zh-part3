### [](#if-statement)If 语句

If 语句可能是编写糟糕代码的根本原因。在编写嵌套 if 条件之前，总是要三思是否有办法简化它。写得不好的`if condition`会降低程序的可读性。
这里先做如下假设。任何干净的代码都将具有以下属性。

*   易读的
*   简明的
*   可扩展——应该能够允许添加更多的条件或业务逻辑，而不影响以上两个因素。
