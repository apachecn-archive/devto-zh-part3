# 新的会议目标

> 原文：<https://dev.to/justzeros/a-new-meeting-goal-3im0>

过去，当我召集我的团队开会时，是因为我有事要告诉他们。

我关起门来构思的一个新项目、方向、技术或想法，我想宣布出来。

当我们分组讨论时，我的想法会改变。变得更好，或者可能完全不同。

不仅仅是通过交流想法，而是通过与我的团队对话的过程。

我记得我在想，“当我召集会议时，我想我知道我想要什么，为什么。但是，到最后，我想要的和为什么要改变了。我想知道这是怎么回事？”

上周，我在研究米哈伊尔·巴赫金(Mikhail Bakhtin)的关系领导力时，无意中发现了这句话，引起了我的共鸣。(强调是我加的。)

> _“这里的对话是**而不是**行动的门槛，**它是行动本身。** _
> 
> _ **它不是一种揭示的手段，**把一个人已经形成的性格展现出来；不，**在对话中，一个人不仅向外展示他自己，而且他第一次成为他自己**——而且，我们重复一遍，不仅是为了别人，也是为了他自己。_
> 
> “存在”意味着对话式的交流。”(巴赫金，1984)

令人兴奋的东西，但值得考虑。

这可能也解释了为什么我整天都在自言自语，有点像个疯子。这也解释了为什么当我写下我的想法时，我发现自己在一个不同的地方。

这些都是与自己对话的形式。

下次你和你的团队坐下来，考虑对话“不是揭示”你想法的手段，而是你第一次真正发现和了解你想法的方式。

记住这一点，与你的团队开会可以有一个全新的目的。
