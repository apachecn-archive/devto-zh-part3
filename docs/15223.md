## [](#1-application)1。应用📝

到处适用！尤其是如果你是一名初级开发人员，想要得到你的第一份工作。你不应该挑剔。适用于每个职位。即使是在你不感兴趣的公司。申请就好！为什么？因为这是很好的练习。就像其他事情一样，变得更好的最好方法是练习。就像如何成为一个更好的网球运动员，你练习。同样的，有了实践和经验，你也能在求职面试中做得更好。

更重要的是，申请你认为自己不适合的工作。为什么？因为你不应该制造自己的路障。这是阻止很多人不采取行动的常见事情。我们喜欢在脑子里编造各种借口来证明我们有资格。当你觉得自己不够好或者觉得自己经验不足的时候。停止✋你不知道，直到你尝试。不要制造自己的路障！不要代表他们说不，那是他们的事，不是你的事。相信自己，对自己的技能有信心，申请申请申请！

## [](#2-technical-interview)2。技术面试💻

通过技术面试是我最纠结的部分😣对我来说，技术面试很像你的高中数学考试。如果他们碰巧问你你学过的问题，那么你会答得很好。但就像生活一样，你永远不知道他们会问什么问题。没关系，通过练习(包括我)你可以变得更好！

以下是 3 类常见的技术问题:

a.白板
b .带回家挑战
c .模型

### [](#a-whiteboard)a .白板

白板是面试官问你一个问题，你在白板上写下你的答案。通常这只是一个算法挑战。你不能在这里使用谷歌。所以我是这样处理的，我把面试官当成谷歌。我会问他们问题。例如，如果你正在使用一种方法，但是你忘记了参数的具体顺序或者你忘记了方法的名称，就去问面试官。他们不是在测试你对语法的记忆程度，而是在测试你能否解决这个问题。

这里要做的另一件事是确保你大声说出你的过程。尽管他们很想看到你解决问题，但他们想看到你的思维过程，并检查你是否能清晰地表达你的解决方案。所以大声说出你脑中的想法👍

### [](#b-take-home-challenge)b .带回家挑战

这些是你可以在家工作的编码挑战。这些挑战总是有时间限制的，即使他们说慢慢来。速度就是游戏！这都是关于在短时间内完成某事。分秒必争。所以你需要超级高效地执行。

对于前端程序员，除非使用普通的 JavaScript，否则通常会使用某种框架。需要记住的是你的设置时间。我用 Vue，所以这对我来说轻而易举。这类似于 jQuery，我只需要删除一行 CDN 代码，然后我就完成了💥如果使用 React 或 Angular，还需要做更多的工作。所以确保你掌握了你的设置。记住，你花在准备上的时间越多，你真正应对挑战的时间就越少。

说到提高效率和速度，唯一能变得更好的方法就是重复和练习。这里有一些你应该能够当场执行的编码挑战:

一些示例编码挑战:

*   构建一个井字游戏
*   构建一个 Connect 4 游戏
*   [FreeCodeCamp 带回家的项目](https://learn.freecodecamp.org/coding-interview-prep/take-home-projects/)

### [](#c-mockups)c .模型

对于前端角色，你可能会得到一个模拟问题。所以他们会给你提供一个 PDF 或者一个网站或者一个网站的一个组件的一些图像版本。你被要求去建立它。它将主要涉及 HTML 和 CSS。根据组件的不同，可能会有一些 JavaScript。

*   构建带验证的注册表单
*   构建一个具有登录/注销功能的响应式导航栏。

## [](#3-cultural-interview)3。文化采访🤝

是什么让一个人适应不同的文化？让我分解一下，用通俗的语言解释一下。说明你不是混蛋！就是这样！一些公司会用一些花哨的术语来解释什么是文化契合，但最终他们只是想确保你是他们喜欢一起工作的人。还是那句话，站在他们的角度想想，你想和一个混蛋一起工作吗？斥责和贬低你的人。绝对不行！没错。所以在这里，你只需要说明你是一个很好的合作者。善良、积极、同情——求职和生活中的制胜法宝！

以下是一些你需要准备好的问题:

a.自我介绍 b .你的优点是什么
c .你的缺点是什么
d .你如何处理矛盾
e .你为什么想在这家公司工作

让我们一起走一遍，我会分享我的建议，告诉你应该如何解决这些问题。

### [](#a-introduce-yourself)a .自我介绍

> 我是一名前端开发人员，在 Angular 工作了 2 年。另外，我还用 Vue 做了一些兼职项目。我能给你看看我在做什么吗？

保持你的介绍简短。1 到 2 句话就足够了。你知道你在简历上写的简介，就用那个吧！你想强调你的技术经验。这里有一个窍门，如果你觉得自己缺乏经验(也许你只有老技术领域的专业经验，或者技术领域不适用于你申请的工作)，这是强调你的兼职项目的好地方。注意我以一个问题结束，如果你认为你正在做一些很酷的副业项目，这是控制谈话和提升你技能的好方法。

### [](#b-what-are-your-weaknesses)b .你的弱点是什么？

> 我的弱点之一是我花太长时间寻求帮助。所以现在我给自己一个时间限制，尝试自己解决。如果我不能在一定时间内解决，我会向团队寻求帮助。

这是一个棘手的问题。但是提示是要确保你有一个解决方案来减轻你的弱点。请务必使用合法的瑕疵。我太努力工作或者我太好了会在房间里得到一些眼球转动🙄。有弱点没关系。每个人都有。不需要为此感到羞耻。承认这一点，并证明你意识到了这一点，并试图解决它。这里的问题是，他们希望看到你努力去弥补自己的弱点，而不一定是真正的弱点。But‼️:你也不想谈论一个太离谱的弱点，以至于他们不想雇用你。例如，我的弱点是我喜欢在工作时睡觉，但我努力工作以获得 8 小时的睡眠，这样我就可以在工作时保持清醒。哈哈，我在写这个的时候都在笑🤣。很好，你有一个解决方案，但我认为这是一个可能太大的弱点😂。就像我说的，这是个棘手的问题。所以请你谨慎行事。

### [](#c-why-do-you-want-to-work-for-the-company)c .你为什么想为公司工作？

> Vue 是一个让我非常兴奋的技术堆栈，我知道这是贵公司当前的技术堆栈。还有，我听说这家公司有很棒的工作文化。这就是为什么我对有机会加入 ABC 公司感到如此兴奋。

在这里，他们只是想看到你已经做了调查，并对公司有所了解。我通常通过他们的网站和社交媒体账户来收集信息。这里有一个可以帮你加分的小技巧，如果公司或创始人获得了任何奖项，把它作为一个理由。公司通常以此为荣，所以这是赞美他们的好方法。

### [](#d-how-do-you-deal-with-conflicts)d .你如何处理矛盾

> 当有冲突时，我总是假定对方是无辜的，并试图与他们沟通。例如，有一次…所以我做的是…

他们在这里寻找的是你在冲突中的表现。解决这个问题的一个好方法是给出一个例子。谈谈你有矛盾的一次经历。然后解释你是怎么处理的。这是最重要的部分，你需要解释你是如何克服冲突的。招聘人员对冲突本身不太感兴趣，而是更关注解决方案👍

## [](#its-okay-to-walk-away-from-a-job)辞掉工作没事！

所以我们总是教人们如何得到这份工作，以及为了得到这份工作我们需要做些什么。我们的心理想法是取悦或迎合我们的采访者。如你所见，这创造了一个非常不均衡的动态。大多数面试官都很专业，不会利用这种情况。但如果他们这样做了，你绝对应该离开。

在加拿大，有些问题[你不能问](http://www.canadahrcentre.com/base/interview-questions/)。我明白了，每个国家都有自己的文化和做生意的规范。但我相信尊重是建立成功关系的核心。

如果他们在面试中不尊重你，发表粗鲁的评论，你认为一旦你得到这份工作会发生什么？如果他们以这种方式行事，他们在工作中肯定也会这样做。你真的想在一个被轻视和成为笑柄的环境中工作吗？他们是混蛋，滚出去！

### [](#how-to-handle-a-rude-interviewer)如何应对粗鲁的面试官

当面试官问或做一些不合适的事情时，知道如何反应或做什么是绝对困难的。所以让我给你一些指导。做好准备或者至少意识到如何应对，会让你在这种情况出现时更容易做出反应。

如果面试官问了一个不合适的问题或者让你不舒服的事情。问他们，这个问题和你申请的职位有什么关系。也许他们确实有一个有效的回应。记住，要不要回答由你自己决定。如果你认为不是。只要说，“我不认为这是一个与我申请的职位相关的问题，你还有其他问题吗？”你完全可以把它扔回去。当然，还是要尊重。仅仅因为他们是个讨厌鬼，并不意味着你需要堕落到他们的水平。如果事情变得很糟糕。走开！你可以这样说:

> “看起来这个职位不太适合我。谢谢你的时间。最适合这个角色的。祝您愉快”

然后走开。还是那句话，你还是可以尊重的。离开，然后在[玻璃门](https://www.glassdoor.com)上留下评论。如果他们是混蛋，那么其他人应该知道。你不必感到羞耻。分享你在[玻璃门](https://www.glassdoor.com)上的经历，让其他人也能知道。市场需要知道，混蛋在职场不受欢迎。我们应该互相尊重。因此，如果公司想要吸引人才和发展，他们需要创造一个健康的环境，淘汰那些混蛋💪

这对女性来说尤其重要。没有一份工作值得你自尊和尊严。你每天都会感觉很糟糕，你会不开心，你肯定会抑郁。还有其他工作充满了令人敬畏的好人。你不需要屈服，接受任何更少的东西。你绝对值得！！！！🙌

找工作从来都不好玩。这可能是一个乏味的过程。但是你必须坚持下去。不要害怕拒绝，拥抱它！随着你做更多的采访，你会变得更好。定义你的不是你的拒绝，而是你的毅力。祝你好运！你走这个！😊
