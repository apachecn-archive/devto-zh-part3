## 试验

使用[节点](https://nodejs.org/)，运行:

`node test/app.js`

并访问 [http://127.0.0.1:8000/](http://127.0.0.1:8000/) 。或者用文件名后的参数指定另一个端口。

## 使变小

要将 instantpage.js 缩小为 instantpage.min.js，请全局安装 uglify-es(`npm i uglify-es -g`)，然后运行`npm run minify`。
