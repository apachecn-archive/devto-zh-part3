# 公共 API

> 原文：<https://dev.to/sicksand/public-apis-16dh>

昨天从我的电报频道，有一个网站列出了所有公共 api。我很喜欢 API。尝试从列表中创建至少一个应用程序(在 Android 上)。引起我注意的一个是建议单([http://api.adviceslip.com/](http://api.adviceslip.com/))

它返回简单 json。

`{
slip: {
advice: "You can fail at what you don't want. So you might as well take a chance on doing what you love.",
slip_id: "184"
}
}`

我想使用 unsplash API([https://unsplash.com](https://unsplash.com))作为背景，图像顶部的文本将是建议单的建议。

所以这里是网站:[https://public-apis.xyz/](https://public-apis.xyz/)为你的开发者在那里做一些 MVP。快乐编码。
