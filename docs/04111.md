“创业长久的秘密”

大约 90%的创业公司都会死掉。你知道为什么吗？？？他们根本没有足够的顾客！没有足够的客户意味着没有足够的收入。

顾客是任何企业的生计。没有付费的顾客，生意就会破产。每个创业公司都需要让他们的客户满意并参与进来，以增加回头客的长期目标，因此，让他们的项目成为成功的经典定义。顾客带来了利润，而大多数兴奋和投入的顾客带来了额外 25%的利润。
你的这些忠实消费者更有可能长期追随你。这正是每个创业者想要的。

那么，如何让你的客户参与进来呢？？很简单，通过他们的电子邮件。即使应用通知的兴起和流行，人们仍然每天查看他们的电子邮件。查看电子邮件已经成为每个人日常生活的一部分。见鬼，甚至你也在查看你的电子邮件。

好吧，那我们开门见山吧。这正是你应该做的。我的建议是，你还是继续使用电子邮件自动回复器回复你的时事通讯，然后开始利用名为“群发电子邮件”的活动...GMass。
GMass 是 Gmail 的 Chrome 扩展，可以将大量电子邮件分开，甚至可以回复您与特定客户的最后一次对话。
GMass 的建立不是为了与你的回答者竞争，而是与之互补。因此，让这两种工具携手合作。

GMass 用于方便地跟踪回复您活动的每一位客户，您可以通过这种方式轻松地与您的订户建立关系。因此，现在你不用在 Gmail 上创建新的消息主题，而是可以给你的特定客户写一封个人回复，并像在 Messenger 或 WhatsApp 上一样将你的邮件作为回复发送出去。

您可以选择使用他们的分发技术，一次将电子邮件活动转发给 500 个以上的客户(或 2000 个 G Suite 用户)。任何人都可以向多达 10，000 名收件人发送活动，GMass 会在多天内自动发送，因此您不会违反 Gmail 的限制。
听起来很酷吧？

这只是冰山一角，我的朋友。这里的杀手锏是 GMass“回复群发邮件”。这绝对是一个巨大的营销游戏改变者。每当你回复之前的对话/邮件时，你的事业的打开率会显著增加。

这里的概念是把你所有的客户名单放入谷歌表。添加所有的姓名和电子邮件地址。然后将这些姓名和电子邮件地址整合到您的 Gmail 邮件列表中。一旦您能够发送邮件合并活动，您现在就可以使用他们的“黑仔功能”,设置自动跟进电子邮件，按顺序发送，直到您收到客户的回复。

让你的顾客了解情况很重要。你的客户必须阅读你的电子邮件和你对他们的电子邮件的回复，这一点至关重要。你的客户不仅想了解你创业的最新情况，他们还想知道，如果他们有什么特殊的问题或顾虑，即使他们有成千上万个问题，你也可以具体回答他们的具体问题。

获得客户很难。维护它们更加困难。没有人想听一个有前途的初创公司的悲惨故事，它们不断扩大规模并获得关注，但仅仅几个月就失去了它们。

让他们大部分时间都参与进来。这是关键！让 GMass 成为你维护珍贵客户的伙伴。他们是可以让你的公司更上一层楼的朋友。
