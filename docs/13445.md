# 刚刚完成了我的第一个电子应用程序

> 原文:[https://dev . to/the 24th ds/just-finished-my-first-app-with electronic-18ho](https://dev.to/the24thds/just-finished-my-first-app-with-electron-18ho)

这些天来，我挑战自己，用电子构建一个应用程序。这并不是一个很大的挑战，因为它仍然是 JavaScript，但它更像是建立一个桌面应用程序而不是 web 应用程序的想法。

很多时候我发现自己处于以下情况:*我在网上找到一篇非常好的文章或视频，我觉得很有趣，但我当时没有时间阅读/观看它*。
我梦想有一个简单的应用程序，可以保存我的发现供以后使用，但同时，我想要一个可以在本地存储数据的东西。而不是去搜索这样的 app，早上起来说自己建一个会更好。所以现在我有了 1.0.0 版本，非常基础，很可能有很多错误，但是我为我所做的感到骄傲😁

我留下了到[产品搜索](https://www.producthunt.com/posts/read-it)页面和 [GitHub](https://github.com/The24thDS/read-IT) 库的链接。如果你能给我一个向上的投票或一个⭐星，或者如果你觉得我的应用程序有趣，甚至尝试一下，我将不胜感激。我在等待建议🤔关于改进应用程序和错误报告😫。帮我做得更好！💻

此外，这是我的第一个开发职位🎉

页（page 的缩写）s:我是 0%的艺术爱好者，所以欢迎任何关于改进 UI 的建议😊
