# App-ideas Github 知识库

> 原文:[https://dev.to/florinpop17/app-ideas-github-repository-40p7](https://dev.to/florinpop17/app-ideas-github-repository-40p7)

你有没有过想造东西却不知道要造什么的经历？就像作者有时会有“作者阻塞”一样，开发者也是如此。我和我的好朋友 Jim 一起创建了一个 Github 知识库，在这里你可以找到一系列的应用和项目想法，它们是:

*   对提高您的编码技能很有帮助💪；
*   非常适合尝试新技术🆕；
*   加入你的投资组合，给你的下一任雇主/客户留下深刻印象📁；
*   非常适合在教程(文章或视频)中用作示例📃；
*   易于完成，也易于扩展新功能👌；

这不仅仅是一个简单的项目列表，而是一个足够详细地描述每个项目的列表，这样你就可以从头开始开发它了！

每个项目都有以下特点:

1.  明确的描述性目标；
2.  应该实现的用户情景列表；(这些故事更像是一个指导方针，而不是一份强制性的待办事项清单。如果你愿意，可以自由添加你自己的特性)；
3.  可以改进基础项目的附加功能列表；
4.  一个 Trello 板，可以克隆并用于跟踪你的进展；

该项目仍处于初始阶段，因此欢迎任何反馈和贡献！

此外，如果你喜欢这个项目，一定要给它一个⭐️之星！
