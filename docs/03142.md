# mailgo，一个不同的 mailto

> 原文:[https://dev.to/manzinello/mailgo-a-different-mailto-1col](https://dev.to/manzinello/mailgo-a-different-mailto-1col)

我正在做一个关于 mailto 链接的开源项目。它叫做 mailgo([https://mailgo.js.org](https://mailgo.js.org))。与 mailto 链接的经典行为不同，这个想法是向用户展示一个具有更多可能性的模式，例如立即复制电子邮件地址。这是 WIP，但在网站上你可以看到一个工作演示。
