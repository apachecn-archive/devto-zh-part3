# 你为什么编码？

> 原文：<https://dev.to/oieeaaaa/why-do-you-code-3i67>

我已经写了将近两年的代码，我仍然不停地问自己同一个问题:**我为什么要写代码？**。

当我开始只用 HTML 和 CSS 编写代码时，我才 16 岁。我在我们学校有这个导师，顾问，老师。他的名字叫克里斯。当我在我们的计算机实验室从 udemy 下载一些盗版教程时。克里斯突然走近我，问我是否想参加我们学校的网页设计竞赛。我对他的回答是*“对不起，我不能，因为我对写代码一无所知”*。

Cris，知道我没有任何编程或设计经验，仍然问我这个问题。然后，他告诉我，有一个现金奖的赢家，有一个微小的机会，你会从比赛中失败，再加上在我们学校的编程和网页设计是遗忘。所以我抓住了这个机会，不是因为他刚刚告诉我的，而是因为他对我的看法。我可以做得更多，所以我开始从 youtube 上搜索与网页设计相关的教程，或者使用 HTML、CSS 将设计转换成静态网站。

长话短说；我赢得比赛不是因为我很好，而是因为我是唯一的参赛者。

过了一个月左右。我还是一直想着从事这个职业(编程)。所以当我走这条路的时候，我开始打开浏览器，搜索一些关于如何编程的在线教程。我已经意识到有一个**无限的可能性**我可以做这件事，我也认为是**超级大国**或**我也可以像克里斯对我做的那样指导某人**。

现在我想问:**你为什么编码？**
请在评论区写下你的答案。
