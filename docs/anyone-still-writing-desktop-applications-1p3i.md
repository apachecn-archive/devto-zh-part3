# 有人还在写桌面应用吗？

> 原文：<https://dev.to/randelramirez/anyone-still-writing-desktop-applications-1p3i>

我只是好奇想知道...这些天来，大部分的应用程序都是以 SaaS、Web 应用程序和移动应用程序的形式编写的，现在，我想知道那些目前仍在积极开发桌面应用程序的人的体验。

1.)你用什么来创建桌面应用？

2.)虽然大多数(如果不是全部的话)桌面应用程序正在被重写为 web/mobile 应用程序，但是对于那些仍在学习成为一名开发人员的人来说，投资学习像 WPF 或一般桌面应用程序这样的东西有什么好处吗？

3.)你认为网络/移动应用程序真的会取代所有的桌面应用程序吗，或者桌面应用程序可能会再次崛起，如果我可以的话...再次走红？

我是一名软件工程师，我参与或亲自创建的大多数项目(个人和工作)都是 Web 应用程序，最近我一直在投资移动应用程序开发。
