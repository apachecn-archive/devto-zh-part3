# 演讲者的需求层次

> 原文:[https://dev.to/wiredferret/speakers-hierarchy-of-needs-2773](https://dev.to/wiredferret/speakers-hierarchy-of-needs-2773)

我一直在想，要成为一个快乐的演讲者，我需要什么，我期望什么，我希望什么，在我看来，这是一个需求层次，就像马斯洛一样。

这绝对不是为了“叫出”任何组织者或者让任何人难受。作为一名会议发言人，我的绝大多数经历都是积极的。

# [](#needs)需求

以下是我真正需要组织者做的事情，以使这种合作能够成功。

## [](#before)之前

*   行为准则。我需要你告诉我这是什么，你的执行方法是什么。如果我告诉你我担心某件特定的事情，我需要你认真对待我，因为作为一名演讲者，我的风险状况与一名与会者不同。
*   你发给我的每一页、每一封电子邮件和每一封通信中都有会议日期。不，更多。对你来说，你的会议是你一年的转折点。对我来说，这是一件让我兴奋的事情，但我需要确保我的时间表是正确的。
*   我应该计划去哪个机场？如果你所在的地区有不止一个，这一点尤为重要。DFW 还是达拉斯-爱？BWI 还是里根？中途还是奥黑尔？LHR 还是其他选择？
*   如果在会议日期之前或之后有活动，在我订票之前，我一接受就告诉我。我讨厌错过海滩日/演讲晚宴/雨林漫步/旅游，因为你在我制定计划后才告诉我。
*   你希望我的幻灯片是什么格式，你的视频输入是什么，我们将使用什么样的音频？
*   实际的发言时间有多长，尤其是如果有可能打乱计算的休息时间？

## [](#during)期间

如果我有机会在演讲开始之外的某个时间测试我的视听能力，每个人都会更高兴。如果房间里有超过 10 个人，我要扩音。如果日程安排有任何变化，如果您能确保我收到这条消息，我将不胜感激。

## [后](#after)

*   老实说，没有多少后续措施是必要的。除非行为准则出了问题。

# [](#wants)想要什么

一些让我更有效率或更开心的事情:

## [](#before)之前

*   我喜欢那些告诉我你所知道的关于你所在地区或场地的事情的电子邮件。是的，请告诉我通常的天气，会议中心的哪个门，以及我离当地名胜有多近。写文档是我第一次看到这个的地方，它太棒了。
*   我也对列出我的日程安排的提醒感到兴奋，尤其是像采访或发言时间这样的额外内容。我开始了。向我发送日历邀请的会议可获得额外的特别奖励积分！有时时区很难，好吗？
*   给我一个发言联络和会议时间。我不需要这个，但能参观会场、演讲厅、后台，并且在我需要帮助的时候有人可以找我，这很好。
*   如果你对你的会议上的听众有所了解，与我分享会让我更准确地调整我的演讲。不一定是完整的人口统计数据，但是“去年 78%的与会者将 Java 列为他们的主要语言”意味着我不会谈论 Python 垃圾收集笑话。

## [](#during)期间

*   演讲者休息室并不是必需的，但拥有它是很好的。我用它来放下我的包，和其他演讲者一起玩，练习，交换 A/V 适配器，吃东西，因为我要回答与会者的问题，所以没有吃午饭。DevOpsDays PDX 在演讲厅的一个角落有一个现场直播的演讲，我很喜欢。
*   实时转录。这对我来说很有用，因为我知道更多的人可以理解我说的所有事情，这对于有语言、理解或注意力困难的与会者来说很有用，有时可以用它作为视频的转录。每个人都赢了。我是白大褂字幕的超级粉丝，因为他们似乎有一批预装技术词汇包的转录员，所以非常准确。
*   请把水放在讲台上。我敢肯定，世界上有人记得随时把他们的私人酒瓶装满，并带到舞台上，但我不是其中之一。
*   我喜欢客房领班或司仪做介绍。没有扬声器当然不是什么大不了的事，但也没那么尴尬，尤其是对新的扬声器来说。一个有经验的演讲者会有一些技巧来吸引注意力，让人们静下心来听，但这实际上是一种不同于演讲的技巧。
*   备用笔记本电脑。我的幻灯片很少失败，但当他们失败时，能借一台笔记本电脑也不错。
*   适配器和滑动点击器。我有自己的，但不是每个人都有，即使你有，有时也会丢失或损坏，所以会议应该有一些可以借出。
*   视听技术。这是一个很大的要求，但在能负担得起的会议上，有人来确保麦克风、幻灯片和信心监视器都正常工作还是挺不错的。
*   信心监视器——这是我直视前方时能看到的东西，它向我展示了我身后的投射。许多演讲者的身后都有演示文稿，他们面前的笔记本电脑上有他们的笔记。对于演讲，尤其是在不同的电脑上进行的闪电式演讲，信心监视器意味着我不会扭来扭去检查我现在在哪一页。
*   我认为，会后聚会的目的是让人们互相交谈。把音乐关小，灯亮着。我们大多数人来这里不是为了炫耀我们病态的舞步，而是为了和其他专业人士建立联系。每 10 分贝意味着我在考虑提前一个小时保释，因为我必须能够保持通话，整晚大喊大叫让这变得很难。

## [后](#after)

*   当一个会议给我发邮件说我的视频上传了，我很感激，这是链接。
*   我也喜欢在会议结束后填写调查问卷。我认为向演讲者发送一般与会者调查和专门的演讲者调查都是有用的。
*   请告诉我明年的 CfP 日期。如果是低量清单，我会继续留在上面。我不会放过你的。我可能没有加入你的行列。我只有这么多内存。
*   如果您已经收集了关于我的表现的反馈，请在过滤后与我分享。我对房间里的空调无能为力，你也不希望你们大会的持久味道变成对我演讲的辱骂。建设性的负面评论没问题，但不要只是浏览评论页面而不看。

# [](#desires)欲望

梦想、愿望和难以置信的昂贵想法:

## [](#before)之前

*   从机场接我。DevOpsDays Chicago 和主要开发商就这么做了，不用去想我旅行的那部分，不用去想我应该去哪家希尔顿，等等，这是一种令人惊异的奢侈。
*   为我预订一个带扬声器的房间，即使我的公司会支付费用。在你的会议上和其他演讲者在一起对我来说是一个巨大的价值，作为一个奖励，你知道我们在哪里，而且它不是在运输罢工期间穿过城镇。
*   提前邮寄扬声器礼物，这样我就不用打包了。ChefConf 做到了这一点，这很棒，因为我经常旅行，这使得一个杯子所占的 4 立方英寸变得很重要。

## [](#during)期间

*   确保演讲者休息室里有适合我饮食需求的食物。特别是如果我们在午餐或晚餐前发言，很可能我们花了很多时间与与会者互动，而不是排队，如果发言者需要专门的一餐，这尤其重要。
*   我不知道该怎么称呼这个——礼宾服务？当我带着*无声*出现在首席开发者奥斯汀面前时，组织者给了我一整套的润喉液、舒缓茶、止痛药等，并把它们带到了我的房间。这意味着我不需要知道如何去药店，也不需要考虑我需要什么。老实说，太甜蜜了，我都哭了。希望演讲者不需要这个，但是如果你给了他们一个他们可以信任的联络人，并授权那个人花一点钱，这将会有很大的不同。它可以是任何东西，从“今天是我的生日”(DevOpsDays Hartford 给我买了纸杯蛋糕！)到“我有胃病”(Immodium，没有它就不要离开家)。
*   给我算一个观众。在我演讲的时候，我没有办法做到这一点，但是我不太擅长估计，知道我演讲的出席者的百分比对我的工作很有用。

## [后](#after)

*   给视频加字幕/转录
*   快速播放视频。第二天视频在这方面做得非常好，ConFreaks 的速度明显更快，也有比静态设置更好地处理起搏器(me)的现场摄像师。
*   做一些关于会谈和整个会议的帖子，这样我们就有链接了。

# [](#selfactualization)自我实现？

我一直在想，根据马斯洛和我自己的标准，我会在会议上实现什么样的自我实现，我认为这是关于对活动的基础感到如此自信，以至于我不必去想它们，就像你们大多数人可能不担心住所或热量充足。能够相信会议运行良好，意味着我可以专注于更高层次的事情，如传递价值和引发讨论。

# [](#my-part)我的部分

这是一个很长(真的很长)的清单，列出了会议应该/可以提供给演讲者的东西，但是会议演讲是一个互利的契约。以下是我认为会议发言人应该承诺提供的内容:

*   要准时。对于组织者来说，没有什么比不知道一个演讲者会在合适的时间出现更让人紧张的了。我不在乎你是否有时差，你是否必须穿着睡衣讲话，无论什么。在你演讲前一个小时到达会场，确保你签到，尽可能快地出现在你的房间。

*   喧闹地失败。如果因为任何原因你不能准时完成你的演讲，请尽快告诉组织者。我知道你很惭愧，但他们的处境更糟，所以忍一忍吧。

*   准备好。我确实经常调整我的演讲，以纳入会议早些时候发生的事情，但开玩笑说你如何在飞过来的飞机上匆匆安排了一次演讲，这是非常不专业的。这样想吧——假设房间里的每个人每小时挣 60 美元(这样计算起来就简单了)。现在用这个数字乘以演讲的时间和房间里的人数。这就是你的演讲在人-小时-美元上的价值。

*   参与。会议演讲的很大一部分价值在于你可以参加会议。如果你只在一天中你演讲的那部分时间出现，而忽略了会议的其他时间，你就错过了很多价值，会议也是如此。我在走廊轨道/午餐线上有很多很棒的对话。有时(五月、六月、九月)，你会遇到日程安排冲突，这种事情时有发生，但我向你保证，我会尽量参加会议的大部分时间，与人们交谈，并帮助组织者解决问题。

*   提升。随着我在越来越多的会议委员会中任职，我明白了演讲者走出去，让他们的社区参与进来是多么的重要。把一个会议想象成一个非常大的文氏图的中间。演讲者带来了他们社区的一部分，这使得整个会议更加丰富。

*   额外收获:我已经讲了足够长的时间，以至于我有一系列的演讲可以在没有通知的情况下进行。我通常会悄悄地告诉组织者，如果他们的日程安排出了问题，我可以帮忙。我非常努力地不把这当成是关于我的，而是关于他们需要玩很多球，以及我如何能提供抓住一些东西。
