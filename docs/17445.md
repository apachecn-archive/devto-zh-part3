# 你的……是🎉新年决心🎉？

> 原文:[https://dev.to/sarthology/-git-commit-m-新年-决议-1775](https://dev.to/sarthology/-git-commit--m-new-year-resolutions-1775)

我知道你们一定在想，“啊...不要再这样了。这些东西从来不管用！”但是他们可以。听我说完！最近我在读《纽约时报》的一篇关于新年决心的文章。读完这篇文章后，我意识到我实现今年决心的原因是

[![](../Images/e4c52666016f0bc5678570671ef399b8.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--JP_ZuZcd--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://i.imgur.com/owNSQki.png)

是的，社区。去年，我和某人分享了我的目标，这帮助我实现了目标。你一定看过那些 Youtube 频道，人们通过视频分享他们的身体改造之旅。他们成功的很大一部分是通过录制这些视频来建立责任感。当他们感觉到成千上万甚至数百万人在看着他们时，他们会变得更加坚定。

所以我们把这一年的最后一个周末用来做点好事怎么样？让我们告诉彼此我们的新年决心，如果我们中的任何人能帮助别人实现他们的决心，然后回复他们的评论。🤓

我的决心:嗯，在创作 [Dev10](https://dev.to/teamxenox/-dev10--a-christmas-gift-for-the-best-community-on--44bg) 的时候，我得到了很多乐趣，也学到了很多东西。所以今年，我的主要决心是*我将创建或贡献至少 12 个开源项目*，还有一些其他的小决心。

## [](#bonus)奖金！！

Github 是个很酷的地方。我在浏览 Github 的探索页面时，发现有人有一个名为 NewYearResolution2019 的公共回购。我想该死，这是一个利用社区的帮助来实现你的目标的好主意。所以我做了“决心板”模板。
叉吧！装满它！犯吧！
