## 前些日子..

[https://w.soundcloud.com/player/?url=https://soundcloud.com/foofighters/everlong&auto_play=false&color=%23000000&hide_related=false&show_comments=true&show_user=true&show_reposts=false&show_teaser=true](https://w.soundcloud.com/player/?url=https://soundcloud.com/foofighters/everlong&auto_play=false&color=%23000000&hide_related=false&show_comments=true&show_user=true&show_reposts=false&show_teaser=true)

我碰到了我的健身教练，聊了聊为什么我很久没有锻炼了。我提到我丢了工作，我想念健身，事实上，我买了一根跳绳给我在家健身的时间。

我做了一个 segway，并向他展示了我建立的网站以及我如何在 AWS S3 上托管它们，我只需要弄清楚如何将我的域名链接到 AWS 提供的 S3 域，以便能够以可承受的价格托管朋友的网站。

又聊了几句后，他提到他在健身房训练的一个人，他拥有一家销售住宅&商业照明和灯具的公司，他建议我在脸书给他发一条信息，收集他们是否需要重新设计、管理或开发网站的信息。令我兴奋的是，我打断了谈话，并提到我有差事，所以我必须走了，但在分手前，我告诉他，我将在未来几天工作，我需要他的帮助。

我立刻打开我的迷你电脑，登录脸书，给那个家伙发了一条信息，告诉他我对开发和设计他们公司的网站很感兴趣。当我在他们的公司页面上做一个快速的背景调查时，他们的产品可以在地铁附近的 8 个商店看到，但是他们的主要办公室离我住的地方有 3 个小时的路程！😑😑

我的随意建议是大约 5-7 页，然后连接一个本地 API，用于他们的本地交付和支付，或者使用 Shopify 或开发一个 [Jekyll 主题](http://wowthemes.pxf.io/c/1348602/430741/7553?u=https%3A%2F%2Fjekyllthemes.io%2Fpremium)，使我的生活更轻松。但我更喜欢自己开发网站来积累经验。

下一站是完成我从 [Proposify](https://vagrys.proposify.com/referral) 的提案，完成我的 UI 的&开发我的开发者技能。

干杯！
