# 2019 年有 IT 专业的学位还有意义吗？

> 原文：<https://dev.to/yafkari/is-having-a-degree-in-it-still-relevant-in-2019-4gnk>

# 简史

我的第一句**“你好，世界！”**现在几岁了....*时间过得真快...😨*

从我还是个孩子的时候，我就对计算机为我们提供的所有可能性感到惊讶，我一直喜欢卡通和间谍电影，在这些电影中，一个戴着小兜帽的黑客只需敲几下键盘就可以进入 NASA 系统。

所以我很快开始自学电脑如何工作，什么是编程语言，...

大约在 12-13 岁时，我决定，**今天**我开始编程。

在不同的论坛上搜索了一种语言供选择后，我开始使用 C。像许多说法语的人一样，我从马蒂厄·内布拉在[开放课堂](https://openclassrooms.com/fr/courses/19980-apprenez-a-programmer-en-c)(以前叫做零起点)上的课程开始。
小时候，我在指针指向前停下来，我跟不上。所以我用 [CodeAcademy](https://www.codecademy.com/catalog/language/html-css) 开始了 HTML/CSS。

一件事接着一件事，我发现自己对计算了解得更多，对区块链(当时实际上是比特币)和用 [Root-me](https://www.root-me.org/) 做黑客越来越感兴趣。

最后，在 16 岁的时候，我决定去一家技术计算机子公司。

有了基础，我希望不必在家里工作，这样我仍然可以学习自己，打赌是一个赢的赌注，我可以学习更多关于计算机的内部工作，不同的系统，以及网络如何工作。

我能够学习如何管理思科设备，并获得 Python 的基础知识。

我还有幸品尝了冒名顶替综合症和其他一些很酷的东西。😩😆

但这对我来说永远不够，我总是在家里学得更多，这是我的热情的负面影响之一，因为在家里我学习，而且很有趣。但是当你到了学校，复习你已经学过的科目是非常令人沮丧的，你认为你在浪费时间。

# 问题

今年是我在大学接受高等教育的第一年。不知道该选择哪一个，我求助于 IT 管理。

但是我又一次面临一个伴随我一生的问题。

现在过了+ - 4 个月，**我仍然没有学到任何东西**，我可以在两周内学会开发课程的主题(JAVA)，这并不是因为我比其他人有更高的智力， **faaar，而是因为我喜欢它，我喜欢从自己身上学习。**

因此，我看到自己获得了全职工作，而且薪水比体面的薪水还要高，但每个人都在攻击我，认为学位比找工作所必需的还要多。

# 所以？学位，相关与否？

对我来说，在知识难以获取的时候，学位是能力的证明，但如今，你可以通过互联网学习一切，所以我认为情况不再是这样了。

我相信有了动力，好的资源(网站，书籍，...)你可以成为一名比**更合格的计算机科学家**。

我因此也是泽维尔·尼尔学派 42 或 19 编码学派(比利时版 42)哲学的支持者。

Xavier Niel 自己就是我所说的完美例子，甚至没有学位，他是免费的创始人。

对我来说，一个至少在比利时有计算机专业学士学位的人，只有基础知识却不直接能进公司工作(虽然这里的教育没那么差)。他将在那里接受**培训**和**辅导**，然后他将完全胜任要求他做的工作。

与拥有 CCNA 或类似证书的人相比，这个人可以直接在任何使用思科系统的公司工作。

# 学校 42、解？

我不会说教育系统，至少在计算机领域是失败的。有些人需要被牵着手，有义务，做评估，有老师在身边。

但是有些人需要自由，这就是学校的力量。在那里，我们可以获得资源，我们可以随心所欲地前进。每个人都以自己的速度前进，如果我们不明白什么，我们就问我们的邻居，如果他不知道，我们就问他的邻居，等等...

对于喜欢按照自己的节奏自由学习而不让其他学生分心的人来说，这是一个完美的环境。

那里的学生也学会自己寻找，我认为这在“标准”学校里教得不够。

# 那我为什么不去？

仅仅是因为在 2018 年，公司更感兴趣的是学位，而不是候选人的**真实技能**。

趋势正在逆转，这是一件积极的事情，希望它会继续朝着这个方向发展！

# 结论

所以，对我来说，一个学位并不能证明一个人在某个特定领域有技能，因为他或她在任何地方都可以获得成功。

我认为最重要的是人们的技能，而不是他们获得这些技能的途径。

但是现在，仍然可以说获得任何工作都需要一个学位。
