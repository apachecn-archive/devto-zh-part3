# GPA 不是一切

> 原文:[https://dev.to/ashleemboyer/gpa-isn-t-everything-1p5c](https://dev.to/ashleemboyer/gpa-isn-t-everything-1p5c)

我在全国最好的本科工程学校之一完成了我的学位。申请人通常在高中班级名列前茅，并且拥有优秀的 SAT 和 ACT 成绩。在高中的最后几年里，我做得相当好，并且非常努力地进入这所学校。

我的第一个季度，考了微积分二(出三)，修辞与作文，普通化学一(我一开始是化工专业*喘息*)，还有大学与生活技能。Calc II 对我来说是一场噩梦。我“跳过”Calc I 是因为我的 AP Calc 分数，似乎有一种疯狂的假设，认为我在这一点上是微积分专家。我迷失得太快了。

至于修辞和作文，我在班上大部分时间都做得相当好。最后一篇论文应该是维基百科风格的，但老实说，我不知道如何写一篇研究论文。小型的乡村高中不会让你准备好写一篇大学水平的论文。😳即使在教授质疑我写论文的能力后，他仍然举报我抄袭，我的论文得了 0 分。

知道故事的下一部分是什么吗？这两门课我都不及格，只得了 D 字母的成绩，这意味着我必须在某个时候重考。这在当时绝对是毁灭性的。在这所学校只上了一节课，我的平均绩点就达到了 2.0，而我在这里的表现本应更好。我怀疑我是否应该在那里，因为谁知道我是否能达到要求？

那是 2013 年的秋天。从那以后，我每年夏天都去实习，而且从去年夏天开始，在完成最后几门课程的同时，我还一直在做全职工作。这对于我们学校的毕业生来说并不典型。大多数人“按时”完成(4 年)，有的甚至做到了 ***三年*** 或与 ***两个专业*** 。恭喜你们，我不可能做到的。

那么，我是如何设法锁定好机会的呢？首先，我为参加秋季校园招聘会的公司数量感到非常幸运——大约有 300 家。

第二点，也是最重要的一点，在我看来，我对自己的处境一直很诚实。不管出于什么原因，我课外的个人生活经常像活地狱一样。我很感激那些时光，因为它们让我获得了精神健康方面的帮助，如果没有那时我学到的东西，我就不会有今天。

不管怎样，我总是确保对人们透明地谈论我的生活。我不会涉及每一个细节，但足以让他们知道，我来自一个与其他大多数学生不同的背景，必须为我想要的东西更加努力地奋斗。

我也总是强调这样一个事实，我有一个计划来纠正我的道路。当我有一门课不及格时(这种情况发生了很多次)，我一定要告诉他们我什么时候再接手这门课。这对我如何获得第三份实习产生了巨大的影响。这位人力资源先生直截了当地说:“很好，你有解决这个问题的计划，而且已经开始了。”

我的平均成绩甚至没有达到 3.0。我非常努力地想把它弄到那里。但是事情一直在发生。生活一次又一次地阻碍我们，但你知道吗？我快到了，我做得很好。GPA 并不是决定一个人是什么样的人的唯一因素，如果一家公司根据 GPA 筛选掉你呢？忘记他们。无论如何，有 90%的可能性他们不是那种你想为之工作的公司。

你有一个**故事**。它是**重要的**。在对你不屑一顾之前，任何不愿意听的人都不值得你在**浪费时间。**
