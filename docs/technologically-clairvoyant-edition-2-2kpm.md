🚇[伦敦地铁网络默认开启 WiFi 追踪](https://techcrunch.com/2019/05/22/mind-the-privacy-gap?utm_source=technologicallyclairvoyant.com&utm_medium=email&utm_campaign=Technologically_Clairvoyant_newsletter)

伦敦交通局宣称 WiFi 追踪的好处大于坏处。伦敦交通局可以看到拥挤的地方，并重定向交通，使管道更有效率。一直以来都可以用 WiFi 追踪人。当你的手机在 WiFi 范围内时，它会 pings 路由器(即使你的 WiFi 是关闭的)。路由器可以使用信号强度来估计您离路由器有多远。计算出地铁站里有多少人是可能的。

仅使用 WiFi 也可以精确定位每个人的确切位置。如果你有 3 个路由器在地铁站里排成一个完美的三角形，每个路由器都会知道有人离那个路由器有多远。结合这些数据，你可以三角定位某人的确切位置。

这让我想起了 IMSI 捕手。IMSI 捕捉器是一种通过伪装成无线电发射塔来缓解移动通信中断的设备。在不久的将来，完全缓解交通压力是可能的。

🏎无人驾驶汽车协同工作可以将交通速度提高 35%

在未来，当所有的汽车都是自动驾驶的时候，交通问题将不复存在。这些汽车将知道哪里有交通，并创建一条不会产生更多交通的路线。在计算机科学中，这个问题被称为[最大流问题](https://en.wikipedia.org/wiki/Maximum_flow_problem?utm_source=technologicallyclairvoyant.com&utm_medium=email&utm_campaign=Technologically_Clairvoyant_newsletter)并且已经解决了。如果你有 10，000 辆汽车驶向高速公路交通堵塞的地方，有可能找到一条所有 10，000 辆汽车都可以向同一个方向行驶但没有交通堵塞的路线。郑重声明，我支持用汽车代替司机。[优步和 Lyft 司机强行加价](https://www.theregister.co.uk/2019/05/20/lyft_uber_surge_pricing/?utm_source=technologicallyclairvoyant.com&utm_medium=email&utm_campaign=Technologically_Clairvoyant_newsletter)赚更多钱。未来，当所有的出租车都是自动驾驶时，消费者就不必担心人为的价格飙升。
