# 塑料文化

> 原文:[https://dev.to/leandrogs/plastic-culture-13im](https://dev.to/leandrogs/plastic-culture-13im)

看到我作为一名开发人员如何积极地改变他人的生活，感觉真好。为此，我一直想在一家以用户为中心的公司工作。

在面试中，我很难理解一家公司的重点及其文化。他们是真的关心用户，还是仅仅认为用户是他们“如何”获利的？

几周前和一个朋友( [@phsilbr](https://twitter.com/phsilbr) )聊起这个话题，我们最终得出结论，这是面试中很难识别的东西。也许最好的方法是询问他们使用什么样的标准来确定任务的优先级，并在面试前在 Linkedin 上与一些开发人员交谈。
