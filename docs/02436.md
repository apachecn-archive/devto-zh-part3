# 为爱你的公司工作...

“倦怠是真实的...它能杀死你！

在过去，每个人都想为谷歌工作。但今天，我一直听到很多人从这家赚了 20 多亿美元的公司辞职和离开的悲伤故事。有趣的是，世界上最受尊敬的雇主不再那么诱人了。人们现在意识到它并不适合所有人。
别误会，我爱谷歌！他们的项目让我激动不已。我总是使用他们的产品，比如谷歌浏览器、YouTube、谷歌驱动、Gmail、谷歌地图、谷歌趋势、Waze 等等。
谷歌不是问题，它只是公司变大的事实...然后就开始有了自己的问题。公司越大，责任越大，这通常会导致更大的问题。
谷歌和任何其他大公司都会在任何给定的项目上与大团队合作。这意味着任何给定的人将只负责项目的一小部分，这通常意味着更多的时间在繁重的工作上，更少的时间在高层决策上。许多程序员宁愿在一个更小的团队中工作，在那里他们有更广泛的责任，即使这意味着放弃一些收入和声望。

让我们面对现实吧。每个人都想为大公司工作。公司越大，工资就越高，对吗？大是好事。大受欢迎。当你拥有一家受欢迎的公司时，每个人都想分你一杯羹。每个人都想知道你是否在招人。现在每个人都想和其他人竞争，以获得每个人都在追求的职位。所以很明显，会有竞争。
事实是，竞争不仅仅是激烈的...这太疯狂了。

看看任何一家大公司的招聘启事就知道了。你会看到有无数的申请人冒着一切风险抓住机会，仅仅是为了被录取。一旦他们进去了，噩梦就开始了。他们将面临大量的工作、编码问题和无限的工作时间。到了第三年，他们已经筋疲力尽，没有足够的精力继续他们曾经充满希望的事业。

正因为如此，人们现在倾向于回避大公司，因为害怕被无休止的工作量榨干。他们宁愿为薪水较低的小公司工作，也不愿为充满危险的大公司工作。他们知道倦怠是真实的！为了给他们带来的工作补充创造力、热情和激情，他们需要休息。公司必须明白“人类需要休息”。我们毕竟都是人。

人们需要休假、长周末、静修和探亲假来保持头脑清醒，并在下周一上班时保持精神饱满。人们现在渴望“远程工作”。靠这个。他们有更多的时间与朋友和家人在一起。

人们现在不仅想挣高薪。他们现在更喜欢更多的自由和一个深爱他们的公司。他们是这样的吗？信不信由你，但确实有。有些公司很关心你的健康。意识到员工是他们的“引擎”的公司，它必须得到适当的润滑和维护，才能持续有效地运行。

Buffer 等公司。一家帮助大忙人管理大部分社交网络账户的公司。这家有爱心的公司有一个团队，他们在世界各地的 50 个城市远程工作。他们拥有超过 450 万注册用户的客户群，年收入超过 1600 万美元。

Buffer 对待他们的员工不完全是工人，而是朋友。他们知道什么会激怒你。你的生活兴趣和目标。他们基本上对你了如指掌。因此，Buffer 给他们的员工提供了有竞争力的薪水、健康保险、401k、利润分成、免费书籍和 Kindle。

不仅如此，他们还为员工提供了一台价值 500 多美元的笔记本电脑，这是一项“成长思维基金”，员工可以从中获得额外的资金用于课程、会议和角色发展。Buffer 有“更聪明地工作”津贴，他们的员工可以通过合作空间或咖啡店工作获得额外的现金。他们还可以选择远程工作，尤其是当他们的位置远离总部时。

但是有一件事绝对会让我们大吃一惊，那就是公司慷慨地给员工充足的假期。除了规定的假期、休养、探亲假等....公司给休假。在 Buffer 工作多年后，享受为期 6 周的全薪休假。

当然，这将会从所有那些慷慨的假期中缓冲一大笔钱。但是由于工作倦怠而失去员工的成本更大。
Buffer 完全理解“暂停的力量”。这些值得拥有的停顿让他们的员工可以选择放松，退后一步，有时间倾听自己...他们内心的自我。
当他们回到工作岗位时，这些休息良好的员工会感到更加自信、充满活力，并对他们的组织有一个更清晰的愿景。

伙计们，听我说完。不要害怕工作。工作可以是美好的。工作实际上可以像婚姻一样。婚姻本质上是爱和承诺。是终身的。你的工作也是如此。假设你选择了正确的伴侣(或者你的公司),他们不仅会尊重你，还会珍惜你，照顾你一辈子。

正如我们所说，Buffer 的员工正在体验幸福。对你来说，找到一家能给你同样的幸福和爱情的公司还不算太晚，因为你一直在寻找这样的公司。
