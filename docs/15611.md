# 不要再做一个有毒的人，开始有同理心。

> 原文：<https://dev.to/bbbbruni/stop-being-a-toxic-person-start-to-have-empathy-54h9>

在我们的日常工作中，我们通常会分享想法，进行一些结对编程，并就我们必须解决的问题或我们正在努力解决的问题进行对话。这很正常，很棒，最后当你解决了问题或者解决了你的疑问时，这真是太棒了。

这种情况涉及到很多贡献，团队合作还有最重要的一点，*同理心*。

很多人一起工作，在一些公司里有很多有毒的人在那里搞垮其他人，尽管有毒的人甚至意识到他/她的行为，他/她在那里，对某人有害。

在我的整个职业生涯中，我一直与有毒的人一起工作，与他们交谈的感觉几乎总是一件坏事，在交谈结束时，我感到悲伤和焦虑，因为我无法不在我的团队或工作中制造尴尬的局面。

这种人有时需要帮助，告诉他们不要再这样了，对某人感同身受可能会有帮助，虽然不是直接帮助，但与你的经理/团队领导交谈是有效的。

## ☔️:我之所以这么说，是因为我想从你们那里了解到，这种情况在你们的工作中是否常见，以及如何告诉那些正在中毒的人停止并开始变得更好？☔️

*(我知道有些人改不了，但很多人不知道他们有毒)*
