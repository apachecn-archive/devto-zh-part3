# 我正在写一本关于词的书

> 原文:[https://dev . to/dmerejkowsky/I-am-writing-a-book-about-ci-52cn](https://dev.to/dmerejkowsky/i-am-writing-a-book-about-ci-52cn)

# [](#why)为什么？

我想谈论 CI 已经很久了(超过 2 年)。但是这是一个很大的话题，我花了一些时间才决定开始写一系列关于它的博客文章。

但后来我意识到这是错误的格式:你看，一次写和发布一篇文章需要你确切地知道说什么和以什么顺序，我发现“来自一个农民的建议”系列不是这样。因此，我打算试着把整件事写成一本书。

# [](#how)如何？

现在，我只专注于*内容*:多亏了 [pandoc](https://pandoc.org/) ，我只有一堆可以转换成大 pdf 的降价文件。

一旦完成，我可能会在互联网上发布 pdf 和源代码，免费提供。

# [](#when)什么时候？

我一点头绪都没有，这是我第一次尝试写书！但是我肯定会在博客上更新的——祝我好运吧！
