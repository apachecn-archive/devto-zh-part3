# 面对冒名顶替综合症如何开始写博客

> 原文:[https://dev . to/seankilleen/how-to-start-blogging-when-face-to-imposter-syndrome-23pf](https://dev.to/seankilleen/how-to-start-blogging-when-faced-with-imposter-syndrome-23pf)

我写了一段时间的草稿，作为我发给一个面临冒名顶替综合症的人的笔记，他不确定自己有什么要补充的，我想我应该把它写在这里，而不是藏在我的草稿里。

## [](#what-do-i-have-to-add-to-the-conversation)“我有什么要补充的吗？”

你有一吨。你的故事是非常宝贵的，你的技能绝对可以帮助某个地方的某个人提升水平。

我们需要你的声音！

## [](#but-where-do-i-get-started)“但是我从哪里开始呢？”

开始时，我采用了一些方法:

*   你得到了哪些好的建议？你能根据主题或上下文对它们进行分类吗？嘣，博文。
*   你对什么新事物感到兴奋？为什么？补充一点研究和繁荣，博客帖子。
*   假设有人作为新员工出现在你的工作岗位上。他们很聪明，但是不熟悉你所有的工具和技术。你会先教他们哪方面的技巧？这些技巧的要点可能是一堆博客文章。
*   你现在在思考什么问题？为什么您认为解决方案不太容易获得？有什么办法能让这个解决方案更容易通过吗？博客帖子。
*   你会做些什么来提升人们到你现在的位置？有人需要一座从他们所在的地方到你所在的地方的桥梁。
*   此外，任何花费你更多时间的事情——绝对是好的博文素材。我最受欢迎的博文仍然是“如何在 Excel 中删除表格格式？”成千上万的点击量。我是为自己写的，这样我就不用再寻找确切的步骤了。
*   我还发现公开保留一个主题列表并寻求反馈很有帮助。例如，我所有的博客文章都是以 github 问题开头的。如果事情还没有准备好，你可以让它搁置，并随时获得反馈。
