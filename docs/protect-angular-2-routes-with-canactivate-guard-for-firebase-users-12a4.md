# 使用 canActivate Guard 为 Firebase 用户保护 Angular 2 路线

> 原文：<https://dev.to/singhdigamber/protect-angular-2-routes-with-canactivate-guard-for-firebase-users-12a4>

在 [positronX.io](https://www.positronx.io) 上最先出现了[为 Firebase 用户用 canActivate Guard 保护 Angular 2 路线的帖子](https://www.positronx.io/protect-angular-2-routes-with-canactivate-guard-for-firebase-users/)。

在本教程中，我将解释如何为 Firebase 用户使用 canActivate Guard 来保护 Angular 2 路线。Route guard 防止未经授权的用户访问 Angular 应用程序。

我创建了这个小型 Angular Firebase 演示应用程序，向您展示如何遵循 Angular app 的路线安全最佳实践。CanActivate 接口为 angular app 的 URL 安全提供了最好的例子。
