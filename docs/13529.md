# 开发者到 wordpress

一个 wordpress 插件来显示你的开发者文章。

* * *

## 使用

### 小部件

侧边栏小部件显示给定用户的最新文章。您可以在主题的任何微件区域使用该微件。

[![](../Images/e0da3ea5f0ceec9b05c546b87cb41364.png)T2】](https://github.com/andercodes/dev-to-wordpresssc1.png)

### 短代码

使用短代码显示特定的帖子。语法:

```
[devtowordpres post="https://dev.to/username/post-url-example"] 
```

[![](../Images/50fb750946924a33fccef0f7ed682de4.png)T2】](https://github.com/andercodes/dev-to-wordpresssc2.png)

## [计] 下载

要下载，请查看[最新发布的](https://github.com/andersonjoseph/dev-to-wordpress/releases)。安装。zip 文件从你的 WordPress 管理面板。
