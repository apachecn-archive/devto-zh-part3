# 介绍 ZephJS

> 原文：<https://dev.to/areinet/introducing-zephjs-h24>

ZephJS 使得编写 Web 组件变得超级简单和容易，不需要 Angular 这样的大型框架，也不需要学习 lit-html 这样复杂的 Shadow DOM。它完全基于标准，重量不到 20k。
