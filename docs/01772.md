# 听听那些鼓励你和特里莎·吉一起前进的人

> 原文:[https://dev . to/samjarman/倾听那些鼓励你与 trisha-gee-1jah 一起进步的人](https://dev.to/samjarman/listen-to-those-who-encourage-you-to-step-up-with-trisha-gee-1jah)

Dev Chats 第二季又回来了！在这一系列中，我大约每周都会与一位出色的开发人员或技术人员交谈。你可以在这里阅读更多。如果你知道下一个我应该和谁聊天，请告诉我。

### [](#introduce-yourself-who-are-you-where-do-you-work)自我介绍！你是谁？你在哪里工作？

我是 Trisha Gee，我是 JetBrains 的开发者倡导者，我主要在 Java 社区讨论 IntelliJ IDEA 和 Java 的最新版本。在此之前，我做了大约 15 年的 Java 开发人员。

### 是谁或什么让你开始编程的？

当我大约 9 岁的时候，我的父母在 BBC Micro 上向我展示了一些基本的编程，我立刻沉迷于让计算机按照我说的去做的想法。在我十几岁的时候，我忘记了编程，但在我大约 16 岁的时候又回到了编程，然后在大学里学习了计算机科学。

### 公众演讲对你的职业生涯产生了怎样的影响？

这对我的职业生涯来说太棒了！早在 2011 年，我是伦敦 Java 社区的成员，那里的许多人，包括 Martijn Verburg 和 Ben Evans，鼓励我在那里做一些演示。我的老板马丁·汤普森也邀请我和他一起出席 JavaOne 大会。在做这个报告的一年内，我已经在六个会议上做了报告，这使我能够参加更多的会议。通过这个和我的博客，我找到了我的第一份开发人员宣传工作，顺便说一下，我的第一份远程工作，从此改变了我的工作和生活方式。

### [](#what-is-the-tech-scene-like-in-seville-spain)西班牙塞维利亚的科技场景是怎样的？

比人们想象的要活跃得多！当我在 2013 年搬到塞维利亚时，我遇到的开发人员会告诉我，那里没有什么进展，没有多少开发人员，他们没有在做好的项目。我的[丈夫](https://twitter.com/israkaos)和我创建了[塞维利亚 Java 用户组](https://www.meetup.com/SVQJUG/)，我的丈夫开始运营一个跨技术[塞维利亚开发者组](https://www.facebook.com/groups/sevilladevelopers/)，我们发现有很多开发者在做很多有趣的事情。从那以后，我的丈夫(特别是)一直努力让开发人员聚在一起分享他们的故事并相互学习，现在对塞维利亚的真实技术场景有了更多的了解。

### [](#how-do-you-think-the-developer-advocaterelations-industry-will-change-over-the-coming-years-and-how-might-that-affect-your-role)您认为开发商代言人/关系行业在未来几年会发生怎样的变化，这会对您的角色产生怎样的影响？

我不是很确定！这是一个有趣且相对较新的角色，对不同的个人和不同的组织来说是不同的。也许这可能会在未来几年内正式化(例如，我看到一些人在技术传道者和开发者拥护者之间做出了区分，尽管我仍然不认为有正式的区分)。到目前为止，虽然我只看到这个角色和关系行业的增长。过去，像谷歌和微软这样面向开发者的大型技术公司拥有开发者拥护者。现在，我看到像银行和小型创业公司这样的企业在招聘倡导者，因为这两类公司经常需要与开发者交流，特别是如果他们有开发者使用的 API 或技术。

### 迄今为止，在你的软件生涯中，你学到的最难的一课是什么？

没有正确的答案。任何事。没有人知道我们真正应该做什么。没关系，这只是软件开发的一个基本事实。

### 对于成功的软件职业生涯，你的第一条建议是什么？

与计算机对话是最简单的部分。学习一项新技术是一件例行公事。真正能让你的生活变得更轻松的技能是沟通技能——找出企业需要什么，弄清楚用户想做什么，与你的同事合作，说服管理层做一些事情，所有这些事情都需要你与人交谈。

### 工作之外，你有什么爱好吗？你认为它们对你的科技事业有任何帮助吗？

我跑步，我经历去健身房的阶段。这两件事对保持我的理智和控制我的压力水平非常重要。我还可以在跑步的时候知道自己目前的压力有多大，这是你坐在电脑前不会经常注意到的。但我现在工作之外的主要活动是做母亲，我可以向你保证，育儿技能绝对可以转化为职业技能！如果你能说服一个三岁的孩子刷牙并及时穿好衣服出门，你可能就能处理很多工作中遇到的人形物品

### 你会向那些想走上与你相似道路的人推荐哪些书籍/资源？

最好的资源就是你身边的人。加入一个用户群，倾听那些鼓励你以某种方式展示、写作或提升你的职业生涯的人。让人们帮助你——我在工作和用户组中遇到的人帮助我找到合适的会议开始，并把我介绍给合适的人，所以 a)我不感到孤独，b)我有一个帮助我的人际网络。
