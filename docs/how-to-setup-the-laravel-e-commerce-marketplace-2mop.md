# 如何设置 laravel 电子商务市场？

> 原文：<https://dev.to/pathaksaurav/how-to-setup-the-laravel-e-commerce-marketplace-2mop>

建立一个 [**多供应商市场**](https://bagisto.com/en/extensions/laravel-multi-vendor-marketplace/) 可以让你**吸引卖家**在你的网上商店销售，并且**为你的客户**提供**多种选择**从不同供应商购买产品。在这里，您可以按照 [**laravel 电子商务市场用户指南**](https://webkul.com/blog/laravel-multi-vendor-marketplace/) 轻松设置您的市场。

[**Laravel 多厂商市场现场演示**](https://demo.bagisto.com/marketplace/)

Laravel 是当今**流行的 web 框架**之一，通过它你可以创建出色的 web 应用。 [**Bagisto**](https://bagisto.com/en/) 是一个**开源的 laravel 电子商务**应用程序，它提供了 **laravel 多供应商市场**扩展来建立您的市场。

[https://www.youtube.com/embed/FAp1NYMBdP4](https://www.youtube.com/embed/FAp1NYMBdP4)

[**Laravel multi-vendor marketplace**](https://store.webkul.com/laravel-multi-vendor-marketplace.html)extension 是迄今为止最便宜的 market place 模块之一，可以与 bagisto 一起使用，可以根据您的需求轻松定制。laravel marketplace 扩展提供以下功能:

*   单独的卖家/供应商资料，带有专门的商店网址。
*   通过为他们的商店提供横幅、社交链接和 SEO 友好的 URL 来管理卖家品牌。
*   卖方/卖方在卖方面板上搜索产品。
*   卖家产品系列的单独视图。
*   卖家的反馈和互动审查系统与星级评定。
*   专门的卖家仪表板，查看收入、支出、订单详情、最新评论和评论。
*   正确检查库存可用性。
*   管理员可以为卖家设定全球佣金。
*   管理员可以启用/禁用供应商/卖家
*   管理员批准的产品
*   有日期过滤器的卖家可享受特殊价格。
*   吸引人的登录页面，上面有最畅销的商品及其相关的顶级产品。
*   卖家可以为个人资料页面、收藏页面和店铺信息编辑店铺 URL。
*   多语言支持/所有语言工作，包括 RTL 语([http://en.wikipedia.org/wiki/Right-to-left 语](http://en.wikipedia.org/wiki/Right-to-left)希伯来语和阿拉伯语)。
*   卖家可以从他们的仪表板上编辑或删除产品。
*   一旦订单完成，管理员可以为卖家创建一个“支付”。
*   管理员可以选择生成发票和运输产品。
*   卖家可以通过展示退货和运输政策以及提供店铺描述来保持他们和顾客之间的一切透明
*   卖家可以添加产品的多个图片。
*   卖家可以从管理端检查总销售额，总支出和剩余支出。
*   创建新产品或提供现有产品详细信息的功能。
*   管理员可以选择是否自动批准订单。
*   管理员可以删除或更新卖家的个人资料和产品。
