# fit plant 的 web 应用版本将首先构建

[![workout overview](../Images/ba212eab1887daa66b4381c2360be742.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--2AcIBb_z--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/cujadknd1tdqetvpbnf0.jpg)

最初，我希望 fitPlant 只是一个 iOS 应用程序，但在我从喜欢 fitPlant 想法的人那里获得一些反馈后，我决定首先制作 web 应用程序，然后是 iOS 和 Android 版本。

到目前为止，我正在设计 web 应用程序。
在上图中，您可以看到健身程序概述。
中间会显示您正在种植的植物。在下面，您可以看到您的可用点(如果您知道我们应该如何称呼这些点，请提出建议)。底部的方框显示您记录的所有锻炼天数、锻炼时间和您获得的积分。在“本周”部分，你可以看到本周你锻炼的所有天数。

因为你在种一株植物，而且如果你不锻炼它会死掉，所以你会有很大的动力去定期锻炼，而不会错过锻炼和改善你的健康。
