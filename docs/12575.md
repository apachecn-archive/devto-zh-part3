# 有哪些你离不开的浏览器插件？

> 原文：<https://dev.to/priteshusadadiya/browser-plugins-that-you-cant-live-without-27l8>

我们都需要工具包，让我们的工作变得简单一点。作为一名 QE，我每天都在使用几个 chrome 扩展，它们提高了我的工作效率，减少了大量的额外工作

[BlazeMeter |连续测试平台](https://chrome.google.com/webstore/detail/blazemeter-the-continuous/mbopgmdnpcbohhpnfglgohlbhfongabi?hl=en) 

Blazemeter 插件是在 jmeter 中记录性能脚本的最简单的方法

[臭虫磁铁](https://chrome.google.com/webstore/detail/bug-magnet/efhedldbjahpgjcneebmbolkalbhckfi?hl=en)

这是进行探索性测试的便利工具。它是 [opensorce](https://github.com/gojko/bugmagnet) 并为你提供一系列测试数据来检查你的应用程序的边缘情况。

[查看我的链接](https://chrome.google.com/webstore/detail/check-my-links/ojkcdipcgfaekbeaelaapakgnjflfglf?hl=en)

它是一个链接检查器，可以浏览你的网页，寻找断开的链接。

[探索性测试 Chrome 扩展](https://chrome.google.com/webstore/detail/exploratory-testing-chrom/khigmghadjljgjpamimgjjmpmlbgmekj?hl=en)

另一个必须有探索性测试插件。你可以一页一页的截图，写错误，想法，笔记等..

[注意任何地方](https://chrome.google.com/webstore/detail/note-anywhere/bohahkiiknkelflnjjlipnaeapefmjbh?hl=en)

有多少次你在浏览一篇文章的时候有了一些想法，过了一段时间就忘记了。不会再有了。

有了这个扩展，你可以在任何网页上创建笔记，当你再次打开那个网页时，笔记会自动加载。

[页面加载时间](https://chrome.google.com/webstore/detail/page-load-time/fploionmjgeclbkemipmkogoaohcdbig?hl=en)

快速需要检查你的网页的加载时间？它有一个扩展。

[屏显-屏显录像机](https://chrome.google.com/webstore/detail/screencastify-screen-vide/mmeijimgabbpbgpdklnllpncmdofkcpn?hl=en)

我用这个插件来记录产品工作流程，错误等...

在免费轮胎，它给你每月 50 录音，最长 10 分钟。每次会话记录。你可以从 Google drive 下载保存的视频或者导出为 gif 格式。

我已经用了一段时间，从来没有需要升级到付费计划。

[什么字体](https://chrome.google.com/webstore/detail/whatfont/jabopobgcpjmedljpbcaablpmlmfcogm?hl=en)

最简单的方法来找到什么样的字体被用在网页的任何部分，而不需要进入 devtools 的细节。

当然也是我们一直以来的最爱

[铬债务工具](https://developers.google.com/web/tools/chrome-devtools/)
