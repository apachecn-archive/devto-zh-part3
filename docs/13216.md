# 如何用普通的 JavaScript 编写主题切换器

> 原文:[https://dev . to/just alever/how-to-code-a-theme-switcher-with-vanilla-JavaScript-1 D4](https://dev.to/justalever/how-to-code-a-theme-switcher-with-vanilla-javascript-1d4)

欢迎来到另一个“让我们构建:用 JavaScript”教程。在这一期中，我们将使用 CSS 变量和普通的 JavaScript 为网页的不同部分构建一个主题切换器。

自从引入现代 CSS 变量以来，构建主题切换器从未如此简单。如果你是一个 Sass 的拥护者，你可能会理解 CSS 代码中变量的意义。

在我自己的工作中，我使用变量来保持我的代码尽可能的干燥(不要重复自己)和一致。使用 Sass，您不能像使用 CSS 变量那样使用 JavaScript 来挂接相同的代码。

在常规 CSS 中的给定元素上声明变量允许您遍历 CSS 以找到那些方便的变量。在本视频指南中，我将向您展示如何在 JavaScript 和 CSS 中使用一些现代技术来实现这一令人惊叹的效果。
