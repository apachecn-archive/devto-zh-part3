你打算学习 Python 还是还不确定为什么？这里有 7 个学习 Python 的理由

1.  **数据科学&机器学习:**这几天机器学习和数据科学的默认选择。
2.  **主要应用&更多:**作为一门通用语言，你可以在很多领域使用它。
3.  **丰富的库&框架:**很多积木
4.  **开源**没有单一的 Python 所有者
5.  **可以与其他编程语言配合使用** Java 版本(Jython)或 Iron Python for net
6.  **提高生产率**由于其清晰的语法，代码变得非常易读
7.  **工作&职业**从 Python 开发者做起，收入颇丰
