我从 2014 年开始编程。我同时自学了 web 和 android 开发。到 2015 年，我已经开始为我的同龄人举办 java、web 设计和 php 入门的开发教程。2016 我以为我准备好自由职业了。或者围绕它创业。因此，我有了第一个客户，他想要一个非营利组织的网站，得到了一份 7 个月的合同，在当地学校辅导信息技术，并注册了全日制计算机科学学位。这是一个可怕的组合，我已经吃不了兜着走了，但这是一次很好的学习经历。我的成绩很差，我的自由职业生涯也以失败告终。唯一的成功是我的高薪家教合同。

我认为我足够优秀，可以成为一名员工，直到我开始参与开发人员社区，从世界各地的其他开发人员的故事中学习。这时我意识到还有很多东西要学。

我有几个 github 项目，大部分尚未完成，但有几个是我引以为豪的。我仍然觉得他们在展示我的技能方面对我不够公平。
