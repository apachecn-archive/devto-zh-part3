# TypeScript 教程-基于参数返回

> 原文:[https://dev . to/aexol/typescript-tutorial-return-based-on-args-1 hbe](https://dev.to/aexol/typescript-tutorial-return-based-on-args-1hbe)

您好，这是高级 typescript 教程系列的第三篇文章。今天我将讲述通用函数的基本用法

```
type Point = {
  x?: number
  y?: number
  z?: number
}
const myFunc = <T extends Point>(args: T): T => {
  return args
} 
```

作为参数，我将提供一个包含点属性的对象。该函数将根据`args`参数中提供的参数仅返回`Point`的一部分；

```
const result = myFunc({
  x: 1,
  y: 1,
}) 
```
