# 然而，利兹编码

> 原文：<https://dev.to/lizzkats/nevertheless-lizz-coded--3l02>

## 我的故事...

我于 2017 年从一个为期 10 个月的编码训练营毕业。后来，我在一份我不喜欢的工作上花了太长时间。我认为这完全是由于冒名顶替综合症。2018 年，我决定碰碰运气，辞去那份没有任何准备的工作，试图获得我想要的工作。

## 结果...

[![](img/703e48afbc4103ab5c8a7c246bd29caf.png)](https://res.cloudinary.com/practicaldev/image/fetch/s--gbB3FZMR--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/5yjs54g006kfrbce1i8r.jpg) 
这是我收到一份我认为是理想工作的拒绝一小时后的照片！为了那次采访，我去了夏威夷的瓦胡岛，所以我觉得很幸运，我可以爬上这座山，克服它。更好的是:同一天，另一家公司给了我一份更令人兴奋的工作。我确信我搞砸了那家公司的面试，但是我错了！

## 我给那些仍在磨砺的人的信息...

不要失去希望！我能够通过练习解决算法和在我最感兴趣的技术中从事个人项目来获得自信。还有，不要害怕申请你认为自己不胜任的工作。你可能最终会去热带地区旅行！
