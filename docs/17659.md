# AnyChart JS Charts 8.5.0 发布了 3D 曲面图、单词树和更多新功能

> 原文:[https://dev . to/andreykh 1985/any chart-js-charts-850-released-with-3d-surface-chart-word-tree-and-more-new-features-375 c](https://dev.to/andreykh1985/anychart-js-charts-850-released-with-3d-surface-chart-word-tree-and-more-new-features-375c)

AnyChart 宣布对其 [JavaScript 数据可视化库](https://www.anychart.com)进行新的重大更新——any chart 8 . 5 . 0 发布。

新版本提供了许多很酷的功能和改进，包括:

*   3D 表面图；
*   单词树，
*   Y scroller，
*   在多系列图表中按系列分组，
*   股价图中的时间刻度间隙，

还有更多。

有关详细信息，请查看 AnyChart 博客上的 AnyChart JS Charts 8.5.0 版本的[亮点。在那里，你也可以分享你关于接下来应该添加到这些 JS 图表库中的内容的建议。](https://www.anychart.com/blog/2018/12/25/javascript-data-visualization-libraries-new-features/)

今年早些时候，AnyChart 被评为 JS 最佳，并在加利福尼亚的 DeveloperWeek 上获得了 DEVIES 奖。
