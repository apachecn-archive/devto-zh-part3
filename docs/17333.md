# 我们开始吧🎉

> 原文:[https://dev.to/mscharl/lets-get-started-3bpj](https://dev.to/mscharl/lets-get-started--3bpj)

那是在 2018 年初，当时我在一家刚刚扩展到维也纳的公司担任高级 Web 开发人员，并在寻找新的开发团队。当我第一天以“Web 开发人员”的身份进来时，我非常惊讶地被问及我是否知道如何用 Swift 做 iOS 应用程序。这是一个非常受欢迎的惊喜，因为一段时间以来我一直想更深入地研究 Swift 和 iOS 开发。在那之后，我为正在进行的项目做了一些支持工作，直到维也纳的团队足够大，能够独立完成我们自己的项目。目前，我们专注于为现有库存管理软件的客户开发在线商店和类似商店的应用程序。一项有趣且有时非常具有挑战性的任务。

我的新工作不仅打开了许多新的可能性，而且给我提供了更多的空闲时间。在我安顿下来后，我开始为副业收集想法，但我从未设法开始，而且许多想法从未被写下来。对于 2019 年，这必须改变！

目前，这是我在 [dev.to](https://dev.to/) 上创建账户的主要原因。

## [迈克尔·沙尔](/mscharl)

我会尽力定期写信。我计划大约每周一次，这听起来很可行。不过话说回来，我尝试了很长时间，但从未成功过😅

我强烈希望这只是因为我使用了错误的平台。但只有时间能证明一切。
