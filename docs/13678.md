# [](#security-threats)安全威胁

WordPress 作为最受欢迎的平台，很容易成为黑客的攻击目标。黑客通常试图访问那些一段时间没有更新的网站，因为过时的软件很容易让他们找到漏洞，并利用它来访问网站。幸运的是，WordPress 团队很快修复了这些问题，为用户推出了一个新的改进版本，并堵住了安全漏洞。继续使用旧版 WP 的网站仍然容易受到攻击。

# [](#additional-functionalities)附加功能

WP 的开发者从一开始就一直在努力改进 WP。早期版本和最新版本之间的差异是巨大的。早期版本功能简单，仅提供博客系统，而新版本有能力开发当今任何类型的网站。

# [](#how-to-update-your-wordpress)如何更新你的 WordPress？

首先，你需要知道什么时候更新你的网站是正确的时间。这里的关键是当你的网站流量低的时候更新网站，因为你不想让你的访问者看到你对你的网站所做的所有改变。也是为了避免万一出现问题时出现的复杂情况。

基本上，要成功升级你的 WordPress 网站需要遵循 6 个步骤。

# [](#1-backup-your-website)1。备份您的网站

备份可以确保在更新过程出错的情况下，您始终有第二个网站副本可供使用。在对你的网站做任何改变之前，备份它总是一个好主意。你可以使用 FTP 客户端手动完成，也可以借助备用的 WP 插件来简化这个过程。在你进入下一步之前，确保你已经备份了你的插件和主题。

# [](#2-turn-caching-off)2。关闭缓存

停用缓存，以避免在升级网站时缓存维护页面。你可以关闭你的缓存插件，比如 W3 Total Cache，如果你正在使用的话。

# [](#3-update-themes-and-plugins)3。更新主题和插件

为了更新你的 WordPress，首先更新你的主题和插件是很重要的。这是为了让你的主题和插件兼容最新版本的 WP。通常，WP 更新伴随着主题和插件更新。开发者经常被建议为更新准备好他们的主题和插件。您可以通过单击仪表板->更新来自动更新它们，或者您也可以通过 FTP 客户端设置到您网站的连接并手动更新。

# [](#4-upgrade-your-wordpress)4。升级你的 WordPress

一旦你更新了你的插件和主题，现在是时候更新你的 WordPress 了。只要点击自动更新按钮，让它为你更新 WP。坐下来放松，因为这可能需要一段时间才能完成。

# [](#5-activate-caching-again)5。再次激活缓存

当您完成 WP 更新后，您可以打开缓存并清空缓存。你需要做的就是再次激活你的插件。

# [](#6-test-your-website)6。测试你的网站

浏览所有网页，确保你的网站运行良好，没有任何遗漏。许多网站管理员通常会忽略这一步。然而，这是确保你成功升级 WordPress 网站的最重要的步骤之一。
