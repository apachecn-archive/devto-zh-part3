# 巴什需要指导！

> 原文:[https://dev.to/rpalo/bash-mentors-needed-4nnp](https://dev.to/rpalo/bash-mentors-needed-4nnp)

你喜欢写剧本吗？Bash 晦涩难懂的语法是否让你觉得自己像个巫师？对我来说是的。

在 exercisem 这个网站上，人们可以进行编码练习，提交解决方案，并得到熟悉该语言习惯用法的人的指导。您帮助学生在提交迭代时改进他们的代码和信心，并试图保持自动化测试通过。(没错，连 Bash 都有 TDD！)

我们需要你的帮助。

我们有成千上万的学生试图得到指导，提交解决方案，我们有 10 个注册的 Bash 导师。我甚至不确定是否所有注册的导师都活跃在 Bash 轨道上。回复 Bash 的学生平均需要一周时间。太逊了。这让我很难过。

因此，如果你是 Bash 大师，想要教导和帮助人们成长，请前往[导师注册页面](https://exercism.io/become-a-mentor)。

我鼓励学生充分利用当前的 Bash 特性，比如关联数组、算术、字符串索引和内置命令。如果学生能够想出一个纯粹的 Bash(仅内置和语言功能)解决方案，将会得到加分，因为 Exercism 是关于学习一种语言的习惯用法的。带有 awk、sed、grep 和 tr 的解决方案很好，但是没有什么能打败纯粹的 BASH！(我多半是开玩笑的。在现实生活中，使用所有可用的工具。但是我通常建议纯 Bash 来帮助学生学习可用的东西。)

如果您还不习惯 Bash，但是想要提供帮助，那么从学生开始，沿着轨道以您的方式工作，一旦您感觉足够舒适(您甚至不必完成它！)成为一名导师，只指导简单的练习！

我们需要你的帮助！

点击上面的链接注册。如果您有任何问题，请发送消息、DM、tweet 或电子邮件给我。有一大堆导师可以给你建议。谢谢！

P.S. Python、Rust、Go 也需要导师。可能是其他语言。这些是我想不出来的。
