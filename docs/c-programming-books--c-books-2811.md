你想开始在这方面的发展，但你没有办法？注意我们为您准备的书籍选择:

### 中书

C 语言书籍向你介绍了 C 编程语言，并提供了一种结构化的方法，面向具有一些高级语言经验的专业人士和学生。除了他关于语言理论的好文章之外，还要复习他的许多实践练习。此外，它还有一个非常有趣的部分，是用 c 编程时最常见的错误。

### 如何像计算机科学家一样思考 C ++版

如何像计算机科学家一样思考 C++版本将教你开始用 C++编程的基础知识，并通过 C++特有的结构化和面向对象的设计，将你的文本集中于思考你的实践。它是在 GNU 自由文档许可证的条款下发布的。如果你决定购买它们，你应该知道筹集的资金将支持自由软件和文档的开发。

### 用 C++思考

[Thinking in C++](https://www.micc.unifi.it/bertini/download/programmazione/TICPP-2nd-ed-Vol-one-printed.pdf) 根据多年来在世界各地教授的研讨会撰写，将 C 程序员带到掌握 C++最简单的途径:在程序员已经知道的范围内一步一步地学习。这是一本在 C 和 C++之间架起桥梁的非常有用的书，它将帮助那些看起来不可思议但在面向对象编程方面没有更新的程序员。

这本书的目标读者是具有 C 语言(或任何其他语法类似于 C 语言的语言，如 Perl 或 Java)知识的用户，他们希望了解更多或过渡到 C++。本文档是荷兰格罗宁根大学每年组织的 C++课程的主要教材。 [C++注释](http://www.icce.rug.nl/documents/cplusplus/)并没有涵盖 C++的所有方面，但无论如何它会帮助你在 C 和 C++之间迈出你非常需要的那一步。你可以回顾一下你以前对 C 语言的处理，因为在本书中，它们是从这门语言的知识库开始的。
