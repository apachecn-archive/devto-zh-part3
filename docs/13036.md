# Magento 2 的最新版本提供了什么？

> 原文：<https://dev.to/bhavikrocktech/what-magento-2s-latest-version-has-to-offer-1o34>

当谈到电子商务，没有比 Magento 2 想到的，为什么不呢，因为它已经采取了电子商务发展到一个全新的水平，其宏伟的能力。Magento 的功能总是令人惊讶，现在随着它的最新更新，Magento 2 的功能列表在很大程度上已经走得很远了。今天，我们将参观它。

Magento 2 有什么特点？

我们已经享受了 Magento 2 中新的文件结构、优化的性能、改进的灵活性、jQuery 库和新的内置技术，现在，随着 Magento 家族的最新成员——Magento 2.3 的加入，可能性进一步增加。

Magento 2 新功能列表

一个全新的管理面板，具有简化的工具导航，可以根据每个用户进行个性化设置。

新的管理员安全性意味着每个管理员用户帐户只能有一个人登录。

反应灵敏，具有触摸友好的导航。通过其拖放式布局编辑，您可以在几分钟内随心所欲地编辑您的在线商店。

新增和改进的产品部分，新增了用户通知选项
增加了“销售-订单网格”页面和“产品-目录”网格页面的定制选项。

出色的 M2 过滤器非常适合组织页面上的结果，并查看产品或订单的特定子集。利用它，你还可以分析数据，监控顾客在你店里的行为。

不太复杂的结账流程。更快的结账过程大大有助于提高店主的转化率。

安全可靠的支付方式。流行的扩展，如 PayPal 和 Braintree 等支付网关，为商店提供更好的功能。
