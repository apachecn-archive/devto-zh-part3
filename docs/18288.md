# 新款 MacBook Pros 定价过高

> 原文:[https://dev . to/rhymes/the-new-macbook-pros-is-over-priced-j5m](https://dev.to/rhymes/the-new-macbook-pros-are-overpriced-j5m)

我使用 Mac 已经十多年了，但我对苹果最近发布的一批电脑缺乏热情(失望至少始于两三年前)。

他们与第一代基于英特尔的 Macbook Pros 在质量和耐用性方面的优势已经不复存在(当时 Windows 也不太好，我厌倦了桌面上的 Linux)。

我的 2012 款 Macbook Pro 13“仍然运行良好，证明了这种质量，但是……”时代在变化”。

我的感觉是，苹果公司忽视了他们的电脑，而是专注于智能手机。这没什么不好，他们是一家私人公司，他们可以专注于他们想要的任何产品。

我的电脑开始显老了，我在考虑 2019 年换一台。

由于一些关于 dev.to、[一些文章](https://medium.com/charged-tech/why-i-left-mac-for-windows-apple-has-given-up-b48c0eaac64)和 Linux 的 Windows 子系统的讨论，我对切换回 PC 和 Windows 10 的想法保持开放的态度。

我不得不承认，我有点害怕回到 Windows。作为一名用户，我很长一段时间都讨厌 Windows，现在每次我坐在 Windows 电脑前，我的第一反应或多或少都是“我该如何操作这个东西？如果我点击这个会发生什么？他们还有病毒吗？”。我坚信 macOS 使用起来要简单得多，但是如果数百万人都喜欢 Windows，我想我肯定也能活下来。

昨天我还看了我最喜欢的 youtubers 用户之一 Sara Dietschy 的评论，我印象深刻:

[https://www.youtube.com/embed/9uIy8n-nsB4](https://www.youtube.com/embed/9uIy8n-nsB4)

(她一直在慢慢放弃苹果笔记本电脑和工作站，因为她主要做视频编辑)

今天，我试图将戴尔 XPS 笔记本电脑(13 英寸和 15 英寸)与 MacBook Pros 和 well 的同等配置进行比较...MacBook Pros 价格在我看来有点太贵了。

这是我用[概念](https://www.notion.so/)做的对比表:

[![](../Images/51c3a3e73be8c2bee2df25d86d1c839d.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--CFpN5Mr4--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/r7x0q7soxb4ccgix6q91.png)

戴尔 XPS 分别为 1549 欧元和 1769 欧元，MacBook Pros 分别为 2270 欧元和 2417 欧元。700 欧元的差额😱戴尔笔记本电脑也是可扩展的，因此您可以放置 32 GB 的 RAM 并在未来增加存储。

戴尔 XPS 最大的硬件怪癖是网络摄像头的位置(在底部！？)，但 MacBook Pros 有那种阴森可怖的触控条和人们喜欢抱怨的怪异蝴蝶键盘。戴尔 XPS 15 英寸机型价格低近 700 欧元，配有 4K 触摸屏、SD 卡和...不同类型的端口(需要较少的适配器和加密狗)。

你觉得 2018 款 MacBook Pros 机型定价过高吗？我错过了什么吗？
