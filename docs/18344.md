# DB2 JDBC 驱动程序

> 原文:[https://dev.to/adzubla/db2-jdbc-driver-2hif](https://dev.to/adzubla/db2-jdbc-driver-2hif)

转到`db2jcc4.jar`所在的文件夹，执行:

```
$ java -cp db2jcc4.jar com.ibm.db2.jcc.DB2Jcc -version
IBM Data Server Driver for JDBC and SQLJ 4.25.13 
```

如果您只需要下载 DB2 驱动程序:[https://www-01.ibm.com/support/docview.wss?uid=swg21363866](https://www-01.ibm.com/support/docview.wss?uid=swg21363866)
