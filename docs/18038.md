# HTML 和 JS 生了一个孩子，名叫 JSX。

> 原文:[https://dev . to/diomedes/html-and-js-had-a-baby-named-jsx-3a7n](https://dev.to/diomedes/html-and-js-had-a-baby-named-jsx-3a7n)

首先，我打开终端，运行 npm，在项目上键入“create-react-app jsx”
，然后删除 SRC 文件夹中的所有内容，并创建 index.js。添加了一些注释来指导我使用双反斜杠(//)。插入 React 和 ReactDOM 导入。然后为新创建的组件创建 Conts 应用程序。最后，使用 ReactDOM 添加一个 render 方法，并使用 querySelector 从 index.html 获取根 ID。

功能组件中混合了 JSX 元素，终端中混合了 NPM 元素，我最终在屏幕上看到了我的第一个组件。“与 React 一起生活”
