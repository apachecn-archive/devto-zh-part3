# MongoDB Atlas 数据浏览器

> 原文:[https://dev.to/kenwalger/mongodb-atlas-data-explorer-4km4](https://dev.to/kenwalger/mongodb-atlas-data-explorer-4km4)

在最近的一次 MongoDB 上。芝加哥当地活动，MongoDB 首席技术官和联合创始人 [Eliot Horowitz](https://www.twitter.com/eliothorowitz) 就 MongoDB Atlas 的数据浏览器功能发布了一个令人兴奋的公告。现在，它适用于所有规模的 Atlas 集群，包括空闲层。

#### [](#the-easiest-way-to-explore-your-data)探索数据的最简单方式

什么是数据浏览器？这个强大的特性允许您直接从 web 浏览器查询、探索和处理 MongoDB Atlas(具有完整的 CRUD 功能)中的数据。当然，我们考虑过安全问题；数据浏览器的访问权限以及用户是否可以修改文档取决于她在 Atlas 项目中的角色。通过数据浏览器执行的操作也记录在 Atlas 报警窗口中。

将这一特性引入“共享的”Atlas 集群大小(免费的 M0s、M2s 和 M5s)可以实现更快的开发速度。现在，您可以在开发应用程序的同时对数据执行操作，这正是这些共享集群大小真正发挥作用的地方。
