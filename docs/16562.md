# 2019 年我的火狐插件

> 原文:[https://dev.to/oliverburn/my-firefox-add-ons-in-2019-130o](https://dev.to/oliverburn/my-firefox-add-ons-in-2019-130o)

这是我目前在工作中使用的 Firefox 插件列表。分享到这里(可能吗？)兴趣(这样我就可以有备份了)。它们都可以通过[https://addons.mozilla.org/en-US/firefox/](https://addons.mozilla.org/en-US/firefox/)买到。

*   [自动丢弃标签](https://addons.mozilla.org/en-US/firefox/addon/auto-tab-discard/) -自动丢弃不活动的标签以释放资源

*   鼠标手势插件，支持 60 多种不同的命令和附加功能，如摇杆和滚轮手势。

*   [HTTPS Everywhere](https://addons.mozilla.org/en-US/firefox/addon/https-everywhere/) -加密您与许多主要网站的通信，使您的浏览更加安全。

*   [隐私獾](https://addons.mozilla.org/en-US/firefox/addon/privacy-badger17/) -自动学习屏蔽隐形追踪器。**被警告——它擅长破坏网站！**

*   使用 goo.gl 服务缩短链接和当前标签的 URL。

*   [树形标签](https://addons.mozilla.org/en-US/firefox/addon/tree-style-tab/) -提供树形标签栏，就像 Windows 资源管理器的文件夹树。从链接(或等)打开的新标签。)会自动附加到父选项卡。如果你经常使用很多很多的标签，这将有助于你的网页浏览，因为你可以了解标签的关系。

*   [ub block Origin](https://addons.mozilla.org/en-US/firefox/addon/ublock-origin/)-高效阻断器。轻松使用 CPU 和内存。

*   [YouTube 高清](https://addons.mozilla.org/en-US/firefox/addon/youtube-high-definition/) -让你播放所有的 YouTube 高清视频，并自动改变视频播放器的大小。
