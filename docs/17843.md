# 您的 ticks 的 ckd.css

> 原文:[https://dev.to/equinusocio/ckdcss-for-your-ticks-5033](https://dev.to/equinusocio/ckdcss-for-your-ticks-5033)

ckd.css v2 已经发布，它将为您的复选框和收音机带来漂亮的小微交互。您甚至可以在自定义输入上使用它。

## [](#why-are-micro-interactions-important)为什么微互动很重要？

> 很可能你的一天是从微互动开始的。通过关掉手机闹铃，你瞬间就进入了用户界面。
> 
> 您将继续用您的数字设备参与这些时刻。每一个都是微互动。每一个都可能小到你都不会去想它。因为这个简单的事实，每一个都行得通。
> 
> 作为一名设计师，认识到微交互的不可见性和设计它们一样重要。你必须创造一些有人情味的东西，完成一项任务，让用户的生活更轻松。你还必须专注于一个可以在各种环境下工作，不需要任何指导或解释的设计。
