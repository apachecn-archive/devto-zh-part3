# 每个人都认为 Web 开发人员拥有的技能

> 原文:[https://dev . to/flipped coding/skills-the-every one-thought-web-developers-has-22 of](https://dev.to/flippedcoding/skills-that-everyone-thinks-web-developers-have-22of)

当你告诉人们你是一名网络开发人员时，他们开始把你看成一个不同的人。你会收到这样的问题，“为什么我的 Wi-Fi 坏了”或者“你能告诉我最好的奶酪网站吗”。当你周围的非技术人员开始认为你是互联网和它所拥有的一切的大师时，这可能会变得非常滑稽。

一旦你展示了你的网络开发能力，你的生活将以最有趣的方式改变。

## 你现在知道如何解决电脑问题了。

如果你从未使用过比闪存盘更复杂的东西，那也没关系。一旦人们知道你以某种形式与计算机打交道，你就会成为解决所有计算机问题的关键人物。准备好向人们解释你不确定为什么他们的鼠标不工作或者他们需要的文件夹发生了什么。

尽管如果你有足够多的朋友、家人和同事反复问你，你会开始知道这些问题的一些答案。很快你可能真的知道如何解决电脑问题。

## [](#you-control-the-internet)你控制了互联网。

网络开发人员制作网站，这意味着他们控制所有网站，对吗？有人甚至会问你为什么网飞会崩溃，或者你是否能加快它的速度。在你向他们解释几次你不能访问所有网站后，他们会暂时放弃。

然后会有别的东西弹出来，你会有另一个关于你甚至不知道的网站的随机问题。好的一面是，你可以向他们解释服务器和网络安全，他们会学到一些新东西，并告诉更多的人。

## [](#you-are-tech-support)你是技术支持。

如果你在一家将 it 和软件混为一谈的小公司工作，这一点尤其正确。所以，当有人向你抱怨网络瘫痪时，不要感到惊讶。此外，如果你不知道该怎么办，当他们有点生气时，不要感到惊讶。

准备好回答用户关于你正在开发的 web 应用的问题，同时你也在解决他们抱怨的问题。准备好磨砺那些关键的人际交往技能，因为这是他们成长最快的时候。

我知道有些事情听起来很荒谬，但是你会对人们问的问题感到惊讶。不是每个人都懂技术，大多数时候，当你向他们解释事情而不显得高人一等时，他们会很感激。

希望你能从中得到一些笑声。因为人们知道你写了某种代码，你遇到过哪些最奇怪、最与 web 开发无关的问题？人们认为我是黑客，只是因为他们在黑色 IDE 中看到了一些 JavaScript。有时候跟着他们走更容易。
