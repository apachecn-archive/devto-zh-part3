# [](#whats-is-a-database-management-systems)什么是数据库管理系统？

数据库管理系统是一种用于正确存储、检索和更新数据库中数据的软件。数据库是相关数据集合。

视图是从查询返回的数据的子集(查询是对数据库中数据的请求)。视图向用户隐藏不必要的详细信息，并提供安全性以防止用户访问受限制的数据。

数据结构在数据库管理系统中非常重要。数据库的整体设计称为模式。模式描述了数据库的逻辑设计。模式是如何构建数据库的蓝图。

数据模型是描述数据、数据中的关系以及数据一致性和约束的一种方式。有四种类型数据模型:

1.  关系模型
2.  实体关系模型
3.  基于对象的数据模型
4.  半结构化数据模型

关系模型和实体关系模型是最广泛使用的数据模型。

数据库系统提供了用于指定数据库模式的数据定义语言和用于与数据库交互的数据操作语言。这不是两种独立的语言，它们形成了一种单一的数据库语言，比如广泛使用的 SQL(我们将在以后的文章中讨论)。

DBMS 需要确保数据在崩溃时是安全的，并防止未经授权的访问。由于应用程序中的数据通常由多个用户共享，数据库管理系统必须避免不一致的结果。

应用程序经常在数据库上执行多个操作，这些操作算作一个工作单元。事务必须包含以下一组属性:

1.  原子数
2.  一致性
3.  隔离
4.  持久性

这些特性保证了可靠性，即使在出错或断电的情况下也是如此。并发执行的事务必须无冲突地执行。

数据对每个应用程序都很重要。数据库可以被认为是应用程序的心脏。
