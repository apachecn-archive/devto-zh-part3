# 大学 vs 训练营 vs 自学

> 原文:[https://dev . to/dylanesque/college-vs-boot camp-vs-自学-41lb](https://dev.to/dylanesque/college-vs-bootcamp-vs-self-taught-41lb)

**2019 年 1 月 25 日**

欢迎来到《自学开发者生存指南》第二期官方词条！今天我将比较和对比学习如何编码的主要方法，以及如何辨别哪种方法最适合你的情况。

### [](#college)学院

大学显然可以为你打开很多大门。仍然有一些公司甚至不会考虑没有大学学位的求职者，如果你有麻省理工学院或斯坦福大学的学位，你将会吸引一些人。计算机科学基础，如数据结构和算法，将在大多数计算机科学或软件工程项目中得到很好的覆盖，通过对这些经常出现的话题记忆犹新，使最近的大学毕业生在面试过程中具有优势。如果你正在读这篇文章，并且还在上高中，你应该强烈地认为大学是首选。此外，拥有一个学位可以让在不同国家找工作的过程不那么令人头疼。

有什么坏处？事实证明，很多。这些年来，大学变得非常昂贵，很容易成为学习编程最昂贵的方式。这些年来，大学里发生的另一件事是，它不再是过去那种有围墙的知识花园了。互联网显然已经使许多信息民主化，这使得支付社区大学的学费变得很难，更不用说像哈佛这样的地方会花费 7 万美元。更糟糕的是，大多数计算机科学项目的课程已经相当过时，让学生更多地准备计算机科学的理论方面，而不是学习如何很好地编码。当你考虑到网络发展的快速步伐时，这变得更加扭曲，这是学术界无法跟上的。最后，还有时间:虽然在全职工作的同时参加在线/兼职课程是相当可行的，但这需要一段时间，即使你只是获得了一个大专学位。

### [](#coding-bootcamp)编码训练营

对程序员的爆炸性需求和获得大学学位所需的时间导致了编码训练营的兴起。训练营在教授编码方面明显优于大学的几个弱点。首先，课程更侧重于在生产环境中更直接有用的材料，而且课程比你在大学里能找到的更现代。紧张的节奏、时间表和与他人一起工作的练习将是一次无价的经历。训练营通常与技术雇主和其他有价值的网络资源有协议。如果你正从另一个职业(如医学或法律)转型，并且你有大量的可支配收入，这无疑是你的最佳选择。

你可能已经注意到，我上面提到的受益于训练营的关键人群非常具体，原因如下:训练营可能会花很多钱，而且在这里列出的所有选项中，他们有最严格的时间要求，通常每周 40 小时或更多，持续 3-6 个月。正因为如此，当你参加训练营时，你需要放下一切，放弃那段时间的收入。另一个可能的理论(但很难明确证明)是，训练营选择性地只录取那些表现出足够能力学习编程并在没有训练营帮助的情况下找到工作的候选人，从而降低了毕业后的就业率。由于大量的时间和金钱投入，未来的学生必须非常仔细地审查项目。我怀疑训练营的好比坏多，但即使是一所普通学校的大学学位也比从训练营毕业更有用，至少从一些雇主的角度来看是这样。

### [](#selftaught)无师自通

相比大学和训练营，自学编程有一些很大的优势。虽然没有训练营快，但这条路线比大学快得多，而且便宜得多。这条道路为学生提供了最灵活和最新的课程。时间安排的灵活性可能是最大的优势，也是为什么它对许多想转行的成年人来说是最有吸引力的(或唯一的)选择。这对那些负担不起大学或训练营所需的时间、日程或资金投入的人来说至关重要。大约有三分之一的程序员是自学的，所以你会在一个好公司里。

自学路线有一些重大的缺点，伴随着它强大的优势。学生对管理他们的学习材料、保持进度、评估他们的进步以及根据需要调整他们的方法负有全部责任。有很多机会会被要学习的技术的广度和深度分散注意力(看看你，JavaScript！)，并被其淹没。他们必须更加努力地工作，以确保他们的知识没有更正规的教育所能提供的巨大差距。找工作时，他们将不得不更加努力地推销自己，以弥补学位或训练营项目的不足。对于那些不寻求像 [freeCodeCamp](https://www.freecodecamp.org/) 或 [Dev.to](https://dev.to/) 这样的支持社区的人来说，这可能是孤立和孤独的。

### [](#why-i-ultimately-choose-the-selftaught-route)为什么我最终选择了自学路线

在对 Code Academy 和 Treehouse 等网站进行了一些在线实验后，我报名参加了当地社区大学的一个 web 开发项目。我年轻时从未完成大学学业，所以我想要一个学位所能提供的合法性。事实是，虽然我在上课的时候确实学到了一些有用的东西(SQL，Linux ),并且有一些非常棒的选修课，但是核心课程已经非常过时了。有多过时？Flash 仍然被列为必修材料，而该课程中最新的技术是 jQuery.......2016 年。除此之外，还有学费、课时短缺和时间安排上的困难，我知道我必须找到一种更好的方法。可笑的是，由于我的经济状况，新兵训练营是不可能的，所以是自学的。

### [](#which-is-best-for-you)哪个最适合你？

显然，在决定哪种编程教育方法最适合你时，有很多个人因素。大学费用可能会因为雇主提供免费学费而减轻，或者如果你在没有具体途径的情况下学习困难，自学途径中缺乏指导可能会带来问题。TL；博士看起来是这样的:

*   大学:有助于求职的合法性，计算机科学基础。不利于成本和时间效率。

*   训练营:适合教授在工作中真正有用的技能，人际关系网，快速提高。不利于调度和成本。

*   自学:有利于灵活性，保持现状，低成本。不利于清晰的结构和指导。
