# 体验、接触和知识的积累

> 原文:[https://dev . to/Stephen Weiss/experience-exposure-and-the-building-of-knowledge-5g7h](https://dev.to/stephencweiss/experience-exposure-and-the-building-of-knowledge-5g7h)

几周前，我从一个完全的新手变成了受邀在五天内就一个主题发表演讲。这种经历令人羞愧，整个过程中我都觉得自己像个骗子。从那以后的几天里，我想了很多关于这件事的事情，以及我似乎是如何建立必要的基础，以便能够把我学到的东西教给我的同龄人。

# [](#the-experienceexposure-cycle)体验-曝光周期

经验暴露循环背后的基本前提是，通过将问题的理论背景与实际动手经验相结合，可以最好地(快速)积累知识。当然，在某些领域，这种结合并不是绝对必要的，但是，特别是在技术领域，我发现这种结合特别有益——即使我并不总是有这样的词汇。

为了理解这个循环是如何工作的，我发现想一个简单的例子是有用的。假设约翰是一个想学微积分的学生。如果他从未接触过代数或解决过问题，他将会陷入困境。但是，即使回到那个水平，仅仅听讲座(即使是最好的讲座)也不足以为他准备微积分打下基础。

暴露是讲座，而经验是以实践问题的形式出现的。

曝光提供了一个观察问题的镜头，并能提供理解问题所必需的词汇。除非你付诸实践，真正体验到这个问题，否则你不可能知道自己对这个话题的理解程度。经验有两个明显的好处:

1.  它强化了你大脑中把概念和问题联系起来的路径。从这里你可以开始看到新的模式，并想象其他应用。
2.  它照亮了你知识的缺口。当你看到一个你不知道如何处理的问题时——不管是不是因为你从来没有接触过它——你知道有新的信息在你的研究中寻找下一步。

## [](#building-a-foundation-takes-time)打基础需要时间

在学校环境之外，大多数学习都是自我指导的，并且通常在头脑中有一个特定的目标。我们不需要知道一切。我们只需要学到足够的东西。

即使我们将研究范围限制在手头的某项具体任务上，我们也常常需要知识基础。根据你带着什么来，这可能需要或多或少的时间。

了解*你在这个过程中的位置*几乎和了解整个过程一样重要，因为你需要的曝光资源会发生巨大的变化。

回到我的案例，直到第四天晚上，当我读到一篇将碎片整理到位的文章时，我才觉得自己完全理解了这个话题。

然而，关键不是我有了突破，而是我在第一天就阅读了同一篇文章！然而，在那个时候，我没有必要的基础来理解作者试图传授的知识。我花了三天的时间阅读、练习和努力寻找这个话题，以便能够回到这个话题并收集一些信息。

读完这篇文章后，我发现我也没有把所有东西都锁起来！我发现，这是因为我采用了这些原则，并试图将它们应用到我的具体用例中，却遇到了困难。但是是接触和经验的结合给了我信心。将一个概念应用于*不同的*情况意味着我已经建立了联系和可扩展的理解基础。

# [](#conclusion)结论

没有曝光的体验是一种令人沮丧的体验。似乎没有什么事情是有意义的，或者按照你想要的方式运行。

没有经验的接触可以刺激智力，但很少有用。

接触可以引入新的想法和熟悉的话题，这样它们就不会陌生了。经验暴露了理解上的差距，建立了解决问题的技巧，并为概念建立了联系。

当你把它们放在一起，然后冲洗并重复时会发生什么？嗯，这简直是魔术，也是我加速学习的秘诀。
