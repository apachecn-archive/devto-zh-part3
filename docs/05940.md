# 德瓦特。网络连接解决方案现在支持 Visual Studio 2019

> 原文：<https://dev.to/devartsoftware/devart-net-connectivity-solutions-now-support-visual-studio-2019-3p03>

Devart 是一家公认的为开发人员和 DBA 提供专业数据库管理软件的供应商，它发布了一个急需的产品更新。从现在开始，dot connect ADO.NET 数据提供程序、LinqConnect 和 Entity Developer 将与微软最新版本的 IDE 完全兼容。

概述的工具将更加有用，因为它们支持 Visual Studio 2019，后者已经为用户提供了诸如改进的性能、更好的搜索、Visual Studio Live Share 等功能。

以上仅代表最新亮点。查看 dot connect ADO.NET 数据提供程序的其他有用功能的完整列表——增强的数据连接解决方案，linq connect——LINQ 到 SQL 兼容的 ORM 解决方案，以及 Entity Developer——ORM 模型的顶级可视化设计器。
