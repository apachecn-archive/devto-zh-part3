# 创建网站前的良好习惯和程序

> 原文：<https://dev.to/yoonsl/good-habits-and-precedure-before-creating-a-website-2gmj>

世界上有如此多的网站和许多方法来创建这些网站。我将在这个博客中谈论的方法是使用 html，css 和 Javascript 来创建一个网页网站。

在创建网站之前，画出网站的基本格式是一个好习惯。比如标题的位置应该在哪里，快捷方式应该在哪里，页面信息应该在哪里更改。这对 html 和 css 部分来说是件好事。在开始创建网站之前，了解网站展示将如何结束非常有帮助。

对于 JavaScript 部分，由于我们已经有了网站的轮廓设计，我们将知道在哪里使用 jquery 或 DOM 呈现 JavaScript。开始使用 JavaScript 时，最好先考虑一下需要哪些简单的特性。例如，当创建待办事项时，需要的基本功能是显示列表、添加到列表、检查待办事项是否已完成以及删除不需要的待办事项。只要在创建网站之前考虑到这一点，我们就可以在创建网站时设定我们的道路应该如何走。

将这些想法写在纸上是一个好习惯，就像头脑风暴一样，这样就很容易看出首先要做什么。

在这个 todolist 例子中，首先我们要决定输入框和按钮框的位置。然后我们应该决定 todolist 应该在哪里呈现。用 CSS，显示应该如何设计。然后我们可以先创建显示功能(渲染)。并创建一个函数来添加 todos。然后让函数检查已完成的待办事项，然后让函数删除已检查的功能。
