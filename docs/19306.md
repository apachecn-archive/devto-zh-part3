# 哈希码，2015

> 原文:[https://dev.to/laneone/hashcode-2015-4lbb](https://dev.to/laneone/hashcode-2015-4lbb)

HashCode 是另一个举办得非常好的 PES 黑客马拉松，有一些非常好的竞争，它是在 2015 年 11 月在 PES 大学举办的。

黑客分为 4 个主题，最后一个主题是一个开放的黑客，因此我们选择了这个主题，并开发了一个产品来帮助你保存移动数据。

其背后的想法很简单，打开移动数据，你请求下载一些东西，Android 上的意图过滤器监听下载，Android 应用程序获取文件的确切链接，它打包链接并将其发送到家中的下载框，连接到您的家庭宽带，下载框只包含一个带有 python 后端的树莓 Pi，它监听下载链接， 收到下载链接后，服务器应用程序使用这个名为 Aria2c 的库来下载正在讨论的特定文件，Aria 已经足够成熟，可以支持多种协议，如 HTTP、FTP、BitTorrent，即使如此，通过使用这个库，我们也可以抽象出应用程序的下载方面。 一旦文件位于本地的 Raspberry Pi 上，它会监听家庭 WiFi 网络上的新成员，一旦检测到 WiFi 表中的状态变化，它会自动尝试将文件发送到设备，如果是加载了客户端应用程序的 android 设备，它会意识到发出了检索文件的请求，并将文件发送到设备，自动将下载的内容无缝、毫不费力地添加到用户的图库中，并最终从本地 SD 卡中删除文件。

评委们很喜欢这个想法，并很快将其与谷歌专有的链接共享应用 pocket 联系起来，Pocket 允许你保存链接，并在你获得可靠的互联网连接时打开它们，我们的应用程序为你实时下载内容，并在你通过 WLAN 访问下载框时发送给你。

我们可以立即改进应用程序的一些方法是挤出 RPi 上的 4 个 USB 端口，并强制它作为一个大文件系统，从而允许用户在标准 SD 卡的大小不适合他们的消费时热插拔额外的存储。向该产品添加指标和分析将大大有助于用户分析他们消费的内容类型以及他们的大部分数据最终被用于何处。可以自动检测新加入者并随后注册下载框，以帮助缓解带宽流量，比如说，您和您的家庭成员都想要的视频可以只下载一次，并从下载框超本地提供，而不是下载两次并浪费带宽。

在一天结束时，这个应用程序是出于一种需要而构建的，当我们在家里有一个可以毫不费力地下载很多内容的应用程序时，我们中的一些住在宿舍的人最终会有非常慢的互联网连接，所以重点是构建这个应用程序，以便我们可以自己使用它，将所有必须下载的内容放在一个列表上并远程执行，当我们回家时，再次无缝地获取它。当你在课堂上冲浪，意识到最新一集的《权力的游戏》出来了，你不想使用你的移动数据下载它，但你使用我们的应用程序下载它，当你回家时，它已经下载了，并通过局域网同步，几乎是瞬间的，它就在你的手机里供你使用。

我们在这次黑客马拉松中获得了第二名，每人获得了一个奖杯和 25k 现金奖励。这是我们真正开始理解他们在黑客马拉松上所期望的错综复杂的东西，以及如何交付给那个特定的利基市场的起点。
