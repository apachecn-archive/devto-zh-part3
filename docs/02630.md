在这篇文章中，我们将为每一个编程新手列出 5 本[必读的](https://fullstackgeek.blogspot.com/p/must-reads.html)书。

**算法导论(东方经济版)(科尔曼)**

这是一本非常著名和非常有价值的书，通常被认为是数据结构和算法的圣经，因此是必读的。这本国际知名的教科书全面介绍了计算机算法的现代研究。它涵盖了广泛的算法深度，但使他们的设计和分析可访问的所有水平的读者。每章是相对独立的，并提出一个算法，一种设计技术，一个应用领域，或一个相关的主题。算法的描述和设计方式让任何稍有编程经验的人都能读懂。这些解释保持了简单性，但没有牺牲覆盖面的深度或数学的严谨性。

**数据结构和算法变得简单:数据结构和算法难题【Narasimha Karumanchi】**

Narasimha Karumanchi 是 CareerMonk Publications 的创始人，也是几本关于数据结构、算法和设计模式的书籍的作者。他是一名软件开发人员，在其漫长的职业生涯中，他既是面试官又是被面试者。最近，他为亚马逊公司、IBM 软件实验室、Mentor Graphics 和微软工作。Narasimha 拥有孟买 IIT 大学的计算机科学硕士学位和 JNT 大学的技术学士学位。他写了以下书籍，并被翻译成国际语言:中文，日文，韩国和台湾。此外，约有 58 所国际大学使用这些书籍作为学术课程的参考。

**使用 C 语言的数据结构**

使用 C 语言的数据结构的第二版已经被开发来提供数据结构的抽象概念以及使用 C 语言的这些概念的实现的全面和一致的覆盖。它首先全面概述了 C 编程的概念，然后介绍了不同的数据结构和方法来分析不同算法的复杂性。然后，它连接这些概念，并将它们应用于各种数据结构的研究，如数组、字符串、链表、堆栈、队列、树、堆和图形。

这本书利用了一种系统的方法，其中每个数据结构的设计都遵循可以对它们执行的不同操作的算法，以及根据它们的运行时间对这些算法的分析。

每章包括各种以 MCQs 形式的结尾章节练习，带有答案、复习题和编程练习，以帮助读者测试他们的知识。

在此购买:[从亚马逊购买](https://amzn.to/2WmhxYF)

数据结构和算法在 Java 中变得简单:数据结构和算法难题

对任何计算机科学专业人员来说都是一种方便的指南,《Java 中的数据结构和算法变得简单:数据结构和算法难题》是各种与数据结构和算法相关的复杂问题的解决方案库。它可以作为计算机科学行业的读者的参考手册。

这本书大约有 21 章，涵盖了递归和回溯、链表、栈、队列、树、优先级队列和堆、不相交集 ADT、图算法、排序、搜索、选择算法[中位数]、符号表、散列、字符串算法、算法设计技术、贪婪算法、分治算法、动态编程、复杂性类和其他各种概念。

Narasimha Karumanchi 的《数据结构和算法在 Java 中变得简单:数据结构和算法难题》于 2011 年出版，它是用 Java 语言编写的。这本书是准备面试、考试和校园工作的指南。它还提供 C/C++版本。简而言之，这本书提供了各种复杂的数据结构和算法问题的解决方案。
