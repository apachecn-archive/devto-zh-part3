## [](#all-hail-tqdm)All hail tqdm

tqdm 来自阿拉伯语“taqadum ”,意思是“进步”,它是众多 Python 库之一，既有用又易于使用。使用 pip 安装 tqdm，然后按照通常的方式进行安装:

我希望这个库能帮助你，希望你今天能学到一些东西。整个文档可在处[获得。](https://pypi.org/project/tqdm/)
