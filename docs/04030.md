# 回想我第一学期的教学

> 原文:[https://dev . to/renegade coder 94/reflecting-on-my-first-session-1e 91](https://dev.to/renegadecoder94/reflecting-on-my-first-semester-of-teaching-1e91)

*文章首次出现在[变节的程序员](https://therenegadecoder.com/teach/reflecting-on-my-first-semester-of-teaching/)上。*

如果你不知道，我现在是俄亥俄州立大学计算机科学和工程系的研究生助教。我在 12 月 5 日写了这篇文章，那是我教授 Java 入门课程的最后一天，当时我觉得试着回顾一下我第一学期的教学会很有趣。

## [](#logistics)物流

为了正确看待一切，我们先来谈谈物流。换句话说，让我们来谈谈时间、作业、学生人数以及诸如此类的事情。

Java 入门课程是一门 3 学分的课程，在大约 15 周的时间里每周一、三、四进行。我很幸运有一节下午课(下午 1:50-2:45 ),所以我的学生们经常很警觉，很投入。

在周一和周三，我和大约 40 名学生定期举行 55 分钟的讲座。与此同时，周四，我和这些学生以及另外 40 名来自另一个班级的学生进行了 55 分钟的实验。像 OSU 这样大规模的学校，肯定有同一个班级的其他部分。事实上，我相信总共有 17 节课，所以总共有大约 680 名学生在上同一门课。

除了教学，我给所有的作业评分，包括实验室、项目和家庭作业。为了客观地看待这个问题，我最终每周批改大约 120 份作业。除此之外，我还参加了三次考试:两次期中考试和一次期末考试。所有这些都是我自己评分的。如果你好奇的话，这是我第一门课期末成绩的高层次分类:

## [](#reflection)倒影

虽然上面所有的数字看起来都很好，但它们并不能让你真正了解我这一学期的实际情况。那么，我们来谈谈吧。

### [](#the-good)好人

我知道几个月前我提到过，我是一时兴起决定从事教学工作的——这要感谢一名实习生——但我对这个决定非常满意。事实上，我热爱教学。这是我重返校园唯一真正享受的部分。

也许这份工作最好的部分是看到学生们真正对学习感兴趣。尽管最近新闻里经常出现一些废话，但很高兴知道这个世界上仍然有聪明、勤奋的人。

此外，我真的很喜欢在教室里教授计算机科学。处于一个能对他人产生积极影响的位置总是感觉很好，我一周有几次这样的机会。

总的来说，我对这次经历感到惊喜，我希望我能很快再次这样做！

### [](#the-bad)坏了

老实说，我对这份工作只有两点不喜欢:评分和课程材料。现在，让我们谈一点关于分级的问题。

虽然我知道对学生来说获得好的反馈很重要，但我发现当活动如此势不可挡时，很难保持积极的态度。有时，我觉得我需要我自己的 GTA 来协助评分。

为了客观地看待我所做的评分，学生们被要求每周完成大约四项作业:一次测验、一次实验、一个项目和一次家庭作业。在这四项作业中，我给其中的三项打分，期望在下一项到期前完成。结果，我一周大约批改了 120 份作业。

考试的时候情况变得更糟，因为我还得给它们打分。通常，我会根据问题给所有的考试打分，这样我就可以限制每次考试的偏差。结果，我同时完成了所有考试的评分，这可能需要几个小时。

在任何给定的一周内，批改作业会占用我大约 10-15 个小时的时间。除此之外，我花了大约 3 个小时在教室，3 个小时在办公室，1 个小时在会议上。换句话说，我的大部分时间都花在了评分上，这并不理想。

### [](#the-ugly)丑陋

好吧，我撒谎了。这份工作还有其他我不喜欢的地方，但很少发生:抄袭和不尊重。

先说第一个，因为简单。当一个学生或一群学生决定从别处借用作品并声称是他们自己的作品时，就发生了剽窃。例如，假设我要求我的学生单独编写一个解决 Fizz Buzz 的方法。剽窃可能属于以下任何一种情况:

*   从网上资源或其他学生那里复制 Fizz Buzz 解决方案
*   稍微修改一下网上资源或其他学生的答案
*   作为一组学生编写代码

我相信还有无数其他的例子，但这些是我发现最常见的。

在过去的一个学期里，我注意到一些学生剽窃了他们的一些作品。幸运的是，他们都退了课，所以我不用处理任何学术诚信的过程。然而，我并不喜欢发现剽窃。作为一名教师，这确实有点伤人，因为你知道他们可以做得更好。当我看我喜欢的运动队表现不佳时，我也有同样的感觉。

另一件让我纠结的事情是某些学生的公然不尊重。我倾向于做一个好人，所以我喜欢时不时地让人们休息一下:

> 这学期过得不好吗？别担心。我能帮什么忙？

这就是我通常的工作方式。不幸的是，一些学生认为这是一个利用我的邀请。特别是，我有学生要求延期，只是为了在延期后交作业。其他学生翘了每一节课，但仍然期待着一个好的出勤率。显然，这些都不是我欣赏的行动。

## [](#anecdotes)轶事趣闻

至此，我想分享几个在我第一学期教学中起到积极作用的故事。

### [](#office-hours)办公时间

大多数时候，我觉得办公时间是一种浪费。毕竟，学生们通常只在临近截止日期和考试的时候，所以我花了大部分时间来评分。

当学生真的来了，他们经常想要作业的答案，他们等到最后一刻才开始。当然，我重视教学，所以我尽量避免分发解决方案。相反，我选择了[苏格拉底式方法](https://www.law.uchicago.edu/socratic-method),我会测试学生对主题的知识，试图引导他们找到可能的解决方案。

然而，为了节省时间，在指导学生解决问题和直接交给他们之间总是有一个平衡。毕竟，我有 40 个学生，对他们每个人单独使用苏格拉底式教学法会浪费很多时间。

幸运的是，一些学生会带着他们自己准备好的问题来找我。特别是，我有两个学生在期末考试前几个小时开始来到我的办公室。他们问了我一些关于他们所学的所有东西的内部运作的问题，尽管他们不会在大部分内容上进行测试。多酷啊。

很高兴看到学生们有永不满足的好奇心；他们想知道原因，而不仅仅是方法。他们没有问“我是如何做到的”，而是问了“为什么会这样？”例如，以下是他们的一些问题:

*   为什么我们要将字段设为私有？
*   上课有什么意义？我们为什么要使用它们？
*   静态方法和实例方法有什么区别？
*   try/catch 是如何工作的？我们什么时候会用到它？
*   捕捉和抛出异常有什么区别？

顺便说一下，如果这是一门理论性更强的课程，所有这些都会是很好的考试题目。不幸的是，我们倾向于教授解决某些类型问题的算法步骤，而没有教授这些算法背后的直觉。

为了了解上一堂只讲“如何”而不讲“为什么”的课是什么感觉，请考虑以下问题:你的家庭成员中有多少人讨厌公共核心数学，因为它不仅仅是教授算法？现在，想想他们中有多少人真正懂数学。我想他们中不会有很多人。

我想这就是为什么看到学生们真的想了解引擎盖下的东西是如何工作的令人耳目一新。

### [](#python)巨蟒

很多人可能都知道，我对 Python 有着爱恨交加的感情。这是一种非常漂亮和强大的语言，但是它在打包和可移植性方面有一些令人讨厌的缺点。也就是说，我很乐意教它。对我来说幸运的是，我上个学期。

我的一个学生碰巧得到了一份 Python 的编码工作，尽管他对这种语言没有任何经验。对他们来说幸运的是，我一直在那里，所以我完全有能力帮助他们解决困难。

首先，我尝试将 Python 与他们或多或少知道的一种语言 Java 进行比较。如果你熟悉这些语言，你会知道它们的不同之处可能多于相似之处，所以我从大的差异开始:

*   编译与解释
*   静态与动态类型
*   大括号与空白

之后，我们深入研究了他们正在进行的项目。我尽最大努力保留我对代码组织和实现的判断，同时帮助我的学生理解这一切。

总的来说，这很有趣！我们甚至聊了一会儿 Git，第一次总是很难。我希望他们一切都好。

除了到目前为止的一切，我还问了我的学生关于课程内容的问题。当然，几乎所有这些都不在我的控制之内，但我很好奇，想看看学生们是否有一些和我一样的感受。我尤其好奇学生们对以下陈述的想法:

*   课程内容组织有序，计划周密

但总的来说，我不确定如何理解这些结果。毕竟我和课程材料有点负面关系。例如，我认为它在实用性的名义下过多地关注了愚蠢的概念(如字符串格式)。具有讽刺意味的是，除了懒惰调试之外，我从未真正使用过字符串格式，但我确信它在某些地方很有用。

我也对材料的搭建方式提出了一些批评。特别是，在完全由对象组成的语言(Java)中，命令式编程受到了很多关注。如果不理解对象，很多语法看起来会很奇怪，我经常不得不回避问题以避免冗长的讨论(例如，什么是私有和公共关键字？).

最后，我觉得 Eclipse 的使用增加了一层复杂性，经常让学生感到沮丧。这就是为什么我经常提倡使用【Java 博士作为学习工具。我觉得当 Java 可以像脚本语言一样使用时，教 Java 就容易多了。

无论如何，让我们继续前进吧！

### [](#value)值

在这一点上，我要求每个学生分享对以下问题的开放式回答:这门课程的哪些方面最有用或最有价值？换句话说，我提出了一个完美的极简主义问题:这门课程是如何给你的生活带来价值的？下面，你会发现一些我最喜欢的:

#### [](#the-value-of-not-teaching-to-the-test)没有教到考试的价值

我最喜欢的一个回答谈到了学习的价值，而不是为了考试而教。

> 我真的很喜欢杰里米教我们的内容，所以我们实际上学习和理解它，而不仅仅是为了通过考试而教授。
> 
> 匿名，2018 年秋季

作为一名教师，我喜欢听到这种事情，但当我得到相反的批评时，我并不感到惊讶。毕竟，我们处在一个以度量为导向的社会。GPA 很重要。SAT，ACT，GRE 成绩很重要。显然，三个字母的缩写也很重要。

#### [](#the-value-of-a-course-website)一个课程网站的价值

另一个我最喜欢的回答是关于课程网站及其相关资源的价值。

> 课堂上复习过的幻灯片被贴在了课程网站上，这样以后就可以很容易地复习了。整个课程网站制作得非常好，包括教学大纲和附加信息。
> 
> 匿名，2018 年秋季

为课程协调员大声欢呼，为学生创造了如此伟大的资源。课程网站确实是组织和展示内容的好方法。

#### [](#the-value-of-handson-learning)动手学习的价值

正如另一名学生提到的，实验室是提供无风险学习的绝佳方式，可以轻松获得教师的帮助。

> 对我来说，项目、讲座和实验是这门课最精彩的部分。我特别喜欢这些实验室，因为很明显它们主要是为我们服务的。我们在实验室投入的是我们每周能从中获得的。我把它们作为一个机会来学习那些内容，并直接和我欣赏的助教一起工作。
> 
> 匿名，2018 年秋季

在这堂课中，我们举行了每周一次的实验，实验完成后或多或少会被评分。当然，我增加了一条额外的规则，强制学生两人一组工作，但是我认为实验室在任何一种情况下都工作得很好。显然，至少有一个学生有同样的感觉。

#### [](#the-value-of-transferable-skills)可转移技能的价值

这学期我试图教的一课是，这门课更多的是解决问题，而不是 Java。事实证明，我的一个学生坚持了这一理念。

> 事实上，我们学到的许多信息将有助于学习其他编程语言
> 
> 匿名，2018 年秋季

正如许多程序员所知道的，学习前两种语言很难，但最终技能会开始转移。我很高兴看到至少有一个学生已经在向前看了。

### [](#testimonials)奖状

除了上面的反馈，我还要求我的学生提交证明。下面，你会发现一些我最喜欢的:

#### [](#coding-tangents)编码切线

也许我最讨厌的品质之一——取决于你问的是谁——是我倾向于根据我当时发现的令人兴奋或有趣的事情从一个话题跳到另一个话题。自然地，这种愚蠢的行为可以在课堂上表现为一些学生所谓的“编码切线”(顺便说一下，播客名称正在申请专利)。对我来说幸运的是，看起来至少我的一个学生从这些编码切线中获得了一些价值。

> 我真的很喜欢杰里米的教学风格。[他]似乎真的喜欢教学，这似乎是一种罕见的现象。我觉得[他]试图教我，让我真正理解 Java，我认为这是最重要的事情。我喜欢[他]详述 java 在现实世界中如何使用的编码切线。
> 
> 托尼，2018 年秋季

既然托尼提到了，我确实认为很难找到真正喜欢教学的老师，这有点奇怪。人们怎么会误入一个他们甚至不喜欢的职业？

#### [](#patience)耐心

我已经培养的技能之一是耐心。老实说，我不确定这种技巧是从哪里来的，但我爸以前叫我禅师。显然，在 80 年代，人们发脾气要常见得多，老实说，他很惊讶我在成长过程中经历的许多情况下都能保持镇定。

事实证明，这种耐心似乎在我目前的讲师角色中得到了回报:

> 我认为格里夫斯基教授教得非常好。我在没有任何编程背景的情况下进入 Java，而我们班几乎一半的人已经有了。然而，他还是花时间仔细检查了每个细节，这让我很快跟上了进度。尽管我无数次通过电子邮件向他提问，他还是及时耐心地回答了。我也很欣赏他没有简单地给我们的作业打分，而是提供了评论，指出错误，以及如何在未来纠正和避免这些错误。
> 
> 匿名，2018 年秋季

我非常感谢学生们感谢他们收到的关于作业的反馈。有时，评分非常具有挑战性，所以我很高兴有些人从中获得了价值。

#### [](#constructive-feedback)建设性反馈

虽然很多评价都是正面的，但我确实收到了一些更实际的评价。例如，我的一个学生指责我过早引入复杂的话题，我认为这是事实:

> 考虑到这是杰里米的第一个学期，我真的很惊讶他教得这么好。虽然让一个教室一直参与显然不容易，但[他]总是鼓励我们独立思考并参与讲座。他总是把学生的最大利益放在第一位，我想我们都很欣赏这一点。[他]尽了最大努力简化这门课程有些复杂和令人困惑的本质，我对此表示感谢。就我个人而言，我是一名生物医学工程专业的学生，打算毕业后进入医疗领域，所以编码很可能不会成为我未来生活的一大部分。然而，当[杰里米]从[他的]职业和学术研究中讨论真实世界的例子时，我总是很欣赏。此外，虽然我个人并不总是感兴趣，但我认为当[他]比课程的某些方面讲得更详细时，这很酷。不过，在未来，我认为可能值得考虑将这些类型的讨论留到本学期晚些时候进行，以避免让学生感到困惑。总的来说，如果教学是杰里米一生的计划，那么我认为俄亥俄州立大学非常幸运有他。
> 
> 匿名，2018 年秋季

比起那句精彩的“俄亥俄州立大学很幸运拥有[他]”来结束一系列的褒奖，这是多么好的方式啊我很幸运被安排在这个位置上，我很兴奋下学期能回到这个位置上。

## [](#student-evaluation-of-instruction)学生对教学的评价

但是等等，还有呢！除了我提供给学生的反馈表之外，这所大学还开展了自己的学生教学评估(SEI)调查。我知道我已经挖掘了很多反馈，但结果刚刚出来(2018 年 12 月 18 日)。

在本次调查中，学生们被要求对以下 10 项与我的调查相似的陈述进行排名:

*   这门课的主题组织得很好
*   教练准备得很充分
*   讲师清楚地传达了主题
*   这位老师真的对教学感兴趣
*   这位老师真诚地想帮助学生
*   老师创造了一个有利于学习的氛围
*   这门课程在智力上很有启发性
*   老师鼓励学生独立思考
*   我从这位老师身上学到了很多东西
*   总的来说，我认为这位讲师……
