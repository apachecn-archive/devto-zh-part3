# 为什么 Python 适合创业公司？

早期创业公司有**有限的时间**、**预算**和对**灵活性的需求**。

每当[雷德·霍夫曼](https://en.wikipedia.org/wiki/Reid_Hoffman)描述一家初创公司时，我都喜欢用这个比喻。

> 把自己从悬崖上扔下去，然后组装一架飞机。

Python 开发人员提供这种能力，以有限的预算创建**最小可行产品(MVP)** ,以允许初创公司测试想法。

在你承诺为产品或服务的特定功能雇佣专业开发人员之前。

到那时，你的 python 开发者将已经奠定了基础，让你能够将用户数量扩大到像 [Disqus](https://blog.disqus.com/scaling-django-to-8-billion-page-views) 或 [Instagram](https://thenewstack.io/instagram-makes-smooth-move-python-3/) 这样的大规模用户

# 创建 MVP

以下是 Python 开发人员可以构建 MVP 来测试初创公司想法的 MVP 列表:

*   **物联网** -使用树莓 Pi &其他 python 支持微控制器创建硬件产品
*   **人工智能** -创造人工智能产品或服务，如聊天机器人、推荐引擎&自动驾驶汽车
*   Web 应用 -通过使用 Flask 或 Django web 框架
*   **Alexa 技能** -通过使用 [Flask 构建游戏、应用或教育 Alexa 技能-向亚马逊 Alexa 扬声器询问](https://flask-ask.readthedocs.io/en/latest/)
*   **数据产品或服务** -数据分析&数据可视化，如 [Plotly](https://plot.ly/) 使用行业数据生成仪表盘或行业报告。
*   **网络爬虫** -使用 [Scrapy](https://scrapy.org/) 或【美丽】从大量网站中抓取数据，以构建数据产品或服务
*   **软件即服务** -使用 Flask & [Django Rest 框架](https://www.django-rest-framework.org/)构建 API，或者定制类似 [Odoo](https://www.odoo.com/) 的 ERP 系统

# 小&精干队伍

当我想成为 Django 或 Flask 开发人员，而不是投身于数据科学领域时，最吸引我的是。

我有一个朋友是本地创业公司的 CTO。他建立了一个由 10 名开发人员组成的团队，他们的工作相当于 30 名开发人员的工作。

这增加了我成为 Django 开发者的动力，因为我正在为初创公司提供价值，并激发了我在未来建立自己的初创公司的愿望。

用一个更小的团队来开发一个产品或服务可以让你在更长的时间里保持活力。

# 那么哪些创业公司在用 Python 呢？

多年来，我知道大量的创业公司、公司或金融机构。

据我多年来在新加坡所知，他正在以某种形式使用 Python。

这是我所知道的新加坡初创公司的名单，他们雇佣懂 Python 的开发人员，专门开发 web 应用程序或数据科学相关的产品或服务。

*   [Microsec](https://www.usec.io/) - Microsec 为受限物联网设备构建安全解决方案，易于集成并能抵御网络攻击。
*   通过每天分析 100 多万份工作，他们提供了最好的劳动力市场洞察&工具
*   Artling 是一个在线艺术画廊，展示亚洲最好的画廊和艺术家的精选作品。
*   [Carousell](https://sg.carousell.com/) - Carousell 是一个智能手机和基于网络的消费者对消费者市场，用于买卖新的和二手商品。
*   [Shopee](https://shopee.sg/) - Shopee 是 Sea 集团(前身为 Garena)旗下总部位于新加坡的电子商务平台
*   StashAway -类似于 Wealthfront 或 Betterment 的机器人顾问平台。
*   [Evie AI](https://www.evie.ai/) - AI 助理，负责排班、招聘&销售
*   [Horangi](https://www.horangi.com/) - Horangi 制造安全产品，为缺乏规模、专业知识或时间的客户快速提供事件响应和威胁检测。

# 结论

除了以上关于为什么你应该使用 Python 来创建你自己的创业公司的原因之外。

我认为任何有抱负的企业家寻求建立自己的创业公司的另一个重要原因是。

Python 对于没有任何 STEM 背景的人来说[很容易上手](https://cacm.acm.org/blogs/blog-cacm/176450-python-is-now-the-most-popular-introductory-teaching-language-at-top-u-s-universities/fulltext)。

拥有一个[生态系统](https://www.jetbrains.com/research/devecosystem-2018/python/)，允许你插入&游戏技术来建立你的 MVP。

尤其是当你没有一个[技术联合创始人](https://medium.freecodecamp.org/non-technical-and-looking-for-a-technical-co-founder-2c212c01d6da)来帮你打造一个 MVP 的时候。

这实际上是我多年来在新加坡创业生态系统中遇到的一个普遍趋势。
