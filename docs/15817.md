大量的选项/选择可能会导致[决策疲劳](https://www.intelligenteconomist.com/decision-fatigue/)。这是很多初学者浪费时间的地方，有些人最终放弃了学习这项技能。也许他们中的一些人成功地克服了决策疲劳，选择了一个学习材料，中途他们发现这并没有给他们所寻求的知识，这又是他们可能放弃的另一点。

在我学习 Javascript 的过程中，我过滤好坏的技术是检查与我从课程/书籍中寻找的内容相关的正面评论。这将有很大的帮助，但这并不保证它会提供你想要的。

所以在这篇文章中，我将分享帮助我学习 Javascript 的学习材料(书籍和课程)。

### [](#codeacademys-learn-javascript-course-free)[Codeacademy 的学习 Javascript 课程](https://www.codecademy.com/learn/learn-javascript)(免费)

Codeacademy 是一个用不同语言教授编程的交互式网站。我喜欢他们教 Javascript 的方法，因为他们解释主题，然后给出容易理解的例子，然后在你进入下一个主题之前完成任务，它建立在你以前的知识基础上。如果你刚开始学习编程，我强烈建议你学习这门课程，因为它是为初学者量身定制的，可以让你熟悉基础知识。

### [](#head-first-javascript-programming-37)[头头 Javascript 编程](https://www.amazon.com/Head-First-JavaScript-Programming-Brain-Friendly/dp/144934013X)(37 美元)

这是我的首选参考书籍，因为有很多例子和插图来传递信息。通常我不喜欢用书来学习，因为我厌倦了书，而且大多数书都不使用基于项目的方法。Head first javascript 编程不是基于项目的，但是他们用大量的例子和谜题来帮助你熟悉 javascript。

### [](#javascript-and-jquery-interactive-frontend-web-development-jon-duckett-34)[JavaScript 和 JQuery:交互式前端 Web 开发——乔恩·达克特](https://www.amazon.com/JavaScript-JQuery-Interactive-Front-End-Development/dp/1118531647/ref=sr_1_4?s=books&ie=UTF8&qid=1547707038&sr=1-4&keywords=jon+duckett)(34 美元)

这是一本独特的书，因为它从一开始就参与进来，构建不同的东西。这本书给了我一次独特的经历，因为在这之前我还没有真正用 Javascript 做过任何东西，而且你也将会学到一点 Jquery。我不能再推荐这本书了，如果在我列出的四种材料中有一种是你应该使用的，那么这就是了。我建议你把这本书从头到尾读一遍。

### [](#watch-and-code-practical-javascript-free)[手表和代码-实用 javascript](https://watchandcode.com/p/practical-javascript) (免费)

这是一个视频教程，带你构建一个待办事项应用程序(是的，另一个待办事项应用程序)。这是最好的免费 Javascript 视频教程之一。讲师 Gordon Zhu 介绍了作为专业开发人员会经常遇到的 Javascript 构建模块。在构建 todo 应用程序时，他涵盖了函数、数组、对象和循环。最棒的是，它并不全是关于代码的，他也给出了关于重构、如何作为初学者解决问题以及如何调试的提示。完成本课程后，你应该准备好选择任何 Javascript 框架(😉反应过来)继续学习。

我要建议的一件事是不要担心记住语法，但要确保你理解语法背后的概念，你可以随时谷歌语法或前往 Stackoverflow。
