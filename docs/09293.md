## 背景

首先介绍一下我的背景，我是[制造技术](https://www.madetech.com/)公司的工程师，专攻[现场可靠性工程](https://landing.google.com/sre/books/)。我的工作是在三大云提供商中的两个上支持我们的软件工程师同事:AWS 和 Azure。我负责供应运行应用程序的基础架构，设置连续交付管道和应用程序监控。

今年，我决定加强我在 AWS 方面的能力，一个很好的方法就是成为 AWS DevOps 专业人员。

我尝试过的学习方法是写博客，这也是我为什么要写这个博客系列的原因！此外，我通过写日记来对我的学习负责。

## 认证

在我们开始之前，让我们明确一下我们指的是哪个版本的考试。这是 2019 年 2 月以后参加的认证考试。

AWS DevOps 专业认证费用为 300 美元，需要 170 分钟完成，评分系统为 100-1000 分:您需要 750 分才能通过。

在这个版本的考试中，他们不再提供问题的数量，但以前是 80 个问题，所以每个问题大约 2 分钟。

考试中的问题信息密集。有两种问题格式:

*   多项选择-选择一项作为正确答案。
*   多选题-两个或两个以上的选择组成正确的答案。

如果你想知道有多密，看看免费的[样题](https://d1.awsstatic.com/training-and-certification/docs-devops-pro/AWS%20Certified%20DevOps%20Engineer%20-%20Professional_Sample%20Questions.pdf)(不能提供例子，因为是版权材料)。

去看看，做完了再回来，我等着。

除非你已经是一个 DevOps 专业人员或使用 AWS 很长时间了，否则你可能会感到震惊。

在我们等待震动平息的同时，让我们覆盖你将被测试的区域。

## 知道什么时候该参加考试

我没有特别的信心能通过考试，也就是说，不用学习。考试费用是 350 美元，所以我不想花那么多钱(即使是由我的雇主支付)去失败。

下一个最好的事情是尝试练习考试，花费 40 美元，有 20 个问题，你需要在一个小时内回答。通过 [AWS 认证网站](https://www.aws.training/certification?src=cert-prep)预约实践考试。这与预订实际考试的方法相同，只是您可以在线参加模拟考试，而实际考试要求您访问考试中心。

我参加了模拟考试，成绩是 40%(

我的学习计划是:

*   创建一系列博客文章
*   使用考试中涵盖的服务进行大量实际练习
*   对照我将作为学习计划的一部分制作的一套抽认卡进行练习

我暂时给自己一个月的时间学习，每天至少花两个小时。之后我会重新参加模拟考试，如果我的分数至少达到 80%，我会考虑预约考试。

敬请关注。
