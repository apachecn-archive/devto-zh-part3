# 从芭蕾短裙到科技

> 原文:[https://dev.to/karaluton/nevertheless-kara-coded-2i78](https://dev.to/karaluton/nevertheless-kara-coded-2i78)

如果三年前你告诉我，我会成为一家科技公司的软件工程师，我会大笑。生活有时很有趣，所以我想利用国际妇女节这个机会展示我是如何进入这个职业的，并希望激励其他试图做同样事情的女性。

我是跳芭蕾长大的。17 岁时，我独自搬到纽约市，在乔佛里芭蕾舞学校接受全日制训练。我热爱我在那里的时光，无论如何我都不会用它来交换，但这让我怀疑我是否想成为专业的芭蕾舞演员。我知道，如果我的芭蕾生涯能持续到 30 岁，那我将是幸运的，所以我决定去上大学，把芭蕾放在次要位置。

[![Me at Joffrey Ballet School](../Images/b36b3bed474f019e45d8383c725bb2e0.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--VM70z_C5--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/crkxlvjj612abyk1ml3m.jpg)

我搬回了家乡田纳西州的纳什维尔，在贝尔蒙特大学注册了公共关系专业。我很难放弃芭蕾。我甚至在当地专业芭蕾舞团做过一段时间的客座艺术家，但随着我开始更加专注于学习芭蕾，这种感觉开始消失。

我在大学里非常努力。我想在音乐行业工作，我知道我必须让自己脱颖而出，所以我在大一后的那个夏天开始实习。毕业后，我在一家音乐营销公司找到了一份宣传助理的工作，这一切都得到了回报。我热爱我的工作，一年后我被提升为公关人员。我去听音乐会，和一些知名人士出去玩，偶尔去红毯，但我也真的感到压力很大。

我的整个工作都依赖于一名记者，他每天会收到*数千封*电子邮件来打开我的邮件，并同意接受采访或报道。我讨厌这种感觉，即我的工作做得有多好完全取决于有人打开电子邮件。当我偶然发现[代码学院](https://www.codecademy.com)时，我开始寻找其他职业选择。

我开始自学 HTML 和 CSS，并且非常喜欢它。我在高中和大学时非常擅长外语，这感觉就像学习另一种语言。就在那时，我决定我真的想从事科技领域的工作。我在乔佛里培训时完成了高中在线课程，所以我知道自学路线不适合我。我开始寻找其他选择，并找到了铁场，一个为期三个月的训练营。

[![My cohort at The Iron Yard](../Images/c7b91ee52cb981a72f9ae2b6e66107dc.png)T2】](https://res.cloudinary.com/practicaldev/image/fetch/s--a7yRQpqW--/c_limit%2Cf_auto%2Cfl_progressive%2Cq_auto%2Cw_880/https://thepracticaldev.s3.amazonaws.com/i/lhvuq4grz242jlusiwqi.jpeg)

在我的训练营中，我学习了所有关于 HTML、CSS 和 JavaScript 的知识，然后专注于学习 [Vue](https://vuejs.org/) 和 [Ember](https://www.emberjs.com/) 。对我来说，在这个行业里尽可能多地认识人也很重要，所以我会在晚上去参加聚会。我非常喜欢参加聚会，以至于当我还在训练营的时候，我就成立了当地的[科技女性](https://www.hiretechladies.com/)纳什维尔分会。我做了很多杂耍，但最终都得到了回报，我在镇上的一家机构找到了我的第一份开发工作。

自从我从训练营毕业以来，经历了很多起起落落，但这确实是我一生中最好的决定之一。有时候我希望我能早点发现编程，但是如果没有走上我走过的路，我也不会成为现在的我，对此我非常感激。

## [](#i-startedcontinued-to-code-in-2019-because)我在 2019 年开始/继续编码是因为...

我喜欢这条职业道路。我喜欢在不知道是什么原因导致错误后修复错误的感觉。我喜欢看到我正在做的网站从一无所有开始，因为我写的东西而变得生机勃勃。

## [](#i-deserve-credit-for)我应得的功劳...

即使在人们怀疑我的时候，我也在追求这条职业道路。当我告诉我的老同事我要离开去学习如何编码时，有人告诉我的第一句话是“但是你看起来不像是编码的人。”我没有让这影响到我，事实上，这给了我更大的压力。

## [](#i-hope-to-see-my-schoolworkdevelopertech-community)我希望看到我的学校/工作/开发者/技术社区...

拥抱那些来自不同背景的人。不要因为某人几个月前没有获得 CS 学位或者没有学会如何编程就认为他不存在。那个人会给你的团队带来完全不同的观点。给他们一个机会。
