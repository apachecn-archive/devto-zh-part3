# 我创建了一个不和谐频道，旨在让开发人员聚在一起/想出一个主意/一起做一个项目。

> 原文：<https://dev.to/pilatdennis/i-created-a-discord-channel-meant-for-developers-to-come-togetherthink-of-an-ideawork-on-a-project-together-3ge3>

你好戴夫.托。

对我来说，总是很难找到志同道合的开发人员，或者找到一个地方，只是为了思考一个想法并一起工作。正因为如此，开发商 Grove 应运而生！我创建了一个不和谐频道，旨在让开发人员聚在一起，思考想法，并作为一个项目一起工作。它在 3 天前开始，现在我们有超过 120 名成员！聊天中许多人告诉我，他们也一直在寻找这样的东西。我真的很高兴我不是唯一一个有这种感觉的人，并且知道这个想法已经被验证了。

我想邀请 Dev.to 社区也加入这个不和谐频道。

下面是不和谐频道的链接:[https://discord.gg/29TK6DW](https://discord.gg/29TK6DW)

来介绍一下自己吧，让别人知道你有什么项目想法，感兴趣，看看你自己要不要加入什么项目！我们会在小树林等你！:)
