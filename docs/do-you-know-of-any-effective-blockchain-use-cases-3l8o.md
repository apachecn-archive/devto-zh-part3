# 你知道任何有效的区块链用例吗？

> 原文：<https://dev.to/ardennl/do-you-know-of-any-effective-blockchain-use-cases-3l8o>

在目前的时间点上，区块链看起来非常像一个解决方案，我们正在拼命寻找与之匹配的问题。在最近发布的来自 [MERLtech](https://twitter.com/MERLTech) 研究人员的报告中，解释说他们找不到任何数据支持区块链技术优于现有解决方案的有效性。

虽然这似乎不是最广泛的研究，但它确实让我想知道是否有任何关于区块链项目的统计数据，这些项目要么
a)解决了在区块链技术
之前无法解决的问题，要么 b)解决问题的方式比使用成熟技术解决问题的方式更有效(无论是在生产力、规模、信任还是其他方面)。
